<h1>שיר השירים - מצודת ציון</h1>
א ( א ) שיר השירים . רוצה לומר שיר המעולה והמובחר שבכל השירים . וכן ותבואי בעדי עדיים ( יחזקאל טז ז ) , ורוצה לומר היפה והמובחר שבעדיים :
<h3>שיר השירים פרק-א</h3>
<h4>{ב}</h4>
<b>דודיך .</b> ענין אהבה וחיבה . כמו נרוה דודים ( משלי ז יח ) :
<h4>{ג}</h4>
<b>לריח .</b> בעבור ריח . כמו פתח פיך לאילם ( שם לא ח ) : <b>תורק .</b> ענין הזלה ושפיכה . וכן על הארץ יריקו ( קהלת יא ג ) : <b>עלמות .</b> נערות . כמו והיה העלמה ( בראשית כד מג ) :
<h4>{ד}</h4>
<b>נרוצה .</b> ענין מהירות ההליכה , ובא בלשון רבים ליחיד לתפארת . וכן ונתנה לך גם את זאת ( שם כט כז ) : <b>הביאני .</b> עבר במקום עתיד וכמוהו רבים במקרא : <b>מישרים .</b> מלשון ישר וראוי :
<h4>{ה}</h4>
<b>ונאוה .</b> מלשון נאה ויופי . כמו נאוה כירושלים ( לקמן ו ד ) : <b>כיריעת .</b> ענין וילון :
<h4>{ו}</h4>
<b>תראני .</b> ענינו הסתכלות דרך בזיון . וכן לראוה בך ( יחוקאל כח וז ) : <b>שחרחורת .</b> מלשון שחרות בהכפל העי''ן והלמ''ד ויורה על רוב השחרות . וכן ירקרק או אדמדם ( ויקרא יג מט ) : <b>ששזפתני .</b> ענין ראיה והבטה . כמו שזפתו עין ( איוב כח ו ) : <b>נחרו .</b> מלשון חרון אף וחמה : <b>נוטרה .</b> ענין שמירה . כמו הינטור לעולם ( ירמיה ג ה ) :
<h4>{ז}</h4>
<b>איכה .</b> הוא כמו אנה . וכן איכה הוא ( מ''ב ו יב ) : <b>תרביץ .</b> ענין השכיבה לנוח : <b>בצהרים .</b> מלשון צוהר ואורה והוא חצי היום : <b>שלמה .</b> אשר למה , השי''ן הוא במקום אשר : <b>כעטיה .</b> ענין הפרחה ושטיפה . כמו ותעט אל השלל ( ש''א טו יט ) :
<h4>{ח}</h4>
<b>בעקבי .</b> רוצה לומר במדרך עקבות רגל . וכן ועקבותיך לא נודעו ( תהלים עז כא ) : <b>ורעי .</b> מלשון מרעה : <b>גדיותיך .</b> מלשון גדי עזים : <b>על .</b> ממעל : <b>משכנות .</b> מלשון משכן :
<h4>{ט}</h4>
<b>לסוסתי .</b> נקבת הסוס תקרא סוסיה , כאשר תקרא נקבת החמור חמורה , והיו''ד נוספת , ובעבור תוספת הי' נהפכה הה''א לתי''ו להסמיכה אל היו''ד . וכן מלאתי משפט ( ישעיה א כא ) , שבעבור תוספת היו''ד נהפכה הה''א לתי''ו . ואמר בלשון נקבה , לפי שדרך המלך והשרים היה לקשור במרכבתם סוסיות נקבות , כי בסוסים זכרים יקשה על הרכב לעצרם ללכת לאט לבל ירוצו : <b>פרעה .</b> כל מלכי מצרים קרואים בשם פרעה : <b>דמיתיך .</b> מלשון דמיון ומשל : <b>רעיתי .</b> חברתי :
<h4>{י}</h4>
<b>נאוו .</b> מלשון נאה ויופי : <b>לחייך .</b> הם הלסתות אצל העינים . וכן בשבט יכו על הלחי ( מיכה ד יד ) : <b>בתורים .</b> הם הפנינים המסודרים בתואר נאה והכונה יפה וקרוב לו כתור האדם המעלה ( דהי''א וז יז ) , ורוצה לומר כתואר ותכונת האדם המעלה : <b>בחרוזים .</b> הם אבנים טובות ומרגליות נקובות מחוברות בחוט , ובדברי רז''ל מחרוזות של דגים ( ב''מ כא : ) :
<h4>{יא}</h4>
<b>נעשה .</b> פעמים דרך היחיד לדבר בלשון רבים לתפארת :
<h4>{יב}</h4>
<b>עד .</b> בעוד . וכן עד היותי על אדמתי ( יונה ד ב ) : <b>במסבו .</b> דרכו היה להסב ולהשען על המטות בשעת האכילה : <b>נרדי .</b> מין בושם :
<h4>{יג}</h4>
<b>צרור .</b> ענין קשור . כמו צרור כספו ( בראשית מב לה ) : <b>המור .</b> מין בושם : <b>דודי .</b> אהובי : <b>שדי .</b> הם דדי האשה : <b>ילין .</b> ענין התמדה . כמו צדק ילין בה ( ישעיה א כא ) :
<h4>{יד}</h4>
<b>אשכול .</b> כן יקרא הענף והגרגרים תלוים בו . וכן אשכול ענבים ( במדבר יג כג ) : <b>הכופר .</b> מין בושם : <b>בכרמי .</b> הוא מושאל לפרדסים בעלי אילנות . וכן כרם זית ( שופטים טו ה ) : <b>עין גדי .</b> שם מקום בארץ ישראל :
<h4>{טו}</h4>
<b>הנך .</b> הנה את . כמו הנך הרה ( בראשית טז יא ) :
<h4>{טז}</h4>
<b>נעים .</b> אהוב וחביב . כמו נטעי נעמנים ( ישעיה יז י ) : <b>ערשנו .</b> כן יקרא המטה . כמו ערש יצועי ( תהלים קלב ג ) : <b>רעננה .</b> ענין לחות ורטיבות כמו דשנים ורעננים ( שם לב ) :
<h4>{יז}</h4>
<b>קורות .</b> כן יקרא העץ העב . וכן ונקחה משם איש קורה אחת ( מ''ב ו ב ) : <b>רהיטנו .</b> הם הבנינים שבעליית הבתים אשר בהם ילכו וירוצו מחדר לחדר . וכן מלך אסור ברהטים ( לקמן ז ו ) , כי התרגום של וירץ הוא ורהט ובלשון הזה יקרא גם שקתות המים . כמו שנאמר ותמלאנה את הרהטים ( שמות ב טז ) , וזה על שם מרוצת המים שבהן : <b>ברותים .</b> כמו ברושים בשי''ן והוא שם אילן סרק חשוב :
<h3>שיר השירים פרק-ב</h3>
<h4>{א}</h4>
<b>חבצלת .</b> הוא הורד המריח : <b>השרון .</b> הוא מקום שמן ודשן . כמו והיה השרון כערבה ( ישעיה לג י ) : <b>שושנת .</b> שם צמח המריח ונקראת כן על כי היא לעולם בעלת ששה עלין :
<h4>{ב}</h4>
<b>החוחים .</b> מין קוצים . כמו חוח עלה ביד שכור ( משלי כו ט ) :
<h4>{ג}</h4>
<b>לחכי .</b> החיך הוא מה שממעל להלשון :
<h4>{ד}</h4>
<b>הביאני .</b> עבר במקום עתיד : <b>ודגלו .</b> מלשון דגל שמרימין בצבא :
<h4>{ה}</h4>
<b>באשישות .</b> הם הלגינים ישימו בהם היין . כמו אשישי ענבים ( הושע ג ב ) : <b>רפדוני .</b> ענין הצעת המטה . כמו רפדתי יצועי ( איוב יז יג ) :
<h4>{ו}</h4>
<b>תחבקני .</b> ענין סבוב הזרוע בגוף :
<h4>{ז}</h4>
<b>השבעתי .</b> מלשון שבועה : <b>בצבאות .</b> נקבות הצבאים : <b>באילות .</b> נקבות האילים : <b>תעירו תעוררו .</b> מלשון התעוררות , וכפל המלה לחזק הענין :
<h4>{ח}</h4>
<b>הנה זה .</b> הלשון הזה יורה על מהירות הענין . וכן והנה זה מלאך נוגע בו ( מ''א יט ה ) : <b>מדלג .</b> מקפץ . שניהם ענין רקוד ומהירות ההליכה והכפל במלות שונות יורה על מרבית המהירות :
<h4>{ט}</h4>
<b>דומה .</b> מלשון דמיון : <b>לעופר כן .</b> יקרא ילד האיל . וכן יקרא ילד הצבי . כמו שנאמר כשני עפרים תאומי צביה ( לקמן ד ה ) : <b>משגיח .</b> ענין הבטה בהתבוננת רב : <b>מציץ .</b> מלשון נצץ והוא גלוי הפרח רוצה לומר מגלה עצמו להיות נראה ונשקף : <b>החרכים .</b> תרגום של חלון חרכא :
<h4>{י}</h4>
<b>ענה .</b> ענין הרמת קול . כמו וענו הלוים כו' קול רם ( דברים כז יד ) :
<h4>{יא}</h4>
<b>הסתיו .</b> זמן החורף כי וקיץ וחורף ( בראשית ח כב ) , תרגם אונקלוס וקיטא וסתוא : <b>חלף .</b> עבר . כמו וחלף ביהודה ( ישעיה ח ח ) :
<h4>{יב}</h4>
<b>הנצנים .</b> ענין פרח . כמו עלתה נצה ( בראשית מ י ) : <b>הזמיר .</b> מלשון זמר ושיר : <b>התור .</b> הוא שם עוף :
<h4>{יג}</h4>
<b>חנטה .</b> יציאת הפירות והראותן קודם גמר בישולן תקרא חנטה . ובדברי רז''ל אילן שחנטו פירותיו ( ר''ה טו . ) : <b>פגיה .</b> הם התאנים שלא בשלו כל צרכן . ובדברי רז''ל פגה שטמנה ( שבת קכג . ) : <b>סמדר .</b> הם הענבים הדקים הניכרים אחר שנפל הפרח . וכן פתח הסמדר ( לקמן ז יג ) : <b>לך .</b> רוצה לומר להנאתך :
<h4>{יד}</h4>
<b>בחגוי .</b> ענין בקוע וחריץ . כמו שוכני בחגוי סלע ( עובדיה א ג ) : <b>המדרגה .</b> ענין מעלות . כמו ונפלו המדרגות ( יחזקאל לח כ ) : <b>ערב .</b> ענין מתיקות ונעימות . כמו וערבה לה' מנחת ( מלאכי ג ד ) : <b>נאוה .</b> נאה ויפה :
<h4>{טו}</h4>
<b>שועלים .</b> שם חיה מה : <b>מחבלים .</b> משחיתים . כמו וחובל עול ( ישעיה י כז ) :
<h4>{יז}</h4>
<b>שיפוח .</b> מלשון הפחה ונשיבה : <b>היום .</b> וכן יקרא השמש . כמ ולקוח היום ( בראשית ג ח ) , וזהו לפי שבעוד השמש על הארץ הוא יום : <b>ונסו .</b> מלשון ניסה ובריחה : <b>הצללים .</b> מלשון צל : <b>סב .</b> מלשון סביב : <b>דמה .</b> מלשון דמיון : <b>לעופר .</b> הוא ילד האיל : <b>בתר .</b> ענין חלוקה והפלגה . כמו ויבתר אותם בתוך ( בראשית טז י ) : <b>ג ( ב ) בשוקים וברחובות .</b> כפל המלה בשמות נרדפים . וכן אדמת עפר ( דניאל יב ב ) :
<h3>שיר השירים פרק-ג</h3>
<h4>{ד}</h4>
<b>כמעט .</b> באה הכ''ף לאמתת הדבר . וכן מכרה כיום ( בראשית כה לא ) : <b>ארפנו .</b> מלשון רפיון : <b>הורתי .</b> מלשון הריון . ורוצה לומר אמי שהרתה אותי והוא כפל ענין במלות שונות :
<h4>{ה}</h4>
<b>תעירו תעוררו .</b> מלשון התעוררות :
<h4>{ו}</h4>
<b>כתמרות .</b> מלשון אילן תמר , ורוצה לומר עמוד עשן זקוף כאילן תמר . וכן ותמרות עשן ( יואל ג ג ) : <b>מקטרת .</b> הבערת הבשמים להריח קרוי קטרת בלשון מקרא : <b>מור ולבונה .</b> שמות מיני בושם : <b>אבקת .</b> רוצה לומר בשמים השחוקים דק דק כאבק . וכן יקרא עפר הדק . כמו יסכך אבקם ( יחזקאל כו י ) : <b>רוכל .</b> כן יקרא מוכר הבושם . כמו הצורפים והרוכלים ( נחמיה ג לב ) :
<h4>{ח}</h4>
<b>מלמדי .</b> ענין הרגל . כמו וילמד לטרוף טרף ( יחזקאל יט ו ) : <b>על ירכו .</b> על צדו :
<h4>{ט}</h4>
<b>אפריון .</b> הוא המטה שנושאין בה הכלות . ובדברי רז''ל הושיבה באפריון ( סוטה יב . ) . וכן קראו רז''ל למטה פוריה , ואמרו על שם שפרין ורבין עליה ( כתובות י : ) : <b>הלבנון .</b> שם יער בארץ ישראל :
<h4>{י}</h4>
<b>רפידתו .</b> ענין הצעה . כמו רפדוני בתפוחים ( לעיל ב ה ) : <b>ן מרכבו .</b> מלשון רכיבה : <b>ארגמן .</b> שם צבע חשוב : <b>רצוף .</b> ענין גחלת אש , ובידו רצפה ( ישעיה ו ו ) . ודרף המקרא לכנות הפלגת האהבה באש הבוער ומתלהב . וכן רשפיה רשפי אש שלהבת יה ( לקמן ח ו ) :
<h4>{יא}</h4>
<b>בעטרה .</b> בכתר : <b>חתונתו .</b> מלשון חתן :
<h3>שיר השירים פרק-ד</h3>
<h4>{א}</h4>
<b>הנך .</b> הנה את : <b>יונים .</b> כיונים : <b>מבעד .</b> בפנים כי בעד ענינו דבר המפסיק . כמו הארץ בריחיה בעדי ( יונה ב ז ) , ומבעד הוא הדבר שהוא מפנים לאותו בעד המפסיק : <b>לצמתך .</b> הוא המסוה המכסה את הפנים . כמו גלי צמתך ( ישעיה מז ב ) , והוא יריעה דקה מאד ומריאת הפנים נראה דרך בה : <b>שגלשו .</b> ענין מריטת השער כי גבח הוא ( ויקרא יג מא ) , תרגום אונקלוס גליש הוא :
<h4>{ב}</h4>
<b>הקצובות .</b> ענין חיתוך וכריתה . כמו מדה אחת וקצב אחד ( מ''א ו כה ) : <b>מתאימות .</b> מלשון תאומים : <b>ושכלה .</b> רוצה לומר חסרון ופגם ואמר בלשון שכלה מלשון הנופל בבעלי חיים הנשכל מחיה רעה :
<h4>{ג}</h4>
<b>כחוט השני .</b> פתיל צבוע אדום . כמו ושני תולעת ( במדבר יט ו ) : <b>ומדברך .</b> דבורך : <b>נאוה .</b> נאה ויפה : <b>כפלח .</b> ענין בקוע . כמו כפלח תחתית ( איוב מא טז ) : <b>רקתך .</b> הוא גובה הפנים שאצל העין מזה ומזה ויקרא כן על מיעוט ורקות הבשר אשר בהם . וכן ותתקע את היתד ברקתו ( שופטים ד כא ) :
<h4>{ד}</h4>
<b>לתלפיות .</b> ענין הוראה ולמוד . כמו מלפנו מבהמות ארץ ( איוב לה י ) , והתי''ו שבתפליות הוא . כמו תי''ו שבתרמית ובתבנית : <b>שלטי .</b> הוא המגן . וכן את שלטי הזהב ( דה''א יח ז ) :
<h4>{ה}</h4>
<b>שדיך .</b> הם דדי האשה : <b>עפרים .</b> כן יקראו ילדי הצביה והאילה :
<h4>{ו}</h4>
<b>שיפוח .</b> מלשון הפחה ונשיבה : <b>היום .</b> השמש : <b>ונסו .</b> מלשון ניסה ובריחה : <b>הצללים .</b> מלשון אצל : <b>המור , הלבונה .</b> שמות מיני בושם :
<h4>{ח}</h4>
<b>מלבנון .</b> שם יער : <b>כלה .</b> תקרא כן על כי אז היא כלולה כל מיני יופי ופאר : <b>תשורי .</b> ענין צעדה והלוך . כמו באשורו אחזה רגלי ( איוב כג יא ) : <b>אמנה , שניר וחרמון .</b> שמות הרים : <b>ממעונות .</b> ענין מדור . כמו ה' מעון אתה היית ( תהלים צ א ) :
<h4>{ט}</h4>
<b>לבבתני .</b> מלשון לב : <b>ענק .</b> כן יקרא עדי הצואר . כמו וענקים לגרגרותיך ( משלי א ט ) :
<h4>{י}</h4>

<h4>{י}</h4>

<h4>{יא}</h4>
<b>כתם פז .</b> זהב טוב ויקר . כמו בכתם אופז ( דניאל י ה ) : <b>קווצותיו .</b> הוא שער הראש התלוי למטה : <b>תלתלים .</b> מלשון תל וגובה : <b>כעורב .</b> שם מין עוף :
<h4>{יב}</h4>

<h4>{יג}</h4>

<h4>{יד}</h4>
<b>גלילי .</b> אופני המרכבה . כמו על גלילי כסף ( אסתר א ו ) : <b>בתרשיש .</b> שם אבן יקר : <b>מעיו .</b> על הבטן יאמר שבו המעים : <b>עשת .</b> ענין בהירות וזכות . וכן ברזל עשות ( יחזקאל חז יט ) : <b>שן .</b> היא שן הפיל : <b>מעולפת .</b> ענין עטוף וכסוי . כמו ותתעלף ( בראשית לח יד ) :
<h4>{טו}</h4>
<b>שוקיו .</b> הוא פרק מפרקי הרגל : <b>שש .</b> אבן שיש : <b>מיוסדים .</b> מלשון יסוד : <b>אדני .</b> הוא הדבר שהעמור תקוע בו . וכן אדני כסף ( שמות כו יט ) : <b>פז .</b> הוא המובחר שבזהבים :
<h4>{טז}</h4>

<h4>{ה}</h4>

<h4>{ו}</h4>
<b>מתאימות .</b> מלשון תאומים : <b>ושכולה .</b> רוצה לומר חסרון ופגם ואמר בלשון השאלה :
<h4>{ז}</h4>
<b>כפלח .</b> ענין בקוע . כמו כפלח תחתית ( איוב מא טז ) : <b>רקתך .</b> הוא גובה הפנים שאצל העין מזה ומזה . וכן ותתקע את היתד ברקתו ( שופטים ד יא ) : <b>מבעד .</b> מבפנים : <b>לצמתך .</b> היא המסוה המכסה את הפנים . וכן גלי צמתך ( ישעיה מז ב ) :

<h4>{י}</h4>
<b>הנשקפה .</b> ענין ראיה והבטה . כמו בעד החלון נשקפה ( שופטים ה כח ) : <b>שחר .</b> כוכב השחר : <b>איומה .</b> מלשון אימה ופחד : <b>כנדגלות .</b> מלשון דגל של מחנות עם :

<h4>{ב}</h4>
<b>פעמיך .</b> כן יקראו הרגלים . וכן פעמי דלים ( ישעיה כו ו ) : <b>בנעלים .</b> מלשון מנעול : <b>חמוקי .</b> ענין סבוב והיקף . כמו חמק עבר ( לעיל ה ו ) : <b>ירכיך .</b> מלשון ירך והוא מפרקי הרגל : <b>חלאים .</b> ענין תכשיט . כמו וחלי כתם ( משלי כה יב ) : <b>אמן .</b> כן יקרא המופלג במלאכה :
<h4>{ג}</h4>

<h4>{ה}</h4>
<b>השן .</b> שן הפיל : <b>ברכות .</b> מקום בנוי באבנים ובסיד ושם מתכנסים המים וכן בתעלת הברכה ( מ''ב יח יז ) : <b>אפך .</b> חוטמך . צופה . ענין הבטה וראיה :
<h4>{ו}</h4>
<b>ודלת .</b> כן יקרא דבר הנתלה בדבר ויורד למטה . ובדברי רז''ל הדלה עליה את הגפן ( סוכה יא . ) , וכן ורעו דליותיו ( ירמיה יא טז ) : <b>כארגמן .</b> מין צבע אדומה : <b>אסור .</b> ענין קשירה : <b>ברהטים .</b> הבנינים שעל העליות מחדר לחדר . וכן רהיטנו ברותים ( לעיל א יז ) : <b>מה יפית .</b> מלת מה הוכחה על הפלגת הענין . וכן מה רב טובך ( תהלים לא כ ) : <b>נעמת .</b> ענין אהבה . כמו אף נעים ( לעיל א טז ) :
<h4>{ח}</h4>
<b>קומתך .</b> מלשון קומה וגבהות : <b>דמתה .</b> מלשון דמיון והשוואה : <b>לאשכלות .</b> כן יקרא הענף והגרגרים תלוים בו . וכן אשכול הכופר ( לעיל א יד ) :
<h4>{ט}</h4>

<h4>{יא}</h4>

<h4>{יב}</h4>
<b>לכה .</b> הוא ענין זרוז : <b>נלינה .</b> מלשון לינה : <b>בכפרים .</b> כן יקראו המקומות הקטנות . כמו בערים ובכפרים ( דה''א כז ) :
<h4>{יג}</h4>
<b>נשכימה .</b> מלשון השכמה : <b>הסמדר .</b> הם הענבים הדקים הניכרים אחר שנפלה הפרח וזהו פתח הסמדר : <b>הנצו .</b> הוא חנטת הפרי : <b>דודי .</b> ענין אהבה וחבה :
<h4>{יד}</h4>

<h3>שיר השירים פרק-ח</h3>
<h4>{א}</h4>
<b>שדי אמי .</b> דדי אמי : <b>בחוץ .</b> בשוק : <b>אשקך .</b> מלשון נשיקה : <b>יבזו .</b> מלשון בזיון :
<h4>{ב}</h4>
<b>אנהגך .</b> מלשון הנהגה והליכה : <b>אשקך .</b> ענין שתיה : <b>הרקח .</b> ענין תערובות בשמים . כמו רוקח מרקחת ( שמות ל כה ) : <b>מעסיס .</b> ענין סחיטה ורוצה לומר המיץ היוצא על ידי סחיטה . וכן וכעסיס דמם ישכרון ( ישעיה מט כו ) :
<h4>{ג}</h4>
<b>תחבקני .</b> ענין סבוב הזרוע בגוף : <b>תעירו , תעוררו .</b> מלשון התעוררות , וכפל המלה לחזוק הענין :

<h4>{ו}</h4>
<b>כחותם .</b> הוא הטבעת העשוי לחותם בו כתבים . וכן חותמך ופתילך ( בראשית לח יח ) : <b>עזה .</b> חזקה : <b>כשאול .</b> הוא בור הקבר : <b>קנאה .</b> ענינו צער מה בראות דבר מה בזולת והוא לא יוכל להשיגו . כמו ותקנא רחל באחותה ( שם ל א ) : <b>רשפיה .</b> ענין גחלת הבוערת . וכן ומקניהם לרשפים ( תהלים עח מח ) : <b>שלהבת יה .</b> שלהבת הוא ענין להב . כמו תיבש שלהבת ( איוב טו לא ) , ורוצה לומר להבת אש גדולה כי כן דרך המקרא כשרוצה להגדיל דבר מה סומכו למלת השם . וכן ארץ מאפליה ( ירמיה ב לא ) ורבים כמוהו :
<h4>{ז}</h4>
<b>לכבות .</b> מלשון כבוי : <b>ישטפוה .</b> ענין גריפה על ידי חוזק הרדיפה . וכן תשטף ספיחיה ( איוב יד יט ) : <b>הון .</b> ענין עושר . וכן הון יקר ( משלי א יג ) : <b>בוז יבוזו .</b> מלשון בזיון :
<h4>{ח}</h4>
<b>בה .</b> רוצה לומר בענינה :
<h4>{ט}</h4>
<b>נבנה .</b> מלשון בנין : <b>עליה .</b> בעבורה : <b>טירת .</b> היכל וארמון . וכן תהי טירתם נשמה ( תהלים סט כו ) : <b>נצור .</b> מלשון מצור ומבצר : <b>לוח .</b> נסר . כמו את כל לחותים ( יחזקאל כז ה ) :
<h4>{י}</h4>
<b>כמגדלות .</b> מלשו מגדל : <b>כמוצאת .</b> מלשון מציאה :
<h4>{יא}</h4>
<b>בבעל .</b> במישור . וכן בעל גד ( יהושע יא יז ) : <b>המון .</b> ענין עם רב :
<h4>{יב}</h4>
<b>לנוטרים .</b> ענין שמירה . כמו נוטרה את הכרמים ( לעיל א ו ) :
<h4>{יג}</h4>
<b>היושבת .</b> בה''א הקריאה . כמו הדור אתם ראו ( ירמיה ב לא ) : <b>מקשיבים .</b> ענין שמיעה והאזנה :
