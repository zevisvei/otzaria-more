<h1>קהלת - מצודת ציון</h1>
<h3>קהלת פרק-א</h3>
<h4>{א}</h4>
<b>קהלת .</b> הוא ענין קבוץ ואסיפה . כמו קהלת יעקב ( דברים לג ד ) , והמלך שלמה היה נקרא כן , על שהקהיל הדעות המנגדות לצרף וללבן הצודק מהבלתי צודק :
<h4>{ב}</h4>
<b>הבל .</b> ענינו דבר שאין בו ממש , והוא מושאל מלשון הבל פה המוזכר בדברי רז''ל . וכן וילכו אחרי ההבל ( ירמיה ב ה ) :
<h4>{ג}</h4>
<b>יתרון .</b> מלשון מותר וריוח : <b>תחת השמש .</b> במקום השמש :
<h4>{ה}</h4>
<b>ובא .</b> ענין שקיעה . כמו כי בא השמש ( בראשית כח י ) : <b>שואף .</b> ענין ההבטה והתאוה אל הדבר . כמו כעבד ישאף צל ( איוב ז ב ) :
<h4>{ו}</h4>
<b>הרוח .</b> ענינו רצון . כמו הנני נותן בו רוח ( ישעיה לז ז ) :
<h4>{ח}</h4>
<b>הדברים .</b> הענינים . כמו על דבר הכסף ( בראשית מג יח ) :
<h4>{ט}</h4>
<b>כל חדש .</b> שום חדש . וכן לא תעשה כל מלאכה ( שמות כ י ) :
<h4>{י}</h4>
<b>לעולמים .</b> כמו בעולמים והלמ''ד במקום הבי''ת . וכן כתוב בוחר אתה לבן ישי ( ש''א כ ל ) ומשפטו בבן ישי . ומלת עולמים יורה על זמנים רבים . כמו ועבדו לעולם ( שמות כא ו ) :
<h4>{יב}</h4>
<b>הייתי .</b> דבר ההוה נופל בו לשון עבר ולשון עתיד . כמו שנאמר והלכו בניו וגו' ככה יעשה איוב ( איוב א ה ) :
<h4>{יג}</h4>
<b>ונתתי .</b> ושמתי . בדברי רז''ל נתן בו עיניו ( שבת לד . ) : <b>ולתור .</b> ענין חפוש ובקור . כמו לתור את הארץ ( במדבר יד לו ) : <b>ענין רע .</b> דבר רע . ובדברי רז''ל יש הרבה : <b>נתן .</b> ענינו עזיבה . כמו ולא נתן סיחון ( שם כא כג ) : <b>לענות .</b> מלשון עינוי ויסורים :
<h4>{יד}</h4>
<b>ורעות .</b> ענין שבירה . כמו תרועם בשבט ברזל ( תהלים ב ט ) : <b>רוח .</b> רצון :
<h4>{טו}</h4>
<b>מעות .</b> ענין קלקול והשחתה . כמו ודרך רשעים יעות ( שם קמו ט ) : <b>להמנות .</b> מלשון מנין ומספר :
<h4>{טז}</h4>
<b>ראה .</b> הוא מושאל על הבנת הדבר . וכן ראה דבריך טובים ( ש''ב טו ג ) :
<h4>{יז}</h4>
<b>הוללות .</b> ענין שיעמום וערבוב הדעת . כמו ושופטים יהולל ( איוב יב וז ) : <b>ושכלות .</b> כמו וסכלות בסמ''ך , והוא ענין שטות : <b>רעיון .</b> ענין שברון . כמו שנאמר למעלה ורעות רוח :
<h3>קהלת פרק-ב</h3>
<h4>{א}</h4>
<b>לכה .</b> הוא ענין זרוז . כמו ועתה לכו ונהרגוהו ( בראשית לז כ ) ויוסף הלא בא אליהם : <b>נא .</b> עתה : <b>אנסכה .</b> מלשון נסיון ובחינה , ורוצה לומר אנסה אותך . כמו למה אככה ( ש''ב ב כב ) ופירושו אכה אותך :
<h4>{ב}</h4>
<b>מהולל .</b> ענין שיעמום :
<h4>{ג}</h4>
<b>תרתי .</b> ענין חיפוש ובקור . כמו ולתור בחכמה ( לעיל א יג ) :
<h4>{ה}</h4>
<b>גנות .</b> מקום זרוע ירקות ופרחים : <b>ופרדסים .</b> מקום נטוע אילנות בעלי פרי :
<h4>{ו}</h4>
<b>ברכות .</b> מקום בנוי באבנים ובסיד ושם מתכנסים המים . וכן בתעלת הברכה ( מ''ב יח יז ) :
<h4>{ח}</h4>
<b>וסגולת .</b> כן יקרא דבר נחמד הגנוז באוצר . וכן והייתם לי סגולה ( שמות יט ה ) : <b>עשיתי .</b> תכנתי והכנתי . כמו ובן הבקר אשר עשה ( בראשית יח ח ) : <b>שרים .</b> משוררים זכרים : <b>ושרות .</b> משוררות נקבות : <b>שדה .</b> מרכבה מוקפת מחיצה ממעל דומה לתיבה . ובדברי רז''ל שידה תיבה ומגדל ( שבת קכ . ) :
<h4>{ט}</h4>
<b>עמדה .</b> נתקיימה . וכן לשמור את בריתו לעמדה ( יחזקאל יז יד ) :
<h4>{י}</h4>
<b>אצלתי .</b> ענין הפרשה והבדלה . כמו ואצלתי מן הרוח ( במדבר יא יז ) :
<h4>{יא}</h4>
<b>ופניתי .</b> מלשון הפנה וסבוב : <b>ורעות .</b> ענין שבירה : <b>רוח .</b> רצון : <b>יתרון .</b> מותר וריוח : <b>תחת .</b> במקום :
<h4>{יב}</h4>
<b>והוללות .</b> שיעמום : <b>וסכלות .</b> שטות :
<h4>{יד}</h4>
<b>שמקרה .</b> מאורע :
<h4>{טו}</h4>
<b>ולמה .</b> על מה , רוצה לומר מה תועלת :
<h4>{כא}</h4>
<b>ובכשרון .</b> ענין יושר :
<h4>{כב}</h4>
<b>הוה .</b> מלשון הויה : <b>וברעיון .</b> ובשברון :
<h4>{כה}</h4>
<b>יחוש .</b> הוא מלשון חשש הידוע בדברי רז''ל :
<h4>{כו}</h4>
<b>ורעות רוח .</b> שבר רצון :
<h3>קהלת פרק-ג</h3>
<h4>{א}</h4>
<b>חפץ .</b> ענינו כמו רצון :
<h4>{ב}</h4>
<b>לטעת .</b> מלשון נטיעה :
<h4>{ד}</h4>
<b>רקוד .</b> ענין דילוג וקפיצה של שמחה :
<h4>{ה}</h4>
<b>כנוס .</b> ענין אסיפה וקבוץ : <b>לחבוק .</b> הוא סביבת הזרוע בגוף בעבור אהבה :
<h4>{ז}</h4>
<b>לחשות .</b> ענין שתיקה . כמו ויחשו גליהם ( תהלים קז כט ) :
<h4>{י}</h4>
<b>נתן .</b> ענין מסירה : <b>לענות .</b> מלשון עינוי ויסורים :
<h4>{יא}</h4>
<b>מבלי .</b> ענינו כמו מלבד :
<h4>{יג}</h4>
<b>מתת .</b> מלשון מתנה :
<h4>{יז}</h4>
<b>חפץ .</b> רצון :
<h4>{יח}</h4>
<b>על דברת .</b> על ענין כמו על דבר הכסף ( בראשית מג יח ) ולפי שהוא דבוק אל בני האדם לזה אמר דברת . כמו עזי וזמרת יה ( שמות טו ב ) : <b>לברם .</b> מלשון ברור דבר ובחינה :
<h4>{יט}</h4>
<b>ומותר .</b> מלשון יתרון : <b>אין .</b> ענינו כמו לא יש : <b>כי .</b> אם :
<h3>קהלת פרק-ד</h3>
<h4>{א}</h4>
<b>ושבתי .</b> מלשון השבה והחזרה : <b>העשוקים .</b> ענין ערור ותגר . כמו והיית אך עשוק וגזול ( דברים כח ) : <b>מנחם .</b> מלשון תנחומין :
<h4>{ב}</h4>
<b>עדנה .</b> עדיין :
<h4>{ג}</h4>
<b>עדן .</b> עדיין :
<h4>{ד}</h4>
<b>כשרון .</b> ענין יושר : <b>כי היא .</b> אשר היא :
<h4>{ה}</h4>
<b>חובק .</b> ענין סבוב היד בדבר מה :
<h4>{ו}</h4>
<b>כף .</b> כף היד : <b>נחת .</b> מלשון מנוחה : <b>חפנים .</b> כן יקראו כפות הידים הדבקות זה בזה :
<h4>{ט}</h4>
<b>שכר .</b> ענין ריוח :
<h4>{י}</h4>
<b>ואילו האחד .</b> ואוי לו . וכן אי לך ארץ ( לקמן י טז ) :
<h4>{יב}</h4>
<b>יתקפו .</b> מלשון תקיף וחוזק ואחיזה בכח רב . כמו ולא יוכל לדון עם שהתקיף ממנו ( לקמן ו י ) : <b>ינתק .</b> ענין העתק וקריעה . כמו וינתק את היתרים ( שופטים טז ט ) :
<h4>{יג}</h4>
<b>מסכן .</b> עני ורש . כמו וחכמת המסכן בזויה ( לקמן ט טז ) : <b>להזהר .</b> ענינו כמו זריזות ושמירה מן הנזק :
<h4>{יד}</h4>
<b>הסורים .</b> חסר האל''ף והראוי האסורים : <b>רש .</b> עני ודל :
<h4>{טו}</h4>
<b>כל החיים .</b> אדם חרוץ וזריז יקרא בפי הבריות אדם חי : <b>המהלכים .</b> ענין טיול :
<h4>{יז}</h4>
<b>מתת .</b> מלשון נתינה . כמו ביום תת ה' ( יהושע י יב ) :
<h3>קהלת פרק-ה</h3>
<h4>{א}</h4>
<b>תבהל .</b> מלשון בהלה ומהירות : <b>על פיך .</b> מלת על באה במקום הבי''ת . כמו וסגרו על מסגר ( ישעיה כד כב ) :
<h4>{ה}</h4>
<b>אל תתן .</b> אל תעזוב . כמו ולא נתן סיחון ( במדבר כא כג ) : <b>לחטיא .</b> ענין חסרון . כמו ולא יחטיא ( שופטים כ טז ) : <b>בשרך .</b> כן יקראו הבנים . כמו שנאמר והיו לבשר אחד ( בראשית ג כד ) , ורוצה לומר בשר האב והאם נעשה אחד בהבנים : <b>המלאך .</b> שליח : <b>וחיבל .</b> ענין השחתה . כמו וחובל עול מפני שמן ( ישעיה י כז ) :
<h4>{ו}</h4>
<b>ודברים .</b> ענינים . כמו על דבר הכסף ( בראשית מג יח ) : <b>כי .</b> אלא :
<h4>{ז}</h4>
<b>עושק .</b> גזל : <b>שומר .</b> ממתין . כמו ואביו שמר את הדבר ( שם לז יא ) :
<h4>{ט}</h4>
<b>בהמון .</b> מלשון המיה . ורוצה לומר מרבית אנשים שגדלה קול המייתם . וכן על ההמון הזה ( דה''ב יד י ) :
<h4>{י}</h4>
<b>כשרון .</b> מלשון כשר וטוב :
<h4>{יד}</h4>
<b>לא ישא .</b> לא יקח . כמו ובתים ונשאו ( מיכה ב ב ) : <b>בעמלו .</b> הבי''ת הוא במקום מ''ם . כמו והנותר בבשר ובלחם ( ויקרא ח לב ) :
<h4>{טו}</h4>
<b>כל עומת .</b> ענינו כמו כנגד , רוצה לומר שוה ודומה . וכן לעומת קלעי החצר ( שמות לח כא ) :
<h4>{טז}</h4>
<b>וקצף .</b> הוא הראות הכעס אל החוץ :
<h4>{יח}</h4>
<b>ונכסים .</b> בה כלול זהב וכסף וכלים ומקנה וכו' . וכן בנכסים רבים שובו ( יהושע כב ח ) : <b>והשליטו .</b> ענין ממשלה ויכולת : <b>ולשאת .</b> ולקחת : <b>מתת .</b> מלשון נתינה . כמו ביום תת ה' ( שם י יב ) :
<h4>{יט}</h4>
<b>מענה .</b> ענין עדות . כמו לא תענה ( שמות כא יד ) :
<h3>קהלת פרק-ו</h3>
<h4>{א}</h4>
<b>ורבה .</b> גדולה . כמו על כל רב ביתו ( אסתר א ח ) :
<h4>{ג}</h4>
<b>ורב .</b> גדול רוצה לומר מכובד : <b>הנפל .</b> כן יקרא הנולד מת :
<h4>{ה}</h4>
<b>נחת .</b> רוצה לומר נחת רוח :
<h4>{ז}</h4>
<b>תמלא .</b> ענינו כריתה וחתוך . כמו התמלא בשכות עורו ( איוב מ לא ) :
<h4>{ח}</h4>
<b>נגד .</b> ענינו השואה ודמיון . וכן ואלכה לנגדך ( בראשית לג יב ) :
<h4>{י}</h4>
<b>שהתקיף .</b> ענין חוזק ואלמות . כמו אם יתקפו האחד ( לעיל ד יב ) :
<h3>קהלת פרק-ז</h3>
<h4>{ו}</h4>
<b>הסירים .</b> ענפי קוצים . כמו עד סירים סבוכים ( נחום א י ) : <b>הסיר .</b> כן יקרא קדרה של נחשת . כמו והיה כל סיר ( זכריה יד כא ) , ואמר בלשון הנופל על הלשון :
<h4>{ז}</h4>
<b>העושק .</b> ענין קנתור וערעור . כמו את כל העשוקים ( לעיל ד א ) : <b>יהולל .</b> ענין ערבוב הדעת . כמו ושופטים יהולל ( איוב יב יז ) : <b>לב מתנה .</b> רוצה לומר לב חכמה שהיא מתנה לאדם מאת המקום . כמו שנאמר כי ה' יתן חכמה ( משלי ב ו ) :
<h4>{ח}</h4>
<b>רוח .</b> ענינו רוגז . כמו הניחו את רוחי ( זכריה ו ח ) : <b>מגבה רוח .</b> ענין גאוה יתירה :
<h4>{ט}</h4>
<b>אל תבהל .</b> אל תמהר :
<h4>{יג}</h4>
<b>עותו .</b> ענין קלקול :
<h4>{יד}</h4>
<b>על דברת .</b> בעבור . כמו על דבר הכסף ( בראשית מג יח ) :
<h4>{טז}</h4>
<b>תשומם .</b> מלשון שממון :
<h4>{יז}</h4>
<b>תרשע .</b> ענין חרדה ובלבול . כמו ובכל אשר יפנה ירשיע ( ש''א יד ) :
<h4>{יח}</h4>
<b>תנח .</b> מלשון הנחה ועזיבה :
<h4>{כג}</h4>
<b>נסיתי .</b> מלשון נסיון :
<h4>{כה}</h4>
<b>ולתור .</b> ענין חפוש ובקור : <b>וחשבון .</b> מלשון מחשבה . וכן כי אין מעשה וחשבון ( לקמן ט י ) : <b>כסל .</b> מלשון כסיל : <b>והסכלות .</b> שטות : <b>הוללות .</b> שעמום :
<h4>{כו}</h4>
<b>מצודים .</b> שם כלי מכלי הציידים . וכן במצודה רעה ( שם ט יב ) : <b>וחרמים .</b> רשתות . כמו יגורהו בחרמו ( חבקוק א טו ) : <b>אסורים .</b> קשורים . כמו וימסו אסוריו ( שופטים טו יד ) :
<h4>{כז}</h4>

<h4>{כח}</h4>
<b>עוד .</b> ענינו הרבה יותר מן הנהוג והרגיל . כמו ויבך על צואריו עוד ( בראשית מו כט ) :
<h3>קהלת פרק-ח</h3>
<h4>{א}</h4>
<b>פשר .</b> ענין פתרון ופירוש . כמו מפשר חלמיין ( דניאל ה יב ) : <b>ועוז .</b> חוזק : <b>ישנא .</b> מלשון השתנות :
<h4>{ב}</h4>
<b>ועל .</b> ואצל . כמו והחונים עליו ( במדבר ב יב ) : <b>דברת .</b> מלשון דבור :
<h4>{ג}</h4>
<b>בדבר רע .</b> בענין רע :
<h4>{ה}</h4>
<b>ומשפט .</b> ענין מנהג . כמו ומשפט הכהנים ( ש''א ב יג ) :
<h4>{ו}</h4>
