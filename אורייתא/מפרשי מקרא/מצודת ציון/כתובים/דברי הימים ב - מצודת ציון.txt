<h1>דברי הימים ב - מצודת ציון</h1>
<h3>דברי הימים ב פרק-א</h3>
<h4>{ט}</h4>
<b>יאמן .</b> מל' אמנה וקיום :
<h4>{יא}</h4>
<b>יען .</b> בעבור : <b>נכסים .</b> הוא כולל למטלטלין ומקנה ושדות :
<h4>{טו}</h4>
<b>כשקמים .</b> מין אילני תאנים :
<h4>{טז}</h4>
<b>ומקוא .</b> התרגום לא תרגם מלת מקוא ולזה נראה כי קוא שם מקום והמ''ם הוא במקום מן : <b>במחיר .</b> ענין ערך הדבר ושויו כמו אקנה מאיתך במחיר ( ש''ב כ''ד ) :
<h3>דברי הימים ב פרק-ב</h3>
<h4>{א}</h4>
<b>סבל .</b> נושא משא כמו והנושא בסבל ( נחמיה ד ) : <b>חוצב .</b> כן יקרא כריתת האבן : <b>ומנצחים .</b> מל' נצחון ונגישה :
<h4>{ה}</h4>
<b>יעצר .</b> ענין עכבה וכן נעצר כח ( דה''א כט ) : <b>יכלכלהו .</b> ענין החזקה כמו אלפים בת יכיל ( מ''א ז ) :
<h4>{ו}</h4>
<b>ובארגון .</b> הוא ארגמן וכן תרגומו : <b>וכרמיל .</b> הוא תולעת שני : <b>לפתח פתוחים .</b> ענין חתירת הצורות באבנים כמו פתוחי חותם ( שמות כח ) :
<h4>{ז}</h4>
<b>ואלגומים .</b> הוא הפוך מן אלמוגים האמור במ''א והוא מין עץ יקר וחשוב :
<h4>{ט}</h4>
<b>כורים .</b> שם מדה של שלשים סאין : <b>בתים .</b> בת היא מדת הלח מחזקת שלשה סאין :
<h4>{יג}</h4>
<b>ובבוץ .</b> הוא פשתן : <b>רפסדות .</b> הם דוברות האמור במלכים א' והם הקורות הקשורות יחד להשיטם במים אל מקומם הנרצה ולשון רפסודות הוזכר בתלמוד ירושלמי ( ברכות פרק ד' ) :
<h4>{יז}</h4>
<b>מנצחים .</b> מל' נצחון והתגברות ונגישה : <b>ויחל .</b> מל' התחלה :
<h3>דברי הימים ב פרק-ג</h3>
<h4>{ה}</h4>
<b>חפה .</b> ענין צפוי וכסוי : <b>ושרשרות .</b> שלשלאות :
<h4>{ו}</h4>
<b>פרוים .</b> שם מקום ורז''ל אמרו שדומה לדם הפרים ( יומא מה ) :
<h4>{ז}</h4>
<b>ויחף .</b> ענין צפוי : <b>כרובים .</b> תרגום של נער הוא רביא כי פרצוף תינוק להם :
<h4>{י}</h4>
<b>צעצעים .</b> כמו צאצאים באל''ף והוא מל' הצאצאים והצפיעות ( ישעיה כב ) ותרגומו בניא ובני בניא ואף כי אמר כרובים אמר לתוספת ביאור מעשה צעצועים :
<h3>דברי הימים ב פרק-ד</h3>
<h4>{ב}</h4>
<b>וקו .</b> חבל :
<h4>{ה}</h4>
<b>יכיל .</b> יחזיק כמו קטן מהכיל ( מ''א ח ) :
<h4>{ו}</h4>
<b>כיורים .</b> כעין דוד ולה דדים המריקים מים : <b>ידיחו .</b> ענין שטיפה :
<h4>{יא}</h4>
<b>הסירות .</b> כעין קדרה כמו והיו הסירות בבית ה' ( זכריה יד ) : <b>היעים .</b> מכבדות לגרף הדשן והוא מל' ויעה ברד ( ישעיה כח ) :
<h4>{יב}</h4>
<b>והגלות .</b> כן יקרא חלק התחתון מן הכותרת על כי מעשהו כספל והוא מל' וגולה על ראשה ( זכריה ד' ) : <b>והשבכות .</b> כן יקרא חלק העליון מן הכותרת על כי מעשהו כעין שבכה ורשת :
<h4>{יג}</h4>
<b>על פני .</b> ר''ל בגבהם ועל כי פני האדם הוא בגובהו נקרא הגובה על פני בל' שאלה :
<h4>{יד}</h4>
<b>המכנות .</b> מל' כן ובסיס :
<h4>{טז}</h4>
<b>מרוק .</b> זכה ובהירה כמו מרקו הרמחים ( ירמיה מו ) :
<h4>{יז}</h4>
<b>בככר .</b> במישור : <b>צרדתה .</b> ובמ''א נאמר צרתן ובשתי השמות נקראה :
<h4>{כ}</h4>
<b>ונרתיהם .</b> הם העשויים כעין בזיכים ובהם השמן והפתילה : <b>לבערם .</b> מל' הבערה והדלקה : <b>הדביר .</b> זהו בית קה''ק : <b>סגור .</b> הנסגר בכור הצורף לטהרו מסיג :
<h4>{כא}</h4>
<b>והפרח .</b> כעין פרחי אילן הנעשים במנורה כמ''ש בתורה : <b>והמלקחים .</b> הצבתים שבהם לוקחים הפתילה למושכה ולהושיבה במקומה : <b>מכלות .</b> מל' כל :
<h4>{כב}</h4>
<b>והמזמרות .</b> כלי הזמר : <b>והכפות .</b> הבזיכין שבהם הלבינה הנתון על מערכת לחם הפנים : <b>והמחתות .</b> הכלים העשוים לחתות ולשאוב בו האש על הקטרת :
<h3>דברי הימים ב פרק-ה</h3>
<h4>{ו}</h4>
<b>הנועדים .</b> מל' וועד ואסיפה כמו הנועדים על ה' ( במדבר טז ) : <b>עליו .</b> כמו אליו :
<h4>{ח}</h4>
<b>בדיו .</b> מוטות עשויים לשאת בהם את הארון :
<h4>{יג}</h4>
<b>ענן .</b> הוא כעין עשן :
<h3>דברי הימים ב פרק-ו</h3>
<h4>{ב}</h4>
<b>זבול .</b> מדור כמו מזבול קדשך ( ישעיה סג ) : <b>ומכון .</b> מל' הכנה :
<h4>{ט}</h4>
<b>מחלציך .</b> הם המתנים והוא כינוי לאבר המשגל על כי הוא סמוך להם :
<h4>{י}</h4>
<b>ויקם .</b> קיים :
<h4>{יג}</h4>
<b>ויברך .</b> ענין כריעה ולתוספת ביאור אמר על ברכיו :
<h4>{טו}</h4>
<b>מלאת .</b> ענין השלמה :
<h4>{יז}</h4>
<b>יאמן .</b> מל' האמנה וקיום :
<h4>{יח}</h4>
<b>האמנם .</b> אמנם הוא ענין אמת והיא בה''א התמיהה : <b>יכלכלוך .</b> ענין החזקה כמו שלשת אלפים יכיל ( לעיל ד ) : <b>אף כי .</b> הוא ענין ק''ו וכ''ש :
<h4>{כב}</h4>
<b>ונשא .</b> ענין חוב כמו כאשר נושא בו ( ישעיה כד ) : <b>אלה .</b> קללת השבועה כמו בשבועת האלה ( במדבר ה ) :
<h4>{כו}</h4>
<b>בהעצר .</b> ענין מניעה : <b>תענם .</b> מל' עניה והשבה :
<h4>{כז}</h4>
<b>תורם .</b> תלמדם כמו יורה דעה ( ישעיה כח ) :
<h4>{כח}</h4>
<b>שדפון .</b> ענין לקות הזרע עד לא נגמר הבשול : <b>ירקון .</b> הכספת פני התבואה ונהפך למראה ירקון : <b>וחסיל .</b> מין ארבה : <b>יצר .</b> מל' מצור : <b>מחלה .</b> מל' חולי :
<h4>{לב}</h4>
<b>הנטויה .</b> מלשון הטיה :
<h4>{לו}</h4>
<b>ואנפת .</b> ענין כעס כמו כי אנפת בי ( שם יב ) : <b>ושבום .</b> מלשון שביה :
<h4>{לז}</h4>
<b>והשיבו .</b> מל' השבה : <b>נשבו שם .</b> מל' שביה :
<h4>{מ}</h4>
<b>קשובות .</b> ענין שמיעה כמו ואין קשב ( מ''א יח ) :
<h4>{מא}</h4>
<b>עזך .</b> ענין חוזק :
<h3>דברי הימים ב פרק-ז</h3>
<h4>{ג}</h4>
<b>הרצפה .</b> הקרקע הרצופה במצע אבנים וכן על רצפת בהט ושש ( אסתר א ) :
<h4>{ה}</h4>
<b>ויחנכו .</b> התחלת הדבר לעמוד בה משם והלאה קרוי חנוך בלשון מקרא :
<h4>{ז}</h4>
<b>להכיל .</b> להחזיק :
<h4>{יג}</h4>
<b>אעצור .</b> ענין מניעה : <b>חגב .</b> מין ארבה :
<h4>{טו}</h4>
<b>קשובות .</b> שומעות :
<h4>{כ}</h4>
<b>ונתשתים .</b> ענין עקירה כמו לנתוש ולנתוץ ( ירמיה א ) : <b>ולשנינה .</b> ענין התמדת הדבור כמו ושננתם לבניך ( דברים ו ) :
<h4>{כא}</h4>
<b>ישום .</b> ענין תמהון כמו ישום וישרוק ( ירמיה יט ) :
<h3>דברי הימים ב פרק-ח</h3>
<h4>{ד}</h4>
<b>ערי המסכנות .</b> ערי אוצרות וכן ויבן ערי מסכנות ( שמות א ) :
<h4>{ה}</h4>
<b>ערי מצור .</b> ערי מבצר נקראת ערי מצור על כי אין לבוא בהן כ''א יצורו עליהן מסביב לבלתי תת יוצא ובא להביא מזון ומים :
<h4>{ו}</h4>
<b>חשק .</b> ענין תאוה : <b>ובלבנון .</b> כן שם היער :
<h4>{ח}</h4>
<b>כלום .</b> מלשון כליון :
<h4>{ט}</h4>
<b>שלישיו .</b> גבוריו :
<h4>{י}</h4>
<b>הנצבים .</b> הממונים כמו נציב פלשתים ( ש''א י''ג ) : <b>הרודים .</b> ענין ממשלה כמו רדה בקרב אויביך ( תהלים קי ) :
<h4>{יד}</h4>
<b>מחלקות .</b> מלשון חלק ור''ל המשמרות :
<h4>{יח}</h4>
<b>אניות .</b> ספינות :
<h3>דברי הימים ב פרק-ט</h3>
<h4>{א}</h4>
<b>לנסות .</b> מלשון נסיון : <b>בחידות .</b> דברים סתומים קשה ההבנה : <b>ובחיל .</b> ענין עושר ורכוש כמו חילך וכל אוצרותיך ( ירמיה יז ) : <b>כבד .</b> ענין רבוי : <b>יקרה .</b> תשובה :
<h4>{ו}</h4>
<b>מרבית .</b> מל' רבוי :
<h4>{י}</h4>
<b>אלגומים .</b> הוא הפוך מן אלמוגים והוא מין עץ יקר :
<h4>{יא}</h4>
<b>מסלות .</b> מל' מסילה ודרך :
<h4>{יד}</h4>
<b>התרים .</b> הוא מל' לתור את הארץ ( במדבר יג ) והם הסוחרים הקטנים המחפשים ומרגלים אחר סחורה לקנותה במקום הזול : <b>ופחות .</b> שלטונים כמו פחת יהודה ( חגי א ) :
<h4>{טו}</h4>
<b>צנה .</b> הוא כעין מלבוש מקיף את הגוף להציל ממכת חרב כמו כצנה רצון תעטרנו ( תהלים ט ) : <b>שחוט .</b> נמשך כמו חץ שחוט ( ירמיה ט ) ור''ל זהב טוב הנמשך כשעוה :
<h4>{טז}</h4>
<b>מגנים .</b> כלים עשוים לאחזם במלחמה להגן :
<h4>{יז}</h4>
<b>שן .</b> שן הפיל :
<h4>{יח}</h4>
<b>מעלות .</b> מדרגות : <b>וכבש .</b> עשוי כעין גשר משופע ובמשנה רצין ועולין בכבש ( יומא כב ) :
<h4>{כ}</h4>
<b>סגור .</b> הנסגר בכור הצורף להסיר הסיג :
<h4>{כא}</h4>
<b>שנהבים .</b> שן הפיל : <b>וקופים .</b> שם חיה קוף ונזכר בדרז''ל ( יומא כט ) : <b>ותוכים .</b> הם העופות יפות הנוצות בחלוף הגוונים וקרויים טווס :
<h4>{כד}</h4>
<b>ושלמות .</b> מלבושים : <b>נשק .</b> כלי זיין :
<h4>{כה}</h4>
<b>אוריות .</b> אבוסים כי אבוס בעליו ( ישעיה א ) תרגומו אוריא והוא מקום מאכל הבהמות :
<h4>{כז}</h4>
<b>כשקמים .</b> מין אילן תאנים : <b>בשפלה .</b> בעמק :
<h4>{כט}</h4>
<b>ובחזות .</b> ענין מראות נבואה ור''ל בספר נבואות יעדו הוא עדו החוזה :
<h3>דברי הימים ב פרק-י</h3>
<h4>{יא}</h4>

<h4>{טו}</h4>

<h4>{יח}</h4>

<h4>{יט}</h4>

<h4>{יד}</h4>

<h4>{יז}</h4>

<h4>{י}</h4>

<h4>{לא}</h4>
<b>במליצי .</b> השליח נקרא מליץ כי מביא דברי המשלח אל מי שנשלח אליו והוא מל' כי המליץ בינותם ( בראשית מב ) :
<h4>{לג}</h4>
