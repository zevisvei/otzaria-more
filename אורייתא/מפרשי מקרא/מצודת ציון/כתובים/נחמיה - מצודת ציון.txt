<b>הפחה .</b> המושל : <b>הסופר .</b> החכם :
<h4>{כז}</h4>
<b>ובחנוכת .</b> התחלת דבר מה לעמוד בה משם והלאה קרוי חנוך בל' מקרא : <b>מצלתים וכו' .</b> מיני כלי נגון :
<h4>{כח}</h4>
<b>הככר .</b> המישור :
<h4>{לז}</h4>
<b>ועל , מעל .</b> ענינם כמו לפני ואצל :
<h4>{מד}</h4>
<b>הנשכות .</b> כמו הלשכות : <b>מנאות .</b> מתנות וחלק כמו ושלחו מנות ( לעיל ח' ) :
<h3>נחמיה פרק-יג</h3>
<h4>{ב}</h4>
<b>לא קדמו .</b> לא באו לפניהם כמו במה אקדם ה' ( מיכה ו' ) :
<h4>{ה}</h4>
<b>ויעש .</b> ענין תקון כמו וימהר לעשות אותו ( בראשית י''ח ) :
<h4>{ו}</h4>
<b>נשאלתי .</b> מל' שאלה ובקשה ועל כי שאלתו היה על עצמו אמר בלשון נפעל וכן נשאל נשאל דוד ( שמואל א' כ' ) :
<h4>{ז}</h4>
<b>נשכה .</b> כמו לשכה :
<h4>{יא}</h4>
<b>ואריבה .</b> מלשון ריב :
<h4>{יד}</h4>
<b>תמח .</b> כמו תמחה והוא ענין מחיקה :
<h4>{טו}</h4>
<b>גתות .</b> הכלי שדורכים בו הענבים להוציא היין קרוי גת וכן כדורך בגת ( ישעיה ס''ג ) : <b>הערמות .</b> כרי לתבואה כמו ויתנו ערמות ( ד''ה ב' ל''א ) : <b>ועומסים .</b> ענין הטענת משא כמו ויעמוס איש על חמורו ( בראשית מ''ד ) : <b>ציד .</b> מלשון צידה ומזון :
<h4>{טז}</h4>
<b>דג .</b> ר''ל קבוצת דגים :
<h4>{יז}</h4>
<b>חורי יהודה .</b> שרי יהודה :
<h4>{יט}</h4>
<b>צללו .</b> מלשון צל :
<h4>{כ}</h4>
<b>הרוכלים .</b> מוכרי הבושם :
<h4>{כא}</h4>
<b>לנים .</b> מל' לינה : <b>תשנו .</b> מל' שנית :
<h4>{כב}</h4>
<b>וחוסה .</b> ענין חמלה כמו אתה חסת על הקיקיון ( יונה ד' ) :
<h4>{כה}</h4>
<b>ואמרטם .</b> ענין תלישת השער כמו ואיש כי ימרט ראשו ( ויקרא י''ג ) :
<h4>{כז}</h4>
<b>למעול .</b> ענין חטא ופשע :
<h4>{כט}</h4>
<b>גאלי .</b> ענין טנוף ומאוס כמו ויגאלו מן הכהונה ( לעיל ז' ) :
<h4>{ל}</h4>
<b>משמרות .</b> כל דבר המוטל על אנשים הרבה לעשות כל אחד בזמנו קרוי משמר כי ישמור לבוא בזמן הקבוע לו ולא יעבור :
<h4>{לא}</h4>
<b>מזומנות .</b> ענין הכנה וקביעות כמו לעתים מזומנים ( לעיל י' ) :
