<h3>תהילים פרק-א</h3>
<h4>{א}</h4>

<h4>{ו}</h4>

<b>נאוה .</b> מלשון נאה ויפה כמו לישרים נאוה תהלה ( לעיל לג ) :
<h4>{ב}</h4>
<b>יכנס .</b> מלשון הכנסה ואסיפה וכן והמסכה צרה כהתכנס ( ישעיה נח ) :
<h4>{ג}</h4>
<b>ומחבש .</b> ענין קשירת המטלית על המכה כמו ולנשברת תחבש ( יחזקאל ל''ד ) :
<h4>{ד}</h4>
<b>מונה .</b> מלשון מנין :
<h4>{ו}</h4>
<b>מעודד .</b> ענין רוממות וחוזק : <b>עדי ארץ .</b> עד ארץ :
<h4>{ז}</h4>
<b>ענו .</b> ענין הרמת קול כמו וענו הלוים וגו' קול רם ( דברים כז ) :
<h4>{ח}</h4>
<b>בעבים .</b> בעננים : <b>חציר .</b> דשא ועשב :
<h4>{י}</h4>
<b>בשוקי .</b> שוק הרגל :
<h4>{יא}</h4>
<b>המיחלים .</b> המקוים :
<h4>{יג}</h4>
<b>בריחי .</b> כעין מטה ישימו אותו ברוחב השער לסגרו :
<h4>{יד}</h4>
<b>חלב .</b> מיטב כמו ואכלו את חלב הארץ ( בראשית מה ) :
<h4>{טו}</h4>
<b>ירוץ .</b> ענין מהירות ההליכה :
<h4>{טז}</h4>
<b>כפור .</b> כמו שלג נופל הוא בשחר בימי הקור וכן וכפור שמים מי ילדו ( איוב לח ) :
<h4>{יז}</h4>
<b>קרחו .</b> הוא הגליד הנקפה והנקרש על פני המים בימי הקור וכן מבטן מי יצא הקרח ( שם ) : <b>כפתים .</b> ענין חתיכות כמו פתות אותה פתים ( ויקרא ב ) : <b>קרתו .</b> מלשון קור וצנה :
<h4>{יח}</h4>
<b>וימסם .</b> מלשון המסה והמגה : <b>ישב .</b> מלשון נשיבה והפחה כמו כי רוח ה' נשבה בו ( ישעיה מ ) : <b>יזלו .</b> מלשון הזלה ונטיפה :
<h4>{כ}</h4>
<b>בל .</b> עניינו כמו לא :
<h3>תהילים פרק-קמח</h3>
<h4>{ו}</h4>
<b>לעד לעולם .</b> כפל המלה בשמות נרדפים וכן אדמת עפר ( דניאל יב ) : <b>חק .</b> עניינו כמו גזרה :
<h4>{ז}</h4>
<b>תנינים .</b> הן מין דגים גדולים שבים : <b>תהומות .</b> עומק הים :
<h4>{ח}</h4>
<b>וקיטור .</b> הוא האד העולה מן הארץ דומה הוא לעשן :
<h4>{י}</h4>
<b>רמש .</b> ענין הזזה ממקום למקום וכן שם רמש ואין מספר ( לעיל קד ) :
<h4>{יא}</h4>
<b>לאמים .</b> עמים :
<h4>{יד}</h4>
<b>קרן .</b> השאילו המלה ההיא להתחזקות וממשלה על כי בעלי הקרן יתחזקו בו בהלחמם עם מי :
<h3>תהילים פרק-קמט</h3>
<h4>{ב}</h4>
<b>בעושיו .</b> בהמגדל ומרומם אותנו כמו הוא עשנו ( לעיל ק ) : <b>יגילו .</b> ישמחו :
<h4>{ג}</h4>
<b>במחול בתוף וכנור .</b> שמות כלי נגון :
<h4>{ד}</h4>
<b>יפאר .</b> מלשון פאר ויופי :
<h4>{ה}</h4>
<b>יעלזו .</b> ישמחו :
<h4>{ו}</h4>
<b>בגרונם .</b> בצוארם : <b>פיפיות .</b> חדוד החרב נקרא פה כמו ולה שתי פיות ( שופטים ג ) :
<h4>{ז}</h4>
<b>תוכחות .</b> ענין משפט יסורים כמו והוכח במכאוב ( איוב לג ) :
<h4>{ח}</h4>
<b>לאסור .</b> ענין קשירת המאסר : <b>בזקים .</b> מין כבלים ושלשלאות כמו והוא אסור בזקים ( ירמיה מ' ) : <b>בכבלי .</b> כעין שלשלאות :
<h4>{ט}</h4>
<b>הדר .</b> ענין יופי :
<h3>תהילים פרק-קנ</h3>
<h4>{א}</h4>
<b>עזו .</b> ענין חוזק :
<h4>{ד}</h4>
<b>במנים ועוגב .</b> שמות כלי נגון :
<h4>{ה}</h4>
<b>בצלצלי שמע .</b> הם שני כלי נחושת ומקישים זה בזה ומשמיעי' קול בהקשתן וכמ''ש במצלתים נחושת להשמיע ( דה''א כו ) ובמשנה הקיש בן ארזא בצלצל ( תמיד פ''ז ) : <b>בצלצלי תרועה .</b> הם החצוצרות המשמיעים הרעשת קול תרועה כי צלצל הוא ענין הרעשת קול וכן לקול צללו שפתי ( חבקוק ג' ) שהיא הרעשה מועטת :
