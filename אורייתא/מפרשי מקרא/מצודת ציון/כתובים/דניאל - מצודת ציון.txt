<h1>דניאל - מצודת ציון</h1>
<h3>דניאל פרק-א</h3>
<h4>{א}</h4>
<b>ויצר .</b> מלשון מצור וענינו הקפת החיל על העיר להציר להם ולכבשם :
<h4>{ב}</h4>
<b>שנער .</b> היא בבל ע''ש שננערו בה מתי המבול כי היא עמוקה מכל העולם :
<h4>{ג}</h4>
<b>סריסיו .</b> שריו : <b>הפרתמים .</b> ענין שררה ועושר וכן חיל פרס ומדי הפרתמים ( אסתר ח ) והיא מלה זרה ואולי היא מלשון כשדים וכמוהו רבים ימצאו בזה הספר אשר אין לדעת ביאורם כ''א לפי הקרוב לענינם :
<h4>{ד}</h4>
<b>מאום .</b> כמו מום : <b>מדע .</b> ענין מחשבה כמו גם במדעך מלך אל תקלל ( קהל' י ) : <b>ספר .</b> ר''ל כתב הנכתב בספר :
<h4>{ה}</h4>
<b>וימן .</b> ענין הזמנה כמו וימן ה' דג ( יונה ב ) : <b>מפת בג .</b> ענין מאכל :
<h4>{ח}</h4>
<b>יתגאל .</b> ענין לכלוך וטנוף כמו לחם מגואל ( מלאכי א ) :
<h4>{י}</h4>
<b>מנה .</b> הזמין : <b>זועפים .</b> ר''ל רעים והוא מל' זעף וכעס כי הכועס נשתנה מראית פניו לרוע : <b>כגילכם .</b> כדמותכם ובדרז''ל בן גילו ( יבמות קכ ) :
<h4>{יא}</h4>
<b>המלצר .</b> הוא המסדר המאכל : <b>אשר מנה .</b> אשר הפקיד כמו ומהם ממונים על הכלים ( דה''א ט' ) :
<h4>{יב}</h4>
<b>נס .</b> מלשון נסיון : <b>הזרועים .</b> מיני זרעונים כקטניות וכיוצא :
<h4>{טו}</h4>
<b>ובריאי .</b> ענין שומן כמו ועגלון איש בריא ( שופטים ג ) :
<h4>{יז}</h4>
<b>חזון .</b> ראיית החלום והוא כפל ענין במ''ש :
<h4>{כ}</h4>
<b>ידות .</b> חלקים כמו וארבע הידות ( בראשי' מז ) : <b>החרטומים .</b> הם חכמי הטבע . האשפים . הם חכמי הרפואה :
<h3>דניאל פרק-ב</h3>
<h4>{א}</h4>
<b>ותתפעם .</b> ענין הכאה וקשקוש וכן ויהי בבוקר ותפעם רוחו ( בראשי' מ''א ) . והוא מלשון פעמון המקשקש : <b>ושנתו .</b> מלשון שינה : <b>נהיתה .</b> ענין כריתה ושבירה כמו נהייתי ונחליתי ( לקמן ח ) :
<h4>{ד}</h4>
<b>ופשרא .</b> ופתרונו כמו ופותר אין אותו ( בראשי' מ ) תרגומו ופשר לית ליה וכן רוב הספר הוא לשון תרגום והוא ארמית ואין צורך להביא ראיה עליהם :
<h4>{ה}</h4>
<b>אזדא .</b> הלכה ובדרז''ל ואזדא לטעמיה : <b>הדמין .</b> בת''י הוא ענין נתחים :
<h4>{ו}</h4>
<b>ונבזבה .</b> ענין דורון גדול בויתור רב ובדרז''ל המבזבז אל יבזבז וכו' ( כתובות נ ) :
<h4>{ח}</h4>
<b>יציב .</b> ענין אמת וכמוהו רבים בזה הספר ובתפלה אמת ויציב והוא מלשון דבר הניצב ועומד ומתקיים :
<h4>{יב}</h4>
<b>בנס .</b> בזעף כי והנם זועפים ( בראשית מ ) תרגומו נסיסין :
<h4>{טו}</h4>
<b>ענה .</b> ענינו הרמת קול כמו וענו הלוים וגו' קול רם ( דברים כז ) : <b>מהחצפה .</b> ממהרת והוא מל' חצוף הממהר להביש ולהחפיר את הבריות :
<h4>{כו}</h4>
<b>כהל .</b> יכול וכן ולא כהלין כתבא למקרא ( לקמן ה ) :
<h4>{כז}</h4>
<b>גזרין .</b> הם בעלי הקסם ההולכים לקראת נחשים :
<h4>{כט}</h4>
<b>רעיוניך .</b> מחשבתך כמו בנתה לרעי ( תהלים קלט ) :
<h4>{ל}</h4>
<b>על דברת .</b> ענינו כמו בעבור וכן על דברת בני האדם ( קהלת ג ) :
<h4>{לא}</h4>
<b>ואלו .</b> והנה וכן ואלו עיר וקדיש ( לקמן ד ) : <b>דכן .</b> זה כי מי האיש הלזה ( בראשית כד ) ת''א דיכי :
<h4>{לב}</h4>
<b>וירכתה .</b> שלשה פרקים יש ברגל העליון קרוי ירך האמצעי קרוי שוק התחתון קרוי רגל :
<h4>{לה}</h4>
<b>כעור .</b> כמוץ או יתכן שתהיה מל' נעורת של פשתן ובא בחסרון הנו''ן ור''ל כדבר אשר ינוער :
<h4>{מ}</h4>
<b>וחשל .</b> ומרדד ובדרז''ל חשלי דודי ( כתובות עז ) :
<h4>{מא}</h4>
<b>די פחר .</b> של יוצר חרס ובדרז''ל במני דפחר' ( תענית ז ) : <b>נצבתא .</b> ענין נטיעה כמו מצבת בם ( ישעיה ו ) ור''ל הכח והיכולת ע''ש שהנטיעה היא כח הפירות והעלין :
<h4>{מח}</h4>
<b>סגנין .</b> ענין שררה מה כמו ויבא סגנים כמו חומר ( ישעיה מא ) :
<h3>דניאל פרק-ג</h3>
<h4>{ב}</h4>
<b>לאחשדרפניא וכו' .</b> כולם שמות מיני שררה זו למעלה מזו וידועים המה בל' כשדים : <b>לחנכת .</b> התחלת תשמיש הדבר להיות משמש והולך קרוי חנוך בל' המקרא :
<h4>{ד}</h4>
<b>וכרוזא .</b> ענין השמעת הדבר לרבים : <b>עממיא .</b> הם האומות שיש להם אלוה ותורה : <b>אומיא .</b> הם בלא אלוה ובלא תורה : <b>ולשניא .</b> הם החלוקים בלשונותם בין מבני אלוה ותורה ובין זולתם :
<h4>{ה}</h4>
<b>משרוקיתא וכו' .</b> כולם שמות מיני כלי זמר וידועים המה בל' כשדים :
<h4>{ח}</h4>
<b>ואכלו קרציהון .</b> הוא מענין רכילות ומלשינות כי לא תלך רכיל ( ויקרא יט ) ת''א לא תיכול קורצין ע''ש שדרך המלשין לרמז דברו ולקרוץ בעיניו למען לא יבינו כל העומדים ובהקריצה ההיא כאלו אוכל בשר מי אשר ילשין עליו :
<h4>{י}</h4>
<b>טעם .</b> ענינו בכ''מ דבר שיש בו ממש הנעשה בטעם ולא דבר של מה בכך :
<h4>{יד}</h4>
<b>הצדא .</b> ענין שממה כי תהו ובהו ( בראשית א ) ת''א צדיא ורקניא :
<h4>{טז}</h4>
<b>חשחין .</b> כמו חוששין הידוע בדרז''ל :
<h4>{יט}</h4>
<b>למזא .</b> ענין הסקה ושריפה וכן מזי רעב ( דברים לב ) :
<h4>{כ}</h4>
<b>לכפתה .</b> ענין קשירה ובדרז''ל עבד כפות לו ( ב''ק כב ) :
<h4>{כא}</h4>
<b>בסרבליהון .</b> הוא שם מלבוש ובדרז''ל בסרבלי חתימי ( שבת נח ) : <b>פטשיהון .</b> הם המכנסים : <b>וכרבלתהון .</b> מין לבוש עשוי להתעטף בו ודוגמתו ודוד מכורבל במעיל ( דה''א טו ) :
<h4>{כב}</h4>
<b>שביבא .</b> ניצוצות כמו ולא יגה שביב אשו ( איוב יח ) :
<h4>{כד}</h4>
<b>להדברוהי .</b> הם היועצים המנהיגים כי וינהג את הצאן ( שמות ג ) ת''א ודבר :
<h4>{כה}</h4>
<b>לבר אלהין .</b> כן יקרא המלאך כאלו הוא מבני בית של השכינה וכן ויבואו בני האלהים ( איוב א ) :
<h4>{כז}</h4>
<b>בגשמיהון .</b> בגופם כי ושבט לגו כסילים ( משלי כו ) ת''י ושבטא לגושמיהון : <b>התחרך .</b> ענין שריפה כמו לא יחרוך רמיה ( שם יב ) : <b>עדת .</b> עברה כמו לא עדה עליו שחל ( איוב כח ) :
<h4>{כט}</h4>
<b>שלו .</b> שגגה ומשגה כמו על השל ( ש''ב ו ) :
<h4>{לא}</h4>
<b>ישגא .</b> יתגדל ויתרבה כמו כארז בלבנון ישגה ( תהלים צב ) :
<h4>{לב}</h4>
<b>שפר .</b> נאה ויאה כמו אמרי שפר ( בראשית מט ) :
<h4>{לג}</h4>
<b>כמה .</b> הוא כענין מאד ובדרז''ל כמה גדולים מעשה חייא ( כתובות קג ) :
<h3>דניאל פרק-ד</h3>
<h4>{א}</h4>
<b>ורענן .</b> ענין לחות ורטיבות כמו דשנים ורעננים ( תהלים צב ) :
<h4>{ו}</h4>
<b>אנס .</b> ענין הכרח ונצוח ועל שם זה נקרא הגזלן אנס :
<h4>{ט}</h4>
<b>עפיה .</b> העלין כמו מבין עפאים ( שם קד ) :
<h4>{י}</h4>
<b>עיר .</b> מלאך המחריב והוא מלשון ערו ערו ( שם קלז ) :
<h4>{יא}</h4>
<b>גדו .</b> כרתו והוא קרוב מלשון שקמים גדעו ( ישעיה ט ) : <b>אתרו .</b> ענין הפלה והשלכה כמו ראה ויתר גוים ( חבקוק ג ) :
<h4>{יד}</h4>
<b>שאלתא .</b> הוא ענין גזירה ע''ש כי נעשה בשאלת פי ה' : <b>עד דברת .</b> כמו על דברת ור''ל בעבור :
<h4>{טז}</h4>
<b>אשתומם .</b> ענין תמהון ושתיקה כמו משמים בתוכם ( יחזקאל ג ) : <b>לערך .</b> לשונאך כמו ויהי ערך ( ש''א כה ) :
<h4>{כב}</h4>
<b>יטעמון .</b> מלשון טעימה ואכילה :
<h4>{כד}</h4>
<b>מלכי .</b> עצתי כמו וימלך לבי עלי ( נחמיה ה ) :
<h4>{לג}</h4>
<b>יבעון .</b> ענין שאלה ובקשה כמו אם תבעיון בעיו ( ישעיה כא ) :
<h4>{לד}</h4>
<b>בגוה .</b> כמו בגאוה ודוגמתו כי השפילו ותאמר גוה ( איוב כב ) :
<h3>דניאל פרק-ה</h3>
<h4>{א}</h4>
<b>לחם רב .</b> כל סעודה קרוי' ע''ש הלחם :
<h4>{ב}</h4>
<b>שגלת' .</b> היא המלכה המיוחדת לו למשכב המשגל :
<h4>{ה}</h4>
<b>נברשתא .</b> מנורה ובדרז''ל עשתה נברשת של זהב ( יומא לז ) : <b>גירא .</b> סיד כמו כאבני גיר ( ישעיה כז ) : <b>פס ידא .</b> כף היד ובדרז''ל שלשים בפיסת היד ( אהלות פ''א ) :
<h4>{ו}</h4>
<b>נקשן .</b> ענין הכאה כמו מקשה תיעשה המנורה ( שמות כה ) :
<h4>{ז}</h4>
<b>ארגונא .</b> ארגמן וכן ובארגון וכרמיל ( דה''ב ב ) והוא שם מין צבע חשוב : <b>והמניכא .</b> ענק רצוף בטבעות לתכשיט כי רביד הזהב ( בראשית מא ) ת''א מניכא דדהבא :
<h4>{ט}</h4>
<b>משתבשין .</b> ענין בלבול ושבוש הדעת :
<h4>{יא}</h4>
<b>נהירו .</b> הארה כמו ואל תופע עליו נהרה ( איוב ג ) :
<h4>{יז}</h4>
<b>ונבזביתך .</b> ענין דורון גדול כמו מתנן ונבזבה ( לעיל ב ) :
<h4>{יט}</h4>
<b>זייעין .</b> ענין חרדה ורתת כמו והיה רק זועה ( ישעיה כח ) :
<h4>{כא}</h4>
<b>ערדיא .</b> פראים כמו ומוסרות ערוד ( איוב לט ) :
<h4>{כד}</h4>
<b>רשים .</b> ענין כתיבה וכן את הרשום בכתב ( לקמן י ) :
<h4>{כח}</h4>
<b>פרס .</b> ענין שבירה כמו הלא פרוס לרעב לחמך ( ישעיה נח ) :
<h3>דניאל פרק-ו</h3>
<h4>{ג}</h4>
<b>סרכין .</b> שם מין שררה :
<h4>{ד}</h4>
<b>עשית .</b> ענין מחשבה כמו אבדו עשתונותיו ( תהלים קמו ) :
<h4>{ה}</h4>
<b>עילה .</b> עלילה , ובדרז''ל עילה מצאו ( כתובות כ ) : <b>שלו .</b> משגה כמו על השל ( ש''ב ו ) :
<h4>{יב}</h4>
<b>הרגשו .</b> ענין קבוץ כי בקבוץ עם רב נשמע קול המיה ורגישה וכן למה רגשו גוים ( תהלים ב' ) :
<h4>{טו}</h4>
<b>בל .</b> הוא מל' מנדה בלו והלך ( עזרא ד' ) וענינה מס וגולגולת , או הוא מענין לאו כמו בל למד צדק ( ישעיה כו ) : <b>משתדר .</b> ענין ספק והשתדלות בדרז''ל השתדל להיות איש ( אבות פ''ב ) :
<h4>{יט}</h4>
<b>ובת טות .</b> לן מבלי סעודת הערב וכן בדרז''ל ובת טות ולא הבדיל ( פסחים קז ) ר''ל לא אכל בלילה על שלא הבדיל : <b>ודחון .</b> הוא הפוך מן חדון ור''ל שמחת כלי זמר :
<h4>{כ}</h4>
<b>בשפרפרא .</b> כן ת''י על השחר : <b>בנגהא .</b> מל' נוגה ואורה :
<h3>דניאל פרק-ז</h3>
<h4>{ב}</h4>
<b>וארו .</b> כמו ואלו ר''ל והנה : <b>מגיחן .</b> ענין יציאה כמו גוחי מבטן ( תהלים כב ) :
<h4>{ד}</h4>
<b>מריטו .</b> תלשו כמו ואמרטה משער ראשי ( עזרא ט ) :
<h4>{ה}</h4>
<b>ולשטר .</b> צלע ת''א סטר :
<h4>{ו}</h4>
<b>באתר .</b> כמו בתר בלא אל''ף :
<h4>{ז}</h4>
<b>רפסה .</b> רמסה כמו מתרפס ברצי כסף ( תהלים סח ) :
<h4>{ט}</h4>
<b>ועתיק .</b> ענין מיושן כמו והדברים עתיקים ( דה''א ד' ) :
<h4>{י}</h4>
<b>ורבו .</b> הוא עשרת אלפים :
<h4>{יב}</h4>
<b>וארכה .</b> ענין המתנה ואריכת זמן : <b>זמן ועידן .</b> הם שמות ( יד ) וליה . ונתן לו הממשלה והכבוד והמלוכה וכל העכו''ם יעבדו לו : נרדפים וכן אדמת עפר ( לקמן יב ) :
<h4>{טו}</h4>
<b>נדנה .</b> ענין תיק ונרתק כמו וישב חרבו אל נדנה ( דה''א ד ) :
<h4>{כג}</h4>
<b>ותדושנה .</b> מל' דישה :
<h4>{כה}</h4>
<b>יבלא .</b> מל' בליה והפסד : <b>ויסבר .</b> ענין מחשבה ובדרז''ל סברוהו רבנן : <b>עדן .</b> כן התרגום של עת ושל שנה :
<h3>דניאל פרק-ח</h3>
<h4>{ב}</h4>
<b>הבירה .</b> ענין ארמון וכן לא לאדם הבירה ( שם כט ) ור''ל שהיתה מטרפולין מקום הארמונות : <b>אובל .</b> נהר והוא מל' יבלי מים ( ישעיה ל ) ובא האל''ף במקום היו''ד כמו ישי ואישי ( דה''א ב ) : <b>אולי .</b> כן שם הנהר :
<h4>{ד}</h4>
<b>מנגח .</b> ענין הכאה ודחיפה בקרן :
<h4>{ה}</h4>
