<h1>עזרא - מצודת ציון</h1>
<h3>עזרא פרק-א</h3>
<h4>{א}</h4>
<b>העיר .</b> מל' התעוררות : <b>רוח .</b> רצון כמו הנני נותן בו רוח ( מ''ב יט ) : <b>ויעבר קול .</b> ענין הכרזה כמו ויעבירו קול במחנה ( שמות לו ) :
<h4>{ב}</h4>
<b>פקד .</b> צוה כמו פקודי ה' ( תהלים י''ט ) :
<h4>{ד}</h4>
<b>וברכוש .</b> כן יקרא כל שוה כסף כמו ואת כל רכושו ( בראשית יד ) :
<h4>{ה}</h4>
<b>ראשי האבות .</b> ראשי המשפחות : <b>לכל העיר .</b> את כל העיר הלמ''ד במקום את וכן הרגו לאבנר ( ש''ב ו ) : <b>רוחו .</b> רצונו :
<h4>{ו}</h4>
<b>ובמגדנות .</b> המלה ההיא יורה על דבר המעולה בין בפירות בין במלבושים בין בכלים וכן ומגדנות לחזקיה ( דה''ב ל''ב ) : <b>לבד על .</b> הוא כפול לתוספ' ביאור וכן הרק אך במשה ( במדב' יב ) :
<h4>{ח}</h4>
<b>על יד .</b> ביד על במקום בי''ת וכן על צבאותם ( שמות יב ) ורבים כמוהו : <b>הגזבר .</b> הפקיד :
<h4>{ט}</h4>
<b>אגרטלי .</b> שם כלי ינתן בו המים לנטילת הידים : <b>מחלפים .</b> סכינים והוא מל' כליל יחלוף ( ישעיה ב ) ע''ש שבהם שוחטים ומחליפים מחיים למיתה ועש''ז קראו חז''ל לצדדי האולם מזה ומזה בית החליפות ( מדות פ''ד ) כי שם גנזו הכהנים סכיניהם :
<h4>{י}</h4>
<b>כפורי .</b> מזרקות וע''ש שהכהן הוורק היה מקנח אצבעו מן הדם בשפת המזרק לזה קרוי כפור שהוא ענין קנוח ובדרז''ל ובעי לכפורי ידיה ( גיטין נו ) : <b>משנים .</b> מל' שניים ור''ל כפולים הראוי להשתמש בהן בשתי עבריהן :
<h3>עזרא פרק-ב</h3>
<h4>{א}</h4>
<b>.</b> :
<h4>{סב}</h4>
<b>המתיחשים .</b> ענין יחוס הוא הודעת סדר הדורות מן המשפחות ממי יצאו ומי אביהם : <b>ויגואלו .</b> ענין טנוף ונמאס כמו לחם מגואל ( מלאכי א ) :
<h4>{סד}</h4>
<b>רבוא .</b> הוא עשרת אלפים :
<h4>{סו}</h4>
<b>פרדיהם .</b> הם הבאים מן הסוס והחמור :
<h4>{סח}</h4>
<b>מכונו .</b> מל' כן ובסיס :
<h4>{סט}</h4>
<b>דרכמונים .</b> שם מטבע ידוע אצלם : <b>מנים .</b> שם משקל ליטרא :
<h3>עזרא פרק-ג</h3>
<h4>{א}</h4>
<b>ויגע .</b> כמו והגיע :
<h4>{ג}</h4>
<b>מכונותיו .</b> מל' כן ובסיס :
<h4>{ז}</h4>
<b>לצדונים ולצורים .</b> אנשי צידון וצור : <b>כרשיון .</b> הוא מל' רשות הידוע בדרז''ל :
<h4>{ח}</h4>
<b>לנצח .</b> ענינו נצוח השיר ע''ש שדרך בעלי השיר לנצח זא''ז בהרמת הקול ובהכרעת הנעימה :
<h4>{ט}</h4>
<b>ואחיו .</b> חבריו ורעיו : <b>לנצח .</b> ענינו נגישה וכפיה והיא מל' נצחון והתגברות :
<h4>{י}</h4>
<b>במצלתים .</b> עשוי הוא משני כלי נחושת להשמיע קול גדול בהקשתן זה בזה וכן בצלצלי שמע ( תהלים קו ) :
<h4>{יא}</h4>
<b>ויענו .</b> ענין הרמת קול כמו וענו הלוים וגו' קול רם ( דברים כז ) :
<h3>עזרא פרק-ד</h3>
<h4>{א}</h4>
<b>צרי .</b> אויבי :
<h4>{ד}</h4>
<b>מרפים .</b> מל' רפיון : <b>לבנות .</b> כמו מלבנות :
<h4>{ה}</h4>
<b>וסוכרים .</b> כמו ושוכרים בשין :
<h4>{ו}</h4>
<b>שטנה .</b> ענין קטרוג :
<h4>{ז}</h4>
<b>כנותיו .</b> ענינו רעיו וחבריו ובדרז''ל שבת ועבוד' כוכבים בכנותהון ( כריתות ג ) כן גרס הערוך ור''ל בכל דיניהם המתחבר אליהם : <b>הנשתון .</b> ענינו מפורש ואין לו דומה כי אם בזה הספר : <b>ומתרגם .</b> גם הוא ענין מפורש ובדרז''ל מאן דמתרגם לי חבית ( ב''מ מא ) :
<h4>{ח}</h4>
<b>בעל טעם .</b> ענינו היודע מיטב הדבור במליצה נאותה וכן וטע' זקנים יקח ( איוב י''ב ) : <b>אגרא .</b> כן יקרא כתב השלוח ע''ש שמאוסף בו דברים הרבה והוא מל' אוגר בקיץ ( משלי י ) : <b>כנמא .</b> ענין אמירה וכן וינאמו נאם ( ירמיה כ''ג ) :
<h4>{יא}</h4>
<b>פרשגן .</b> ענין טופס הדבר וענינו כי משנה תורת משה ( יהושע ח ) ת''י פרשגן אורייתא :
<h4>{יב}</h4>
<b>ושוריא .</b> חומות וכן אדלג שור ( תהלים י''ח ) : <b>שכלילו .</b> ענין גמר והשלמה וכת''א על ויכלו השמים ( בראשית ב' ) : <b>ואושיא .</b> ענין יסוד כמו לאשישי קיר חרשת ( ישעיה ט''ז ) : <b>יחיטו .</b> מל' חוט ר''ל יחברו כאלו תפרם בחוט ע''י המחט :
<h4>{יג}</h4>
<b>מנדה .</b> זה מס המלך המוטל על כל בני המדינה ביחד : <b>בלו .</b> הכסף גולגלתא . והלך . הוא תשודה המביאים למלך בעת יעבור אליהם : <b>ואפתום .</b> הוא ענין הכנסה ואין לו דומה :
<h4>{יד}</h4>
<b>מלח .</b> ענין חרבן כמו ארץ מלחה ( ירמיה י''ז ) : <b>וערות .</b> מל' ערוה ר''ל בזיון : <b>אריך .</b> מענין יפה וכשר ובדרז''ל מאה פלפלין במאה ועשרים ואריך ( ב''מ עה ) ור''ל והדבר יפה וישר ואין בו משום רבית :
<h4>{טו}</h4>
<b>יבקר .</b> יחפש כמו אבקר את צאני ( יחזקאל ל''ד ) : <b>ואשתדור .</b> ענינו סרבנות ומרידה והוא מלשון שדר שהוא שליחות בלשון ארמי כי כן דרך המורדים לשלוח שלוחים לשאול עזר להחזיק ידם :
<h4>{יט}</h4>
<b>טעם .</b> ענינו צווי וגזירה ועצה כמו מטעם המלך ( יונה ג ) :
<h4>{כב}</h4>
<b>וזהירין .</b> הוא מל' זהירות הידוע בדרז''ל שהוא כענין זריזות אלא שזריזות הוא להשמר שלא תבוא הדבר לידו וזהירות הוא להשמר ממנה בעת באה לידו : <b>שלו .</b> משגה ושגגה וכן על השל ( ש''ב ו' ) :
<h3>עזרא פרק-ה</h3>
<h4>{ב}</h4>
<b>ושריו .</b> ענין התחלה כי וזה החילם ( בראשית י''א ) ת''א ודין דשריאו : <b>מסעדין .</b> מל' סעד וסיוע :
<h4>{ג}</h4>
<b>ושתר בוזני .</b> כך שמו : <b>ואשרנא .</b> כמו ושוריא ור''ל החומות :
<h4>{ח}</h4>
<b>אספרנא .</b> הוא ענין מהירות רב ואין לו דומה כי אם בזה הספר :
<h3>עזרא פרק-ו</h3>
<h4>{ב}</h4>
<b>באחמתא .</b> הוא מל' וחמת מים ( בראשית כ''א ) והוא נאד של עור : <b>מגלה .</b> ספר :
<h4>{ג}</h4>
<b>מסובלין .</b> ענין משא כבדה כמו והנושאים בסבל ( נחמיה ד ) :
<h4>{ד}</h4>
<b>נדבכין .</b> שורות ובדרז''ל קורין וכו' ובראש הנדבך ( ברכות ט''ו ) :
<h4>{ז}</h4>
<b>ולשבי .</b> הלמ''ד הוא במקום את ור''ל ועם סבי :
<h4>{ט}</h4>
<b>חשחן .</b> כמו חוששין וכן לא חשחין אנחנא ( דניאל ג ) : <b>שלו .</b> ענין משגה ושכחה כמו על השל ( ש''ב ו' ) :
<h4>{יא}</h4>
<b>יתנסח .</b> ענין עקירה כמו בית גאים יסח ה' ( משלי ט''ו ) : <b>וזקיף .</b> הוא העץ שתולין עליו ובדרז''ל דאית ליה זקיפתא וכו' זקיף ביניתא ( ב''מ נ''ט ) : <b>יתמחא .</b> יושם ובדרז''ל המחוהו אצל חנוני ( שם קי''א ) :
<h4>{יב}</h4>
<b>ימגר .</b> ענין הפלה כמו וכסאו לארץ מגרת ( תהלים פ''ט ) :
<h4>{טו}</h4>
<b>ושיציא .</b> ענין גמר והשלמה כי כאשר כלו ( בראשית מ''ג ) ת''א כד שציאו :
<h4>{טז}</h4>
<b>חנכת .</b> התחלת דבר מה לעמוד בה משם והלאה נקרא חנוך בלשון המקרא :
<h4>{יח}</h4>
<b>בפלוגתהון .</b> ענין חלוקה :
<h4>{כב}</h4>
<b>והסב .</b> מל' סבוב ור''ל הטה ופנה :
<h3>עזרא פרק-ז</h3>
<h4>{ו}</h4>
<b>סופר .</b> ר''ל חכם ובדרז''ל סופר מברך ובור יוצא ( ברכות מ''ה ) : <b>מהיר .</b> הוא הפוך מן קשה ההבנה :
<h4>{ז}</h4>
<b>והשוערים .</b> שומרי השערים . והנתינים . הגבעונים . ( יב ) גמיר . כולה כי כליל תקטר ( ויקרא ו ) ת''א גמיר מתסק :
<h4>{יג}</h4>
<b>מתנדב .</b> ענין רצון כי כל המתנדב מתנדב בדצון :
<h4>{כ}</h4>
<b>השהות .</b> כמו חששות והוא ענין צורך : <b>יפל .</b> ענין נטיה ודוגמתו כי יפול לא יוטל ( תהלים ל''ז ) . ( כב ) : <b>בתין .</b> שם מדת הלח והיא ג' סאין :
<h4>{כג}</h4>
<b>אדרזדא .</b> הוא ענין זריזות ואין לו דומה :
<h3>עזרא פרק-ח</h3>
<h4>{כ}</h4>
<b>נקבו .</b> נתפרשו וכן ונוקב שם ( ויקרא כ''ד ) :
<h4>{כב}</h4>
<b>ופרשים .</b> הם רוכבי הסוסים בהרגל רב :
<h4>{כג}</h4>
<b>ויעתר .</b> ענין קבלת תפלה וכן ויעתר לו ( בראשית כ''ה ) :
<h4>{כז}</h4>
<b>כיפורי .</b> מזרקות : <b>לאדרכונים .</b> שם מטבע ידוע אצלם : <b>מוצהב .</b> מראה כעין גוון הזהב וכן שער צהוב ( ויקרא יג ) :
<h4>{כט}</h4>
<b>שקדו .</b> ענין מהירות כמו כי שוקד אני ( ירמיה א' ) :
<h4>{לא}</h4>
<b>ואורב .</b> שונא היושב במארב להסתר :
<h4>{לה}</h4>
<b>צפירי .</b> שעירי :
<h3>עזרא פרק-ט</h3>
<h4>{א}</h4>
<b>לאחשדרפני .</b> מין שררה ידוע אצלם : <b>ופחוות .</b> מל' פחה ומושל :
<h4>{ב}</h4>
<b>נבדלו .</b> ענין הפרשה :
<h4>{ג}</h4>
<b>והתערבו .</b> מל' תערובות ובלבול : <b>והסגנים .</b> מין שררה : <b>במעל .</b> ענין חטא ופשע :
<h4>{ד}</h4>
<b>ומעילי .</b> הוא מין מלבוש עליון : <b>ואמרטה .</b> ענין תלישה כמו ואיש כי ימרט ראשו ( ויקרא י''ג ) : <b>משומם .</b> משותק ונאלם כמו משמים בתוכם ( יחזקאל ג' ) :
<h4>{ה}</h4>
<b>חרד .</b> ענין מהירות כמו ויחרדו זקני העיר ( ש''א י''ו ) :
<h4>{ז}</h4>
<b>ונכלמתי .</b> מלשון כלימה ובושה : <b>להרים .</b> לרומם ולהגביה : <b>ואשמתנו .</b> מל' אשמה ופשע :
<h4>{ט}</h4>
<b>תחנה .</b> חנינה וחן וכן לבלתי היות להם תחנה ( יהושע י''א ) : <b>פליטה .</b> ענין הצלה ושיור : <b>יתד .</b> מסמר : <b>מחיה .</b> מל' חיות :
<h4>{י}</h4>
<b>חרבותיו .</b> מלשון חורבה : <b>גדר .</b> כותל מקיף :
<h4>{יב}</h4>
<b>נדה .</b> מל' נדידה ורחוק :
<h4>{יד}</h4>
<b>חשכת .</b> מנעת כמו ולא חשכת ( בראשית כ''ב ) :
<h4>{טו}</h4>
<b>להפר .</b> לבטל : <b>להתחתן .</b> מלשון חתון : <b>תאנף .</b> מל' אף וחמה :
<h3>עזרא פרק-י</h3>
<h4>{א}</h4>
