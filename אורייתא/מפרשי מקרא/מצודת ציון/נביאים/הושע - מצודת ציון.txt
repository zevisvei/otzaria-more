<h1>הושע - מצודת ציון</h1>
<h3>הושע פרק-א</h3>
<h4>{ב}</h4>
<b>זנונים .</b> מל' זונה : <b>זנה תזנה .</b> ענין הסרה ועש''ז תקרא המנאפת זונה כי סרתה מדרכי הצניעות :
<h4>{ד}</h4>
<b>ופקדתי .</b> ענין זכרון כמו וה' פקד ( ראשית כ''א ) : <b>והשבתי .</b> ענין בטול כמו שבת נוגש ( ישעיה י''ד ) :
<h4>{ה}</h4>
<b>קשת .</b> ר''ל חוזק וגבורה :
<h4>{ו}</h4>
<b>נשא אשא .</b> ענין מחילת העון כמו ונשאתי לכל המקום ( בראשית י''ח ) :
<h4>{ז}</h4>
<b>ובפרשים .</b> הם הרגילים לרכוב על הסוסים :
<h4>{ח}</h4>
<b>ותגמול .</b> כן יקרא זמן השלמת היניקה כמו ביום הגמל את יצחק ( שם כ''א ) :
<h3>הושע פרק-ב</h3>
<h4>{א}</h4>
<b>ימד .</b> מלשון מדידה : <b>להם .</b> עליהם :
<h4>{ב}</h4>
<b>יזרעאל .</b> היא מלה מורכבת מן זריעה ומן אל :
<h4>{ד}</h4>
<b>אישה .</b> בעלה : <b>ונאפופיה .</b> מלשון ניאוף :
<h4>{ה}</h4>
<b>אפשיטנה .</b> ענין הסרת המלבוש כמו ופשט את בגדיו ( ויקרא ו' ) : <b>והצגתיה .</b> ענין העמידה כמו תציג אותו לבד ( שופטים ז' ) : <b>ושתיה .</b> ענין שימה כמו למען שיתי ( שמות י' ) : <b>ציה .</b> שממה :
<h4>{ז}</h4>
<b>הובישה .</b> מלשון בושה : <b>הורתם .</b> מלשון הריון : <b>צמרי ופשתי .</b> מלשון צמר ופשתים : <b>ושקויי .</b> מל' משקה :
<h4>{ח}</h4>
<b>שך .</b> ענין גדר כמו הסר משוכתו ( ישעיה ה' ) : <b>בסירים .</b> קוצים כמו כקול הסירים ( קהלת ז' ) : <b>וגדרתי .</b> מלשון גדר : <b>ונתיבותיה .</b> מלשון נתיב ושביל :
<h4>{ט}</h4>
<b>תשיג .</b> ענין הקרוב : <b>אישי .</b> בעלי :
<h4>{י}</h4>
<b>והתירוש .</b> כן יקרא ענבי היין : <b>והיצהר .</b> הוא השמן כמו תירושך ויצהרך ( דברים י''א ) :
<h4>{יא}</h4>
<b>במועדו .</b> בזמנו : <b>והצלתי .</b> ענין הפרשה כמו הלא אצלת לי ברכה ( בראשית כ''ז ) : <b>ערותה .</b> גלוי הבשר :
<h4>{יב}</h4>
<b>נבלותה .</b> ענינו דבר כעור ומגונה וכן והשלכתי עליך שקוצים ונבלתיך ( נחום ג' ) :
<h4>{יג}</h4>
<b>והשבתי .</b> ענין בטול : <b>משושה .</b> שמחתה :
<h4>{יד}</h4>
<b>והשמותי .</b> מלשון שממון : <b>אתנה .</b> מלשון אתנן וכן יקרא מתן הזונה כמו אתנן זונה ( דברים כ''ג ) :
<h4>{טו}</h4>
<b>ופקדתי .</b> מלשון זכרון : <b>ותעד .</b> מלשון עדי קשוט כמו ועדית עדי ( יחזקאל כ''ג ) : <b>נזמה .</b> שם עדי כמו נזם זהב ( משלי כ''ה ) : <b>וחליתה .</b> ענין תכשיט מה כמו וחלי כתם ( , שם ) :
<h4>{טז}</h4>
<b>מפתיה .</b> מלשון פתוי והסתה :
<h4>{יז}</h4>
<b>עכור .</b> ענין השחתה כמו ועמק עכור ( ישעיה ס''ה ) : <b>וענתה .</b> ענינו הרמת קול שירה כמו ותען להם מרים ( שמות ט''ו ) :
<h4>{כ}</h4>
<b>ורמש .</b> הם בריות קטנות קצרי הרגל וכן רמש וצפור כנף ( תהלים קמ''ח ) :
<h4>{כא}</h4>
<b>וארשתיך .</b> מל' ארוס' :
<h4>{כג}</h4>
<b>אענה .</b> מלשון עניה והשבת שאלה :
<h3>הושע פרק-ג</h3>
<h4>{א}</h4>
<b>רע .</b> חבר ועל הבעל יאמר ודוגמתו והיא חברתך ( מלאכי ב' ) : <b>פונים .</b> מלשון הפנה וסבוב : <b>אשישי .</b> כוסות וגביעים כמו סמכוני באשישות ( ש''ה ב' ) : <b>ענבים .</b> ר''ל יין ענבים :
<h4>{ב}</h4>
<b>ואכרה .</b> ענין קנין כמו וגם מים תכרו ( דברים ב' ) : <b>כסף .</b> הוא מלה משותפת על הכסף ממש ועל התאוה כמו למעשה ידיך תכסוף ( איוב י''ד ) : <b>וחומר .</b> הוא הכור בת עשרה איפות והאיפה ג' סאים : <b>ולתך .</b> הוא חצי הכור חמשה איפות :
<h4>{ג}</h4>
<b>תשבי .</b> ענין עכבה והמתנה כמו אשב עד שובך ( שופטים ו' ) :
<h4>{ד}</h4>
<b>מצבה .</b> הוא מזבח של אבן אחת ועושים אותה להקריב בה עכו''ם : <b>אפוד .</b> זה יאמר על האפוד והחשן עם האורים ותומים וכן אפוד ירד בידו ( שמואל א' כ''ג ) :
<h4>{ה}</h4>
<b>אל ה' .</b> את ה' וכן ואל טובו כמו וירדפו אל מדין ( שופטי' ז' ) :
<h3>הושע פרק-ד</h3>
<h4>{ב}</h4>
<b>אלה .</b> ענין שבועה : <b>וכחש .</b> שקר : <b>פרצו .</b> ענין שבר הגדר :
<h4>{ג}</h4>
<b>תאבל .</b> ענין השחתה כמו אבלה נבלה הארץ ( ישעיה כ''ד ) : <b>ואומלל .</b> ענין כריתה כמו כי אמלל אני ( תהלים ו' ) : <b>ואספו .</b> ענין כליון ואבדון כמו אסף אסיפם ( ירמיה ח' ) :
<h4>{ד}</h4>
<b>אך .</b> ת''י ברם ר''ל באמת וכן אך עצמי ובשרי אתה ( בראשית כ''ט ) : <b>ירב .</b> מלשון מריבה :
<h4>{ה}</h4>
<b>ודמיתי .</b> ענין כריתה כמו ער מואב נדמה ( ישעיה ט''ו ) : <b>אמך .</b> הכנסייא :
<h4>{ז}</h4>
<b>כרובם .</b> ענין גדולה כמו כי רבה היא ( אסתר א' ) : <b>בקלון .</b> מלשון קלות ובזיון : <b>אמיר .</b> מלשון תמורה וחלוף :
<h4>{ח}</h4>
<b>ישאו נפשו .</b> זה הלשון יאמר על המצפה לדבר מה וכן מנשאים את נפשם ( ירמיה כ''ב ) :
<h4>{ט}</h4>
<b>ופקדתי .</b> ענין זכרון : <b>ומעלליו .</b> מעשיו :
<h4>{י}</h4>
<b>יפרצו .</b> ענין התחזקות הרבוי וכן פרצו לרוב ( ד''ה א' ד' ) :
<h4>{יא}</h4>
<b>ותירוש .</b> הוא היין החדש :
<h4>{יב}</h4>
<b>בעצו .</b> מלשון עץ : <b>ומקלו .</b> מטה עץ : <b>התעה .</b> מלשון תועה :
<h4>{יג}</h4>
<b>אלון ולבנה ואלה .</b> שמות אילנות :
<h4>{יד}</h4>
<b>יפרדו .</b> ענין הפרשה כמו הפרד נא מעלי ( בראשית י''ג ) : <b>הקדשות .</b> הזונות כמו לא תהיה קדשה ( דברים כ''ג ) : <b>ילבט .</b> ענין כשלון ויגיעה כמו ואויל שפתי' ילבט ( משלי י ) :
<h4>{טו}</h4>
<b>יאשם .</b> מלשון אשמה :
<h4>{טז}</h4>
<b>סוררה .</b> ענין מרידה : <b>ירעם .</b> מל' מרעה : <b>במרחב .</b> במקום רחב :
<h4>{יז}</h4>
<b>עצבים .</b> כן יקראו העכו''ם :
<h4>{יח}</h4>
<b>סר .</b> מלשון הסרה : <b>סבאם .</b> כן יקראו הדברים המשכרים כמו סבאך מהול במים ( ישעיה א' ) : <b>הבו .</b> ענין נתינה כמו הבו שכרי ( זכריה יא ) : <b>קלון .</b> ענין בזיון : <b>מגניה .</b> מלכיה :
<h4>{יט}</h4>
<b>צרר .</b> ענין קשירה כמו מי צרר מים ( משלי ל ) : <b>בכנפיה .</b> מל' כנף והמשילה לעוף שהרוח נושאה :
<h3>הושע פרק-ה</h3>
<h4>{ב}</h4>
<b>ושחטה .</b> מל' שחיטה : <b>שטים .</b> ענין הסרה מדרך הטוב כמו ואת כי שטית ( במדבר ה ) : <b>העמיקו .</b> מל' עומק : <b>מוסר .</b> מלשון יסורין :
<h4>{ג}</h4>
<b>נכחד .</b> ענין מניעה כמו ולא כחדו מאבותם ( איוב טו ) :
<h4>{ד}</h4>
<b>לא יתנו .</b> לא יעזבו וכן ולא נתן סיחון ( במדבר כא ) : <b>מעלליהם .</b> מעשיהם :
<h4>{ה}</h4>
<b>וענה .</b> ענין הכנעה והשפלה כמו מאנת לענות מפני ( שמות י ) : <b>גאון .</b> מלשון גאוה ורוממות :
<h4>{ו}</h4>
<b>חלץ .</b> ענין השמטה והסרה כמו צדיק מצרה נחלץ ( משלי יא ) :
<h4>{ז}</h4>
<b>חלקיהם .</b> כן יקרא אחוזת השדה כמו ואכלה את החלק ( עמוס ז ) :
<h4>{ח}</h4>
<b>חצוצרה .</b> שם כלי נגון : <b>הריעו .</b> מל' תרועה והרמת קול :
<h4>{ט}</h4>
<b>לשמה .</b> מלשון שממון : <b>תוכחה .</b> מל' ויכוח :
<h4>{י}</h4>
<b>כמסיגי גבול .</b> אמר בל' שאלה כמי שמחזיר הסימן לתוך שדה חבירו להרחיב גבולו כמ''ש לא תסיג גבול רעך ( דברים יט ) : <b>עברתי .</b> מל' עברה וזעם :
<h4>{יא}</h4>
<b>עשוק .</b> ענין שדידה וגזל : <b>רצוץ .</b> ענין שבירה וכתיתה כמו עשוק ורצוץ ( שם כח ) : <b>הואיל .</b> ענין רצון כמו ויואל משה ( שמות ב ) : <b>צו .</b> מל' צואה :
<h4>{יב}</h4>
<b>כעש .</b> היא הרמה האוכלת הבגדים כמו כבגד יאכלם עש ( ישעיה נא ) : <b>וכרקב .</b> תולעת הגדלה בעץ נרקב :
<h4>{יג}</h4>
<b>מזורו .</b> כן נקרא המכה ע''ש שזורים עליה אבק מסממני' המרפאין וכן ישימו מזור תחתיך ( עובדיה א ) : <b>ירב .</b> הוא אשור ויקרא בשני השמות וכן מנחה למלך ירב ( לקמן י ) : <b>יגהה .</b> ענין רפואה כמו ייטב גהה ( משלי יז ) :
<h4>{יד}</h4>
<b>כשחל .</b> וככפיר . הם שמות האריה :
<h4>{טו}</h4>
<b>ישחרונני .</b> ענין דרישה כמו שוחר טוב ( שם יא ) :
<h3>הושע פרק-ו</h3>
<h4>{א}</h4>
<b>לכו .</b> הוא ענין לשון זרוז : <b>יך .</b> מלשון הכאה : <b>ויחבשנו .</b> ענין קשירת המטלית על המכה והשבירה כמו ולנשברת לא חבשתם ( יחזקאל לד ) :
<h4>{ב}</h4>
<b>יחיינו .</b> ענין רפואה כמו עד חיותם ( יהושע ה ) : <b>מיומים .</b> ר''ל מהעתים :
<h4>{ג}</h4>
<b>נרדפה .</b> אמר בלשון שאלה על מרבית ההשתדלות : <b>כשחר .</b> היא אור הבוקר : <b>נכון .</b> ברור ומובטח : <b>מוצאו .</b> מל' יציאה : <b>כמלקוש .</b> הוא המטר המאוחר היורד על הקשין ועל המלילות כמו כעב מלקוש ( משלי טז ) : <b>יורה .</b> ענין שביעה כת''י והוא הפוך כמו ירוה שהוא ענין שביעה כמו כגן רוה ( ישעיה נח ) :
<h4>{ה}</h4>

<h4>{ח}</h4>
<b>עקובה .</b> ענין מארב ואורב כמו ואת עקבו מים ( יהושע ח ) :
<h4>{ט}</h4>
<b>וכחכי .</b> ענין תקוה כמו וחכיתי לה' ( ישעיה ח ) : <b>גדודים .</b> מרבית אנשים יחד קרוי גדוד : <b>חבר .</b> מל' חבורה וסיעה : <b>שכמה .</b> ר''ל צד אחד ודעה אחת כמו לעבדו שכם אחד ( צפניה ג ) : <b>זמה .</b> מעשה תעוב ומגונה :
<h4>{י}</h4>
<b>שערוריה .</b> ענין דבר מגונה ומאוס כמו שמה ושערוריה ( ירמיה ח ) :
<h4>{יא}</h4>
<b>שת .</b> ענין שימה : <b>קציר .</b> ענף וסעיף כמו ועשה קציר ( איוב יד ) : <b>שבות .</b> מל' שבי הגולה :
<h3>הושע פרק-ז</h3>
<h4>{א}</h4>
<b>פשט .</b> ענין הסרת המלבוש כמו ופשט את בגדיו ( ויקרא ו ) :
<h4>{ג}</h4>
<b>ובכחשיהם .</b> בשקריהם :
<h4>{ד}</h4>

<h4>{ה}</h4>
<b>החלו .</b> מל' חלי : <b>חמת .</b> הוא נאד וכן וחמת מים ( בראשית כא ) : <b>לוצצים .</b> מל' לץ :
<h4>{ו}</h4>

<h4>{ח}</h4>

<h4>{ט}</h4>

<h4>{י}</h4>
<b>וענה .</b> ענין הכנעה כמו מאנת לענות מפני ( שמות י' ) : <b>גאון .</b> מלשון גאוה ורוממות :
<h4>{יא}</h4>
<b>פותה .</b> מל' פתוי ולכן יקרא הסכל פתי כי הוא נפתה בקל :
<h4>{יב}</h4>
<b>איסירם .</b> מל' יסורין :
<h4>{יג}</h4>
<b>נדדו .</b> ענין הנעה : <b>שוד .</b> ענין עשק וגזל : <b>אפדם .</b> מלשון פדיון :
<h4>{יד}</h4>
<b>יילילו .</b> מל' יללה : <b>על דגן .</b> בעבור דגן : <b>יתגוררו .</b> מלשון גרות : <b>יסורו .</b> מל' סרירה ומרידה :

<b>בעברתי .</b> מל' עברה וזעם :
<h4>{יב}</h4>
<b>צרור .</b> ענין קשירה כמו צרור כספו ( בראשית מב ) : <b>צפונה .</b> ענין הטמנה כמו יצפון לצדיק ( משלי יג ) :
<h4>{יג}</h4>
<b>חבלי .</b> כן יקראו מכאובי היולדת וכן תזעק בחבליה ( ישעיה כו ) : <b>במשבר .</b> הוא מקום מושב היולדת כשכורעת ללדת וכן כי באו בנים עד משבר ( שם לז ) :
<h4>{יד}</h4>
<b>קטבך .</b> ענין כריתה וכן מקטב ישוד צהרים ( תהלים צא ) : <b>נחם .</b> מלשון תנחומין :
<h4>{טו}</h4>
<b>יפריא .</b> מל' פרי : <b>ויבוש .</b> מל' יבשות : <b>מקורו .</b> הוא המעיין : <b>ויחרב .</b> מל' חורב ויובש : <b>ישסה .</b> ענין בזה ושלל כמו ומשנאינו שסו למו ( שם מד ) :
<h3>הושע פרק-יד</h3>
<h4>{א}</h4>
<b>תאשם .</b> מל' שממון : <b>מרתה .</b> מל' מרי ומרד : <b>עולליהם .</b> קטניהם : <b>ירוטשו .</b> ענין בקיעה כמו ועולליהם תרטש ( מ''ב ח ) : <b>והריותיו .</b> הנשים הדרות :
<h4>{ה}</h4>
<b>משובתם .</b> ענין מרידה והליכה בדרכי הלב כמו שובו בנים שובבים ( ירמיה ג ) :
<h4>{ו}</h4>
<b>כשושנה .</b> הוא הורד : <b>ויך .</b> יאמר כן בדרך השאלה על התפשטות ודוגמתו ומחה על כתף ים כנרת ( במדבר לד ) ומחה הוא תרגום של מכה : <b>כלבנון .</b> שם יער :
<h4>{ז}</h4>
<b>יונקותיו .</b> ענפיו הרכים וכן ויעל כיונק לפניו ( ישעיה נג ) : <b>הודו .</b> ענין הדר ונאה :
<h4>{ח}</h4>
<b>ישובו .</b> ענין ההשקט ומרגוע כמו בשובה ונחת ( שם ל ) : <b>יחיו .</b> ענין השבת הנפש מן החולשה ודומה לו עד חיותם ( יהושע ה ) : <b>דגן .</b> כדגן ותחסר כ''ף הדמיון וכן ייטיב גהה ( משלי יז ) ומשפטו כגהה :
<h4>{ט}</h4>
<b>לעצבים .</b> הם העכו''ם : <b>עניתי .</b> מלשון עניים ותשובת שאלה : <b>ואשורנו .</b> ענין הבטה וראיה כמו אשורנו ולא קרוב ( במדבר כד ) : <b>כברוש .</b> שם אילן : <b>רענן .</b> ענין לחות ורטיבות כמו בשמן רענן ( תהלים צב ) :
<h3>הושע פרק-טו</h3>
<h4>{ב}</h4>
<b>והאזינו .</b> הטו אוזן לשמוע :
<h4>{ג}</h4>
<b>עליה .</b> בעבורה :
<h4>{ד}</h4>
<b>יתר .</b> מל' נותר : <b>הגזם .</b> הארבה הילק החסיל . כולם שמות מיני חגבים :
<h4>{ה}</h4>
<b>הקיצו .</b> ענין הערה מן השינה : <b>והילילו .</b> מל' יללה : <b>עסיס .</b> כן נקרא היץ והן וכעסיס דמם ישכרון ( ישעיה מט ) :
<h4>{ו}</h4>
<b>עצום .</b> חזק בכח : <b>ומתלעות .</b> הם השינים הגסות כמו מתלעות עול ( איוב כט ) :
<h4>{ז}</h4>
<b>לקצפה .</b> מל' קצף ור''ל למארה : <b>חשף חשפה .</b> קלף קליפת' והוא מל' מחשף הלבן ( בראשית ל ) שהוא ענין גלוי כי בהקלף קליפות העץ נעשה מגולה : <b>הלבינו .</b> מל' לבן : <b>שריגיה .</b> ענביה כמו ובגפן ג' שריגים ( שם מ ) :
<h4>{ח}</h4>
<b>אלי .</b> ענין קינה
