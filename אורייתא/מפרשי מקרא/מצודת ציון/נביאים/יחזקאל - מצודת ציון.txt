<h1>יחזקאל - מצודת ציון</h1>
<h3>יחזקאל פרק-א</h3>
<h4>{א}</h4>
<b>כבר .</b> כן שם הנהר :
<h4>{ד}</h4>
<b>מתלקחת .</b> מתלהב ובוער כמו ואש מתלקחת בתוך הברד ( שמות ט ) והוא מלשון לקיחה כי הלהב נלקח ונאחז בדבר הנשרף : <b>ונוגה .</b> ענין הארה וזריחה כמו מנוגה נגדו ( תהלים יח ) : <b>כעין .</b> כגוון כמו ועינו כעין הבדולח ( במדבר יא ) : <b>ההשמל .</b> הוא הזך אשר באש המנוקה מן העשן , אף היא מלה מורכבת חש מל וחש הוא ענין שתיקה כמו עת לחשות ( קהלת ג ) ומל הוא מלשון מלה ודבור :
<h4>{ה}</h4>
<b>חיות .</b> ר''ל ממין החי וכת''י בריין : <b>דמות .</b> צורות : <b>להנה .</b> להן :
<h4>{ז}</h4>
<b>ישרה .</b> שוה מבלי עקום פרקים : <b>וכף .</b> הוא תחתיות הרגל : <b>ונוצצים .</b> מל' ניצוץ אש : <b>וקלל .</b> מזוקק ומזהיר וכן ומרגלותיו כעין נחושת קלל ( דניאל י ) :
<h4>{ח}</h4>
<b>רבעיהם .</b> ת''י סטריהון והוא מלשון רבע כי כל צד הוא רבע כי ארבע צדדים יש :
<h4>{ט}</h4>
<b>חוברות .</b> ר''ל נוגעות : <b>אשה אל אחותה .</b> כן יאמר בדברים השווים וכן ופניהם איש אל אחיו ( שמות כה ) : <b>יסבו .</b> מלשון סבוב והפנה : <b>עבר .</b> צד :
<h4>{יא}</h4>
<b>וכנפיהם .</b> באה הוי''ו במקום עם וכן ויוסף היה במצרים ( שם א ) ור''ל עם יוסף שהיה במצרים : <b>פרודות .</b> ענין פרישה כמו מפוזר ומפורד ( אסתר ג ) : <b>גויתיהנה .</b> גופיהן כמו גוית שאול ( ש''א ל''א ) :
<h4>{יב}</h4>
<b>הרוח .</b> ענין רצון כמו הנני נותן בו רוח ( ישעיה לז ) :
<h4>{יג}</h4>
<b>בוערות .</b> דולקות : <b>הלפידים .</b> כן יקרא העץ שהשלהבת קשורה בו וכן כלפיד יבער ( שם סב ) :
<h4>{יד}</h4>
<b>הבזק .</b> כמו הברק כי זסשר''ץ מתחלף :
<h4>{טו}</h4>
<b>אופן .</b> גלגל כעין של עגלה :
<h4>{טז}</h4>
<b>תרשיש .</b> שם אבן יקר :
<h4>{יח}</h4>
<b>וגביהן .</b> מל' גבוהות וכן וגבותם : <b>עינים .</b> גוונים הרבה והוא מלשון כעין תרשיש :
<h4>{כ}</h4>
<b>לעומתם .</b> ר''ל שוה להם וכן לעומת קלעי החצר ( שמות לח ) : <b>החיה .</b> ר''ל כל חיה או שאמר שם הכלל :
<h4>{כב}</h4>
<b>רקיע .</b> עגין שטיחה ופרישה כמו לרוקע הארץ ( תהלים קלז ) : <b>הקרח .</b> הוא הגליד הנקפה והנקרש ע''פ המים בימי הקור וכן משליך קרחו ( שם קמז ) :
<h4>{כג}</h4>
<b>ישרות .</b> מלשון ישר ושוה :
<h4>{כד}</h4>
<b>המולה .</b> ענין המיה ושאון וכן לקול המולה ( ירמיה יא ) : <b>תרפינה .</b> מלשון רפיון והשקטה :
<h3>יחזקאל פרק-ב</h3>
<h4>{א}</h4>
<b>אותך .</b> כמו אתך והתי''ו בדג''ש ור''ל עמך :
<h4>{ג}</h4>
<b>עצם היום .</b> גוף היום וכן בעצם היום הזה ( בראשית ז ) :
<h4>{ד}</h4>
<b>קשי פנים .</b> הוא הפוך מן בושת הפנים כי זהו פניו רכות וממהר להלבין בשביל בושה קטנה ולא כן קשה הפנים :
<h4>{ה}</h4>
<b>מרי .</b> ענין מרד כמו ממרים הייתם ( דברים ט ) :
<h4>{ו}</h4>
<b>סרבים .</b> מל' סרוב ומיאון : <b>וסלונים .</b> מין קוצים כמו סלון ממאיר ( לקמן כח ) : <b>עקרבים .</b> הם מין קוצים כמו ואני בעקרבים ( דה''ב י ) : <b>תחת .</b> ענין שבר ופחד כמו אל תחת ( ירמיה א ) :
<h4>{ח}</h4>
<b>פצה .</b> פתח כמו פציתי פי ( שופטים יא ) :
<h4>{ט}</h4>
<b>שלוחה .</b> ענין הושטה כמו ידה ליתד תשלחנה ( שם ה ) : <b>מגלת .</b> מל' גליון ואגרת ויקרא כן ע''ש שגוללין אותו : <b>ספר .</b> ענין כתיבה ע''ש שיש בו ספורי דברים :
<h4>{י}</h4>
<b>קינים .</b> מל' קינה : <b>והגה .</b> ענין המיה כעין יללה וכן אהגה כיונה ( ישעיה לח ) : <b>והי .</b> הוא ענין מספד ונהי :
<h3>יחזקאל פרק-ג</h3>
<h4>{ה}</h4>
<b>עמקי .</b> ר''ל קשה : <b>שפה .</b> כן יקרא לשון עם כמו ושפה אחת לכולם ( בראשית יא ) :
<h4>{ו}</h4>
<b>תשמע .</b> ענין הבנה : <b>אם לא .</b> הוא ענין שבועה כאדם האומר אם לא כן הוא יהיה כזאת וכזאת וגזם ולא אמר וכן אם לא בתים ( ישעיה ה ) :
<h4>{ז}</h4>
<b>לא יאבו .</b> לא ירצו כמו לא אבה יבמי ( דברים כה ) :
<h4>{ח}</h4>
<b>לעומת .</b> נוכח :
<h4>{ט}</h4>
<b>כשמיר .</b> הוא ברזל החזק וכן כצפורן שמיר ( ירמיה יו ) : <b>מצור .</b> מסלע :
<h4>{יג}</h4>
<b>משיקות .</b> מל' נשיקה כי בהגעתן זו לזו כאלו נושקות זו לזו :
<h4>{יד}</h4>
<b>בחמת .</b> מל' חמה וכעס :
<h4>{טו}</h4>
<b>משמים .</b> ענין תמהון ושתיקה כמו אשתומם כשעה חדא ( דניאל ד ) :
<h4>{טז}</h4>
<b>מקצה .</b> מסוף :
<h4>{יז}</h4>
<b>צופה .</b> הוא העומד על מגדל העיר ומביט מי ומי הבאים אל העיר : <b>והזהרת .</b> מל' אזהרה :
<h4>{יח}</h4>
<b>אבקש .</b> ר''ל אדרוש לשלם גמול עליה :
<h4>{כה}</h4>
<b>עבותים .</b> חבלים עבות בת ג' יתרים : <b>ואסרוך .</b> ענין קשור :
<h4>{כו}</h4>
<b>חכך .</b> החיך מה שמעל הלשון :
<h3>יחזקאל פרק-ד</h3>
<h4>{א}</h4>
<b>לבנה .</b> עשוי מן הטיט ומייבשים בחמה או שורפים אותו באש וכן נלבנה לבנים ( בראשית יא ) : <b>וחקות .</b> מלשון חקיקה :
<h4>{ב}</h4>
<b>מצור .</b> האויבים המקיפים העיר ללכדה יקרא מצור כי מצירים להם : <b>דיק .</b> הוא מגדל הנבנה ללכוד העיר וכן ויבנו עליה דיק ( ירמיה נב ) : <b>סוללה .</b> הוא תל העפר ששופכים מול העיר לעלות עליו ללחום על העיר וכן וישפוך סוללה ( דניאל ) יא ) : <b>מחנות .</b> צבאות עם : <b>כרים .</b> הם אילי ברזל עשויים לנגח בהם בתומה להפילה כמוזכר ביוסיפון :
<h4>{ג}</h4>
<b>מחבת .</b> שם כלי מה : <b>קיר .</b> כותל : <b>והכינותה .</b> מל' הכנה : <b>אות .</b> סימן : <b>לבית ישראל .</b> על בני יהודה יאמר :
<h4>{ד}</h4>
<b>תשא .</b> ענין כפרה וסליחה כמו כל תשא עון ( הושע יד ) :
<h4>{ה}</h4>
<b>שני .</b> מל' שנה : <b>ונשאת .</b> תכפר :
<h4>{ו}</h4>
<b>וכלית .</b> ענין השלמה :
<h4>{ז}</h4>
<b>תכין .</b> מל' הכנה : <b>חשופה .</b> גלויה כמו חשפי שובל ( ישעיה מו ) :
<h4>{ט}</h4>
<b>חטין וגו' .</b> שמות מיני תבואה וקטניות : ( , א ) במשורה . כן נקרא מדת הלח כמו במשקל ובמשורה ( ויקרא יט ) : <b>ההין .</b> שם מדה :
<h4>{יב}</h4>
<b>ועוגת .</b> היא חררה האפויה ע''ג גחלים וכן עוגה בלי הפוכה ( הושע ז ) : <b>בגללי .</b> כן יקרא הצואה כמו כגללו לנצח יאבד ( איוב כ ) ולתוספת ביאור אמר צאת האדם : <b>תעגנה .</b> מל' עוגה ר''ל תאפינה לעשותה עוגה :
<h4>{יג}</h4>
<b>טמא .</b> ר''ל נתעב ומאוס : <b>אדיחם .</b> מל' דחיה :
<h4>{יד}</h4>
<b>אהה .</b> ענין לשון צעקת יללה : <b>מטומאה .</b> תעובה ומאוס : <b>פגול .</b> מתועב כמו ומרק פגולים ( ישעיה סו ) :
<h4>{טו}</h4>
<b>צפיעי .</b> הוא הגלל כי כל דבר היוצא מהגוף קרוי צפיעי כמ''ש הצאצאים והצפיעות ( שם כב ) : <b>תחת .</b> במקום :
<h4>{טז}</h4>
<b>מטה לחם .</b> על שנשענים על הלחם לאכול לשבעה אמר מטה בל' שאלה וכן כל מטה לחם שבר ( תהלים קה ) : <b>ובשממון .</b> בתמהון לבב והוא ענין בהלה ופחד :
<h4>{יז}</h4>
<b>ונשמו .</b> ענין תמהון : <b>ונמקו .</b> ענין המסה והמגה :
<h3>יחזקאל פרק-ה</h3>
<h4>{א}</h4>
<b>תער .</b> הוא סכין העשוי לגלח בו וכן והעבירו תער ( במדבר ה ) : <b>הגלבים .</b> הם הספרים ודוגמתו בתרגום ירושלמי כי כל זקן גרועה ( ישעיה טו ) תרגמו ירושלמי גליבא : <b>והעברת .</b> מל' העברה ועל הגלוח יאמר :
<h4>{ב}</h4>
<b>באור .</b> להב אש : <b>תבעיר .</b> מל' הבערה ושריפה : <b>כמלאות .</b> ענין השלמה כמו ובמלאת ימי טהרה ( ויקרא יב ) : <b>תזרה .</b> ענין פזור כמו וזריתי פרש ( מלאכי ב ) : <b>אריק .</b> מל' ריק ור''ל ארוקן אותה מתערה וכן שמן תורק ( ש''ה א ) :
<h4>{ג}</h4>
<b>וצרת .</b> ענין קשירה כמו מי צרר מים בשמלה ( משלי ל ) : <b>בכנפיך .</b> שולי הבגד כמו על כנפי בגדיהם ( במדבר ט ) :
<h4>{ו}</h4>
<b>ותמר .</b> מל' תמורה וחלוף :
<h4>{ז}</h4>
<b>יען .</b> בעבור : <b>המנכם .</b> מל' המון עם רב :
<h4>{י}</h4>
<b>וזריתי .</b> אפזר : <b>רוח .</b> פאה ועבר :
<h4>{יא}</h4>
<b>אם לא .</b> הוא ענין לשון שבועה כאדם האומר אם לא כן הוא יהיה כזאת וכזאת וגזם ולא אמר וכן אם לא בתים רבים ( ישעיה ה ) : <b>שקוציך .</b> מל' שקץ ותעוב ועל העכו''ם יאמר : <b>אגרע .</b> מל' גרעון וחסרון : <b>תחוס .</b> מל' חס וחמלה :
<h4>{יב}</h4>
<b>אזרה .</b> אפזר : <b>אריק .</b> מל' ריק :
<h4>{יג}</h4>
<b>וכלה .</b> ענין השלמה : <b>והניחותי .</b> מל' הנחה : <b>והנחמתי .</b> מל' תנחומין : <b>בקנאתי .</b> ענין כעס כמו הס קנאוני בלא אל ( דברים לב ) :
<h4>{יד}</h4>
<b>לחרבה .</b> מל' חורבן :
<h4>{טו}</h4>
<b>וגדופה .</b> ענין בזיון כמו את מי חרפת וגדפת ( ישעיה לז ) : <b>מוסר .</b> ענין תוכחה : <b>ומשמה .</b> ענין תמהון : <b>ובתוכחות .</b> ענין גמול עונש כמו והוכח במכאוב ( איוב לג ) :
<h4>{טז}</h4>
<b>חצי .</b> ר''ל מכאובי ואמר בל' שאלה על השדפון והירקון והארבה : <b>מטה לחם .</b> משען לחם ועל כי נשענים על הלחם לאכול לשבעה אמר מטה בלשון שאלה :
<h4>{יז}</h4>
<b>ושכלוך .</b> ענין מיתה והריגה כמו ושכלה אתכם ( ויקרא כו ) :
<h3>יחזקאל פרק-ו</h3>
<h4>{ג}</h4>
<b>לאפיקים .</b> הוא מקום שנגרים שם המים בחוזק וכן כאפיקים בנגב ( תהלים קכו ) : <b>ולגאיות .</b> מל' גיא ועמק : <b>במותיכם .</b> מל' במה והיא מזבח אבנים :
<h4>{ד}</h4>
<b>ונשמו .</b> מל' שממון : <b>חמניכם .</b> מל' חמה : <b>גלוליכם .</b> דמות עובדי גלולים המאוס כגלל של רעי :
<h4>{ה}</h4>
<b>פגרי .</b> כן נקרא גוף ההרוג : <b>וזריתי .</b> אפזר :
<h4>{ו}</h4>
<b>תחרבנה .</b> מל' חורבן : <b>תישמנה .</b> מל' שממון : <b>למען .</b> ענינו כמו באופן ודוגמתו למען ינוח שורך וחמורך ( שמות כג ) ופירושו באופן שינוח : <b>יחרבו .</b> מל' חורבן : <b>ויאשמו .</b> מל' שממון : <b>ונשבתו .</b> ענין בטול כמו שבת נוגש ( ישעיה יד ) : <b>ונגדעו .</b> ענין כריתה כמו שקמים גדעו ( שם ט ) : <b>ונמחו .</b> ענין המסה והשחתה כמו וימח את כל היקום ( בראשי' ז ) :
<h4>{ח}</h4>
<b>והותרתי .</b> מל' נותר : <b>פליטי .</b> ענין המלטה כמו ויבא הפליט ( שם יד ) : <b>בהזרותיכם .</b> ענין פזור :
<h4>{ט}</h4>
<b>נשבו .</b> מל' שבי : <b>נשברתי .</b> ענינו שברון רוח ועצבון : <b>את לבם .</b> מלבם וכן ואת עיניהם כמו כצאתי את העיר ( שמות ט ) ופירושו מהעיר : <b>הזונה .</b> הוא ענין הסרה ולכן נקראת מנאפת בשם זונה על כי סרתה מדרכי הצניעות : <b>ונקוטו .</b> ענין כריתה כמו ונקטתם בפניכם ( לקמן כ ) : <b>אל הרעות .</b> על הרעות ר''ל בעבור הרעות :
<h4>{י}</h4>
<b>אל חנם .</b> בחנם כמו ואל הארון ( שמות כה ) :
<h4>{יא}</h4>
<b>ורקע .</b> ענין שטיחה ורדוד כמו כסף מרוקע ( ירמיה י ) : <b>אח .</b> כמו האח ויאמר על השמחה ועל הצער :
<h4>{יב}</h4>
<b>והנצור .</b> מלשון מצור :
<h4>{יג}</h4>
