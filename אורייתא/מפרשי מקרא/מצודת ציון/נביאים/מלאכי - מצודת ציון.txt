<h1>מלאכי - מצודת ציון</h1>
<h3>מלאכי פרק-א</h3>
<h4>{א}</h4>
<b>משא .</b> נבואה ולתוספת ביאור אמר דבר ה' :
<h4>{ג}</h4>
<b>לתנות .</b> מלשון תנין והוא מין נחש :
<h4>{ד}</h4>
<b>רששנו .</b> מלשון רש ועני וכן מתרושש והון רב ( משלי י''ג ) : <b>גבול .</b> ענין מחוז : <b>זעם .</b> ענין כעס :
<h4>{ו}</h4>
<b>אדונים .</b> דרך המקרא להזכיר שם אדנות בלשון רבים וכן אדוני יוסף ( בראשית ל''ט ) :
<h4>{ז}</h4>
<b>מגישים .</b> מקריבים : <b>לחם .</b> כן יקרא כל הקרב על המזבח וכן את קרבני לחמי ( במדבר כח ) : <b>מגואל .</b> מתועב ומטונף כמו מוראה ונגאלה ( צפניה ג ) : <b>גאלנוך .</b> ר''ל גאלנו ותעבנו קרבניך : <b>שלחן .</b> ר''ל המזבח שמסדרין עליו הקרבן לה' כעין שמסדרין על השולחן וכמ''ש וידבר אלי זה השלחן וגו' ( יחזקאל מ''א ) :
<h4>{ח}</h4>
<b>פסח .</b> חגר וצולע : <b>לפחתך .</b> ענין שר ושליט כמו פחת יהודה ( חגי א ) : <b>הישא פניך .</b> זה הלשון יאמר על הכבוד וכן זקן ונשוא פנים ( ישעיה ג ) :
<h4>{ט}</h4>
<b>חלו .</b> ענין תפלה כמו ויחל משה ( שמות לב ) : <b>ויחננו .</b> מלשון חנינה וחמלה :
<h4>{י}</h4>
<b>תאירו .</b> תבעירו כמו נשים באות מאירות אותה ( ישעיה כז ) : <b>חפץ .</b> ענין רצון : <b>ומנחה .</b> כן יקראו כל הקרבנות שבאים למנחה ולדורון וכן וערבה לה' מנחת ( לקמן ג ) :
<h4>{יא}</h4>
<b>מבואו .</b> ענין שקיעה כמו כי בא השמש ( ראשית כא ) : <b>מקטר .</b> מל' הקטרה : <b>מגש .</b> ענין הקרבה :
<h4>{יב}</h4>
<b>וניבו .</b> ענין אמירה כמו בורא ניב שפתים ( ישעיה נז ) : <b>אכלו .</b> מאכלו :
<h4>{יג}</h4>
<b>מתלאה .</b> ענין עיפות ויגיעה כמו את כל התלאה ( שמות יח ) : <b>והפחתם .</b> ענין דאבון ותוגה כמו מפח נפש ( איוב יא ) או הוא מל' הפחה ונשיבת רוח :
<h4>{יד}</h4>
<b>וארור .</b> ענין קללה : <b>נוכל .</b> ענין ערמימות כמו ויתנכלו אותו ( בראשית לז ) :
<h3>מלאכי פרק-ב</h3>
<h4>{ב}</h4>
<b>המארה וארותי .</b> מל' ארירה וקללה :
<h4>{ג}</h4>
<b>גוער .</b> ענין זעקת נזיפה : <b>וזריתי .</b> אפזר כמו מזרה ישראל ( ירמיה לא ) : <b>פרש .</b> זבל וצואה כמו וקרבו ופרשו ( ויקרא ד ) : <b>חגיכם .</b> כן יקראו הקרבנות לפי שרובם באים בחג וכן אסרו חג בעבותים ( תהלים קיח ) :
<h4>{ה}</h4>
<b>נחת .</b> ענין פחד כמו ויהי חתת אלהים ( בראשית לה ) :
<h4>{ו}</h4>
<b>ובמישור .</b> מל' ישר :
<h4>{ז}</h4>
<b>מלאך .</b> שליח כמו ויאמר חגי מלאך ה' ( חגי א ) :
<h4>{ח}</h4>
<b>סרתם .</b> מל' הסרה : <b>הכשלתם .</b> מל' מכשול :
<h4>{ט}</h4>
<b>כפי .</b> כשיעור כמו איש כפי אכלו ( שמות יב ) : <b>ונושאים פנים .</b> זה הלשון יאמר על הכבוד והחנופה וכן ופני רשעים תשאו ( תהלים פב ) :
<h4>{י}</h4>
<b>נבגד .</b> ענין פשע ומרד : <b>באחיו .</b> ענינו אחוה וקורבה : <b>לחלל .</b> ענין הבטול וההפרה :
<h4>{יב}</h4>
<b>ער .</b> מל' התעוררות : <b>ועונה .</b> מל' עליה ותשובה : <b>מאהלי .</b> ר''ל מאהלי תורה : <b>ומגיש .</b> מקריב : <b>מנחה .</b> תשורה וקרבן :
<h4>{יג}</h4>
<b>כסות .</b> מל' כסוי : <b>ואנקה .</b> ענין צעקת אנחה כמו אנקת אסיר ( תלים קב ) : <b>פנות .</b> מל' הפנה וסבוב :
<h4>{טו}</h4>
<b>ושאר .</b> הנה יתרון מעלה כי כמו שמלת יותר היא משותפת להשארה וליתרון מעלה כמ''ש יתר מדעהו צדיק ( משלי יב ) כן מלת שאר משותפ' לשניה' : <b>רוח .</b> ר''ל רוח חכמה ויראת ה' : <b>ברוחכם .</b> ענין רצון , כמו הנני נותן בו רוח ( ישעיה לז ) :
<h4>{טז}</h4>
<b>חמס .</b> ענין מחשבה רעה ושנאה :
<h4>{יז}</h4>
<b>הוגעתם .</b> מל' יגיעה ועייפות : <b>חפץ .</b> ענין רצון :
<h3>מלאכי פרק-ג</h3>
<h4>{א}</h4>
<b>ופנה .</b> ענין הסרה ובעור כמו פנית לפניה ( תהלים פ ) :
<h4>{ב}</h4>
<b>מכלכל .</b> יסבול כמו ומי יכילנו ( יואל ב ) : <b>מצרף .</b> ענין זכוך מהסיג כמו צרפתיך ולא בכסף ( ישעיה מח ) : <b>וכבורית .</b> שם עשב המנקה וכן ותרבי לך בורית ( ירמיה ב ) : <b>מכבסים .</b> מלשון כבוס :
<h4>{ג}</h4>
<b>כסף .</b> כמו ככסף : <b>וזקק .</b> ענין זכוך וצרוף כמו מזוקק שבעתים ( תהלים יב ) : <b>מגישי .</b> ענין הקרבה : <b>מנחה .</b> ר''ל תשורה וכל הקרבנות בכלל : <b>בצדקה .</b> ר''ל כדת וכראוי :
<h4>{ד}</h4>
<b>וערבה .</b> ענין מתיקות וכן ושנתי ערבה לי ( ירמיה לא ) : <b>קדמוניות .</b> מלשון קודם :
<h4>{ה}</h4>
<b>ובעושקי .</b> ענין גזל : <b>ומטי .</b> מל' הטייה :
<h4>{ו}</h4>
<b>שניתי .</b> מלשון השתנות וחלוף : <b>כליתם .</b> מל' תכלית וסוף :
<h4>{ז}</h4>
<b>למימי .</b> הלמ''ד נוספת :
<h4>{ח}</h4>
<b>היקבע .</b> ענין עושק כמו וקבע את קבעיהם נפש ( משלי כד ) :
<h4>{ט}</h4>
<b>במארה .</b> נארים . מל' ארירה וקללה :
<h4>{י}</h4>
<b>טרף .</b> ענין מזון והוא מושאל ממזון החיה הבאה ע''י טריפה : <b>ובחנוני .</b> מל' בחינה ונסיון : <b>נא .</b> עתה : <b>ארובות .</b> חלונות כמו וארובות השמים נפתחו ( ראשית ז ) : <b>והריקותי .</b> ענין הזלה ושפיכה כמו על הארץ יריקו ( קהלת יא ) : <b>בלי .</b> הוא כמו אל ולא : <b>די .</b> ענין הספוק :
<h4>{יא}</h4>
<b>וגערתי .</b> ענין זעקת נזיפה : <b>תשכל .</b> ענין חסרון הפרי והוא מושאל ממחוסר בנים הנקרא שכול :
<h4>{יב}</h4>
<b>ואשרו .</b> ענין הלול ושבח כמו אשרי האיש ( ת''א ) : <b>ארץ .</b> ר''ל בארץ : <b>חפץ .</b> ענין רצון :
<h4>{יג}</h4>
<b>חזקו .</b> ענין קושי :
<h4>{יד}</h4>
<b>שוא .</b> ענין חנם : <b>בצע .</b> ענין תועלת כמו מה בצע בדמי ( שם ל ) : <b>קדורנית .</b> ענין שחרות כמו שמש וירח קדרו ( יואל ד ) ור''ל שחוח ובהכנעה :
<h4>{טו}</h4>
<b>מאשרים .</b> משבחים : <b>נבנו .</b> מלשון בנין : <b>בחנו .</b> מלשון בחינה ונסיון : <b>וימלטו .</b> ענין הצלה :
<h4>{טז}</h4>
<b>ויקשב .</b> ענין האזנה : <b>ספר .</b> מגילה : <b>ולחושבי .</b> מל' מחשבה :
<h4>{יז}</h4>
<b>סגלה .</b> ענין אוצר כמו וסגולת מלכים ( קהלת ב ) : <b>וחמלתי .</b> ענין רחמנות :
<h4>{יט}</h4>
<b>בוער .</b> ענין שרפה : <b>קש .</b> הוא התבן הדק : <b>ולהט .</b> ענין שרפה והתלהבות כמו אש לוהט ( תהלים קד ) : <b>לא יעזוב .</b> לא יניח :
<h4>{כ}</h4>
<b>ופשתם .</b> ענין רבוי ור''ל רבוי הבשר וכן כי תפושו כעגלה דשה ( ירמיה ג ) : <b>מרבק .</b> כן יקרא דיר העגלים מקום שמתפטמים שם וכן ועגלים מתוך מרבק ( עמוס ו ) :
<h4>{כא}</h4>
<b>ועסותם .</b> ענין מיעוך וכתישה וכן מעסיס רמוני ( ש''ה ה ) והוא המיץ מן הרמונים אשר יקרא כן ע''ש שממעכים וכותשים הרמונים להוציא המיץ : <b>כפות .</b> כמו ששטחית היד נקרא כף היד כן שטחית הרגל נקרא כף הרגל :
<h4>{כב}</h4>
<b>בחורב .</b> הוא הר סיני : <b>על כל ישראל .</b> בעבור כל ישראל :
<h4>{כג}</h4>
<b>לפני .</b> קודם :
<h4>{כד}</h4>
<b>והשיב .</b> מל' תשובה : <b>על בנים .</b> ר''ל על ידי בנים , או על הוא כמו עם : <b>חרם .</b> ענין כריתה ואבדון כמו וחרם לא יהיה עוד וישבה ירושלים לבטח ( זכריה י''ד ) :
