<h1>מיכה - מצודת ציון</h1>
<h3>מיכה פרק-א</h3>
<h4>{א}</h4>
<b>המורשתי .</b> מן מורישה והיא מערי יהודה :
<h4>{ב}</h4>
<b>הקשיבי .</b> ענין שמיעה :
<h4>{ג}</h4>
<b>ודרך .</b> מלשון דריכה ורמיסה : <b>במתי .</b> ענין גובה כמו ועל במותי ידריכני ( חבקוק ג ) :
<h4>{ד}</h4>
<b>ונמסו .</b> ענין המגה והמסה : <b>יתבקעו .</b> מל' בקיעה : <b>כדונג .</b> הוא השעוה וכן היה לבי כדונג ( תהלים כד ) : <b>מוגרים .</b> ענין הזלה ושפיכה כמו וכמים הנגרים ( ש''ב יד ) : <b>במורד .</b> במקום מדרון והוא מלשון ירידה וכן ויכום במורד ( יהושע ז ) :
<h4>{ה}</h4>
<b>במות .</b> מזבחות של עכו''ם :
<h4>{ו}</h4>
<b>לעי .</b> ענין תל וגל כמו שמו את ירושלים לעיים ( תהלים עט ) : <b>למטעי .</b> מלשון נטיעה : <b>והגרתי .</b> ענין הזלה ושפיכה ואמר בלשון שאלה על ההפלה : <b>לגי .</b> לעמק :
<h4>{ז}</h4>
<b>יכתו .</b> מל' כתיתה ושבירה : <b>אתנניה .</b> מל' אתנן : <b>עצביה .</b> העכו''ם תקרא כן על כי מעצבת לב עובדיה קורא ואינו נענה :
<h4>{ח}</h4>
<b>ואילילה .</b> מל' יללה : <b>שולל .</b> ענין שטות וטרוף הדעת וכן מוליך יועצים שולל ( איוב יב ) : <b>כתנים .</b> מין נחש : <b>כבנות יענה .</b> שם עוף בת היענה :
<h4>{ט}</h4>
<b>אנושה .</b> כאובה ביותר כמו אנוש חצי ( שם לד ) : <b>שער עמי .</b> עיר עמי וכן באחד שעריך ( דברים יז ) :
<h4>{י}</h4>
<b>התפלשי .</b> ענין הגלגול והלכלוך וכן והתפלשו אדירי הצאן ( ירמיה כה ) :
<h4>{יא}</h4>
<b>עריה .</b> ענין גלוי כמו ואת ערום ועריה ( יחזקאל טז ) : <b>בשת .</b> הוא כנוי לערוה וכן והחזיקה במבושיו ( דברים כה ) :
<h4>{יב}</h4>
<b>חלה .</b> מל' חיל וחלחלה :
<h4>{יג}</h4>
<b>רתום .</b> הוא ענין אסור וקשירה והוא לשון ערבי : <b>לרכש .</b> שם מין בהמה הקלה לרוץ כמו רכבי הרכש ( אסתר ח ) : <b>לבת ציון .</b> לעדת ציון : <b>פשעי .</b> כן יקראו הפסילים כי מפשעת אנשים וכן והפשע שומם ( דניאל ח ) :
<h4>{יד}</h4>
<b>שלוחים .</b> כן יקרא מתן גדול וכן ויתנה שלוחים לבתו ( מ''א ט ) : <b>על .</b> כמו אל : <b>מורשת .</b> מל' ירושה : <b>לאכזב .</b> ענין עזיבה והפסק כמו כי תכזבי ( ישעיה נז ) :
<h4>{טו}</h4>
<b>אבי .</b> כמו אביא באלף וכן אל יני ראשי ( תהלים קמא ) ומשפטו יניא באל''ף :
<h4>{טז}</h4>
<b>קרחי .</b> ענין מריטת השער וכן והקריחי עליך ( יחזקאל כז ) : <b>וגוזי .</b> מל' גזיזה וחתוך וכן גזי נזרך ( ירמיה ז ) : <b>תענוגיך .</b> מל' עונג ותפנוק :
<h3>מיכה פרק-ב</h3>
<h4>{א}</h4>
<b>לאל .</b> ענין כח ואומץ כמו יש לאל ידי ( בראשית לא ) :
<h4>{ב}</h4>
<b>ונשאו .</b> לקחו כמו ישא ברכה ( תהלים כד ) :
<h4>{ג}</h4>
<b>תמישו .</b> ענין הסרה כמו לא ימיש עמוד הענן ( שמות יג ) : <b>רומה .</b> מלשון התרוממות :
<h4>{ד}</h4>
<b>ישא .</b> ענין הרמת קול : <b>ונהה נהי נהיה .</b> עניינו קינה ויללה כמו אשא בכי ונהי ( ירמיה ט ) : <b>שדוד נשדונו .</b> מלשון שדידה ועושק : <b>חלק .</b> אחוזת נחלה כמו חלקת יואב ( ש''ב יד ) : <b>ימיר .</b> מל' תמורה וחלוף : <b>ימיש .</b> ענין הסרה : <b>לשובב .</b> מל' השבה :
<h4>{ה}</h4>
<b>חבל .</b> מל' קו המדה :
<h4>{ו}</h4>
<b>תטיפו יטיפון .</b> הוא מל' נטיפה והזלה והושאל בענין הדבור של נבואה וכן והטף אל דרום ( יחזקאל כא ) : <b>יסג .</b> מל' השגה וקרוב ועם היא בסמ''ך וכן ותסג ולא תפליט ( לקמן ו ) : <b>רוח .</b> רצון : <b>מעלליו .</b> מעשיו :
<h4>{ח}</h4>
<b>ואתמול .</b> ענינו כמו אתמול בחול''ם וכן כי ערוך מאתמול תפתה ( ישעיה ל ) : <b>ממול .</b> מנגד : <b>שלמה .</b> מלבוש : <b>אדר .</b> מלבוש יקר כמו ויעבר אדרתו ( יונה ג ) : <b>תפשיטון .</b> ענין הסרת המלבוש :
<h4>{ט}</h4>
<b>עולליה .</b> בנים הקטנים : <b>הדרי .</b> מל' הדר ויופי ועל העושר יאמר :
<h4>{י}</h4>
<b>המנוחה .</b> מקום מנוח ר''ל נחלה : <b>תחבל וחבל .</b> מל' חבלה והשחתה : <b>נמרץ .</b> ענין חוזק כמו קללה נמרצת ( מ''א ב ) :
<h4>{יא}</h4>
<b>לו .</b> ענינו כמו אם וכן לו חיה רעה ( יחזקאל יד ) : <b>אטיף .</b> הושאל בענין הדבור של נבואה :
<h4>{יב}</h4>
<b>בצרה .</b> בלשון שאלה אמר על דיר הצאן שהיא להם למבצר . הדברו . ענין הנהגה וכן ורעו כבשים כדברם ( ישעיה ה ) : <b>תהימנה .</b> מל' המייה ושאון :
<h4>{יג}</h4>
<b>הפורץ .</b> מלשון פרצה ושבירה :
<h3>מיכה פרק-ג</h3>
<h4>{א}</h4>
<b>נא .</b> עתה : <b>וקציני .</b> ענין שררה :
<h4>{ב}</h4>
<b>ושארם .</b> כן תקרא הבשר כמו אם יכין שאר לעמו ( תהלים עח ) :
<h4>{ג}</h4>
<b>הפשיטו .</b> ענין הסרה : <b>פצחו .</b> ענין פתיחה כמו פצחו ורננו וזמרו ( שם צח ) : <b>יפרשו .</b> מל' פרוסה ושבירה ועם היא בשי''ן וכן פורש אין להם ( איכה ד ) : <b>בסיר .</b> קדרה כמו שפות הסיר ( מ''ב ד ) : <b>קלחת .</b> גם הוא כלי בשול כמו או בקלחת או בפרור ( ש''א ב ) :
<h4>{ד}</h4>
<b>יענה .</b> ענין השבה : <b>הרעו .</b> מל' רעה :
<h4>{ה}</h4>
<b>המתעים .</b> מל' תועה : <b>הנושכים .</b> חותכים לאכול וכן אם ישוך הנחש ( קהלת י ) : <b>בשניהם .</b> מל' שן : <b>וקדשו .</b> ענין הזמנה כמו קדשו קהל ( יואל ב ) :
<h4>{ו}</h4>
<b>מחזון .</b> ענין נבואה : <b>מקסום .</b> מין כשוף : <b>ובאה .</b> ענין שקיעה כמו כי בא השמש ( ברא' כח ) : <b>וקדר .</b> ענין חשך ושחרות כמו שמש וירח קדרו ( יואל ד ) :
<h4>{ז}</h4>
<b>וחפרו .</b> ענין כלימה כמו וחפרה הלבנה ( ישעיה כד ) : <b>ועטו .</b> ענין עטיפה : <b>שפם .</b> הוא שער השפתים כמו ועל שפם יעטה ( ויקרא יג ) : <b>מענה .</b> ענין אמירה וכן ומה' מענה לשון ( משלי טז ) :
<h4>{ח}</h4>
<b>את רוח .</b> מן רוח וכן כצאתי את העיר ( שמות ט ) :
<h4>{ט}</h4>
<b>המתעבים .</b> מל' תעוב ושקץ : <b>יעקשו .</b> ענין עקום ומעוקל כמו ועם עקש תתפל ( ש''ב כב ) :
<h4>{יא}</h4>
<b>במחיר .</b> עניינו דמי ערך השוי וכן יקנה מקנה במחיר ( מ''א י ) : <b>יורו .</b> מל' הוראה ולמוד : <b>ישענו .</b> מל' השענה ובטחון :
<h4>{יב}</h4>
<b>בגללכם .</b> בעבורכם וכן בגלל יוסף ( בראשית לט ) : <b>שדה .</b> כשדה : <b>עיין .</b> כמו עיים במ''ם וכן לקץ הימין ( דניאל יב ) ועיים הוא גלי אבנים וכן לעיי השדה ( לעיל א ) : <b>לבמות .</b> מל' במה :
<h3>מיכה פרק-ד</h3>
<h4>{א}</h4>
<b>באחרית .</b> בסוף : <b>נכון .</b> מוכן ומתוקן : <b>ונהרו .</b> ימשכו כנהר וכן ונהרו אל טוב ה' ( ירמיה לא ) :
<h4>{ב}</h4>

<h4>{ג}</h4>
<b>והוכיח .</b> ענין ברור דברים : <b>עצומים .</b> חזקים : <b>וכתתו .</b> ענין כתישה : <b>לאתים .</b> שם כלי חפירה וכן את מחרשתו ואת אתו ( ש''א י''ג ) : <b>למזמרות .</b> שם הכלי שכורתין בו הומורות מל' וכרמך לא תזמור ( ויקרא כה ) :
<h4>{ד}</h4>
<b>מחריד .</b> מל' חרדה :
<h4>{ו}</h4>
<b>הצולעה .</b> החגרת כמו והוא צולע על ירכו ( בראשית לב ) :
<h4>{ז}</h4>
<b>והנהלאה .</b> ענין יגיעה כמו נלאה מואב ( ישעי' טז ) :
<h4>{ח}</h4>
<b>עופל .</b> ענין מגדל ומבצר כמו עופל ובחן ( שם לב ) : <b>עדיך .</b> אליך כמו תגע עדיך ( איוב ד' ) : <b>תאתה .</b> ענין ביאה כמו אתא בוקר ( ישעיה כא ) :
<h4>{ט}</h4>
<b>תריעי רע .</b> ענין חבר ואוהב : <b>החזיקך .</b> ענין אחיזה : <b>חיל .</b> חלחלה ורתת :
<h4>{י}</h4>
<b>חולי .</b> מל' חלחלה : <b>וגוחי .</b> ענין האנחה כי האנחה היא הוצאת הנשימה והוא מענין כי אתה גוחי מבטן ( תהלים כב ) שהוא ענין המשכה והוצאה וקרוב הוא אל לשון גניחה המוזכר בדרז''ל : <b>מקריה .</b> מעיר : <b>תנצלי .</b> ענין ריקות כמו ויתנצלו בני ישראל את עדים ( שמות ל''ג ) : <b>מכף .</b> מיד :
<h4>{יא}</h4>
<b>תחנף .</b> תתחייב כמו ולא תחניפו את הארץ ( במדבר ל''ה ) : <b>ותחז .</b> כמו ותחזה והוא ענין ראיה והסתכלות כמו ואתה תחזה ( שמות י''ח ) :
<h4>{יב}</h4>
<b>כעמיר .</b> כן יקרא אגודת השבלים כמו וחצנו מעמר ( תהלים קכ''ט ) : <b>גורנה .</b> הוא המקום שמכניסין שם התבואה :
<h4>{יג}</h4>
<b>ודושי .</b> מלשון דישה : <b>ופרסותיך .</b> הם כעין מנעלים שברגלי הבהמות כמו בפרסות סוסיו ( יחזקאל כ''ז ) ואמר בדרך שאלה : <b>נחושה .</b> נחושת : <b>והדקות .</b> ענין שחיקה כמו אדיקם ארקעם ( ש''ב כ''ב ) : <b>והחרמתי .</b> ענין הקדש כמו כל חרם בישראל ( במדבר י''ח ) : <b>בצעם .</b> חמדת עשרם כמו בצע כסף ( שופטים ה ) : <b>וחילם .</b> גם הוא ענין עושר כמו ושללו חילך ( יחזקאל כ''ו ) :
<h4>{יד}</h4>
<b>תתגודדי .</b> ענין שריטת הבשר כמו לא תתגודדו ( דברים י''ד ) : <b>גדוד .</b> ענין מחנה וכן פשט גדוד ( הושע ו ) : <b>מצור .</b> כן יקרא הקפת גייסות סביב העיר ללכדה : <b>בשבט .</b> בשרביט : <b>הלחי .</b> כן נקרא המקום שאצל העין :

<h4>{יא}</h4>

<h4>{טו}</h4>

<h4>{טז}</h4>
<b>במועצותם .</b> מלשון עצה : <b>לשמה .</b> ענין תמהון : <b>לשרקה .</b> הוא קול הבאה בקבוץ השפתים והדרך לעשותם כשרואים דבר חשוב שנחרב וכן לשמה ולשרקה ( ירמיה יט ) : <b>תשאו .</b> מלשון משא וסבל :
<h3>מיכה פרק-ז</h3>
<h4>{א}</h4>

<h4>{ב}</h4>

<h4>{ג}</h4>
<b>להיטיב .</b> ענין חוזק כמו ההיטיב חרה לך ( יונה ד' ) : <b>הות .</b> ענין שברון נפש וכן הות לאביו בן כסיל ( משלי י''ט ) : <b>ויעבתוה .</b> ר''ל יחזקו כחבל עב :

<h4>{ו}</h4>

<h4>{ז}</h4>
