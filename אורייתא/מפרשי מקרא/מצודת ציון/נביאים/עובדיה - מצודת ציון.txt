<h1>עובדיה - מצודת ציון</h1>
<h3>עובדיה פרק-א</h3>
<h4>{א}</h4>
<b>וציר .</b> ענין שליח כמו וציר אמונים ( משלי יג ) :
<h4>{ב}</h4>
<b>בזוי .</b> מבוזה :
<h4>{ג}</h4>
<b>השיאך .</b> ענין הסתה כמו הנחש השיאני ( בראשית ג ) : <b>שוכני .</b> היו''ד יתירה : <b>בחגוי .</b> ענין בקוע וחריץ כמו יונתי בחגוי הסלע ( ש''ה ב ) :
<h4>{ד}</h4>
<b>קנך .</b> מל' קן והוא מדור העופות :
<h4>{ה}</h4>
<b>שודדי .</b> ענין גזל ועושק : <b>נדמיתה .</b> ענין כריתה ואבדון כמו ער מואב נדמה ( ישעיה טו ) : <b>דים .</b> די הספוק כמו והמלאכה היתה דים ( שמות לו ) : <b>בוצרים .</b> כן יקראו תולשי ענבי כרם כמו כי תבצור כרמך ( דברי' כד ) : <b>עוללות .</b> הם הענבים הקטנים הגרועים כמו לא תעולל ( , שם ) :
<h4>{ו}</h4>
<b>נחפשו .</b> מלשון חפוש וחקירה כמו משם אחפש ( עמוס ט ) : <b>נבעו .</b> ענין דרישה וחפוש ודומה לו אם תבעיון בעיו ( ישעיה כא ) שהוא ענין דרישת שאלה וכמו שמלת בקשה הונח לדריש' שאלה ולחפוש כן מלת בעה הונח לשניהם : <b>מצפוניו .</b> מל' צפון ונסתר :
<h4>{ז}</h4>
<b>שלחוך .</b> ענין לויה כמו ואברה' הולך עמם לשלחם ( בראשי' יח ) : <b>השיאך .</b> ענין הסתה : <b>יכלו .</b> מל' יכולת ונצחון : <b>לחמך .</b> ר''ל אוכלי לחמך : <b>מזור .</b> ענין מכה ותקרא כן ע''ש שזורים עליה אבק מסממנים המרפאים וכן ולא יגהה מכם מזור ( הושע ה ) : <b>תחתיך .</b> במקומך :
<h4>{ט}</h4>
<b>וחתו .</b> ענין שבירה כמו והחתתי את עילם ( ירמיה מט ) : <b>תימן .</b> הוא אדום היושב בהר שעיר בדרומה של א''י : <b>מקטל .</b> ענין הרג כמו הן יקטלני ( איוב יג ) :
<h4>{י}</h4>
<b>מחמס .</b> ענין עושק :
<h4>{יא}</h4>
<b>מנגד .</b> ממרחק : <b>שבות .</b> מל' שבי : <b>זרים .</b> מלשון זר ונכרי : <b>חילו .</b> ענין עושר כמו ועזבו לאחרים חילם ( תהלים מט ) : <b>ידו .</b> ענין השלכה והפלה כמו ידו אליה ( ירמיה נ ) :
<h4>{יב}</h4>
<b>נכרו .</b> מל' נכר וזר :
<h4>{יג}</h4>
<b>אידם .</b> ענין צער ומקרה רע כמו הלא איד לעול ( איוב לא ) : <b>בחילו .</b> בעשרו :
<h4>{יד}</h4>
<b>תעמוד .</b> ענינו התחזקות על הדבר וההשתדלות עליו ודומה לו אך יונתן וגו' עמדו על זאת ( עזרא י ) : <b>הפרק .</b> ענין שביר' כמו ופרסיהן יפרק ( זכריה יא ) . פליטיו . ענין שארית כמו פליט ושריד ( איכה ב ) : <b>תסגר .</b> ר''ל מסגר המאסר . <b>שרידיו .</b> ענין שארית :
<h4>{טו}</h4>
<b>קרוב .</b> ר''ל נכון ומזומן לפניו וכן כי קרוב יום אידם ( דברים לב ) :
<h4>{טז}</h4>
<b>ולעו .</b> ענין השחתה ובלבול וכן על כן דברי לעו ( איוב ו ) :
<h4>{יז}</h4>
<b>מורשיהם .</b> מל' ירושה :
<h4>{יח}</h4>
<b>להבה .</b> שלהבת : <b>לקש .</b> הוא תבן הדק : <b>ודלקו .</b> ענין שרפה והבערה :
<h4>{יט}</h4>
<b>והשפלה .</b> העמק :
<h4>{כ}</h4>
<b>החל .</b> כמו החיל ונפלה היו''ד :
