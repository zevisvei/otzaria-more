<h1>צפניה - מצודת ציון</h1>
<h3>צפניה פרק-א</h3>
<h4>{ב}</h4>
<b>אסוף אסף .</b> ענין כליון כמו ואספת את נפשך ( שופטים יח ) והיה ראוי לומר אסף אאסף ותחס' האל''ף :
<h4>{ג}</h4>
<b>את הרשעים .</b> עם הרשעים :
<h4>{ד}</h4>
<b>שאר .</b> שיור : <b>הכמרים .</b> כן יקראו כהני עבודת כוכבים וכן וכמריו עליו יגילו ( הושע י ) : <b>הכהנים .</b> הם מין אחר מתחלפים בלבושיהם :
<h4>{ו}</h4>
<b>הנסוגים .</b> ענין החזרה לאחור כמו ולא נסוג ממך ( תהלים פ ) :
<h4>{ז}</h4>
<b>הס .</b> ענין שתיקה כמו ויהס כלב ( במדבר יג ) : <b>הקדיש .</b> ענין הזמנה כמו קדשו קהל ( יואל ב ) : <b>קרואיו .</b> המזומנים לסעודה וכן בראש הקרואים ( ש''א ט ) :
<h4>{ח}</h4>
<b>ופקדתי .</b> ענין השגחה : <b>נכרי .</b> ענין זרות :
<h4>{ט}</h4>
<b>הדולג .</b> ענין קפיצה : <b>המפתן .</b> אסקופה כמו מתחת מפתן הבית ( יחזקאל מז ) : <b>חמס .</b> עושק :
<h4>{י}</h4>
<b>המשנה .</b> מל' שנים ור''ל בין שתי החומות שהיא משנה לעיר וכן והיא יושבת בירושלים במשנה ( מ''ב כב ) :
<h4>{יא}</h4>
<b>הילילו .</b> מל' יללה : <b>המכתש .</b> שם מקום בירושלים עמוקה ככלי המכתשת שדוכין בה בשמים : <b>נדמה .</b> ענין כריתה כמו ער מואב נדמה ( ישעיה טו ) : <b>עם כנען .</b> ר''ל עדת התגרים כמו כנען בידו מאזני מרמה ( הושע יב ) : <b>נטילי .</b> ענין משא כמו ונטל החול ( משלי כו ) :
<h4>{יב}</h4>
<b>אחפש .</b> ענין בקור ודרישה כמו תמנו חפש מחפש ( תהלים סד ) : <b>הקופאים .</b> הצפים ממעל כי ויצף הברזל ( מ''ב ו ) תרגומו וקפא פרזלא : <b>שמריהם .</b> הוא עכירת המשקה הנופל לשולי הכלי כמו ושקט הוא אל שמריו ( ירמיה מח ) : <b>ירע .</b> מלשון רעה :
<h4>{יג}</h4>
<b>חילם .</b> עשרם כמו ישא את חיל דמשק ( ישעיה ח ) : <b>למשסה .</b> ענין רמיסה כמו למשסה ואין אומר השב ( שם מב ) :
<h4>{יד}</h4>
<b>צורח .</b> ענין שאגת קול גדול כמו יריע אף יצריח ( , שם ) :
<h4>{טו}</h4>
<b>עברה .</b> ענין כעס : <b>ומצוקה .</b> עניינו כמו צרה וכן בארץ צרה וצוקה ( שם ל ) וכפל המלה בשמות נרדפים להפלגות הענין : <b>שואה ומשואה .</b> ענין חושך ושממון כמו אמש שואה ומשואה ( איוב ל ) : <b>וערפל .</b> הוא עב הענן :
<h4>{טז}</h4>
<b>הפנות .</b> הם המגדלים שדרכם לעמוד בפנות וזויות המבצר :
<h4>{יז}</h4>
<b>והצרותי .</b> מל' צרה : <b>ולחומם .</b> כן יקרא גם הבשר כמו כי לחמם לנפשם ( הושע ט ) : <b>כגללים .</b> כצואה כמו כגללו לנצח יאבד ( איוב כ ) :
<h4>{יח}</h4>
<b>קנאתו .</b> ענין כעס וחמה כמו הם קנאוני בלא אל ( דברים לב ) : <b>תאכל .</b> ענין השחתה : <b>נבהלה .</b> מל' בהלה :
<h3>צפניה פרק-ב</h3>
<h4>{א}</h4>
<b>התקוששו וקושו .</b> ענין החפוש והלקיטה היטב כמו לקושש קש ( שמות ה' ) : <b>נכסף .</b> ענין חשק וחמוד כמו נכסוף נכספתה ( בראשית לא ) :
<h4>{ב}</h4>
<b>חוק .</b> גזירת המקום : <b>כמוץ .</b> היא פסולת התבואה כמו כמוץ יסוער מגורן ( הושע יג ) :
<h4>{ג}</h4>
<b>בקשו .</b> ענין דרישה :
<h4>{ה}</h4>
<b>חבל .</b> כן יקרא המחוז כמו חבלי מנשה ( יהושע יז ) :
<h4>{ו}</h4>
<b>נות .</b> מל' נוה ומדור : <b>כרות .</b> ענין חפירה כמו בור כרה ( תהלים ז ) : <b>וגדרות .</b> בנין גדר למרבץ צאן :
<h4>{ז}</h4>
<b>ירעון .</b> מל' מרעה : <b>ירבצון .</b> ענין השכיבה לנוח : <b>ושב .</b> ענין ההשקט ומרגוע כמו בשובה ונחת ( ישעיה ל ) : <b>שביתם .</b> מל' שבי הגולה :
<h4>{ח}</h4>
<b>וגדופי .</b> ענין חרפה כמו וישראל לגדופים ( שם מג ) :
<h4>{ט}</h4>
<b>ממשק .</b> ענין השמעת קול כמו כמשק גבים ( שם לג ) : <b>חרול .</b> מין קוצים כמו תחת חרול יספחו ( איוב ל ) : <b>ומכרה .</b> מל' כרייה וחפירה : <b>יבזום .</b> מל' בזה :
<h4>{י}</h4>
<b>גאונם .</b> מל' גאוה :
<h4>{יא}</h4>
<b>רזה .</b> ענין כחשות כמו במשמניו רזון ( ישעיה ו ) :
<h4>{יג}</h4>
<b>ציה .</b> ענין חורב ויובש :
<h4>{יד}</h4>
<b>חיתו .</b> גם הבהמה תקרא בשם חיה וכן וחיתם לדבר הסגיר ( תהלים עח ) : <b>קאת .</b> קפוד . שמות עופות המצויות בחרבות : <b>בכפתוריה .</b> כן יקרא המשקוף לפי שעושין בו ציורי כפתורים וכן הך הכפתור ( עמוס ט ) : <b>ילינו .</b> ענין התמדה וכן צדק ילין בה ( ישעיה א ) : <b>ישורר .</b> מל' שיר ורנן : <b>חרב .</b> מל' חורבן : <b>בסף .</b> מזוזה כמו וסף השער ( יחזקאל מ ) : <b>ארזה .</b> תקרת הארז : <b>ערה .</b> ענין גלוי כמו וקיר ערה מגן ( ישעיה כ''ב ) :
<h4>{טו}</h4>
<b>העליזה .</b> ענין שמחה כמו יעלזו חסידים ( תהלים קמט ) : <b>ואפסי .</b> כמו ואין : <b>ישרוק .</b> ענין השמעת קול הבאה בקבוץ השפתים כמו שרקו ויניעו ראשם ( איכה ב' ) : <b>יניע .</b> מלשון הנעה והנדה :
<h3>צפניה פרק-ג</h3>
<h4>{א}</h4>
<b>מוראה ונגאלה .</b> ענין שניהם מלוכלך ומטונף כמו ושמתיך כרואי ( נחום ג ) וכמו נגאלו בדם ( ישעיה נט ) וכפל המלה בשמות נרדפות : <b>היונה .</b> מל' אונאה ועניינו קחת ההון וכן מפני חרון היונה ( ירמיה כה ) :
<h4>{ג}</h4>
<b>גרמו .</b> ענין עצם כמו גרמיו כמטיל ברזל ( איוב מ ) :
<h4>{ד}</h4>
<b>פוחזים .</b> קלי הדעת וכן אנשים ריקים ופוחזים ( שופטים ט ) : <b>בוגדות .</b> מל' בוגד : <b>חללו .</b> ענין בזיון : <b>חמסו .</b> ענין הסרה כמו ויחמוס כגן שכו ( איכה ב ) :
<h4>{ה}</h4>
<b>לאור .</b> ר''ל ברור כאור ודומה לו : <b>נעדר .</b> ענין חסרון כמו איש לא נעדר ( ישעיה מ ) :
<h4>{ו}</h4>
<b>נשמו .</b> מל' שממון : <b>פנותם .</b> הם המגדלים הגבוהים כמו ועל הפנות הגבוהות ( לעיל א ) : <b>החרבתי .</b> מל' חרבן : <b>חוצותם .</b> ענין שוק ורחוב : <b>נצדו .</b> ענין שממון כי והשמותי אני את הארץ ( ויקרא כו ) ת''א ואצדי :
<h4>{ז}</h4>
<b>אך .</b> עניינו כמו באמת וכן אך עצמי ובשרי אתה ( בראשית כט ) : <b>מעונה .</b> מדורה כמו ה' מעון אתה היית ( תהלים צ ) : <b>פקדתי .</b> צויתי כמו והוא פקד עלי ( עזרא א ) : <b>עלילותם .</b> מעשיהם :
<h4>{ח}</h4>
<b>חכו .</b> ענין המתנה ואיחור כמו ונסתה ולא תחכה ( מ''ב ט ) : <b>לעד .</b> ענין בזה ושלל כמו בבוקר יאכל עד ( בראשית מט ) :
<h4>{ט}</h4>
<b>שפה .</b> כן יקרא לשון עם ועם וכן שפת רעהו ( שם יא ) : <b>ברורה .</b> כלפי שקורא לשון העכו''ם עמקי שפה כמ''ש עם עמקי שפה ( ישעיה לג ) קורא למולם לשון הקדש שפה ברורה ונקיה : <b>שכם .</b> ר''ל צד אחד ודעה אחת כמו דרך ירצחו שכמה ( הושע ו ) :
<h4>{י}</h4>
<b>עתרי .</b> שם אומה : <b>בת פוצי .</b> עדת פוצי והוא שם אומה : <b>יובילון .</b> ענין הבאה כמו יובל שי ( ישעיה יח ) :
<h4>{יא}</h4>
<b>עלילותיך .</b> מעשיך : <b>עליזי .</b> ענין שמחה :
<h4>{יב}</h4>
<b>עני ודל .</b> ר''ל מוכנע כעני ודל : <b>וחסו .</b> ענין בטחון :
<h4>{יג}</h4>
<b>תרמית .</b> מל' רמיה ומרמה : <b>ירעו .</b> מל' מרעה : <b>ורבצו .</b> ענין השכיבה לנוח :
<h4>{טו}</h4>
<b>פנה .</b> ענין הסרה וביעור כמו פנית לפניה ( תהלים פ ) :
<h4>{טז}</h4>
<b>ירפו ידיך .</b> כן נקרא המתפחד ומתייאש וכן ורפה ידים ( ש''ב יז ) :
<h4>{יז}</h4>
<b>יחריש .</b> ענין שתיקה כמו והחריש לה ( במדבר ל ) :
<h4>{יח}</h4>
<b>נוגי .</b> מל' יגון ותוגה : <b>ממועד .</b> ענין זמן ועת כמו למועד אשוב אליך ( בראשית יח ) : <b>אספתי .</b> ענין הכנסה כמו ואין איש מאסף אותם ( שופטים יט ) : <b>משא .</b> מל' משא וסבל :
<h4>{יט}</h4>
<b>עושה .</b> ענין כתישה ומעוך כמו בעשות ממצרים דדיך ( יחזקאל כג ) : <b>מעניך .</b> מל' עינוי : <b>הצולעה .</b> החגרת כמו והוא צולע על ירכו ( בראשית לב ) : <b>והנדחה .</b> מלשון דחיה : <b>לתהלה .</b> מל' הלול ושבח : <b>לשם .</b> ר''ל לפרסום שם טוב :
<h4>{כ}</h4>
<b>בשובי .</b> ענין השקט ומרגוע כמו בשובה ונחת ( ישעיה ל ) : <b>שבותיכם .</b> מל' שבי הגולה : וסגנים ( יחזקאל כג ) :
<h3>צפניה פרק-ד</h3>
<h4>{ד}</h4>
<b>ספונים .</b> ענין תקרת הכסוי כמו וספון בארז ( ירמיה כב ) : <b>חרב .</b> מלשון חורבן :
<h4>{ה}</h4>
<b>דרכיכם .</b> ר''ל עסקיכם
