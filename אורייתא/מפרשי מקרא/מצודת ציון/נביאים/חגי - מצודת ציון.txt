<h1>חגי - מצודת ציון</h1>
<h3>חגי פרק-א</h3>
<h4>{א}</h4>
<b>פחת .</b> שליט ומושל כמו פחות וסגנים ( יחזקאל כג ) :
<h4>{ד}</h4>
<b>ספונים .</b> ענין תקרת הכסוי כמו וספון בארז ( ירמיה כב ) : <b>חרב .</b> מלשון חורבן :
<h4>{ה}</h4>
<b>דרכיכם .</b> ר''ל עסקיכם וכן מעשות דרכיך ( ישעיה נח ) :
<h4>{ו}</h4>
<b>לשכרה .</b> מלשון שכרות : <b>לחום .</b> מל' חמימות : <b>והמשתכר משתכר .</b> מלשון שכר וריוח : <b>צרור .</b> ענין קשור כמו צרור הכסף ( משלי ז ) : <b>נקוב .</b> מל' נקב וחור :
<h4>{ח}</h4>
<b>וארצה .</b> מלשון רצוי :
<h4>{ט}</h4>
<b>ונפחתי .</b> מלשון הפחה נשיבת רוח הפה : <b>יען .</b> בעבור :
<h4>{י}</h4>
<b>כלאו .</b> ענין מניעה כמו ויכלא העם מהביא ( שמות לו ) : <b>יבולה .</b> פריה כמו ואין יבול בגפנים ( חבקוק ג ) :
<h4>{יא}</h4>
<b>חורב .</b> יובש כמו חרבו המים ( בראשית ח ) : <b>התירוש .</b> ענבי היין : <b>היצהר .</b> זיתי השמן :
<h4>{יב}</h4>
<b>ועל דברי .</b> ואל דברי :
<h4>{יג}</h4>
<b>מלאך .</b> שליח וכן אשר מלאך שלוח עליהם ( יחזקאל כג ) :
<h4>{יד}</h4>
<b>ויער .</b> מל' תהעוררות : <b>רוח .</b> רצון כמו אל אשר יהיה שמה הרוח ללכת ( שם א ) :
<h3>חגי פרק-ב</h3>
<h4>{ו}</h4>
<b>החרבה .</b> היבשה וכן אל החרבה ( יהושע ד ) :
<h4>{ז}</h4>
<b>כבוד .</b> על העושר יאמר :
<h4>{יב}</h4>
<b>ישא .</b> מל' משא וסבל : <b>קדש .</b> ר''ל מובדל מפני הטומאה כי לשון קדש יאמר על דבר המובדל בין לטהרה בין לטומאה בין לאיסור כמ''ש פן תקדש המלאה ( דברים כב ) : <b>בכנף בגדו .</b> בשולי בגדו : <b>הנזיד .</b> התבשיל כמו ונזיד עדשים ( בראשית כה ) :
<h4>{טז}</h4>
<b>ערמת .</b> כן יקרא כרי תבואה כמו בקצה ( רות ג ) : <b>היקב .</b> הוא הבור שהיין יורד שם מן הגת : <b>לחשוף .</b> ענין שאיבה כמו ולחשוף מים מגבא ( ישעיה ל' ) : <b>פורה .</b> כן יקרא הגת וכן פורה דרכתי לבדי ( שם סג ) :
<h4>{יז}</h4>
<b>בשדפון .</b> ענין לקות הזרע עד לא נגמר הבשול : <b>ובירקון .</b> היא הכספת פני התבואה ונהפך לירקון :
<h4>{יט}</h4>
<b>במגורה .</b> כן יקרא אוצר התבואה וכן נהרסו ממגורות ( יואל א ) : <b>ועד .</b> ענינו כמו אפי' וכן ועד כפר הפרזי ( ש''א ו ) : <b>נשא .</b> ענין גדול והצמחה :
<h4>{כב}</h4>
<b>והשמדתי .</b> ענין כליון :
<h4>{כג}</h4>
<b>כחותם .</b> טבעת העשוי לחתום בו אגרות :
