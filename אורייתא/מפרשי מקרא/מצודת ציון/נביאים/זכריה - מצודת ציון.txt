<h1>זכריה - מצודת ציון</h1>
העשוי לחתום בו אגרות :
<h3>זכריה פרק-א</h3>
<h4>{ו}</h4>
<b>זמם .</b> ענין מחשבה כמו זממו אל תפק ( תהלים קמ ) :
<h4>{ח}</h4>
<b>ההדסים .</b> שם אילן המריח : <b>במצולה .</b> מקום רבוי המים עניינו כמו ותשליכני מצולה ( יונה ב ) : <b>שרוקים .</b> מצויירים בצבעים מחולפים ודוגמתו בדרז''ל אשה לא תעביר שרק על פניה ( שבת צה ) :
<h4>{יא}</h4>
<b>יושבת .</b> רצה לומר יושבת בשלוה : <b>ושוקטת .</b> על השקט ומרגוע :
<h4>{יב}</h4>
<b>ויען .</b> ענין הרמת קול כמו וענית ואמרת ( דברים כו ) : <b>זעמתה .</b> מל' זעם וכעס :
<h4>{יג}</h4>
<b>נחומים .</b> מל' נחמה :
<h4>{יד}</h4>
<b>קנאתי .</b> ענין כעס כמו הם קנאוני בלא אל ( שם לב ) :
<h4>{טו}</h4>
<b>השאננים .</b> מל' שאנן והשקט :
<h4>{טז}</h4>
<b>וקו .</b> ענינו חוט המשקולת של הבנאי וכן קו לקו ( ישעיה כח ) : <b>ינטה .</b> מל' הטיה :
<h4>{יז}</h4>
<b>תפוצנה .</b> ענין פזור כמו ומאלה נפצה כל הארץ ( בראשית ט ) :
<h3>זכריה פרק-ב</h3>
<h4>{א}</h4>
<b>קרנות .</b> מל' קרן :
<h4>{ב}</h4>

<h4>{ג}</h4>
<b>חרשים .</b> אומנים כמו חרש עצים ( ישעיה מד ) :
<h4>{ד}</h4>
<b>כפי .</b> כשיעור כמו איש כפי אכלו ( שמות יב ) : <b>לידות .</b> ענין הפלה והשלכה כמו ידו אליה ( ירמיה נ ) : <b>הנושאים .</b> ענין הרמה : <b>לזרותה .</b> לפזרם :
<h4>{ה}</h4>
<b>חבל מדה .</b> קו המדה :
<h4>{ו}</h4>
<b>אנה .</b> להיכן כמו אנה אנחנו עולים ( דברים א ) :
<h4>{ח}</h4>
<b>רוץ .</b> ענין מהירת ההליכה : <b>הלז .</b> הזה וכן מה הציון הלז ( מ''ב כג ) : <b>פרזות .</b> ר''ל מבלי חומה סביב וכן ראש פרזיו ( חבקוק ג ) :
<h4>{י}</h4>
<b>הוי .</b> הוא ענין קריאה וכן הוי כל צמא ( ישעיה נה ) : <b>ונוסו .</b> ענין בריחה ומהירות ההליכה : <b>רוחות .</b> פאה ועבר :
<h4>{יא}</h4>
<b>המלטי .</b> ענין הצלה : <b>בת בבל .</b> עדת בבל :
<h4>{יב}</h4>
<b>השוללים .</b> ענין בזה : <b>בבבת עינו .</b> הוא גלגל העין וכן כאישון בת עין ( תהלים יז ) :
<h4>{יג}</h4>
<b>מניף .</b> ענין הרמה כמו והניף הכהן ( ויקרא כג ) :
<h4>{טו}</h4>
<b>ונלוו .</b> ענין התחברות כמו ונלוה הגר ( ישעיה יד ) :
<h4>{יז}</h4>
<b>הס .</b> ענין שתיקה כמו ויהס כלב ( במדבר יג ) : <b>כל בשר .</b> כל אדם : <b>נעור .</b> מלשון הערה והקיצה כמו אשר יעור משנתו ( לקמן ד' ) : <b>ממעון .</b> ענין מדור כמו ה' מעון אתה היית לנו ( תהלי' צ ) :
<h3>זכריה פרק-ג</h3>
<h4>{א}</h4>
<b>השטן .</b> מלאך מקטרג : <b>לשטנו .</b> ענין קטרוג וכן כתבו שטנה ( עזרא ד' ) :
<h4>{ב}</h4>
<b>יגער .</b> ענין צעקה נזיפה : <b>אוד .</b> הוא העץ שמנדנדים בו האש כמו כאוד מוצל משרפה ( עמוס ד' ) :
<h4>{ג}</h4>
<b>צואים .</b> ר''ל מלוכלכים ולפי שהצואה הוא לכלוך קורא לבגדים מלוכלכים בגדים צואים :
<h4>{ד}</h4>
<b>ויען .</b> ענין הרמת קול כמו וענית ואמרת ( דברים כו ) : <b>מחלצות .</b> כן יקראו מין מלבושים נאים וכן המחלצות והמעטפות ( ישעיה ג ) :
<h4>{ה}</h4>
<b>צניף .</b> מל' מצנפת העשוי לסבב את הראש : <b>טהור .</b> ר''ל ברור ונקי :
<h4>{ו}</h4>
<b>ויעד .</b> ענין התראה כי רוב המתרים מתרים בעדים למען לא יכחש לאחר זמן :
<h4>{ז}</h4>
<b>תדין .</b> מל' דין והוא ענין היישרת הדבר : <b>מהלכים .</b> מל' הלוך וטיול :
<h4>{ח}</h4>
<b>מופת .</b> ענין נס ואות :
<h4>{ט}</h4>
<b>שבעה עינים .</b> ר''ל הרבה וכן שבע יפול צדיק ( משלי כד ) : <b>מפתח פתוחה .</b> ענין חקיקת הציור כמו ויודע לפתח פתוחים ( ד''ה ב' ב' ) : <b>ומשתי .</b> ענין הסרה כמו לא ימיש עמוד הענן ( שמות יג ) :
<h3>זכריה פרק-ד</h3>
<h4>{א}</h4>
<b>ויעירני .</b> מל' הערה והקצה :
<h4>{ב}</h4>
<b>וגולה .</b> ספל עגול וכן וגולת הכותרות ( מ''א ז' ) : <b>נרותיה .</b> כעין בזיכין על השמן והפתילה כמו ועשית את נרותיה ( שמות כה ) : <b>מוצקות .</b> מל' יציקה והם צינורות ודרך שם ניצוק ונמשך השמן :
<h4>{ד}</h4>
<b>ואען .</b> ענין הרמת קול :
<h4>{ו}</h4>
<b>בחיל .</b> ענין צבאות עם : <b>ברוחי .</b> ענין רצון :
<h4>{ז}</h4>
<b>למישור .</b> מל' ישר ושוה : <b>הראשה .</b> ר''ל טובה וחשובה וכן וראשית שמנים ימשחו ( עמוס ז' ) : <b>תשואות .</b> ענין המית קול כמו תשואות נוגש ( איוב לט ) :
<h4>{ט}</h4>
<b>תבצענה .</b> ענין השלמה כמו כי יבצע ה' ( ישעיה י ) :
<h4>{י}</h4>
<b>בז .</b> מל' בזיון : <b>האבן הבדיל .</b> הוא האבן התלוי בקו המשקולת וע''י מיישר הבנאי את הבנין ועל שע''פ רוב דרכם לעשותה מאבן אמר האבן הבדיל וכן אבן העופרת ( לקמן ה ) : <b>משוטטים .</b> ענין התנועה הנה והנה וכן שוטטו בחוצות ירושלים ( ירמיה ה ) :
<h4>{יב}</h4>
<b>שבלי .</b> מל' השבלים הטובות ( בראשית מא ) ובדרך שאלה קורא כן ענפי הזית לפי שהיו מלאים גרגרי זיתים כמו השבולת המלאה גרגרי תבואה : <b>ביד .</b> ר''ל בתוך וכן ביד כל אדם יחתום ( איוב ל''ז ) : <b>צנתרות .</b> הוא שם כלי מה : <b>המריקים .</b> ענין הזלה ושפיכה כמו על הארץ יריקו ( קהלת י''א ) : <b>הזהב .</b> ר''ל שמן מזוקק ונקי כזהב וכן מצפון זהב יאתה ( איוב ל''ז ) :
<h4>{יד}</h4>
<b>היצהר .</b> הוא השמן כמו תירוש ויצהר ( דברים כ''ח ) : <b>על אדון .</b> אצל אדון :
