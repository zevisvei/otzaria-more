<h1>עמוס</h1>

<h2>עמוס פרק-א</h2>
<h3>{א}</h3>
דִּבְרֵ֣י עָמ֔וֹס אֲשֶׁר־הָיָ֥ה בַנֹּקְדִ֖ים מִתְּק֑וֹעַ אֲשֶׁר֩ חָזָ֨ה עַל־יִשְׂרָאֵ֜ל בִּימֵ֣י ׀ עֻזִּיָּ֣ה מֶֽלֶךְ־יְהוּדָ֗ה וּבִימֵ֞י יָרָבְעָ֤ם בֶּן־יוֹאָשׁ֙ מֶ֣לֶךְ יִשְׂרָאֵ֔ל שְׁנָתַ֖יִם לִפְנֵ֥י הָרָֽעַשׁ׃
<b style="color: gray;">רש"י</b> <b>אשר היה בנוקדים .</b> תרגום יונתן דהוה מרי גיתין כמו ומישע מלך מואב היה נוקד ( מלכים ב ג ) : <b>שנתים לפני הרעש .</b> שנתים קודם שנתנגע עוזיה יום שנתנבא ישעיה שנא' וינועו אמות הסיפים ( ישעיה ו ) ואומר כאשר נסתם מפני הרעש וגו' ( זכריה יד ) :
<b style="color: gray;">מצודת דוד</b> <b>אשר היה בנוקדים .</b> ר''ל שהיה הגדול שבנוקדים : <b>מתקוע .</b> שם עיר בנחלת אשר : <b>ובימי ירבעם .</b> שמלך בימי עוזיה ( ולפי שניבא גם עליו בארצו כי היה מבני עשרת השבטים לכן אמר בימי ירבעם ) : <b>לפני הרעש .</b> שהיה כשנכנס עוזיה בהיכל להקטיר וכמ''ש כאשר נסתם מפני הרעש בימי עוזיה ( זכריה יד ) :
<b style="color: gray;">מצודת ציון</b> <b>בנוקדים .</b> לפי שרוב הצאן המה נקודים יקרא המתעסק בהם בשם נוקד וכן ומישע מלך מואב היה נוקד ( מ''ב ג ) : <b>חזה .</b> ניבא :
<h3>{ב}</h3>
וַיֹּאמַ֓ר ׀ יְהוָה֙ מִצִּיּ֣וֹן יִשְׁאָ֔ג וּמִירוּשָׁלִַ֖ם יִתֵּ֣ן קוֹל֑וֹ וְאָֽבְלוּ֙ נְא֣וֹת הָרֹעִ֔ים וְיָבֵ֖שׁ רֹ֥אשׁ הַכַּרְמֶֽל׃
<b style="color: gray;">רש"י</b> <b>מציון ישאג .</b> מבית קדשי הקדשים הדיבור יוצא : <b>ואבלו נאות הרועים .</b> ויצדון מדורי מלכיא ויחרוב תקיף כרכיהון :
<b style="color: gray;">מצודת דוד</b> <b>מציון ישאג .</b> ר''ל ממקום שכנו ישאג לאיים ולהחריד : <b>ומירושלים וגו' .</b> כפל הדבר במ''ש : <b>ואבלו .</b> מקול שאגתו ישחתו מדורי הרועים הם היכלי העכו''ם : <b>ראש הכרמל .</b> מיטב הכרמל :
<b style="color: gray;">מצודת ציון</b> <b>ואבלו .</b> ענין השחתה כמו אבלה אדמה ( יואל א ) : <b>נאות .</b> מדור : <b>ראש .</b> ענין מיטב כמו ראשית גבורתם ( ירמיה מט ) : <b>הכרמל .</b> הוא שם כולל למקום שדות וכרמים וכן הדר הכרמל ( ישעיה לה ) :
<h3>{ג}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י דַמֶּ֔שֶׂק וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־דּוּשָׁ֛ם בַּחֲרֻצ֥וֹת הַבַּרְזֶ֖ל אֶת־הַגִּלְעָֽד׃
<b style="color: gray;">רש"י</b> <b>על שלשה פשעי וגו' .</b> דכתיב הן כל אלה יפעל אל פעמים שלש עם גבר ( איוב לג ) ואם יותר יותר וכאן כך הוא כבר נהגתי עמהם כמדותי לוותר להם על ג' פעמים ואם יש יותר כל שכן שיש בכלל המרובה המועט : <b>ועל ארבעה לא אשיבנו .</b> על פשע רביעי אין עלי עוד להשיבו ריקם מלהשיב לו גמול וכל שכן שיש לו פשעים הרבה עתה את זה אפקוד עליהם אשר דשו יושבי גלעד הסמוכין להם : <b>בחרוצות הברזל .</b> ת''י במוריגא דפרזלא והוא כעין שופינ''א שקורין לימ''א שעשוי חרוצים חרוצים ורודים בהם בני אדם ביסורי' ובהם מחתכין את קשי התבואה לעשותן תבן :
<b style="color: gray;">מצודת דוד</b> <b>על שלשה פשעי דמשק .</b> יתכן שהם הפשעים הראשיים בדמשק והחמורים מכולם והם עכו''ם וג ''ע וש''ד הנמצאים בכל אלה הנזכרים פה ור''ל האם מהראוי שעל שלשת הפשעי' הראשיים ועל הוספת פשע הרביעי שלא אשיב לו גמולו בראשו בתמיה : <b>על דושם .</b> עתה יפרש מהו הפשע הרביעי שישלם עליו הגמול ואמר שהוא על שדשו את אנשי גלעד בכלי ברזל מלא חריצים עשוי לדוש בו התבואה :
<b style="color: gray;">מצודת ציון</b> <b>דושם .</b> מל' דישה : <b>בחרוצות .</b> שם כלי דישה מלאה חריצים כמו כי לא בחרוץ יודש קצח ( שם כח ) :
<h3>{ד}</h3>
וְשִׁלַּ֥חְתִּי אֵ֖שׁ בְּבֵ֣ית חֲזָאֵ֑ל וְאָכְלָ֖ה אַרְמְנ֥וֹת בֶּן־הֲדָֽד׃
<b style="color: gray;">רש"י</b> <b>חזאל .</b> ובן הדד . מלכי ארם וראש ארם דמשק :
<b style="color: gray;">מצודת דוד</b> <b>ושלהתי .</b> לכן אשלח אש : <b>בבית חזאל .</b> ר''ל אויב חזק יחריב בית חזאל מלך דמשק : <b>ארמנות בן הדד .</b> ההיכלות שבנה בן הדד מלך ארם :
<b style="color: gray;">מצודת ציון</b> <b>ואכלה .</b> ותשרף :
<h3>{ה}</h3>
וְשָֽׁבַרְתִּי֙ בְּרִ֣יחַ דַּמֶּ֔שֶׂק וְהִכְרַתִּ֤י יוֹשֵׁב֙ מִבִּקְעַת־אָ֔וֶן וְתוֹמֵ֥ךְ שֵׁ֖בֶט מִבֵּ֣ית עֶ֑דֶן וְגָל֧וּ עַם־אֲרָ֛ם קִ֖ירָה אָמַ֥ר יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>בקעת און .</b> מקום הוא בארם : <b>ותומך שבט .</b> ועביד שולטן כמו לא יסור שבט ( בראשית מט ) על שם שהמושל רודה את העם במקלות : <b>מבית עדן .</b> מקום : <b>קירה .</b> מקום ושם הגלם סנחריב בימי אחז שנאמר ויעל דמשק ויתפשם ויגלה את ארם קירה ( מלכים ב טז ) כל הפורענות שנתנבאו נביאי ישראל לבא חרב על העכו''ם נביאים שהיו קודם סנחריב כגון ישעיה ועמוס נבאו על חרב סנחריב וירמיה ויחזקאל נבאו על חרב נבוכדנאצר :
<b style="color: gray;">מצודת דוד</b> <b>בריח דמשק .</b> ר''ל האויב יבוא בה כאלו נשברו הבריחים ונפתחו השערים : <b>ותומך שבט .</b> ר''ל אכרות תומך שבט מבית מדן ועל המושל יאמר שהוא אוחז השבט לרדות בו את המחוייב וכן לא יסור שבט מיהודה ( בראשית מט ) : <b>קירה .</b> אל קיר והוא שם מקום וכן נאמר ויגלם קירה ( מ''ב ט''ז ) :
<b style="color: gray;">מצודת ציון</b> <b>בריח .</b> מטה העשוי לסגור בו השער : <b>מבקעת און .</b> שם עיר : <b>ותומך .</b> ומחזיק וכן וכפיה תמכו פלך ( משלי לא ) : <b>שבט .</b> שרביט : <b>מבית עדן .</b> שם עיר :
<h3>{ו}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י עַזָּ֔ה וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־הַגְלוֹתָ֛ם גָּל֥וּת שְׁלֵמָ֖ה לְהַסְגִּ֥יר לֶאֱדֽוֹם׃
<b style="color: gray;">רש"י</b> <b>עזה .</b> מפלשתים היתה : <b>גלות שלמה .</b> גלות ישראל שתהא הגלות שלימה שלא ימלט איש שהיו עומדים על הדרכים ותופשין הנמלטים ומסגירין אותם וכן עשו אנשי צור וכן לארבע רוחות בחורבן בית שני כמו שמפורש באגדת האזינו עזה מן הדרום צור מן הצפון דמשק מן המזרח ערביים מן המערב שנאמר ( ישעיה כא ) משא בערב ביער בערב תלינו :
<b style="color: gray;">מצודת דוד</b> <b>על הגלותם וגו' .</b> כי בזמן חורבן בית השני ע''י טיטוס ברחו קצתם מישראל דרך ארץ פלשתים ותפשום בני עזה ומסרום ביד האויב וז''ש על הגלותם וגו' ר''ל על שהגלו את ישראל גלות שלימה איש לא נעדר וזהו במה שתפסום להסגיר לעכו''ם :
<b style="color: gray;">מצודת ציון</b> <b>להסגיר .</b> ענין מסירה כמו לא תסגיר עבד ( דברים כג ) :
<h3>{ז}</h3>
וְשִׁלַּ֥חְתִּי אֵ֖שׁ בְּחוֹמַ֣ת עַזָּ֑ה וְאָכְלָ֖ה אַרְמְנֹתֶֽיהָ׃
<h3>{ח}</h3>
וְהִכְרַתִּ֤י יוֹשֵׁב֙ מֵֽאַשְׁדּ֔וֹד וְתוֹמֵ֥ךְ שֵׁ֖בֶט מֵֽאַשְׁקְל֑וֹן וַהֲשִׁיב֨וֹתִי יָדִ֜י עַל־עֶקְר֗וֹן וְאָֽבְדוּ֙ שְׁאֵרִ֣ית פְּלִשְׁתִּ֔ים אָמַ֖ר אֲדֹנָ֥י יְהוִֽה׃
<b style="color: gray;">רש"י</b> <b>אשדוד .</b> אשקלון עקרון . סרני פלשתים הם : <b>והשיבותי ידי .</b> אכה ואשנה :
<b style="color: gray;">מצודת דוד</b> <b>והשיבותי .</b> ר''ל אכה ואשיב ידי להכות עוד : <b>ואבדו שארית .</b> ר''ל אף שארית מה לא ישאר מהם :
<h3>{ט}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵי־צֹ֔ר וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַֽל־הַסְגִּירָ֞ם גָּל֤וּת שְׁלֵמָה֙ לֶאֱד֔וֹם וְלֹ֥א זָכְר֖וּ בְּרִ֥ית אַחִֽים׃
<b style="color: gray;">רש"י</b> <b>ולא זכרו ברית אחים .</b> אשר היה בין חירם ובין שלמה שנאמר ויכרתו ברית שניהם ( מלכים א ה ) והיו קורין זה לזה אחי שנ' ( שם ט ) מה הערים אשר נתת לי אחי :
<b style="color: gray;">מצודת דוד</b> <b>על הסגירם .</b> על שהם מסרו לאדום את הבורחים מישראל להיות גלות שלימה כדרך שעשה עזה : <b>ולא זכרו .</b> אנשי צור לא זכרו ברית האחוה שהיה להם עם ישראל וחירם מלך צור קרא לשלמה אח כמ''ש מה הערים האלה אשר נתתה לי אחי ( מ''א ט ) :
<h3>{י}</h3>
וְשִׁלַּ֥חְתִּי אֵ֖שׁ בְּח֣וֹמַת צֹ֑ר וְאָכְלָ֖ה אַרְמְנֹתֶֽיהָ׃
<h3>{יא}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י אֱד֔וֹם וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־רָדְפ֨וֹ בַחֶ֤רֶב אָחִיו֙ וְשִׁחֵ֣ת רַחֲמָ֔יו וַיִּטְרֹ֤ף לָעַד֙ אַפּ֔וֹ וְעֶבְרָת֖וֹ שְׁמָ֥רָה נֶֽצַח׃
<b style="color: gray;">רש"י</b> <b>על רדפו בחרב אחיו .</b> שנאמר ( מדבר כ ) פן בחרב אצא לקראתך וגם עודנו טורף אפו ושומר עברתו ולא חזר בו : <b>ויטרוף לעד אפו .</b> החזיק בה ולא הניחה :
<b style="color: gray;">מצודת דוד</b> <b>על רדפו בחרב אחיו .</b> כי הלא את עשו ליעקב ומדוע א''כ רדפו בחרב בזמן חורבן בית השני : <b>ושחת רחמיו .</b> ביטל הרחמנות הראוי להיות לו עליו כדרך האח על אחיו : <b>ויטרוף לעד אפו .</b> טרף את ישראל בכל עת שהיה היכולת בידו : <b>ועברתו .</b> העברה שיש לו על בני ישראל שמורה אצלו לעולם ולא יעזבנה בשום פעם :
<b style="color: gray;">מצודת ציון</b> <b>ועברתו .</b> מל' עברה וזעם : <b>נצח .</b> לעולם :
<h3>{יב}</h3>
וְשִׁלַּ֥חְתִּי אֵ֖שׁ בְּתֵימָ֑ן וְאָכְלָ֖ה אַרְמְנ֥וֹת בָּצְרָֽה׃
<b style="color: gray;">רש"י</b> <b>ארמנות בצרה .</b> בין מואב ובין אדום היא ומארץ מואב ונמשכה אחר אדום והיא העמידה לו מלך במות מלך הראשון שנאמר יובב בן זרח מבצרה ( בראשית לז ) :
<b style="color: gray;">מצודת דוד</b> <b>בתימן .</b> היא ארץ אדום הסמוכה לא''י בדרומה ותלקה גם היא : <b>בצרה .</b> עיר הבצורה והוא כרך גדול וגו' :
<h3>{יג}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י בְנֵֽי־עַמּ֔וֹן וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־בִּקְעָם֙ הָר֣וֹת הַגִּלְעָ֔ד לְמַ֖עַן הַרְחִ֥יב אֶת־גְּבוּלָֽם׃
<b style="color: gray;">רש"י</b> <b>בקעם הרות הגלעד .</b> השחיתו אותה וגם הרותיהם בקעו שלא ישאר יורש שיערער על הנחלה וכן אומר ( בירמיה מט ) לבני עמון כה אמר ה' הבנים אין לישראל אם יורש אין לו מדוע ירש מלכם :
<b style="color: gray;">מצודת דוד</b> <b>על בקעם .</b> על אשר בקעו כרסי הנשים ההרות של גלעד להוציא הולדות למען יוכלו להרחיב את גבולם ממה שגזלו מבני גלעד על כי לא ישאר יורש מה לערער על הגבול :
<h3>{יד}</h3>
וְהִצַּ֤תִּי אֵשׁ֙ בְּחוֹמַ֣ת רַבָּ֔ה וְאָכְלָ֖ה אַרְמְנוֹתֶ֑יהָ בִּתְרוּעָה֙ בְּי֣וֹם מִלְחָמָ֔ה בְּסַ֖עַר בְּי֥וֹם סוּפָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>בתרועה וגו' .</b> בעת יריע האויב עליהם ביום המלחמה : <b>בסער .</b> בסערת האויב אשר יבוא : <b>ביום סופה .</b> הוא יום המלחמה וכפל הדבר במ''ש :
<b style="color: gray;">מצודת ציון</b> <b>והצתי .</b> מל' הצתה והדלקה : <b>בסער .</b> מל' רוח סערה ואמר בדרך שאלה על התגברות האויב : <b>סופה .</b> רוח סופה :
<h3>{טו}</h3>
וְהָלַ֥ךְ מַלְכָּ֖ם בַּגּוֹלָ֑ה ה֧וּא וְשָׂרָ֛יו יַחְדָּ֖ו אָמַ֥ר יְהוָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>מלכם .</b> מלך בני עמון :
<h2>עמוס פרק-ב</h2>
<h3>{א}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י מוֹאָ֔ב וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־שָׂרְפ֛וֹ עַצְמ֥וֹת מֶֽלֶךְ־אֱד֖וֹם לַשִּֽׂיד׃
<b style="color: gray;">רש"י</b> <b>על שרפו עצמות וגו' .</b> פעם אחת נפל מלך אדום ביד מלך מואב ושרפו עצמותיו וטחום בקירות הבית ותבע הקב''ה אונאת המלך שנהגו בו בזיון :
<b style="color: gray;">מצודת דוד</b> <b>על שרפו וגו' .</b> הוא מה שנאמר ויקח את בנו הבכור אשר ימלוך תחתיו ויעלהו עולה על החומה ויהי קצף גדול על ישראל ( מ''ב ו ) ופי' שמלך מואב לקח בן מלך אדום הראוי למלוך ושרפו כליל כעולה ואז קצף מלך אדום על ישראל כל כי בעבורם באה לו הרעה ההיא ולא ראו להצילו א''כ ע''י מלך מואב נתעורר קצף על ישראל לכן יקבל מואב גמול : <b>לשיד .</b> ר''ל עד שנעשו דק ולבן כסיד כדרך העצם אחר השריפה :
<b style="color: gray;">מצודת ציון</b> <b>לשיד .</b> כמו לסיד בסמ''ך :
<h3>{ב}</h3>
וְשִׁלַּחְתִּי־אֵ֣שׁ בְּמוֹאָ֔ב וְאָכְלָ֖ה אַרְמְנ֣וֹת הַקְּרִיּ֑וֹת וּמֵ֤ת בְּשָׁאוֹן֙ מוֹאָ֔ב בִּתְרוּעָ֖ה בְּק֥וֹל שׁוֹפָֽר׃
<b style="color: gray;">רש"י</b> <b>קריות .</b> שם מקום כד''א ( שם מח ) על קריות ועל בצרה : <b>בשאון .</b> קול המון מלחמה :
<b style="color: gray;">מצודת דוד</b> <b>ומת בשאון מואב .</b> מואב ימות בשאון המלחמה בעת יריע האויב בקול שופר :
<b style="color: gray;">מצודת ציון</b> <b>הקריות .</b> שם מחוז : <b>בשאון .</b> ענין המייה :
<h3>{ג}</h3>
וְהִכְרַתִּ֥י שׁוֹפֵ֖ט מִקִּרְבָּ֑הּ וְכָל־שָׂרֶ֛יהָ אֶהֱר֥וֹג עִמּ֖וֹ אָמַ֥ר יְהוָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>עמו .</b> עם השופט ההוא :
<h3>{ד}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י יְהוּדָ֔ה וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַֽל־מָאֳסָ֞ם אֶת־תּוֹרַ֣ת יְהוָ֗ה וְחֻקָּיו֙ לֹ֣א שָׁמָ֔רוּ וַיַּתְעוּם֙ כִּזְבֵיהֶ֔ם אֲשֶׁר־הָלְכ֥וּ אֲבוֹתָ֖ם אַחֲרֵיהֶֽם׃
<b style="color: gray;">מצודת דוד</b> <b>ויתעום כזביהם .</b> הנביאים המכזבים להם התעום מדרך הישר : <b>אשר הלכו וגו' .</b> הם הם הנביאים המכזבים אשר הלכו אבותיהם אחריהם וכאומר הלא אבותיכם הלכו אחריהם ונכשלו והם לא לקחו מוסר :
<b style="color: gray;">מצודת ציון</b> <b>ויתעום .</b> מלשון תועה : <b>נעלים .</b> מנעלי הרגל :
<h3>{ה}</h3>
וְשִׁלַּ֥חְתִּי אֵ֖שׁ בִּֽיהוּדָ֑ה וְאָכְלָ֖ה אַרְמְנ֥וֹת יְרוּשָׁלִָֽם׃
<h3>{ו}</h3>
כֹּ֚ה אָמַ֣ר יְהוָ֔ה עַל־שְׁלֹשָׁה֙ פִּשְׁעֵ֣י יִשְׂרָאֵ֔ל וְעַל־אַרְבָּעָ֖ה לֹ֣א אֲשִׁיבֶ֑נּוּ עַל־מִכְרָ֤ם בַּכֶּ֙סֶף֙ צַדִּ֔יק וְאֶבְי֖וֹן בַּעֲב֥וּר נַעֲלָֽיִם׃
<b style="color: gray;">רש"י</b> <b>על מכרם בכסף צדיק .</b> הדיינין היו מוכרי' את מי שהיה זכאי בדין בכסף שוחד שהיו מקבלים מיד בעל דינו : <b>ואביון בעבור נעלים .</b> תרגם יונתן בשני מקומות ( לקמן ח ו ) בדיל דיחסנון ואומר אני שכך פירושו מטין משפט האביון כדי שיצטרך למכור שדהו שהיה לו בין שדות הדיינין וזה עוקף עליו ונוטלה בדמים קלים כדי לגדור ולנעול כל שדותיו יחד ולא יפסיק ביניהם :
<b style="color: gray;">מצודת דוד</b> <b>על מכרם וגו' .</b> ר''ל מטין דין הצדיק ומוכרים אותו במחיר שוחד הכסף : <b>ואביון .</b> מטין דין האביון בעבור מחיר שוחד מנעלים לרגליהם ר''ל אף במעט שוחד מטין את הדין :
<h3>{ז}</h3>
הַשֹּׁאֲפִ֤ים עַל־עֲפַר־אֶ֙רֶץ֙ בְּרֹ֣אשׁ דַּלִּ֔ים וְדֶ֥רֶךְ עֲנָוִ֖ים יַטּ֑וּ וְאִ֣ישׁ וְאָבִ֗יו יֵֽלְכוּ֙ אֶל־הַֽנַּעֲרָ֔ה לְמַ֥עַן חַלֵּ֖ל אֶת־שֵׁ֥ם קָדְשִֽׁי׃
<b style="color: gray;">רש"י</b> <b>השואפים על עפר ארץ .</b> על עפר ארץ שהם הולכים עליהם כל שאפם וכל מחשבתם בראש דלים הוא איך יגזלו אותם ויטלו את שלהם , שאיפה גולושי''ר בלע''ז : <b>ודרך ענוים יטו וגו' .</b> החלשים נוטים מדרכם והולכין דרך עקלתון מפני יראתם כמה דאת אמר ( איוב כד ) יטו אביונים מדרך : <b>אל הנערה .</b> המאורסה :
<b style="color: gray;">מצודת דוד</b> <b>השואפים .</b> אשר מביטים אל עפר הארץ להיות מושבם בראש הדלים כי השופטים ההם משימים שוטרים תחתיהם לכוף הדלים לקיים דבר המשפט ואם ימאנו יקחום בשער ראשם ויפילום בארץ וירמסום ברגליהם והנה עפר הארץ בראש הדלים : <b>ודרך ענוים יטו .</b> מטים דרך ענוים להסב מדרך הפשוטה להטות דרך עקלתון כי יפחדו מהם ויסתתרו לבל יראום : <b>ואיש ואביו .</b> שניהם יחד ולא יבושו זה מזה : <b>אל הנערה .</b> ר''ל אל נערה המאורסה לזנות עמה : <b>למען חלל .</b> כאלו יתכוונו לחלל שם קדשי כי אם היה בעבור הנאה לבד מה להם ללכת אל ארוסה ושניהם יחד :
<b style="color: gray;">מצודת ציון</b> <b>השואפים .</b> ענין הבטה אל הדבר והתוחלת לה וכן כעבד ישאף צל ( איוב ז ) :
<h3>{ח}</h3>
וְעַל־בְּגָדִ֤ים חֲבֻלִים֙ יַטּ֔וּ אֵ֖צֶל כָּל־מִזְבֵּ֑חַ וְיֵ֤ין עֲנוּשִׁים֙ יִשְׁתּ֔וּ בֵּ֖ית אֱלֹהֵיהֶֽם׃
<b style="color: gray;">רש"י</b> <b>ועל בגדים חבולים יטו .</b> תרגם יונתן על מטות שוויין דמשכון מסתחרין זוקפין מלוה על העניים וממשכנים אותם ועושים בגדיהם מצעות ומסובין עליהם בהטייה בעת סעודתם : <b>בית אלהיהם .</b> בבית עכו''ם שהיתה להם אצל מזבח : <b>יטו .</b> לשון מסיבה שכל מסיבת סעוד' בהטייה היא שהוא נסמך על שמאלו : <b>ויין ענושים .</b> שעונשים אותם ממון ושותים בו יין :
<b style="color: gray;">מצודת דוד</b> <b>ועל בגדים חבולים יטו .</b> ממשכנין בגדי המסרבים במשפטם ומטים עצמן לסמוך עליהם בעת הסעודה מה שהם סועדים אצל כל מזבח העשוי לפסילים : <b>ויין ענושים ישתו .</b> מענשים את המסרב לתת כסף ובו יקנו יין ושותין בבית העכו''ם :
<b style="color: gray;">מצודת ציון</b> <b>חבולים .</b> ענין משכון כמו לא יחבול רחים ( דברים כד ) : <b>יטו .</b> מל' הטייה ר''ל ישענו : <b>ענושים .</b> מל' עונש ממון :
<h3>{ט}</h3>
וְאָ֨נֹכִ֜י הִשְׁמַ֤דְתִּי אֶת־הָֽאֱמֹרִי֙ מִפְּנֵיהֶ֔ם אֲשֶׁ֨ר כְּגֹ֤בַהּ אֲרָזִים֙ גָּבְה֔וֹ וְחָסֹ֥ן ה֖וּא כָּֽאַלּוֹנִ֑ים וָאַשְׁמִ֤יד פִּרְיוֹ֙ מִמַּ֔עַל וְשָׁרָשָׁ֖יו מִתָּֽחַת׃
<b style="color: gray;">רש"י</b> <b>כאלונים .</b> קיישנ''ש בלע''ז : <b>ואשמיד פריו ממעל .</b> שרים העליונים : <b>ושרשיו מתחת .</b> שרים התחתונים ד''א הצרעה היתה מסמא עיניהם מלמעלה ומסרסתן מלמטן :
<b style="color: gray;">מצודת דוד</b> <b>ואנכי השמדתי .</b> ר''ל הלא אף את העכו''ם אשר לא ידעוני אני מענשם בעונם כי הלא השמדתי מפניהם את האמורי שהיה גבהו כארז וחזק כאילן אלון : <b>ואשמיד וגו' .</b> ר''ל השמדתיו בתכלית ולפי שהמשילו לאילן אמר לשון הנופל בהשחתת עץ פרי שמשירים הפירות ועוקרים השרשים :
<b style="color: gray;">מצודת ציון</b> <b>השמדתי .</b> ענין כליון ואבדון : <b>וחסון .</b> ענין חוזק כמו והיה החסון לנעורת ( ישעיה א ) : <b>כאלונים .</b> שם אילן החזק ביותר :
<h3>{י}</h3>
וְאָנֹכִ֛י הֶעֱלֵ֥יתִי אֶתְכֶ֖ם מֵאֶ֣רֶץ מִצְרָ֑יִם וָאוֹלֵ֨ךְ אֶתְכֶ֤ם בַּמִּדְבָּר֙ אַרְבָּעִ֣ים שָׁנָ֔ה לָרֶ֖שֶׁת אֶת־אֶ֥רֶץ הָאֱמֹרִֽי׃
<b style="color: gray;">מצודת דוד</b> <b>ואנכי העליתי .</b> ר''ל ואתם הלא ידעתם אותי כי אנכי העליתי אתכם וגו' וספקתי צרכיכם במדבר ארבעים שנה לבוא לרשת את ארץ האמורי :
<h3>{יא}</h3>
וָאָקִ֤ים מִבְּנֵיכֶם֙ לִנְבִיאִ֔ים וּמִבַּחוּרֵיכֶ֖ם לִנְזִרִ֑ים הַאַ֥ף אֵֽין־זֹ֛את בְּנֵ֥י יִשְׂרָאֵ֖ל נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>לנזרים .</b> תרגם יונתן למלפין שהיו פרושין מדרכי עם הארץ ועוסקים בתורה אין נזירה בכל מקום אלא ל' פרישה : <b>האף אין זאת .</b> בתמיה כלומ' כלום אתם יכולים להכחיש את זאת :
<b style="color: gray;">מצודת דוד</b> <b>ואקים וגו' .</b> בכל ימי הדורות הקמתי מבניכם לנביאים והשריתי שכינתי עליהם : <b>ומבחוריכם לנזירים .</b> נתתי בלבם רוח טהרה להיות נזירים : <b>האף אין זאת .</b> ר''ל וכי גם זאת תכחשו לומר שלא היה זאת וכאומר הלא הוא דבר מפורסם וא''א לכחש בו :
<b style="color: gray;">מצודת ציון</b> <b>לנזירים .</b> הוא ענין פרישה כי פורש עצמו מן היין והטומאה :
<h3>{יב}</h3>
וַתַּשְׁק֥וּ אֶת־הַנְּזִרִ֖ים יָ֑יִן וְעַל־הַנְּבִיאִים֙ צִוִּיתֶ֣ם לֵאמֹ֔ר לֹ֖א תִּנָּבְאֽוּ׃
<b style="color: gray;">רש"י</b> <b>ותשקו את הנזירים יין .</b> שלא יורו אתכם שהשיכור אסור להורות , ותשקו את הנזירים יין , תרגם יונתן ואטעיתון ית מלפיכון בחמר : <b>לא תנבאו .</b> לעמוס צוה כן אמציה כהן בית אל חוזה ברח לך אל ארץ יהודה ( לקמן ז ) :
<b style="color: gray;">מצודת דוד</b> <b>ותשקו .</b> ר''ל פתיתם את הנזירים לשתות יין ומחיתם ביד הנביאים שלא ינבאו :
<h3>{יג}</h3>
הִנֵּ֛ה אָנֹכִ֥י מֵעִ֖יק תַּחְתֵּיכֶ֑ם כַּאֲשֶׁ֤ר תָּעִיק֙ הָעֲגָלָ֔ה הַֽמְלֵאָ֥ה לָ֖הּ עָמִֽיר׃
<b style="color: gray;">רש"י</b> <b>מעיק .</b> תרגום של מציק : <b>תחתיכם .</b> את חנייתכם : <b>תעיק העגלה .</b> מכובד משא שעליה :
<b style="color: gray;">מצודת דוד</b> <b>הנה אנכי .</b> מוסב למעלה שאמר אם שלמתי גמול להעכו''ם אשר לא ידעוני כ''ש אתם שהרביתי להטיב עמכם ועכ''ז פשעתם בי כ''ש שאשלם לכם גמול ולכן הנה אני מביא עליכם צוקה במקומכם ולא תוכלו לברוח כי בכל מקום שתהיו תשיג לכם הצוקה : <b>כאשר תעיק העגלה .</b> אמר דרך משל כי העגלה אינה בעלת חיים שתעיק לה ור''ל כמו העגלה המלאה עמיר שהמשא קשה ונפוחה רב וכאילו תעיק אל העגלה בכל מקום מהלכתה כן יעיק להם במקומם בכל מקום שיהיו :
<b style="color: gray;">מצודת ציון</b> <b>מעיק .</b> ענין צוקה וצרה כמו מפני עקת רשע ( תהלים נה ) : <b>תחתיכם .</b> במקומכם : <b>עמיר .</b> אלומות התבואה כמו ושכחת עומר ( דברים כד ) :
<h3>{יד}</h3>
וְאָבַ֤ד מָנוֹס֙ מִקָּ֔ל וְחָזָ֖ק לֹא־יְאַמֵּ֣ץ כֹּח֑וֹ וְגִבּ֖וֹר לֹא־יְמַלֵּ֥ט נַפְשֽׁוֹ׃
<b style="color: gray;">מצודת דוד</b> <b>ואבד מנוס מקל .</b> אף הקל ברגליו לא יציל עצמו במה שינוס מפני האויב הואיל וימצא אויב בכל מקום שיהיה : <b>לא יאמץ .</b> לא יוכל לחזק כחו במלחמה :
<b style="color: gray;">מצודת ציון</b> <b>מנוס .</b> ענין בריחה : <b>יאמץ .</b> ענין חוזק כמו יוסף אומץ ( איוב טז ) :
<h3>{טו}</h3>
וְתֹפֵ֤שׂ הַקֶּ֙שֶׁת֙ לֹ֣א יַעֲמֹ֔ד וְקַ֥ל בְּרַגְלָ֖יו לֹ֣א יְמַלֵּ֑ט וְרֹכֵ֣ב הַסּ֔וּס לֹ֥א יְמַלֵּ֖ט נַפְשֽׁוֹ׃
<b style="color: gray;">רש"י</b> <b>וקל ברגליו לא ימלט .</b> את נפשו :
<b style="color: gray;">מצודת דוד</b> <b>לא יעמוד .</b> רצה לומר לא יעמוד על עמדו בקשרי המלחמה כי ירך לבבו וינוס לנפשו : <b>לא ימלט נפשו .</b> ע''י מרוצת הסוס :
<b style="color: gray;">מצודת ציון</b> <b>ותופש .</b> ענין אחוזה :
<h3>{טז}</h3>
וְאַמִּ֥יץ לִבּ֖וֹ בַּגִּבּוֹרִ֑ים עָר֛וֹם יָנ֥וּס בַּיּוֹם־הַה֖וּא נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>ערום ינוס וגו' .</b> ערטילאי בלא כלי זיין :
<b style="color: gray;">מצודת דוד</b> <b>ואמיץ לבו בגבורים .</b> ר''ל אף מי שדרכו לאמץ ולחזק לבבו גם בין הגבורים הנה אז ינוס ערום כי יפשוט כליו להיות קל לברוח כי המורך יבוא בלבו :
<h2>עמוס פרק-ג</h2>
<h3>{א}</h3>
שִׁמְע֞וּ אֶת־הַדָּבָ֣ר הַזֶּ֗ה אֲשֶׁ֨ר דִּבֶּ֧ר יְהוָ֛ה עֲלֵיכֶ֖ם בְּנֵ֣י יִשְׂרָאֵ֑ל עַ֚ל כָּל־הַמִּשְׁפָּחָ֔ה אֲשֶׁ֧ר הֶעֱלֵ֛יתִי מֵאֶ֥רֶץ מִצְרַ֖יִם לֵאמֹֽר׃
<b style="color: gray;">מצודת דוד</b> <b>על כל המשפחה .</b> כפל הדבר במ''ש כי לשון משפחה יאמר על כל העם וכן ואם משפחת מצרים ( זכריה יד ) :
<h3>{ב}</h3>
רַ֚ק אֶתְכֶ֣ם יָדַ֔עְתִּי מִכֹּ֖ל מִשְׁפְּח֣וֹת הָאֲדָמָ֑ה עַל־כֵּן֙ אֶפְקֹ֣ד עֲלֵיכֶ֔ם אֵ֖ת כָּל־עֲוֹנֹֽתֵיכֶֽם׃
<b style="color: gray;">רש"י</b> <b>רק אתכם ידעתי .</b> אהבתי ואתם פשעתם בי על כן וגו' , ורבותינו דרשוהו לצד אחר במסכת עכו''ם :
<b style="color: gray;">מצודת דוד</b> <b>רק אתכם ידעתי .</b> לבד אתכם אהבתי מכל העכו''ם ואתם פשעתם בי לכן אזכור עליכם את כל עונותיכם לשלם גמול על כולם :
<b style="color: gray;">מצודת ציון</b> <b>ידעתי .</b> ענין אהבה וכן מה אדם ותדעהו ( תהלים קמד ) : <b>אפקוד .</b> ענין זכרון :
<h3>{ג}</h3>
הֲיֵלְכ֥וּ שְׁנַ֖יִם יַחְדָּ֑ו בִּלְתִּ֖י אִם־נוֹעָֽדוּ׃
<b style="color: gray;">רש"י</b> <b>הילכו שנים יחדיו .</b> שאתם אומרים לנביאים לא תנבאו כלום הנביאים מתנבאים מלבם אלא א''כ נצטוו כלום דברים נעשי' אלא ע''פ דרכם : <b>בלתי אם נועדו .</b> אלא א''כ קבעו מועד ללכת יחדיו למקום פלוני :
<b style="color: gray;">מצודת דוד</b> <b>הילכו .</b> וכי ילכו שני אנשים יחד בדרך אחד אם לא שקבעו והזמינו מתחילה זמן קבוע ללכת יחד :
<b style="color: gray;">מצודת ציון</b> <b>נועדו .</b> ענין קביעות זמן והזמנה וכן נועדו עברו יחדיו ( שם מח ) :
<h3>{ד}</h3>
הֲיִשְׁאַ֤ג אַרְיֵה֙ בַּיַּ֔עַר וְטֶ֖רֶף אֵ֣ין ל֑וֹ הֲיִתֵּ֨ן כְּפִ֤יר קוֹלוֹ֙ מִמְּעֹ֣נָת֔וֹ בִּלְתִּ֖י אִם־לָכָֽד׃
<b style="color: gray;">רש"י</b> <b>הישאג אריה ביער וטרף אין לו .</b> כשאריה אוחז טרף דרכו לשאוג ואינו שואג אלא א''כ לכד וכן הוא אומר ( ישעיה ה ) ישאג ככפירים וינהום ויאחז טרף וכן ( שם לא ) כאשר יהגה הכפיר והאריה על טרפו : <b>הישאג אריה .</b> הוא נבוכדנאצר , ד''א הנביאים מדמין את רוח הקודש הבא עליהם לשאגת אריה כמו שאמרו למטה אריה שאג מי לא יירא וכאן זה פירוש הדוגמא היבא רוח הקודש בפי הנביאים לדבר דבר מאת הקב''ה לרעה אלא אם כן הפורענות נגזר מלפניו זהו הטרף , היתן הקב''ה קולו לדבר קשה בלתי אם לכד אתכם במוקש עון :
<b style="color: gray;">מצודת דוד</b> <b>הישאג .</b> וכי ישאג אריה ביער ולא ימצא טרף כי בשמוע החיות קול שאגתו יעמדו במקומם מפחדו ויבוא בהם לטרוף טרף כרצונו : <b>היתן .</b> וכי הכפיר ישמיע קול שמחה ממעונתו אם לא שלכד מה כי אז יגיל וישמח ולא כשעדיין לא לכד :
<b style="color: gray;">מצודת ציון</b> <b>כפיר .</b> ארי בחור : <b>ממעונתו .</b> ענין מדור כמו ה' מעון אתה היית לנו ( שם צ ) :
<h3>{ה}</h3>
הֲתִפֹּ֤ל צִפּוֹר֙ עַל־פַּ֣ח הָאָ֔רֶץ וּמוֹקֵ֖שׁ אֵ֣ין לָ֑הּ הֲיַֽעֲלֶה־פַּח֙ מִן־הָ֣אֲדָמָ֔ה וְלָכ֖וֹד לֹ֥א יִלְכּֽוֹד׃
<b style="color: gray;">רש"י</b> <b>התפול צפור על פח הארץ ומוקש אין לה .</b> בפח שנפלה עליו ומוקש אין לה : <b>היעלה פח מן האדמה .</b> ממקום שהוטל שם אלא א''כ נאחז בו העוף כשהוא רוצה לברוח עוקר את הפח ומעלהו ממקומו מעט ואותה שעה כבר נלכד : <b>התפול צפור על פח .</b> ואין הפח מכשילה כלומר וכי אפשר שאתם עוברים עבירות ולא יהיה לכם למוקש : <b>היעלה פח וגו' .</b> כך היעלה עונותיכם למעלה ולא ילכדו אתכם :
<b style="color: gray;">מצודת דוד</b> <b>התפול .</b> וכי תשכון צפור על הפח הפרושה בארץ ולא מצאה מוקש להלכד בה וכי לחנם מזורה הרשת : <b>היעלה פח .</b> דרך הצפור כשהוא נלכד ורוצה להציל נפשו מנדנד הפח ומעלהו קצת מן האדמה ואמר וכי הפח יעלה מן האדמה ולא לכד דבר וכי מי מעלה את הפח :
<b style="color: gray;">מצודת ציון</b> <b>התפול .</b> התשכון וכן על פני כל אחיו נפל ( בראשית כה ) : <b>פח .</b> הוא הרשת :
<h3>{ו}</h3>
אִם־יִתָּקַ֤ע שׁוֹפָר֙ בְּעִ֔יר וְעָ֖ם לֹ֣א יֶחֱרָ֑דוּ אִם־תִּהְיֶ֤ה רָעָה֙ בְּעִ֔יר וַיהוָ֖ה לֹ֥א עָשָֽׂה׃
<b style="color: gray;">רש"י</b> <b>אם יתקע שופר .</b> שהצופה רוא' גייסות באים בעיר ותוקע בשופר להזהיר את העם אין אלו אלא דברי משל ודמיון : <b>אם יתקע שופר בעיר .</b> כך היה לכם להיות חרדים אל דברי הנביאים שהם צופים לכם למלט אתכם מן הרעה שלא תבא וכשתבא לכם הרעה הלא תדעו כי הקב''ה עושה לכם על שלא נזהרתם בנביאיו :
<b style="color: gray;">מצודת דוד</b> <b>אם יתקע .</b> בעת המלחמה מעמידים הצופה לראות אם בא האויב וכאשר יבוא יתקע בשופר להזהיר את העם ואמר האם יתקע שופר בעיר ועם לא יחרדו הלא בודאי למאד יחרדו : <b>אם תהיה רעה בעיר וה' לא עשה .</b> ר''ל כמו שכל אלו הדברי' הנזכרים א''א שיהיו בזולת הדבר הראוי לכל אחד ואחד כן א''א שתהיה רעה בעיר במקרה וה' לא עשהו בגמול המעשה :
<b style="color: gray;">מצודת ציון</b> <b>אם יתקע .</b> האם יתקע ותחסר ה''א התימה וכן אם תהיה :
<h3>{ז}</h3>
כִּ֣י לֹ֧א יַעֲשֶׂ֛ה אֲדֹנָ֥י יְהוִ֖ה דָּבָ֑ר כִּ֚י אִם־גָּלָ֣ה סוֹד֔וֹ אֶל־עֲבָדָ֖יו הַנְּבִיאִֽים׃
<b style="color: gray;">רש"י</b> <b>כי לא יעשה אדני .</b> אלהים דבר . אלא א''כ גלה סודו אותו סוד אל עבדיו הנביאים :
<b style="color: gray;">מצודת דוד</b> <b>כי לא יעשה .</b> מביא ראיה שהכל בא בהשגחה כי אין ה' עושה דבר כ''א יגלה סודו בתחילה אל הנביאים :
<h3>{ח}</h3>
אַרְיֵ֥ה שָׁאָ֖ג מִ֣י לֹ֣א יִירָ֑א אֲדֹנָ֤י יְהוִה֙ דִּבֶּ֔ר מִ֖י לֹ֥א יִנָּבֵֽא׃
<b style="color: gray;">רש"י</b> <b>אריה שאג מי לא יירא .</b> כך הקב''ה דבר אל הנביאי' להנבא מי לא ינבא :
<b style="color: gray;">מצודת דוד</b> <b>אריה שאג .</b> ר''ל והלא הנביאים בודאי מתנבאים הדבר אל העם ואינם כובשים הנבואה כי כמו שאם ישאג האריה לא ימצא מי שלא יירא מקול שאגתו כן אם ה' מדבר מה אל הנביאים לא ימצא מי שלא ינבא א''כ כל דבר הבא נשמע מפי הנביא עד לא בא א''כ אין זה מקרה כ''א בהשגחה ומוסב לתחילת הענין שאמר על כן אפקוד עליכם וגו' לשלם גמול כי הואיל והכל בהשגחה יש גמול על המעשה :
<h3>{ט}</h3>
הַשְׁמִ֙יעוּ֙ עַל־אַרְמְנ֣וֹת בְּאַשְׁדּ֔וֹד וְעַֽל־אַרְמְנ֖וֹת בְּאֶ֣רֶץ מִצְרָ֑יִם וְאִמְר֗וּ הֵאָֽסְפוּ֙ עַל־הָרֵ֣י שֹׁמְר֔וֹן וּרְא֞וּ מְהוּמֹ֤ת רַבּוֹת֙ בְּתוֹכָ֔הּ וַעֲשׁוּקִ֖ים בְּקִרְבָּֽהּ׃
<b style="color: gray;">רש"י</b> <b>השמיעו .</b> אתם בני פלשתים ובני מצרים והכריזו זאת ואמרו בארמנותיכ' שיאספו על הרי שומרון לראות מעל ההרים בתוך העיר את דרכם הרעה :
<b style="color: gray;">מצודת דוד</b> <b>השמיעו .</b> אתם בני אשדוד השמיעו כרוז בארמנותיכם : <b>האספו .</b> ללכת למלחמה בעזרת אשור : <b>וראו .</b> ר''ל אל תיראו מהם כי תראו בתוכה מהומות רבות המיית מלחמה איש באחיו ועושק רב תראו בקרבה וחלוק לבם ובקל יהיו נמסרים ביד האויב :
<b style="color: gray;">מצודת ציון</b> <b>ועשוקים .</b> נעשקים מיד העושקים :
<h3>{י}</h3>
וְלֹֽא־יָדְע֥וּ עֲשׂוֹת־נְכֹחָ֖ה נְאֻם־יְהוָ֑ה הָאֽוֹצְרִ֛ים חָמָ֥ס וָשֹׁ֖ד בְּאַרְמְנֽוֹתֵיהֶֽם׃
<b style="color: gray;">רש"י</b> <b>נכוחה .</b> ישרה : <b>חמס ושוד .</b> ממון של גזל :
<b style="color: gray;">מצודת דוד</b> <b>ולא ידעו .</b> ר''ל גם תראו אשר אינם יודעים לעשות נכוחה וא''כ בודאי לא יעזרם ה' : <b>האוצרים .</b> תראו אשר חומסים וגוזלים ואוצרים בארמנותיהם וכאומר אלו גזלו לצורך הספוק למלאות נפשם לא היה הדבר רע בתכלית אבל הם יגזלו מבלי צורך והכרח :
<b style="color: gray;">מצודת ציון</b> <b>נכוחה .</b> דבר ישר וראוי כמו הולך נכוחו ( ישעיה נז ) : <b>האוצרים .</b> מל' אוצר ר''ל טומנים :
<h3>{יא}</h3>
לָכֵ֗ן כֹּ֤ה אָמַר֙ אֲדֹנָ֣י יְהוִ֔ה צַ֖ר וּסְבִ֣יב הָאָ֑רֶץ וְהוֹרִ֤ד מִמֵּךְ֙ עֻזֵּ֔ךְ וְנָבֹ֖זּוּ אַרְמְנוֹתָֽיִךְ׃
<b style="color: gray;">רש"י</b> <b>צר וסביב הארץ .</b> הצר מוכן לבא וסביב לארץ הזאת יבא :
<b style="color: gray;">מצודת דוד</b> <b>צר וסביב הארץ .</b> ר''ל הצר יבא עליך וסביב הארץ יחנה לבל תוכל להמלט : <b>והוריד .</b> ישפיל ממך חזקך וארמנותיך יהיו נבוזים מן הצר הבא :
<b style="color: gray;">מצודת ציון</b> <b>צר .</b> אויב : <b>עוזך .</b> חזקך : <b>ונבוזו .</b> מלשון בזה ושלל :
<h3>{יב}</h3>
כֹּה֮ אָמַ֣ר יְהוָה֒ כַּאֲשֶׁר֩ יַצִּ֨יל הָרֹעֶ֜ה מִפִּ֧י הָאֲרִ֛י שְׁתֵּ֥י כְרָעַ֖יִם א֣וֹ בְדַל־אֹ֑זֶן כֵּ֣ן יִנָּצְל֞וּ בְּנֵ֣י יִשְׂרָאֵ֗ל הַיֹּֽשְׁבִים֙ בְּשֹׁ֣מְר֔וֹן בִּפְאַ֥ת מִטָּ֖ה וּבִדְמֶ֥שֶׁק עָֽרֶשׂ׃
<b style="color: gray;">רש"י</b> <b>כאשר יציל הרועה .</b> דבר מועט : <b>בדל אוזן .</b> חסחוס דאודן : <b>כן ינצלו בני ישראל .</b> מתי מעט : <b>בפאת מטה ובדמשק ערש .</b> ת''י בתקוף שולטן ועל דמשק רחיצן ואומר אני שפתרון התרגום לפי שבימי יהואחז בן יהוא כתיב ( מלכים ב יג ) כי אבדם מלך ארם וירבעם בן בנו הושיעם במקצת שנאמר ( שם יד ) ולא דבר ה' למחות את שם ישראל ויושיעם ביד ירבעם וגו' ( שם ) הוא השיב את גבול ישראל וזהו בפאת מטה פנת הבית היא חזקו של בית לכן תרגם בתקוף שולטן של ירבעם ועמוס נתנבא בימי ירבעם ועוד נבא שסופן להסמך על מלכי ארם וכן עשו בימי פקח בן רמליהו שנתחבר עם רצין וזהו ובדמשק ערש עיקר מנוחתם ומבטחם תהי על דמשק , ורבותינו אמרו בסדר עולם ר' נהוראי אומר משום רבי יהושע אלו עשרת השבטי' שנסמכו על חזקיהו מלך יהודה ועל יהודה ופלטו עמהם : <b>בפאת מטה .</b> מלמד שלא נשתייר מהם אלח א' משמונה שבהם ושאר ערש היכן הוא בדמשק לקיים מה שנאמר ( לקמן ה' ) והגליתי אתכם מהלאה לדמשק ושמעתי משמו של ר' מנחם זצ''ל שהיה מפרש זו א' משמונה שלא נאמר פאת אלא בפאת מטה במקצתה בא' משתי רוחות של פאה והיא שמינית של מטה שהרי ארבע פאות יש ושתי רוחות לפאה , ד''א מצאתי בדברי ר' מאיר זצ''ל בפאת מטה בעון אחזיה בן אחאב ששלח לדרוש בבעל זבוב אלהי עקרון האחיה מחלי זה ( מלכים ב א ) ובדמשק ערש בן הדד מלך ארם שלח לדרוש באלהי ישראל ( שם ח' ) :
<b style="color: gray;">מצודת דוד</b> <b>כאשר יציל הרועה .</b> דרך הרועה כשהארי טורף שה מן העדר הוא מחזיר להציל מה להביא לראיה לבעליו שהארי טרפו ושהוא לא גנבו ולאח''ז משליכו כי לא הצילו כ''א להיות לעד ולא למאכל : <b>כן ינצלו .</b> ר''ל כמו שאין תועלת בעצם הדבר שמציל הרועה מן הארי כן לא יהיה תועלת באותן הנשארים בשומרון כי רק ישארו החולים השוכבים בפאת המטה ובערש המטונף ממשקה החולה :
<b style="color: gray;">מצודת ציון</b> <b>בדל אזן .</b> ת''י חסחוס והוא הקשה שבאזן ואין לו דמיון : <b>בפאת .</b> ענין עבר וצד : <b>ובדמשק ערש .</b> ערש היא מטה כמו ערשו ערש ברזל ( דברים ג ) ויקרא מטת החולה דמשק ערש כי כולה משקה מכוחו וניעו מן הזיעה וכן העיר דמשק נקראת בזה השם על כי היא כולה משקה במרבית יאורים ואגמי מים ועם כי היא בשי''ן שמאלית :
<h3>{יג}</h3>
שִׁמְע֥וּ וְהָעִ֖ידוּ בְּבֵ֣ית יַֽעֲקֹ֑ב נְאֻם־אֲדֹנָ֥י יְהוִ֖ה אֱלֹהֵ֥י הַצְּבָאֽוֹת׃
<b style="color: gray;">מצודת דוד</b> <b>שמעו .</b> אתם הנביאים שמעו דברי והתרו בבית יעקב :
<b style="color: gray;">מצודת ציון</b> <b>והעידו .</b> ענין התראה כי ע''פ רוב מתרין בעדים לבל יכחש לאח''ז :
<h3>{יד}</h3>
כִּ֗י בְּי֛וֹם פָּקְדִ֥י פִשְׁעֵֽי־יִשְׂרָאֵ֖ל עָלָ֑יו וּפָֽקַדְתִּי֙ עַל־מִזְבְּח֣וֹת בֵּֽית־אֵ֔ל וְנִגְדְּעוּ֙ קַרְנ֣וֹת הַמִּזְבֵּ֔חַ וְנָפְל֖וּ לָאָֽרֶץ׃
<b style="color: gray;">מצודת דוד</b> <b>כי ביום פקדי .</b> ר''ל וזהו דברי אשר ביום שאזכור פשעי ישראל עליו לשלם לו הגמול אז אשגיח על המזבחות העומדות בבית אל לכרות קרנותם ויפלו על פני הארץ :
<b style="color: gray;">מצודת ציון</b> <b>פקדי .</b> ענין זכרון : <b>ופקדתי .</b> עוין השגחה כמו פקדו נא וראו ( ש''א יד ) : <b>ונגדעו .</b> ענין כריתה כמו ואשריהם תגדעון ( דברים ז ) : <b>קרנות .</b> הבליטות היוצאות כעין קרנים :
<h3>{טו}</h3>
וְהִכֵּיתִ֥י בֵית־הַחֹ֖רֶף עַל־בֵּ֣ית הַקָּ֑יִץ וְאָבְד֞וּ בָּתֵּ֣י הַשֵּׁ֗ן וְסָפ֛וּ בָּתִּ֥ים רַבִּ֖ים נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>בית החורף על בית הקיץ .</b> מתוך שהיו מעונגים היו בונים להם העשירי' שתי בתים אחד לחורף ואחד לקיץ : <b>בתי השן .</b> דמכבשין בשן דפילא :
<b style="color: gray;">מצודת דוד</b> <b>בית החורף .</b> להתענג היו עושים בית מיוחד לחורף והיה שמור מן הקרירות ומנשיבת הרוח ובית מיוחד לקיץ במקום שנושב הרוח ואמר שיכה זה עם זה ר''ל שיחרבו ויאבדו :
<b style="color: gray;">מצודת ציון</b> <b>על בית .</b> עם בית : <b>השן .</b> שן הפיל : <b>וספו .</b> ענין כליון כמו ספתה בהמות ( ירמיה יב ) :
<h2>עמוס פרק-ד</h2>
<h3>{א}</h3>
שִׁמְע֞וּ הַדָּבָ֣ר הַזֶּ֗ה פָּר֤וֹת הַבָּשָׁן֙ אֲשֶׁר֙ בְּהַ֣ר שֹֽׁמְר֔וֹן הָעֹשְׁק֣וֹת דַּלִּ֔ים הָרֹצְצ֖וֹת אֶבְיוֹנִ֑ים הָאֹמְרֹ֥ת לַאֲדֹֽנֵיהֶ֖ם הָבִ֥יאָה וְנִשְׁתֶּֽה׃
<b style="color: gray;">רש"י</b> <b>פרות הבשן .</b> נשי השרים והסגנים וי''ת עתירי נכסיא :
<b style="color: gray;">מצודת דוד</b> <b>פרות הבשן .</b> הם נשי השרים בשמרון ועל כי היו מתענגות ושמנות מרוב כל המשילם אל הפרות הרועות בבשן במרעה שמן שהן שמנות ובריאות בשר : <b>העושקות דלים וגו' .</b> וחוזר ומפרש שאינם עושקות בידיהן רק הן האומרות לבעליהן הביאה יין ונשתה ולפי שאין להם ממה להביא עושקים הדלים תשושי הכח ואם יעמדו נגדם רוצצים אותם ברדוי ומלקות א''כ העושק והרציצה באה ע''י הנשים וכאלו יעשום הם :
<b style="color: gray;">מצודת ציון</b> <b>הבשן .</b> הוא מקום מרעה שמן כמ''ש ורעה הכרמל והבשן ( שם נ ) : <b>העושקות .</b> מל' עושק וגזל : <b>הרוצצות .</b> ענין שבירה כמו קנה רצוץ ( ישעיה מב ) : <b>לאדניהם .</b> ר''ל לבעליהן :
<h3>{ב}</h3>
נִשְׁבַּ֨ע אֲדֹנָ֤י יְהוִה֙ בְּקָדְשׁ֔וֹ כִּ֛י הִנֵּ֥ה יָמִ֖ים בָּאִ֣ים עֲלֵיכֶ֑ם וְנִשָּׂ֤א אֶתְכֶם֙ בְּצִנּ֔וֹת וְאַחֲרִיתְכֶ֖ן בְּסִיר֥וֹת דּוּגָֽה׃
<b style="color: gray;">רש"י</b> <b>ונשא אתכם בצנות .</b> ויטלון יתכון עממיא על תריסהון : <b>ואחריתכן בסירות דוגה .</b> ובנתיכון בדוגית ציידין היא ספינה קטנה של ציידי דגים ולפי שקטנה היא קורא לה סיר וקורא לה בלשון משנה דוגית בב''ב , ואחריתכן פי' דונש בן לברט מצינו זרעו של אדם נקראים אחריתו כמו ( דניאל יא ) ובעמדו תשבר מלכותו ותחץ לד' רוחות השמים ולא לאחריתו כלומר ולא יניח מלכותו לבנו :
<b style="color: gray;">מצודת דוד</b> <b>בקדשו .</b> בקדושתו : <b>ונשא אתכם .</b> האויב ישא ויביא אתכם באניות קטנות עשויות בתמונת צנה והדרך להוליך בהן השפחות לא נשים מכובדות : <b>ואחריתכן .</b> בניכן ובנותיכ' יביאו בספינו' העשויות בתמונת קדרה שמביאים בהן הדגים ואינן עשויות להוליך בהן אנשים והאויב לא יחוש עליהם ויביאם בהם :
<b style="color: gray;">מצודת ציון</b> <b>ונשא .</b> מל' משא וסבל : <b>בצנות .</b> צנה הוא המגן כמו צנה וסוחרה ( תהלים צא ) ור''ל אניות בדמות צנות : <b>ואחריתכן .</b> כן יקראו הבנים והבנות הנשארים אחרי מות האדם וכן ולא לאחריתו ( דניאל יא ) : <b>בסירות .</b> מל' סיר והוא כעין קדרה כמו סיר הבשר ( שמות טז ) : <b>דוגה .</b> מל' דגים :
<h3>{ג}</h3>
וּפְרָצִ֥ים תֵּצֶ֖אנָה אִשָּׁ֣ה נֶגְדָּ֑הּ וְהִשְׁלַכְתֶּ֥נָה הַהַרְמ֖וֹנָה נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>ופרצים תצאנה אשה נגדה .</b> ובפרצי החומה תצאנה שם לנוס : <b>אשה נגדה .</b> בפרצה שתמצא נגדה כמו ויעלו כל העם איש נגדו ( יהושע ו ) שירבו פרצות בחומה ולא יהיו צריכות לבקש פתחים : <b>והשלכתנה ההרמונה .</b> והשלכתן מעליכון הגאוה והשררה שנטלתם ול' ארמי הוא בגמרא ( בבא מציעא פ''ג ) הרמנא דמלכא הוא ציווי המלך וי''ת ויגלון יתכון להלאה מן טורי חרמיני והם הרי חשך ל' חרב וחרבה :
<b style="color: gray;">מצודת דוד</b> <b>ופרצים תצאנה .</b> כאשר ינוסו למלט נפשם אל הארמון שמחוץ לחומת העיר ולהשגב במעוז הארמון הנה תצאנה מן העיר בדרך פרצי החומה הידועות להן כי לא יוכלו ללכת דרך השערים כי האויב ישמור מול השערים וישבום בצאתן ולכן תצאנה דרך הפרצות : <b>אשה נגדה .</b> כ''א תצא בפרצה שתמצא תחילה לפניה : <b>והשלכתנה ההרמונה .</b> תהיינה רדופות ונשלכות אל הארמון להשגב שמה :
<b style="color: gray;">מצודת ציון</b> <b>ופרצים .</b> שבר החומה : <b>והשלכתנה .</b> אמר בלשון הרחבה על מהירת הרדיפה . ההרמונה . אל הארמון הה''א האחרונה באה במקום הלמ''ד או אל בתחלתה והה''א השנייה באה במקום האל''ף :
<h3>{ד}</h3>
בֹּ֤אוּ בֵֽית־אֵל֙ וּפִשְׁע֔וּ הַגִּלְגָּ֖ל הַרְבּ֣וּ לִפְשֹׁ֑עַ וְהָבִ֤יאוּ לַבֹּ֙קֶר֙ זִבְחֵיכֶ֔ם לִשְׁלֹ֥שֶׁת יָמִ֖ים מַעְשְׂרֹֽתֵיכֶֽם׃
<b style="color: gray;">רש"י</b> <b>באו בית אל וגו' .</b> כאדם האומר לבן בליעל עשה עשה עד שתתמלא סאתך : <b>והביאו לבקר זבחיכם .</b> זאת אומרים להם כומרי הבעל עכו''ם וותרנית היא ואינה מחמרת עליכם התורה אמרה ( שמות כג ) לא ילין לבקר אבל לעכו''ם והביאו לבקר תורה אמרה בקדשים קלים ביום זבחכם יאכל ממחרת ( ויקרא יט ) ועכו''ם לג' ימים מעשר בהמותיכם תורה אמרה שאור לא תקטירו ( שם ב ) :
<b style="color: gray;">מצודת דוד</b> <b>באו בית אל .</b> הוא דרך גזום כאומר לבן בליעל עשה כחפצך ותראה מה יקרה לך באחרית הימים ודומה לו שמח בחור בילדותיך וגו' ואחריו נאמר ודע כי על כל אלה יביאך האלהים במשפט ( קהלת יא ) : <b>ופשעו .</b> להקריב שם אל העגל : <b>הגלגל .</b> מוסב על באו לומר באו לגלגל ושם הרבו לפשוע כי הזביחה שמה אל העכו''ם הוא עון גדול ורב לפני המקום לפי ששם עמד המשכן בתחילה וכן נאמר כל רעתם בגלגל ( הושע ט ) : <b>והביאו לבוקר .</b> בכל בוקר הקדימו להביא שם זבחיכם : <b>לשלשת ימים .</b> בכל כלות שלשה שנים הביאו שמה מעשרותיכם וכדרך שצותה התורה להביא לירושלים מקום המקדש כמ''ש כי תכלה לעשר וגו' בשנה השלישית וגו' ואמרת לפני ה' וגו' ( דברים כו ) הנה אתם הביאום לבית אל והגלגל מקום העגל והפסל :
<b style="color: gray;">מצודת ציון</b> <b>ימים .</b> ענינו שנים כמו ימים תהיה גאולתו ( ויקרא כה ) :
<h3>{ה}</h3>
וְקַטֵּ֤ר מֵֽחָמֵץ֙ תּוֹדָ֔ה וְקִרְא֥וּ נְדָב֖וֹת הַשְׁמִ֑יעוּ כִּ֣י כֵ֤ן אֲהַבְתֶּם֙ בְּנֵ֣י יִשְׂרָאֵ֔ל נְאֻ֖ם אֲדֹנָ֥י יְהוִֽה׃
<b style="color: gray;">רש"י</b> <b>וקטר מחמץ תודה .</b> תורה אמרה לא תאחר לשלמו ( דברי' כג ) ועכו''ם וקראו נדבות השמיעו הזכירו כל מי שמתנדב ושכח יתן אל לבו ויזכור וי''ת וקטר מחמץ מן אונס כמו אשרו חמוץ ( ישעיה א ) מכף מעול וחומץ ( תהלים עא ) : <b>וקראו נדבות השמיעו .</b> בקול זמר ואמרו שקרבן זה מרוצה ומקובל בנדבה כן ת''י :
<b style="color: gray;">מצודת דוד</b> <b>וקטר מחמץ תודה .</b> הקטירו לחמי תודה מן לחם החמץ שבהן כי כן היה דרכם אבל התורה לא צותה להקטיר מלחמי התודה ומכ''ש לחם החמץ שאין החמץ נקטר כלל בכ''מ כמ''ש כי כל שאור וכל דבש לא תקטירו וכו' ( ויקרא ב ) : <b>וקראו נדבות השמיעו .</b> הכריזו להביא שמה נדבות והשמיעו ברבים : <b>כי כן אהבתם .</b> בזה יפרש אמריו לומר לא שאני מצוה אתכם לעשות כן באמת אבל אמרי המה כי כן אהבתם ולכן אני אומר עשו כאשר אהבתם ותראו מה יקרה אתכם :
<h3>{ו}</h3>
וְגַם־אֲנִי֩ נָתַ֨תִּי לָכֶ֜ם נִקְי֤וֹן שִׁנַּ֙יִם֙ בְּכָל־עָ֣רֵיכֶ֔ם וְחֹ֣סֶר לֶ֔חֶם בְּכֹ֖ל מְקוֹמֹֽתֵיכֶ֑ם וְלֹֽא־שַׁבְתֶּ֥ם עָדַ֖י נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>נקיון שנים .</b> אקהיות שנים , ד''א נקיון שנים זהו חוסר אכילת בשר שהוא נכנס בין השנים כמו שנאמ' הבשר עודנו בין שניהם ( מדבר יא ) :
<b style="color: gray;">מצודת דוד</b> <b>וגם אני .</b> אבל גם אני שלמתי מדה במדה לפי שמנעתם ממזבחי בשר קרבנות ומנחות והנה מזבחי נקי אין עולה ואין מנחה כן אנכי החסרתי מכם בכל עריכם הבשר והלחם באופן שיהיה לכם נקיון שנים מהבשר כי דרך הבשר ליכנס בין השנים בעת האכילה וכמ''ש הבשר עודנו בין שניהם ( במדבר יב ) ובהעדר הבשר השנים נקיים וכן היו חסירי לחם כי האויב בא בארצם שנה בשנה ואכל תבואות השדה ושלל את המקנה : <b>ולא שבתם עדי .</b> עם כל זה לא לקחתם מוסר ולא שבתם אלי :
<b style="color: gray;">מצודת ציון</b> <b>נקיון .</b> מל' נקי : <b>עדי .</b> אלי כמו תגע עדיך ( איוב ד ) :
<h3>{ז}</h3>
וְגַ֣ם אָנֹכִי֩ מָנַ֨עְתִּי מִכֶּ֜ם אֶת־הַגֶּ֗שֶׁם בְּע֨וֹד שְׁלֹשָׁ֤ה חֳדָשִׁים֙ לַקָּצִ֔יר וְהִמְטַרְתִּי֙ עַל־עִ֣יר אֶחָ֔ת וְעַל־עִ֥יר אַחַ֖ת לֹ֣א אַמְטִ֑יר חֶלְקָ֤ה אַחַת֙ תִּמָּטֵ֔ר וְחֶלְקָ֛ה אֲשֶֽׁר־לֹֽא־תַמְטִ֥יר עָלֶ֖יהָ תִּיבָֽשׁ׃
<b style="color: gray;">רש"י</b> <b>בעוד שלשה חדשים לקציר .</b> עד הקציר כלומר ג' חדשים לפני הקציר : <b>חלקה .</b> בקעת שדה קמפנייא בלע''ז : <b>תמטר .</b> תהא מקום מטר ( תעני' כ ) אמר מר שתיהן לקללה זו תרקב וזו תיבש : <b>וחלקה אשר לא תמטיר עליה .</b> לא תראה נמטרת את עצמה :
<b style="color: gray;">מצודת דוד</b> <b>וגם אנכי מנעתי מכם וגו' .</b> ועוד גם זאת עשיתי אני שמנעתי מכם את הגשם כשהיה עוד שלשה חדשים לזמן הקציר ואז הצורך רב אל הגשמים : <b>והמטרתי .</b> ולמען לא יאמרו שבמקרה קרה לשנה שאיננה גשומה לכן המטרתי על עיר אחת ועל עיר אחת הסמוכה לה לא המטרתי למען דעת שבא בהשגחה : <b>חלקה .</b> ואפילו בעיר שירדה המטר לא ירדה בכל המקומות אבל חלקה אחת לבד היתה נמטרת והחלקה שאצלה אשר לא תמטיר העב עליה היתה יבשה ורטיבות החלקה שאצלה לא הועיל לה להצילה מן היבשות לדעת שבא בהשגחה :
<b style="color: gray;">מצודת ציון</b> <b>חלקה .</b> כן יקרא השדה כמו ואכלה את החלק ( לקמן ז ) :
<h3>{ח}</h3>
וְנָע֡וּ שְׁתַּיִם֩ שָׁלֹ֨שׁ עָרִ֜ים אֶל־עִ֥יר אַחַ֛ת לִשְׁתּ֥וֹת מַ֖יִם וְלֹ֣א יִשְׂבָּ֑עוּ וְלֹֽא־שַׁבְתֶּ֥ם עָדַ֖י נְאֻם־יְהוָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>ונעו .</b> אנשים משתים או משלש ערים שלא ירד שם הגשם היו נעים ללכת אל עיר אחת שהגשם ירד בה למצוא שם מים לשתות ולא היו שבעים מן השתיה כי מעט מים מצאו בה ולא היה בו די לרוות הצמאון : <b>ולא שבתם .</b> עכ''ז לא לקחתם מוסר ולא שבתם אלי :
<b style="color: gray;">מצודת ציון</b> <b>ונעו .</b> מל' תנועה והנדה :
<h3>{ט}</h3>
הִכֵּ֣יתִי אֶתְכֶם֮ בַּשִּׁדָּפ֣וֹן וּבַיֵּרָקוֹן֒ הַרְבּ֨וֹת גַּנּוֹתֵיכֶ֧ם וְכַרְמֵיכֶ֛ם וּתְאֵנֵיכֶ֥ם וְזֵיתֵיכֶ֖ם יֹאכַ֣ל הַגָּזָ֑ם וְלֹֽא־שַׁבְתֶּ֥ם עָדַ֖י נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>הרבות גנותיכם .</b> שאתם עוסקי' להרבות גנותיכם : <b>יאכל הגזם .</b> אכל הגזם :
<b style="color: gray;">מצודת דוד</b> <b>הרבות .</b> מרבית פרי גנותיכם וגו' אכל ארבה ולא קבלתם הנאה ממרבית הפירות :
<b style="color: gray;">מצודת ציון</b> <b>בשדפון .</b> הוא לקיות הזרע בשבלים ע''י הרוח : <b>ובירקון .</b> בעבור מרבית היובש נהפכים השבלים למראה ירוק ונפסדים כמו יככה וגו' ובשדפון ובירקון ( דברים כח ) : <b>הרבות .</b> מל' רבוי : <b>גנותיכם .</b> מל' גן : <b>הגזם .</b> מין ארבה :
<h3>{י}</h3>
שִׁלַּ֨חְתִּי בָכֶ֥ם דֶּ֙בֶר֙ בְּדֶ֣רֶךְ מִצְרַ֔יִם הָרַ֤גְתִּי בַחֶ֙רֶב֙ בַּח֣וּרֵיכֶ֔ם עִ֖ם שְׁבִ֣י סֽוּסֵיכֶ֑ם וָאַעֲלֶ֞ה בְּאֹ֤שׁ מַחֲנֵיכֶם֙ וּֽבְאַפְּכֶ֔ם וְלֹֽא־שַׁבְתֶּ֥ם עָדַ֖י נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>בדרך מצרים .</b> כדרך ששלחתי עליכם במדבר בצאתכם ממצרים : <b>עם שבי סוסיכם .</b> שיהרגו גם הסוסים : <b>ואעלה באוש מחניכם .</b> ריח פגריכם מבאיש באפכם העליתי אותו :
<b style="color: gray;">מצודת דוד</b> <b>בדרך מצרים .</b> כשהלכו בחוריכם רוכבי סוסים לרדת מצרימה לשבור בר כי שם יש הרבה תבואה הנה בדרך אלה בתחילת מלכותו כמ''ש ( בס''ע ) : <b>ותהיו .</b> אתם בני שומרון לבד תשארו כאוד המוצל מן השרפה : <b>ולא שבתם .</b> ר''ל ידעתי שלא תקחו מוסר ולא תשובו אלי ואמר בלשון עבר כדרך הנבואה בהרבה מן המקומות כי הנבואה ברורה כאלו כבר נעשתה :
<b style="color: gray;">מצודת ציון</b> <b>עם שבי .</b> כמו ושבי וכן עם יפה עינים ( ש''א טז ) : <b>באש .</b> ענין סרחון : <b>מחניכם .</b> חבורת אנשים קרויים מחנה : <b>ובאפכם .</b> מל' אף וחוטם :
<h3>{יא}</h3>
הָפַ֣כְתִּי בָכֶ֗ם כְּמַהְפֵּכַ֤ת אֱלֹהִים֙ אֶת־סְדֹ֣ם וְאֶת־עֲמֹרָ֔ה וַתִּהְי֕וּ כְּא֖וּד מֻצָּ֣ל מִשְּׂרֵפָ֑ה וְלֹֽא־שַׁבְתֶּ֥ם עָדַ֖י נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>כאוד .</b> טיסו''ן בלע''ז כלומר שחורים וחשוכים וקדרים :
<b style="color: gray;">מצודת ציון</b> <b>כאוד .</b> הוא העץ שמנדנדים בו האש הנחרך מסביב וכן זנבות האודים ( ישעיה ז ) :
<h3>{יב}</h3>
לָכֵ֕ן כֹּ֥ה אֶעֱשֶׂה־לְּךָ֖ יִשְׂרָאֵ֑ל עֵ֚קֶב כִּֽי־זֹ֣את אֶֽעֱשֶׂה־לָּ֔ךְ הִכּ֥וֹן לִקְרַאת־אֱלֹהֶ֖יךָ יִשְׂרָאֵֽל׃
<b style="color: gray;">רש"י</b> <b>לכן .</b> יען אשר לא שבתם עדי : <b>כה אעשה לך .</b> ככל הרעות האלה : <b>עקב .</b> כי אתה שומע מפי נביאי אשר זאת אעשה לך : <b>הכון לקראת .</b> לשוב מרשעתך :
<b style="color: gray;">מצודת דוד</b> <b>לכן .</b> הואיל ולא שבתם עד הנה לכן אעשה לך כה כמו שאמרתי שאהפוך בכם כמהפכת וגו' : <b>עקב .</b> בעבור אשר אעשה לך את זאת האמור למעלה וידעת א''כ מזה לכן הכון עצמך אתה ישראל לקראת אלהיך להלחם בו לעכב על ידו ואמר כמהתל ומלעיג וכאומר אף שידעת הדבר אין בידך להיות נשמר ממנה :
<b style="color: gray;">מצודת ציון</b> <b>כה .</b> ענינו כמו כן : <b>עקב .</b> בעבור כמו עקב תשמעון ( דברים ז ) : <b>הכין .</b> מל' הכנה :
<h3>{יג}</h3>
כִּ֡י הִנֵּה֩ יוֹצֵ֨ר הָרִ֜ים וּבֹרֵ֣א ר֗וּחַ וּמַגִּ֤יד לְאָדָם֙ מַה־שֵּׂח֔וֹ עֹשֵׂ֥ה שַׁ֙חַר֙ עֵיפָ֔ה וְדֹרֵ֖ךְ עַל־בָּ֣מֳתֵי אָ֑רֶץ יְהוָ֥ה אֱלֹהֵֽי־צְבָא֖וֹת שְׁמֽוֹ׃
<b style="color: gray;">רש"י</b> <b>מה שיחו .</b> כל מעשיו פורטין לפניו בשעת מיתתו : <b>עושה שחר .</b> לצדיקים כאור נוגה : <b>עיפה .</b> הופך נוגהן של רשעים לחשך אפלה כמו ( ישעיה ח ) והנה צרה וחשכה מעוף צוקה וכן ( איוב י ) ארץ עיפתה תעופה כבקר תהיה ( שם יא ) חשכך כבקר יהי מאיר : <b>ודורך על במתי ארץ .</b> משפיל גבוהים וגסי הרוח :
<b style="color: gray;">מצודת דוד</b> <b>כי הנה יוצר הרים .</b> כי הנה ה' הוא היוצר את ההרים והוא הבורא את הרוח והכל שלו ובידו והוא מגיד לאדם מה יהיה דבורו כי יודע הוא תעלומות לב ובידו להפך אור השחר להיות חושך והוא דורך על במתי ארץ ר''ל משפיל הוא את הרמים , כי שמו ה' אלהי צבאות כי הוא מושל בצבאות מעלה ומטה וכאומר אה כל אלה בו מי א''כ יוכל להיות נשמר מהרעה שהוא מייעד להביא :
<b style="color: gray;">מצודת ציון</b> <b>שחו .</b> דבורו כמו או שיח לארץ ( איוב יב ) : <b>שחר .</b> אור השחר : <b>עיפה .</b> ענין חושך כמו ארץ עיפתה ( שם י ) : <b>ודורך .</b> מל' דריכה ורמיסה : <b>במתי .</b> ענין גבהות ורוממות כמו ועל במותי יעמידני ( תהלים יח ) :
<h2>עמוס פרק-ה</h2>
<h3>{א}</h3>
שִׁמְע֞וּ אֶת־הַדָּבָ֣ר הַזֶּ֗ה אֲשֶׁ֨ר אָנֹכִ֜י נֹשֵׂ֧א עֲלֵיכֶ֛ם קִינָ֖ה בֵּ֥ית יִשְׂרָאֵֽל׃
<b style="color: gray;">מצודת דוד</b> <b>נושא עליכם .</b> מרים קול קינה עליכם אתם בית ישראל :
<b style="color: gray;">מצודת ציון</b> <b>נושא .</b> ענין הרמת קול :
<h3>{ב}</h3>
נָֽפְלָה֙ לֹֽא־תוֹסִ֣יף ק֔וּם בְּתוּלַ֖ת יִשְׂרָאֵ֑ל נִטְּשָׁ֥ה עַל־אַדְמָתָ֖הּ אֵ֥ין מְקִימָֽהּ׃
<b style="color: gray;">רש"י</b> <b>בתולת ישראל .</b> משיגלו בימי הושע בן אלה ( מלכים ב טז ) לא יעמוד עוד מלך מעשרת השבטים : <b>נטשה על אדמתה .</b> כל דבר המושלך ומוטל ונופץ על השדה קורהו נטישה :
<b style="color: gray;">מצודת דוד</b> <b>נפלה .</b> ר''ל הנה עתה עדת ישראל נפלה בשפל המצב ולא תוסיף עוד לקום להחזיר לקדמותה לממשלתה הראשונה עד אשר יגלו מארצם ( אבל בגאולה העתידה ישובו גם המה כמו יהודה ובנימין ומלך אחד יהיה לכולם ויחזרו א''כ גם הם לממשלתם הראשונה ) : <b>נטשה על אדמתה .</b> עודנה בארצה היא עזובה ואין מי מקימה כי אין הכח ביד מלכיה ושריה להקימה מעפרה :
<b style="color: gray;">מצודת ציון</b> <b>בתולת .</b> קראה בתולה לפי שלא נכבשה עדיין ולא באה ברשות אחר כבתולה שלא באה ברשות בעל וכן בתולת בת ציון ( ישעי' כג ) : <b>נטשה .</b> ענין עזיבה כמו נטשתה עמך ( שם ב ) :
<h3>{ג}</h3>
כִּ֣י כֹ֤ה אָמַר֙ אֲדֹנָ֣י יְהוִ֔ה הָעִ֛יר הַיֹּצֵ֥את אֶ֖לֶף תַּשְׁאִ֣יר מֵאָ֑ה וְהַיּוֹצֵ֥את מֵאָ֛ה תַּשְׁאִ֥יר עֲשָׂרָ֖ה לְבֵ֥ית יִשְׂרָאֵֽל׃
<b style="color: gray;">רש"י</b> <b>היוצאת אלף .</b> המוציאה אלף איש :
<b style="color: gray;">מצודת דוד</b> <b>העיר היוצאת אלף .</b> שהיו יוצאים ממנה אלף אנשים לצבא לא ישארו בה כ''א מאה : <b>לבית ישראל .</b> מוסב על העיר לומר העיר של בית ישראל היוצאת אלף וגו' :
<h3>{ד}</h3>
כִּ֣י כֹ֥ה אָמַ֛ר יְהוָ֖ה לְבֵ֣ית יִשְׂרָאֵ֑ל דִּרְשׁ֖וּנִי וִֽחְיֽוּ׃
<b style="color: gray;">רש"י</b> <b>דרשוני וחיו .</b> עלו לרגל לשלם נדריכם בירושלים :
<b style="color: gray;">מצודת דוד</b> <b>דרשוני .</b> דרשו אותי ותשארו בחיים :
<h3>{ה}</h3>
וְאַֽל־תִּדְרְשׁוּ֙ בֵּֽית־אֵ֔ל וְהַגִּלְגָּל֙ לֹ֣א תָבֹ֔אוּ וּבְאֵ֥ר שֶׁ֖בַע לֹ֣א תַעֲבֹ֑רוּ כִּ֤י הַגִּלְגָּל֙ גָּלֹ֣ה יִגְלֶ֔ה וּבֵֽית־אֵ֖ל יִהְיֶ֥ה לְאָֽוֶן׃
<b style="color: gray;">רש"י</b> <b>ואל תדרשו בית אל וגו' ובאר שבע לא תעבורו .</b> לא נאמר כאן אל תבואו ואל תדרשו אלא אל תעבורו למדנו שלא היתה שם עכו''ם אלא כל זמן שהיה א' מבני עשרת השבטים יוצא לדרוש ולא עבר את באר שבע עודנו יכול לפנות לבית המקדש לירושלי' ומשעבר את באר שבע הוא הדרך לעגל בית אל או לעגל שבדן וכן הוא במקום אחר ( לקמן ח ) חי אלהיך דן וחי וגו' ותדע שגזר הכתוב גזירת פורענות על גלגל ובית אל ולא גזר על באר שבע : <b>הגלגל גלה יגלה .</b> לפי שמו קללתו וכן בית אל יש עיר שסמוכה לה ששמה בית און בספר ( שופטים ז ) : <b>יהיה לאון .</b> יהון ללמא והוא תרגום של הבל וזה דוגמת אם גלעד און אך שוא היו ( הושע יב ) :
<b style="color: gray;">מצודת דוד</b> <b>בית אל .</b> ר''ל את העגל אשר בבית אל : <b>והגלגל .</b> אל הגלגל לא תבאו אל העכו''ם העומדת שם : <b>ובאר שבע .</b> לבאר שבע לא תעבורו כי גם שם בנו מזבחות לעכו''ם כמ''ש וחי דרך באר שבע ( לקמן ח ) : <b>כי הגלגל .</b> כי בסבת עבודת הגלגל גלה יגלה ישראל מעל אדמתו והוא לשון הנופל על לשון : <b>ובית אל .</b> בסבת עבודת בית אל יהיה ישראל לעמל ויגיעה וגם זה הוא לשון נופל על הלשון כי בית אל כינה הכתוב לגנאי בשם בית און כמ''ש ואל תעלו בית און ( הושע ד ) :
<b style="color: gray;">מצודת ציון</b> <b>גלה יגלה .</b> מל' גלות : <b>לאון .</b> ענין עמל ויגיעה וכן אם גלעד און ( הושע יב ) :
<h3>{ו}</h3>
דִּרְשׁ֥וּ אֶת־יְהוָ֖ה וִֽחְי֑וּ פֶּן־יִצְלַ֤ח כָּאֵשׁ֙ בֵּ֣ית יוֹסֵ֔ף וְאָכְלָ֥ה וְאֵין־מְכַבֶּ֖ה לְבֵֽית־אֵֽל׃
<b style="color: gray;">רש"י</b> <b>בית יוסף .</b> כמו לבית יוסף . פן יצלח כאש . פן תעלה חמתו כאש לבית יוסף :
<b style="color: gray;">מצודת דוד</b> <b>פן יצלח .</b> ר''ל פן חמת ה' יצלח ויתגבר על בית יוסף כאש המצליח ומתגבר והולך ושורף : <b>בית יוסף .</b> היא מלכות אפרים שהם מבני יוסף : <b>ואכלה .</b> ותשרוף אנשי שומרון העובדים לעגל שבבית אל ולא ימצא מי מכבה השרפה ר''ל לא ימצאו מושיע ומציל :
<b style="color: gray;">מצודת ציון</b> <b>יצלח .</b> ענין התגברות : <b>ואכלה .</b> ותשרף :
<h3>{ז}</h3>
הַהֹפְכִ֥ים לְלַעֲנָ֖ה מִשְׁפָּ֑ט וּצְדָקָ֖ה לָאָ֥רֶץ הִנִּֽיחוּ׃
<b style="color: gray;">רש"י</b> <b>ההופכים ללענה משפט .</b> ואינם זוכרין הפיכותי שאני הופך לבקר צלמות יש ששרויין עתה בחשך ואני מאיר להם :
<b style="color: gray;">מצודת דוד</b> <b>ההופכים .</b> אשר הם מהפכים משפט התורה המתוק כדבש לקחת תמורתו את העגל המרה כלענה : <b>לארץ הניחו .</b> משפילים הצדק' עדי ארץ :
<b style="color: gray;">מצודת ציון</b> <b>ללענה .</b> שם עשב מר :
<h3>{ח}</h3>
עֹשֵׂ֨ה כִימָ֜ה וּכְסִ֗יל וְהֹפֵ֤ךְ לַבֹּ֙קֶר֙ צַלְמָ֔וֶת וְי֖וֹם לַ֣יְלָה הֶחְשִׁ֑יךְ הַקּוֹרֵ֣א לְמֵֽי־הַיָּ֗ם וַֽיִּשְׁפְּכֵ֛ם עַל־פְּנֵ֥י הָאָ֖רֶץ יְהוָ֥ה שְׁמֽוֹ׃
<b style="color: gray;">רש"י</b> <b>ויום לילה החשיך .</b> ויש ששעתו עומדת לו כאור היום ואני מחשיכו ללילה : <b>הקורא למי הים .</b> יונתן תרגם דאמר לכנשא משריין סגיאין כמי ימא ומבדר להון על אפי ארעא רבותינו פירשו על דור אנוש שהציף עליהם את מי אוקינוס : <b>ה' שמו .</b> לו היה לכם לעבוד והוא היה מגביר אתכם על שונאיכם :
<b style="color: gray;">מצודת דוד</b> <b>עושה כימה וכסיל .</b> מוסב למקרא שלפני פניו שאמר דרשו את ה' והוא זה העושה כימה וכסיל עם כי המה מתחלפים בפעולותיהם כימה מקרר וכסיל מחמם : <b>והופך וגו' .</b> את הצלמות מהפך להיות בוקר ואת היום מחשיך להיות לילה : <b>הקורא למי הים .</b> להעלותם בעבים ואחר זמן שופכים ע''פ הארץ : <b>ה' שמו .</b> המורה שהוא אדון הכל :
<b style="color: gray;">מצודת ציון</b> <b>כימה וכסיל .</b> שמות מזלות : <b>צלמות .</b> צלו של מות והוא חשכת הקבר :
<h3>{ט}</h3>
הַמַּבְלִ֥יג שֹׁ֖ד עַל־עָ֑ז וְשֹׁ֖ד עַל־מִבְצָ֥ר יָבֽוֹא׃
<b style="color: gray;">רש"י</b> <b>המבליג .</b> המגביר כמו ואבליגה מעט ( איוב י ) מבליגיתי עלי יגון ( ירמיה ח ) : <b>שוד .</b> אדם שדוד וחלש דמגבר חלשין על תקיפין ובזוזין על כרכי תקיפין משליט :
<b style="color: gray;">מצודת דוד</b> <b>המבליג .</b> הלא הוא מחזק שוד וחרבן על עם עז : <b>ושוד .</b> ר''ל וכן יבוא חרבן על המבצר שאתם בוטחים בו וכאומר הואיל וכן דרשו את ה' וחיו :
<b style="color: gray;">מצודת ציון</b> <b>המבליג .</b> ענין חוזק כמו ואבליגה מעט ( איוב י ) : <b>שוד .</b> ענין חרבן ואבדון כמו משוד כי יבוא ( שם ה ) :
<h3>{י}</h3>
שָׂנְא֥וּ בַשַּׁ֖עַר מוֹכִ֑יחַ וְדֹבֵ֥ר תָּמִ֖ים יְתָעֵֽבוּ׃
<b style="color: gray;">רש"י</b> <b>שנאו בשער מוכיח .</b> אתם לא יראתם מפניו ומפני נביאיו ושנאתם את המוכיחים אתכם בשערי בתי דינא :
<b style="color: gray;">מצודת דוד</b> <b>שנאו .</b> והם עכ''ז שנאו בשער את הנביא המוכיח ר''ל אף בשער העיר במקום פרסום המה מראים שנאה על המוכיח : <b>ודובר תמים .</b> ומי שמדבר דברים תמימים וישרים אותו מתעבים :
<b style="color: gray;">מצודת ציון</b> <b>יתעבו .</b> מל' תעוב ושקוץ :
<h3>{יא}</h3>
לָ֠כֵן יַ֣עַן בּוֹשַׁסְכֶ֞ם עַל־דָּ֗ל וּמַשְׂאַת־בַּר֙ תִּקְח֣וּ מִמֶּ֔נּוּ בָּתֵּ֥י גָזִ֛ית בְּנִיתֶ֖ם וְלֹא־תֵ֣שְׁבוּ בָ֑ם כַּרְמֵי־חֶ֣מֶד נְטַעְתֶּ֔ם וְלֹ֥א תִשְׁתּ֖וּ אֶת־יֵינָֽם׃
<b style="color: gray;">רש"י</b> <b>בושסכם .</b> כמו בוססכם על דל שאתם רומסין ורופסים על ראשו כמו בוססו את חלקתי ( שם יב ) : <b>ומשאת בר תקחו .</b> זקיפת מלווה שהפקעתם עליהם השערים למכור להם תבואה בריבית כדי ליטול נחלתם מהם לפיכך בתי' וכרמים אשר בניתם ונטעתם על אותן הנחלות לא יתקיימו בידכם ומשאת ל' משא שמכבידי' עליהם : <b>כרמי חמד .</b> כרמי חמדה :
<b style="color: gray;">מצודת דוד</b> <b>יען בושסכם .</b> בעבור שרמיסת רגליכם הוא על הדל ותגזלו ממנו המשא של תבואה שעל שכמו לזה בתי הגזית שבניתם והרי הם שלכם ואעפ''כ לא תשבו בם כי האויב יקחום וכן כרמים חמודי' שנטעתם ושלכם המה ועכ''ז לא תשתו אתם את יינם כ''א האויב :
<b style="color: gray;">מצודת ציון</b> <b>בושסכם .</b> כמו בוססכם והוא ענין רמיסה כמו בוססו את חלקתי ( ירמיה יב ) : <b>ומשאת .</b> מל' משא : <b>בר .</b> תבואה כמו לשבור בר ( בראשית מב ) : <b>גזית .</b> אבנים מסותתים ביושר כמו ולא תבנה אתהן גזית ( שמות יז ) :
<h3>{יב}</h3>
כִּ֤י יָדַ֙עְתִּי֙ רַבִּ֣ים פִּשְׁעֵיכֶ֔ם וַעֲצֻמִ֖ים חַטֹּֽאתֵיכֶ֑ם צֹרְרֵ֤י צַדִּיק֙ לֹ֣קְחֵי כֹ֔פֶר וְאֶבְיוֹנִ֖ים בַּשַּׁ֥עַר הִטּֽוּ׃
<b style="color: gray;">רש"י</b> <b>צוררי צדיק .</b> מעיקין ליה לזכאה : <b>בשער הטו .</b> בשערי בתי דיניכם הטו משפט אביוני' :
<b style="color: gray;">מצודת דוד</b> <b>רבים פשעיכם .</b> אשר פשעיכם רבים וחטאתיכם המה חזקים אתם צוררים את הצדיק ולוקחים כופר לפטור מעונש מיתה את המחויב ובשער מקום המשפט מטים דין האביונים :
<b style="color: gray;">מצודת ציון</b> <b>ועצומים .</b> חזקים : <b>צוררי .</b> מל' צר ואויב : <b>כופר .</b> פדיון :
<h3>{יג}</h3>
לָכֵ֗ן הַמַּשְׂכִּ֛יל בָּעֵ֥ת הַהִ֖יא יִדֹּ֑ם כִּ֛י עֵ֥ת רָעָ֖ה הִֽיא׃
<b style="color: gray;">רש"י</b> <b>המשכיל .</b> מי שהוא חכם ידום בבא הרעה ולא יהרהר אחר מה''ד שהרי כל אלה עשו : <b>כי עת רעה היא .</b> עת המזומנת לקבל רעתם :
<b style="color: gray;">מצודת דוד</b> <b>לכן .</b> בעבור המשפטים האלה הנה המשכיל ידום בעת בוא הפורעניות ולא יהרהר אחר מדת הדין כי בא העת שיקבלו הרעה כי נתמלאה הסאה :
<b style="color: gray;">מצודת ציון</b> <b>ידום .</b> ענין שתיקה כמו וידום אהרן ( ויקרא י ) :
<h3>{יד}</h3>
דִּרְשׁוּ־ט֥וֹב וְאַל־רָ֖ע לְמַ֣עַן תִּֽחְי֑וּ וִיהִי־כֵ֞ן יְהוָ֧ה אֱלֹהֵֽי־צְבָא֛וֹת אִתְּכֶ֖ם כַּאֲשֶׁ֥ר אֲמַרְתֶּֽם׃
<b style="color: gray;">רש"י</b> <b>ויהי כן וגו' .</b> אם זאת תעשו אז יהי המקום עמכם כאשר אמרתם בלבבכם שתתקיים לכם גדולתכם :
<b style="color: gray;">מצודת דוד</b> <b>דרשו טוב .</b> לכן דרשו לעשות טוב ואל תדרשו לעשות רע למען תשיגו את החיים : <b>ויהי כן וגו' .</b> ר''ל כן יהיה ה' עמכם תקוע בלבבכם כאשר אמרתם ונדרתם בעת שקבלתם אותו לאלוה :
<h3>{טו}</h3>
שִׂנְאוּ־רָע֙ וְאֶ֣הֱבוּ ט֔וֹב וְהַצִּ֥יגוּ בַשַּׁ֖עַר מִשְׁפָּ֑ט אוּלַ֗י יֶֽחֱנַ֛ן יְהוָ֥ה אֱלֹהֵֽי־צְבָא֖וֹת שְׁאֵרִ֥ית יוֹסֵֽף׃
<b style="color: gray;">רש"י</b> <b>שארית יוסף .</b> כל ישראל נקראין על שמו לפי שכלכל אותם :
<b style="color: gray;">מצודת דוד</b> <b>שנאו רע .</b> דבר הרע בעיני ה' שנאו ואהבו את הטוב בעיניו ואת המשפט העמידו בשער המשפט להיות מתקיים ולעמוד על עמדו : <b>אולי יחנן .</b> אם כן תעשו אולי יחנן ה' את השארית אשר ישארו מיוסף וקרא את כל ישראל ע''ש יוסף לפי שכלכלם במצרים :
<b style="color: gray;">מצודת ציון</b> <b>והציגו .</b> ענין העמדה כמו כמו תציג אותו לבד ( שופי' ז ) : <b>אולי .</b> עניינו שמא : <b>יחנן .</b> מל' חנינה וחמלה :
<h3>{טז}</h3>
לָ֠כֵן כֹּֽה־אָמַ֨ר יְהוָ֜ה אֱלֹהֵ֤י צְבָאוֹת֙ אֲדֹנָ֔י בְּכָל־רְחֹב֣וֹת מִסְפֵּ֔ד וּבְכָל־חוּצ֖וֹת יֹאמְר֣וּ הוֹ־ה֑וֹ וְקָרְא֤וּ אִכָּר֙ אֶל־אֵ֔בֶל וּמִסְפֵּ֖ד אֶל־י֥וֹדְעֵי נֶֽהִי׃
<b style="color: gray;">רש"י</b> <b>הו הו .</b> ל' צעקה : <b>וקראו איכר אל אבל .</b> יפגעו שיירות האיכרים החורשים בשדות בקול מספד האבלים הצועקים בחוצות : <b>ומספד אל יודעי נהי .</b> ועביד מספדא יערע בדעביד אילייא ואינו אלא ל' כפול כמו משפטי ודיני ( תהלים ט' ) :
<b style="color: gray;">מצודת דוד</b> <b>לכן .</b> ר''ל הואיל ולא כן עשיתם לכן כה אמר וגו' : <b>מספד .</b> יהיה מספד : <b>וקראו איכר .</b> יקראו אל האיכר להתאבל על מלאכתו שהיתה בחנם : <b>ומספד .</b> ובמספד יקראו אל יודעי נהי שהם יעוררו המספד :
<b style="color: gray;">מצודת ציון</b> <b>חוצות .</b> שווקים : <b>הו הו .</b> כמו הוי הוי והוא ענין צעקת יללה : <b>אכר .</b> כן יקרא העושה מלאכת השדה וכן הובישו אכרים ( יואל א ) : <b>נהי .</b> ענין קינה ויללה :
<h3>{יז}</h3>
וּבְכָל־כְּרָמִ֖ים מִסְפֵּ֑ד כִּֽי־אֶעֱבֹ֥ר בְּקִרְבְּךָ֖ אָמַ֥ר יְהוָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>מספד .</b> יהיה מספד על חסרון היין : <b>כי אעבור בקרבך .</b> להכות בך :
<h3>{יח}</h3>
ה֥וֹי הַמִּתְאַוִּ֖ים אֶת־י֣וֹם יְהוָ֑ה לָמָּה־זֶּ֥ה לָכֶ֛ם י֥וֹם יְהוָ֖ה הוּא־חֹ֥שֶׁךְ וְלֹא־אֽוֹר׃
<b style="color: gray;">רש"י</b> <b>הוי המתאוים וגו' .</b> האומרים ימהר יחישה מעשהו ( ישעיה ה ) : <b>למה זה לכם וגו' .</b> משל לתרנגול ועטלף שהיו מצפין לאורה כדאיתא בחלק :
<b style="color: gray;">מצודת דוד</b> <b>הוי המתאוים .</b> על בני יהודה יאמר והוא מ''ש האומרים ימהר יחישה מעשהו וגו' ( ישעיה ה ) שהיו בני יהודה מלעיגים בדברי הנבואה ואמרו ימהר להביא יעודי הפורעניות למען נראה אם יוכל עשוהו וכאלו יתאוו לה ואמר הוי על אלה המתאוים יום הפורעניות הבא מה' : <b>למה זה וגו' .</b> ר''ל למה אתם מתאווים : <b>יום ה' .</b> הלא הוא יום חושך ולא יום אור :
<b style="color: gray;">מצודת ציון</b> <b>המתאוים .</b> מל' תאוה :
<h3>{יט}</h3>
כַּאֲשֶׁ֨ר יָנ֥וּס אִישׁ֙ מִפְּנֵ֣י הָאֲרִ֔י וּפְגָע֖וֹ הַדֹּ֑ב וּבָ֣א הַבַּ֔יִת וְסָמַ֤ךְ יָדוֹ֙ עַל־הַקִּ֔יר וּנְשָׁכ֖וֹ הַנָּחָֽשׁ׃
<b style="color: gray;">מצודת דוד</b> <b>כאשר .</b> היום ההוא יהיה כאשר ינוס איש מפני הארי הרודף אחריו ובדרך נוסו פגע בו הדוב וכאשר יבוא אל הבית להטמין עצמו וסמך ידו אל הקיר לחפש אחר מקום נסתר ישכנו הנחש היושב בחור הקיר ר''ל כשתנצלו מצרה האחת תבוא השניה :
<b style="color: gray;">מצודת ציון</b> <b>הקיר .</b> הכותל : <b>ונשכו .</b> מל' נשיכת השנים :
<h3>{כ}</h3>
הֲלֹא־חֹ֛שֶׁךְ י֥וֹם יְהוָ֖ה וְלֹא־א֑וֹר וְאָפֵ֖ל וְלֹא־נֹ֥גַֽהּ לֽוֹ׃
<b style="color: gray;">מצודת דוד</b> <b>הלא חושך .</b> ר''ל א ''כ היום ההוא יהיה חושך ועל מה תתאוו לו :
<b style="color: gray;">מצודת ציון</b> <b>נוגה .</b> הארה וזריחה :
<h3>{כא}</h3>
שָׂנֵ֥אתִי מָאַ֖סְתִּי חַגֵּיכֶ֑ם וְלֹ֥א אָרִ֖יחַ בְּעַצְּרֹֽתֵיכֶֽם׃
<b style="color: gray;">רש"י</b> <b>בעצרותיכם .</b> קורבן כנישתכון כשאתם קוראי' עצר' :
<b style="color: gray;">מצודת דוד</b> <b>שנאתי .</b> ר''ל אל תבטחו בהבאת הקרבנות כי באמת שנאתיו מאסתי קרבנות חגיכם ולא אריח בהקטרת העולות שאתם מביאים בעת שאתם נעצרים ומעוכבים לפני בבה''מ ר''ל אינם מקובלים לפני :
<b style="color: gray;">מצודת ציון</b> <b>חגיכם .</b> קרבנות חגיכם : <b>אריח .</b> מל' ריח : <b>בעצרותיכם .</b> ענין עכבה במקום כמו נעצר לפני ה' ( ש''א כא ) :
<h3>{כב}</h3>
כִּ֣י אִם־תַּעֲלוּ־לִ֥י עֹל֛וֹת וּמִנְחֹתֵיכֶ֖ם לֹ֣א אֶרְצֶ֑ה וְשֶׁ֥לֶם מְרִיאֵיכֶ֖ם לֹ֥א אַבִּֽיט׃
<b style="color: gray;">רש"י</b> <b>ושלם מריאיכם .</b> ושלמי פטימיכון בהמות מפוטמות :
<b style="color: gray;">מצודת דוד</b> <b>ומנחותיכם .</b> ואף המנחות שאתם מעלים לא ארצה בהם : <b>ושלם מריאיכם .</b> השלמים מה שאתם מביאים מריאים לא אביט אליהם ואינם לרצון :
<b style="color: gray;">מצודת ציון</b> <b>ארצה .</b> מל' רצון : <b>ושלם .</b> מל' שלמים : <b>מריאכם .</b> בהמות פטומות וכן ויזבח שור ומריא ( ש''ב ז ) :
<h3>{כג}</h3>
הָסֵ֥ר מֵעָלַ֖י הֲמ֣וֹן שִׁרֶ֑יךָ וְזִמְרַ֥ת נְבָלֶ֖יךָ לֹ֥א אֶשְׁמָֽע׃
<b style="color: gray;">רש"י</b> <b>הסר מעלי המון שריך .</b> שהלוים משוררים על הדוכן אחרי שאתם חמסנים ומעוותי משפט :
<b style="color: gray;">מצודת דוד</b> <b>המון שיריך .</b> המיית השיר מה שהלוים משוררים לפני : <b>וזמרת נבליך .</b> מה שהם מזמרים לפני בכלי נבל לא אשמע כי אינם ערבים עלי :
<b style="color: gray;">מצודת ציון</b> <b>המון .</b> מל' המייה : <b>שריך .</b> מל' שיר : <b>וזמרת .</b> מל' זמר : <b>נבליך .</b> הוא כלי נגון :
<h3>{כד}</h3>
וְיִגַּ֥ל כַּמַּ֖יִם מִשְׁפָּ֑ט וּצְדָקָ֖ה כְּנַ֥חַל אֵיתָֽן׃
<b style="color: gray;">רש"י</b> <b>ויגל כמים משפט .</b> וזאת תעשו וארצה לכם המשפט אשר כבשתם והסתרתם יגולה וישטף ביניכם כמים נובעים : <b>וצדקה .</b> תגלה כנחל איתן :
<b style="color: gray;">מצודת דוד</b> <b>ויגל כמים משפט .</b> ר''ל זאת עשו המשפט אשר כבשתם בעמקי המסתור יגולה כמים הנובעים היוצאים מבטן הארץ ממקום מסתור והצדקה שפסקתם ממנה התמידו בה כנחל חזק השוטף והולך ואינו פוסק :
<b style="color: gray;">מצודת ציון</b> <b>ויגל .</b> מל' גלוי : <b>איתן .</b> חזק כמו נהרות איתן ( תהלים עד ) :
<h3>{כה}</h3>
הַזְּבָחִ֨ים וּמִנְחָ֜ה הִֽגַּשְׁתֶּם־לִ֧י בַמִּדְבָּ֛ר אַרְבָּעִ֥ים שָׁנָ֖ה בֵּ֥ית יִשְׂרָאֵֽל׃
<b style="color: gray;">רש"י</b> <b>הזבחים ומנחה .</b> וכי לזבחים ומנחות רצוני והלא במדבר לא דברתי את אבותיכם להקריב לי זבחים אני אמרתי אדם כי יקריב ( ויקרא א' ) כשירצה להקריב ואומר ויעשו בני ישראל את הפסח במועדו ( מדב' ט ) מגיד שלא עשו ישרא' במדבר אלא אותו פסח לבדו :
<b style="color: gray;">מצודת דוד</b> <b>הזבחים .</b> ר''ל וכי הכל תלוי בזבחים וכי הגשתם לפני במדבר זבחים ומנחות בכל ימי משך מ' שנה שהלכתם בו הלא באותן הימים לא עשו אלא פסח אחד וגם עולות לא קרבו כן ארז''ל וכאומר וכי לא הייתי אז מרוצה לכם בזולת הקרבנות :
<b style="color: gray;">מצודת ציון</b> <b>הגשתם .</b> ענין הקרבה :
<h3>{כו}</h3>
וּנְשָׂאתֶ֗ם אֵ֚ת סִכּ֣וּת מַלְכְּכֶ֔ם וְאֵ֖ת כִּיּ֣וּן צַלְמֵיכֶ֑ם כּוֹכַב֙ אֱלֹ֣הֵיכֶ֔ם אֲשֶׁ֥ר עֲשִׂיתֶ֖ם לָכֶֽם׃
<b style="color: gray;">רש"י</b> <b>ונשאתם את סכות מלככם .</b> חוזר לענין הפורעניות שלמעלה חשך להם יום ה' ולא אור ונשאתם בגולה עמכם את גלוליכם וירכיבום האויבים על צואריכם לשאת עמכם בגולה : <b>סכות .</b> כיון : <b>כוכב .</b> שמות עכו''ם הן :
<b style="color: gray;">מצודת דוד</b> <b>ונשאתם .</b> ר''ל הואיל ואין הכל תלוי בקרבנות א''כ לא יגינו עליכם ותלכו גולה ועל צואריכם תטענו את הפסל של העכו''ם סכות שקבלתם עליכם למלך ולאלוה ואת העכו''ם כיון שעשיתם צלמו לעבוד לפניו ואת העכו''ם כוכב וכו' כי האויב יכריחום להביאם עמהם בגולה :
<b style="color: gray;">מצודת ציון</b> <b>ונשאתם .</b> מל' משא וסבל : <b>סכות , כיון , כוכב .</b> הם שמות מיני עכו''ם :
<h3>{כז}</h3>
וְהִגְלֵיתִ֥י אֶתְכֶ֖ם מֵהָ֣לְאָה לְדַמָּ֑שֶׂק אָמַ֛ר יְהוָ֥ה אֱלֹהֵֽי־צְבָא֖וֹת שְׁמֽוֹ׃
<b style="color: gray;">רש"י</b> <b>והגלתי אתכם מהלאה לדמשק .</b> כלומר לא לשכנותיכם תגלו כאשר עד הנה שנמסרתם ביד מלכי ארם ולא היו שובים אתכם למקום רחוק אלא עד דמשק ומעתה ביד סנחרב אמסור אתכם והוא יגלה אתכם למרחוק :
<b style="color: gray;">מצודת דוד</b> <b>מהלאה לדמשק .</b> רחוק מדמשק משם ולהלן :
<b style="color: gray;">מצודת ציון</b> <b>מהלא' .</b> להלן ורחוק :
<h2>עמוס פרק-ו</h2>
<h3>{א}</h3>
ה֚וֹי הַשַּׁאֲנַנִּ֣ים בְּצִיּ֔וֹן וְהַבֹּטְחִ֖ים בְּהַ֣ר שֹׁמְר֑וֹן נְקֻבֵי֙ רֵאשִׁ֣ית הַגּוֹיִ֔ם וּבָ֥אוּ לָהֶ֖ם בֵּ֥ית יִשְׂרָאֵֽל׃
<b style="color: gray;">רש"י</b> <b>נקובי ראשית הגוים .</b> לכם היו קוראים גוי המשובח שבעכו''ם : <b>ובאו להם בית ישראל .</b> הם בחרו במשפטיהם ובאו לכללם :
<b style="color: gray;">מצודת דוד</b> <b>הוי השאננים .</b> הוי על אלו היושבים שאנן בציון ועל אלו היושבים בטח בהר שומרון : <b>נקובי .</b> מתחלה היו נקובים בשם שהם העם המשובחים שבכל העכו''ם במשפט הנהגת דרכם עתה באו כל בית ישראל לכללם לאחוז במשפטי הנהגת דרכם :
<b style="color: gray;">מצודת ציון</b> <b>השאנני' .</b> ענין השקט ומנוחה כמו שאנן מואב ( ירמיה מח ) : <b>נקובי .</b> כן יאמר על הפרשת השם כמו נקבו בשמות ( במד' א ) : <b>ראשית .</b> ענין מיטב ומבחר כמו ראשי' גבורתם ( בירמיה מט ) :
<h3>{ב}</h3>
עִבְר֤וּ כַֽלְנֵה֙ וּרְא֔וּ וּלְכ֥וּ מִשָּׁ֖ם חֲמַ֣ת רַבָּ֑ה וּרְד֣וּ גַת־פְּלִשְׁתּ֗ים הֲטוֹבִים֙ מִן־הַמַּמְלָכ֣וֹת הָאֵ֔לֶּה אִם־רַ֥ב גְּבוּלָ֖ם מִגְּבֻלְכֶֽם׃
<b style="color: gray;">רש"י</b> <b>עברו כלנה וראו .</b> למה מאסתם בתורתי לדבק בחוקות עכו''ם עברו וראו בעכו''ם החשובים אם טובים הם מן הממלכות האלה שבשומרון ושבירושלים : <b>כלנה .</b> בבל : <b>חמת .</b> אנטוכיא : <b>גת פלשתים .</b> חשובה שבחמשת סרניהם שהיא קרויה מתג האמה ( שמואל ב ח ) על שם שרודה את כולם והיא אם המלכות : <b>אם רב גבולם מגבולכם .</b> ומה מצאתם בי עול :
<b style="color: gray;">מצודת דוד</b> <b>עברו כלנה .</b> עברו לכלנה וראו טוב המקום ומשם לכו אל חמת רבה ורדו אל גת של פלשתים לראות גודל גבולם ושבחם : <b>הטובים .</b> וכי המה טובים מן הממלכות האלה הם מלכות יהודה ומלכות אפרים וכי גבולם גדול מגבולכם ומה א''כ מצאתם בי עול למאס משפטי :
<b style="color: gray;">מצודת ציון</b> <b>כלנה .</b> שם מקום : <b>חמת רבה .</b> חמת הגדולה :
<h3>{ג}</h3>
הַֽמְנַדִּ֖ים לְי֣וֹם רָ֑ע וַתַּגִּישׁ֖וּן שֶׁ֥בֶת חָמָֽס׃
<b style="color: gray;">רש"י</b> <b>המנדים ליום רע .</b> אתם מנידים עצמיכם ליום רע : <b>ותגישון .</b> עצמיכם לשבת בגלות אשר יחמס אתכם שנאמר בו מחמס אחיך יעקב ( עובדי' א ) זה מדרשו של רבי שמשון זצ''ל , ד''א המנדים ליום רע המבזים על נבואת ימי רע שהנביאים מתנבאים להם . ותגישון שבת חמס . זהו כמו שאמר ישעיה הוי מגיעי בית בבית ותגישון שבת בתוכם זו אצל זו על ידי חמס :

<b style="color: gray;">מצודת ציון</b> <b>המנדים .</b> מל' הנדה והנעה : <b>ותגישון .</b> ענין הקרוב : <b>שבת .</b> מל' ישיבה : <b>חמס .</b> עושק וגזל :

<b style="color: gray;">מצודת דוד</b> <b>לכן .</b> הואיל וכן עשו לכן עתה בזמן קרוב יגלו בראש גולים ר''ל במבחר הגולים בבזיון רב ( וכי כמו בדברים טובים יאמר לשון ראש ומבחר על הטוב שבכולן כן בדברים רעים יאמר לשון ראש ומבחר על הרע שבכולן ) : <b>וסר מרזח סרוחים .</b> אז יסיר מהם המשתאות הגדולות העודפות מהשיעור הראוי , או אז יסור אליהם אבלות גדולות ועודפות בשיעור :
<b style="color: gray;">מצודת ציון</b> <b>בראש .</b> ענין מובחר : <b>וסר .</b> מל' הסרה : <b>מרזח .</b> כן יקראו המשתאות ובספרי בפרשת וישב ישראל בשטים יש וחזרו לעשות מרזחים והיו קוראים להם ואוכלים , או הוא ענין אבל וכן אל תבוא בית מרזח ( ירמיה טו ) וארז''ל בית אבל : <b>סרוחים .</b> ענין עודף ויתרון :

<h3>{י}</h3>

<b style="color: gray;">רש"י</b> <b>ונשאו דודו ומסרפו .</b> קרובו ואוהבו המציל את עצמותיו מן השריפה כן תרגום יונתן ויטיליניה קריביה מיקידא : <b>ואמר .</b> המוציא מן השריפה לאשר בירכתי הבית לחבירו שהלך לבדוק בתוך הבית : <b>העוד עמך .</b> אדם חי זולתך שנמלט מן החבואים : <b>ואמר אפס .</b> אין חי אלא מתים ואמר זה אליו הס את הפגרים ואמר הס השלך וסלק הכל כי זאת להם על אשר לא רצו להזכיר בשם ה' כן ת''י :
<b style="color: gray;">מצודת דוד</b> <b>ונשאו .</b> את כל אחד ישא דודו ומסרפו להוציא העצמים מן הבית לבל ישרפו גם הם עם הבשר : <b>ואומר .</b> הנושא הזה יאמר לאשר הוא בסוף הבית המחפש אחר עצמות כמוהו : <b>העוד עמך .</b> אם יש עוד מה עמך מן העצמות להוציאם מן השרפה : <b>ואמר אפס .</b> ר''ל וזה ישיב כמתרעם כבר נשרפו גם העצמות וכלה הכל : <b>ואמר הס .</b> ויחזיר זה ויאמר לו שתוק ואל תהרהר אחר מדת הדין כי אלו הנשרפים לא אבו להזכיר בשם ה' כ''א הזכירו בשם הבעל וראויים הם לזה הגמול :
<b style="color: gray;">מצודת ציון</b> <b>ונשאו .</b> מל' משא וסבל : <b>דודו .</b> הוא אחי האב : <b>ומסרפו .</b> הוא אחי האם , או ומסרפו הוא כמו ומשרפו בשי''ן כי זסשר''ץ מתחלף ור''ל אוהבו המציל עצמותיו מן השריפה : <b>בירכתי .</b> בסוף : <b>אפס .</b> ענין כליון כמו אפס כסף ( בראשית מ''ז ) : <b>הס .</b> ענין שתיקה כמו ויהס כלב ( במדבר י''ג ) :
<h3>{יא}</h3>

<b style="color: gray;">מצודת דוד</b> <b>והכה .</b> מי שיצוה לו הוא האויב : <b>הבית הגדול .</b> על מלכות אפרים יאמר שהיא גדולה ממלכות יהודה כי בהם יש עשרת שבטים : <b>רסיסים .</b> לבקעיות מרובות ותכופות שאין לתקנם כי אם לאחר זמן רב ר''ל שהם לא ישובו לארצם עד הגאולה העתידה : <b>והבית הקטן .</b> על מלכות יהודה יאמר שהיא קטנה ממלכות אפרים : <b>בקיעים .</b> לבקעים יחידים שאפשר לתקנם מהר ר''ל שהם ישובו לארצם בזמן בית השני :

<h3>{יב}</h3>

<b style="color: gray;">מצודת ציון</b> <b>הירוצון .</b> מל' ריצה ורדיפה : <b>לראש .</b> ללענה . מיני עשבים מרים כמו פורה ראש ולענה ( דברים כ''ט ) :
<h3>{יג}</h3>
הַשְּׂמֵחִ֖ים לְלֹ֣א דָבָ֑ר הָאֹ֣מְרִ֔ים הֲל֣וֹא בְחָזְקֵ֔נוּ לָקַ֥חְנוּ לָ֖נוּ קַרְנָֽיִם׃
<b style="color: gray;">רש"י</b> <b>ללא דבר .</b> על גדולתם ועושרם שאינו כלום שלא יתקיים : <b>קרנים .</b> חוזק וגבורה לנגח בקרנים עניים ואביונים :
<b style="color: gray;">מצודת דוד</b> <b>השמחים .</b> הם שמחים על לא דבר ר''ל שמחים על הצלחתם שלא תתקיים לזמן מרובה : <b>האומרים .</b> הם אומרים בחוזק ידינו לקחנו לנו אומץ וחוזק וכאומר אבל לא כן הוא כי הכל בהשגחה :
<b style="color: gray;">מצודת ציון</b> <b>קרנים .</b> ענין חוזק והוא שאול מקרני בעל חי שמתחזקין בהם :
<h3>{יד}</h3>
כִּ֡י הִנְנִי֩ מֵקִ֨ים עֲלֵיכֶ֜ם בֵּ֣ית יִשְׂרָאֵ֗ל נְאֻם־יְהוָ֛ה אֱלֹהֵ֥י הַצְּבָא֖וֹת גּ֑וֹי וְלָחֲצ֥וּ אֶתְכֶ֛ם מִלְּב֥וֹא חֲמָ֖ת עַד־נַ֥חַל הָעֲרָבָֽה׃
<b style="color: gray;">רש"י</b> <b>מלבא חמת .</b> שהוא במקצוע צפונית מערבית של ארץ ישראל : <b>עד נחל הערבה .</b> נחל מצרים שהוא במקצוע דרומית מערבית :
<b style="color: gray;">מצודת דוד</b> <b>כי הגני .</b> כי עתה שחטאתם לי הנני מעמיד עליכם אתם בית ישראל : <b>גוי .</b> מוסב על הנני מקים עליכם : <b>ולחצו אתכם .</b> העכו''ם ההם : <b>מלבוא חמת .</b> מן הדרך שבאים בו אל חמת שהוא בפאת הצפוני של א''י : <b>עד נחל הערבה .</b> שהוא בפאת הדרומי של א''י מן הקצה אל הקצה :
<b style="color: gray;">מצודת ציון</b> <b>ולחצו .</b> ענין דחק :
<h2>עמוס פרק-ז</h2>
<h3>{א}</h3>
כֹּ֤ה הִרְאַ֙נִי֙ אֲדֹנָ֣י יְהוִ֔ה וְהִנֵּה֙ יוֹצֵ֣ר גֹּבַ֔י בִּתְחִלַּ֖ת עֲל֣וֹת הַלָּ֑קֶשׁ וְהִ֨נֵּה־לֶ֔קֶשׁ אַחַ֖ר גִּזֵּ֥י הַמֶּֽלֶךְ׃
<b style="color: gray;">רש"י</b> <b>כה הראני .</b> פורענות שהיה רוצה להביא גובאי : <b>בתחלת עלות הלקש .</b> ל' מלקוש שהתבואה עולה בקנה שלה : <b>והנה לקש .</b> היה צומח אחרי גיזי המלך אחר שגזזו תבואת המלך עד שלא עלתה בקנה היו גוזזין אותה שחת להאכיל לבהמות וכן ת''י דאיתגזיזת שיחתא דמלכא ול' משנה היא קוצר לשחת ומאכיל לבהמה :
<b style="color: gray;">מצודת דוד</b> <b>כה הראני .</b> במראה הנבואה : <b>יוצר גובי .</b> שבורא ארבה בתחלת צמיחת העשב המאוחר : <b>והנה לקש .</b> נדמה לי כאלו אחר שנגזז ונקצר העשב למאכל בהמות המלך הצמיחה וגדלה הלקש הוא עשב המאוחר :
<b style="color: gray;">מצודת ציון</b> <b>יוצר .</b> בורא : <b>גובי .</b> ארבה כי ויעל הארבה ( שמות ט' ) ת''א גובא , וכן כגוב גובי ( נחום ג' ) : <b>הלקש .</b> כן יקרא העשב המאוחר כמו שנקרא מטר המאוחר בשם מלקוש : <b>גזי .</b> מל' גזיזה וקצירה :
<h3>{ב}</h3>
וְהָיָ֗ה אִם־כִּלָּה֙ לֶֽאֱכוֹל֙ אֶת־עֵ֣שֶׂב הָאָ֔רֶץ וָאֹמַ֗ר אֲדֹנָ֤י יְהוִה֙ סְֽלַֽח־נָ֔א מִ֥י יָק֖וּם יַֽעֲקֹ֑ב כִּ֥י קָטֹ֖ן הֽוּא׃
<b style="color: gray;">רש"י</b> <b>והיה אם כלה .</b> כאשר כלה לאכול : <b>את עשב הארץ .</b> באותה שדה : <b>ואמר אדני ה' סלח נא .</b> ואל תשחת הארץ : <b>מי יקום יעקב .</b> מאן יקום ויבעי על חטאיהון : <b>כי קטן .</b> ארי מטלטלין :
<b style="color: gray;">מצודת דוד</b> <b>והיה אם כלה לאכול .</b> כי ראה במראה הנבואה כאלו הארבה אכל כל העשב וכאשר כלה לאכול את הכל הבין אז שהמקום מייעד להביא הארבה לכלות הכל : <b>ואמר .</b> אמרתי בתפלה לה' סלח נא עון ישראל ולא תענשם בהבאת הארבה כי מי יקום את יעקב לעמוד על עמדו לכלכלו ברעב כי הוא קטן ואין בו כח ויכולת לכלכל את עצמו למצוא די ספוקו וכל העכו''ם שונאים אותו ולא יאבו לכלכלו וימות א''כ ברעב :
<h3>{ג}</h3>
נִחַ֥ם יְהוָ֖ה עַל־זֹ֑את לֹ֥א תִהְיֶ֖ה אָמַ֥ר יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>נחם ה' על זאת .</b> נתרצה בתפלתו ונעתר לי :
<b style="color: gray;">מצודת דוד</b> <b>נחם ה' .</b> והנה נתחרט ה' על הגזירה הזאת ואמר שלא תהיה עוד :
<b style="color: gray;">מצודת ציון</b> <b>נחם .</b> ענין הפוך מחשבה :
<h3>{ד}</h3>
כֹּ֤ה הִרְאַ֙נִי֙ אֲדֹנָ֣י יְהוִ֔ה וְהִנֵּ֥ה קֹרֵ֛א לָרִ֥ב בָּאֵ֖שׁ אֲדֹנָ֣י יְהוִ֑ה וַתֹּ֙אכַל֙ אֶת־תְּה֣וֹם רַבָּ֔ה וְאָכְלָ֖ה אֶת־הַחֵֽלֶק׃
<b style="color: gray;">רש"י</b> <b>הראני .</b> עוד פורענות להביא עליכם : <b>והנה קורא .</b> את פמליא שלו : <b>לריב .</b> אתכם בפורענות דליקת אש : <b>החלק .</b> חלקת השדה :
<b style="color: gray;">מצודת דוד</b> <b>קורא לריב באש וגו' .</b> ה' היה קורא למלאכי מרום שהם יריבו עם ישראל בהבאת אש ר''ל להביא עליהם חום רב שורף כאש : <b>ותאכל .</b> נדמה לי כאלו שרפה והובישה את מי התהום וכאלו שרפה את צמח קבוצת השדות :
<b style="color: gray;">מצודת ציון</b> <b>תהום .</b> עומק המים : <b>רבה .</b> גדולה בעומק : <b>החלק .</b> כן יקרא השדה וכן חלקה אחת תמטר ( לעיל ד' ) :
<h3>{ה}</h3>
וָאֹמַ֗ר אֲדֹנָ֤י יְהוִה֙ חֲדַל־נָ֔א מִ֥י יָק֖וּם יַעֲקֹ֑ב כִּ֥י קָטֹ֖ן הֽוּא׃
<b style="color: gray;">מצודת דוד</b> <b>חדל נא .</b> מנע נא את הגזירה ההיא כי מי יקום את יעקב לעמוד על עמדו לכלכלו ברעב כי הוא קטן ואין בו יכולת לכלכל עצמו :
<b style="color: gray;">מצודת ציון</b> <b>חדל .</b> ענין מניעה :
<h3>{ו}</h3>
נִחַ֥ם יְהוָ֖ה עַל־זֹ֑את גַּם־הִיא֙ לֹ֣א תִֽהְיֶ֔ה אָמַ֖ר אֲדֹנָ֥י יְהוִֽה׃
<b style="color: gray;">מצודת דוד</b> <b>גם היא .</b> גם הגזרה ההיא לא תהיה :
<h3>{ז}</h3>
כֹּ֣ה הִרְאַ֔נִי וְהִנֵּ֧ה אֲדֹנָ֛י נִצָּ֖ב עַל־חוֹמַ֣ת אֲנָ֑ךְ וּבְיָד֖וֹ אֲנָֽךְ׃
<b style="color: gray;">רש"י</b> <b>על חומת אנך .</b> על שורא דדינא וכן פתר דונש , אנך ל' ערבי חוט קו המשקולת העופרת אשר בה יביט בונה החומה ליישר הדימוס אף כאן הוא לשון קו משקולת משפט :
<b style="color: gray;">מצודת דוד</b> <b>חומת אנך .</b> על החומה שבונין אותה בקו המשקולת : <b>ובידו .</b> ביד המקום היה קו המשקולת :
<b style="color: gray;">מצודת ציון</b> <b>אנך .</b> כן יקרא קו הבנין ובו משקולת הבדיל וע''י מיישר הבנאי בנין החומה ואנך הוא בדיל בל' ערבי :
<h3>{ח}</h3>
וַיֹּ֨אמֶר יְהוָ֜ה אֵלַ֗י מָֽה־אַתָּ֤ה רֹאֶה֙ עָמ֔וֹס וָאֹמַ֖ר אֲנָ֑ךְ וַיֹּ֣אמֶר אֲדֹנָ֗י הִנְנִ֨י שָׂ֤ם אֲנָךְ֙ בְּקֶ֙רֶב֙ עַמִּ֣י יִשְׂרָאֵ֔ל לֹֽא־אוֹסִ֥יף ע֖וֹד עֲב֥וֹר לֽוֹ׃
<b style="color: gray;">רש"י</b> <b>הנני שם אנך .</b> הנני שופט אותם בקו שורת הדין : <b>עבור לו .</b> להמתין להם על פשעיהם :
<b style="color: gray;">מצודת דוד</b> <b>מה אתה רואה .</b> לא שאל לו על המראות הראשונות לפי שהיו מבוארים מעצמם ולא הוצרך לבארו אבל במראה ההיא הוצרך לבארו לכן שאלו : <b>שם אנך וגו' .</b> ר''ל אשפוט אותם בקו משקולת של שורת הדין כפי הגמול ולא אוסיף עוד לעבור על פשעם למחול אותם :
<b style="color: gray;">מצודת ציון</b> <b>עבור .</b> מל' העברה :
<h3>{ט}</h3>
וְנָשַׁ֙מּוּ֙ בָּמ֣וֹת יִשְׂחָ֔ק וּמִקְדְּשֵׁ֥י יִשְׂרָאֵ֖ל יֶחֱרָ֑בוּ וְקַמְתִּ֛י עַל־בֵּ֥ית יָרָבְעָ֖ם בֶּחָֽרֶב׃
<b style="color: gray;">רש"י</b> <b>ומקדשי ישראל .</b> הוא בית אל שנאמר כי מקדש מלך הוא : <b>על בית ירבעם .</b> בן יואש :
<b style="color: gray;">מצודת דוד</b> <b>ונשמו .</b> הבמות שהעמידו לעכו''ם יהיו שוממין כי מגודל הצרות אין מי יבוא אליהם : <b>ומקדשי .</b> הארמונות החזקות של ישראל יהיו לחרבה : <b>וקמתי .</b> אקום על בית ירבעם להפרע ממנו בחרב ועל זכריה בנו יאמר שהרגו שלום בן יבש כמ''ש ( במ''ב ) :
<b style="color: gray;">מצודת ציון</b> <b>ונשמו .</b> מל' שממון : <b>במות .</b> מלשון במה ומזבח : <b>ישחק .</b> כמו יצחק ויתכן שאמר ישחק לרמז שהבמות יהיו לשחוק וללעג : <b>ומקדשי .</b> ענין ארמון משגב וכן למטה מקדש מלך הוא וכמו והיה למקדש ( ישעיה ח' ) : <b>יחרבו .</b> מל' חורבן :
<h3>{י}</h3>
וַיִּשְׁלַ֗ח אֲמַצְיָה֙ כֹּהֵ֣ן בֵּֽית־אֵ֔ל אֶל־יָרָבְעָ֥ם מֶֽלֶךְ־יִשְׂרָאֵ֖ל לֵאמֹ֑ר קָשַׁ֨ר עָלֶ֜יךָ עָמ֗וֹס בְּקֶ֙רֶב֙ בֵּ֣ית יִשְׂרָאֵ֔ל לֹא־תוּכַ֣ל הָאָ֔רֶץ לְהָכִ֖יל אֶת־כָּל־דְּבָרָֽיו׃
<b style="color: gray;">רש"י</b> <b>כהן בית אל .</b> כומר לע''ז שלו :
<b style="color: gray;">מצודת דוד</b> <b>קשר עליך עמוס .</b> ר''ל מדבר דברים המביאים קשר למרוד בך ולהרגך כי כשישמעו שונאיו שהנביא מייעד עליו החרב יתעוררו להרגו בדבר ה' : <b>בקרב בית ישראל .</b> ר''ל בפרסום רב במעמד כולם : <b>לא תוכל הארץ .</b> עם הארץ לא תוכל לסבול את כל דבריו מה שמאיים בשממון וחורבן כאומר הואיל וגם נפש העם מרה עליו א''כ תוכל לעשות בו משפט חרוץ כי גם יד העם יהיה עמך :
<b style="color: gray;">מצודת ציון</b> <b>כהן .</b> ר''ל כהן הבעל וכומר לעכו''ם : <b>קשר .</b> ענין אגודת מרד כמו ויהי הקשר אמיץ ( ש''ב טז ) : <b>להכיל .</b> לסבול כמו נלאיתי הכיל ( ירמי' ו' ) :
<h3>{יא}</h3>
כִּי־כֹה֙ אָמַ֣ר עָמ֔וֹס בַּחֶ֖רֶב יָמ֣וּת יָרָבְעָ֑ם וְיִ֨שְׂרָאֵ֔ל גָּלֹ֥ה יִגְלֶ֖ה מֵעַ֥ל אַדְמָתֽוֹ׃
<b style="color: gray;">רש"י</b> <b>בחרב ימות ירבעם .</b> והוא לא אמר כן אלא וקמתי על בית ירבעם והוא בנו זכריה שהרגו שלום בן יבש ( מלכים ב טו ) :
<b style="color: gray;">מצודת דוד</b> <b>בחרב ימות ירבעם .</b> באמת כחש לו כי הוא אמר על בית ירבעם והוא זכריה בנו : <b>וישראל גלה יגלה .</b> גם בזה כחש כי לא אמר כן עד לבסוף :
<h3>{יב}</h3>
וַיֹּ֤אמֶר אֲמַצְיָה֙ אֶל־עָמ֔וֹס חֹזֶ֕ה לֵ֥ךְ בְּרַח־לְךָ֖ אֶל־אֶ֣רֶץ יְהוּדָ֑ה וֶאֱכָל־שָׁ֣ם לֶ֔חֶם וְשָׁ֖ם תִּנָּבֵֽא׃
<b style="color: gray;">רש"י</b> <b>חוזה .</b> אתה שמתנבא עתידות : <b>לך ברח לך אל ארץ יהודה .</b> ששם מקדשו של הקב''ה שאתה מתנבא בשמו : <b>ואכל שם לחם .</b> לשון בזיון אמר לו שם יתנו לך פתיתי לחם בשכר שתתנבא להם :
<b style="color: gray;">מצודת דוד</b> <b>חוזה .</b> ר''ל אתה הוא חוזה מה תעשה פה ברח לך אל ארץ יהודה כי שם מקדש ה' מקום הנבואה : <b>ואכל שם לחם .</b> שם יפרנסו אותך : <b>ושם תנבא .</b> ר''ל שם תקבל שכר על הנבואה שאתה מתנבא רעה על מלכות אפרים כי לאויבים המה לו :
<b style="color: gray;">מצודת ציון</b> <b>חוזה .</b> רואה ונביא :
<h3>{יג}</h3>
וּבֵֽית־אֵ֔ל לֹֽא־תוֹסִ֥יף ע֖וֹד לְהִנָּבֵ֑א כִּ֤י מִקְדַּשׁ־מֶ֙לֶךְ֙ ה֔וּא וּבֵ֥ית מַמְלָכָ֖ה הֽוּא׃
<b style="color: gray;">רש"י</b> <b>ובית אל לא תוסיף עוד להנבא .</b> שאין מזכירין כאן אלא שם ע''ז :
<b style="color: gray;">מצודת דוד</b> <b>ובית אל .</b> אבל בבית אל לא תוסיף עוד להנבא כזאת כי פה הוא ארמון מלך הארץ ההיא ופן אנשיו ימיתוך בשמעם שאתה מתנבא כזאת : <b>ובית ממלכה הוא .</b> ר''ל הממלכה מוחזקת ביד המלך מכמה דורות וידו תקיפה להכותך נפש :
<b style="color: gray;">מצודת ציון</b> <b>מקדש .</b> ארמון משגב :
<h3>{יד}</h3>
וַיַּ֤עַן עָמוֹס֙ וַיֹּ֣אמֶר אֶל־אֲמַצְיָ֔ה לֹא־נָבִ֣יא אָנֹ֔כִי וְלֹ֥א בֶן־נָבִ֖יא אָנֹ֑כִי כִּֽי־בוֹקֵ֥ר אָנֹ֖כִי וּבוֹלֵ֥ס שִׁקְמִֽים׃
<b style="color: gray;">רש"י</b> <b>לא נביא אנכי .</b> איני מנביאי שקר שלכם הנוטלים שכרם להנבא איני צריך לכל זאת ולא נהגתי כך אני שאני עשיר ובעל מקנה ונכסים כמו שת''י ארי מרי גיתי אנא ושקמין אית לי בשפלתא : <b>בוקר אנכי .</b> רועה בקר : <b>ובולס שקמים .</b> מחפש בשקמים לראו' איזה עתו לקוץ כדי להוסיף ענפים ואיזה ראוי לקורות שכן דרך שקוצצין בתולות השקמה ובולס כמו ובולש אלא שעמוס מגמגם בלשונו שכך אמרו למה נקרא שמו עמוס שהיה עמוס בלשונו וישראל קוראים אותו פסילוס כדאיתא בפסיקתא :
<b style="color: gray;">מצודת דוד</b> <b>לא נביא אנכי .</b> ר''ל אני לא הכנתי עצמי לנבואה ולא נתעסקתי בלמודה לשתחשוב שאני מקבל שכר על הנבואה ובעבור השכר הכנתי עצמי לה : <b>ולא בן נביא אנכי .</b> וכאומר ולא ראיתי ג''כ מאבי קבלת שכר נבואה והוא ענין מליצה והרחבה : <b>כי בוקר אנכי .</b> אני בעל בקר ואינני צריך לשכר ואני כל היום מערבב שקמים עם דברים אחרים למאכל הבקרים ואין אני עוסק בהכנת הנבואה ולמודה :
<b style="color: gray;">מצודת ציון</b> <b>בוקר .</b> מל' בקר : <b>ובולס .</b> ענין בלילה וערבוב ובדרז''ל שכן עני אוכל פתו מעיסה בלוסה ( שבת עו ) : <b>שקמים .</b> שם מין ממיני התאנים וכן ושקמותם בחנמל ( תהלים ע''ח ) :
<h3>{טו}</h3>
וַיִּקָּחֵ֣נִי יְהוָ֔ה מֵאַחֲרֵ֖י הַצֹּ֑אן וַיֹּ֤אמֶר אֵלַי֙ יְהוָ֔ה לֵ֥ךְ הִנָּבֵ֖א אֶל־עַמִּ֥י יִשְׂרָאֵֽל׃
<b style="color: gray;">מצודת דוד</b> <b>ויקחני ה' .</b> אבל מבלי הכנה לקחני ה' ממה שהייתי הולך אחר הצאן לרעות אותם וצוה אותי ללכת ולנבאות :
<h3>{טז}</h3>
וְעַתָּ֖ה שְׁמַ֣ע דְּבַר־יְהוָ֑ה אַתָּ֣ה אֹמֵ֗ר לֹ֤א תִנָּבֵא֙ עַל־יִשְׂרָאֵ֔ל וְלֹ֥א תַטִּ֖יף עַל־בֵּ֥ית יִשְׂחָֽק׃
<b style="color: gray;">רש"י</b> <b>ולא תטיף .</b> ל' נבואה הוא והרבה יש במקרא הטף אל דרום ( יחזקאל כ''א ) :
<b style="color: gray;">מצודת דוד</b> <b>ועתה .</b> כאומר הואיל ואין לי צורך אל שכר הנבואה עתה שמע דבר ה' עם כי אין מקום לקבל ממך שכר על הנבואה ההיא :
<b style="color: gray;">מצודת ציון</b> <b>תטיף .</b> הוא מל' נטיפה והזלה ומושאל מענין הדבור וכן הטף אל דרום ( יחזקאל כ''א ) :
<h3>{יז}</h3>
לָכֵ֞ן כֹּה־אָמַ֣ר יְהוָ֗ה אִשְׁתְּךָ֞ בָּעִ֤יר תִּזְנֶה֙ וּבָנֶ֤יךָ וּבְנֹתֶ֙יךָ֙ בַּחֶ֣רֶב יִפֹּ֔לוּ וְאַדְמָתְךָ֖ בַּחֶ֣בֶל תְּחֻלָּ֑ק וְאַתָּ֗ה עַל־אֲדָמָ֤ה טְמֵאָה֙ תָּמ֔וּת וְיִ֨שְׂרָאֵ֔ל גָּלֹ֥ה יִגְלֶ֖ה מֵעַ֥ל אַדְמָתֽוֹ׃
<b style="color: gray;">מצודת דוד</b> <b>בעיר תזנה .</b> אשר חזקתה ברצון משא''כ בשדה : <b>בחבל תחולק .</b> בקו המדה תחולק בין העכו''ם : <b>על אדמה וגו' .</b> בארץ העכו''ם : <b>וישראל וגו' .</b> ר''ל כמו שכחשת לדבר בשמי הנה באמת כן יהיה וכן יקום :
<b style="color: gray;">מצודת ציון</b> <b>בחבל .</b> קו המדה :
<h2>עמוס פרק-ח</h2>
<h3>{א}</h3>
כֹּ֥ה הִרְאַ֖נִי אֲדֹנָ֣י יְהוִ֑ה וְהִנֵּ֖ה כְּל֥וּב קָֽיִץ׃
<b style="color: gray;">רש"י</b> <b>כלוב קיץ .</b> סל מלא סופי תאנים שאינם טובות :
<b style="color: gray;">מצודת דוד</b> <b>כלוב קיץ .</b> סל מלא פירות הקיץ :
<b style="color: gray;">מצודת ציון</b> <b>כלוב .</b> הוא כעין סל וכן ככלוב מלא עוף ( ירמיה ה' ) : <b>קיץ .</b> ר''ל פירות המייבשים בימות הקיץ וכן והלחם והקיץ ( ש''ב ט''ז ) :
<h3>{ב}</h3>
וַיֹּ֗אמֶר מָֽה־אַתָּ֤ה רֹאֶה֙ עָמ֔וֹס וָאֹמַ֖ר כְּל֣וּב קָ֑יִץ וַיֹּ֨אמֶר יְהוָ֜ה אֵלַ֗י בָּ֤א הַקֵּץ֙ אֶל־עַמִּ֣י יִשְׂרָאֵ֔ל לֹא־אוֹסִ֥יף ע֖וֹד עֲב֥וֹר לֽוֹ׃
<b style="color: gray;">מצודת דוד</b> <b>בא הקץ .</b> ר''ל המראה מרמז שבא הקץ והסוף על עמי ישראל כי קיץ ראית וקץ בא שהוא לשון הנופל על לשון : <b>עבור לו .</b> לעבור ולמחול לו על פשעיו :
<h3>{ג}</h3>
וְהֵילִ֜ילוּ שִׁיר֤וֹת הֵיכָל֙ בַּיּ֣וֹם הַה֔וּא נְאֻ֖ם אֲדֹנָ֣י יְהוִ֑ה רַ֣ב הַפֶּ֔גֶר בְּכָל־מָק֖וֹם הִשְׁלִ֥יךְ הָֽס׃
<b style="color: gray;">רש"י</b> <b>שירות היכל .</b> שהם משוררים בתוך היכלי הם במשתאותיהם : <b>רב הפגר .</b> סגי פגרי קטולין : <b>בכל מקום השליך הס .</b> בכל אתר יימר רמי וסליק השליכו מכאן ( בס''א הסירו מכאן ) הפגרים האלה :
<b style="color: gray;">מצודת דוד</b> <b>והילילו .</b> יהפכו ליללה השירות והזמירות שהיו מזמרים בהיכליהם במקום השירות יילילו : <b>רב הפגר .</b> אז ירבו פגרי הרוגים ובכל מקום אשר השליך המשליך את הפגרים הרבים ההם יהיה שתיקה ודמימות קול כי הרואים לא יצעקו ולא ירימו קול למען לא ירך לב העם :
<b style="color: gray;">מצודת ציון</b> <b>והילילו .</b> מל' יללה : <b>שירות .</b> מל' שיר וזמר : <b>הפגר .</b> כן יקרא גוף המת כמו וכובד פגר ( נחום ג' ) : <b>הס .</b> ענין שתיקה כמו ואמר הס ( לעיל ו ) :
<h3>{ד}</h3>
שִׁמְעוּ־זֹ֕את הַשֹּׁאֲפִ֖ים אֶבְי֑וֹן וְלַשְׁבִּ֖ית (ענוי־)[עֲנִיֵּי־]אָֽרֶץ׃
<b style="color: gray;">רש"י</b> <b>השואפים .</b> הבולעים אביון : <b>ולשבית .</b> כמו ולהשבית ודוגמא לשמיד מעוזניה ( ישעיה כג ) ללכת לגיד ביזרעאל ( מלכים ב ט ) :
<b style="color: gray;">מצודת דוד</b> <b>השואפים אביון .</b> אתם הבולעים את האביון ועוסקים להשבית מן העולם עניי ארץ :
<b style="color: gray;">מצודת ציון</b> <b>השואפים .</b> ענין בליעה וההמשכה אל הפה כמו שאפני אנוש ( תהלים נ''ו ) : <b>ולשבית .</b> כמו ולהשבית וכן ללכת לגיד ( מ''ב ט' ) ומשפטו להגיד ולהשבית הוא ענין בטול ומניעה כמו שבת נוגש ( ישעיה י''ד ) :
<h3>{ה}</h3>
לֵאמֹ֗ר מָתַ֞י יַעֲבֹ֤ר הַחֹ֙דֶשׁ֙ וְנַשְׁבִּ֣ירָה שֶּׁ֔בֶר וְהַשַּׁבָּ֖ת וְנִפְתְּחָה־בָּ֑ר לְהַקְטִ֤ין אֵיפָה֙ וּלְהַגְדִּ֣יל שֶׁ֔קֶל וּלְעַוֵּ֖ת מֹאזְנֵ֥י מִרְמָֽה׃
<b style="color: gray;">רש"י</b> <b>מתי יעבור החדש .</b> זו היא השאיפה מצפים עת תייקר התבואה וימכרו לעניים ברבית ויטלו שדותיהם , יעבור החדש ת''י מתי תתעבר השנה ויבא חדש העיבור ול' העברה הוא כמו ( ירמיה מו ) העביר המועד ולפי שדוחין את הקרבת העומר והשנה נמשכת והישן מתייקר : <b>והשבת ונפתחה בר .</b> מתי תבוא השמיטה ותייקר התבואה ונפתח אוצרות בר שלנו : <b>להקטין איפה .</b> למכור במדה קטנה ואת כספינו נקבל בשקל גדול :
<b style="color: gray;">מצודת דוד</b> <b>לאמר .</b> שהמה אומרים מתי יעבור החודש ר''ל החודש הידוע של הקציר כי בימי הקציר מוצאין העניים לאכול מהלקט והשכח' והפאה ואינם קונים תבואה למאכלם : <b>ונשבירה שבר .</b> כשיעבור החדש לא ימצאו מה לאכול ואז נמכור להם התבואה כפי אות נפשינו : <b>והשבת .</b> מלת מתי יעבור משמשת בשתים כאלו אמר מתי יעבור השבת הוא שנת השמיטה כי אז מתפרנסי' העניים מתבואת הארץ מן המופקר ואינם קונים תבואה למאכלם : <b>ונפתחה בר .</b> כשיעבור השבת אז נפתח אוצרות התבואה : <b>להקטין איפה .</b> ר''ל לא נמכר באמת ובמשפט כ''א נעסוק לרמותם להקטין מדת האיפה ולהגדיל משקל השקל לעות עוד כף המאזנים לעשות מרמה להכריע למטה את הכף שבו אבן המשקל למען יהא נראה שהכסף חסר הרבה : <b>מאזני מרמה .</b> לפי שמאזני משקל שלהם היו עשויות באופן שיכולים להכריע את הכף למה שהם רוצים לכן קראם מאזני מרמה :
<b style="color: gray;">מצודת ציון</b> <b>ונשבירה .</b> מוכר התבואה נקרא משביר כמו המשביר לכל עם הארץ ( בראשית מ''ב ) שבר כן תקרא התבואה וכן נאמר שבר רעבון בתיכם ( שם מ''ג ) וזהו לפי ששוברת הרעב על דרך ישברו פראים צמאם ( תהלים ק''ד ) : <b>והשבת .</b> כן תקרא שנת השמטה כמ''ש ובשנה השביעית שבת וגו' ( ויקרא כ''ה ) : <b>בר .</b> תבואה כמו לשבור בר ( בראשית מ''ב ) : <b>איפה .</b> שם מדה בת ג' סאים : <b>ולעות .</b> ענין עקום ודבר שאינו ראוי והוא מל' עון :
<h3>{ו}</h3>
לִקְנ֤וֹת בַּכֶּ֙סֶף֙ דַּלִּ֔ים וְאֶבְי֖וֹן בַּעֲב֣וּר נַעֲלָ֑יִם וּמַפַּ֥ל בַּ֖ר נַשְׁבִּֽיר׃
<b style="color: gray;">רש"י</b> <b>בעבור נעלים .</b> בדיל דיחסנון : <b>ומפל בר נשביר .</b> ומהממת עיבורא נזבין , את הפסולת שנפל מן החטים בנפה נמכור ביוקר לעניים :
<b style="color: gray;">מצודת דוד</b> <b>לקנות בכסף דלים .</b> ר''ל נקח מהם כל כספם ברמיה עד שלא ישאר מאומה בידם ונקנה אותם בכסף לעשות עמהם עבודה : <b>בעבור נעלים .</b> ר''ל נקנה אותה בדבר מועט בעבור זוג מנעלים : <b>ומפל בר נשביר .</b> עוד היו אומרים לא נמכור להם תבואה נקיה אלא נערב בה הפסולת שמפילין מן התבואה בשעה שמרקדין אותה בנפה וכברה והם בע''כ יקנו עם הפסולת :
<b style="color: gray;">מצודת ציון</b> <b>נעלים .</b> מנעלי הרגל : <b>ומפל .</b> מל' הפלה :
<h3>{ז}</h3>
נִשְׁבַּ֥ע יְהוָ֖ה בִּגְא֣וֹן יַעֲקֹ֑ב אִם־אֶשְׁכַּ֥ח לָנֶ֖צַח כָּל־מַעֲשֵׂיהֶֽם׃
<b style="color: gray;">מצודת דוד</b> <b>בגאון יעקב .</b> בה''מ יקרא כן כמ''ש וצבי עדיו לגאון שמהו ( יחזקאל ז ) :
<h3>{ח}</h3>
הַ֤עַל זֹאת֙ לֹֽא־תִרְגַּ֣ז הָאָ֔רֶץ וְאָבַ֖ל כָּל־יוֹשֵׁ֣ב בָּ֑הּ וְעָלְתָ֤ה כָאֹר֙ כֻּלָּ֔הּ וְנִגְרְשָׁ֥ה (ונשקה) [וְנִשְׁקְעָ֖ה] כִּיא֥וֹר מִצְרָֽיִם׃
<b style="color: gray;">רש"י</b> <b>העל זאת לא תרגז הארץ .</b> וכי אין עון זה כדאי שתחרב עליו הארץ : <b>ועלתה כאור כלה .</b> ובמקום אחר הוא אומר ועלתה כיאור כלה ( לקמן ט ) , וי''ת שניהם בשוה ויסק עלה מלך במשריתי' דסגיאין כמו נהרא ויחפינין כולה וקשה בעיני לפרש כאור כמו כיאור ויש לפרש כמו ( איוב לז ) יפיץ ענן אורו על כפים כסה אור ( שם לו ) ועלתה כענן גשם כולה אפילה וחשיכה : <b>ונגרשה ונשקעה כיאור מצרים .</b> נילוס עולה אחת למ' שנה ומשקה את הארץ וכשהוא עולה מעלה רפש וטיט על שפתו וכשחוזר מהשקות חוזר על גדותיו ומימיו מצטללין והוא קרוי שיקוע שהטיט העוכר את מימיו נשקע ונגרש כמו ( ישעיה כז ) ויגרשו מימיו רפש וטיט אף הארץ תקיא רשעים שבתוכה ואחר כך תנוח :
<b style="color: gray;">מצודת דוד</b> <b>העל זאת .</b> וכי על מעשה כזאת לא תרעד הארץ ולא תשחת כל היושבים בה : <b>ועלתה כאור .</b> ר''ל דין הוא שתעלה המים על כלה כמו היאור המכסה כל הארץ בצאתו להשקות השדות : <b>ונגרשה , ונשקעה .</b> המים השוטפים יגרשו רפש וטיט ואח''ז יהיה נשקע הרפש והטיט לרדת אל קרקעית המים כמו יאור מצרים שבעת שמתגבר ועולה מעלה רפש וטיט ואח''ז שוקטים למטה ר''ל מהדין להשמיד הרשעים מן הארץ ואח''ז תהיה ההרוחה :
<b style="color: gray;">מצודת ציון</b> <b>תרגז .</b> ענין תנועת הרעדה : <b>ואבל .</b> ענין השחתה כמו אבלה אדמה ( יואל א' ) : <b>כאור .</b> כמו כיאור הוא הנילוס ונפלה היו''ד : <b>ונגרשה .</b> מל' גרושין ותנודה . ונשקעה . ענין נפילת הדבר בעומק כמו אז אשקיע מימיהם ( יחזקאל לב ) :
<h3>{ט}</h3>
וְהָיָ֣ה ׀ בַּיּ֣וֹם הַה֗וּא נְאֻם֙ אֲדֹנָ֣י יְהוִ֔ה וְהֵבֵאתִ֥י הַשֶּׁ֖מֶשׁ בַּֽצָּהֳרָ֑יִם וְהַחֲשַׁכְתִּ֥י לָאָ֖רֶץ בְּי֥וֹם אֽוֹר׃
<b style="color: gray;">רש"י</b> <b>והבאתי השמש בצהרים .</b> כשיהא להם שלום גדול תבא מפלה פתאום ואמרו רבותינו ( מ''ק כה ) זה יומו של יאשיהו שמת בחרב של שלום כמו שאמרו רבותינו ( תענית כב ) אין לך חרב של שלום יותר מפרעה נכה שאמר לו ליאשיה ( ד''ה ב ל''ה ) מה לי ולך לא עליך אתה היום לא עליך אני בא היום כי אם לעבור דרך ארצך אל בית מלחמתי וגו' : <b>השמש .</b> מלכות בית דוד נמשלה כשמש שנא' ( תהלים פט ) וכסאו כשמש נגדי :
<b style="color: gray;">מצודת דוד</b> <b>והבאתי .</b> אשקיע את השמש בעת צהרים בזמן תקפה ר''ל בעת ישבו בהשקט ושאנן תבוא הצרה פתאום : <b>והחשכתי .</b> אביא חושך על הארץ ביום שיש בו אור רב והוא כפל ענין במ''ש :
<b style="color: gray;">מצודת ציון</b> <b>והבאתי .</b> ענין שקיעה כמו ובא השמש וטהר ( ויקרא כ''ב ) : <b>בצהרים .</b> הוא חצי היום :
<h3>{י}</h3>
וְהָפַכְתִּ֨י חַגֵּיכֶ֜ם לְאֵ֗בֶל וְכָל־שִֽׁירֵיכֶם֙ לְקִינָ֔ה וְהַעֲלֵיתִ֤י עַל־כָּל־מָתְנַ֙יִם֙ שָׂ֔ק וְעַל־כָּל־רֹ֖אשׁ קָרְחָ֑ה וְשַׂמְתִּ֙יהָ֙ כְּאֵ֣בֶל יָחִ֔יד וְאַחֲרִיתָ֖הּ כְּי֥וֹם מָֽר׃
<b style="color: gray;">רש"י</b> <b>והפכתי חגיכם לאבל .</b> כענין שנאמר ( ד''ה ב לה ) וכל יהודה וירושלים מתאבלים על יאשיהו : <b>וכל שיריכם לקינה .</b> כענין שנאמר ( שם ) ויאמרו כל השרים והשרות בקינותיהם : <b>כאבל יחיד .</b> כאב שמתאבל על בן יחיד :
<b style="color: gray;">מצודת דוד</b> <b>והפכתי .</b> תמורת שמחת החג יהיה אבל ותמורת השירה יהיה קינה : <b>והעליתי .</b> ר''ל אביא עליהם צרה עד שכולם ילבשו שק על מתניהם ויקרחו קרחה בראשם והם ענין צער ואבלות : <b>ושמתיה .</b> אשים עדת ישראל להיות מתאבלת כדרך האבילות שעושין על בן יחיד שמת : <b>כיום מר .</b> הוא יום מיתת בן יחיד וכפל הדבר במ''ש :
<b style="color: gray;">מצודת ציון</b> <b>קרחה .</b> ענין נפילה ותלישת השער כמו קרח הוא ( שם יג ) :
<h3>{יא}</h3>
הִנֵּ֣ה ׀ יָמִ֣ים בָּאִ֗ים נְאֻם֙ אֲדֹנָ֣י יְהוִ֔ה וְהִשְׁלַחְתִּ֥י רָעָ֖ב בָּאָ֑רֶץ לֹֽא־רָעָ֤ב לַלֶּ֙חֶם֙ וְלֹֽא־צָמָ֣א לַמַּ֔יִם כִּ֣י אִם־לִשְׁמֹ֔עַ אֵ֖ת דִּבְרֵ֥י יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>כי אם לשמוע .</b> שתפסוק מכם רוח הקדש :
<b style="color: gray;">מצודת דוד</b> <b>כי אם לשמוע וגו' .</b> ר''ל יהיו תאבים אל הנבואה ולא ימצאו כי תפסק הנבואה בהיותם בגולה :
<h3>{יב}</h3>
וְנָעוּ֙ מִיָּ֣ם עַד־יָ֔ם וּמִצָּפ֖וֹן וְעַד־מִזְרָ֑ח יְשֽׁוֹטְט֛וּ לְבַקֵּ֥שׁ אֶת־דְּבַר־יְהוָ֖ה וְלֹ֥א יִמְצָֽאוּ׃
<b style="color: gray;">מצודת דוד</b> <b>ונעו .</b> ינועו ללכת מים הדרומי הוא ים סוף שהוא בדרומה של א''י עד הים המערבי ומגבול הצפוני עד גבול המזרחי ר''ל אפי' בא''י תפסק הנבואה : <b>ישוטטו .</b> ילכו אנה ואנה לבקש דברי נבואה ולא ימצאו :
<b style="color: gray;">מצודת ציון</b> <b>ונעו .</b> מלשון תנועה : <b>ישוטטו .</b> ענין ההליכה אנה ואנה כמו שוטטו בחוצות ירושלים ( ירמיה ה' ) :
<h3>{יג}</h3>
בַּיּ֨וֹם הַה֜וּא תִּ֠תְעַלַּפְנָה הַבְּתוּלֹ֧ת הַיָּפ֛וֹת וְהַבַּחוּרִ֖ים בַּצָּמָֽא׃
<b style="color: gray;">רש"י</b> <b>תתעלפנה .</b> תפרח רוחם כמו ויתעלף ( יונה ד ) וכן עצי השדה עולפו ( יחזקאל לא ) וכן בלשון משנה יתעלפה פשמיי''ר בלע''ז :
<b style="color: gray;">מצודת דוד</b> <b>ביום ההוא .</b> הוא יום הפורעניו' : <b>בצמא .</b> בעבור הצמאון יתעלפו הבחורים והבתולות כי לא ימצאו מים לשתות כאשר ילכו בגולה :
<b style="color: gray;">מצודת ציון</b> <b>תתעלפנה .</b> ענין החלשות והעדר ההרגשה כמו בניך עלפו ( ישעיה נ''א ) :
<h3>{יד}</h3>
הַנִּשְׁבָּעִים֙ בְּאַשְׁמַ֣ת שֹֽׁמְר֔וֹן וְאָמְר֗וּ חֵ֤י אֱלֹהֶ֙יךָ֙ דָּ֔ן וְחֵ֖י דֶּ֣רֶךְ בְּאֵֽר־שָׁ֑בַע וְנָפְל֖וּ וְלֹא־יָק֥וּמוּ עֽוֹד׃
<b style="color: gray;">רש"י</b> <b>חי אלהיך דן .</b> אחד מן העגלים שהעמיד ירבעם בדן :
<b style="color: gray;">מצודת דוד</b> <b>הנשבעים .</b> ר''ל כי הם הנשבעים בעגל של שומרון ובשבועת' יאמרו זה לזה הריני נשבע חי אלהיך העגל העומד בדן : <b>וחי דרך .</b> ר''ל חי הולכי דרך באר שבע אל המזבחות שעמדו שם לעכו''ם כמ''ש ובאר שבע לא תעבורו ( לעיל ה' ) : <b>ונפלו .</b> לכן יפלו ולא יקומו עוד ר''ל בזמן בית השני לא יקומו אבל בגאולה העתידה יקומו כולם יהודה ואפרים :
<b style="color: gray;">מצודת ציון</b> <b>באשמת .</b> מל' אשם וכן תקרא העכו''ם כן מאשמת עובדיה וכן תקרא פשע עש''ז כמ''ש והפשע שמה ( דניאל ח ) : <b>עוד .</b> ענין זמן רב ודוגמתו ויבך על צואריו עוד ( בראשית מ''ו ) :
<h2>עמוס פרק-ט</h2>
<h3>{א}</h3>
רָאִ֨יתִי אֶת־אֲדֹנָ֜י נִצָּ֣ב עַֽל־הַמִּזְבֵּ֗חַ וַיֹּאמֶר֩ הַ֨ךְ הַכַּפְתּ֜וֹר וְיִרְעֲשׁ֣וּ הַסִּפִּ֗ים וּבְצַ֙עַם֙ בְּרֹ֣אשׁ כֻּלָּ֔ם וְאַחֲרִיתָ֖ם בַּחֶ֣רֶב אֶהֱרֹ֑ג לֹֽא־יָנ֤וּס לָהֶם֙ נָ֔ס וְלֹֽא־יִמָּלֵ֥ט לָהֶ֖ם פָּלִֽיט׃
<b style="color: gray;">רש"י</b> <b>נצב על המזבח .</b> מסתלק מכרובא למדבחא דרך יציאת חוץ על מזבח הזהב שהוא בהיכל והוא מעשר מסעות שנסעה שכינה : <b>הך הכפתור .</b> שהוא בראש הגג : <b>וירעשו הספים .</b> איסקופות הבית התחתונות כך יהרג המלך וירעדו השרים : <b>ובצעם בראש כולם .</b> כמו ופצעם וי''ת כפתור המנורה טפי מנרתא יתקטל מלכא יאשיהו וירעשו הסיפים ויחרב ביתא ויתפגרון עזרתא : <b>ובצעם בראש כולם .</b> ומאני מקדשא בשביא יהכון ובצעם כמו ממונם יגלה בראש כולם :
<b style="color: gray;">מצודת דוד</b> <b>הך הכפתור .</b> הכה את הכפתור הוא המשקוף ומהכאה ההיא ירעשו המזוזות ר''ל באבדן המלך יחרדו השרים ועל יאשיהו יאמר כי אחר שנהרג הוא הושפלה המלוכה ולא היה ביד השרים להמליך מלך מדעתם וכאשר המליכו את יהואחז הסירו מלך מצרים מן המלוכה ונתן עונש על הארץ כמ''ש במ''ב ומאז לא המליכו מלך מדעתם : <b>ובצעם .</b> חמדת עשרם יגלה בראש כולם כי עד שלא גלו הם בימי צדקיה הוציא נ''נ אוצרות בית ה' ואוצרות בית המלך כמ''ש במ''ב : <b>ואחריתם .</b> בניהם ובנותיהם אהרוג בחרב העכו''ם : <b>לא ינוס וגו' .</b> הרוצה להניס לא יוכל : <b>ולא ימלט וגו' .</b> הוא כפל ענין במ''ש :
<b style="color: gray;">מצודת ציון</b> <b>הכפתור .</b> הוא המשקוף וקראו כפתור לפי שעושין בו ציורי כפתורים העגולים כעין תפוחים וכן בכפתורים ילינו ( צפניה ב ) : <b>וירעשו .</b> ענין תנועה חזקה : <b>הספים .</b> הם המזוזות כמו וסף השער ( יחזקאל ט ) : <b>ובצעם .</b> חמדת עשרם כמו בצע כסף ( שופטים ה ) : <b>ואחריתם .</b> כן יקראו הבנים והבנות כמו ולא לאחריתו ( דניאל י''א ) : <b>ינוס .</b> ענין בריחה : <b>להם .</b> כמו מהם : <b>ימלט .</b> ענין השמטה : <b>פליט .</b> ענין שארית כמו פליט ושריד ( איכה ב' ) :
<h3>{ב}</h3>
אִם־יַחְתְּר֣וּ בִשְׁא֔וֹל מִשָּׁ֖ם יָדִ֣י תִקָּחֵ֑ם וְאִֽם־יַעֲלוּ֙ הַשָּׁמַ֔יִם מִשָּׁ֖ם אוֹרִידֵֽם׃
<b style="color: gray;">רש"י</b> <b>אם יחתרו בשאול .</b> אם ידמון לאיטמרא כדבשאול : <b>ידי תקחם .</b> במימרי ידברינון שנאיהון : <b>ואם יעלו השמים .</b> ואם יסקון בטורין רמין עד צית שמיא מתמן במימרי יחתונון :
<b style="color: gray;">מצודת דוד</b> <b>אם יחתרו בשאול .</b> אם יעשו לעצמם מחתרת בשאול ר''ל בעומק הארץ להסתתר שמה הנה ידי תקחם משם למסרם ביד האויב והוא ענין מליצה ור''ל שלא יועילו לעצמם בשום תחבולה להנצל מיד העכו''ם : <b>משם אורידם .</b> להיות נמצאים ביד העכו''ם :
<b style="color: gray;">מצודת ציון</b> <b>יחתרו .</b> ענין חפירה בקרקע כמו חתר בחשך בתים ( איוב כ''ד ) : <b>בשאול .</b> בקבר עמוק :
<h3>{ג}</h3>
וְאִם־יֵחָֽבְאוּ֙ בְּרֹ֣אשׁ הַכַּרְמֶ֔ל מִשָּׁ֥ם אֲחַפֵּ֖שׂ וּלְקַחְתִּ֑ים וְאִם־יִסָּ֨תְר֜וּ מִנֶּ֤גֶד עֵינַי֙ בְּקַרְקַ֣ע הַיָּ֔ם מִשָּׁ֛ם אֲצַוֶּ֥ה אֶת־הַנָּחָ֖שׁ וּנְשָׁכָֽם׃
<b style="color: gray;">רש"י</b> <b>בראש הכרמל .</b> בריש מגדלי כרכיא : <b>משם אחפש .</b> מתמן אפקיד בלושין ויבלשונונון : <b>בקרקע הים .</b> בניסי ימא איי הים : <b>אצוה את הנחש .</b> י''ת עממיא דתקיפין כחויא :
<b style="color: gray;">מצודת דוד</b> <b>ואם יחבאו .</b> ואם יסתירו עצמם בראש הר הכרמל שהיה שם מקומות מסתרים : <b>משם אחפש .</b> ממקום מחבואם אחפשם למצוא אותם ולקחתים למסרם ביד העכו''ם : <b>ואם יסתרו .</b> ואם יחשבו להסתר מפני בקרקעית הים : <b>משם .</b> מקרקעית הים אצוה את הנחש הוא הלויתן שהוא ינשך אותם והכל הוא ענין מליצה :
<b style="color: gray;">מצודת ציון</b> <b>יחבאו .</b> ענין הסתרה כמו חבי כמעט רגע ( ישעיה לב ) : <b>אחפש .</b> ענין חקירה כמו ויחפש רוחי ( תהלים עז ) : <b>הנחש .</b> הוא הלויתן הדומה למראה נחש וכמ''ש על לויתן נחש בריח ( ישעיה כז ) : <b>ונשכם .</b> מל' נשיכה בשינים :
<h3>{ד}</h3>
וְאִם־יֵלְכ֤וּ בַשְּׁבִי֙ לִפְנֵ֣י אֹֽיבֵיהֶ֔ם מִשָּׁ֛ם אֲצַוֶּ֥ה אֶת־הַחֶ֖רֶב וַהֲרָגָ֑תַם וְשַׂמְתִּ֨י עֵינִ֧י עֲלֵיהֶ֛ם לְרָעָ֖ה וְלֹ֥א לְטוֹבָֽה׃
<b style="color: gray;">מצודת דוד</b> <b>ואם ילכו בשבי .</b> ר''ל אם יכנעו לפני העכו''ם וימסרו עצמם בשבי למען ירחמו עליהם : <b>משם .</b> ממקום שאחר שגורשין מימיו רפש וטיט אח''ז נשקע למטה ר''ל אחר השמדת הרשעי' תהיה ההרווחה :
<h3>{ה}</h3>
וַאדֹנָ֨י יְהוִ֜ה הַצְּבָא֗וֹת הַנּוֹגֵ֤עַ בָּאָ֙רֶץ֙ וַתָּמ֔וֹג וְאָבְל֖וּ כָּל־י֣וֹשְׁבֵי בָ֑הּ וְעָלְתָ֤ה כַיְאֹר֙ כֻּלָּ֔הּ וְשָׁקְעָ֖ה כִּיאֹ֥ר מִצְרָֽיִם׃
<b style="color: gray;">רש"י</b> <b>הנוגע בארץ ותמוג .</b> דנזיף בארעא וזיע ותמוג לשון נוע וה' הוא הנוגע בארץ ותמוג כלומר גזירה זאת מאתי יצאת :
<b style="color: gray;">מצודת ציון</b> <b>ותמוג .</b> מל' המגה והמסה : <b>ואבלו .</b> ענין השחתה כמו אבלה אדמה ( יואל א ) : <b>ושקעה .</b> ענין נפילת הדבר בעומק כמו אשקיע מימיהם ( יחזקאל לב ) :
<h3>{ו}</h3>
הַבּוֹנֶ֤ה בַשָּׁמַ֙יִם֙ (מעלותו) [מַעֲלוֹתָ֔יו] וַאֲגֻדָּת֖וֹ עַל־אֶ֣רֶץ יְסָדָ֑הּ הַקֹּרֵ֣א לְמֵֽי־הַיָּ֗ם וַֽיִּשְׁפְּכֵ֛ם עַל־פְּנֵ֥י הָאָ֖רֶץ יְהוָ֥ה שְׁמֽוֹ׃
<b style="color: gray;">רש"י</b> <b>ואגודתו .</b> קבוצת בריותיו , ד''א קבוצות הישרים היא היתה יסוד הארץ לתקנת השמים שבשבילם הכל קיים : <b>הקורא למי הים .</b> דאמר לכנשא משריין סגיאין כמי ימא :
<b style="color: gray;">מצודת דוד</b> <b>הבונה .</b> הוא הבונה בשמים מעלותיו זה למעלה מזה ועל גלגלים יאמר שהם זה למעלה מזה : <b>ואגודתו .</b> קבוצת בריותיו יסדה על הארץ : <b>הקורא למי הים .</b> להעלותם בהעבים ואח''ז שופכם ומורידם ע''פ הארץ : <b>ה' שמו .</b> המורה שהכל שלו ובידו ומוסב למעלה לומר הואיל וכן הוא לא תמצאו א''כ שום תחבולה להנצל לו מהפורענות שהוא מייעד להביא :
<b style="color: gray;">מצודת ציון</b> <b>מעלותיו .</b> מלשון עלייה הבנוי ממעל ואמר בלשון שאלה על הגלגלים : <b>ואגודתו .</b> ענין קבוצה אחת וכן יהיו לאגודה אחת ( ש''ב ב ) :
<h3>{ז}</h3>
הֲל֣וֹא כִבְנֵי֩ כֻשִׁיִּ֨ים אַתֶּ֥ם לִ֛י בְּנֵ֥י יִשְׂרָאֵ֖ל נְאֻם־יְהוָ֑ה הֲל֣וֹא אֶת־יִשְׂרָאֵ֗ל הֶעֱלֵ֙יתִי֙ מֵאֶ֣רֶץ מִצְרַ֔יִם וּפְלִשְׁתִּיִּ֥ים מִכַּפְתּ֖וֹר וַאֲרָ֥ם מִקִּֽיר׃
<b style="color: gray;">רש"י</b> <b>הלא כבני כושיים אתם לי .</b> למה אמנע מלהפרע מכם אחרי אשר אינכם שבים אלי הלא מן בני נח באתם כשאר עכו''ם ככושיים אשר נדמיתם להם כענין שנאמר ( ירמיה יג ) היהפוך כושי עורו כך אתם תוכלו להטיב : <b>הלא .</b> בטובתי תחילת בוחרי בכם היא הוצאה שהוצאתי אתכם מארץ מצרים ומה לי בכך אף פלשתים כך הוצאתי מכפתור כשבאו כפתורים על עוים כענין שנאמר ( דברים ב ) והעוים היושבים בחצרים עד עזה וגו' כבשו גם בני עזה ושאר סרני פלשתים תחתיהם והוצאתים מידם ואעפ''כ לא עשיתים לי לעם : <b>וארם מקיר .</b> וכן אני עתיד להעלות את ארם מקיר שיגלם שם סנחריב ולקץ הימים משתבטל מלכות אשור יצאו :
<b style="color: gray;">מצודת דוד</b> <b>הלא כבני כושיים וגו' .</b> כמו בני כושיים שהמה עבדים לאדוניהם עבדות עולם כן אתם לי עבדים כל ימי עולם כי הלוא העליתי אתכם מארץ מצרים מבית עבדים לכן אתם לי לעבדים : <b>ופלשתיים מכפתור .</b> ר''ל וכי העליתי את פלשתיים בני כפתור מן המקום שגלו שמה : <b>וארם מקיר .</b> ר''ל וכי אעלה את ארם מקיר שיגלו שמה ע''י מלך אשור כמ''ש ( במ''ב ) וכאומר הואיל ולא עשיתי ולא אעשה לאומה אחרת כפי מה שעשיתי עמכם לכן קנוים אתם לי לעבדים :
<b style="color: gray;">מצודת ציון</b> <b>כבני כושיים .</b> הם האנשים השחורים מבני כוש בן חם :
<h3>{ח}</h3>
הִנֵּ֞ה עֵינֵ֣י ׀ אֲדֹנָ֣י יְהוִ֗ה בַּמַּמְלָכָה֙ הַֽחַטָּאָ֔ה וְהִשְׁמַדְתִּ֣י אֹתָ֔הּ מֵעַ֖ל פְּנֵ֣י הָאֲדָמָ֑ה אֶ֗פֶס כִּ֠י לֹ֣א הַשְׁמֵ֥יד אַשְׁמִ֛יד אֶת־בֵּ֥ית יַעֲקֹ֖ב נְאֻם־יְהוָֽה׃
<b style="color: gray;">רש"י</b> <b>והשמדתי אותה .</b> הממלכה הם בית יהוא אבל בית יעקב לא אשמיד :
<b style="color: gray;">מצודת דוד</b> <b>( הנה עיני אד' .</b> ר''ל הואיל וכן הוא ואתם פשעתם בי לכן הנה עיני ה' והשגחתו בממלכה החטאה היא מלכות אפרים להשמיד אותה : <b>אפס וגו' .</b> רק לא אשמיד את בית יעקב את המון העם וכל וכל כי לבד המלוכה אשמיד ולא יהיה בהם עוד מלך :
<b style="color: gray;">מצודת ציון</b> <b>אפס .</b> עניינו כמו רק וכן ואפס את הדבר ( במדבר כב ) :
<h3>{ט}</h3>
כִּֽי־הִנֵּ֤ה אָֽנֹכִי֙ מְצַוֶּ֔ה וַהֲנִע֥וֹתִי בְכָֽל־הַגּוֹיִ֖ם אֶת־בֵּ֣ית יִשְׂרָאֵ֑ל כַּאֲשֶׁ֤ר יִנּ֙וֹעַ֙ בַּכְּבָרָ֔ה וְלֹֽא־יִפּ֥וֹל צְר֖וֹר אָֽרֶץ׃
<b style="color: gray;">רש"י</b> <b>כי הנה אנכי מצוה .</b> להגלותם ולהניעם בכל הגוים הנעה רבה ויתירה כאשר ינוע מה שכוברין בכברה לאחר שנפלו ממנו סובין הדקין ונותרו הגסין שאין יכולין לצאת ואז המנענע מנענע בכל כחו : <b>ינוע .</b> על ידי אחר ולא יתכן לקרות ינוע שהוא דבר הנע מאליו :
<b style="color: gray;">מצודת דוד</b> <b>והניעותי וגו' .</b> אניע בגולה את ישראל בין העכו''ם כמו שמנענעין תבואה בככרה להפיל העפר דרך נקביה אשר עם כל זה הצרורות הגסות לא יפלו לארץ דרך נקבי הכברה כי צר להם נקבי הכברה כן ישראל עם כי יהיו נעים ונדים בגולה לא יתמו שם הצדיקים כ''א הפושעים והמורדים :
<b style="color: gray;">מצודת ציון</b> <b>והניעותי .</b> מל' הנעה והנדה : <b>בכברה .</b> היא כעין נפה שמרקדין בו הקמח ודומה לו ועשית לו מכבר ( שמות כז ) : <b>צרור .</b> חתיכה גסה של עפר וכן אשר לא נמצא שם גם צרור ( ש''ב יז ) :
<h3>{י}</h3>
בַּחֶ֣רֶב יָמ֔וּתוּ כֹּ֖ל חַטָּאֵ֣י עַמִּ֑י הָאֹמְרִ֗ים לֹֽא־תַגִּ֧ישׁ וְתַקְדִּ֛ים בַּעֲדֵ֖ינוּ הָרָעָֽה׃
<b style="color: gray;">רש"י</b> <b>לא תגיש ותקדים בעדינו הרעה .</b> בשביל עוונינו לא תמהר הרעה לגשת ולבא :
<b style="color: gray;">מצודת דוד</b> <b>בחרב ימותו .</b> המתים בחרב העכו''ם יהיו כל החוטאים שבעמי אשר אומרים לא תבוא עלינו הרעה כי אין גמול ועונש אבל הצדיקים ישארו ולא יאכלם החרב :
<b style="color: gray;">מצודת ציון</b> <b>תגיש .</b> ענין הקרוב : <b>ותקדים .</b> ענין מהירות : <b>בעדינו .</b> ר''ל עלינו :
<h3>{יא}</h3>
בַּיּ֣וֹם הַה֔וּא אָקִ֛ים אֶת־סֻכַּ֥ת דָּוִ֖יד הַנֹּפֶ֑לֶת וְגָדַרְתִּ֣י אֶת־פִּרְצֵיהֶ֗ן וַהֲרִֽסֹתָיו֙ אָקִ֔ים וּבְנִיתִ֖יהָ כִּימֵ֥י עוֹלָֽם׃
<b style="color: gray;">רש"י</b> <b>ביום ההוא .</b> אחרי בא עליהם כל אלה יבא יום ההוא המוכן לגאולה ובו אקים סוכת דוד הנופלת , י''ת מלכותא דבית דוד :
<b style="color: gray;">מצודת דוד</b> <b>ביום ההוא .</b> כאשר יתמו הפושעים : <b>אקים וגו' .</b> ר''ל אחזיר המלוכה לבית דוד אשר נפלה והושפלה בימי הגולה : <b>וגדרתי וגו' .</b> את הפרצה שהיה בה אגדור או ר''ל אז לא ימלוך מלך אחר על אפרים להיות פרצה מה במלכות בית דוד : <b>והריסותיו .</b> מה שהיה הריסה במלכות בית דוד כל ימי הגולה שלא מלכו כלל הנה אז אקים אותה ואבננה להיות בנוי ומתוקן כימי עולם ר''ל כמו שהיתה מאז בימי דוד עצמו :
<b style="color: gray;">מצודת ציון</b> <b>וגדרתי .</b> ענין בנין : <b>פרציהן .</b> ענין שבר חומה : <b>והריסותיו .</b> מל' הריסה ונתיצה :
<h3>{יב}</h3>
לְמַ֨עַן יִֽירְשׁ֜וּ אֶת־שְׁאֵרִ֤ית אֱדוֹם֙ וְכָל־הַגּוֹיִ֔ם אֲשֶׁר־נִקְרָ֥א שְׁמִ֖י עֲלֵיהֶ֑ם נְאֻם־יְהוָ֖ה עֹ֥שֶׂה זֹּֽאת׃
<b style="color: gray;">רש"י</b> <b>למען יירשו .</b> ישראל את שארית וגו' ואת כל העכו''ם כי נקרא שמי עליהם . אשר . כמו כי :
<b style="color: gray;">מצודת דוד</b> <b>למען יירשו וגו' .</b> הוא הפוך כאשר אמר למען יירשו אשר נקרא וכו' ר''ל הממשלה תהיה באופן שיירשו העם אשר שמי נקרא עליהם להיות נקראים עם ה' הם יירשו את השארית הנשאר מכל העכו''ם להיות להם לעבדים : <b>עושה זאת .</b> העושה הטובה ההיא :
<b style="color: gray;">מצודת ציון</b> <b>למען .</b> ענינו כמו באופן כמו למען ינוח שורך ( שמות כג ) :
<h3>{יג}</h3>
הִנֵּ֨ה יָמִ֤ים בָּאִים֙ נְאֻם־יְהוָ֔ה וְנִגַּ֤שׁ חוֹרֵשׁ֙ בַּקֹּצֵ֔ר וְדֹרֵ֥ךְ עֲנָבִ֖ים בְּמֹשֵׁ֣ךְ הַזָּ֑רַע וְהִטִּ֤יפוּ הֶֽהָרִים֙ עָסִ֔יס וְכָל־הַגְּבָע֖וֹת תִּתְמוֹגַֽגְנָה׃
<b style="color: gray;">רש"י</b> <b>ונגש חורש בקוצר .</b> והשיג דיש את בציר ובציר ישיג את זרע ( ויקרא כו ) לא יספיקו לחרוש עד שיבא הקציר ולא יספיקו לבצור עד שיבא עת הזרע : <b>תתמוגגנה .</b> תרגם יונתן יתפלחן , קרקע עבודה מתמוגגת בבא גשמים : <b>עסיס .</b> יין טוב ומתוק :
<b style="color: gray;">מצודת דוד</b> <b>ונגש חורש בקוצר .</b> ר''ל הרבה תבואה תצמח ויהיו עוסקים בקצירה עד זמן החרישה והחורש והקוצר יגשו ויפגשו זה בזה וכן יפגוש דורך ענבים במי שמושך ומוליך התבואה לזריעה כי ירבו הענבים ויהיו עוסקים בדריכת הענבים עד הזריעה : <b>והטיפו .</b> הרבה ענבים יגדלו בהרים ויהיה היין נוטף מהם והוא ענין מליצה : <b>תתמוגגנה .</b> ר''ל יהיה מרעה שמן והחלב יזוב מן הצאן וכאלו נמסו הגבעות משפע הזלת החלב :
<b style="color: gray;">מצודת ציון</b> <b>ונגש .</b> ענין הקרוב והדבוק : <b>ודורך .</b> הדרך לדרוך על הענבים להוציא היין : <b>במושך .</b> מל' משיכה והולכה : <b>והטיפו .</b> ענין הזלה : <b>עסיס .</b> כן יקרא היין וכן על עסיס כי נכרת מפיכם ( יואל א ) : <b>תתמוגגנה .</b> מל' המגה והמסה :
<h3>{יד}</h3>
וְשַׁבְתִּי֮ אֶת־שְׁב֣וּת עַמִּ֣י יִשְׂרָאֵל֒ וּבָנ֞וּ עָרִ֤ים נְשַׁמּוֹת֙ וְיָשָׁ֔בוּ וְנָטְע֣וּ כְרָמִ֔ים וְשָׁת֖וּ אֶת־יֵינָ֑ם וְעָשׂ֣וּ גַנּ֔וֹת וְאָכְל֖וּ אֶת־פְּרִיהֶֽם׃
<b style="color: gray;">מצודת דוד</b> <b>ושבתי .</b> אעמיד בהשקט ומרגוע את בני השבי של עמי ישראל : <b>ערים נשמות .</b> שהיו שוממות בעת הגולה : <b>וישבו .</b> בהערים ההם ישבו ולא יגלו עוד מהן : <b>ושתו .</b> הם ישתו היין ולא יבואו עוד עכו''ם לשתותה : <b>ועשו גנות .</b> יתקנו גנות ויזרעום :
<b style="color: gray;">מצודת ציון</b> <b>ושבתי .</b> ענין השקט ומרגוע כמו בשובה ונחת תושעון ( ישעיה ל ) : <b>שבות .</b> מלשון שבי הגולה : <b>נשמות .</b> מלשון שממון :
<h3>{טו}</h3>
וּנְטַעְתִּ֖ים עַל־אַדְמָתָ֑ם וְלֹ֨א יִנָּתְשׁ֜וּ ע֗וֹד מֵעַ֤ל אַדְמָתָם֙ אֲשֶׁ֣ר נָתַ֣תִּי לָהֶ֔ם אָמַ֖ר יְהוָ֥ה אֱלֹהֶֽיךָ׃
<b style="color: gray;">מצודת דוד</b> <b>ונטעתים .</b> יהיו משורשים כאילן הנטוע : <b>ולא ינתשו .</b> ולא יהיו נעקרים עוד מעל אדמתם אשר נתתי להם מאז ר''ל לא יגלו עוד ממנה כי שמה ישבו עד עולם :
<b style="color: gray;">מצודת ציון</b> <b>ינתשו .</b> ענין עקירה כמו לנתוש ולנתוץ ( ירמיה א ) :
