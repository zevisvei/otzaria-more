<h1>סדר סעודה וברכותיה</h1>

<h2>סדר ברכת המזון</h2>
<div id="q1" style="border-bottom: 2px dotted blue; margin: 20px 0;">שיר המעלות ועל נהרות</div>
<div id="a1" style="padding:3px;background-color:#CCCCCC;display:none;"><span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בשבת ובמועד קודם ברכת המזון אומרים:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁיר</span> הַמַּעֲלוֹת. בְּשׁוּב יְהֹוָה אֶת שִׁיבַת צִיּוֹן הָיִינוּ כְּחֹלְמִים: אָז יִמָּלֵא שְׂחוֹק פִּינוּ וּלְשׁוֹנֵנוּ רִנָּה. אָז יאמְרוּ בַגּוֹיִם הִגְדִּיל יְהֹוָה לַעֲשׁוֹת עִם אֵלֶּה: הִגְדִּיל יְהֹוָה לַעֲשׁוֹת עִמָּנוּ. הָיִינוּ שְׂמֵחִים: שׁוּבָה יְהֹוָה אֶת שְׁבִיתֵנוּ כַּאֲפִיקִים בַּנֶּגֶב: הַזּרְעִים בְּדִמְעָה בְּרִנָּה יִקְצרוּ: הָלוֹךְ יֵלֵךְ וּבָכה נשֵׂא מֶשֶׁךְ הַזָּרַע. בּא יָבא בְרִנָּה. נשֵׂא אֲלֻמּתָיו:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחול קודם ברכת המזון אומרים:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">עַל נַהֲרוֹת בָּבֶל שָׁם יָשַׁבְנוּ גַּם בָּכִינוּ בְּזָכְרֵנוּ אֶת צִיּוֹן: עַל עֲרָבִים בְּתוֹכָהּ תָּלִינוּ כִּנֹּרוֹתֵינוּ: כִּי שָׁם שְׁאֵלוּנוּ שׁוֹבֵינוּ דִּבְרֵי שִׁיר וְתוֹלָלֵינוּ שִׂמְחָה שִׁירוּ לָנוּ מִשִּׁיר צִיּוֹן: אֵיךְ נָשִׁיר אֶת שִׁיר יְהֹוָה עַל אַדְמַת נֵכָר: אִם אֶשְׁכָּחֵךְ יְרוּשָׁלִָם תִּשְׁכַּח יְמִינִי: תִּדְבַּק לְשׁוֹנִי לְחִכִּי אִם לֹא אֶזְכְּרֵכִי אִם לֹא אַעֲלֶה אֶת יְרוּשָׁלִַם עַל רֹאשׁ שִׂמְחָתִי: זְכֹר יְהֹוָה לִבְנֵי אֱדוֹם אֵת יוֹם יְרוּשָׁלִָם הָאֹמְרִים עָרוּ עָרוּ עַד הַיְסוֹד בָּהּ: בַּת בָּבֶל הַשְּׁדוּדָה אַשְׁרֵי שֶׁיְשַׁלֶּם לָךְ אֶת גְּמוּלֵךְ שֶׁגָּמַלְתְּ לָנוּ: אַשְׁרֵי שֶׁיֹּאחֵז וְנִפֵּץ אֶת עֹלָלַיִךְ אֶל הַסָּלַע:</span></span></span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מים אחרונים חובה קודם ברכת המזון, ולא להפסיק בין נטילה לברכת המזון אף בדברי תורה. וישים מגבעתו על ראשו כשמברך ברכת המזון משום הכון לקראת אלהיך ישראל. כשנוטל כוס לברך ברהמ"ז נוהגין לומר וקודם שיברך יאמר:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">המזמן אומר:</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהִי</span> שֵׁם יְהֹוָה מְבֹרָךְ מֵעַתָּה וְעַד עוֹלָם: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בִּרְשׁוּת</span> מָרָנָן וְרַבּוֹתַי נְבָרֵךְ <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בעשרה או יותר</span></span> אֱלֹהֵינוּ) שֶׁאָכַלְנוּ מִשֶּׁלּוֹ: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">המסובים עונים:</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">והמזמן חוזר:</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בעשרה או יותר</span></span> אֱלֹהֵינוּ) שֶׁאָכַלְנוּ מִשֶּׁלּוֹ וּבְטוּבוֹ חָיִינוּ: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">מי שלא אכל עונה:</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בעשרה או יותר</span></span> אֱלֹהֵינוּ) וּמְבוֹרָךְ שְׁמוֹ תָּמִיד לְעוֹלָם וָעֶד: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ברכת המזון לנשואין:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">דְּוַי הָסֵר וְגַם חָרוֹן. וְאָז אִלֵּם בְּשִׁיר יָרוֹן. נְחֵנוּ בְּמַעְגְּלֵי צֶדֶק. שְׁעֵה בִּרְכַּת יְשׁוּרוּן. בְּנֵי אַהֲרֹן:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בִּרְשׁוּת מָרָנָן וְרַבָּנָן וְרַבּוֹתַי. נְבָרֵךְ אֱלֹהֵינוּ שֶׁהַשִּׂמְחָה בִּמְעוֹנוֹ. וְשֶׁאָכַלְנוּ מִשֶּׁלוֹ:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">והמזמן חוזר:</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אֱלֹהֵינוּ שֶׁהַשִּׂמְחָה בִּמְעוֹנוֹ. וְשֶׁאָכַלְנוּ מִשֶּׁלּוֹ. וּבְטוּבוֹ חָיִינוּ:</div>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלהֵינוּ מֶלֶךְ הָעוֹלָם. הַזָּן אֶת הָעוֹלָם כֻּלּוֹ. בְּטוּבוֹ בְּחֵן בְּחֶסֶד וּבְרַחֲמִים. הוּא נוֹתֵן לֶחֶם לְכָל בָּשָׂר. כִּי לְעוֹלָם חַסְדּוֹ: וּבְטוּבוֹ הַגָּדוֹל תָּמִיד לֹא חָסַר לָנוּ וְאַל יֶחְסַר לָנוּ מָזוֹן לְעוֹלָם וָעֶד. בַּעֲבוּר שְׁמוֹ הַגָּדוֹל. כִּי הוּא אֵל זָן וּמְפַרְנֵס לַכּל וּמֵטִיב לַכּל וּמֵכִין מָזוֹן לְכָל בְּרִיּוֹתָיו אֲשֶׁר בָּרָא. כָּאָמוּר פּוֹתֵחַ אֶת יָדֶךָ וּמַשְׂבִּיעַ לְכָל חַי רָצוֹן: בָּרוּךְ אַתָּה יְהֹוָה . הַזָּן אֶת הַכּל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">נוֹדֶה</span> לְּךָ יְהֹוָה אֱלהֵינוּ. עַל שֶׁהִנְחַלְתָּ לַאֲבוֹתֵינוּ אֶרֶץ חֶמְדָה טוֹבָה וּרְחָבָה. וְעַל שֶׁהוֹצֵאתָנוּ יְהֹוָה אֱלהֵינוּ מֵאֶרֶץ מִצְרַיִם. וּפְדִיתָנוּ מִבֵּית עֲבָדִים. וְעַל בְּרִיתְךָ שֶׁחָתַמְתָּ בִּבְשָׂרֵנוּ. וְעַל תּוֹרָתְךָ שֶׁלִּמַּדְתָּנוּ. וְעַל חֻקֶּיךָ שֶׁהוֹדַעְתָּנוּ. וְעַל חַיִּים חֵן וָחֶסֶד שֶׁחוֹנַנְתָּנוּ. וְעַל אֲכִילַת מָזוֹן שָׁאַתָּה זָן וּמְפַרְנֵס אוֹתָנוּ תָּמִיד. בְּכָל יוֹם וּבְכָל עֵת וּבְכָל שָׁעָה:

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על היין:</span> עַל הַגֶּפֶן וְעַל פְּרִי הַגֶּפֶן:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על פירות משבעת המינים:</span> עַל הָעֵץ וְעַל פְּרִי הָעֵץ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על מזונות:</span> עַל הַמִחְיָה וְעַל הַכַּלְכָּלָה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על מזונות ויין ביחד:</span> עַל הַמִּחְיָה וְעַל הַכַּלְכָּלָה וְעַל הַגֶּפֶן וְעַל פְּרִי הַגֶּפֶן:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְעַל תְּנוּבַת הַשָּׂדֶה</span> וְעַל אֶרֶץ חֶמְדָּה טוֹבָה וּרְחָבָה שֶׁרָצִיתָ וְהִנְחַלְתָּ לַאֲבוֹתֵינוּ לֶאֱכֹל מִפִּרְיָהּ וְלִשְׂבּוֹעַ מִטּוּבָהּ, רַחֵם (נָא) יְהֹוָה אֱלֹהֵינוּ עַל יִשְׂרָאֵל עַמֶּךָ וְעַל יְרוּשָׁלַיִם עִירֶךָ וְעַל צִיּוֹן מִשְׁכַּן כְּבוֹדֶךָ וְעַל מִזְבְּחֶךָ וְעַל הֵיכָלֶךָ, וּבְנֵה יְרוּשָׁלַיִם עִיר הַקֹּדֶשׁ בִּמְהֵרָה בְּיָמֵינוּ וְהַעֲלֵנוּ לְתוֹכָהּ וְשַׂמְּחֵנוּ בְּבִנְיָנָהּ וְנֹאכַל מִפִּרְיָּהּ וְנִשְׂבַּע מִטּוּבָהּ וּנְבָרֶכְךָ עָלֶיהָ בִּקְדֻשָּׁה וּבְטָהֳרָה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(בשבת:)</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">וּרְצֵה וְהַחֲלִיצֵנוּ בְּיוֹם הַשַּׁבָּת הַזֶּה:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(בר"ח:)</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">וְזָכְרֵנוּ לְטוֹבָה בְּיוֹם רֹאשׁ הַחֹדֶשׁ הַזֶּה:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(בר"ה:)</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">וְזָכְרֵנוּ לְטוֹבָה בְּיוֹם הַזִּכָּרוֹן הַזֶּה:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ביו"ט:)</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">וְשַׂמְחֵנוּ בְּיוֹם חַג (פלוני) הַזֶּה:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">כִּי אַתָּה</span> יְהֹוָה טוֹב וּמֵטִיב לַכֹּל וְנוֹדֶה לְךָ עַל הָאָרֶץ וְעַל:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(על היין:)</span></span> פְּרִי הַגֶּפֶן: בָּרוּךְ אַתָּה יְהֹוָה עַל הָאָרֶץ וְעַל פְּרִי הַגֶּפֶן: (בא"י וְעַל פְּרִי גַפְנָהּ):
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(על הפירות:)</span></span> הַפֵּרוֹת: בָּרוּךְ אַתָּה יְהֹוָה עַל הָאָרֶץ וְעַל הַפֵּרוֹת: (בא"י וְעַל פֵּרוֹתֶיהָ):
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(על מזונות:)</span></span> הַמִחְיָה: בָּרוּךְ אַתָּה יְהֹוָה עַל הָאָרֶץ וְעַל הַמִחְיָה וְעַל הַכַּלְכָּלָה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(על מזונות ויין ביחד:)</span></span> הַמִּחְיָה וְעַל פְּרִי הַגֶּפֶן: בָּרוּךְ אַתָּה יְהֹוָה עַל הָאָרֶץ וְעַל הַמִחְיָה וְעַל הַכַּלְכָּלָה וְעַל פְּרִי הַגֶּפֶן: (בא"י וְעַל פְּרִי גַפְנָהּ):

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם בּוֹרֵא פְּרִי הָאֲדָמָה:

<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על פירות האילן חוץ מז' מינים ופירות האדמה וירקות ועל כל המשקין ועל דבר שאין גידולו מן הארץ מברכין אחר אכילתן זה:</span>
