<h1>סדר ערוב תבשילין/חצרות/תחומין</h1>

<h2>עירובי תבשילין</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">סדר עירובי תבשילין:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יום טוב שחל להיות בערב שבת אסור לאפות או לבשל בקדרה מיוחדת לשבת אלא על ידי עירובי תבשילין שעושה בערב יום טוב. היינו שלוקח איזה תבשיל או צלי שראוי ללפת בו את הפת. ומברך:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ עַל מִצְוַת עֵרוּב:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בַּהֲדֵין עֵרוּבָא יְהֵא שָׁרֵא לָנָא לְמֵפָא וּלְבַשָּׁלָא וּלְאַטְמָנָא וּלְאַדְלָקָא שְׁרָגָא וּלְמֶעְבַּד כָּל צָרְכָנָא מִיּוֹמָא טָבָא לְשַׁבַּתָּא לָנוּ וּלְכָל יִשְׂרָאֵל הַדָּרִים בָּעִיר הַזֹּאת: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מי שאינו מבין את השפה הארמית אומר בלשׁון הקודש:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בָּזֶה הָעֵרוּב יְהֵא מֻתָּר לָנוּ לֶאֱפוֹת וּלְבַשֵּׁל וּלְהַטְּמִין וּלְהַדְלִיק נֵר וְלַעֲשׂוֹת כָּל צָרְכֵינוּ מִיּוֹם טוֹב לְשַׁבָּת לָנוּ וּלְכָל יִשְׂרָאֵל הַדָּרִים בָּעִיר הַזֹּאת:</span></span></span>
<h2>עירובי תחומין</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">סדר עירובי תחומין:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ עַל מִצְוַת עֵרוּב:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּזֶה הָעֵרוּב יְהֵא מֻתָּר לִי (אוֹ לִפְּלוֹנִי וּפְלוֹנִי וְלִבְּנֵי מְקוֹם פְּלוֹנִי) לֵילֵךְ מִמָּקוֹם זֶה אַלְפַּיִם אַמָּה לְכָל רוּחַ בְּשַׁבָּת (בְיוֹם טוֹב) (אוֹ בְיוֹם כִּפּוּר) זֶה: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">סדר עירובי חצרות:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מי שמערב בשביל כמה בעלי בתים, צריך לזכות הפת על ידי אחר: ויאמר המזכה להזוכה בלשון זה:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכֵה בְּעֵרוּב זֶה</span> לַאַחֵינוּ בְּנֵי יִשְׂרָאֵל הַדָּרִים פֹּה עִמָּנוּ. וְשֶׁיָּבוֹאוּ לָדוּר בָּעִיר הַזֹּאת <span style="font-size:95%; color:RGB(0,0,255); text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחצר הזאת, במבוי הזאת</span></span> שֶׁיִּסְמְכוּ עָלָיו לְטַלְטֵל עַל יָדוֹ מֵרְשׁוּת לִרְשׁוּת בַּשַּׁבָּת וְיוֹם טוֹב הַבָּאִים: וְלֹא יְזַכֶּה עַל יְדֵי בְּנוֹ הַקָּטָן. וְלֹא עַל יְדֵי בְּנוֹ הַגָּדוֹל הַסָּמוּךְ עַל שֻׁלְחָנוֹ. וְלֹא עַל יְדֵי אִשְׁתּוֹ. וּמִי שֶׁזָּכָה נוֹטֵל בְּיָדוֹ וְיַגְבִּיהַּ טֶפַח: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">המערב יברך ברכה זו:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ עַל מִצְוַת עֵרוּב: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בַּהֲדֵין</span> עֵרוּבָא יְהֵא שָׁרֵא לָנָא לְאַפוּקֵי וּלְעִיוּלֵי מִבַּיִת לְבַּיִת. וּמִחָצֵר לְחָצֵר. מִגַג לְגַג. מִבָּתִּים לְחָצֵר. מִחָצֵר לְבָּתִּים כָּל מֵאי דְצָרִיךְ לָן לִשְׁתֵּיהֶן בְּכָל שַׁבְּתוֹת הַשָּׁנָה וְיָמִים טוֹבִים לָנוּ בִּנֵי יִשְׂרָאֵל וּלְכָל הַדָּרִים בָּעִיר הַזֹּאת: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">אם אינו מבין את השפה הארמית יאמר בלשון הקודש:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּזֶה</span> הָעֵרוּב יְהֵא מֻתָּר לָנוּ לְהוֹצִיא וּלְהַכְנִיס מִן הַמְּבוֹאוֹת לַחֲצֵרוֹת וּמִן הַחֲצֵרוֹת לַמְּבוֹאוֹת וּמִשְׁתֵּיהֶן לְבָתִּים וּמִבָּתִּים לִשְּׁתֵּיהֶן בְּכָל שַׁבְּתוֹת הַשָּׁנָה וְיָמִים טוֹבִים לָנוּ וּלְכָל יִשְׂרָאֵל הַיּוֹשְׁבִים בָּעִיר הַזֹּאת:</span>
