<h1>סדר ליל שבת</h1>

<h2>ברכת הבנים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מנהג ישראל לברך את הילדים בלילי שבת כשבאים מבית הכנסת:</span>
<span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לזכר:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְשִׂמְךָ אֱלֹהִים כְּאֶפְרַיִם וְכִמְנַשֶּׁה:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְבָרֶכְךָ יְהֹוָה וְיִשְׁמְרֶךָ: יָאֵר יְהֹוָה פָּנָיו אֵלֶיךָ וִיחֻנֶּךָּ: יִשָּׂא יְהֹוָה פָּנָיו אֵלֶיךָ וְיָשֵׂם לְךָ שָׁלוֹם:</span></span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לנקבה:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְשִׂמְךְ אֱלֹהִים כְּשָׂרָה רִבְקָה רָחֵל וְלֵאָה:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְבָרֶכְךָ יְהֹוָה וְיִשְׁמְרֶךָ: יָאֵר יְהֹוָה פָּנָיו אֵלֶיךָ וִיחֻנֶּךָּ: יִשָּׂא יְהֹוָה פָּנָיו אֵלֶיךָ וְיָשֵׂם לְךָ שָׁלוֹם:</span></span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשיבא מבית הכנסת לביתו יאמר זה:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">שַׁבָּת שָלוֹם וּמְבֹרָךְ</span>
<h2>שלום עליכם</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שָׁלוֹם עֲלֵיכֶם</span> מַלְאֲכֵי הַשָּׁרֵת, מַלְאֲכֵי עֶלְיוֹן, מִמֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בּוֹאֲכֶם לְשָלוֹם</span> מַלְאֲכֵי הַשָּׁלוֹם, מַלְאֲכֵי עֶלְיוֹן, מִמֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרְכוּנוּ לְשָלוֹם</span> מַלְאֲכֵי הַשָּׁלוֹם, מַלְאֲכֵי עֶלְיוֹן, מִמֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">צֵאתְכֶם לְשָלוֹם</span> מַלְאֲכֵי הַשָּׁלוֹם, מַלְאֲכֵי עֶלְיוֹן, מִמֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כִּי</span> מַלְאָכָיו יצַוֶּה לָּךְ, לִשְׁמָרְךָ בְּכָל דְּרָכֶיךָ: יְהֹוָה יִשְׁמֹר צֵאתְךָ וּבוֹאֶךָ, מֵעַתָּה וְעַד עוֹלָם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רִבּוֹן</span> כָּל הָעוֹלָמִים אֲדוֹן כָּל הַנְּשָׁמוֹת אֲדוֹן הַשָּׁלוֹם. מֶלֶךְ אַבִּיר. מֶלֶךְ בָּרוּךְ. מֶלֶךְ גָּדוֹל. מֶלֶךְ דּוֹבֵר שָׁלוֹם. מֶלֶךְ הָדוּר. מֶלֶךְ וָתִיק. מֶלֶךְ זָךְ. מֶלֶךְ חֵי הָעוֹלָמִים. מֶלֶךְ טוֹב וּמֵטִיב. מֶלֶךְ יָחִיד וּמְיוּחָד. מֶלֶךְ כַּבִּיר. מֶלֶךְ לוֹבֵשׁ רַחֲמִים. מֶלֶךְ מַלְכֵי הַמְּלָכִים. מֶלֶךְ נִשְׂגָּב. מֶלֶךְ סוֹמֵךְ נוֹפְלִים. מֶלֶךְ עֹשֶׂה מַעֲשֵׂה בְרֵאשִׁית. מֶלֶךְ פּוֹדֶה וּמַצִיל. מֶלֶךְ צַח וְאָדוֹם. מֶלֶךְ קָדוֹשׁ. מֶלֶךְ רָם וְנִשָּׂא. מֶלֶךְ שׁוֹמֵעַ תְּפִלָּה. מֶלֶךְ תָּמִים דַּרְכּוֹ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מוֹדֶה</span> אֲנִי לְפָנֶיךָ יְהֹוָה אֱלֹהַי וֵאלֹהֵי אֲבוֹתַי. עַל כָּל הַחֶסֶד אֲשֶׁר עָשִׂיתָ עִמָּדִי וַאֲשֶׁר אַתָּה עָתִיד לַעֲשׂוֹת עִמִּי וְעִם כָּל בְּנֵי בֵיתִי וְעִם כָּל בְּרִיּוֹתֶיךָ בְּנֵי בְרִיתִי. וּבְרוּכִים הֵם מַלְאָכֶיךָ הַקְּדוֹשִׁים וְהַטְּהוֹרִים שֶׁעֹשִׂים רְצוֹנֶךָ. אֲדוֹן הַשָּׁלוֹם מֶלֶךְ שֶׁהַשָּׁלוֹם שֶׁלּוֹ בָּרְכֵנִי בַשָּׁלוֹם. וְתִפְקוֹד אוֹתִי וְאֶת כָּל בְּנֵי בֵיתִי וְכָל עַמְּךָ בֵּית יִשְׂרָאֵל לְחַיִּים טוֹבִים וּלְשָׁלוֹם: מֶלֶךְ עֶלְיוֹן עַל כָּל צְבָא מָרוֹם. יוֹצְרֵנוּ יוֹצֵר בְּרֵאשִׁית. אֲחַלֶּה פָנֶיךָ הַמְּאִירִים שֶׁתְּזַכֶּה אוֹתִי וְאֶת כָּל בְּנֵי בֵיתִי לִמְצֹא חֵן וְשֵׂכֶל טוֹב בְּעֵינֶיךָ וּבְעֵינֵי כָל בְּנֵי אָדָם וְחַוָה וּבְעֵינֵי כָל רוֹאֵינוּ לַעֲבוֹדָתֶךָ. וְזַכֵּנוּ לְקַבֵּל שַׁבָּתוֹת מִתּוֹךְ רוֹב שִׂמְחָה וּמִתּוֹךְ עֹשֶׁר וְכָבוֹד וּמִתּוֹךְ מִעוּט עֲוֹנוֹת. וְהָסֵר מִמֶּנִּי וּמִכָּל בְּנֵי בֵיתִי וּמִכָּל עַמְּךָ בֵּית יִשְׂרָאֵל כָּל מִינֵי חוֹלִי וְכָל מִינֵי מַדְוֶה וְכָל מִינֵי דַלּוּת וַעֲנִיּוּת וְאֶבְיוֹנוּת. וְתֵן בָּנוּ יֵצֶר טוֹב לְעָבְדְּךָ בֶּאֱמֶת וּבְיִרְאָה וּבְאַהֲבָה. וְנִהְיֶה מְכֻבָּדִים בְּעֵינֶיךָ וּבְעֵינֵי כָל רוֹאֵינוּ. כִּי אַתָּה הוּא מֶלֶךְ הַכָּבוֹד כִּי לְךָ נָאֶה כִּי לְךָ יָאֶה: אָנָּא מֶלֶךְ מַלְכֵי הַמְּלָכִים צַוֵּה לְמַלְאָכֶיךָ מַלְאֲכֵי הַשָּׁרֵת מְשָׁרְתֵי עֶלְיוֹן שֶׁיִּפְקְדוּנִי בְּרַחֲמִים וִיבָרְכוּנִי בְּבוֹאָם לְבֵיתִי בְּיוֹם קָדְשֵׁנוּ. כִּי הִדְלַקְתִּי נֵרוֹתַי וְהִצַּעְתִּי מִטָּתִי וְהֶחֱלַפְתִּי שִׂמְלוֹתַי לִכְבוֹד יוֹם הַשַּׁבָּת. וּבָאתִי לְבֵיתְךָ לְהַפִּיל תְּחִנָּתִי לְפָנֶיךָ שֶׁתַּעֲבִיר אַנְחָתִי. וָאָעִיד אֲשֶׁר בָּרָאתָ בְּשִׁשָּׁה יָמִים כָּל הַיְּצוּר וָאֶשְׁנֶה. וָאֲשַׁלֵּשׁ עוֹד לְהָעִיד עַל כּוֹסִי בְּתוֹךְ שִׂמְחָתִי. כַּאֲשֶׁר צִוִּיתַנִי לְזָכְרוֹ וּלְהִתְעַנֵּג בְּיֶתֶר נִשְׁמָתִי אֲשֶׁר נָתַתָּ בִּי: בּוֹ אֶשְׁבּוֹת כַּאֲשֶׁר צִוִּיתַנִי לְשָׁרְתֶךָ וְכֵן אַגִּיד גְּדֻלָּתְךָ בְּרִנָּה. וְשִׁוִּיתִי יְהֹוָה לִקְרָאתִי שֶׁתְּרַחֲמֵנִי עוֹד בְּגָלוּתִי לְגָאֳלֵנִי. וּלְעוֹרֵר לִבִּי לְאַהֲבָתֶךָ. וְאָז אֶשְׁמוֹר פִּקּוּדֶיךָ וְחֻקֶּיךָ בְּלִי עֶצֶב וְאֶתְפַּלֵּל כַּדָּת כָּרָאוּי וּכְנָכוֹן: מַלְאֲכֵי הַשָּׁלוֹם בּוֹאֲכֶם לְשָׁלוֹם. בָּרְכוּנִי לְשָׁלוֹם וְאִמְרוּ בָּרוּךְ לְשֻׁלְחָנִי הֶעָרוּךְ. וְצֵאתְכֶם לְשָׁלוֹם מֵעַתָּה וְעַד עוֹלָם אָמֵן סֶלָה:
<h2>אשת חיל</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֵשֶׁת חַיִל</span> מִי יִמְצָא וְרָחֹק מִפְּנִינִים מִכְרָהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּ</span>טַח בָּהּ לֵב בַּעְלָהּ וְשָׁלָל לֹא יֶחְסָר: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">גְּ</span>מָלַתְהוּ טוֹב וְלֹא רָע כָּל יְמֵי חַיֶּיה: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דָּ</span>רְשָׁה צֶמֶר וּפִשְׁתִּים וַתַּעַשׂ בְּחֵפֶץ כַּפֶּיהָ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הָ</span>יְתָה כָּאֳנִיּוֹת סוֹחֵר מִמֶּרְחָק תָּבִיא לַחְמָהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַ</span>תָּקָם בְּעוֹד לַיְלָה וַתִּתֵּן טֶרֶף לְבֵיתָהּ וְחֹק לְנַעֲרֹתֶיהָ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">זָ</span>מְמָה שָׂדֶה וַתִּקָּחֵהוּ מִפְּרִי כַפֶּיהָ נָטְעָה כָּרֶם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">חָ</span>גְרָה בְעוֹז מָתְנֶיהָ וַתְּאַמֵּץ זְרֹעוֹתֶיהָ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">טָ</span>עֲמָה כִּי טוֹב סַחְרָהּ לֹא יִכְבֶּה בַלַּיְלָה נֵרָהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יָ</span>דֶיהָ שִׁלְּחָה בַכִּישׁוֹר וְכַפֶּיהָ תָּמְכוּ פָלֶךְ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כַּ</span>פָּהּ פָּרְשָׂה לֶעָנִי וְיָדֶיהָ שִׁלְּחָה לָאֶבְיוֹן: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לֹ</span>א תִירָא לְבֵיתָהּ מִשָּׁלֶג כִּי כָל בֵּיתָהּ לָבֻשׁ שָׁנִים: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מַ</span>רְבַדִּים עָשְׂתָה לָּהּ שֵׁשׁ וְאַרְגָּמָן לְבוּשָׁהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נ</span>וֹדָע בַּשְּׁעָרִים בַּעְלָהּ בְּשִׁבְתּוֹ עִם זִקְנֵי אָרֶץ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">סָ</span>דִין עָשְׂתָה וַתִּמְכֹּר וַחֲגוֹר נָתְנָה לַכְּנַעֲנִי: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עֹ</span>ז וְהָדָר לְבוּשָׁהּ וַתִּשְׂחַק לְיוֹם אַחֲרוֹן: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">פִּ</span>יהָ פָּתְחָה בְחָכְמָה וְתוֹרַת חֶסֶד עַל לְשׁוֹנָהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">צ</span>וֹפִיָּה הֲלִיכוֹת בֵּיתָהּ וְלֶחֶם עַצְלוּת לֹא תֹאכֵל: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">קָ</span>מוּ בָנֶיהָ וַיְאַשְּׁרוּהָ בַּעְלָהּ וַיְהַלְלָהּ: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רַ</span>בּוֹת בָּנוֹת עָשׂוּ חָיִל וְאַתְּ עָלִית עַל כֻּלָּנָה: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שֶׁ</span>קֶר הַחֵן וְהֶבֶל הַיֹּפִי אִשָּׁה יִרְאַת יְהֹוָה הִיא תִתְהַלָּל: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">תְּ</span>נוּ לָהּ מִפְּרִי יָדֶיהָ וִיהַלְלוּהָ בַשְּׁעָרִים מַעֲשֶׂיהָ:
<h2>אתקינו סעודתא</h2>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אַתְקִינוּ סְעֻדָּתָא</span> דִּמְהֵימְנוּתָא שְׁלֵימָתָא, חֶדְוָתָא דְמַלְכָּא קַדִּישָׁא: אַתְקִינוּ סְעֻדָּתָא דְמַלְכָּא, דָּא הִיא סְעֻדָּתָא דַּחֲקַל תַּפּוּחִין קַדִּישִׁין (וּזְעֵיר אַנְפִּין ועַתִּיקָא קַדִּישָׁא אַתְיָן לסַעֲדָא בַּהֲדֵיהּ):
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כל אדם יאמר זה המאמר מהזוהר המסוגל לנשמה המדבר מכבוד סעודה זו:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זָכוֹר</span> אֶת יוֹם הַשַּׁבָּת לְקַדְּשׁוֹ. רַבִּי יִצְחָק פָּתַח וְאָמַר כְּתִיב וַיְבָרֶךְ אֱלֹהִים אֶת יוֹם הַשְּׁבִיעִי וּכְתִיב בַּמָּן שֵׁשֶׁת יָמִים תִּלְקְטוּהוּ וּבַיּוֹם הַשְּׁבִיעִי שַׁבָּת לֹא יִהְיֶה בּוֹ. וְכֵיוָן דְּלָא מִשְׁתְּכַח בֵּהּ מְזוֹנֵי מַה בִּרְכָתָא אִשְׁתְּכַח בֵּהּ אֶלָּא הָכִי תָאנָא כָּל בִּרְכָאָן דִּלְעֵלָּא וְתַתָּא בְּיוֹמָא שְׁבִיעָאָה תַּלְיָן וְתָאנָא אַמַּאי לֹא אִשְׁתְּכַח מַנָּא בְּיוֹמָא שְׁבִיעָאָה מִשּׁוּם דְּהַהוּא יוֹמָא מִתְבָּרְכָאָן מִנֵּהּ כָּל שִׁתָּא יוֹמִין עִלָּאִין וְכָל חַד וְחַד יָהִיב מְזוֹנָא לְתַתָּא כָּל חַד בְּיוֹמוֹי מֵהַהִיא בְּרָכָה דְּמִתְבָּרְכָאָן בְּיוֹמָא שְׁבִיעָאָה. בְּגִין כַּךְ מַאן דְּאִיהוּ בְּדַרְגָּא דִמְהֵימְנוּתָא בָּעֵי לְסַדְרָא פְּתוֹרָא. וּלְאַתְקָנָא סְעוּדְתָא בְּלֵילְיָא דְשַׁבַּתָּא בְּגִין דְּיִתְבָּרֵךְ פְּתוֹרֵהּ כָּל אִינוּן שִׁתָּא יוֹמִין דְהָא בְּהַהוּא זִמְנָא אִזְדַּמֵּן בְּרָכָה לְאִתְבָּרְכָא כָּל שִׁתָּא יוֹמִין דְּשַׁבַּתָּא וּבִרְכְּתָא לֹא אִשְׁתְּכַח בִּפְתוֹרָא רֵיקַנְיָא וְעַל כַּךְ בָּעֵי לְסַדְּרָא פְּתוֹרֵהּ בְּלֵילְיָא דְשַׁבַּתָּא בְּנַהֲמָא וּבִמְזוֹנֵי. רַבִּי יִצְחָק אָמַר אֲפִילוּ בְּיוֹמָא דְּשַׁבַּתָּא נַמֵּי. רַבִּי יְהוּדָה אָמַר בָּעֵי לְאִתְעַנָּגָא בְּהַאי יוֹמָא וּלְמֵיכַל תְּלַת סְעוּדְתֵי בְּשַׁבַּתָּא בְּגִין דְּיִשְׁתְּכַח שָׂבְעָה וְעִנּוּגָא בְּהַאי יוֹמָא בְּעַלְמָא: רַבִּי אַבָּא אָמַר לְאִזְדַּוְגָא בְּאִינוּן יוֹמִין דִּלְעֵילָא דְּמִתְבָּרְכָאָן מֵהַאי יוֹמָא וְהַאי יוֹמָא מַלְיָא רֵישֵׁיהּ דִּזְעֵר אַנְפִּין מִטַּלָּא דְנָחִית מֵעַתִּיקָא קַדִּישָׁא סְתִימָא דְכלָא וְאַטֵּיל לְחַקְלָא דְתַפּוּחִין קַדִּישִׁין תְּלַת זִמְנֵי מִכַּד עַיֵּל שַׁבְּתָא בְּגִין דְּיִתְבָּרְכוּן כֻּלְּהוֹן כַּחֲדָא. וְעַל דָּא בָּעֵי בַּר נַשׁ לְאִתְעַנָּגָא תְּלַת זִמְנִין אִלֵּין דְּהָא בְהָא תַּלְיָא מְהֵימְנוּתָא דִלְעֵלָּא בְּעַתִּיקָא קַדִּישָׁא וּבִזְעֵיר אַנְפִּין וּבְחַקְלָא דְתַפּוּחִין וּבָעֵי בַּר נַשׁ לְאִתְעַנָּגָא בְּהוּ וּלְמֶחֱדֵי בְּהוּ. וּמַאן דְּגָרַע סְעוּדְתָא מִנַּיְהוּ אִתְחֲזֵי פְּגִימוּתָא לְעֵלָּא וְעוֹנְשֵׁהּ דְּהַהוּא בַּר נַשׁ סַגִּי. בְּגִינֵי כַּךְ בָּעֵי לְסַדְּרָא פְּתוֹרֵהּ תְּלַת זִמְנֵי מִכַּד עַיֵּל שַׁבְּתָא וְלֹא יִשְׁתְּכַח פְּתוֹרֵהּ רֵיקַנְיָא וְתִשְׁרֵי בִּרְכָתָא עֲלֵהּ כָּל שְׁאָר יוֹמָא דְּשַׁבְּתָא וּבְהַאי מִלָּה אַחֲזֵי וְתָלֵי מְהֵימְנוּתָא לְעֵלָּא: רַבִּי שִׁמְעוֹן אָמַר הַאי מַאן דְּאַשְׁלִים תְּלַת סְעוּדְתֵי בְּשַׁבְּתָא קָלָא נָפִיק וּמַכְרְזָא עֲלֵהּ אָז תִּתְעַנַּג עַל יְהֹוָה. דָּא סְעוּדְתָא חֲדָא לָקֳבֵל עַתִּיקָא קַדִּישָׁא דְּכָל קַדִּישִׁין. וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ. דָּא סְעוּדְתָא תִּנְיָנָא לָקָבֵל חַקְלָא דְתַפּוּחִין קַדִּישִׁין. וְהַאֲכַלְתִּיךָ נַחֲלַת יַעֲקֹב אָבִיךָ. דָּא הִיא שְׁלִימוּ דְאִשְׁתַּלִּים בִּזְעֵיר אַנְפִּין לְקִבְלֵהּ בָּעֵי לְאַשְׁלָמָא סְעוּדְתֵהּ וּבָעֵי לְאִתְעַנָּגָא בְּכֻלְהוּ סְעוּדְתֵי וּלְמֶחֱדֵי בְּכָל חַד וְחַד מִנַּיְהוּ מִשׁוּם דְּאִיהוּ מְהֵימְנוּתָא בְּכֻלְּהוּ שְׁלֵימָתָא. וּבְגִין כַּךְ שַׁבְּתָא אִתְיַקַּר מִכָּל שְׁאָר זִמְנִין וְחַגִּין מִשׁוּם דְּכלָא בֵּהּ אִשְׁתְּכַח וְלֹא אִשְׁתְּכַח הָכִי בְּכֻלְהוּ זִמְנִין וְחַגִּין. אָמַר רַבִּי חִיָּא בְּגִין כַּךְ מִשּׁוּם דְּאִשְׁתְּכַח כֹּלָא בֵּהּ אִדְכַּר תְּלַת זִמְנִין דִּכְתִיב וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי וַיִּשְׁבּוֹת בַּיּוֹם הַשְּׁבִיעִי וַיְבָרֶךְ אֱלֹהִים אֶת יוֹם הַשְּׁבִיעִי. רַבִּי אַבָּא (נ"א: רַב הַמְנוּנָא סָבָא) כַּד הֲוָה יָתִיב בִּסְעוּדָתָא דְשַׁבַּתָּא הֲוֵי חָדֵי בְּכָל חַד וְחַד וַהֲוָה אָמַר דָּא הִיא סְעוּדָתָא קַדִּישָׁא דְעַתִּיקָא קַדִּישָׁא סְתִימָא דְכלָא. בִּסְעוּדָתָא אַחֲרָא הֲוָה אָמַר דָּא הִיא סְעוּדָתָא דְקוּדְשָׁא בְּרִיךְ הוּא וְכֵן בְּכֻלְּהוּ סְעוּדְתֵי וַהֲוָה חָדֵי בְּכָל חַד וְחַד. כַּד הֲוָה אַשְׁלִים סְעוּדְתֵי אָמַר אַשְׁלִימוּ סְעוּדְתֵי דִמְהֵימְנוּתָא. רַבִּי שִׁמְעוֹן הֲוָה אָמַר הָכִי. אַתְקִינוּ סְעוּדָתָא דִמְהֵימְנוּתָא. אַתְקִינוּ סְעוּדָתָא דְמַלְכָּא. כַּד הֲוָה יָתִיב בֵּהּ. הֲווּ מַכְרְזֵי עֲלֵהּ אָז תִּתְעַנַּג עַל יְהֹוָה וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ וְהַאֲכַלְתִּיךָ נַחֲלַת יַעֲקֹב אָבִיךָ. אָמַר רַבִּי אֶלְעָזָר לַאֲבוּהִי. אִלֵּין סְעוּדְתֵי הֵיךְ מִתְתַּקְנָן. אָמַר לֵהּ לֵילְיָא דְשַׁבְּתָא כְּתִיב בֵּהּ וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ. בֵּהּ בְּלֵילְיָא מִתְבָּרְכָא מַטְרוּנִיתָא וְכֻלְּהוּ חֲקַל תַּפּוּחִין וּמִתְבָּרְכָא פְּתוֹרֵהּ דְּבַר נַשׁ וְנִשְׁמָתָא אִתּוֹסְפַת וְהַהִיא לֵילְיָא חֶדְוָה דְמַטְרוּנִיתָא הֲוִי וּבָעֵי בַּר נַשׁ לְמֶחֱדֵי בְּחֶדְוָתָא לְמֵיכַל סְעוּדָתָא דְמַטְרוּנִיתָא:
<br/>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">פיוט משובח מהאר"י ז"ל ושמא דמריה עליה אני יצחק לוריא בן שלמה (והשי"ן כפול).</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַתְקִינוּ</span> סְעוּדָתָא דִמְהֵימְנוּתָא שְׁלֵמָתָא חֶדְוָתָא דְּמַלְכָּא קַדִּישָׁא: אַתְקִינוּ סְעוּדָתָא דְמַלְכָּא. דָּא הִיא סְעוּדָתָא דַּחֲקַל תַּפּוּחִין קַדִּישִׁין וּזְעֵר אַנְפִּין וְעַתִּיקָא קַדִּישָׁא. אַתְיָן לְסַעֲדָא בַּהֲדָהּ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֲ</span>זַמֵּר בִּשְׁבָחִין. לְמֵיעַל גּוֹ פִתְחִין. דְּבַחֲקַל תַּפּוּחִין. דְּאִנּוּן קַדִּישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">נְ</span>זַמִּין לָהּ הַשְׁתָּא. בִּפְתוֹרָא חַדְתָּא. וּבִמְנַרְתָּא טַבְתָּא. דְּנַהֲרָא עַל רֵישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְ</span>מִינָא וּשְׂמָאלָא. וּבֵנַיְהוּ כַלָּה. בְּקִשּׁוּטִין אָזְלָא. וּמָאנִין וּלְבוּשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְ</span>חַבֵּק לָהּ בַּעְלָהּ. וּבִיסוֹדָא דִילָהּ. דְּעָבִיד נַיְחָא לָהּ. יְהֵא כַּתִּישׁ כַּתִּישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">צְ</span>וָחִין אַף עַקְתִין. בְּטֵלִין וּשְׁבִיתִין. בְּרַם אַנְפִּין חַדְתִּין. וְרוּחִין עִם נַפְשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">חֲ</span>דוּ סַגִּי יֵיתֵי. וְעַל חֲדָא תַּרְתֵּי. נְהוֹרָא לָהּ יִמְטֵי. וּבִרְכָאָן דִּנְפִישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">קְ</span>רִיבוּ שׁוּשְׁבִינִין. עֲבִידוּ תִּקּוּנִין. לְאַפָּשָׁא זִינִין. וְנוּנִין עִם רַחֲשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְ</span>מֶעְבַד נִשְׁמָתִין. וְרוּחִין חַדְתִּין. בְּתַרְתֵּין וּתְלָתִין. וּבִתְלָתָא שִׁבְשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְ</span>עִטּוּרִין שַׁבְעִין לָהּ. וּמַלְכָּא דִּלְעֵלָּא. דְּיִתְעַטֵּר כּלָא. בְּקַדִּישׁ קַדִּישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רְ</span>שִׁימִין וּסְתִימִין. בְּגוֹ כָּל עָלְמִין. בְּרַם עַתִּיק יוֹמִין. הֲלָא בַּטִּישׁ בַּטִּישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְ</span>הֵא רַעֲוָא קַמֵּיהּ. דְּיִשְׁרֵי עַל עַמֵיהּ. דְּיִתְעַנַּג לִשְׁמֵיהּ. בִּמְתִיקִין וְדוּבְשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֲ</span>סַדֵּר לִדְרוֹמָא. מְנַרְתָּא דִסְתִימָא. וְשֻׁלְחָן עִם נַהֲמָא. בִּצְפוֹנָא אַרְשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בְּ</span>חַמְרָא גּוֹ כַסָּא. וּמְדָאנֵי אַסָּא. לְאָרוּס וַאֲרוּסָה. לְהִתַּקְפָא חַלָּשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">נְ</span>עַטֵּר לְהוֹן כִּתְרִין. בְּמִלִּין יַקִּירִין. בְּשַׁבְעִין עִטּוּרִין. דְּעַל גַּבֵּי חַמְשִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שְׁ</span>כִינְתָּא תִּתְעַטָּר. בְּשִׁית נַהֲמֵי לִסְטָר. בְּוָוִין תִּתְקַטָּר. וְזִינִין דִּכְנִישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שְׁ</span>בִיתִין וּשְׁבִיקִין. מְסָאֲבִין דִּרְחִיקִין. חֲבִילִין דִּמְעִיקִין. וְכָל זִינֵי חַרְשִׁין:
<br/>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְמִבְצַע</span> עַל רִפְתָּא. כְּזֵיתָא וּכְבֵיעָתָא. תְּרֵין יוּדִין נַקְטָא. סְתִימִין וּפְרִישִׁין: מְשַׁח זֵיתָא דַכְיָא. דְטַחֲנִין רֵחַיָּא. וְנַגְדִּין נַחֲלַיָּא. בְּגַוָּהּ בִּלְחִישִׁין: הֲלָא נֵימָא רָזִין. וּמִלִין דִּגְנִיזִין. דְּלֵיתְהוֹן מִתְחַזִין. טְמִירִין וּכְבִישִׁין: אִתְעַטְרָא כַלָּה. בְּרָזִין דִּלְעֵלָא. בְּגוֹ הַאי הִלּוּלָא. דְּעִירִין קַדִּישִׁין:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וִיהֵא</span> רַעֲוָא מִן קֳדָם עַתִּיקָא קַדִּישָׁא דְכָל קַדִּישִׁין. טְמִירָא דְּכָל טְמִירִין סְתִימָא דְכלָא דְיִתְמַשֵׁךְ טַלָּא עִלָּאָה מִינֵהּ לְמַלְיָא רֵישֵׁיהּ דִזְעֵיר אַנְפִּין וּלְהַטִּיל לַחֲקַל תַּפּוּחִין קַדִּישִׁין בִּנְהִירוּ דְאַנְפִּין בְּרַעֲוָא וּבְחֶדְוָתָא דְכלָא וְיִתְמַשֵּׁךְ מִן קֳדָם עַתִּיקָא קַדִּישָׁא דְכָל קַדִּישִׁין טְמִירָא דְכָל טְמִירִין סְתִּימָא דְכלָא רְעוּתָא וְרַחֲמֵי חִנָּא וְחִסְדָּא בִּנְהִירוּ עִלָּאָה בִּרְעוּתָא וְחֶדְוָתָא עָלַי וְעַל כָּל בְּנֵי בֵּיתִי וְעַל כָּל הַנִּלְוִים אֵלַי וְעַל כָּל בְּנֵי יִשְׂרָאֵל עַמֵיהּ וְיִפְרְקִינָנָא מִכָּל עַקְתִין בִּישִׁין דְיֵיתוּן לְעַלְמָא וְיַזְמִין וְיִתְיַהֵב לָנָא מְזוֹנָא וּפַּרְנָסָתָא טַבְתָּא בְּלִי צָרָה וְעַקְתָא מִמַזְלָא דְכָל מְזוֹנָא בֵּיהּ תַּלְיָא וִישֵׁזְבִינָנָא מֵעֵינָא בִּישָׁא וּמֵחַרְבָּא דְמַלְאַךְ הַמָּוֶת וּמִדִּינָהּ שֶׁל גֵיהִנּם. וְיֵתֵי לָנָא וּלְכָל נַפְשָׁתָנָא חִנָּא וְחִסְדָּא וְחַיֵי אֲרִיכֵי וּמְזוֹנֵי רְוִיחֵי וְרַחֲמֵי מִן קֳדָמֵיהּ אָמֵן כֵּן יְהִי רָצוֹן אָמֵן וְאָמֵן:
<h2>קידוש ליל שבת</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְשֵׁם</span> יִחוּד קוּדְשָׁא בְּרִיךְ הוּא וּשְׁכִינְתֵּיהּ, בִּדְחִילוּ וּרְחִימוּ, הֲרֵינִי מוּכָן וּמְזוּמָן לְקַיֵּם מִצְוַת עֲשֵׂה לְקַדֵּשׁ עַל הַיַּיִן, כִּדְכְתִיב, זָכוֹר וְשָׁמוֹר, זָכְרֵהוּ עַל הַיַּיִן: וִיהִי נוֹעַם אֲדֹנָי אֱלֹהֵינוּ עָלֵינוּ. וּמַעֲשֵׂה יָדֵינוּ כּונְנָה עָלֵינוּ. וּמַעֲשֵׂה יָדֵינוּ כּונְנֵהוּ:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נותן עיניו בכוס ומסתכל על הנרות</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בלחש:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וַיְהִי עֶרֶב וַיְהִי בֹקֶר</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יוֹם</span> הַשִּׁשִּׁי. וַיְכֻלּוּ הַשָּׁמַיִם וְהָאָרֶץ וְכָל צְבָאָם: וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי מְלַאכְתּוֹ אֲשֶׁר עָשָׂה. וַיִּשְׁבֹּת בַּיּוֹם הַשְּׁבִיעִי מִכָּל מְלַאכְתּוֹ אֲשֶׁר עָשָׂה: וַיְבָרֶךְ אֱלֹהִים אֶת יוֹם הַשְּׁבִיעִי וַיְקַדֵּשׁ אֹתוֹ. כִּי בוֹ שָׁבַת מִכָּל מְלַאכְתּוֹ אֲשֶׁר בָּרָא אֱלֹהִים לַעֲשׂוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">סַבְרִי מָרָנָן וְרַבָּנָן וְרַבּוֹתַי:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם בּוֹרֵא פְּרִי הַגָּפֶן:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם. אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְרָצָה בָנוּ. וְשַׁבַּת קָדְשׁוֹ בְּאַהֲבָה וּבְרָצוֹן הִנְחִילָנוּ. זִכָּרוֹן לְמַעֲשֵׂה בְרֵאשִׁית. <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כִּי הוּא יוֹם</span> תְּחִלָּה לְמִקְרָאֵי קֹדֶשׁ זֵכֶר לִיצִיאַת מִצְרָיִם. <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כִּי בָנוּ בָחַרְתָּ וְאוֹתָנוּ קִדַּשְׁתָּ מִכָּל הָעַמִּים</span> וְשַׁבַּת קָדְשְׁךָ בְּאַהֲבָה וּבְרָצוֹן הִנְחַלְתָּנוּ: בָּרוּךְ אַתָּה יְהֹוָה מְקַדֵּשׁ הַשַּׁבָּת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בשבת חוה"מ סוכות יאמר זה תיכף אחר קידוש:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם. אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָנוּ לֵישֵׁב בַּסֻּכָּה:</span>
<h2>זמירות לליל שבת</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">זה הזמר מיוסד מן חרוז שני והלאה ע"פ א"ב וכל חלק הרביעי מן כל חרוז לקוח מן פסוק אחד. ובחרוז הראשון מפני ענותנותו של המחבר חתם שמו בראשי תיבות שניות של ג' חלקי החרוז שהם מקדש שומר הרבה ר"ת משה:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">כָּל</span> מְקַדֵּשׁ שְׁבִיעִי כָּרָאוּי לוֹ. כָּל שׁוֹמֵר שַׁבָּת כַּדָּת מֵחַלְּלוֹ. שְׂכָרוֹ הַרְבֵּה מְאֹד עַל פִּי פָעֳלוֹ. אִישׁ עַל מַחֲנֵהוּ וְאִישׁ עַל דִּגְלוֹ: אוֹהֲבֵי יְהֹוָה הַמְחַכִּים בְּבִנְיַן אֲרִיאֵל. בְּיוֹם הַשַׁבָּת שִׂישׁוּ וְשִׂמְחוּ כִּמְקַבְּלֵי מַתַּן נַחֲלִיאֵל. גַּם שְׂאוּ יְדֵיכֶם קוֹדֶשׁ וְאִמְרוּ לָאֵל. בָּרוּךְ יְהֹוָה אֲשֶׁר נָתַן מְנוּחָה לְעַמּוֹ יִשְׂרָאֵל: דוֹרְשֵׁי יְהֹוָה זֶרַע אַבְרָהָם אוֹהֲבוֹ. הַמְאַחֲרִים לָצֵאת מִן הַשַּׁבָּת וּמְמַהֲרִים לָבֹא. שְׂמֵחִים לְשָׁמְרוֹ וּלְעָרֵב עֵרוּבוֹ. זֶה הַיּוֹם עָשָׂה יְהֹוָה נָגִילָה וְנִשְׂמְחָה בוֹ: זִכְרוּ תּוֹרַת משֶׁה בְּמִצְוַת שַׁבָּת גְּרוּסָה. חֲרוּתָה לְיוֹם הַשְּׁבִיעִי כְּכַלָּה בֵּין רֵעוֹתֶיהָ מְשֻׁבָּצָה. טְהוֹרִים יִירָשׁוּהָ וִיקַדְּשׁוּהָ בְּמַאֲמַר כָּל אֲשֶׁר עָשָׂה. וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי מְלַאכְתּוֹ אֲשֶׁר עָשָׂה: יוֹם קָדוֹשׁ הוּא מִבּוֹאוֹ וְעַד צֵאתוֹ. כָּל זֶרַע יַעֲקֹב יְכַבְּדוּהוּ כִּדְבַר הַמֶּלֶךְ וְדָתוֹ. לָנוּחַ בּוֹ וְלִשְׂמוֹחַ בְּתַעֲנוּג אָכוֹל וְשָׁתוֹ כָּל עֲדַת יִשְׂרָאֵל יַעֲשׂוּ אוֹתוֹ: מְשֹׁךְ חַסְדְּךָ לְיֹדְעֶיךָ אֵל קַנֹּא וְנוֹקֵם. נוֹטְרֵי לַיּוֹם הַשְּׁבִיעִי זָכוֹר וְשָׁמוֹר לְהָקֵם. שַׂמְּחֵם בְּבִנְיַן שָׁלֵם בְּאוֹר פָּנֶיךָ תַּבְהִיקֵם. יִרְוְיֻן מִדֶּשֶׁן בֵּיתֶךָ וְנַחַל עֲדָנֶיךָ תַשְׁקֵם: עֲזוֹר לַשּׁוֹבְתִים בַּשְּׁבִיעִי בֶּחָרִישׁ וּבַקָּצִיר עוֹלָמִים. פּוֹסְעִים בּוֹ פְּסִיעָה קְטַנָּה. סוֹעֲדִים בּוֹ לְבָרֵךְ שָׁלשׁ פְּעָמִים. צִדְקָתָם תַּצְהִיר כְּאוֹר שִׁבְעַת הַיָּמִים. יְהֹוָה אֱלֹהֵי יִשְׂרָאֵל אַהֲבַת תָּמִים. יְהֹוָה אֱלֹהֵי יִשְׂרָאֵל תְּשׁוּעַת עוֹלָמִים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">זה הזמר שם המחבר משה וחתם שמו בג' ראשי החרוזים הראשונים מנוחה שמי הוא:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מְנוּחָה</span> וְשִׂמְחָה אוֹר לַיְּהוּדִים יוֹם שַׁבָּתוֹן יוֹם מַחֲמַדִּים. שׁוֹמְרָיו וְזוֹכְרָיו הֵמָּה מְעִידִים. כִּי לְשִׁשָּׁה כּל בְּרוּאִים וְעוֹמְדִים: שְׁמֵי שָׁמַיִם אֶרֶץ וְיַמִּים. כָּל צְבָא מָרוֹם גְּבוֹהִים וְרָמִים. תַּנִּין וְאָדָם וְחַיַּת רְאֵמִים. כִּי בְּיָהּ יְהֹוָה צוּר עוֹלָמִים: הוּא אֲשֶׁר דִּבֶּר לְעַם סְגֻלָּתוֹ. שָׁמוֹר לְקַדְּשׁוֹ מִבֹּאוֹ וְעַד צֵאתוֹ. שַׁבָּת קֹדֶשׁ יוֹם חֶמְדָּתוֹ. כִּי בוֹ שָׁבַת אֵל מִכָּל מְלַאכְתּוֹ: בְּמִצְוַת שַׁבָּת אֵל יַחֲלִיצָךְ. קוּם קְרָא אֵלָיו יָחִישׁ לְאַמְּצָךְ. נִשְׁמַת כָּל חַי וְגַם נַעֲרִיצָךְ. אֱכוֹל בְּשִׂמְחָה כִּי כְּבָר רָצָךְ: בְּמִשְׁנֶה לֶחֶם וְקִדּוּשׁ רַבָּה. בְּרוֹב מַטְעַמִּים וְרוּחַ נְדִיבָה. יִזְכּוּ לְרַב טוּב הַמִּתְעַנְּגִים בָּהּ. בְּבִיאַת גּוֹאֵל לְחַיֵּי הָעוֹלָם הַבָּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">זה הזמר שם המחבר מנחם והסימן מתחיל מהחרוז השני מערב נחלת חפציך מעין:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מַה</span> יְּדִידוּת מְנוּחָתֵךְ. אַתְּ שַׁבָּת הַמַּלְכָּה. בְּכֵן נָרוּץ לִקְרָאתֵךְ. בּוֹאִי כַלָּה נְסוּכָה. לְבוּשׁ בִּגְדֵי חֲמוּדוֹת. לְהַדְלִיק נֵר בִּבְרָכָה. וַתֵּכֶל כָּל הָעֲבוֹדוֹת. לֹא תַעֲשׁוּ מְלָאכָה: לְהִתְעַנֵּג בְּתַעֲנוּגִים. בַּרְבּוּרִים וּשְׂלָיו וְדָגִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מֵ</span>עֶרֶב מַזְמִינִים כָּל מִינֵי מַטְעַמִּים. מִבְּעוֹד יוֹם מוּכָנִים. תַּרְנְגוֹלִים מְפֻטָּמִים. וְלַעֲרוֹךְ בּוֹ כַּמָּה מִינִים שְׁתוֹת יֵינוֹת מְבֻשָּׂמִים. וְתַפְנוּקֵי מַעֲדַנִּים. בְּכָל שָׁלשׁ פְּעָמִים: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">להתענג</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">נַ</span>חֲלַת יַעֲקֹב יִירָשׁ. בְּלִי מְצָרִים נַחֲלָה. וִיכַבְּדוּהוּ עָשִׁיר וָרָשׁ. וְתִזְכּוּ לִגְאֻלָּה. יוֹם שַׁבָּת אִם תִּשְׁמרוּ. וִהְיִיתֶם לִי סְגֻלָּה. שֵׁשֶׁת יָמִים תַּעֲבוֹדוּ. וּבַשְּׁבִיעִי נָגִילָה: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">להתענג</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">חֲ</span>פָצֶיךָ אֲסוּרִים. וְגַם לַחֲשׁוֹב חֶשְׁבּוֹנוֹת. הִרְהוּרִים מֻתָּרִים וּלְשַׁדֵּךְ הַבָּנוֹת. וְתִנּוֹק לְלַמְּדוֹ סֵפֶר. לַמְנַצֵּחַ בִּנְגִינוֹת. וְלַהֲגוֹת בְּאִמְרֵי שֶׁפֶר. בְּכָל פִּנּוֹת וּמַחֲנוֹת: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">להתענג</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">הִ</span>לּוּכָךְ תְּהֵא בְנַחַת. עוֹנֶג קְרָא לַשַּׁבָּת. וְהַשֵּׁנָה מְשֻׁבַּחַת. כְּדַת נֶפֶשׁ מְשִׁיבַת. בְּכֵן נַפְשִׁי לְךָ עָרְגָה. וְלָנוּחַ בְּחִבַּת. כַּשּׁוֹשַׁנִּים סוּגָה. בּוֹ יָנוּחוּ בֵּן וּבַת: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">להתענג</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מֵ</span>עֵין עוֹלָם הַבָּא. יוֹם שַׁבָּת מְנוּחָה. כָּל הַמִּתְעַנְּגִים בָּהּ. יִזְכּוּ לְרוֹב שִׂמְחָה. מֵחֶבְלֵי מָשִׁיחַ יֻצָּלוּ לִרְוָחָה. פְּדוּתֵינוּ תַצְמִיחַ וְנָס יָגוֹן וַאֲנָחָה: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">להתענג</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">זה הזמר שם המחבר מרדכי בר יצחק חזק ושמו חתום בראשי החרוזים</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מַה</span> יָּפִית וּמַה נָּעַמְתְּ אַהֲבָה בְּתַעֲנוּגִים. אַתְּ שַׁבָּת מְשׁוֹשׁ נוּגִים. לְךָ בָּשָׂר וְגַם דָּגִים. נְכוֹנִים מִבְּעוֹד יוֹם: מֵעֶרֶב עַד עֶרֶב לֵב חָדִים. בְּבֹא עִתֵּךְ עֵת דּוֹדִים. גִּיל וְשָׂשׁוֹן לַיְהוּדִים. לִמְצוֹא פִדְיוֹם: וְאַתְּ עוֹנֶג לְהִתְעַנֵּג בְּמַמְתַּקִּים. בְּתַעֲנוּגוֹת בְּנֵי אָדָם. וְיַיִן כִּי יִתְאַדָּם. וּשְׁאָר מַשְׁקִים: רְאֵה וְקַדֵּשׁ בְּיוֹם קוֹדֶשׁ. עֲלֵי יַיִן זָכְרֵהוּ. וְאִם אַיִן עֲלֵי לֶחֶם בְּצַע. בְּעַיִן יָפָה לְקַדְּשׁוֹ: שָׁמְרֵהוּ כְּהִלְכוֹתָיו מֵעֲבוֹדוֹת. מֵאָבוֹת וְתוֹלָדוֹת. שִׁיר לָאֵל תֵּן לְהוֹדוֹת. זֵכֶר לְקַדְּשׁוֹ: כָּל שׁוֹמֵר יוֹם שַׁבָּת מֵחַלְלוֹ. מְחוֹלְלוֹ יִמְחוֹל מַעֲלוֹ. וּבָא גוֹאֲלוֹ שְׁמוֹ שִׁילֹה. יוּבַל שַׁי לוֹ: דְּבַר סִימָּן טוֹב לָךְ בְּמָן וְאוֹת דֶּגֶל. סַמְבַּטְיוֹן הַמִתְגַּלְגֵּל. בְּכָל יוֹם נָח כְּעַם סֶגֶל. יִשְׁבְּתוּ וְיִשְׁקוֹטוּ: הַטּוֹרְחִים וְלֹא נָחִים עֲדֵי שִׁשִּׁי. וַתְּשַׁלְּחֵם לַחָפְשִׁי. בִּגְדֵי שֵׁשׁ וְגַם מֶשִׁי. לְכַבְּדֵךְ יַעֲטוּ: וְיאֹמְרוּ בּוֹאִי כַּלָּה מַה תְּאַחֲרִי. הֵן שֻׁלְחָן וְגַם נֵרֵךְ עֲרוּכִים. כִּי בָא אוֹרֵךְ. קוּמִי אוֹרִי: כְּבוֹד יְהֹוָה עָלַיִךְ כְּאוֹר זָרַח. בְּעֵת יָצַר בְּלִי טוֹרַח. קְצוֹת מַעֲרָב וְגַם מִזְרָח. צָפוֹן וְיָמִין: עֶלְיוֹנִים וְתַחְתּוֹנִים בְּמַאֲמָרוֹ. עֲלֵי תֵבֵל שָׂם אוֹרוֹ. וְכַחֲטוֹא יְצִיר לְיוֹצְרוֹ. בִקְּשׁוֹ לְהַטְמִין: וְאַתְּ
