<h1>פיוט אקדמות לשבועות</h1>

<h2>פיוט אקדמות לשבועות</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחג השבועות לפני קריאת התורה נוהגים לומר אקדמות:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַקְדָמוּת</span> מִילִין, וְשָׁרָיוּת שׁוּתָא:
אַוְלָא שָׁקִילְנָא, הַרְמָן וּרְשׁוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בְּבָבֵי תְּרֵי וּתְלַת, דְאֶפְתַּח בְּנַקְשׁוּתָא: בְּבָרֵי דְבָרֵי וְטָרֵי, עֲדֵי לְקַשִׁשׁוּתָא:</span>
גְבוּרָן עָלְמִין לֵיה, וְלֹא סִפֵּק פְּרִישׁוּתָא:
גְוִיל אִלּוּ רְקִיעֵי, קְנֵי כָּל חוּרְשָׁתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">דְיוֹ אִלּוּ יַמֵּי, וְכָל מֵי כְּנִישׁוּתָא: דַיְירֵי אַרְעָא סַפְרֵי, וְרַשְׁמֵי רַשְׁוָתָא:</span>
הֲדַר מָרֵי שְׁמַיָא, וְשַׁלִיט בְּיַבֶּשְׁתָּא:
הָקֵם עָלְמָא יְחִידָאֵי, וְכַבְּשֵׁיהּ בְּכַבְּשׁוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">וּבְלָא לֵאוּ שַׁכְלְלֵיה, וּבְלָא תְשָׁשׁוּתָא: וּבְאָתָא קַלִילָא, דְלֵית בָּהּ מְשָׁשׁוּתָא:</span>
זַמִין כָּל עֲבִידְתֵּיהּ, בְּהַךְ יוֹמֵי שִׁתָּא:
זֵהוֹר יְקָרֵיה עֲלִי, עֲלֵי כּוּרְסֵיהּ דֶּאֱשָׁתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חֲיָל אֶלֶף אַלְפִין וְרִבּוֹא, לְשַׁמְשׁוּתָא: חַדְתִּין נְבוֹט לְצַפְרִין, סַגִיאָה טְרָשׁוּתָא:</span>
טְפֵי יְקִידִין שְׂרָפִין, כְּלוּל גַפֵּי שִׁתָּא:
טְעֵם עַד יִתְיְהֵב לְהוֹן, שְׁתִּיקִין בְּאַדִשְׁתָּא
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְקַבְּלוּן דֵין מִן דֵין שָׁוֵי, דְלָא בְּשַׁשְׁתָּא: יְקַר מְלֵי כָל אַרְעָא, לִתְלוֹתֵי קְדוּשְׁתָּא:</span>
כְּקָל מִן קֳדָם שַׁדַי, כְּקָל מֵי נְפִישׁוּתָא:
כְּרוּבִין קֳבֵל גַלְגַלִין, מְרוֹמְמִין בְּאַוְשָׁתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לְמֶחֱזֵי בְּאַנְפָּא עֵין, כְּוַת גִירֵי קַשְׁתָּא: לְכָל אֲתַר דְמִשְׁתַּלְחִין, זְרִיזִין בְּאַשְׁוָתָא:</span>
מְבָרְכִין בְּרִיךְ יְקָרֵהּ, בְּכָל לְשָׁן לְחִישׁוּתָא:
מֵאֲתַר בֵּית שְׁכִינְתֵּהּ, דְלָא צָרִיךְ בְּחִישׁוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נְהִים כָּל חֵיל מְרוֹמָא, מְקַלְסִין בַּחֲשַׁשְׁתָּא: נְהִירָא מַלְכוּתֵהּ, לְדָר וְדָר לְאַפְרַשְׁתָּא:</span>
סְדִירָא בְּהוֹן קְדוּשְׁתָּא, וְכַד חַלְפָא שַׁעְתָּא:
סִיוּמָא דִלְעָלַם, וְאוֹף לָא לִשְׁבוּעָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">עֲדַב יְקַר אַחֲסַנְתֵּהּ, חֲבִיבִין דְבִקְבַעְתָּא: עֲבִידִין לֵיהּ חֲטִיבָא, בִּדְנַח וּשְׁקַעְתָּא:</span>
פְּרִישָׁן לְמָנָתֵהּ, לְמֶעְבַּד לֵיהּ רְעוּתָא:
פְּרִישׁוּתֵהּ שְׁבָחֵהּ, יְחַווּן בְּשָׁעוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">צְבִי וְחָמִיד וְרָגִיג, דְיִלְאוֹן בְּלָעוּתָא: צְלוֹתְהוֹן בְּכֵן מְקַבֵּל, וְהַנְיָא בָּעוּתָא:</span>
קְטִירָא לְחֵי עָלְמָא, בְּתָגָא בִּשְׁבוּעָתָא:
קֳבֵל יְקַר טוֹטֶפְתָּא, יְתִיבָא בִקְבִיעוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">רְשִׁימָא הִיא גוּפָא, בְּחָכְמְתָא וּבְדַעְתָּא: רְבוּתְהוֹן דְיִשְׂרָאֵל, קְרָאֵי בִשְׁמַעְתָּא:</span>
שְׁבַח רִבּוֹן עַלְמָא, אֲמִירָא דַכְוָתָא:
שְׁפַר עֲלֵיהּ לְחַווּיֵהּ, בְּאַפֵּי מַלְכְּוָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">תָּאִין וּמִתְכַּנְשִׁין, כְּחֵיזוּ אַדְוָתָא: תְּמֵהִין וְשַׁיְילִין לֵיהּ, בְּעֵסֶק אַתְוָתָא:</span>
מְנָן וּמָאן הוּא רְחִימָךְ, שַׁפִּירָא בְּרֵיוָתָא:
אֲרוּם בְּגִינֵהּ סָפִית, מִדוּר אַרְיְוָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְקָרָא וְיָאָה אַתְּ אִין, תַּעַרְבִי לְמַרְוָתָא: רְעוּתֵךְ נַעֲבֵיד לִיךְ, בְּכָל אַתְרְוָתָא:</span>
בְּחָכְמְתָא מְתִיבָתָא לְהוֹן, קְצַת לְהוֹדָעוּתָא:
יְדַעְתּוּן חַכְּמִין לֵיהּ, בְּאִשְׁתְּמוֹדָעוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">רְבוּתְכוֹן מָה חֲשִׁיבָא, קֳבֵל הַהִיא שְׁבַחְתָּא: רְבוּתָא דְיַעֲבֵיד לִי, כַּד מַטְיָא שַׁעְתָּא:</span>
בְּמֵיתֵי לִי נְהוֹרָא, וְתַחֲפֵי לְכוֹן בַּהֲתָא:
יְקָרֵיהּ כַּד אִתְגְלֵי, בְּתוּקְפָא וּבִגְבוּרְתָּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְשַׁלֵם גְמֻלַיָא, לְסַנְאֵי וְנַגְוָתָא: צִדְקָתָא לְעַם חָבִיב, וְסַגְיָא זַכְוָתָא:</span>
חֲדוּ שְׁלֵמָא בְּמֵיתֵי, וּמָנָא דַכְוָתָא:
קִרְיְתָא דִירוּשְׁלֵם, כַּד יְכַנֵשׁ גַלְוָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יְקָרֵיהּ מָטֵיל עֲלֵיהּ, בְּיוֹמֵי וְלֵילָותָא: גְנוּנֵיהּ לְמֶעְבַּד בָּהּ, בְּתוּשְׁבְּחָן כְּלִילָתָא:</span>
דְזֵיהוֹר עֲנָנַיָּא, לְמִשְׁפַּר כִּילָתָא:
לְפוּמֵיהּ דַּעֲבִידְתָּא, עֲבִידָן מְטַלַּלְתָּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בְּתַכְתַּקֵי דְהַב פִּיזָא, וּשְׁבַע מַעֲלָתָא: תְּחִימִין צַדִיקֵי, קָדָם רַב פָּעֳלָתָא:</span>
וְרֵיוֵיהוֹן דָמֵי, לְשָׂבְעָא חֶדְוָתָא:
רְקִיעָא בְּזֵיהוֹרֵיהּ, וְכוֹכְבֵי זִיוָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הֲדָרָא דְלָא אֶפְשַׁר, לְמִפְרַט בִּשִׂפְוָתָא: וְלָא אִשְׁתְּמַע וְחָמֵי, נְבִיאָן חֶזְוָתָא:</span>
בְּלָא שָׁלְטָא בֵיהּ עֵין, בְּגוֹ עֵדֶן גִנְתָא:
מְטַיְילֵי בֵיהּ חִנְגָּא, לְבַהֲדֵי דִשְׁכִינְתָּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">עֲלֵיהּ רַמְזֵי דֵין הוּא, בְּרַם בְּאֶמְתָנוּתָא: שַׂבַּרְנָא לֵיהּ בְּשִׁבְיָן, תְּקוֹף הֵמָנוּתָא:</span>
יַדְבֵּר לָן עָלְמִין, עֲלֵמִין מְדַמוּתָא:
מְנָת דִילָן דְּמִלְּקַדְמִין, פָּרֵשׁ בַּאֲרָמוּתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">טְלוּלָא דְלִוְיָתָן, וְתוֹר טוּר רָמוּתָא: וְחַד בְּחַד כִּי סָבִיךְ, וְעָבִיד קְרָבוּתָא:</span>
בְּקַרְנוֹהִי מְנַגַּח בְּהֵמוֹת, בְּרַבְרְבוּתָא:
יְקַרְטַע נוּן לְקִבְלֵהּ, בְּצִיצוֹי בִּגְבוּרְתָּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מְקָרֵב לֵיה בָּרְיֵהּ, בְּחַרְבֵּהּ בְּרַבְרְבוּתָא: אֲרִיסְטוֹן לְצַדִּיקֵי יְתַקֵן, וְשֵׁרוּתָא:</span>
מְסַחֲרִין עֲלֵי תַכֵּי, דְכַדְכּוֹד וְגוּמַרְתָּא:
נְגִידִין קָמֵיהוֹן, אֲפַרְסְמוֹן נַהֲרָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">וּמִתְפַּנְקִין וְרָוֵי, בְּכַסֵי רְוָיָתָא: חֲמַר מְרַת דְמִבְּרֵאשִׁית, נְטִיר בֵּי נַעֲוָתָא:</span>
זַכָּאִין כַּד שְׁמַעְתּוּן, שְׁבַח דָא שִׁירָתָא:
קְבִיעִין כֵּן תֶּהֱווּן, בְּהַנְהוּ חֲבוּרָתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">וְתִזְכּוּן דִי תֵיתְבוּן, בְּעֵילָא דָרָתָא: אֲרֵי תְצִיתוּן לְמִילָי, דְנָפְקִין בְּהַדְרָתָא:</span>
מְרוֹמָם הוּא אֱלָהִין, בְּקַדְמָא וּבַתְרַיְתָא:
צְבִי וְאִתְרְעִי בָן, וּמְסַר לָן אוֹרַיְתָא:
