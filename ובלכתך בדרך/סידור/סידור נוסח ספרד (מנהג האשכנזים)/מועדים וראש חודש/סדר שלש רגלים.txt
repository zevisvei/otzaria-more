<h1>סדר שלש רגלים</h1>

<h2>קידוש לשלש רגלים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשחל יו"ט בשבת מתחילים כאן:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נותן עיניו בכוס ומסתכל על הנרות</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בלחש:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">וַיְהִי עֶרֶב וַיְהִי בֹקֶר</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יוֹם</span> הַשִּׁשִּׁי. וַיְכֻלּוּ הַשָּׁמַיִם וְהָאָרֶץ וְכָל צְבָאָם: וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי מְלַאכְתּוֹ אֲשֶׁר עָשָׂה. וַיִּשְׁבֹּת בַּיּוֹם הַשְּׁבִיעִי מִכָּל מְלַאכְתּוֹ אֲשֶׁר עָשָׂה: וַיְבָרֶךְ אֱלֹהִים אֶת יוֹם הַשְּׁבִיעִי וַיְקַדֵּשׁ אֹתוֹ. כִּי בוֹ שָׁבַת מִכָּל מְלַאכְתּוֹ אֲשֶׁר בָּרָא אֱלֹהִים לַעֲשׂוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשחל יו"ט בחול מתחילין כאן:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">סַבְרִי מָרָנָן וְרַבָּנָן וְרַבּוֹתַי:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם בּוֹרֵא פְּרִי הַגָּפֶן:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם אֲשֶׁר בָּחַר בָּנוּ מִכָּל עָם וְרוֹמְמָנוּ מִכָּל לָשׁוֹן וְקִדְּשָׁנוּ בְּמִצְוֹתָיו. וַתִּתֶּן לָנוּ יְהֹוָה אֱלֹהֵינוּ בְּאַהֲבָה <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">(<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> שַׁבָּתוֹת לִמְנוּחָה וּ)מוֹעֲדִים לְשִׂמְחָה. חַגִּים וּזְמַנִּים לְשָׂשׂוֹן. אֶת יוֹם <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">(<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> הַשַּׁבָּת הַזֶּה וְאֶת יוֹם):
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לפסח:</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">חַג הַמַּצּוֹת הַזֶּה. זְמַן חֵרוּתֵנוּ:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבועות:</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">חַג הַשָׁבוּעוֹת הַזֶּה. זְמַן מַתַּן תּוֹרָתֵינוּ:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לסוכות:</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">חַג הַסֻּכּוֹת הַזֶּה. זְמַן שִׂמְחָתֵנוּ:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשמ"ע ולש"ת:</span> <span style="font-weight:bold;text-shadow:0 0px 0 white;">שְׁמִינִי עֲצֶרֶת הַחַג הַזֶּה. זְמַן שִׂמְחָתֵנוּ:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> בְּאַהֲבָה) מִקְרָא קֹדֶשׁ זֵכֶר לִיצִיאַת מִצְרָיִם. כִּי בָנוּ בָחַרְתָּ וְאוֹתָנוּ קִדַּשְׁתָּ מִכָּל הָעַמִּים. <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> וְשַׁבָּת) וּמוֹעֲדֵי קָדְשֶׁךָ <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> בְּאַהֲבָה וּבְרָצוֹן) בְּשִׂמְחָה וּבְשָׂשׂוֹן הִנְחַלְתָּנוּ: בָּרוּךְ אַתָּה יְהֹוָה. מְקַדֵּשׁ <span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span> הַשַּׁבָּת וְ) יִשְׂרָאֵל וְהַזְּמַנִּים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בליל ראשון של סוכות אומרים תחלה לישב בסוכה ואח"כ שהחינו:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם שֶׁהֶחֱיָנוּ וְקִיְּמָנוּ וְהִגִּיעָנוּ לַזְּמַן הַזֶּה:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם. אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו. וְצִוָּנוּ לֵישֵׁב בַּסֻּכָּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשחל יו"ט במוצאי שבת מקדשין לפי סדר יקנה"ז, דהיינו יין, קידוש, נר, הבדלה, זמן:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם. בּוֹרֵא מְאוֹרֵי הָאֵשׁ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם. הַמַּבְדִּיל בֵּין קֹדֶשׁ לְחוֹל. בֵּין אוֹר לְחֹשֶׁךְ. בֵּין יִשְׂרָאֵל לָעַמִּים. בֵּין יוֹם הַשְּׁבִיעִי לְשֵׁשֶׁת יְמֵי הַמַּעֲשֶׂה. בֵּין קְדֻשַּׁת שַׁבָּת לִקְדֻשַּׁת יוֹם טוֹב הִבְדַּלְתָּ. וְאֶת יוֹם הַשְּׁבִיעִי מִשֵּׁשֶׁת יְמֵי הַמַּעֲשֶׂה קִדַּשְׁתָּ. הִבְדַּלְתָּ וְקִדַּשְׁתָּ אֶת עַמְּךָ יִשְׂרָאֵל בִּקְדֻשָּׁתֶךָ: בָּרוּךְ אַתָּה יְהֹוָה. הַמַּבְדִּיל בֵּין קֹדֶשׁ לְקֹדֶשׁ: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם שֶׁהֶחֱיָנוּ וְקִיְּמָנוּ וְהִגִּיעָנוּ לַזְּמַן הַזֶּה:</span>
<h2>תפילת שלש רגלים לערבית, לשחרית ולמנחה</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מתפללים ערבית כמו בשבת ואחר השכיבנו אוֹמרים "וידבר משה", ובשבת אומרים קודם: "ושמרו בני יִשְׂרָאֵל"</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיְדַבֵּר</span> מֹשֶׁה אֶת מוֹעֲדֵי יְהֹוָה, אֶל בְּנֵי יִשְׂרָאֵל:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חצי קדיש:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא. יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא ( בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן:</span>
<h2>שמונה עשרה לשחרית, מנחה וערבית של שלוש רגלים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">לערבית ולשחרית:</span></span> <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֲדֹנָי</span>, שְׂפָתַי תִּפְתָּח וּפִי יַגִּיד תְּהִלָּתֶךָ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">למנחה:</span></span> <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כִּי</span> שֵׁם יְהֹוָה אֶקְרָא הָבוּ גֹדֶל לֵאלֹהֵינוּ: אֲדֹנָי, שְׂפָתַי תִּפְתָּח וּפִי יַגִּיד תְּהִלָּתֶךָ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, אֱלֹהֵי אַבְרָהָם, אֱלֹהֵי יִצְחָק וֵאלֹהֵי יַעֲקֹב. הָאֵל הַגָּדוֹל, הַגִּבּוֹר וְהַנּוֹרָא, אֵל עֶלְיוֹן, גּוֹמֵל חֲסָדִים טוֹבִים, וְקוֹנֵה הַכֹּל, וְזוֹכֵר חַסְדֵי אָבוֹת, וּמֵבִיא גוֹאֵל לִבְנֵי בְנֵיהֶם לְמַעַן שְׁמוֹ, בְּאַהֲבָה. מֶלֶךְ עוֹזֵר וּמוֹשִׁיעַ וּמָגֵן, בָּרוּךְ אַתָּה יְהֹוָה, מָגֵן אַבְרָהָם:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַתָּה</span> גִּבּוֹר לְעוֹלָם, אֲדֹנָי. מְחַיֵּה מֵתִים אַתָּה, רַב לְהוֹשִׁיעַ.
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בקיץ:</span> מוֹרִיד הַטָּל. <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחורף:</span> מַשִּׁיב הָרוּחַ וּמוֹרִיד הַגֶּשֶׁם.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מְכַלְכֵּל</span> חַיִּים בְּחֶסֶד, מְחַיֵּה מֵתִים בְּרַחֲמִים רַבִּים, סוֹמֵךְ נוֹפְלִים, וְרוֹפֵא חוֹלִים, וּמַתִּיר אֲסוּרִים, וּמְקַיֵּם אֱמוּנָתוֹ לִישֵׁנֵי עָפָר. מִי כָמוֹךָ בַעַל גְּבוּרוֹת, וּמִי דוֹמֶה לָּךְ, מֶלֶךְ מֵמִית וּמְחַיֶּה וּמַצְמִיחַ יְשׁוּעָה. וְנֶאֱמָן אַתָּה לְהַחֲיוֹת מֵתִים. בָּרוּךְ אַתָּה יְהֹוָה, מְחַיֵּה הַמֵּתִים:
<span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><span style="text-align: center;"><i>קדושה</i></span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">בחזרת הש"ץ</span>: אוֹמרים קדושה</span> <br/><span style="color:RGB(0,127,127); font-family: arial; <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מוֹדִים</span> אֲנַחְנוּ לָךְ, שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ לְעוֹלָם וָעֶד, צוּרֵנוּ צוּר חַיֵּינוּ וּמָגֵן יִשְׁעֵנוּ אַתָּה הוּא לְדֹר וָדֹר. נוֹדֶה לְּךָ וּנְסַפֵּר תְּהִלָּתֶךָ, עַל חַיֵּינוּ הַמְּסוּרִים בְּיָדֶיךָ, וְעַל נִשְׁמוֹתֵינוּ הַפְּקוּדוֹת לָךְ, וְעַל נִסֶּיךָ שֶׁבְּכָל יוֹם עִמָּנוּ, וְעַל נִפְלְאוֹתֶיךָ וְטוֹבוֹתֶיךָ שֶׁבְּכָל עֵת, עֶרֶב וָבֹקֶר וְצָהֳרָיִם. הַטּוֹב, כִּי לֹא כָלוּ רַחֲמֶיךָ, הַמְּרַחֵם, כִּי לֹא תַמּוּ חֲסָדֶיךָ, כִּי מֵעוֹלָם קִוִּינוּ לָךְ:
<span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><span style="text-align: center;"><i>מודים דרבנן</i></span><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחזרת הש"ץ</span></span> כשהוא אוֹמר מודים, אוֹמרים הקהל:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מוֹדִים אֲנַחְנוּ לָךְ, שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, אֱלֹהֵי כָל בָּשָׁר, יוֹצְרֵנוּ יוֹצֵר בְּרֵאשִׁית. בְּרָכוֹת וְהוֹדָאוֹת לְשִׁמְךָ הַגָּדוֹל וְהַקָּדוֹשׁ, עַל שֶׁהֶחֱיִיתָנוּ וְקִיַּמְתָּנוּ. כֵּן תְּחַיֵּנוּ וּתְקַיְמֵנוּ, וְתֶאֱסוֹף גָּלֻיּוֹתֵינוּ לְחַצְרוֹת קָדְשֶׁךָ, לִשְׁמֹר חֻקֶּיךָ וְלַעֲשׂוֹת רְצוֹנְךָ וּלְעָבְדְךָ בְלֵבָב שָׁלֵם, עַל שֶׁאֲנַחְנוּ מוֹדִים לָךְ, בָּרוּךְ אֵל הַהוֹדָאוֹת.</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֻּלָּם יִתְבָּרַךְ וְיִתְרוֹמַם וְיִתְנַשֵּׂא שִׁמְךָ מַלְכֵּנוּ תָמִיד לְעוֹלָם וָעֶד.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְכָל</span> הַחַיִּים יוֹדוּךָ סֶּלָה: וִיהַלְלוּ וִיבָרְכוּ אֶת שִׁמְךָ הַגָּדוֹל, בֶּאֱמֶת, לְעוֹלָם כִּי טוֹב, הָאֵל יְשׁוּעָתֵנוּ וְעֶזְרָתֵנוּ סֶלָה, הָאֵל הַטּוֹב. בָּרוּךְ אַתָּה יְהֹוָה, הַטּוֹב שִׁמְךָ וּלְךָ נָאֶה לְהוֹדוֹת:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שִׂים שָׁלוֹם</span> טוֹבָה וּבְרָכָה, חַיִּים, חֵן וָחֶסֶד, וְרַחֲמִים, עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל עַמֶּךָ. בָּרְכֵנוּ אָבִינוּ, כֻּלָּנוּ כְאֶחָד (יַחַד) בְּאוֹר פָּנֶיךָ, כִּי בְאוֹר פָּנֶיךָ נָתַתָּ לָּנוּ יְהֹוָה אֱלֹהֵינוּ, תּוֹרַת חַיִּים, וְאַהֲבָת חֶסֶד, וּצְדָקָה וּבְרָכָה וְרַחֲמִים, וְחַיִּים וְשָׁלוֹם. (וְכָל טוּב) וְטוֹב יִהְיֶה בְּעֵינֶיךָ לְבָרְכֵנוּ וּלְבָרֵךְ אֶת כָּל עַמְּךָ יִשְׂרָאֵל בְּכָל עֵת וּבְכָל שָׁעָה בְּרֹב עֹז וְשָׁלוֹם. בָּרוּךְ אַתָּה יְהֹוָה, הַמְּבָרֵךְ אֶת עַמּוֹ יִשְׂרָאֵל בַּשָּׁלוֹם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִהְיוּ לְרָצוֹן</span> אִמְרֵי פִי וְהֶגְיוֹן לִבִּי לְפָנֶיךָ יְהֹוָה צוּרִי וְגֹאֲלִי:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֱלֹהַי.</span> נְצֹר לְשׁונִי מֵרָע וּשְׂפָתַי מִדַּבֵּר מִרְמָה. וְלִמְקַלְלַי נַפְשִׁי תִדֹּם. וְנַפְשִׁי כֶּעָפָר לַכֹּל תִּהְיֶה. פְּתַח לִבִּי בְּתוֹרָתֶךָ. וְאַחֲרֵי מִצְוֹתֶיךָ תִּרְדֹּף נַפְשִׁי. וְכָל הַקָּמִים וְהַחוֹשְׁבִים עָלַי לְרָעָה. מְהֵרָה הָפֵר עֲצָתָם וְקַלְקֵל מַחֲשַׁבְתָּם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יש אומרים כאן תפילה זו:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהִי</span> רָצוֹן מִלְּפָנֶיךָ יְהֹוָה אֱלֹהַי וֵאלֹהֵי אֲבוֹתַי. שֶׁלֹּא תַעֲלֶה קִנְאַת אָדָם עָלַי וְלֹא קִנְאָתִי עַל אֲחֵרִים. וְשֶׁלֹּא אֶכְעוֹס הַיֹּום וְשֶׁלֹּא אַכְעִיסֶךָ. וְתַצִּילֵנִי מִיֵּצֶר הָרָע. וְתֵן בְּלִבִּי הַכְנָעָה וַעֲנָוָה. מַלְכֵּנוּ וֵאלֹהֵינוּ אֱלֹהַּ כָּל בָּשָׂר. יַחֵד שִׁמְךָ בְּעוֹלָמֶךָ בְּנֵה עִירֶךָ יַסֵּד בֵּיתֶךָ וְשַׁכְלֵל הֵיכָלֶךָ. וְקַבֵּץ קִבּוּץ גָּלֻיּוֹת וּפְדֵה צֹאנֶךָ וְשַׂמַּח עֲדָתֶךָ:
עֲשֵׂה לְמַעַן שְׁמֶךָ. עֲשֵׂה לְמַעַן יְמִינֶךָ. עֲשֵׂה לְמַעַן תּוֹרָתֶךָ. עֲשֵׂה לְמַעַן קְדֻשָּׁתֶךָ. לְמַעַן יֵחָלְצוּן יְדִידֶיךָ הושִׁיעָה יְמִינְךָ וַעֲנֵנִי:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נוהגים לומר כאן פסוק המתחיל וגומר באוֹת שבה מתחיל וגומר שמו הפרטי</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יִהְיוּ לְרָצוֹן</span> אִמְרֵי פִי וְהֶגְיוֹן לִבִּי לְפָנֶיךָ יְהֹוָה צוּרִי וְגֹאֲלִי:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם</span> (בעשי"ת הַשָּׁלום) בִּמְרוֹמָיו. הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל. וְאִמְרוּ אָמֵן:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהִי</span> רָצוֹן מִלְּפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ. שֶׁיִּבָּנֶה בֵּית הַמִּקְדָּשׁ בִּמְהֵרָה בְיָמֵינוּ. וְתֵן חֶלְקֵנוּ בְּתוֹרָתֶךָ: וְשָׁם נַעֲבָדְךָ בְּיִרְאָה כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמוֹנִיוֹת:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעָרְבָה</span> לַיהֹוָה מִנְחַת יְהוּדָה וִירוּשָׁלָיִם. כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמוֹנִיוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קדיש שלם</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא (בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">תִּתְקַבַּל</span> צְלוֹתְהוֹן וּבָעוּתְהוֹן דְכָל בֵּית יִשְׂרָאֵל קֳדָם אֲבוּהוֹן דִּי בִשְׁמַיָּא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בערבית ליל יום טוב, אחרי שמונה עשרה, אם חל בשבת אומרים "ויכולו", והחזן אומר ברכת מעין שבע וקדיש שלם. ואם חל בחול אומרים קדיש שלם. ובליל יום טוב ראשון של פסח, אומרים הלל שלם, ואחר כך אומר החזן קדיש שלם, אם חל בשבת אומרים ויכולו ולאחר מכן הלל שלם</span>

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֵּן נְקַוֶּה לָּךְ יְהֹוָה אֱלֹהֵינוּ, לִרְאוֹת מְהֵרָה בְּתִפְאֶרֶת עֻזָּךְ, לְהַעֲבִיר גִּלּוּלִים מִן הָאָרֶץ, וְהָאֱלִילִים כָּרוֹת יִכָּרֵתוּן, לְתַקֵּן עוֹלָם בְּמַלְכוּת שַׁדַּי, וְכָל בְּנֵי בָשָׂר יִקְרְאוּ בִשְׁמֶךָ, לְהַפְנוֹת אֵלֶיךָ כָּל רִשְׁעֵי אָרֶץ. יַכִּירוּ וְיֵדְעוּ כָּל יוֹשְׁבֵי תֵבֵל, כִּי לְךָ תִּכְרַע כָּל בֶּרֶךְ, תִּשָּׁבַע כָּל לָשׁוֹן. לְפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ יִכְרְעוּ וְיִפֹּלוּ, וְלִכְבוֹד שִׁמְךָ יְקָר יִתֵּנוּ, וִיקַבְּלוּ כֻלָּם אֶת עֹל מַלְכוּתֶךָ, וְתִמְלֹךְ עֲלֵיהֶם מְהֵרָה לְעוֹלָם וָעֶד. כִּי הַמַּלְכוּת שֶׁלְּךָ הִיא, וּלְעוֹלְמֵי עַד תִּמְלֹךְ בְּכָבוֹד. כַּכָּתוּב בְּתוֹרָתָךְ: יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד: וְנֶאֱמַר: וְהָיָה יְהֹוָה לְמֶלֶךְ עַל כָּל הָאָרֶץ בַּיּוֹם הַהוּא יִהְיֶה יְהֹוָה אֶחָד וּשְׁמוֹ אֶחָד:

<h2>סדר הזכרת נשמות</h2>

<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">מי שאין לו אב יאמר זה:</span></span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִזְכּוֹר</span> אֱלֹהִים נִשְׁמַת אָבִי מוֹרִי (פב"פ) שֶׁהָלַךְ לְעוֹלָמוֹ בַּעֲבוּר שֶׁאֶתֵּן בְּלִי נֶדֶר צְדָקָה בַּעֲדוֹ, בִּשְׂכַר זֶה תְּהֵא נַפְשׁוֹ צְרוּרָה בִּצְרוֹר הַחַיִּים עִם נִשְׁמוֹת אַבְרָהָם יִצְחָק וְיַעֲקֹב שָׂרָה רִבְקָה רָחֵל וְלֵאָה וְעִם שְׁאָר צַדִּיקִים וְצִדְקָנִיּוֹת שֶׁבְּגַן עֵדֶן. וְנֹאמַר אָמֵן:

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִזְכּוֹר</span> אֱלֹהִים נִשְׁמַת (פב"פ) וְנִשְׁמַת כָּל קְרוֹבַי וּקְרוֹבוֹתַי דוֹדַי וְדוֹדוֹתַי אַחַי וְאַחְיוֹתַי. הֵן מִצַּד אָבִי וְהֵן מִצַּד אִמִּי. הֵן שֶׁהוּמְתוּ הֵן שֶׁנֶהֱרְגוּ הֵן שֶׁנִשְׁחֲטוּ וְשֶׁנִשְׂרְפוּ וְשֶׁנִטְבְּעוּ וְשֶׁנֶּחְנְקוּ עַל קִדוּשׁ הַשֵּׁם. בַּעֲבוּר שֶׁאֶתֵּן בְּלִי נֶדֶר צְדָקָה בְּעַד הַזְכָּרַת נִשְׁמוֹתֵיהֶם. וּבִשְׂכַר זֶה תִּהְיֶינָה נַפְשׁוֹתֵיהֶם צְרוּרוֹת בִּצְרוֹר הַחַיִּים עִם נִשְׁמוֹת אַבְרָהָם יִצְחָק וְיַעֲקֹב שָׂרָה רִבְקָה רָחֵל וְלֵאָה וְעִם שְׁאָר צַדִּיקִים וְצִדְקָנִיּוֹת שֶׁבְּגַן עֵדֶן. וְנֹאמַר אָמֵן:

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֵל</span> מָלֵא רַחֲמִים שׁוֹכֵן בַּמְּרוֹמִים הַמְצֵא מְנוּחָה נְכוֹנָה עַל כַּנְפֵי הַשְּׁכִינָה בְּמַעֲלוֹת קְדוֹשִׁים וּטְהוֹרִים כְּזֹהַר הָרָקִיעַ מַזְהִירִים אֶת נִשְׁמַת (פב"פ) שֶׁהָלַךְ לְעוֹלָמוֹ בַּעֲבוּר (שפלוני) נָדַב לִצְדָקָה בְּעַד הַזְכָּרַת נִשְׁמָתוֹ, לָכֵן בַּעַל הָרַחֲמִים יַסְתִּירֵהוּ בְּסֵתֶר כְּנָפָיו לְעוֹלָמִים וְיִצְרֹר בִּצְרוֹר הַחַיִּים אֶת נִשְׁמָתוֹ יְהֹוָה הוּא נַחֲלָתוֹ וְיָנוּחַ בְּשָּׁלוֹם עַל מִשְׁכָּבוֹ וְנֹאמַר אָמֵן:

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מִי</span> שֶׁבֵּרַךְ אֲבוֹתֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקֹב הוּא יְבָרֵךְ אֶת ר' (פב"פ) בַּעֲבוּר שֶׁנָּדַב צְדָקָה בְּעַד הַנְּשָׁמוֹת שֶׁהִזְכִּיר הַיּוֹם וְלִכְבוֹד הַמָּקוֹם וְלִכְבוֹד הַתּוֹרָה וְלִכְבוֹד הָרֶגֶל. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אָב</span> הָרַחֲמִים שׁוֹכֵן מְרוֹמִים. בְּרַחֲמָיו הָעֲצוּמִים הוּא יִפְקוֹד בְּרַחֲמִים הַחֲסִידִים וְהַיְשָׁרִים וְהַתְּמִימִים. קְהִלּוֹת הַקֹּדֶשׁ שֶׁמָּסְרוּ נַפְשָׁם עַל קְדֻשַּׁת הַשֵּׁם. הַנֶּאֱהָבִים וְהַנְּעִימִים בְּחַיֵּיהֶם וּבְמוֹתָם לֹא נִפְרָדוּ. מִנְּשָׁרִים קַלּוּ וּמֵאֲרָיוֹת גָּבֵרוּ לַעֲשׂוֹת רְצוֹן קוֹנָם וְחֵפֶץ צוּרָם. יִזְכְּרֵם אֱלֹהֵינוּ לְטוֹבָה עִם שְׁאָר צַדִּיקֵי עוֹלָם. וְיִנְקוֹם בְּיָמֵינוּ לְעֵינֵינוּ נִקְמַת דַּם עֲבָדָיו הַשָּׁפוּךְ. כַּכָּתוּב בְּתוֹרַת משֶׁה אִישׁ הָאֱלֹהִים. הַרְנִינוּ גוֹיִם עַמּוֹ כִּי דַם עֲבָדָיו יִקּוֹם וְנָקָם יָשִׁיב לְצָרָיו וְכִפֶּר אַדְמָתוֹ עַמּוֹ: וְעַל יְדֵי עֲבָדֶיךָ הַנְּבִיאִים כָּתוּב לֵאמֹר. וְנִקֵּיתִי דָּמָם לֹא נִקֵּיתִי וַיהֹוָה שֹׁכֵן בְּצִיּוֹן: וּבְכִתְבֵי הַקֹּדֶשׁ נֶאֱמַר לָמָּה יֹאמְרוּ הַגּוֹיִם אַיֵּה אֱלֹהֵיהֶם. יִוָּדַע בַּגּוֹיִם לְעֵינֵינוּ נִקְמַת דַּם עֲבָדֶיךָ הַשָּׁפוּךְ: וְאוֹמֵר, כִּי דוֹרֵשׁ דָּמִים אוֹתָם זָכָר לֹא שָׁכַח צַעֲקַת עֲנָוִים: וְאוֹמֵר, יָדִין בַּגּוֹיִם מָלֵא גְוִיּוֹת מָחַץ רֹאשׁ עַל אֶרֶץ רַבָּה. מִנַּחַל בַּדֶּרֶךְ יִשְׁתֶּה עַל כֵּן יָרִים רֹאשׁ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יָהּ</span> אֵלִי וְגוֹאֲלִי אֶתְיַצְבָה לִקְרָאתֶךָ. הָיָה וְיִהְיֶה. הָיָה וְהֹוֶה. כָּל גּוֹי אַדְמָתֶךָ:

<h2>תפילת טל ליום ראשון של פסח</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">פותחין הארון:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">ברוך אתה וכו' עד ומושיע ומגן:</span></span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בְּדַעְתּוֹ.</span> אַבִּיעָה חִידוֹת. בְּעַם זוּ בְּזוּ בְּטַל לְהַחֲדוֹת. טַל גֵיא וּדְשָׁאֶיהָ לַחֲדוֹת. דָצִים בְּצִלּוֹ לְהֵחָדוֹת. אוֹת יַלְדוּת טַל לְהָגֵן לְתוֹלָדוֹת:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה מָגֵן אַבְרָהָם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אַתָּה</span> גִבּוֹר לְעוֹלָם אֲדֹנָי. מְחַיֵּה מֵתִים אַתָּה רַב לְהוֹשִׁיעַ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">תְּהוֹמוֹת.</span> הֲדוֹם לִרְסִיסוֹ כְּסוּפִים. וְכָל נְאוֹת דֶשֶׁא לוֹ נִכְסָפִים. טַל זִכְרוֹ גְבוּרוֹת מוֹסִפִים. חָקוּק בְּגִישַׁת מוּסָפִים. טַל לְהַחֲיוֹת בּוֹ נְקוּקֵי סְעִיפִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> תֵּן לְרַצּוֹת אַרְצָךְ. שִׁיתֵנוּ בְרָכָה בְּדִיצָךְ. רוֹב דָגָן וְתִירוֹשׁ בְּהַפְרִיצָךְ. קוֹמֵם עִיר בָּהּ חֶפְצָךְ. בְּטַל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> צַוֵּה שָׁנָה טוֹבָה וּמְעֻטֶרֶת. פְּרִי הָאָרֶץ לְגָאוֹן וּלְתִפְאֶרֶת. עִיר כְּסֻּכָּה נוֹתֶרֶת. שִׂימָהּ בְּיָדְךָ עֲטֶרֶת. בְּטַל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> נוֹפֶת עֲלֵי אֶרֶץ בְּרוּכָה. מִמֶּגֶד שָׁמַיִם שַׂבְּעֵנוּ בְרָכָה. לְהָאִיר מִתּוֹךְ חֲשֵׁכָה. כַּנָה אַחֲרֶיךָ מְשׁוּכָה. בְּטַל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> יַעֲסִיס צוּף הָרִים. טְעֵם בִּמְאוֹדֶךָ מֻבְחָרִים. חֲנוּנֶיךָ חַלֵץ מִמַסְגֵרִים. זִמְרָה נַנְעִים וְקוֹל נָרִים. בְּטַל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> וָשׂוֹבַע מַלֵּא אֲסָמֵינוּ. הֲכָעֵת תְּחַדֵּשׁ יָמֵינוּ. דוֹד כְּעֶרְכְּךָ הַעֲמֵד שְׁמֵנוּ. גַּן רָוֶה שִׂימֵנוּ. בְּטַל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">טַל</span> בּוֹ תְבָרֵךְ מָזוֹן. בְּמַשְׁמַנֵּינוּ אַל יְהִי רָזוֹן. אֲיֻמָה אֲשֶׁר הִסַּעְתָּ כַצֹּאן. אָנָא תָּפֵק לָהּ רָצוֹן. בְּטַל:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">חזן</span></span> שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ מַשִּׁיב הָרוּחַ וּמוֹרִיד הַטָּל:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לִבְרָכָה</span> וְלֹא לִקְלָלָה: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְחַיִּים</span> וְלֹא לַמָּוֶת: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְשׂוֹבַע</span> וְלֹא לְרָזוֹן: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">מכלכל חיים וכו':</span></span>
<h2>תפילת גשם לשמיני עצרת</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">פותחין הארון:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">ברוך אתה וכו' עד ומושיע ומגן:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַף</span> בְּרִי אֻתַּת שֵׁם שַׂר מָטָר. לְהַעֲבִיב וּלְהַעֲנִין לְהָרִיק וּלְהַמְטַר. מַיִם אִבִּים בָּם גֵּיא לַעֲטַר. לְבַל יוּעֲצָרוּ בְּנִשְׁיוֹן שְׁטַר. אֱמוּנִים גְּנוֹן בָּם שׁוֹאֲלֵי מָטָר:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה מָגֵן אַבְרָהָם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אַתָּה</span> גִבּוֹר לְעוֹלָם אֲדֹנָי. מְחַיֵּה מֵתִים אַתָּה רַב לְהוֹשִׁיעַ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יַטְרִיחַ</span> לְפַלֵּג מִפֶּלֶג גֶּשֶׁם. לְמוֹגֵג פְּנֵי נֶשִׁי בְּצַחוֹת לֶשֶׁם. מַיִם לְאַדְרָךְ כִּנִּיתָ בְּרֶשֶׁם. לְהַרְגִיעַ בְּרַעֲפָם לִנְפוּחֵי נֶשֶׁם. לְהַחֲיוֹת מַזְכִּירִים גְּבוּרוֹת הַגָּשֶׁם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר</span> אָב נִמְשַׁךְ אַחֲרֶיךָ כַּמַּיִם. בֵּרַכְתּוֹ כְּעֵץ שָׁתוּל עַל פַּלְגֵי מַיִם. גְּנַנְתּוֹ הִצַּלְתּוֹ מֵאֵשׁ וּמִמַּיִם. דְּרַשְׁתּוֹ בְּזָרְעוֹ עַל כָּל מָיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בַּעֲבוּרוֹ אַל תִּמְנַע מָיִם:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר</span> הַנּוֹלָד בִּבְשׂוֹרַת יֻקַּח נָא מְעַט מַיִם. וְשַׂחְתָּ לְהוֹרוֹ לְשָׁחֲטוֹ לִשְׁפּוֹךְ דָּמוֹ כַּמַּיִם. זִהֵר גַּם הוּא לִשְׁפּוֹךְ לֵב כַּמַּיִם. חָפַר וּמָצָא בְּאֵרוֹת מָיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּצִדְקוֹ חוֹן חַשְׁרַת מָיִם:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר</span> טָעַן מַקְלוֹ וְעָבַר יַרְדֵּן מַיִם. יִחַד לֵב וְגָל אֶבֶן מִפִּי בְאֵר מַיִם. כְּנֶאֱבַק לוֹ שַׂר בָּלוּל מֵאֵשׁ וּמִמַּיִם. לָכֵן הִבְטַחְתּוֹ הֱיוֹת עִמּוֹ בָּאֵשׁ וּבַמָּיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בַּעֲבוּרוֹ אַל תִּמְנַע מָיִם:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר</span> מָשׁוּי בְּתֵיבַת גּוֹמֶא מִן הַמַּיִם. נָמוּ דָּלֹה דָלָה וְהִשְׁקָה צֹאן מַיִם. סְגוּלֶיךָ עֵת צָמְאוּ לְמַּיִם. עַל הַסֶּלַע הָךְ וַיֵּצְאוּ מָיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּצִדְקוֹ חוֹן חַשְׁרַת מָיִם:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר פָּקִיד שָׁתוֹת טוֹבֵל חָמֵשׁ טְבִילוֹת בַּמַּיִם. צוֹעֶה וּמַרְחִיץ כַּפָּיו בְּקִדּוּשׁ מַיִם. קוֹרֵא וּמַזֶּה טָהֳרַת מַיִם. רוּחַק מֵעַם פַּחַז כַּמָּיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בַּעֲבוּרוֹ אַל תִּמְנַע מָיִם:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זְכוֹר</span> שְׁנֵים עָשָׂר שְׁבָטִים שֶׁהֶעֱבַרְתָּ בְּגִזְרַת מַיִם. שֶׁהִמְתַּקְתָּ לָמוֹ מְרִירוּת מַיִם. תּוֹלְדוֹתָם נִשְׁפַּךְ דָּמָם עָלֶיךָ כַּמַּיִם. תֵּפֶן כִּי נַפְשֵׁנוּ אָפְפוּ מָיִם: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּצִדְקָם חוֹן חַשְׁרַת מָיִם:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">חזן</span></span> שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ מַשִּׁיב הָרוּחַ וּמוֹרִיד הַגֶשֶׁם: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לִבְרָכָה</span> וְלֹא לִקְלָלָה: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְחַיִּים</span> וְלֹא לַמָּוֶת: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְשׂוֹבַע</span> וְלֹא לְרָזוֹן: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אמן:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">מכלכל חיים וכו':</span></span></span>
<h2>מוסף לשלש רגלים</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כִּי</span> שֵׁם יְהֹוָה אֶקְרָא הָבוּ גֹדֶל לֵאלֹהֵינוּ: אֲדֹנָי, שְׂפָתַי תִּפְתָּח וּפִי יַגִּיד תְּהִלָּתֶךָ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, אֱלֹהֵי אַבְרָהָם, אֱלֹהֵי יִצְחָק וֵאלֹהֵי יַעֲקֹב. הָאֵל הַגָּדוֹל, הַגִּבּוֹר וְהַנּוֹרָא, אֵל עֶלְיוֹן, גּוֹמֵל חֲסָדִים טוֹבִים, וְקוֹנֵה הַכֹּל, וְזוֹכֵר חַסְדֵי אָבוֹת, וּמֵבִיא גוֹאֵל לִבְנֵי בְנֵיהֶם לְמַעַן שְׁמוֹ, בְּאַהֲבָה. מֶלֶךְ עוֹזֵר וּמוֹשִׁיעַ וּמָגֵן, בָּרוּךְ אַתָּה יְהֹוָה, מָגֵן אַבְרָהָם:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַתָּה</span> גִּבּוֹר לְעוֹלָם, אֲדֹנָי. מְחַיֵּה מֵתִים אַתָּה, רַב לְהוֹשִׁיעַ.
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בקיץ:</span> מוֹרִיד הַטָּל. <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחורף:</span> מַשִּׁיב הָרוּחַ וּמוֹרִיד הַגֶּשֶׁם.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מְכַלְכֵּל</span> חַיִּים בְּחֶסֶד, מְחַיֵּה מֵתִים בְּרַחֲמִים רַבִּים, סוֹמֵךְ נוֹפְלִים, וְרוֹפֵא חוֹלִים, וּמַתִּיר אֲסוּרִים, וּמְקַיֵּם אֱמוּנָתוֹ לִישֵׁנֵי עָפָר. מִי כָמוֹךָ בַעַל גְּבוּרוֹת, וּמִי דוֹמֶה לָּךְ, מֶלֶךְ מֵמִית וּמְחַיֶּה וּמַצְמִיחַ יְשׁוּעָה. וְנֶאֱמָן אַתָּה לְהַחֲיוֹת מֵתִים. בָּרוּךְ אַתָּה יְהֹוָה, מְחַיֵּה הַמֵּתִים:
<span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><span style="text-align: center;"><i>כתר</i></span><br/><span style="color:RGB(0,127,127); font-family: arial; <span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">בחזרת הש"ץ אומרים כאן ותערב:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חו"ק:</span></span> וְתֵעָרֵב לְפָנֶיךָ עֲתִירָתֵנוּ כְּעוֹלָה וּכְקָרְבָּן. אָנָא רַחוּם. בְּרַחֲמֶיךָ הָרַבִּים. הָשֵׁב שְׁכִינָתְךָ לְצִיּוֹן עִירְךָ וְסֵדֶר הָעֲבוֹדָה לִירוּשָׁלָיִם. וְתֶחֱזֶינָה עֵינֵינוּ בְּשׁוּבְךָ לְצִיּוֹן בְּרַחֲמִים. וְשָׁם נַעֲבָדְךָ בְּיִרְאָה כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמוֹנִיּוֹת: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">והחזן מסיים:</span></span> בָּרוּךְ אַתָּה יְהֹוָה. שֶׁאוֹתְךָ לְבַדְּךָ בְּיִרְאָה נַעֲבֹד:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְתֶחֱזֶינָה</span> עֵינֵינוּ בְּשׁוּבְךָ לְצִיּוֹן בְּרַחֲמִים. בָּרוּךְ אַתָּה יְהֹוָה, הַמַּחֲזִיר שְׁכִינָתוֹ לְצִיּוֹן:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מוֹדִים</span> אֲנַחְנוּ לָךְ, שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ לְעוֹלָם וָעֶד, צוּרֵנוּ צוּר חַיֵּינוּ וּמָגֵן יִשְׁעֵנוּ אַתָּה הוּא לְדֹר וָדֹר. נוֹדֶה לְּךָ וּנְסַפֵּר תְּהִלָּתֶךָ, עַל חַיֵּינוּ הַמְּסוּרִים בְּיָדֶיךָ, וְעַל נִשְׁמוֹתֵינוּ הַפְּקוּדוֹת לָךְ, וְעַל נִסֶּיךָ שֶׁבְּכָל יוֹם עִמָּנוּ, וְעַל נִפְלְאוֹתֶיךָ וְטוֹבוֹתֶיךָ שֶׁבְּכָל עֵת, עֶרֶב וָבֹקֶר וְצָהֳרָיִם. הַטּוֹב, כִּי לֹא כָלוּ רַחֲמֶיךָ, הַמְּרַחֵם, כִּי לֹא תַמּוּ חֲסָדֶיךָ, כִּי מֵעוֹלָם קִוִּינוּ לָךְ:
<span style="padding:10; margin:10; background-color=RGB(244,244,244);text-shadow:0 0px 0 white;"><span style="text-align: center;"><i>מודים דרבנן</i></span><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בחזרת הש"ץ</span></span> כשהוא אוֹמר מודים, אוֹמרים הקהל:</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מוֹדִים אֲנַחְנוּ לָךְ, שָׁאַתָּה הוּא יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, אֱלֹהֵי כָל בָּשָׁר, יוֹצְרֵנוּ יוֹצֵר בְּרֵאשִׁית. בְּרָכוֹת וְהוֹדָאוֹת לְשִׁמְךָ הַגָּדוֹל וְהַקָּדוֹשׁ, עַל שֶׁהֶחֱיִיתָנוּ וְקִיַּמְתָּנוּ. כֵּן תְּחַיֵּנוּ וּתְקַיְמֵנוּ, וְתֶאֱסוֹף גָּלֻיּוֹתֵינוּ לְחַצְרוֹת קָדְשֶׁךָ, לִשְׁמֹר חֻקֶּיךָ וְלַעֲשׂוֹת רְצוֹנְךָ וּלְעָבְדְךָ בְלֵבָב שָׁלֵם, עַל שֶׁאֲנַחְנוּ מוֹדִים לָךְ, בָּרוּךְ אֵל הַהוֹדָאוֹת.</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֻּלָּם יִתְבָּרַךְ וְיִתְרוֹמַם וְיִתְנַשֵּׂא שִׁמְךָ מַלְכֵּנוּ תָמִיד לְעוֹלָם וָעֶד.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְכָל</span> הַחַיִּים יוֹדוּךָ סֶּלָה: וִיהַלְלוּ וִיבָרְכוּ אֶת שִׁמְךָ הַגָּדוֹל, בֶּאֱמֶת, לְעוֹלָם כִּי טוֹב, הָאֵל יְשׁוּעָתֵנוּ וְעֶזְרָתֵנוּ סֶלָה, הָאֵל הַטּוֹב. בָּרוּךְ אַתָּה יְהֹוָה, הַטּוֹב שִׁמְךָ וּלְךָ נָאֶה לְהוֹדוֹת:

<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִהְיוּ לְרָצוֹן</span> אִמְרֵי פִי וְהֶגְיוֹן לִבִּי לְפָנֶיךָ יְהֹוָה צוּרִי וְגֹאֲלִי:

<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">יש אומרים כאן תפילה זו:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהִי</span> רָצוֹן מִלְּפָנֶיךָ יְהֹוָה אֱלֹהַי וֵאלֹהֵי אֲבוֹתַי. שֶׁלֹּא תַעֲלֶה קִנְאַת אָדָם עָלַי וְלֹא קִנְאָתִי עַל אֲחֵרִים. וְשֶׁלֹּא אֶכְעוֹס הַיֹּום וְשֶׁלֹּא אַכְעִיסֶךָ. וְתַצִּילֵנִי מִיֵּצֶר הָרָע. וְתֵן בְּלִבִּי הַכְנָעָה וַעֲנָוָה. מַלְכֵּנוּ וֵאלֹהֵינוּ אֱלֹהַּ כָּל בָּשָׂר. יַחֵד שִׁמְךָ בְּעוֹלָמֶךָ בְּנֵה עִירֶךָ יַסֵּד בֵּיתֶךָ וְשַׁכְלֵל הֵיכָלֶךָ. וְקַבֵּץ קִבּוּץ גָּלֻיּוֹת וּפְדֵה צֹאנֶךָ וְשַׂמַּח עֲדָתֶךָ:
עֲשֵׂה לְמַעַן שְׁמֶךָ. עֲשֵׂה לְמַעַן יְמִינֶךָ. עֲשֵׂה לְמַעַן תּוֹרָתֶךָ. עֲשֵׂה לְמַעַן קְדֻשָּׁתֶךָ. לְמַעַן יֵחָלְצוּן יְדִידֶיךָ הושִׁיעָה יְמִינְךָ וַעֲנֵנִי:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נוהגים לומר כאן פסוק המתחיל וגומר באוֹת שבה מתחיל וגומר שמו הפרטי</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יִהְיוּ לְרָצוֹן</span> אִמְרֵי פִי וְהֶגְיוֹן לִבִּי לְפָנֶיךָ יְהֹוָה צוּרִי וְגֹאֲלִי:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם</span> (בעשי"ת הַשָּׁלום) בִּמְרוֹמָיו. הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל. וְאִמְרוּ אָמֵן:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהִי</span> רָצוֹן מִלְּפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ. שֶׁיִּבָּנֶה בֵּית הַמִּקְדָּשׁ בִּמְהֵרָה בְיָמֵינוּ. וְתֵן חֶלְקֵנוּ בְּתוֹרָתֶךָ: וְשָׁם נַעֲבָדְךָ בְּיִרְאָה כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמוֹנִיוֹת:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעָרְבָה</span> לַיהֹוָה מִנְחַת יְהוּדָה וִירוּשָׁלָיִם. כִּימֵי עוֹלָם וּכְשָׁנִים קַדְמוֹנִיוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קדיש שלם</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא (בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">תִּתְקַבַּל</span> צְלוֹתְהוֹן וּבָעוּתְהוֹן דְכָל בֵּית יִשְׂרָאֵל קֳדָם אֲבוּהוֹן דִּי בִשְׁמַיָּא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<h2>קוה</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">קַוֵּה</span> אֶל יְהֹוָה חֲזַק וְיַאֲמֵץ לִבֶּךָ וְקַוֵּה אֶל יְהֹוָה: אֵין קָדוֹשׁ כַּיְהֹוָה כִּי אֵין בִּלְתֶּךָ וְאֵין צוּר כֵּאלֹהֵינוּ: כִּי מִי אֱלוֹהַּ מִבַּלְעֲדֵי יְהֹוָה וּמִי צוּר זוּלָתִי אֱלֹהֵינוּ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֵין כֵּאלֹהֵינוּ</span>, אֵין כַּאדוֹנֵנוּ, אֵין כְּמַלְכֵּנוּ, אֵין כְּמוֹשִׁיעֵנוּ: מִי כֵאלֹהֵינוּ, מִי כַאדוֹנֵנוּ, מִי כְמַלְכֵּנוּ, מִי כְמוֹשִׁיעֵנוּ: נוֹדֶה לֵאלֹהֵינוּ, נוֹדֶה לַאדוֹנֵנוּ, נוֹדֶה לְמַלְכֵּנוּ, נוֹדֶה לְמוֹשִׁיעֵנוּ: בָּרוּךְ אֱלֹהֵינוּ, בָּרוּךְ אֲדוֹנֵנוּ, בָּרוּךְ מַלְכֵּנוּ, בָּרוּךְ מוֹשִׁיעֵנוּ: אַתָּה הוּא אֱלֹהֵינוּ, אַתָּה הוּא אֲדוֹנֵנוּ, אַתָּה הוּא מַלְכֵּנוּ, אַתָּה הוּא מוֹשִׁיעֵנוּ אַתָּה תוֹשִׁיעֵנוּ: אַתָּה תָקוּם תְּרַחֵם צִיּוֹן, כִּי עֵת לְחֶנְנָהּ, כִּי בָא מוֹעֵד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">פִּטּוּם</span> הַקְּטֹרֶת הַצֳּרִי. וְהַצִּפֹּרֶן. הַחֶלְבְּנָה. וְהַלְּבוֹנָה. מִשְׁקַל שִׁבְעִים שִׁבְעִים מָנֶה. מוֹר. וּקְצִיעָה. שִׁבֹּלֶת נֵרְדְּ. וְכַרְכֹּם. מִשְׁקַל שִׁשָּׁה עָשָׂר שִׁשָּׁה עָשָׂר מָנֶה. הַקֹּשְׁטְ שְׁנֵים עָשָׂר. וְקִלּוּפָה שְׁלֹשָׁה. וְקִנָּמוֹן תִּשְׁעָה. בֹּרִית כַּרְשִׁינָה תִּשְׁעָה קַבִּין. יֵין קַפְרִיסִין סְאִין תְּלָתָא וְקַבִּין תְּלָתָא. וְאִם אֵין לוֹ יֵין קַפְרִיסִין מֵבִיא חֲמַר חִוַּרְיָן עַתִּיק. מֶלַח סְדוֹמִית רוֹבַע מַעֲלֶה עָשָׁן כָּל שֶׁהוּא. רַבִּי נָתָן הַבַּבְלִי אוֹמֵר. אַף כִּפַּת הַיַּרְדֵּן כָּל שֶׁהוּא. וְאִם נָתַן בָּהּ דְּבַשׁ פְּסָלָהּ. וְאִם חִסַּר אַחַת מִכָּל סַמָּנֶיהָ חַיָּב מִיתָה: רַבָּן שִׁמְעוֹן בֶּן גַּמְלִיאֵל אוֹמֵר. הַצֳּרִי אֵינוֹ אֶלָּא שְׂרָף הַנּוֹטֵף מֵעֲצֵי הַקְּטָף. בֹּרִית כַּרְשִׁינָה שֶׁשָּׁפִין בָּהּ אֶת הַצִּפֹּרֶן. כְּדֵי שֶׁתְּהֵא נָאָה. יֵין קַפְרִיסִין שֶׁשּׁוֹרִין בּוֹ אֶת הַצִּפֹּרֶן. כְּדֵי שֶׁתְּהֵא עַזָּה. וַהֲלֹא מֵי רַגְלַיִם יָפִין לָהּ. אֶלָּא שֶׁאֵין מַכְנִיסִין מֵי רַגְלַיִם בַּמִּקְדָּשׁ מִפְּנֵי הַכָּבוֹד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַשִׁיר</span> שֶׁהַלְוִיִּם הָיוּ אוֹמְרִים בְּבֵית הַמִּקְדָּשׁ: בַּיּוֹם הָרִאשׁוֹן הָיוּ אוֹמְרִים. לַיהֹוָה הָאָרֶץ וּמְלוֹאָהּ תֵּבֵל וְיוֹשְׁבֵי בָהּ: בַּשֵּׁנִי הָיוּ אוֹמְרִים. גָּדוֹל יְהֹוָה וּמְהֻלָּל מְאֹד בְּעִיר אֱלֹהֵינוּ הַר קָדְשׁוֹ: בַּשְּׁלִישִׁי הָיוּ אוֹמְרִים. אֱלֹהִים נִצָּב בַּעֲדַת אֵל בְּקֶרֶב אֱלֹהִים יִשְׁפֹּט: בָּרְבִיעִי הָיוּ אוֹמְרִים. אֵל נְקָמוֹת יְהֹוָה אֵל נְקָמוֹת הוֹפִיעַ: בַּחֲמִישִׁי הָיוּ אוֹמְרִים. הַרְנִינוּ לֵאלֹהִים עוּזֵּנוּ הָרִיעוּ לֵאלֹהֵי יַעֲקֹב: בַּשִּׁשִּׁי הָיוּ אוֹמְרִים. יְהֹוָה מָלָךְ גֵּאוּת לָבֵשׁ לָבֵשׁ יְהֹוָה עֹז הִתְאַזָּר אַף תִּכּוֹן תֵּבֵל בַּל תִּמּוֹט: בַּשַּׁבָּת הָיוּ אוֹמְרִים. מִזְמוֹר שִׁיר לְיוֹם הַשַּׁבָּת. מִזְמוֹר שִׁיר לֶעָתִיד לָבֹא לְיוֹם שֶׁכֻּלּוֹ שַׁבָּת וּמְנוּחָה לְחַיֵּי הָעוֹלָמִים:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">תָּנָא</span> דְבֵי אֵלִיָּהוּ. כָּל הַשּׁוֹנֶה הֲלָכוֹת בְּכָל יוֹם. מֻבְטָח לוֹ שֶׁהוּא בֶּן עוֹלָם הַבָּא. שֶׁנֶּאֱמַר. הֲלִיכוֹת עוֹלָם לוֹ. אַל תִּקְרֵי הֲלִיכוֹת אֶלָּא הֲלָכוֹת:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָמַר</span> רַבִּי אֶלְעָזָר אָמַר רַבִּי חֲנִינָא. תַּלְמִידֵי חֲכָמִים מַרְבִּים שָׁלוֹם בָּעוֹלָם. שֶׁנֶּאֱמַר. וְכָל בָּנַיִךְ לִמּוּדֵי יְהֹוָה וְרַב שְׁלוֹם בָּנָיִךְ. אַל תִּקְרֵי בָּנָיִךְ. אֶלָּא בּוֹנָיִךְ: שָׁלוֹם רָב לְאֹהֲבֵי תוֹרָתֶךָ. וְאֵין לָמוֹ מִכְשׁוֹל: יְהִי שָׁלוֹם בְּחֵילֵךְ. שַׁלְוָה בְּאַרְמְנוֹתָיִךְ: לְמַעַן אַחַי וְרֵעָי אֲדַבְּרָה נָּא שָׁלוֹם בָּךְ: לְמַעַן בֵּית יְהֹוָה אֱלֹהֵינוּ אֲבַקְשָׁה טוֹב לָךְ: יְהֹוָה עֹז לְעַמּוֹ יִתֵּן. יְהֹוָה יְבָרֵךְ אֶת עַמּוֹ בַשָּׁלוֹם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קדיש דרבנן</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא, בְּעָלְמָא דִּי בְרָא כִרְעוּתֵהּ, וְיַמְלִיךְ מַלְכוּתֵהּ, וְיַצְמַח פּוּרְקָנֵהּ וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא.</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְבָּרֵךְ,</span> וְיִשְׁתַּבַּח, וְיִתְפָּאֵר, וְיִתְרוֹמָם, וְיִתְנַשֵּׂא, וְיִתְהַדָּר, וְיִתְעַלֶּה, וְיִתְהַלָּל, שְׁמֵהּ דְּקֻדְשָׁא בְּרִיךְ הוּא. לְעֵלָּא מִן כָּל בִּרְכָתָא (בעשי"ת לְעֵלָּא וּלְעֵלָּא מִכָּל בִּרְכָתָא) וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עַל יִשְׂרָאֵל</span> וְעַל רַבָּנָן, וְעַל תַּלְמִידֵיהוֹן וְעַל כָּל תַּלְמִידֵי תַלְמִידֵיהוֹן, וְעַל כָּל מָאן דְּעָסְקִין בְּאוֹרַיְתָא, דִּי בְאַתְרָא הָדֵין וְדִי בְכָל אֲתַר וַאֲתַר. יְהֵא לְהוֹן וּלְכוֹן שְׁלָמָא רַבָּא חִנָּא וְחִסְדָּא וְרַחֲמִין וְחַיִּין אֲרִיכִין וּמְזוֹנָא רְוִיחָא וּפוּרְקָנָא מִן קֳדָם אֲבוּהוֹן דְּבִשְׁמַיָּא וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרְכוּ</span> אֶת יְהֹוָה הַמְּבֹרָךְ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קהל וחזן:</span> <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> יְהֹוָה הַמְּבֹרָךְ לְעוֹלָם וָעֵד:
<h2>עלינו לשבח</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עָלֵינוּ</span> לְשַׁבֵּחַ לַאֲדוֹן הַכֹּל, לָתֵת גְּדֻלָּה לְיוֹצֵר בְּרֵאשִׁית, שֶׁלֹּא עָשָׂנוּ כְּגוֹיֵי הָאֲרָצוֹת, וְלֹא שָׂמָנוּ כְּמִשְׁפְּחוֹת הָאֲדָמָה. שֶׁלֹּא שָׂם חֶלְקֵנוּ כָּהֶם, וְגוֹרָלֵנוּ כְּכָל הֲמוֹנָם, שֶׁהֵם מִשְׁתַּחֲוִים לְהֶבֶל וָרִיק, וּמִתְפַּלְּלִים אֶל אֵל לֹא יוֹשִׁיעַ. וַאֲנַחְנוּ כּוֹרְעִים וּמִשְׁתַּחֲוִים וּמוֹדִים, לִפְנֵי מֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא, שֶׁהוּא נוֹטֶה שָׁמַיִם וְיֹסֵד אָרֶץ, וּמוֹשַׁב יְקָרוֹ בַּשָּׁמַיִם מִמַּעַל, וּשְׁכִינַת עֻזּוֹ בְּגָבְהֵי מְרוֹמִים. הוּא אֱלֹהֵינוּ אֵין עוֹד, אֱמֶת מַלְכֵּנוּ, אֶפֶס זוּלָתוֹ. כַּכָּתוּב בַּתּוֹרָתוֹ: וְיָדַעְתָּ הַיּוֹם וַהֲשֵׁבֹתָ אֶל לְבָבֶךָ, כִּי יְהֹוָה הוּא הָאֱלֹהִים בַּשָּׁמַיִם מִמַּעַל וְעַל הָאָרֶץ מִתָּחַת אֵין עוֹד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֵּן נְקַוֶּה לָּךְ יְהֹוָה אֱלֹהֵינוּ, לִרְאוֹת מְהֵרָה בְּתִפְאֶרֶת עֻזָּךְ, לְהַעֲבִיר גִּלּוּלִים מִן הָאָרֶץ, וְהָאֱלִילִים כָּרוֹת יִכָּרֵתוּן, לְתַקֵּן עוֹלָם בְּמַלְכוּת שַׁדַּי, וְכָל בְּנֵי בָשָׂר יִקְרְאוּ בִשְׁמֶךָ, לְהַפְנוֹת אֵלֶיךָ כָּל רִשְׁעֵי אָרֶץ. יַכִּירוּ וְיֵדְעוּ כָּל יוֹשְׁבֵי תֵבֵל, כִּי לְךָ תִּכְרַע כָּל בֶּרֶךְ, תִּשָּׁבַע כָּל לָשׁוֹן. לְפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ יִכְרְעוּ וְיִפֹּלוּ, וְלִכְבוֹד שִׁמְךָ יְקָר יִתֵּנוּ, וִיקַבְּלוּ כֻלָּם אֶת עֹל מַלְכוּתֶךָ, וְתִמְלֹךְ עֲלֵיהֶם מְהֵרָה לְעוֹלָם וָעֶד. כִּי הַמַּלְכוּת שֶׁלְּךָ הִיא, וּלְעוֹלְמֵי עַד תִּמְלֹךְ בְּכָבוֹד. כַּכָּתוּב בְּתוֹרָתָךְ: יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד: וְנֶאֱמַר: וְהָיָה יְהֹוָה לְמֶלֶךְ עַל כָּל הָאָרֶץ בַּיּוֹם הַהוּא יִהְיֶה יְהֹוָה אֶחָד וּשְׁמוֹ אֶחָד:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קדיש יתום</span>
<br/> <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא (בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<h2>סדר קידושא רבא לשלש רגלים ולראש השנה</h2>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">(<span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לשבת</span></span>) <span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אִם</span> תָּשִׁיב מִשַּׁבָּת רַגְלֶךָ עֲשׂוֹת חֲפָצֶךָ בְּיוֹם קָדְשִׁי. וְקָרָאתָ לַשַּׁבָּת עֹנֶג לִקְדוֹשׁ יְהֹוָה מְכֻבָּד. וְכִבַּדְתּוֹ מֵעֲשׂוֹת דְּרָכֶיךָ מִמְּצוֹא חֶפְצְךָ וְדַבֵּר דָּבָר: אָז תִּתְעַנַּג עַל יְהֹוָה וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ. וְהַאֲכַלְתִּיךָ נַחֲלַת יַעֲקֹב אָבִיךָ. כִּי פִּי יְהֹוָה דִּבֵּר:
