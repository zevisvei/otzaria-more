<h1>ברכות</h1>

<h2>ברכת שהחיינו</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הלובש מלבוש חדש וכן האוכל פרי חדש הגדל בזמן קבוע בשנה ומתחדש משנה לשנה מברך:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם שֶׁהֶחֱיָנוּ וְקִיְּמָנוּ וְהִגִּיעָנוּ לַזְמַן הַזֶּה:
<h2>ריח בשמים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על ריח בשמים מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא מִינֵי בְשָׂמִים:
<h2>ריח טוב שבעשבים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על ריח טוב שבעשבים מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא עִשְׂבֵּי בְשָׂמִים:
<h2>ריח טוב שבעצים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על ריח טוב שבעצים ושיחים מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא עֲצֵי בְשָׂמִים:
<h2>ריח טוב שבפירות</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על ריח טוב שבפירות מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, הַנּוֹתֵן רֵיחַ טוֹב בַּפֵּירוֹת:
<h2>שמן אפרסמון</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על שמן אפרסמון מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא שֶׁמֶן עָרֵב:
<h2>בִּרְכַּת ברקים ורעמים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על הברקים מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, עוֹשֶׂה מַעֲשֶׂה בְּרֵאשִׁית:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על הרעמים מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁכֹּחוֹ וּגְבוּרָתוֹ מָלֵא עוֹלָם:
<h2>בִּרְכַּת הקשת</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה קשת צריך לברך ברכה זו:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, זוֹכֵר הַבְּרִית נֶאֱמָן בִּבְרִיתוֹ וְקַיָּם בְּמַאֲמָרוֹ:
<h2>הים הגדול</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה את הים הגדול מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, עֹשֶׂה אֶת הַיָּם הַגָּדוֹל:
<h2>בִּרְכַּת הָאִילָנוֹת</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה אילני מאכל (לפחות שני אילנות) בפריחתם בימי ניסן מברך פעם אחת בשנה:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁלֹּא חִסֵּר בְּעוֹלָמוֹ כְּלוּם, וּבָרָא בוֹ בְּרִיוֹת טוֹבוֹת וְאִילָנוֹת טוֹבוֹת, לֵהָנוֹת בָּהֶם בְּנֵי אָדָם:
<h2>ברכות שונות</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה חכם גדול מישראל אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁחָלַק מֵחָכְמָתוֹ לִירֵיאָיו:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה חכם גדול מאומות העולם אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁנָתַן מֵחָכְמָתוֹ לְבָשָׂר וָדָם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה מלך מישראל אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁחָלַק מִכְּבוֹדוֹ לִירֵיאָיו:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה מלך מאומות העולם אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁנָתַן מִכְּבוֹדוֹ לְבָשָׂר וָדָם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה בתי כנסיות בחורבנן מברך דיין האמת, ובישובן מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, מַצִיב גְּבוּל אַלְמָנָה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה מקום שנעשו בו ניסים לישראל, כגון מעברות הירדן וחומות יריחו אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁעָשָׂה נִיסִּים לַאֲבוֹתֵינוּ בַּמָּקוֹם הַזֶּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה מקום שנעשה בו נס לו, או לרבו, או לאבותיו, או לאבותינו בכלל, מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁעָשָׂה לִי (לְרַבִּי, לַאֲבוֹתַי, לַאֲבוֹתֵינוּ) נֵס בַּמָּקוֹם הַזֶּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשרואין בריה משונה, כגון גמד, וכושי וכדומה אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, מְשַׁנֶּה הַבְּרִיּוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה חבירו החביב לו אחר ל' יום מברך שהחיינו, ואחר י"ב חודש מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, מְחַיֶה הַמֵּתִים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הרואה אילנות טובים ובריות נאות, מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁכָּכָה לוֹ בְּעוֹלָמוֹ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מי שלא ראה קברי ישראל שלשים יום יברך בראייתם:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר יָצַר אֶתְכֶם בְּדִּין, וְזָן וְכִלְכֵּל אֶתְכֶם בְּדִּין, וְהֵמִית אֶתְכֶם בְּדִּין, וְיוֹדֵעַ מִסְפַּר כֻּלְּכֶם בְּדִין, וְעָתִיד לְהַחֲזִיר וּלְהַחֲיוֹתכֶם בַּדִּין. בָּרוּךְ אַתָּה יְהֹוָה מְחַיֵּה הַמֵּתִים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בפעם הראשונה שיורדים גשמים הרבה בארץ ישראל, ובחו"ל לאחר עצירת גשמים שהיה העולם בצער מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מוֹדִים</span> אֲנַחְנוּ לָךְ יְהֹוָה אֱלֹהֵינוּ על כל טפה וטפה שהורדת לנו וְאִלּוּ פִינוּ מָלֵא שִׁירָה כַּיָּם. וּלְשׁוֹנֵנוּ רִנָּה כַּהֲמוֹן גַּלָּיו. וְשִׂפְתוֹתֵינוּ שֶׁבַח כְּמֶרְחֲבֵי רָקִיעַ. וְעֵינֵינוּ מְאִירוֹת כַּשֶּׁמֶשׁ וְכַיָּרֵחַ. וְיָדֵינוּ פְרוּשׂוֹת כְּנִשְׁרֵי שָׁמָיִם. וְרַגְלֵינוּ קַלּוֹת כָּאַיָּלוֹת. אֵין אֲנַחְנוּ מַסְפִּיקִים לְהוֹדוֹת לְךָ יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ. וּלְבָרֵךְ אֶת שִׁמְךָ מַלְכֵּנוּ. עַל אַחַת מֵאָלֶף אֶלֶף אַלְפֵי אֲלָפִים וְרִבֵּי רְבָבוֹת פְּעָמִים. הַטּוֹבוֹת נִסִּים וְנִפְלָאוֹת שֶׁעָשִׂיתָ עִם אֲבוֹתֵינוּ וְעִמָּנוּ: מִלְּפָנִים מִמִּצְרַיִם גְּאַלְתָּנוּ יְהֹוָה אֱלֹהֵינוּ. וּמִבֵּית עֲבָדִים פְּדִיתָנוּ. בְּרָעָב זַנְתָּנוּ. וּבְשָׂבָע כִּלְכַּלְתָּנוּ. מֵחֶרֶב הִצַּלְתָּנוּ. וּמִדֶּבֶר מִלַּטְתָּנוּ. וּמֵחֳלָיִם רָעִים וְרַבִּים וְנֶאֱמָנִים דִּלִּיתָנוּ: עַד הֵנָּה עֲזָרוּנוּ רַחֲמֶיךָ. וְלֹא עֲזָבוּנוּ חֲסָדֶיךָ יְהֹוָה אֱלֹהֵינוּ. וְאַל תִּטְּשֵׁנוּ יְהֹוָה אֱלֹהֵינוּ לָנֶצַח: עַל כֵּן אֵבָרִים שֶׁפִּלַּגְתָּ בָּנוּ. וְרוּחַ וּנְשָׁמָה שֶׁנָּפַחְתָּ בְּאַפֵּינוּ. וְלָשׁוֹן אֲשֶׁר שַׂמְתָּ בְּפִינוּ. הֵן הֵם. יוֹדוּ וִיבָרְכוּ וִישַׁבְּחוּ וִיפָאֲרוּ וִישׁוֹרְרוּ וִירוֹמְמוּ וְיַעֲרִיצוּ וְיַקְדִּישׁוּ וְיַמְלִיכוּ אֶת שִׁמְךָ מַלְכֵּנוּ. בָּרוּךְ אַתָּה יְהֹוָה אֵל רוֹב הַהוֹדָאוֹת:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ובא"י אם השנים כתיקונן יברך בלא שם ומלכות (עי' מ"ב סי' רכא סק"א ובבה"ל שם):</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">על פדיון פטר חמור יברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו, וְצִוָּנוּ עַל פִּדְיוֹן פֶּטֶר חֲמוֹר:
<h2>בִּרְכַּת טְבִילַת כְּלִים</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">כשטובל כלים חדשים מברך:</span>

<h2>בִּרְכַּת הַמְזוּזָה</h2>

<h2>סֵדֶר הַפְרָשַׁת חַלָּה</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">א. בזמנינו שהחלה אסורה באכילה לכהנים מפני הטומאה, לֹא נותנים אוֹתה לכהנים אלא שורפים אוֹתה. ומפרישים רק כל שהוא מבלי כל שיעור. ב. את החלה שמפרישים צריך לשרוף, ואסור לזורקה לאשפה ולזלזל בקדושתה. ורק לאחר ששרפה אוֹתה מותר לזורקה לאשפה, כיון שכבר נעשה מצוותה, ואין בזה משום זלזול. ואם אין אפשרות לשורפה, תכרוך את החלה פעמיים בנייר ותכניסה לפח הזבל. ג. טוב להמתין מלהפריש חלה עד לאחר גמר הלישה - כשתֵעשה העיסה גוש אחד. ואם לֹא הפרישה מן העיסה ואפתה - מפרישה מן הפת לאחר האפיה. ד. אין חיוב הפרשת חלה אלא מעיסה העומדת לאפיה. אבל העושה עיסה כדי לבשלה אוֹ לטגנה, מפרישה ממנה חלה בלי ברכה, ורק אם רוצה לאפות קצת ממנה - מפרישה ממנה חלה בברכה. ה. שעור הפרשת חלה בברכה הוא כשיש בקמח שיעור של 1560 גר', וממדת חסידות טוב להפריש חלה בלי ברכה משיעור קילו וחצי לערך (שו"ת חזו"ע עמ' תקיח). ולשיטת החזו"א השעור הוא 2250 גר'.</span>

<h2>תפילת הדרך לטסים באוירון</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">באוירון מוסיפים תפילה זו:</span></span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְיְהִי</span> רָצוֹן מִלְפָנֶיךָ שֶׁתִּשְׁמְרֵנוּ מֵרוּחַ סוֹעָה וְסַעַר וּמִצַּעַר. וְתוֹצִיא רוּחַ טוֹב מֵאוֹצְרוֹתֶיךָ לְהַנְהִיג אֶת סְפִינַת הָאֲוִיר, וְחַזֵּק כָּל מַנְהִיגֶיהָ שֶׁיִּנְהֲגוּ אוֹתָהּ כָּרָאוּי וְהַגִּיעֵנוּ אֶל מְחוֹז חֶפְצֵנוּ לְחַיִּים וּלְשָׁלוֹם בְּלִי שׁוּם פַּחַד. שָׁמְרָה נַפְשִׁי וְהַצִּילֵנִי, אַל אֵבוֹשׁ כִּי חָסִיתִי בָּךְ, וַאֲנַחְנוּ נְבָרֵךְ יָהּ מֵעַתָּה וְעַד עוֹלָם הַלְלוּיָהּ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ואחר כך יאמר <span style="font-weight:bold;text-shadow:0 0px 0 white;">ויעקב הלך לדרכו</span> עד <span style="font-weight:bold;text-shadow:0 0px 0 white;">שכינת אל</span> כדלעיל:</span>
