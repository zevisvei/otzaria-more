<h1>תפילת ערבית לימי החול</h1>

<h2>תפילת ערבית</h2>
<h3>שיר המעלות</h3>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שִׁיר הַמַּעֲלוֹת.</span> הִנֵּה בָּרְכוּ אֶת יְהֹוָה כָּל עַבְדֵי יְהֹוָה. הָעֹמְדִים בְּבֵית יְהֹוָה בַּלֵּילוֹת: שְׂאוּ יְדֵיכֶם קוֹדֶשׁ וּבָרְכוּ אֶת יְהֹוָה: יְבָרֶכְךָ יְהֹוָה מִצִּיּוֹן. עֹשֵׂה שָׁמַיִם וָאָרֶץ: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span> יְהֹוָה צְבָאוֹת עִמָּנוּ. מִשְׂגָּב לָנוּ אֱלֹהֵי יַעֲקֹב סֶלָה: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span> יְהֹוָה צְבָאוֹת אַשְׁרֵי אָדָם בּוֹטֵחַ בָּךְ: <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(ג"פ:)</span></span> יְהֹוָה הוֹשִׁיעָה. הַמֶּלֶךְ יַעֲנֵנוּ בְיוֹם קָרְאֵנוּ:<br/> הושִׁיעָה אֶת עַמֶּךָ וּבָרֵךְ אֶת נַחֲלָתֶךָ וּרְעֵם וְנַשְּׂאֵם עַד הָעוֹלָם: מִי יִתֵּן מִצִּיּוֹן יְשׁוּעַת יִשְׂרָאֵל בְּשׁוּב יְהֹוָה שְׁבוּת עַמּוֹ. יָגֵל יַעֲקֹב יִשְׂמַח יִשְׂרָאֵל: בְּשָׁלוֹם יַחְדָּו אֶשְׁכְּבָה וְאִישָׁן. כִּי אַתָּה יְהֹוָה לְבָדָד לָבֶטַח תּוֹשִׁיבֵנִי: יוֹמָם יְצַוֶּה יְהֹוָה חַסְדּוֹ. וּבַלַּיְלָה שִׁירֹה עִמִּי. תְּפִלָּה לְאֵל חַיָּי: וּתְשׁוּעַת צַדִּיקִים מֵיְהֹוָה. מָעוּזָּם בְּעֵת צָרָה: וַיַּעְזְרֵם יְהֹוָה וַיְפַלְּטֵם. יְפַלְּטֵם מֵרְשָׁעִים וְיוֹשִׁיעֵם כִּי חָסוּ בוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חצי קדיש:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא. יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא ( בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְהוּא רַחוּם</span> יְכַפֵּר עָוֹן וְלֹא יַשְׁחִית, וְהִרְבָּה לְהָשִׁיב אַפּוֹ, ולֹא יָעִיר כָּל חֲמָתוֹ: יְהֹוָה הוֹשִׁיעָה, הַמֶּלֶךְ יַעֲנֵנוּ בְיוֹם קָרְאֵנוּ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חזן:</span> <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרְכוּ</span> אֶת יְהֹוָה הַמְּבֹרָךְ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קהל וחזן:</span> <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> יְהֹוָה הַמְּבֹרָךְ לְעוֹלָם וָעֵד:
<h3>ק''ש וברכותיה</h3>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר בִּדְבָרוֹ מַעֲרִיב עֲרָבִים, בְּחָכְמָה פּוֹתֵחַ שְׁעָרִים, וּבִּתְבוּנָה מְשַׁנֶּה עִתִּים, וּמַחֲלִיף אֶת הַזְּמַנִּים, וּמְסַדֵּר אֶת הַכּוֹכָבִים בְּמִשְׁמְרוֹתֵיהֶם בָּרָקִיעַ, כִּרְצוֹנוֹ. בּוֹרֵא יוֹמָם וָלַיְלָה, גּוֹלֵל אוֹר מִפְּנֵי חֹשֶׁךְ וְחֹשֶׁךְ מִפְּנֵי אוֹר, וּמַעֲבִיר יוֹם וּמֵבִיא לַיְלָה, וּמַבְדִּיל בֵּין יוֹם וּבֵין לָיְלָה. יְהֹוָה צְבָאוֹת שְׁמוֹ. אֵל חַי וְקַיָּם תָּמִיד יִמְלוֹךְ עָלֵינוּ לְעוֹלָם וָעֶד: בָּרוּךְ אַתָּה יְהֹוָה, הַמַּעֲרִיב עֲרָבִים:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַהֲבַה רַבָּה</span> (אַהֲבַת עוֹלָם) בֵּית יִשְׂרָאֵל עַמְּךָ אָהַבְתָּ, תּוֹרָה וּמִצְוֹת חֻקִּים וּמִשְׁפָּטִים אוֹתָנוּ לִמַּדְתָּ. עַל כֵּן יְהֹוָה אֱלֹהֵינוּ, בְּשָׁכְבֵנוּ וּבְקוּמֵנוּ נָשִׂיחַ בְּחֻקֶּיךָ, וְנִשְׂמַח בְּדִבְרֵי תַלְמוּד תּוֹרָתֶךָ, וּבְמִצְוֹתֶיךָ לְעוֹלָם וָעֶד, כִּי הֵם חַיֵּינוּ וְאֹרֶךְ יָמֵינוּ, וּבָהֶם נֶהְגֶּה יוֹמָם וָלָיְלָה. וְאַהֲבָתְךָ אַל תָסוּר מִמֶּנּוּ לְעוֹלָמִים, בָּרוּךְ אַתָּה יְהֹוָה, אוֹהֵב עַמּוֹ יִשְׂרָאֵל:
<span style="text-align: center;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">( (יחיד אומר: <span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֵל מֶלֶךְ נֶאֱמָן:</span> ) )</span></span>
<span style="text-align: center;"><big>שְׁמַע יִשְׂרָאֵל יְהֹוָה אֱלֹהֵינוּ יְהֹוָה אֶחָד:</big></span>
<br/><span style="text-align: center;"><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בלחש: בָּרוּךְ, שֵּׁם כְּבוֹד מַלְכוּתוֹ, לְעוֹלָם וָעֶד:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְאָהַבְתָּ</span> אֵת יְהֹוָה אֱלֹהֶיךָ בְּכָל לְבָבְךָ וּבְכָל נַפְשְׁךָ וּבְכָל מְאֹדֶךָ. וְהָיוּ הַדְּבָרִים הָאֵלֶּה אֲשֶׁר אָנֹכִי מְצַוְּךָ הַיּוֹם עַל לְבָבֶךָ. וְשִׁנַּנְתָּם לְבָנֶיךָ וְדִבַּרְתָּ בָּם בְּשִׁבְתְּךָ בְּבֵיתֶךָ וּבְלֶכְתְּךָ בַדֶּרֶךְ וּבְשָׁכְבְּךָ וּבְקוּמֶךָ. וּקְשַׁרְתָּם לְאוֹת עַל יָדֶךָ וְהָיוּ לְטֹטָפֹת בֵּין עֵינֶיךָ. וּכְתַבְתָּם עַל מְזוּזֹת בֵּיתֶךָ וּבִשְׁעָרֶיךָ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְהָיָה</span> אִם שָׁמֹעַ תִּשְׁמְעוּ אֶל מִצְוֹתַי אֲשֶׁר אָנֹכִי מְצַוֶּה אֶתְכֶם הַיּוֹם לְאַהֲבָה אֶת יְהֹוָה אֱלֹהֵיכֶם וּלְעָבְדוֹ בְּכָל לְבַבְכֶם וּבְכָל נַפְשְׁכֶם. וְנָתַתִּי מְטַר אַרְצְכֶם בְּעִתּוֹ יוֹרֶה וּמַלְקוֹשׁ וְאָסַפְתָּ דְגָנֶךָ וְתִירֹשְׁךָ וְיִצְהָרֶךָ. וְנָתַתִּי עֵשֶׂב בְּשָׂדְךָ לִבְהֶמְתֶּךָ וְאָכַלְתָּ וְשָׂבָעְתָּ. הִשָּׁמְרוּ לָכֶם פֶּן יִפְתֶּה לְבַבְכֶם וְסַרְתֶּם וַעֲבַדְתֶּם אֱלֹהִים אֲחֵרִים וְהִשְׁתַּחֲוִיתֶם לָהֶם. וְחָרָה אַף יְהֹוָה בָּכֶם וְעָצַר אֶת הַשָּׁמַיִם וְלֹא יִהְיֶה מָטָר וְהָאֲדָמָה לֹא תִתֵּן אֶת יְבוּלָהּ וַאֲבַדְתֶּם מְהֵרָה מֵעַל הָאָרֶץ הַטֹּבָה אֲשֶׁר יְהֹוָה נֹתֵן לָכֶם. וְשַׂמְתֶּם אֶת דְּבָרַי אֵלֶּה עַל לְבַבְכֶם וְעַל נַפְשְׁכֶם וּקְשַׁרְתֶּם אֹתָם לְאוֹת עַל יֶדְכֶם וְהָיוּ לְטוֹטָפֹת בֵּין עֵינֵיכֶם. וְלִמַּדְתֶּם אֹתָם אֶת בְּנֵיכֶם לְדַבֵּר בָּם בְּשִׁבְתְּךָ בְּבֵיתֶךָ וּבְלֶכְתְּךָ בַדֶּרֶךְ וּבְשָׁכְבְּךָ וּבְקוּמֶךָ. וּכְתַבְתָּם עַל מְזוּזוֹת בֵּיתֶךָ וּבִשְׁעָרֶיךָ. לְמַעַן יִרְבּוּ יְמֵיכֶם וִימֵי בְנֵיכֶם עַל הָאֲדָמָה אֲשֶׁר נִשְׁבַּע יְהֹוָה לַאֲבֹתֵיכֶם לָתֵת לָהֶם כִּימֵי הַשָּׁמַיִם עַל הָאָרֶץ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיֹּאמֶר</span> יְהֹוָה אֶל מֹשֶׁה לֵּאמֹר. דַּבֵּר אֶל בְּנֵי יִשְׂרָאֵל וְאָמַרְתָּ אֲלֵהֶם וְעָשׂוּ לָהֶם צִיצִת עַל כַּנְפֵי בִגְדֵיהֶם לְדֹרֹתָם וְנָתְנוּ עַל צִיצִת הַכָּנָף פְּתִיל תְּכֵלֶת. וְהָיָה לָכֶם לְצִיצִת וּרְאִיתֶם אֹתוֹ וּזְכַרְתֶּם אֶת כָּל מִצְוֹת יְהֹוָה וַעֲשִׂיתֶם אֹתָם וְלֹא תָתֻרוּ אַחֲרֵי לְבַבְכֶם וְאַחֲרֵי עֵינֵיכֶם אֲשֶׁר אַתֶּם זֹנִים אַחֲרֵיהֶם. לְמַעַן תִּזְכְּרוּ וַעֲשִׂיתֶם אֶת כָּל מִצְוֹתָי וִהְיִיתֶם קְדֹשִׁים לְאלֹהֵיכֶם. אֲנִי יְהֹוָה אֱלֹהֵיכֶם אֲשֶׁר הוֹצֵאתִי אֶתְכֶם מֵאֶרֶץ מִצְרַיִם לִהְיוֹת לָכֶם לֵאֱלֹהִים אֲנִי יְהֹוָה אֱלֹהֵיכֶם : אֱמֶת

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֶת צֶמַח</span> דָּוִד עַבְדְּךָ מְהֵרָה תַצְמִיחַ, וְקַרְנוּ תָרוּם בִּישׁוּעָתֶךָ. כִּי לִישׁוּעָתְךָ קִוִּינוּ כָּל הַיּוֹם. (וּמְצַפִּים לְךָ לִיְשׁוּעָה). בָּרוּךְ אַתָּה יְהֹוָה, מַצְמִיחַ קֶרֶן יְשׁוּעָה:

<div id="a9" style="padding:3px;background-color:#CCCCCC;display:none;">בברכת שמע קולינו קודם שמסיים כי אתה שומע יתוודה בהכנעה ובשברון לב ויאמר: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">אָנָּא</span> יְהֹוָה חָטָאתִי. עָוִיתִי. וּפָשַׁעְתִּי לְפָנֶיךָ. מִיּוֹם הֱיוֹתִי עַל הָאֲדָמָה עַד הַיּוֹם הַזֶּה (ובפרט בחטא פלוני): אָנָּא יְהֹוָה. עֲשֵׂה לְמַעַן שִׁמְךָ הַגָּדוֹל. וּתְכַפֵּר לִי עַל חֲטָאַי וַעֲוֹנַי וּפְשָׁעַי. שֶׁחָטָאתִי. וְשֶׁעָוִיתִי. וְשֶׁפָּשַׁעְתִּי לְפָנֶיךָ. מִנְּעוּרַי עַד הַיּוֹם הַזֶּה. וּתְמַלֵּא כָּל הַשֵּׁמוֹת שֶׁפָּגַמְתִּי בְּשִׁמְךָ הַגָּדוֹל:</span></div>
<div id="q10" style="border-bottom: 2px dotted blue; margin: 20px 0;">תפילה על פרנסה</div>
<div id="a10" style="padding:3px;background-color:#CCCCCC;display:none;"><br/>תפילה שיתפלל על מזונו קודם כי אתה שומע: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">אַתָּה</span> הוּא יְהֹוָה הָאֱלֹהִים. הַזָּן וּמְפַרְנֵס וּמְכַלְכֵּל מִקַרְנֵי רְאֵמִים עַד בֵּיצֵי כִנִּים. הַטְרִיפֵנִי לֶחֶם חֻקִּי. וְהַמְצֵא לִי וּלְכָל בְּנֵי בֵיתִי מְזוֹנוֹתַי קוֹדֶם שֶׁאֶצְטָרֵךְ לָהֶם. בְּנַחַת וְלֹא בְצַעַר. בְּהֶתֵּר וְלֹא בְאִסּוּר. בְּכָבוֹד וְלֹא בְּבִזָּיוֹן. לְחַיִּים וּלְשָׁלוֹם. מִשֶּׁפַע בְּרָכָה וְהַצְלָחָה. מִשֶּׁפַע בְּרֵכָה עֶלְיוֹנָה. כְּדֵי שֶׁאוּכַל לַעֲשׂוֹת רְצוֹנֶךָ. וְלַעֲסוֹק בְּתוֹרָתֶךָ. וּלְקַיֵּם מִצְוֹתֶיךָ. וְאַל תַּצְרִיכֵנִי לִידֵי מַתְּנַת בָּשָׂר וָדָם. וִיקֻיַּם בִּי מִקְרָא שֶׁכָּתוּב. פּוֹתֵחַ אֶת יָדֶיךָ וּמַשְׂבִּיעַ לְכָל חַי רָצוֹן:</span></div>

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רְצֵה</span> יְהֹוָה אֱלֹהֵינוּ בְּעַמְּךָ יִשְׂרָאֵל, וְלִתְפִלָּתָם שְׁעֵה, וְהָשֵׁב הָעֲבוֹדָה לִדְבִיר בֵּיתֶךָ, וְאִשֵּׁי יִשְׂרָאֵל וּתְפִלָּתָם מְהֵרָה בְּאַהֲבָה תְּקַבֵּל בְּרָצוֹן, וּתְהִי לְרָצוֹן תָּמִיד עֲבוֹדַת יִשְׂרָאֵל עַמֶּךָ:

<h3>עלינו לשבח</h3>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עָלֵינוּ</span> לְשַׁבֵּחַ לַאֲדוֹן הַכֹּל, לָתֵת גְּדֻלָּה לְיוֹצֵר בְּרֵאשִׁית, שֶׁלֹּא עָשָׂנוּ כְּגוֹיֵי הָאֲרָצוֹת, וְלֹא שָׂמָנוּ כְּמִשְׁפְּחוֹת הָאֲדָמָה. שֶׁלֹּא שָׂם חֶלְקֵנוּ כָּהֶם, וְגוֹרָלֵנוּ כְּכָל הֲמוֹנָם, שֶׁהֵם מִשְׁתַּחֲוִים לְהֶבֶל וָרִיק, וּמִתְפַּלְּלִים אֶל אֵל לֹא יוֹשִׁיעַ. וַאֲנַחְנוּ כּוֹרְעִים וּמִשְׁתַּחֲוִים וּמוֹדִים, לִפְנֵי מֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא, שֶׁהוּא נוֹטֶה שָׁמַיִם וְיֹסֵד אָרֶץ, וּמוֹשַׁב יְקָרוֹ בַּשָּׁמַיִם מִמַּעַל, וּשְׁכִינַת עֻזּוֹ בְּגָבְהֵי מְרוֹמִים. הוּא אֱלֹהֵינוּ אֵין עוֹד, אֱמֶת מַלְכֵּנוּ, אֶפֶס זוּלָתוֹ. כַּכָּתוּב בַּתּוֹרָתוֹ: וְיָדַעְתָּ הַיּוֹם וַהֲשֵׁבֹתָ אֶל לְבָבֶךָ, כִּי יְהֹוָה הוּא הָאֱלֹהִים בַּשָּׁמַיִם מִמַּעַל וְעַל הָאָרֶץ מִתָּחַת אֵין עוֹד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֵּן נְקַוֶּה לָּךְ יְהֹוָה אֱלֹהֵינוּ, לִרְאוֹת מְהֵרָה בְּתִפְאֶרֶת עֻזָּךְ, לְהַעֲבִיר גִּלּוּלִים מִן הָאָרֶץ, וְהָאֱלִילִים כָּרוֹת יִכָּרֵתוּן, לְתַקֵּן עוֹלָם בְּמַלְכוּת שַׁדַּי, וְכָל בְּנֵי בָשָׂר יִקְרְאוּ בִשְׁמֶךָ, לְהַפְנוֹת אֵלֶיךָ כָּל רִשְׁעֵי אָרֶץ. יַכִּירוּ וְיֵדְעוּ כָּל יוֹשְׁבֵי תֵבֵל, כִּי לְךָ תִּכְרַע כָּל בֶּרֶךְ, תִּשָּׁבַע כָּל לָשׁוֹן. לְפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ יִכְרְעוּ וְיִפֹּלוּ, וְלִכְבוֹד שִׁמְךָ יְקָר יִתֵּנוּ, וִיקַבְּלוּ כֻלָּם אֶת עֹל מַלְכוּתֶךָ, וְתִמְלֹךְ עֲלֵיהֶם מְהֵרָה לְעוֹלָם וָעֶד. כִּי הַמַּלְכוּת שֶׁלְּךָ הִיא, וּלְעוֹלְמֵי עַד תִּמְלֹךְ בְּכָבוֹד. כַּכָּתוּב בְּתוֹרָתָךְ: יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד: וְנֶאֱמַר: וְהָיָה יְהֹוָה לְמֶלֶךְ עַל כָּל הָאָרֶץ בַּיּוֹם הַהוּא יִהְיֶה יְהֹוָה אֶחָד וּשְׁמוֹ אֶחָד:

<br/> <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא (בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<h3>ספירת העומר</h3>
<h3>קידוש לבנה</h3>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">עיקר קידוש הלבנה על פי האריז"ל משיעברו עליה שבעת ימים מהמולד מעת לעת, ומיום השמיני והלאה הוא זמן קידוש הלבנה, ויש לקדש בבגדים נאים ובשמחה, כי הוא קבלת פני השכינה הק', ורגליהם רגל ישרה ולצד מזרח דווקא, וכשמגיע לברוך יוצרך, כשם שאני רוקד וכו' אזי יהפוך פניו נגד הלבנה:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַלְלוּיָהּ</span> הַלְלוּ אֶת יְהֹוָה מִן הַשָּׁמַיִם הַלְלוּהוּ בַּמְּרוֹמִים: הַלְלוּהוּ כָל מַלְאָכָיו הַלְלוּהוּ כָּל צְבָאָיו: הַלְלוּהוּ שֶׁמֶשׁ וְיָרֵחַ הַלְלוּהוּ כָּל כּוֹכְבֵי אוֹר: הַלְלוּהוּ שְׁמֵי הַשָּׁמָיִם וְהַמַּיִם אֲשֶׁר מֵעַל הַשָּׁמָיִם: יְהַלְלוּ אֶת שֵׁם יְהֹוָה כִּי הוּא צִוָּה וְנִבְרָאוֹ: וַיַּעֲמִידֵם לָעַד לְעוֹלָם חָק נָתַן וְלֹא יַעֲבוֹר:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">(הֲרֵינִי</span> מוּכָן וּמְזֻמָן לְקַיֵּם הַמִצְוָה לְקַדֵּשׁ הַלְּבָנָה, לְשֵׁם יִחוּד קוּדְשָׁא בְּרִיךְ הוּא וּשְׁכִינְתֵּיהּ עַל יְדֵי הַהוּא טָמִיר וְנֶעְלָם בְּשֵׁם כָּל יִשְׂרָאֵל: )</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">ישים פניו למזרח ויישר רגליו ויראה ראיה אחת בפני הלבנה קודם שיתחיל לברך ואח"כ לֹא יביט בה כלל:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר בְּמַאֲמָרוֹ בָּרָא שְׁחָקִים, וּבְרוּחַ פִּיו כָּל צְבָאָם, חֹק וּזְמַן נָתַן לָהֶם שֶׁלּא יְשַׁנּוּ אֶת תַּפְקִידָם, שָׂשִׂים וּשְׂמֵחִים לַעֲשׂוֹת רְצוֹן קוֹנָם, פּוֹעֵל אֱמֶת שֶׁפְּעֻלָּתוֹ אֱמֶת, וְלַלְּבָנָה אָמַר שֶׁתִּתְחַדֵּשׁ, עֲטֶרֶת תִּפְאֶרֶת לַעֲמוּסֵי בָטֶן שֶׁהֵם עֲתִידִים לְהִתְחַדֵּשׁ כְּמוֹתָהּ, וּלְפָאֵר לְיוֹצְרָם עַל שֵׁם כְּבוֹד מַלְכוּתו: בָּרוּךְ אַתָּה יְהֹוָה מְחַדֵּשׁ חֳדָשִׁים:

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַלְלוּיָהּ</span> הַלְלוּ אֵל בְּקָדְשׁוֹ הַלְלוּהוּ בִּרְקִיעַ עֻזּוֹ: הַלְלוּהוּ בִגְבוּרֹתָיו הַלְלוּהוּ כְּרֹב גֻּדְלוֹ: הַלְלוּהוּ בְּתֵקַע שׁוֹפָר הַלְלוּהוּ בְּנֵבֶל וְכִנּוֹר: הַלְלוּהוּ בְתֹף וּמָחוֹל הַלְלוּהוּ בְּמִנִּים וְעוּגָב: הַלְלוּהוּ בְצִלְצְלֵי שָׁמַע הַלְלוּהוּ בְּצִלְצְלֵי תְרוּעָה: כָּל הַנְּשָׁמָה תְּהַלֵּל יָהּ הַלְלוּיָהּ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">תָּנָא</span> דְבֵי רַבִּי יִשְׁמָעֵאל אִלְמָלֵי לֹא זָכוּ יִשְׂרָאֵל אֶלָּא לְהַקְבִּיל פְּנֵי אֲבִיהֶם שֶׁבַּשָׁמַיִם פַּעַם אַחַת בַּחֹדֶשׁ דַּיָם. אָמַר אַבַּיֵי הִלְכָּךְ צָרִיךְ לְמֵימְרָא מְעוּמָד: מִי זֹאת עולָה מִן הַמִּדְבָּר מִתְרַפֶּקֶת עַל דּוֹדָהּ: וִיהִי רָצוֹן מִלְּפָנֶיךָ יְהֹוָה אֱלֹהַי וֵאלֹהֵי אֲבוֹתַי לְמַלֹּאת פְּגִימַת הַלְּבָנָה וְלֹא יִהְיֶה בָּהּ שׁוּם מִעוּט. וִיהִי אוֹר הַלְּבָנָה כְּאוֹר הַחַמָּה וּכְאוֹר שִׁבְעַת יְמֵי בְרֵאשִׁית כְּמו שֶׁהָיְתָה קוֹדֶם מִעוּטָהּ. שֶׁנֶּאֱמַר אֶת שְׁנֵי הַמְּארוֹת הַגְּדוֹלִים: וְיִתְקַיֶם בָּנוּ מִקְרָא שֶׁכָּתוּב וּבִקְשׁוּ אֶת יְהֹוָה אֱלֹהֵיהֶם וְאֵת דָּוִיד מַלְכָּם. אָמֵן:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לַמְנַצֵּח</span> בִּנְגִינֹת מִזְמוֹר שִׁיר: אֱלֹהִים יְחָנֵּנוּ וִיבָרְכֵנוּ יָאֵר פָּנָיו אִתָּנוּ סֶלָה: לָדַעַת בָּאָרֶץ דַּרְכֶּךָ בְּכָל גּוֹיִם יְשׁוּעָתֶךָ: יוֹדוּךָ עַמִּים אֱלֹהִים יוֹדוּךָ עַמִּים כֻּלָּם: יִשְׂמְחוּ וִירַנְּנוּ לְאֻמִּים כִּי תִשְׁפֹּט עַמִּים מִישׁוֹר וּלְאֻמִּים בָּאָרֶץ תַּנְחֵם סֶלָה: יוֹדוּךָ עַמִּים אֱלֹהִים יוֹדוּךָ עַמִּים כֻּלָּם: אֶרֶץ נָתְנָה יְבוּלָהּ יְבָרְכֵנוּ אֱלֹהִים אֱלֹהֵינוּ: יְבָרְכֵנוּ אֱלֹהִים וְיִירְאוֹ אֹתוֹ כָּל אַפְסֵי אָרֶץ:

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֵּן נְקַוֶּה לָּךְ יְהֹוָה אֱלֹהֵינוּ, לִרְאוֹת מְהֵרָה בְּתִפְאֶרֶת עֻזָּךְ, לְהַעֲבִיר גִּלּוּלִים מִן הָאָרֶץ, וְהָאֱלִילִים כָּרוֹת יִכָּרֵתוּן, לְתַקֵּן עוֹלָם בְּמַלְכוּת שַׁדַּי, וְכָל בְּנֵי בָשָׂר יִקְרְאוּ בִשְׁמֶךָ, לְהַפְנוֹת אֵלֶיךָ כָּל רִשְׁעֵי אָרֶץ. יַכִּירוּ וְיֵדְעוּ כָּל יוֹשְׁבֵי תֵבֵל, כִּי לְךָ תִּכְרַע כָּל בֶּרֶךְ, תִּשָּׁבַע כָּל לָשׁוֹן. לְפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ יִכְרְעוּ וְיִפֹּלוּ, וְלִכְבוֹד שִׁמְךָ יְקָר יִתֵּנוּ, וִיקַבְּלוּ כֻלָּם אֶת עֹל מַלְכוּתֶךָ, וְתִמְלֹךְ עֲלֵיהֶם מְהֵרָה לְעוֹלָם וָעֶד. כִּי הַמַּלְכוּת שֶׁלְּךָ הִיא, וּלְעוֹלְמֵי עַד תִּמְלֹךְ בְּכָבוֹד. כַּכָּתוּב בְּתוֹרָתָךְ: יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד: וְנֶאֱמַר: וְהָיָה יְהֹוָה לְמֶלֶךְ עַל כָּל הָאָרֶץ בַּיּוֹם הַהוּא יִהְיֶה יְהֹוָה אֶחָד וּשְׁמוֹ אֶחָד:
