<h1>ברכות ותפילות שונות</h1>

<h2>סדר ברכות אירוסין ונשואין</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">נוהגין לעשות החופה תחת השמים לסימן ברכה. מוליכין את החתן תחת החופה ומעמידים אותו כשפניו למזרח, ומעמידין את הכלה לימין החתן, והרב המסדר את הקידושין מצדד את עצמו ופניו למזרח ומברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא פְּרִי הַגֶּפֶן:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו, וְצִוָּנוּ עַל הָעֲרָיוֹת, וְאָסַר לָנוּ אֶת הָאֲרוּסוֹת. וְהִתִּיר לָנוּ אֶת הַנְּשׂוּאוֹת לָנוּ, עַל יְדֵי חֻפָּה וְקִדּוּשִׁין. בָּרוּךְ אַתָּה יְהֹוָה, מְקַדֵּשׁ עַמּוֹ יִשְׂרָאֵל עַל יְדֵי חֻפָּה וְקִדּוּשִׁין:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ושותה המברך, ושותין מעט החתן והכלה. ואחר כך מקדש החתן את הכלה בטבעת לפני עדים כשרים ואומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֲרֵי</span> אַתְּ מְקֻדֶּשֶׁת לִי בְּטַבַּעַת זוֹ כְּדַת מֹשֶׁה וְיִשְׂרָאֵל:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קוראים הכתובה, ואחר כך מוסרים הכתובה לכלה, ומברכים על כוס שניה ואומר אלו שבע ברכות:</span>
<h2>סֵדֶר שֶׁבַע בְּרָכוֹת</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא פְּרִי הַגֶּפֶן:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בשבע ברכות שלאחר ברכת המזון מתחילים כאן:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁהַכֹּל בָּרָא לִכְבוֹדוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, יוֹצֵר הָאָדָם:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר יָצַר אֶת הָאָדָם בְּצַלְמוֹ, בְּצֶלֶם דְמוּת תַּבְנִיתוֹ, וְהִתְקִין לוֹ מִמֶּנּוּ בִּנְיַן עֲדֵי עַד. בָּרוּךְ אַתָּה יְהֹוָה יוֹצֵר הָאָדָם:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שׂוֹשׂ</span> תָּשִׂישׂ וְתָגֵל הָעֲקָרָה, בְּקִבּוּץ בָּנֶיהָ לְתוֹכָהּ בִּמְהֵרָה בְּשִׂמְחָה. בָּרוּךְ אַתָּה יְהֹוָה מְשַׂמֵּחַ צִיּוֹן בְּבָנֶיהָ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שַׂמֵּחַ</span> תְּשַׂמַּח רֵעִים אֲהוּבִים כְּשַׂמֵּחֲךָ יְצִירְךָ בְּגַן עֵדֶן מִקֶּדֶם. בָּרוּךְ אַתָּה יְהֹוָה מְשַׂמֵּחַ חָתָן וְכַלָּה:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר בָּרָא שָׂשׂוֹן וְשִׂמְחָה, חָתָן וְכַלָּה, גִּילָה רִנָּה, דִּיצָה וְחֶדְוָה, אַהֲבָה וְאַחְוָה, שָׁלוֹם וְרֵעוּת, מְהֵרָה יְהֹוָה אֱלֹהֵֽינוּ יִשָּׁמַע בְּעָרֵי יְהוּדָה וּבְחוּצוֹת יְרוּשָׁלַיִם, קוֹל שָׂשׂוֹן וְקוֹל שִׂמְחָה, קוֹל חָתָן וְקוֹל כַּלָּה, קוֹל מִצְהֲלוֹת חֲתָנִים מֵחֻפָּתָם, וּנְעָרִים מִמִּשְׁתֵּה נְגִינָתָם, בָּרוּךְ אַתָּה יְהֹוָה מְשַׂמֵּחַ הֶחָתָן עִם הַכַּלָּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ובברכת המזון מברך עתה על הכוס:</span>
<h2>סֵדֶר בְּרִית מִילָה</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">אין מלים עד נץ החמה של יום שמיני ללידתו, וכל היום כשר למילה אלא שזריזין מקדימים למצוות ומלים בבקר. ומדליקים הרבה נרות בבית הכנסת בשעת המילה. ואין חולצין התפילין עד אחרי המילה. כשמביאים התינוק לבית הכנסת למול אומרים הקהל:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ הַבָּא</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">האב אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַשְׁרֵי</span> תִּבְחַר וּתְקָרֵב יִשְׁכֹּן חֲצֵרֶיךָ.
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">והעומדים שם אומרים:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נִשְׂבְּעָה</span> בְּטוּב בֵּיתֶךָ, קְדֹשׁ הֵיכָלֶךָ:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מנהג ארץ ישראל תובב"א:</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שְׁמַע</span> יִשְׂרָאֵל, יְהֹוָה אֱלֹהֵינוּ יְהֹוָה אֶחָד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהֹוָה</span> מֶלֶךְ יְהֹוָה מָלָךְ יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְהֹוָה</span> מֶלֶךְ יְהֹוָה מָלָךְ יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָֽנָּא</span> יְהֹוָה הוֹשִׁיעָה נָא: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָֽנָּא</span> יְהֹוָה הוֹשִׁיעָה נָא:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָֽנָּא</span> יְהֹוָה הַצְלִיחָה נָא: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָֽנָּא</span> יְהֹוָה הַצְלִיחָה נָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">המוהל אומר אלו הפסוקים:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיְדַבֵּר</span> יְהֹוָה אֶל מֹשֶׁה לֵּאמֹר: פִּינְחָס בֶּן אֶלְעָזָר בֶּן אַהֲרֹן הַכֹּהֵן הֵשִׁיב אֶת חֲמָתִי מֵעַל בְּנֵי יִשְׂרָאֵל בְּקַנְאוֹ אֶת קִנְאָתִי בְּתוֹכָם וְלֹא כִלִּיתִי אֶת בְּנֵי יִשְׂרָאֵל בְּקִנְאָתִי: לָכֵן אֱמֹר הִנְנִי נֹתֵן לוֹ אֶת בְּרִיתִי שָׁלוֹם:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ויניח התינוק על הכסא של אליהו, ויאמר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">זֶה</span> הַכִּסֵּא שֶׁל אֵלִיָּהוּ הַנָּבִיא זָכוּר לַטּוֹב:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">המוהל אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לִישׁוּעָתְךָ</span> קִוִּיתִי יְהֹוָה: שִׂבַּרְתִּי לִישׁוּעָתְךָ יְהֹוָה וּמִצְוֹתֶיךָ עָשִׂיתִי: אֵלִיָּהוּ הַנָּבִיא מַלְאַךְ הַבְּרִית, הִנֵּה שֶׁלְּךָ לְפָנֶיךָ, עֲמוֹד עַל יְמִינִי וְסָמְכֵנִי. שִׂבַּרְתִּי לִישׁוּעָתְךָ יְהֹוָה. שָׂשׂ אָנֹכִי עַל אִמְרָתֶךָ, כְּמוֹצֵא שָׁלָל רָב: שָׁלוֹם רָב לְאֹהֲבֵי תוֹרָתֶךָ וְאֵין לָמוֹ מִכְשׁוֹל, אַשְׁרֵי תִבְחַר וּתְקָרֵב יִשְׁכּוֹן חֲצֵרֶיךָ, נִשְׂבְּעָה בְּטוּב בֵּיתֶךָ קְדֹשׁ הֵיכָלֶךָ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">בשעה שהמוהל מוכן למול, יאמר האב:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הִנְנִי</span> מוּכָן וּמְזֻמָּן לְקַיֵּם מִצְוַת עֲשֵׂה, שֶׁצִוַּנִי הַבּוֹרֵא יִתְבָּרַךְ לָמוּל אֶת בְּנִי:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">והמוהל קודם שמל אומר:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָמַר</span> הַקָּדוֹשׁ בָּרוּךְ הוּא לְאַבְרָהָם אָבִינוּ הִתְהַלֵּךְ לְפָנַי וֶהְיֵה תָמִים: הִנְנִי מוּכָן וּמְזֻמָּן לְקַיֵּם מִצְוַת עֲשֵׂה, שֶׁצִוַּנִי הַבּוֹרֵא יִתְבָּרַךְ לָמוּל. <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ואם האב בעצמו מל יאמר:</span> אֶת בְּנִי: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ויכון ישקבל התינוק מאליהו ז"ל:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ויקח את התינוק ויניחהו על ברכי הסנדק, והאב עומד על יד המוהל להודיע שהוא שלוחו, והמוהל מברך הברכה בקול רם:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ לְהַכְנִיסוֹ בִּבְרִיתוֹ עַל הַמִּילָה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ואחר כך יחתוך הערלה ואבי הבן מברך, ואם אין לו אב מברך הסנדק:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ לְהַכְנִיסוֹ בִּבְרִיתוֹ שֶׁל אַבְרָהָם אָבִינוּ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁהֶחֱיָֽיָנוּ וְקִיְּמָנוּ וְהִגִּיעָנוּ לַזְּמַן הַזֶּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">והקהל עונים:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כְּשֵׁם</span> שֶׁנִּכְנַס לַבְּרִית, כָּךְ יִכָּנֵס לְתּוֹרָה וּלְחֻפָּה וּלְמַעֲשִׂים טוֹבִים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ובעשותו החיתוך ימהר מאד הפריעה והמציצה, וישים הערלה בחול או בעפר ואחר כך יטול המוהל את ידיו כדי שיברך על הכוס בנקיות:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא פְּרִי הַגֶּפֶן:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדֵּשׁ יְדִיד מִבֶּטֶן וְחוֹק בִּשְׁאֵרוֹ שָׂם. וְצֶאֱצָאָיו חָתַם בְּאוֹת בְּרִית קֹדֶשׁ. עַל כֵּן בִּשְׂכַר זֹאת אֵל חַי חֶלְקֵנוּ צוּרֵנוּ צַוֵּה לְהַצִּיל יְדִידוּת שְׁאֵרֵנוּ מִשַּׁחַת, לְמַעַן בְּרִיתוֹ אֲשֶׁר שָׂם בִּבְשָׂרֵנוּ. בָּרוּךְ אַתָּה יְהֹוָה כּוֹרֵת הַבְּרִית:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֱלֹהֵינוּ</span> וֵאלֹהֵי אֲבוֹתֵינוּ קַיֵּים אֶת הַיֶּלֶד הַזֶּה לְאָבִיו וּלְאִמּוֹ וְיִקָּרֵא שְׁמוֹ בְּיִשְׂרָאֵל <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">פְּלוֹנִי בֶּן פְּלוֹנִי</span></span> יִשְׂמַח הָאָב <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">אם האב בעצמו מברך יאמר: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהִי רָצוֹן שֶׁאֶשְׂמַח בְּיוֹצֵא חֲלָצַי</span></span> בְּיוֹצֵא חֲלָצָיו, <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ליתום מאב יאמר: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִשְׂמַח הָאָב בְּגַן עֵדֶּן בְּיוֹצֵא חֲלָצָיו</span></span> וְתָגֵל אִמּוֹ בִּפְרִי בִטְנָהּ <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ליתום מהאם יאמר: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְתָגֵל אִמּוֹ בְּגַן עֵדֶּן בִּפְרִי בִטְנָהּ</span></span> כַּכָּתוּב: יִשְׂמַח אָבִיךָ וְאִמֶּךָ, וְתָגֵל יוֹלַדְתֶּךָ: וְנֶאֱמַר: וָאֶעֱבֹר עָלַיִךְ וָאֶרְאֵךְ מִתְבּוֹסֶסֶת בְּדָמָיִךְ וָאֹמַר לָךְ בְּדָמַיִךְ חֲיִי <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">נותן יין באצבעו בפי התינוק, וכן בפעם השניה כשחוזר לומר "ואוֹמר לך"</span></span> וָאֹמַר לָךְ בְּדָמַיִךְ חֲיִי: וְנֶאֱמַר: זָכַר לְעוֹלָם בְּרִיתוֹ, דָּבָר צִוָּה לְאֶלֶף דּוֹר: אֲשֶׁר כָּרַת אֶת אַבְרָהָם, וּשְׁבוּעָתוֹ לְיִשְׂחָק: וַיַּעֲמִידֶהָ לְיַעֲקֹב לְחֹק, לְיִשְׂרָאֵל בְּרִית עוֹלָם: וְנֶאֱמַר וַיָּמָל אַבְרָהָם אֶת יִצְחָק בְּנוֹ בֶּן שְׁמֹנַת יָמִים כַּאֲשֶׁר צִוָּה אֹתוֹ אֱלֹהִים: הוֹדוּ לַיהֹוָה כִּי טוֹב, כִּי לְעוֹלָם חַסְדּוֹ הוֹדוּ לַיהֹוָה כִּי טוֹב, כִּי לְעוֹלָם חַסְדּוֹ: <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">פְּלוֹנִי בֶּן פְּלוֹנִי</span></span> זֶה הַקָּטוֹן גָּדוֹל יִהְיֶה, כְּשֵׁם שֶׁנִּכְנַס לַבְּרִית כֵּן יִכָּנֵס לַתוֹרָה וּלְחֻפָּה וּלְמַעֲשִׂים טוֹבִים:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">ישתה הסנדק מכוס של ברכה, יתן גם לתינוק הנימול מעט מכוס של ברכה, כמו שעשה כשאמר בדמיך חיי, ואחר כך יאמר המוהל זה:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;"><br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מִי שֶׁבֵּרַךְ אֲבוֹתֵינוּ אַבְרָהָם יִצְחָק וְיַעֲקֹב, הוּא יְבָרֵךְ אֶת הַיֶּלֶד רַךְ הַנִּמּוֹל, <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">פְּלוֹנִי בֶּן פְּלוֹנִי</span></span> וְיִשְׁלַח לוֹ רְפוּאָה שְׁלֵמָה, בַּעֲבוּר שֶׁנִּכְנַס לַבְּרִית. וּכְשֵׁם שֶׁנִּכְנַס לַבְּרִית כֵּן יִכָּנֵס לְתּוֹרָה וּלְחֻפָּה וּלְמַעֲשִׂים טוֹבִים, וְנֹאמַר אָמֵן:</span></span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">אחר כך יעמוד המוהל ויתפלל תפילה זו, גם האב יתפלל תפילה זו תיכף אחר המילה:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רִבּוֹנוֹ</span> שֶׁל עוֹלָם, יְהִי רָצוֹן מִלְּפָנֶיךָ, שֶׁיְּהֵא חָשׁוּב וּמְרֻצֶּה וּמְקֻבָּל לְפָנֶיךָ, כְּאִלּוּ הִקְרַבְתִּיהוּ לִפְנֵי כִסֵּא כְבוֹדֶךָ, וְאַתָּה בְּרַחֲמֶיךָ הָרַבִּים, שְׁלַח עַל יְדֵי מַלְאָכֶיךָ הַקְּדוֹשִׁים נְשָׁמָה קְדוֹשָׁה וּטְהוֹרָה <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְפְּלוֹנִי בֶּן פְּלוֹנִי</span></span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">האב יאמר: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לִבְנִי</span></span> הַנִּמוֹל עַתָּה לְשִׁמְךָ הַגָּדוֹל, וְשֶׁיִּהְיֶה לִבּוֹ פָּתוּחַ כְּפִּתְחוֹ שֶׁל אוּלָם בְּתוֹרָתְךָ הַקְּדוֹשָׁה, לִלְמוֹד וּלְלַמֵּד לִשְׁמוֹר וְלַעֲשׂוֹת, וְתֵן לוֹ אֲרִיכוּת יָמִים וְשָׁנִים, חַיִּים שֶׁל יִרְאַת חֵטְא, חַיִּים שֶׁל עשֶׁר וְכָבוֹד, חַיִּים שֶׁתְּמַלֵּא מִשְׁאֲלוֹת לִבּוֹ לְטוֹבָה, אָמֵן וְכֵן יְהִי רָצוֹן.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עָלֵינוּ</span> לְשַׁבֵּחַ לַאֲדוֹן הַכֹּל, לָתֵת גְּדֻלָּה לְיוֹצֵר בְּרֵאשִׁית, שֶׁלֹּא עָשָׂנוּ כְּגוֹיֵי הָאֲרָצוֹת, וְלֹא שָׂמָנוּ כְּמִשְׁפְּחוֹת הָאֲדָמָה. שֶׁלֹּא שָׂם חֶלְקֵנוּ כָּהֶם, וְגוֹרָלֵנוּ כְּכָל הֲמוֹנָם, שֶׁהֵם מִשְׁתַּחֲוִים לְהֶבֶל וָרִיק, וּמִתְפַּלְּלִים אֶל אֵל לֹא יוֹשִׁיעַ. וַאֲנַחְנוּ כּוֹרְעִים וּמִשְׁתַּחֲוִים וּמוֹדִים, לִפְנֵי מֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא, שֶׁהוּא נוֹטֶה שָׁמַיִם וְיֹסֵד אָרֶץ, וּמוֹשַׁב יְקָרוֹ בַּשָּׁמַיִם מִמַּעַל, וּשְׁכִינַת עֻזּוֹ בְּגָבְהֵי מְרוֹמִים. הוּא אֱלֹהֵינוּ אֵין עוֹד, אֱמֶת מַלְכֵּנוּ, אֶפֶס זוּלָתוֹ. כַּכָּתוּב בַּתּוֹרָתוֹ: וְיָדַעְתָּ הַיּוֹם וַהֲשֵׁבֹתָ אֶל לְבָבֶךָ, כִּי יְהֹוָה הוּא הָאֱלֹהִים בַּשָּׁמַיִם מִמַּעַל וְעַל הָאָרֶץ מִתָּחַת אֵין עוֹד:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> כֵּן נְקַוֶּה לָּךְ יְהֹוָה אֱלֹהֵינוּ, לִרְאוֹת מְהֵרָה בְּתִפְאֶרֶת עֻזָּךְ, לְהַעֲבִיר גִּלּוּלִים מִן הָאָרֶץ, וְהָאֱלִילִים כָּרוֹת יִכָּרֵתוּן, לְתַקֵּן עוֹלָם בְּמַלְכוּת שַׁדַּי, וְכָל בְּנֵי בָשָׂר יִקְרְאוּ בִשְׁמֶךָ, לְהַפְנוֹת אֵלֶיךָ כָּל רִשְׁעֵי אָרֶץ. יַכִּירוּ וְיֵדְעוּ כָּל יוֹשְׁבֵי תֵבֵל, כִּי לְךָ תִּכְרַע כָּל בֶּרֶךְ, תִּשָּׁבַע כָּל לָשׁוֹן. לְפָנֶיךָ יְהֹוָה אֱלֹהֵינוּ יִכְרְעוּ וְיִפֹּלוּ, וְלִכְבוֹד שִׁמְךָ יְקָר יִתֵּנוּ, וִיקַבְּלוּ כֻלָּם אֶת עֹל מַלְכוּתֶךָ, וְתִמְלֹךְ עֲלֵיהֶם מְהֵרָה לְעוֹלָם וָעֶד. כִּי הַמַּלְכוּת שֶׁלְּךָ הִיא, וּלְעוֹלְמֵי עַד תִּמְלֹךְ בְּכָבוֹד. כַּכָּתוּב בְּתוֹרָתָךְ: יְהֹוָה יִמְלֹךְ לְעֹלָם וָעֶד: וְנֶאֱמַר: וְהָיָה יְהֹוָה לְמֶלֶךְ עַל כָּל הָאָרֶץ בַּיּוֹם הַהוּא יִהְיֶה יְהֹוָה אֶחָד וּשְׁמוֹ אֶחָד:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">קדיש יתום:</span>
<br/> <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִתְגַּדַּל</span> וְיִתְקַדַּשׁ שְׁמֵהּ רַבָּא. בְּעָלְמָא דִּי בְרָא, כִּרְעוּתֵה, וְיַמְלִיךְ מַלְכוּתֵה, וְיַצְמַח פֻּרְקָנֵה, וִיקָרֵב מְשִׁיחֵהּ. בְּחַיֵּיכוֹן וּבְיוֹמֵיכוֹן וּבְחַיֵּי דְכָל בֵּית יִשְׂרָאֵל, בַּעֲגָלָא וּבִזְמַן קָרִיב, וְאִמְרוּ אָמֵן. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא שְׁמֵיהּ רַבָּא מְבָרַךְ לְעָלַם וּלְעָלְמֵי עָלְמַיָּא יִתְבָּרַךְ.</span> וְיִשְׁתַּבַּח. וְיִתְפָּאַר. וְיִתְרוֹמַם. וְיִתְנַשֵּׂא. וְיִתְהַדָּר. וְיִתְעַלֶּה. וְיִתְהַלָּל שְׁמֵהּ דְּקֻדְשָׁא, בְּרִיךְ הוּא. לְעֵלָּא (בעשי"ת וּלְעֵלָּא מִכָּל) מִן כָּל בִּרְכָתָא וְשִׁירָתָא, תֻּשְׁבְּחָתָא וְנֶחֱמָתָא, דַּאֲמִירָן בְּעָלְמָא, וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֵא</span> שְׁלָמָא רַבָּא מִן שְׁמַיָּא וְחַיִּים טוֹבִים עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֹשֶׂה שָׁלוֹם (בעשי"ת הַשָּׁלוֹם) בִּמְרוֹמָיו הוּא יַעֲשֶׂה שָׁלוֹם עָלֵינוּ וְעַל כָּל יִשְׂרָאֵל וְאִמְרוּ אָמֵן:</span></span>
<h2>גאולה לברית מילה</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יום ליבשה</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">חיברו ר' יהודה הלוי נהגו לאומרו בסעודת הברית:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יוֹם</span> לְיַבָּשָׁה נֶהֶפְּכוּ מְצוּלִים. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">הִטְבַּעְתָּ בְּתַרְמִית.</span> רַגְלֵי בַּת עֲנָמִית. וּפַעֲמֵי שׁוּלַמִּית. יָפוּ בַּנְּעָלִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְכָל רוֹאֵי יְשׁוּרוּן.</span> בְּבֵית הוֹדִי יְשֹׁרְרוּן. אֵין כָּאֵל יְשׁוּרוּן. וְאוֹיְבֵינוּ פְלִילִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">דְּגָלַי כֵּן תָּרִים.</span> עַל הַנִּשְׁאָרִים. וּתְלַקֵּט נִפְזָרִים. כִּמְלַקֵּט שִׁבָּלִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">הַבָּאִים עִמְּךָ.</span> בִּבְרִית חוֹתָמְךָ. וּמִבֶּטֶן לְשִׁמְךָ. הֵמָּה נִמּוֹלִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">הַרְאֵה אוֹתוֹתָם.</span> לְכָל רוֹאֵי אוֹתָם. וְעַל כַּנְפֵי כְסוּתָם. יַעֲשוּ גְדִילִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְמִי זֹאת נִרְשֶׁמֶת.</span> הַכֶּר נָא דְבַר אֱמֶת. לְמִי הַחוֹתֶמֶת. וּלְמִי הַפְּתִילִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְשׁוּב שֵׁנִית לְקַדְּשָׁהּ.</span> וְאַל תּוֹסִיף לְגָרְשָׁהּ. וְהַעֲלֵה אוֹר שִׁמְשָׁהּ. וְנָסוּ הַצְּלָלִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְדִידִים רוֹמְמוּךָ.</span> בְּשִׁירָה קִדְּמוּךָ. מִי כָמוֹכָה. יְהֹוָה בָּאֵלִים:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בִּגְלַל אָבוֹת</span> תּוֹשִׁיעַ בָּנִים וְתָבִיא גְאֻלָּה לִבְנֵי בְנֵיהֶם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">שִׁירָה חֲדָשָׁה שִׁבְּחוּ גְאוּלִים:</span>
<h2>סדר פדיון הבן</h2>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">מצות עשה על כל איש מישראל שיפדה את בנו שהוא בכור לאמו בחמשה סלעים, ויכול ליתן לכהן שאר החפצים שיהיו שוים כך, אבל לֹא קרקעות או שטרות או עבדים,ולכן אין פודים בשטרות כסף. ונוהגים לעשות סעודת מצוה. ועושים את הפדיון בתוך הסעודה: אין פודין את הבכור עד שיעברו עליו שלושים יום, וביום שלושים ואחד יפדוהו מיד שלא להשהות את המצוה:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">האב מביא את הבכור לפני הכהן עם הפדיון ואומר לו:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זֶה</span> בְּנִי בְּכוֹרִי וְהוּא פֶּטֶר רֶחֶם לְאִמוֹ הַיִשְׂרְאֵלִית, וְהַקָדוֹשׁ בָּרוּךְ הוּא צִוָה לִפְדוֹתוֹ שֶׁנֶאֱמַר וּפְדוּיָו מִבֶּן חוֹדֶשׁ תִּפְדֶה בְּעֶרְכְּךָ כֶּסֶף חֲמֵשֶׁת שְׁקָלִים בְּשֶׁקֶל הַקוֹדֶשׁ עֶשְׂרִים גֵרָה הוּא וְנֶאֱמַר: קַדֶשׁ לִי כָל בְּכוֹר פֶטֶר כָּל רֶחֶם בִּבְנֵי יִשְׂרָאֵל בָּאָדָם וּבַבְּהֵמָה לִי הוּא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הכהן שואל את האב:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">מַאי</span> בָּעִית טְפֵי, לִיתֵּן לִי בִּנְךָ בְּכוֹרֶךָ, שֶׁהוּא פֶּטֶר רֶחֶם לְאִמּוֹ, אוֹ בָעִית לִפְדוֹתוֹ בְּעַד חָמֵשׁ סְלָעִים כְּדִמְחַיַיבְתָּ מִדְאוֹרַיְיתָא:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">האב אומר:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">חָפֵץ</span> אֲנִי לִפְדוֹת אֶת בְּנִי וְהֵילָךְ דְמֵי פִדְיוֹנוֹ כְּדִמְחַיַיבְתִּי מִדְאוֹרַיְיתָא:
<br/> <span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">האב מברך ברכה זו ושהחיינו:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו וְצִוָּנוּ עַל פִּדְיוֹן הַבֵּן:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם שֶׁהֶחֱיָנוּ וְקִיְּמָנוּ וְהִגִּיעָנוּ לַזְמַן הַזֶּה:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הכהן מעביר את דמי הפדיון מעל ראש הבכור ואומר:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">זֶה תַחַת זֶה,</span> זֶה חִילוּף זֶה, זֶה מָחוּל עַל זֶה, וְיִכָּנֵס זֶה הַבֵּן לְחַיִים לְתּוֹרָה וּלְיִרְאַת שָׁמַיִם. יְהִי רָצוֹן שֶׁכְּשֵׁם שֶׁנִּכְנַס לְפִדְיוֹן כֵּן יִכָּנֵס לְתוֹרָה וּלְחֻפָּה וּלְמַעֲשִׂים טוֹבִים אָמֵן:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הכהן שם את ידיו על ראש הילד ומברכו:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְשִׂימְךָ</span> אֱלֹהִים כְּאֶפְרַיִם וְכִמְנַשֶּׁה: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְבָרֶכְךָ</span> יְהֹוָה וְיִשְׁמְרֶךָ: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יָאֵר</span> יְהֹוָה פָּנָיו אֵלֶיךָ וִיחֻנֶּךָּ: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יִשָּׂא</span> יְהֹוָה פָּנָיו אֵלֶיךָ וְיָשֵׂם לְךָ שָׁלוֹם:
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">כִּי</span> אוֹרֶךְ יָמִים וּשְׁנוֹת חַיִּים וְשָׁלוֹם יוֹסִיפוּ לָךְ: <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְהֹוָה</span> יִשְׁמָרְךָ מִכָּל רָע יִשְׁמֹר אֶת נַפְשֶׁךָ:
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">הכהן מברך על הכוס:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם בּוֹרֵא פְּרִי הַגָּפֶן:
