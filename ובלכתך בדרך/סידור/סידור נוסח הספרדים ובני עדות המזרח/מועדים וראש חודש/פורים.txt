<h1>פורים</h1>

<h2>שַׁבַּת זָכוֹר</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">בשבת זכור, קודם הוצאת ספר תורה <span style="font-size:95%; color:RGB(0,0,255); text-shadow:0 0px 0 white;">ויש נוהגים לאומרו אחר קריאת התורה (עין כה"ח סימן סח)</span> נוהגים לומר פיוט "מי כמוך":</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מִי כָמוֹךָ</span>, וְאֵין כָּמוֹךָ, מִי דֽוֹמֶה־לָּךְ, וְאֵין דּוֹמֶה לָךְ:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">על סדר א"ב</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אָ</span>דוֹן חַסְדְּךָ בַּל יֶחְדָּל, אֲמִתְּךָ מִבְצָר עֹז וּמִגְדָּל, כִּי־הָיִיתָ מָעוֹז לַדָּל, מָעוֹז לָאֶבְיוֹן בַּצַּר־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בִּ</span>ימֵי חָרְפִי מִקַּדְמוֹנָי, בִּי דִבֶּר רוּחַ יְהֹוָה , והַיּוֹם אִם גֵּרְשׁוּנִי עֲוֹנָי, דּוֹדִי לִי וַאֲנִי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">גְּ</span>מָלַנִי מֵאָז טוֹבוֹת, גְּלוּיוֹת יְדוּעוֹת לַלְּבָבוֹת, גַּם אִם יִהְיוּ הַצָּרוֹת קְרוֹבוֹת, אַשְׁרֵי כָּל־חוֹכֵי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דְּ</span>מֵי מִתְקוֹמְמַי אַזֶּה, וְדִגְלִי אָרִים כִּדְבַר כָּל־חוֹזֶה, הִנֵּה אֱלֹהֵֽינוּ זֶה, קִוִּֽינוּ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַ</span>יּוֹם אַבִּיעָה חִידוֹת, הָיוּ מִנִּי קֶדֶם עֲתִידוֹת, נִפְלָא בָּם הַמֵּבִין צְפוּנֵי סוֹדוֹת, יוֹצְרִי מִבֶּטֶן לְעֶבֶד לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַ</span>יְהִי בִּימֵי אֲחַשְׁוֵרוֹשׁ, מֶלֶךְ הוּקָם עָל כִּבְרוֹשׁ, נִשָּׂא וָרָם עַל כָּל־קָצִין וָרֹאשׁ, וַיהֹוָה הֵנִיחַ־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">זִ</span>מֵּן בִּשְׁנַת שָׁלֹשׁ לְמַלְכוּתוֹ, לְהַרְאוֹת לְכָל־עֲבָדָיו תְּכוּנָתוֹ, וְאַבְנֵי נֶזֶר מִתְנוֹססוֹת עַל אַדְמָתוֹ, וְעַפְרֹת זָהָב לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">חֵ</span>לֶק לְכָל־בְּנֵי שׁוּשָׁן נָתַן, וְהִקְדִּישׁ קְרוּאָיו כְּחָתָן, בַּחֲצַר גִּנַּת בִּיתָן, אִוָּהּ לְמוֹשָׁב לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">טָ</span>עַם עֲשׂוֹת רְצוֹן כָּל־אִישׁ וְיִצְרוֹ, וְאֵין אוֹנֵס לְאִישׁ יַיִן עֲבָרוֹ, וְלָתֵת לְכָל־שׁוֹאֵל דֵּי מַחְסֹרוֹ, אֲשֶׁר יֶחְסַר לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יַ</span>חַד אֻסּפוּ כָּל־הַנִּדְרָשִׁים, גַּם וַשְׁתִּי הַמַּלְכָּה עָֽשׂתָה מִשְׁתֵּה נָשִׁים, וּמִשְׁתֵּה הַנָּשִׁים כַּאֲנָשִׁים, שִׁבְעַת יָמִים יִסְפְּרוּ־ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כְּ</span>טוֹב לֵב הַמֶּלֶךְ בַּיָּיִן, אָמַר לְהָבִיא לְפָנָיו אֶת יְפַת הָעָיִן, וַתּמָאֵן בִּדְבָרוֹ לֵאמֹר אָיִן, הֵעֵזָה פָנֶיהָ וַתֹּאמַר לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לַ</span>קְּרוֹבִים אֵלָיו הַיּוֹשְׁבִים עֲלֵי כָן, שָׁאַל וַיְּעָצֵהוּ מְמוּכָן, הוּא הָמָן לַפֻּרְעָנוּת מוּכָן, לִבּוֹ יִקְבָּץ־אָוֶן לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מ</span>וֹעֲצוֹתָיו כַּדָּת נְתוּנוֹת, הָיוּ לְעַמִּי רְפוּאוֹת צְפוּנוֹת, כִּי לוּלֵי אִגָּרוֹת רִאשׁוֹנוֹת, גַּם־פְּלֵיטָה לֹא־הָֽיְתָה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נ</span>וֹעָץ לִקְרֹא לְכָל־הֲמוֹנָיו, לְהָבִיא כָלּ יְּפַתּ תֹּאַר לְפָנָיו, וְהַנַּעֲרָה אֲשֶׁר תִּיטַב בְּעֵינָיו, מָהֹר יִמְהָרֶנָּה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">סִ</span>נְסָן לְיָאִיר הֵאִיר וְזָרַח, יָשִׂישׂ כָּגִבּוֹר לָרוּץ אֹרַח, טֶרֶם מַכָּה צִיץ רְפוּאָה פָרַח, כִּי־הִפְלָה יְהֹוָה חָסִיד לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עִ</span>מּוֹ נֶאֶמְנָה הֲדַסָּה תַמָּה, הִיא אֶסְתֵּר בַּת דּוֹדוֹ בָּרָה כַּחַמָּה, וּבְמוֹת אָבִיהָ וְאִמָּהּ, לְקָחָהּ מָרְדְּכַי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">פָּֽ</span>גְעָה חֵן וַתֵּלֶךְ וַהֲלוֹם, וַתִּלָּקַח אֶסְתֵּר אֶל בֵּית־הַמֶּלֶךְ גִּזְרַת יָהֲלוֹם, וּמָרְדְּכַי צִוָּה לָהּ לְכִי לְשָׁלוֹם, כִּי־הוּא אֲדֹנַיִךְ וְהִשְׁתַּחֲוִי־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">צֶ</span>דֶק לָֽבשָׁה בְּמַמְלַכְתָּהּ, לֹא הִגִּידָה אֶסְתֵּר עַמָּהּ וּמוֹלַדְתָּהּ, וְהַמֶּלֶךְ אֲהֵבָהּ וַיֶּרֶב תִּפְאַרְתָּהּ, כִּי מֵיְהֹוָה הָֽיְתָה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">קֶ</span>שֶׁב רַב קָשַׁב צְרוֹר הַמּוֹר, מִסָּרִיסֵי הַמֶּלֶךְ הַקְּבוּעִים לִשְׁמוֹר, וְהִנָּם חוֹשׁבִים עַל הַמֶּלֶךְ לֵאֹמֹר, הָבָה נִּתְחַכְּמָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רָ</span>חֲשָׁה אֶסְתֵּר לַמֶּלֶךְ בְּאִמְרֵי שֶׁפֶר, בַּשֵּׁם מָרְדְּכַי וְנִכְתַּב בַּסֵּפֶר, בֻּקַּשׁ וְנִמְצָא לִפְנֵי צְבִי עוֹפֶר, כִּי־בוּל הָרִים יִשְׂאוּ־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שְׁ</span>נֵיהֶם נִתְלוּ עַל הָעֵץ, כִּי שָׁמַע מָרְדְּכַי לְשׁוֹנָם בְּהִוָּעֵץ, נַשְׁקֶה לַמֶּלֶךְ סַם הַמָּוֶת רוֹעֵץ, אוּלַי יפֻתֶּה וְנוּכלָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">תְּ</span>שׁוּעָה זֹאת צפוּנָה לְדוֹר אַחֲרוֹן, נִכְתּבָה עַל סֵפֶר הַזִּכָּרוֹן, לִהְיוֹת לוֹ מִן הַמֶּלֶךְ תַּגְמוּל וְיִתְרוֹן, כִּי פֹעַל אָדָם יְשַׁלֶּם־לוֹ:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">סימן: אני יהודה הלוי הקטן ברבי שמואל הלוי</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אַחַר</span> כָּל־אֵלֶּה הַדּבָרִים, אֲחַשְׁוֵרוֹשׁ אֶת־הָמָן הֵרִים, וַיְנַשּׂאֵהוּ מֵעַל כָּל־הַשָּׂרִים, וְתוֹעֲפוֹת הָרִים לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נִמְנַע</span> מָרְדְּכַי מִסְּגוֹד לָרָשָׁע, נִין עֲמָלֵק אֲשֶׁר מִבֶּטֶן פָּשַׁע, וְזֶה הוֹסִיף עַל חַטָּאתוֹ פֶּשַׁע, לֹא־יֹאבֶה יְהֹוָה סְלֹחַ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יוֹם</span> כָּֽרעוּ לוֹ כּל־עַבְדֵּי הַמֶּלֶךְ, וּמָרְדְּכַי בְּתֻמּוֹ הוֹלֵךְ, בִּקֵּשׁ לְחַרְחֵר רִיב עַל עַם דָּל וָהֵלֶךְ, עֹבֵר מִתְעַבֵּר עַל־רִיב לֹּא־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יָעַץ</span> תַּחְבּוּלוֹת כְּשִׂכִּים בְּצִדּוֹ, וַיִּבֶז בְּעֵינָיו בִּגְאוֹן חֶמְדּוֹ, לִשְׁלוֹחַ יָד בְּמָרְדְּכַי לְבַדּוֹ, בָּז לְדָבָר יֵחָבֶל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הִרְבָּה</span> כֶסֶף וְזָהָב לִבְלִי חוֹק, וַיֹּאמֶר בְּלִבּוֹ זֶה עֵת לִשְׂחוֹק, וַיָּבֵא עֵצוֹת מֵרָחוֹק, לָרֶשֶׁת מִשְׁכָּנוֹת לֹא־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיַּפֵּל</span> פּוּר וְיָדוֹ בַגּוֹרָל הָדָה, וַיַּרְא כִּי בַאֲדָר מֵת אָבִי הַתְּעוּדָה, וְלֹא זָכַר כִּי הִיא עֵת הַלֵּידָה, חֶבְלֵי יוֹלֵדָה יָבֹאוּ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דּוֹב</span> אוֹרֵב יְעָצוּהוּ רַעְיוֹנָיו, וַיִּשְׁאַל שְׁאֵלָה מֵאֲדוֹנָיו, וְגַם הוּא שִׁנֵּי אֲרָיוֹת שִׁנָּיו, וּמְתַלּעוֹת לָבִיא לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֵן</span> עַם אָחוֹר נָזוֹרוּ, וְדָתֵיהֶם שׁוֹנוֹת וְדָֽתךָ בַּל יִנְצוֹרוּ, רוּחַ יִזְרָעוּ וְסוּפָתָהּ יִקְצוֹרוּ, קָמָה אֵין־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֲכִינוֹתִי</span> כֶסֶף לִגְנָזֶיךָ יַעַל, לְאַבֵּד זֶה הַגּוֹי הַנִּגְעַל, כּל־הַבִּרְכַּיִם אֲשֶׁר לֹא־כָֽרְעוּ לַבַּעַל, וְכָל־הַפֶּה אֲשֶׁר לֹא־נָשַׁק לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לְהָבִיא</span> אֶל גִּנְזֵי הַמֶּלֶךְ אֹסֵף, עֲשֶׂרֶת אֲלָפִים כִּכְּרֵי כֶסֶף, וְנֶפֶשׁ אַכֶּה לְכָל־בֵּית־יוֹסֵף, פַּעַם אַחַת וְלֹא אֶשְׁנֶה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיֹּאמֶר</span> לוֹ הַמֶּלֶךְ יְהִי לְךָ קִנְיָנֶיךָ, וְזֹאת טַבַּעְתִּי עַל יַד יְמִינֶךָ, וְהָעָם לַעֲשׂוֹת בּוֹ כַּטּוֹב בְּעֵינֶיךָ, וּבָא עַד־קִצּוֹ וְאֵין עוֹזֵר לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יָצָא</span> וְסוֹפרֵי הַמֶּלֶךְ נִקְרָאוּ, וַיִּכְתּבוּ כְּכָל־אֲשֶׁר הוֹרָהוּ, וּבְכל־עַם וָעָם רָצִים יָצָאוּ, שָׂרֵי הָרֶכֶב אֲשֶׁר־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הָיָה</span> דבָרוֹ נָחוּץ לֵאמֹר, כֶּרֶם חֶמֶר בְּיוֹם אֶחָד לִזְמוֹר, בִּשְׁלוֹשָׁה עָשָׂר בַּאֲדָר הָחֵל וְגָמוֹר, לֹא־אוֹסִיף עוֹד עֲבוֹר לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">קָרַע</span> מָרְדְּכַי בּגָדָיו לִפְנֵי אֵל, וַיִּצְעַק מַר עַל גּוֹלַת אֲרִיאֵל, הֲבָנִים אֵין ליִשְׂרָאֵל, אִם־יוֹרֵשׁ אֵין לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">טָרֹף</span> טֹרַף שְׁאָר הֲמוֹנַי, מִי נָתַן לִמְשִׁסָּה יַעֲקֹב לְמוֹנַי, וְיִשְׂרָאֵל לְבֹזזִים - הֲלוֹא יְהֹוָה , זוּ חָטָאנוּ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נַעֲרוֹת</span> אֶסְתֵּר הִגִּידוּהָ דָבָר, שָׁמַעְנוּ צְעָקָה כַּיּעֵנִים בַּמִּדְבָּר, וּמָרְדְּכַי בִּלְבוּשׁ שַׂק עָבַר, לֹא יָדַעְנוּ מֶה־הָיָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּגָדִים</span> שָֽׁלחָה לְהַלְבִּישׁוֹ, וְלֹא קִבֵּל מִמְּרִירוּת נַפְשׁוֹ, וַתִּשְׁלַח לַהֲתָךְ לְחָקְרוֹ וּלְדרְשׁוֹ, לְדֵעָה מַה־יֵּעָשֶׂה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רוּחוֹ</span> הוֹצִיא לַהֲתָךְ וְחִוָּה, וּפָרָשַׁת הַכֶּסֶף לְפָנָיו שִׁוָּה, וְהַפַּתְשֶׁגֶן שָׁלַח וְעַל אֶסְתֵּר צִוָּה, לָבוֹא אֶל־הַמֶּלֶךְ לְהִתְחַנֶּן־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בִּתְשׁוּבָתָהּ</span> אָֽמרָה לַהֲשִׁיבוֹ, דַּע כִּי בֶן־מָוֶת אֲשֶׁר לֹא יִקָּרֵא וְיָבֹא, לְבַד מֵאֲשֶׁר יוֹשִׁיט לוֹ הַמֶּלֶךְ אֶת־שַׁרְבִיט זְהָבוֹ, בִּקְרב־אִישׁ לְהִשְׁתַּחֲוֹת לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יַקִּיר</span> כְּשָׁמְעוֹ דִּבְרֵי הֲדַסָּה, זָעַק אֲהָהּ כִּי אֵין מְנוּסָה, זַעַף יְהֹוָה אֶשָּׂא, כִּי חָטָאתִי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שָׁלַח</span> לָהּ לֵאמֹר אַל תּדַמִּי, לְהִמָּלֵט בֵּית־הַמֶּלֶךְ מִכּל־לְאוּמִי, כִּי אַתְּ תֹּאבְדִי וְאוּלָם עַמִּי, אַחֲרֵי נִמְכַּר גְּאֻלָּה תִּהְיֶה־לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מִמָּקוֹם</span> אַחֵר יִשְׁלַח שְׁלוֹמָיו, עֹשֶׂה שָׁלוֹם בִּמְרוֹמָיו, כִּי לֹא כָלוּ רַחֲמָיו, עַל־כֵּן אוֹחִיל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַתַּעֲנֵהוּ</span> לֵךְ כּנוֹס כָּל־חוֹכֵי פִדְיוֹם, וְצוּמוּ עָלַי שְׁלֹשֶׁת יָמִים לַיְלָה וָיוֹם, וְרַחֲמִים בַּקְּשׁוּ מֵאֵל נוֹרָא וְאָיוֹם, וְאַל־תִּתּנוּ דֳמִי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֵל</span> נִדְרָשׁ בְּכָל־לֵב יִדְרְשֶׁנּוּ, כִּי נִכְמרוּ רַחֲמָיו לְעַם יִקְרָאֶנּוּ, וַיֹּאמֶר עוֹד זָכֹר אֶזְכְּרֶנּוּ, עַל־כֵּן הָמוּ מֵעַי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לָֽבְשָׁה</span> חֵן בַּיּוֹם הַשְּׁלִישִׁי וַתֵּלֶךְ, וַתִּיקַר עַד מאֹד בְּעֵינֵי הַמֶּלֶךְ, וַיִּבֶז בְּכָל־אֲשֶׁר הוּא מוֹלֵךְ, וְעַל כָּל־אֲשֶׁר יֶשׁ־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַיָּפָה</span> בַנָּשִׁים נֶטַע נַעֲמָן, מַה־שְּׁאֵלָתֵך כִּי הַכֹּל מְזוּמָן, וַתֹּאמֶר יָבוֹא הַמֶּלֶךְ וְהָמָן, אֶל־הַמִּשְׁתֶּה אֲשֶׁר־עָשִׂיתִי לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לְמָחָר</span> קְרָאתָם לְסוֹד נִכְמָס, וַיֵּצֵא הָמָן בַּיּוֹם הַהוּא שָׂמֵחַ וְנֶעְלָס, וְהוּא בַּמְּלָכִים יִתְקַלָּס, וְרֹזְנִים מִשְׂחָק לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיָּבֵא</span> אֹהֲבָיו וְזֶרֶשׁ אִשְׁתּוֹ, וַיְסַפֵּר לָהֶם כְּבוֹדוֹ וּמֶמְשַׁלְתּוֹ, וּגְזֵרָה עוֹמֶדֶת לְעוּמָתוֹ, הוֹי הַמַּרְבֶּה לֹּא־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יְעָצוּהוּ</span> עֲשׂוֹת עֵץ גָּבוֹהַּ חֲמִשִּׁים, לִתְלוֹת עָלָיו קֹדֶשׁ קָדָשִׁים, וּלְהָכִינוֹ כְּחֶפְצוֹ מִכּל־חֲרָשִׁים, חָרָשׁ חָכָם יְבַקֶּשׁ־לוֹ:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">על סדר א"ב</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אֱלֹהִים</span> עֵדֶר צֹאנוֹ בִּקֵּר, וַתִּדַּד שְׁנַת הַמֶּלֶךְ עַד יהִי חוֹקֵר, בְּסֵפֶר הַזִּכְרוֹנוֹת וְהָיָה בֹקֶר, וְיֹדַע יְהֹוָה אֶת־אֲשֶׁר־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּמָצְאוֹ</span> דְּבַר מָרְדְּכַי מְפוֹרָשׁ, מַה־נַּעֲשָׂה יְקָר וּגְדֻלָּה דָּרַשׁ, וַיֹּאמרוּ כִּי לֹא פֹרַשׁ, מַה־יֵּעָשֶׂה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">גִּלָּה</span> אֹזֶן הָמָן בְּדַבְּרוֹ, מַה־לַּעֲשׂוֹת בָּאִישׁ אֲשֶׁר הַמֶּלֶךְ חָפֵץ בִּיקָרוֹ, וַיַּעַן וַיִּנָּקֵשׁ בְּמַאֲמָרוֹ, פִּי־כְסִיל מְחִתָּה־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דִּבֶּר</span> לַעֲדוֹתוֹ בַּעֲדָיֵי מְלָכִים, וְלָלֶכֶת לְפָנָיו אֶחָד מֵהַנְסִיכִים, וְלִבּוֹ בְּמַחֲשָׁבָיו הוֹלֵךְ חֲשֵׁכִים, וְאֵין נֹגַהּ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֱשִׁיבוֹ</span> הַמֶּלֶךְ כֵּן דִּבַּרְתָּ, עֲשֵׂה לְמָרְדְּכַי הַיְּהוּדִי כּכָל־אֲשֶׁר אָמַרְתָּ, כֵּן מִשְׁפָּטֶיךָ אַתָּה חָרַצְתָּ, כַּמִּשְׁפָּט הַזֶּה יֵעָשֶׂה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיָּשָׁב</span> מָרְדְּכַי אֶל מִשְׁמַרְתּוֹ, וְהָמָן נִדְחַף אֶל בֵּיתוֹ, וַיִּוָּעֲדוּ אֹהֲבָיו וְזֶרֶשׁ אִשְׁתּוֹ, לָבוֹא לָנוּד־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">זֶרַע</span> יְשׁוּרוּן זָֽכרוּ נְבוֹנָיו, אִם יֵשׁ מָרְדְּכַי אֶחָד מִבָּנָיו, אֲשֶׁר הַחִלּוֹתָ לִנְפֹּל לְפָנָיו, לֹא־תוּכַל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">חֲכָמָיו</span> עוֹדָם מדַבְּרִים עִמּוֹ, וְסָרִיסֵי הַמֶּלֶךְ הִבְהִילוּהוּ לַהֲקִימוֹ, וְלֹא רָאָה כִּי־יָבֹא יוֹמוֹ, אֲדֹנָי יִשְׂחַק־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">טֶרֶם</span> כְּלוֹת מִשְׁתֶּה הַיַּיִן, אָמַר הַמֶּלֶךְ אֶל יְפַת הָעַיִן, שַׁאֲלִי כִּי הַכָּל לְנֶגְדֶּךְ כְּאַיִן, וַתֵּבְךְּ וַתִּתְחַנֶּן־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יִנָּתֵן</span> לִי נַפְשִׁי בִּשְׁאֵלָתִי, וְעַמִּי אֲדֹנִי הַמֶּלֶךְ בְּבַקָּשָׁתִי, כִּי נִמְכַּרְנוּ לְצוֹרֵר וְלֹא מִבְּרִיתִי, תַּאֲוַת לִבּוֹ נָתַתָּה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">כִּי</span> מַה־חֶפְצִי אַחַר זֶה הַצַּעַר, וּכְלֵי הַקְּרָב שׁוֹת שָׁתוּ הַשַּׁעַר, הֲיִשְׁאַג אַרְיֵה בַּיַּעַר, וְטֶרֶף אֵין לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">לָהּ</span> אָמַר מִי הוּא זֶה מִכָּל־אֲנָשַׁי, וְאֵי זֶה הוּא מִכּל־מִגְרָשַׁי, אוֹ מִי מִנּוֹשַׁי, אֲשֶׁר־מָכַרְתִּי אֶתְכֶם לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">מִהֲרָה</span> לַעֲנוֹתוֹ הָמָן זֶה הָרָע, אֲשֶׁר פְּרָעוֹת בּגוֹי צַדִּיק פָּרַע, אוֹי לְרָשָׁע רָע, כִּי־גְמוּל יָדָיו יֵעָשֶׂה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נִמְלָא</span> חֵמָה וְקָם אֶל גַּנוֹ, וּבְשׁוּבוֹ רָאָהוּ נוֹפֵל עַל כַּנּוֹ, יְגַלּוּ שָׁמַיִם עֲוֹנוֹ, וְאֶרֶץ מִתְקוֹמָמָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">סָרִיס</span> אֶחָד גִּלָּה חַטָּאתוֹ, הֵן עֵץ עָשָׂה לְמָרְדְּכַי בְּבֵיתוֹ, וַיְצַו הַמֶּלֶךְ לִתְלוֹתוֹ, עַל־הָעֵץ אֲשֶׁר־הֵכִין לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">עָֽמדָה</span> אֶסְתֵּר לְבַקֵּשׁ עַל עַמָּהּ, לְהָשִׁיב סִפְרֵי הָאַף וְהַחֵמָה, כִּי נִחַם יְהֹוָה עַל מַכָּה עֲצוּמָה, וְשָׁב ורָפָא לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">פַּתְשֶׁגֶן</span> הַכְּתָב לִהְיוֹת הַיּהוּדִים, לַהֲרוֹג בְּשׂוֹנְאֵיהֶם עֲתִידִים, כִּי נָפַל פַּחַד מָרְדְּכַי עַל הַמּוֹרְדִים, וְשָׁלוֹם הָיָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">צֻוָּה</span> מָרְדְּכַי עַל עַם לֹא אַלְמָן, מוֹשִׁיעַ וְרַב נָגִיד וְנֶאֱמָן, והָפְקַד עַל בֵּית הָמָן, וְעַל כָּל־אֲשֶׁר יֶשׁ־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">קִדַשְׁתִּי</span> לְצֹרְרַי מִזְבֵּחַ, וְאָכִין לְבָנָיו מַטְבֵּחַ, בַּעֲוֹן אֲבִיהֶם הַמְרַצֵּחַ, וְאֵלֶּה שְׁמוֹת הַיִּלֹּדִים לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">רִצַּצְתִּי</span> פַּרְשַׁנְדָּתָא דַלְפוֹן אַסְפָּתָא, פּוֹרָתָא אֲדַלְיָא אֲרִידָתָא פַּרְמַשְׁתָּא, אֲרִיסַי אֲרִידַי וַיְזָתָא, וְגַם־קבוּרָה לֹא־הָֽיתָה לּוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">שָׁלַלְתִּי</span> שֹׁללַי וְאָרִיעַ רֵעַ, כִּי עָלָה מִשּׁאֹל אֶבְיוֹן גֹּוֵעַ, וַיְמַלֵּט עָנִי מְשַׁוֵּעַ, וְיָתוֹם וְלֹא־עֹזֵר לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">תִּכָּתֵב</span> זֹאת לְדוֹר אַחֲרוֹן, וְלִבְנֵי בָנִים תִּהְיֶה לְזִכָּרוֹן, וְכָל־הַמַּזְכִּיר אֹתָהּ יָרֹן, אַשְׁרֵי הָעָם שֶׁכָּכָה לּוֹ:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">סימן: אני יהודה</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">אִכְלוּ</span> רֵעִים שְׁתוּ וְשִׁכְרוּ, וִימֵי הַפּוּרִים בְּשִׂמְחָה שִׁמְרוּ, וְעִם שִׂמְחַתְכֶם הָאֶבְיוֹנִים זִכְרוּ, וְשִׁלְחוּ מָנוֹת לְאֵין נָכוֹן לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">נִסֵּי</span> אֵל מֵאָז נְשָׂאוּנִי, וּבְנִבְכֵי הַיָּם הֶעֱבִירוּנִי, וְלָכֵן כִּלְיוֹתַי יִסּרוּנִי, דּוֹם לַיהֹוָה וְהִתְחוֹלֵל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יוֹם</span> צֵאת פַּרְעֹה אַחֲרַי, לִטְרוֹף כְּאַרְיֵה עֲדָרַי, נִצְּבוּ כַּחוֹמָה מֵימֵי מִשְׁבָּרַי, וַיִּתְגָּעֲשׁוּ כִּי־חָרָה לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יָצָא</span> צָר אַחֲרֵי עַם קְדוֹשׁוֹ, וַיִּשְׁלַח צוּר עֲנָנוֹ לְהַפְרִישׁוֹ, לִהְיוֹת צֵל עַל־רֹאשׁוֹ, לְהַצִּיל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֶעֱבִירוֹ</span> בַיַּבָּשָׁה וְצָרָיו הֶחֱרִיד, בֵּינוֹ וּבֵין רוֹדְפָיו הִפְרִיד, וּלְתַחְתִּיּוֹת מְצוּלָה אוֹתָם הוֹרִיד, חֹקְקִי בַסֶּלַע מִשְׁכָּן לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וּבְשׁוּב</span> הַיָּם לְאֵיתָנוֹ, נִטְבַּע פַּרְעֹה וְכָל־הֲמוֹנוֹ, כִּי הַיָּם הִכְבִּיד מַרְכְּבוֹת גְּאוֹנוֹ, וַיַּרְא כִּי לֹא יָכֹל לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דַּהֲרוֹת</span> אַבִּירִים בְּאַדִּירִים נָֽפלוּ, וְכַעוֹפֶרֶת בְּמַיִם רַבִּים צָֽללוּ, וְדֹרשֵׁי יְהֹוָה שְׁמוֹ יְהַלְּלוּ, בְּנֵבֶל עָשׂוֹר זַמְּרוּ־לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הֶרְאָנוּ</span> יָדוֹ הַנִּפְלָאָה, עַל שְׂפַת הַיָּם נוֹרָאָה, אָשִׁירָה לַּיהֹוָה כִּי־גָאֹה גָּאָה, בִּזְמִרוֹת נָרִיעַ לוֹ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">דְּגָלִים</span> עָֽברוּ בַיַּבָּשָׁה, וְלִפְנֵיהֶם אֵל נַעֲרָץ בִּקְדֹֻשָּׁה, אָז שׁוֹררוּ שִׁירָה חֲדָשָׁה, מִי־כָמֹכָה בָּאֵלִם יְהֹוָה:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;"><br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">ומוציאים ספר התורה וקוראים בפרשת השבוע</span>, ולמפטיר קוראים בפרשת זכור. <span style="font-size:95%; color:RGB(0,0,255); text-shadow:0 0px 0 white;">ואם אמרו 'מי כמוך' לאחר קריאת ספר התורה אז אומרים "אשרי" וגומרים התפלה ככל שבת.</span></span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">זָכוֹר</span> אֵת אֲשֶׁר־עָשָׂה לְךָ עֲמָלֵק בַּדֶּרֶךְ בְּצֵאתְכֶם מִמִּצְרָיִם: אֲשֶׁר קָרְךָ בַּדֶּרֶךְ וַיְזַנֵּב בְּךָ כָּל־הַנֶּחֱשָׁלִים אַחַרֶיךָ וְאַתָּה עָיֵף וְיָגֵעַ וְלֹא יָרֵא אֱלֹהִים: וְהָיָה בְּהָנִיחַ יְהוָה אֱלֹהֶיךָ לְךָ מִכָּל־אֹיְבֶיךָ מִסָּבִיב בָּאָרֶץ אֲשֶׁר יְהוָה־אֱלֹהֶיךָ נֹתֵן לְךָ נַחֲלָה לְרִשְׁתָּהּ תִּמְחֶה אֶת־זֵכֶר עֲמָלֵק מִתַּחַת הַשָּׁמָיִם לֹא תִּשְׁכָּח:
<h2>קריאת המגילה</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">לפני קריאת המגילה מברך החזן בעמידה, וגם הקהל יעמדו בשעת הברכה:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו, וְצִוָּנוּ עַל מִקְרָא מְגִלָּה:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁעָשָׂה נִסִּים לַאֲבוֹתֵֽינוּ בַּיָּמִים הָהֵם בַּזְּמַן הַזֶּה:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">בליל פורים, כשקורא בפעם הראשונה, מברך ברכת "שהחיינו", יְכַוֵּן בברכת "שהחיינו" זו לפטור סעודת פורים ומשלוח מנות.</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה , אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, שֶׁהֶחֱיָֽינוּ וקִיְּמָנוּ וְהִגִּיעָנוּ לַזּמַן הַזֶּה:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">קוראים את המגילה</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">אחר קריאת המגילה גוללה ואחר־כך מברך:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה , אֱלֹהֵֽינוּ מֶלֶךְ הָעוֹלָם, <span style="font-size:95%; color:RGB(0,0,255); text-shadow:0 0px 0 white;">הַאֵל</span> הָרָב אֶת־רִיבֵנוּ, וְהַדָּן אֶת־דִּינֵנוּ, וְהַנּוֹקֵם אֶת־נִקְמָתֵנוּ, וְהַמְשַׁלֵּם גְּמּוּל לְכָל־אוֹיְבֵי נַפְשֵׁנוּ, וְהַנִּפְרָע לָנוּ מִצָּרֵֽינוּ. בָּרוּךְ אַתָּה יְהֹוָה , הַנִּפְרָע לְעַמּוֹ יִשְׂרָאֵל מִכּל־צָרֵיהֶם הָאֵל הַמּוֹשִׁיעַ:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">אחר־כך אומרים:</span>
אָרוּר הָמָן בָּרוּךְ מרְדּכַי.
אֲרוּרָה זֶרֶשׁ בְּרוּכָה אֶסְתֵּר.
אֲרוּרִים כָּל־הָרְשָׁעִים בְּרוּכִים כָּל־יִשְׂרָאֵל, וְגַם חַרְבוֹנָה זָכוּר לטוֹב: <br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">שלש פעמים</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">אומרים "ואתה קדוש" והחזן אומר קדיש תתקבל, (ויש מוסיפים "שיר המעלות לדָוִד, לולי ה' שהיה לָנוּ" וגו') ואח"כ "שיר למעלות, אשא עיני" וגו', ואומרים קדיש יהא שלמא, ברכו, ו"עָלֵינוּ לשבח".</span>
<h2>סֵדֶר יוֹם פּוּרִים</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ביום פורים מתפללים כמו כל יום חול ואומרים בהודאה על הנִסים, ולאחר העמידה חצי קדיש, ומוציאים ס"ת וקוראים ג' עולים בפרשת בשלח, ונוהגים לחזור על הפסוק האחרון פעם נוספת כְּדֵי להשלים לעשרה פסוקים:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַיָּבֹא עֲמָלֵק</span> וַיִּלָּחֶם עִם־יִשְׂרָאֵל בִּרְפִידִם: וַיֹּאמֶר מֹשֶׁה אֶל־יְהוֹשֻׁעַ בְּחַר־לָנוּ אֲנָשִׁים וְצֵא הִלָּחֵם בַּעֲמָלֵק מָחָר אָנֹכִי נִצָּב עַל־רֹאשׁ הַגִּבְעָה וּמַטֵּה הָאֱלֹהִים בְּיָדִי: וַיַּעַשׂ יְהוֹשֻׁעַ כַּאֲשֶׁר אָמַר־לוֹ מֹשֶׁה לְהִלָּחֵם בַּעֲמָלֵק וּמֹשֶׁה אַהֲרֹן וְחוּר עָלוּ רֹאשׁ הַגִּבְעָה: וְהָיָה כַּאֲשֶׁר יָרִים מֹשֶׁה יָדוֹ וְגָבַר יִשְׂרָאֵל וְכַאֲשֶׁר יָנִיחַ יָדוֹ וְגָבַר עֲמָלֵק: וִידֵי מֹשֶׁה כְּבֵדִים וַיִּקְחוּ־אֶבֶן וַיָּשִׂימוּ תַחְתָּיו וַיֵּשֶׁב עָלֶיהָ וְאַהֲרֹן וְחוּר תָּמְכוּ בְיָדָיו מִזֶּה אֶחָד וּמִזֶּה אֶחָד וַיְהִי יָדָיו אֱמוּנָה עַד־בֹּא הַשָּׁמֶשׁ: וַיַּחֲלֹשׁ יְהוֹשֻׁעַ אֶת־עֲמָלֵק וְאֶת־עַמּוֹ לְפִי־חָרֶב: <br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ישראל</span> וַיֹּאמֶר יְהוָה אֶל־מֹשֶׁה כְּתֹב זֹאת זִכָּרוֹן בַּסֵּפֶר וְשִׂים בְּאָזְנֵי יְהוֹשֻׁעַ כִּי־מָחֹה אֶמְחֶה אֶת־זֵכֶר עֲמָלֵק מִתַּחַת הַשָּׁמָיִם: וַיִּבֶן מֹשֶׁה מִזְבֵּחַ וַיִּקְרָא שְׁמוֹ יְהוָה נִסִּי: וַיֹּאמֶר כִּי־יָד עַל־כֵּס יָהּ מִלְחָמָה לַיהוָה בַּעֲמָלֵק מִדֹּר דֹּר:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ואומר החזן חצי קדיש, אשרי, ובא לציון עד מעתה ועד עולם, וקורין המגילה בברכותיה לפניה ולאחריה. אחר המגילה אומרים ואתה קדוש, קדיש תתקבל, יהללו, ומחזירין הס"ת למקומו, ואומרים בית יעקב, שיר של יום, הושיענו, למנצח על אילת השחר, קדיש יהא שלמא, וגומרים התפלה כשאר ימים:</span>
