<h2>שלום עליכם</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">כשיבוא לביתו ינשק ידי אביו ואמו, כי יש סוד בדבר, ואח"כ יסתכל באור הנר אשר ברכו עליו, ויסתכל בשני הנרות ויכוין שהם אחד כנגד זכור ושמור. ואח"כ יעמוד במקום שישב שם אצל השולחן, ויאמר בקול רם: דָּא הִיא סעֻדָּתָא דַּחֲקַל תַּפּוּחִין קַדִּישִׁין, ויקיף את השלחן פעם אחת, ויקח שתי אגודות הדס (אחד כנגד זכור ואחד כנגד שמור) ויברך עליהם ויריח, ויאמר בפיו: זָכוֹר ושָׁמוֹר בּדִבּוּר אֶחָד נֶאֱמָרוּ, וגם יאמר בלחש: רֵיחַ נִיחֹחַ אִשֶּׁה לַיהֹוָה, ויקיף בשתיקה פעם שניה את שלחנו בעוד ההדס בידו (אם אין לו הדס יקח מיני בשמים אחרים). ויאמר בשמחה תפלה זו המהוה את קבלת הפנים למלאכים המלוים את היהודי בדרכו מבית הכנסת לביתו (שבת קי"ט).</span>

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרְכוּנוּ לְשָלוֹם</span> מַלְאֲכֵי הַשָּׁלוֹם, מַלְאֲכֵי עֶלְיוֹן, מֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ג' פעמים</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בְּשִׁבְתְּכֶם לְשָלוֹם</span> מַלְאֲכֵי הַשָּׁלוֹם, מַלְאֲכֵי עֶלְיוֹן, מֶלֶךְ מַלְכֵי הַמְּלָכִים הַקָּדוֹשׁ בָּרוּךְ הוּא: <br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ג' פעמים - בבית שאין שלום בית רח''ל לא יאמרו - רבי מרדכי שרעבי זצוק''ל</span>

<h2>אשת חיל</h2>

<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">י</span>וֹם <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַ</span>שִּׁשִּׁי: <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">וַ</span>יְכֻלּוּ <br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">הַ</span>שָּׁמַיִם וְהָאָרֶץ וְכָל צְבָאָם: וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי מְלַאכְתּוֹ אֲשֶׁר עָשָׂה וַיִּשְׁבֹּת בַּיּוֹם הַשְּׁבִיעִי מִכָּל מְלַאכְתּוֹ אֲשֶׁר עָשָׂה: וַיְבָרֶךְ אֱלֹהִים אֶת יוֹם הַשְּׁבִיעִי וַיְקַדֵּשׁ אֹתוֹ כִּי בוֹ שָׁבַת מִכָּל מְלַאכְתּוֹ אֲשֶׁר בָּרָא אֱלֹהִים לַעֲשׂוֹת:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ואומר המקדש</span> סַבְרֵי מָרָנָן, <br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ועונים</span> <br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;">לחַיִּים</span>.
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, בּוֹרֵא פְּרִי הַגֶּפֶן:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">יכוון לקיים מצות עשה דאורייתא של הקידוש, ועל היין מצוה דרבנן.</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בְּמִצְוֹתָיו, וְרָצָה בָנוּ, וְשַׁבַּת קָדְשׁוֹ בְּאַהֲבָה וּבְרָצוֹן הִנְחִילָנוּ, זִכָּרוֹן לְמַעֲשֵׂה בְרֵאשִׁית, תְּחִלָּה לְמִקְרָאֵי קֹדֶשׁ, זֵכֶר לִיצִיאַת מִצְרַיִם. וְשַׁבַּת קָדְשְׁךָ בְּאַהֲבָה וּבְרָצוֹן הִנְחַלְתָּנוּ. בָּרוּךְ אַתָּה יְהֹוָה, מְקַדֵּשׁ הַשַּׁבָּת:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ישב וישתה רביעית או מלוא לוגמיו, ולא יברך אחריו ברכה אחרונה, אף אם שתה רביעית.</span>
<h2>ברכת הבנים</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">על הבנים לנשק ידי הוריהם, וטוב שהאב יניח ידיו על ראשם ויברכם</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;"><br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לבן</span>:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְשִׂמְךָ</span> אֱלֹהִים כְּאֶפְרַיִם וְכִמְנַשֶּׁה: יְבָרֶכְךָ יְהֹוָה וְיִשְׁמְרֶךָ: יָאֵר יְהֹוָה | פָּנָיו אֵלֶיךָ וִיחֻנֶּךָּ: יִשָּׂא יְהֹוָה | פָּנָיו אֵלֶיךָ, וְיָשֵׂם לְךָ שָׁלוֹם: וְשָׂמוּ אֶת שְׁמִי עַל בְּנֵי יִשְׂרָאֵל, וַאֲנִי אֲבָרְכֵם.
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;"><br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">לבת</span>:</span>
<br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">יְשִׂימֵךְ</span> אֱלֹהִים כְּשָׂרָה, רִבְקָה, רָחֵל וְלֵאָה: יְבָרֶכְךָ יְהֹוָה וְיִשְׁמְרֶךָ: יָאֵר יְהֹוָה | פָּנָיו אֵלֶיךָ וִיחֻנֶּךָּ: יִשָּׂא יְהֹוָה | פָּנָיו אֵלֶיךָ, וְיָשֵׂם לְךָ שָׁלוֹם: וְשָׂמוּ אֶת שְׁמִי עַל בְּנֵי יִשְׂרָאֵל, וַאֲנִי אֲבָרְכֵם.
<h2>סעודה ראשונה</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">ימהר ליטול ידיו לסעודה אחר הקידוש. וקודם נטילת ידים יש נוהגים לומר לשם יחוד זה, שהובא בבא"ח</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְשֵׁם יִחוּד</span> קֻדְשָׁא בְּרִיךְ הוּא וּשְכִינְתֵּהּ בִּדְחִילוּ וּרְחִימוּ וּרְחִימוּ וּדְחִילוּ לְיַחֲדָא שֵׁם אוֹת יוּ"ד אוֹת הֵ"א בּאוֹת וָא"ו אוֹת הֵ"א, בְּיִחוּדָא שְׁלִים בְּשֵׁם כָּל יִשְׂרָאֵל, לַאֲקָמָא שְׁכִינְתָּא מֵעַפְרָא וּלְעִלוּי שְׁכִינַת עֻזֵּנוּ, הֲרֵינִי בָא לְקַיֵּם מִצְוַת עֲשֵׂה דְּרַבָּנָן לִטּוֹל יָדַי וּלְשַׁפְשְׁפָן הֵיטֵב קֹדֶם סְעֻדָה, כָּדֵי לַעֲשׂוֹת נַחַת רוּחַ לְיוֹצְרִי וְלַעֲשׂוֹת רְצוֹן בּוֹרְאִי, לְתַקֵּן שֹׁרֶשׁ מִצְוָה זוֹ בְּמָקוֹם עֶלְיוֹן. וִיהִי רָצוֹן מִלְּפָנֶיךָ, יְהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, שֶׁתִּהְיֶה חֲשׁוּבָה וּמְקֻבֶּלֶת וּרְצוּיָה לְפָנֶיךָ מִצְוָה זוֹ שֶׁל נְטִילַת יָדַיִם וְהַשִּׁפְשׁוּף קֹדֶם סעֻדָּה, כְּאִלּוּ כִוַּנְתִּי בְּכָל הַכַּוָּנוֹת הָראוּיוֹת לכַוֵּן בְּמִצְוָה זוֹ. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וִיהִי רָצוֹן</span> מִלְּפָנֶיךָ, יהֹוָה אֱלֹהֵינוּ וֵאלֹהֵי אֲבוֹתֵינוּ, שֶׁבִּזְכוּת מִצְוַת הַבְּרָכָה אֲשֶׁר נְבָרֵךְ עַל נְטִילַת יָדַיִם קֹדֶם סעֻדָה, יִהְיֶה עַתָּה עֵת רָצוֹן לפָנֶיךָ, וְיִמָּשֵׁךְ מִן אוֹר אֵין סוֹף שֶׁפַע מֹחִין ואוֹרוֹת גְּדוֹלִים לִשְׁנֵים עָשָׂר פַּרְצוּפִים, הָרְמוּזִים בְּעֶשֶׂר סְפִירוֹת הַכּוֹללוֹת, שֶׁהֵן רְמוּזוֹת בְּעֶשֶׂר אֶצְבּעוֹת הַיָּדַיִם, וּמִשָּׁם נְקַבֵּל שֶׁפַע עֶשֶׂר הַבְּרָכוֹת, כַּכָּתוּב: ויִתֶּן לְךָ הָאֱלֹהִים מִטַּל הַשָּׁמַיִם וּמִשְׁמַנֵּי הָאָרֶץ ורֹב דָּגָן וְתִירֹשׁ: יַעַבְדוּךָ עַמִּים ויִשְׁתַּחֲווּ לְךָ לְאֻמִּים, הֱוֵה גְּבִיר לְאַחֶיךָ, וְיִשְׁתַּחֲווּ לְךָ בְּנֵי אִמֶּךָ. אוֹרֲרֶיךָ אָרוּר, וּמְבָרְכֶיךָ בָּרוּךְ:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, אֲשֶׁר קִדְּשָׁנוּ בּמִצְוֹתָיו, וצִוָּנוּ עַל נְטִילַת יָדָיִם:
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">יאחז שתי ככרות פת בשתי ידיו, ויברך המוציא. ויש נוהגים לומר פסוקים אלו קודם הברכה</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">עֵינֵי כֹל</span> אֵלֶיךָ ישַׂבֵּרוּ ואַתָּה נוֹתֵן לָהֵם אֶת אכְלָם בּעִתוֹ: פּוֹתֵחַ אֶת יָדֶךָ וּמַשְׂבִּיעַ לכל חַי רָצוֹן:</span>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בָּרוּךְ</span> אַתָּה יְהֹוָה, אֱלֹהֵינוּ מֶלֶךְ הָעוֹלָם, הַמּוֹצִיא לֶחֶם | מִן הָאָרֶץ.
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">יבצע פרוסה ויטבול אותה במלח שלש פעמים ויכוין שלש פעמים הוי"ה, ויטעם ממנה, ואח"כ יתן פרוסות לחם לכל המסובים.</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">לפני הסעודה טוב לומר:</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">לְשֵׁם יִחוּד</span> קֻדְשָׁא בְּרִיךְ הוּא וּשְכִינְתֵּהּ בִּדְחִילוּ וּרְחִימוּ וּרְחִימוּ וּדְחִילוּ לְיַחֲדָא שֵׁם אוֹת יוּ"ד אוֹת הֵ"א בּאוֹת וָא"ו אוֹת הֵ"א, בְּיִחוּדָא שְׁלִים בּשֵׁם כָּל יִשְׂרָאֵל, לַאֲקָמָא שְׁכִינְתָּא מֵעַפְרָא וּלְעִלוּי שְׁכִינַת עֻזֵּנוּ, הֲרֵינִי מוּכָן לְקַיֵּם מִצְוַת סְעֻדָּה רִאשׁוֹנָה שֶׁל שַׁבָּת, לְתַקֵּן אֶת שָׁרְשָׁהּ בְּמָקוֹם עֶלְיוֹן, והִיא כנֶגֶד יִצְחָק אַבָּא קַדִּישָׁא. וּבִזְכוּתוֹ נִנָּצֵל מֵחֶבְלוֹ שֶׁל מָשִׁיחַ, וִיקֻיַּם בָּנוּ מִקְרָא שֶׁכָּתוּב: "וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ". וִיהִי נֹעַם אֲדֹנָי אֱלֹהֵינוּ עָלֵינוּ, וּמַעֲשֵׂה יָדֵינוּ כּוֹננָה עָלֵינוּ, וּמַעֲשֵׂה יָדֵינוּ כּוֹנְנֵהוּ:</span>
<h2>זוהר לסעודה ראשונה</h2>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">טוב לומר זוהר זה מדבר בכבוד השבת - יחרו פ''ח ע''א</span>
<br/><span style="color:RGB(0,127,127); font-family: arial; font-size:85%;text-shadow:0 0px 0 white;"><span style="font-weight:bold;text-shadow:0 0px 0 white;">זָכוֹר</span> אֶת יוֹם הַשַּׁבָּת לְקַדְּשׁוֹ. רַבִּי יִצְחָק אָמַר, כְּתִיב וַיְבָרֶךְ אֱלהִים אֶת יוֹם הַשְּׁבִיעִי, וּכְתִיב בַּמָּן שֵׁשֶׁת יָמִים תִּלְקְטֻהוּ וּבְיוֹם הַשְּׁבִיעִי שַׁבָּת לֹא יִהְיֶה בּוֹ. כֵּיוָן דְּלָא מִשְׁתְּכַח בֵּיהּ מְזוֹנֵי, מַה בִּרְכְתָא אִשְׁתְּכַח בֵּיהּ. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אֶלָּא</span> הָכִי תָּאנָא, כָּל בִּרְכָּאן דִּלְעֵילָּא וְתַתָּא, בְּיוֹמָא שְׁבִיעָאָה תַּלְיָין. וְתָאנָא, אֲמַאי לָא אִשְׁתְּכַח מָנָּא בְּיוֹמָא שְׁבִיעָאָה, מִשּׁוּם דְּהַהוּא יוֹמָא, מִתְבָּרְכָאן מִינֵּיהּ כָּל שִׁיתָא יוֹמִין עִלָּאִין, וְכָל חַד וְחַד יָהִיב מְזוֹנֵיהּ לְתַתָּא, כָּל חַד בְּיוֹמוֹי, מֵהַהִיא בְּרָכָה דְּמִתְבָּרְכָאן בְּיוֹמָא שְׁבִיעָאָה. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וּבְגִינֵי</span> כַּךְ, מַאן דְּאִיהוּ בְּדַרְגָּא דִּמְהֵימָנוּתָא, בָּעֵי לְסַדְּרָא פָּתוֹרָא, וּלְאַתְקְנָא סְעוּדָתָא בְּלֵילְיָא דְּשַׁבְּתָא, בְּגִין דְּיִתְבָּרֵךְ פָּתוֹרֵיהּ, כָּל אִינּוּן שִׁיתָא יוֹמִין, דְּהָא בְּהַהוּא זִמְנָא, אִזְדָּמַן בְּרָכָה, לְאִתְבָּרְכָא כָּל שִׁיתָא יוֹמִין דְּשַׁבְּתָא, וּבִרְכָתָא לָא אִשְׁתְּכַח בְּפָתוֹרָא רֵיקָנַיָּא. וְעַל כַּךְ, בָּעֵי לְסַדְּרָא פָּתוֹרֵיהּ בְּלֵילְיָא דְּשַׁבְּתָא, בְּנַהֲמֵי וּבִמְזוֹנֵי. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רַבִּי יִצְחָק</span> אָמַר, אֲפִילוּ בְּיוֹמָא דְּשַׁבְּתָא נָמֵי. רַבִּי יְהוּדָה אָמַר, בָּעֵי לְאִתְעַנְגָּא בְּהַאי יוֹמָא, וּלְמֵיכַל תְּלַת סְעוּדָתֵי בְּשַׁבְּתָא, בְּגִין דְּיִשְׁתְּכַח שָׂבְעָא וְעִנּוּגָא בְּהַאי יוֹמָא בְּעָלְמָא. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רַבִּי אַבָּא</span> אָמַר, לְאִזְדַּמְּנָא בִּרְכְתָא בְּאִינּוּן יוֹמִין דִּלְעֵילָּא, דְּמִתְבָּרְכָאן מֵהַאי יוֹמָא. וְהַאי יוֹמָא, מַלְיָא רֵישֵׁיהּ דִּזְעֵיר אַנְפִּין, מִטַּלָּא דְּנָחִית מֵעַתִּיקָא קַדִּישָׁא סְתִימָא דְּכֹלָּא, וְאָטִיל לְחַקְלָא דְּתַפּוּחִין קַדִּישִׁין, תְּלַת זִמְנֵי, מִכַּד עָיִיִל שַׁבְּתָא, דְּיִתְבָּרְכוּן כֻּלְּהוּ כַּחֲדָא. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וְעַל</span> דָּא בָּעֵי בַּר נָשׁ, לְאִתְעַנְגָּא תְּלַת זִמְנִין אִלֵּין, דְּהָא בְּהָא תַּלְיָא מְהֵימָנוּתָא דִּלְעֵילָּא, בְּעַתִּיקָא קַדִּישָׁא, וּבִזְעֵיר אַפִּין, וּבְחַקְלָא דְּתַפּוּחִין. וּבָעֵי בַּר נָשׁ לְאִתְעַנְגָּא בְּהוּ, וּלְמֵחדֵי בְּהוּ. וּמַאן דְּגָרַע סְעוּדָתָא מִנַיְיהוּ, אַחְזֵי פְּגִימוּתָא לְעֵילָא וְעוֹנְשֵׁיהּ דְּהַהוּא בַּר נָשׁ סַגִּי. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">בְּגִינֵי</span> כַּךְ, בָּעֵי לְסַדְּרָא פָּתוֹרֵיהּ, תְּלַת זִמְנֵי, מִכַּד עָיִיִל שַׁבְּתָא, וְלָא יִשְׁתְּכַּח פָּתוֹרֵיהּ רֵיקָנַיָּא, וְתִשְׁרֵי בִּרְכְתָא עֲלֵיהּ, כָּל שְׁאַר יוֹמֵי דְּשַׁבְּתָא. וּבְהַאי מִלָּה, אַחְזֵי, וְתָלֵי מְהֵימָנוּתָא לְעֵילָּא. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רַבִּי שִׁמְעוֹן</span> אָמַר, הַאי מַאן דְּאַשְׁלִים תְּלַת סְעוּדָתֵי בְּשַׁבָּתָא, קָלָא נָפִיק וּמַכְרְזָא עָלֵיהּ, אָז תִּתְעַנַּג עַל יְיָ', דָּא סְעוּדָתָא חֲדָא, לָקֳבֵל עַתִּיקָא קַדִּישָׁא דְּכָל קַדִּישִׁין. וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ, דָּא סְעוּדָתָא תִּנְיָינָא, לָקֳבֵל חַקְלָא דְּתַפּוּחִין קַדִּישִׁין. וְהַאֲכַלְתִּיךָ נַחֲלַת יַעֲקֹב אָבִיךָ, דָּא הוּא שְׁלִימוּ דְּאִשְׁתְּלִים בִּזְעֵיר אַפִּין. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">וּלֳקָבְלַיְיהוּ</span> בָּעֵי לְאַשְׁלְמָא סְעוּדָתֵיהּ, וּבָעֵי לְאִתְעַנְגָּא בְּכֻלְּהוּ סְעוּדָתֵי, וּלְמֵחדֵי בְּכָל חַד וְחַד מִנַּיְיהוּ, מִשּׁוּם דְּאִיהוּ מְהֵימָנוּתָא שְׁלֵימָתָא. וּבְגִין כַּךְ, שַׁבְּתָא אִתְיָקָּר, מִכָּל שְׁאַר זִמְנִין וְחַגִּין, מִשּׁוּם דְּכֹלָּא בֵּיהּ אִשְׁתְּכַח, וְלָא אִשְׁתְּכַח הָכִי בְּכֻלְּהוּ זִמְנֵי וְחַגֵּי. אָמַר רַבִּי חִיָּיא, בְּגִין כָּךְ, מִשּׁוּם דְּאִשְׁתְּכַח כֹּלָּא בֵּיהּ, אִידְכַּר תְּלַת זִמְנִין. דִּכְתִּיב, וַיְכַל אֱלֹהִים בַּיּוֹם הַשְּׁבִיעִי. וַיִּשְׁבּוֹת בַּיּוֹם הַשְּׁבִיעִי. וַיְבָרֶךְ אֱלהִים אֶת יוֹם הַשְּׁבִיעִי. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רַבִּי אַבָּא</span>, כַּד הֲוָה יָתִיב בִּסְעוּדָתָא דְּשַׁבְּתָא, הֲוֵי חַדֵּי, בְּכָל חַד וְחַד, וַהֲוָה אָמַר, דָּא הִיא סְעוּדָתָא קַדִּישָׁא, דְּעַתִּיקָא קַדִּישָׁא סְתִימָא דְּכֹלָּא. בִּסְעוּדָתָא אַחֲרָא הֲוָה אָמַר, דָּא הִיא סְעוּדָתָא דְּקוּדְשָׁא בְּרִיךְ הוּא. וְכֵן בְּכֻלְּהוּ סְעוּדָתֵי, וַהֲוָה חַדֵּי בְּכָל חַד וְחַד. כַּד הֲוָה אַשְׁלִים סְעוּדָתֵי, אָמַר אַשְׁלִימוּ סְעוּדָתֵי דִּמְהֵימָנוּתָא. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">רַבִּי שִׁמְעוֹן</span>, כַּד הֲוָה אָתֵי לִסְעוּדָתָא, הֲוָה אָמַר הָכִי, אַתְקִינוּ סְעוּדָתָא דִּמְהֵימְנוּתָא עִלָּאָה, אַתְקִינוּ סְעוּדָתָא דְּמַלְכָּא, וַהֲוָה יָתִיב וְחַדֵּי. כַּד אַשְׁלִים סְעוּדָתָא תְּלִיתָאָה, הֲווֹ מַכְרִזֵי עָלֵיהּ, אָז תִּתְעַנַּג עַל יְיָ' וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ וְהַאֲכַלְתִּיךָ נַחֲלַת יַעֲקֹב אָבִיךָ. <br/><span style="font-weight:bold;text-shadow:0 0px 0 white;">אָמַר רַבִּי אֶלְעָזָר</span> לְאָבוּי, אִלֵּין סְעוּדָתֵי הֵיךְ מִתְתַּקְנִין. אָמַר לֵיהּ, לֵילְיָא דְּשַׁבְּתָא, כְּתִיב, וְהִרְכַּבְתִּיךָ עַל בָּמֳתֵי אָרֶץ. בֵּיהּ בְּלֵילְיָא, מִתְבָּרְכָא מַטְרוֹנִיתָא, וְכֻלְּהוּ חֲקַל תַּפּוּחִין, וּמִתְבָּרְכָא פָּתוֹרֵיהּ דְּבַר נָשׁ, וְנִשְׁמְתָא אִתּוֹסְפַת, וְהַהוּא לֵילְיָא, חֶדְוָה דְּמַטְרוֹנִיתָא הֲוֵי. וּבָעֵי בַּר נָשׁ לְמֶחדֵי בְּחֶדְוָותָא, וּלְמֵיכַל סְעוּדָתָא דְּמַטְרוֹנִיתָא. בָּרוּךְ יְהֹוָה לְעוֹלָם אָמֵן ואָמֵן:</span>
<h2>שירי שבת</h2>
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">בר יוחאי</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">סימן שמעון לביא</span>
<br/>בַּר יוֹחָאי, נִמְשַׁחְתָּ אַשְׁרֶיךָ, שֶׁמֶן שָׂשׂוֹן מֵחֲבֵרֶיךָ:
<br/>בַּר יוֹחָאי, שֶׁמֶן מִשְׁחַת קֹדֶשׁ, נִמְשַׁחְתָּ מִמִּדַּת הַקֹּדֶשׁ, נָשָׂאתָ צִיץ נֵזֶר הַקֹּדֶשׁ, חָבוּשׁ עַל רֹאשׁךָ פּאֵרֶךָ:
<br/>בַּר יוֹחָאי, מוֹשַׁב טוֹב יָשַׁבְתָּ, יוֹם נַסְתָּ יוֹם אֲשֶׁר בָּרַחְתָּ, בִּמְעָרַת צוּרִים שֶׁעָמַדְתָּ, שָׁם קָנִיתָ הוֹדךָ וַהֲדָרֶךָ:
<br/>בַּר יוֹחָאי, עֲצֵי שִׁטִּים עוֹמדִים, לִמּוּדֵי יְהֹוָה הֵם לוֹמדִים, אוֹר מֻפְלֶא אוֹר הַיְקוֹד הֵם יוֹקדִים, הֲלֹא הֵמָּה יוֹרוּךָ מוֹרֶיךָ:
<br/>בַּר יוֹחָאי, ולִשְׂדֵה תַּפּוּחִים, עָלִיתָ לִלְקוֹט בּוֹ מֶרְקָחִים, סוֹד תּוֹרָה כּצִיצִים וּפְרָחִים, נַעֲשֶׂה אָדָם נֶאֱמַר בַּעֲבוּרֶךָ:
<br/>בַּר יוֹחָאי, נֶאֱזַרְתָּ בִּגְבוּרָה, וּבְמִלְחֶמֶת אֵשׁ דַּת הַשַּׁעְרָה, וחֶרֶב הוֹצֵאתָ מִתַּעְרָהּ, שָׁלַפְתָּ נֶגֶד צֹררֶיךָ:
<br/>בַּר יוֹחָאי, לִמְקוֹם אַבְנֵי שַׁיִשׁ, הִגַּעְתָּ לִפְנֵי אַרְיֵה לַיִשׁ, גַּם גֻּלַּת כּוֹתֶרֶת עַל עַיִשׁ, תָּשׁוּרִי וּמִי ישׁוּרֶךָ:
<br/>בַּר יוֹחָאי, בּקֹדֶשׁ הַקֳּדָשִׁים, קַו יָרוֹק מחַדֵּשׁ חֳדָשִׁים, שֶׁבַע שַׁבָּתוֹת סוֹד חֲמִשִּׁים, קָשַׁרְתָּ קִשְׁרֵי שִׁי"ן קשָׁרֶיךָ:
<br/>בַּר יוֹחָאי, יוֹ"ד חָכמָה קדוּמָה, הִשְׁקַפְתָּ לִכְבוֹדוֹ פּנִימָה, שׁלֹשִׁים וּשְׁתַּיִם נתִיבוֹת רֵאשִׁית תּרוּמָה, אַתְּ כּרוּב מִמְשַׁח זִיו אוֹרֶךָ:

<br/>הִלּוּכְךָ יְהֵא בְנַחַת, עֹנֶג קְרָא לַשַּׁבָּת. וְהַשֵּׁנָה מְשֻׁבַּחַת, כְּדָת נֶפֶשׁ מְשִׁיבַת. בְּכֵן נַפְשִׁי לךָ עָרְגָה, וְלָנוּחַ בְּחִבַּת. כַּשּׁוֹשַׁנִּים סוּגָה, בּוֹ יָנוּחוּ בֵּן וּבַת:
<br/>לְהִתְעַנֵּג בְּתַעֲנוּגִים בַּרְבּוּרִים וּשְׂלָו וְדָגִים:
<br/>מֵעֵין עוֹלָם הַבָּא, יוֹם שַׁבַּת מְנוּחָה. כּל הַמִּתְעַנְּגִים בָּהּ, יִזְכּוּ לְרֹב שִׂמְחָה. מֵחֶבְלֵי מָשִׁיחַ, יֻצָּלוּ לִרְוָחָה. פְּדוּתֵנוּ תַּצְמִיחַ, וְנָס יָגוֹן וַאֲנָחָה:
<br/>לְהִתְעַנֵּג בְּתַעֲנוּגִים בַּרְבּוּרִים וּשְׂלָו וְדָגִים:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יודוך רעיוני</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">מלים: רבי ישראל נג'ארה זצ"ל.</span>
<br/>יוֹדוּךָ רַעְיוֹנַי יְהֹוָה רוֹעִי. בְּיוֹם שַׁבַּת קֹדֶשׁ יוֹם הַשְּׁבִיעִי:
<br/>יוֹם אֲשֶׁר כִּלִּיתָ, בּוֹ כּל מְלַאכְתֶּךָ. אוֹמַר כִּי שָׂרִיתָ, עַל כּל זוּלָתֶךָ. וּמַעֲשִׂים עָשִׂיתָ, אֵין לְבַלּוֹתֶךָ. לִי בֶן אֲמָתֶךָ, חִישׁ לְהַרְגִּיעִי:
<br/>שְׁבִיעִי בָּחַרְתָּ, מִכּל הַמִּנְיָנִים. וְאוֹתוֹ קִדַּשְׁתָּ, בְּשָׁבוּעוֹת וְשָׁנִים. חִישׁ אֲשֶׁר נָשָׂאתָ, לְגֶזַע אֱמוּנִים. וּפְדֵם מֵאֲסוֹנִים, אָרְחִי וְרִבְעִי:
<br/>רְצֵה בִמְנוּחָתִי, יוֹם זֶה יוֹם מְנוּחָה. וּבְיוֹם עֲבוֹדָתִי, הַמְצִא לִי הָרְוָחָה. וְהָכֵן לִשְׁבִיתָתִי, מַשְׂאֵת וַאֲרוּחָה. וְשָׂשׂוֹן וְשִׂמְחָה, יִהְיֶה שַׁעֲשׁוּעִי:
<br/>אֶל עוֹלָם שֶׁכֻּלּוֹ, שַׁבָּת תְּזַכֵּנִי. וְנֵרְךָ בְהִלּוֹ, שִׂים אוֹר בַּעֲדֵנִי. וְאֶל מִשְׁכַּן שִׁילֹה, תָּשׁוּב תַּעֲלֵנִי. מְהֵרָה עֲנֵנִי, אוֹרִי וְיִשְׁעִי:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">ברוך אל עליון</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">סימן: ברוך חזק.</span>
<br/>בָּרוּךְ אֵל עֶלְיוֹן, אֲשֶׁר נָתַן מְנוּחָה. לְנַפְשֵׁנוּ פִּדְיוֹן, מִשֵּׁאת וַאֲנָחָה. וְהוּא יִדְרֹשׁ לְצִיּוֹן, עִיר הַנִּדָּחָה. עַד אָנָה תוּגְיוֹן, נֶפֶשׁ נֶאֱנָחָה. הַשּׁוֹמֵר שַׁבָּת, הַבֵּן עִם הַבַּת. לָאֵל יֵרָצוּ, כְּמִנְחָה עַל מַחֲבַת:
<br/>רוֹכֵב בָּעֲרָבוֹת, מֶלֶךְ עוֹלָמִים. אֶת עַמּוֹ לִשְׁבּוֹת, אִזֵּן בַּנְּעִימִים. בְּמַאֲכָלֵי עֲרֵבוֹת, בְּמִינֵי מַטְעַמִּים. בְּמַלְבּוּשֵׁי כָבוֹד, זֶבַח מִשְׁפָּחָה: השומר
<br/>וְאַשְׁרֵי כָל חוֹכֶה, לְתַשְׁלוּמֵי כֶפֶל. מֵאֵת כָּל סוֹכֶה, שׁוֹכֵן בָּעֲרָפֶל. נַחֲלָה לוֹ יִזְכֶּה, בָּהָר וּבַשָּׁפֶל. נַחֲלָה וּמְנוּחָה, כַּשֶּׁמֶשׁ לוֹ זָרְחָה: השומר
<br/>כּל שׁוֹמֵר שַׁבָּת, כַּדָּת מֵחַלְּלוֹ. הֵן הֶכְשֵׁר חִבַּת, קֹדֶשׁ גּוֹרָלוֹ. וְאִם יֵצֵא חוֹבַת, הַיּוֹם אַשְׁרֵי לוֹ. אֶל אֵל אָדוֹן מְחוֹלְלוֹ, מִנְחָה הִיא שְׁלוּחָה: השומר
<br/>חֶמְדַּת הַיָּמִים, קְרָאוֹ אֵלִי צוּר. וְאַשְׁרֵי לִתְמִימִים, אִם יִהְיֶה נָצוּר. כֶּתֶר הִלּוּמִים, עַל רֹאשָׁם יָצוּר. צוּר הָעוֹלָמִים, רוּחוֹ בָם נָחָה: השומר
<br/>זָכוֹר אֶת יוֹם הַשַּׁבָּת, לְקַדְּשׁוֹ. קַרְנוֹ כִּי גָבְהָה, נֵזֶר עַל רֹאשׁוֹ. עַל כֵּן יִתֵּן הָאָדָם, לְנַפְשׁוֹ. עֹנֶג וְגַם שִׂמְחָה, בָּהֶם לְמשְׁחָה: השומר
<br/>קֹדֶשׁ הִיא לָכֶם, שַׁבָּת הַמַּלְכָּה. אֶל תּוֹךְ בָּתֵּיכֶם, לְהָנִיחַ בְּרָכָה. בְּכל מוֹשְׁבוֹתֵיכֶם, לֹא תַעֲשׂוּ מְלָאכָה. בְּנֵיכֶם וּבְנוֹתֵיכֶם, עֶבֶד וְגַם שִׁפְחָה: השומר
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">יום השבת</span>
<br/>יוֹם הַשַּׁבָּת אֵין כָּמוֹהוּ. בֵּרֲכוֹ אֱלֹהִים וַיְקַדְּשֵׁהוּ:
<br/>מִיּוֹם רִאשׁוֹן עַד יוֹם שִׁשִּׁי. לִקְרַאת שַׁבָּת נִכְסְפָה נַפְשִׁי. כִּי בַשַּׁבָּת אֵצֵא חָפְשִׁי. וְכִי יוֹם מְנוּחָה אֶקְרָאֵהוּ:
<br/>נָגִיל וְנִשְׂמַח כִּי טוֹב לְהוֹדוֹת. וּלְזַמֵּר עֶלְיוֹן עַל כּל אוֹדוֹת. כִּי הַשַּׁבָּת לָנוּ חֲמוּדוֹת. אִישׁ בַּעַר לֹא יְדָעֵהוּ:
<br/>צִוָּה לַעֲשׂוֹת שַׁבָּת כַּהֲלָכָה. אָז מִסִּינַי מִצְוָה עֲרוּכָה. גַּם הִיא לְשׁוֹמְרָהּ. כֶּתֶר מְלוּכָה. וְכָבוֹד וְהָדָר תְּעַטְּרֵהוּ:
<br/>וּמְאַהֲבֶיהָ חַיִּים יַאֲרִיכוּ. וְגַם טוֹעֲמֶיהָ חַיִּים זָכוּ. וּבִנְתִיב יוֹשֶׁר הֵמָּה יֵלֵכוּ. עֵדוֹתָיו בְּכלּ לֵב יִדְרְשׁוּהוּ:
<br/>רָצָה יְהֹוָה וּבָחַר בָּנוּ. וּמִכּל לָשׁוֹן הִבְדִּילָנוּ. וּקְדֻשַּׁת הַשַׁבָּת הִנְחִיל לָנוּ. כּל זֶרַע יַעֲקֹב כַּבְּדוּהוּ:
<br/><span style="font-size:110%; font-weight:bold;text-shadow:0 0px 0 white;">צור משלו אכלנו</span>
<br/><span style="color:RGB(127,127,127); font-family: arial; font-size:80%;text-shadow:0 0px 0 white;">פזמון לפני ברכת המזון</span>
<br/>צוּר מִשֶּׁלּוֹ אָכַלְנוּ, בָּרְכוּ אֱמוּנַי, שָׂבַעְנוּ והוֹתַרְנוּ, כִּדְבַר יְהֹוָה:
<br/>הַזָּן אֶת עוֹלָמוֹ, רוֹעֵנוּ אָבִינוּ, אָכַלְנוּ מִלַּחְמוֹ, וּמִיֵּינוֹ שָׁתִינוּ, עַל כֵּן נוֹדֶה לִשְׁמוֹ, וּנְהַלּלוֹ בּפִינוּ, אָמַרְנוּ ועָנִינוּ, אֵין קָדוֹשׁ כַּיהֹוָה:
<br/>בְּשִׁיר וְקוֹל תּוֹדָה, נְבָרֵךְ לֵאלֹהֵינוּ, עַל אֶרֶץ חֶמְדָּה טוֹבָה, שֶׁהִנְחִיל לַאֲבוֹתֵינוּ, וּמָזוֹן וְצֵדָה, הִשְׂבִּיעַ לְנַפְשֵׁנוּ, חַסְדּוֹ גָבַר עָלֵינוּ, וֶאֱמֶת יְהֹוָה:
<br/>רַחֵם בּחַסְדֶּךָ, עַל עַמְּךָ צוּרֵנוּ, עַל צִיּוֹן מִשְׁכַּן כּבוֹדֶךָ, זְבוּל בֵּית תִּפְאַרְתֵּנוּ, וּבֶן דָּוִד עַבְדֶּךָ, יָבֹא ויִגְאָלֵנוּ, רוּחַ אַפֵּנוּ מְשִׁיחַ יְהֹוָה:
<br/>יִבָּנֶה הַמִּקְדָּשׁ, עִיר צִיּוֹן תּמַלֵּא, וְשָׁם נָשִׁיר שִׁיר חָדָשׁ, וּבִרְנָנָה שָׁם נַעֲלֶה, הָרַחֲמָן הַנִּקְדָּשׁ, יִתְבָּרַךְ ויִתְעַלֶּה, עַל כּוֹס יַיִן מָלֵא, כְּבִרְכַּת יְהֹוָה:
