<h1>[ומפורשים]</h1>
אלעזר בירבי קליר
<blockquote><br/> <p><strong>ח</strong></p></blockquote>
<blockquote><p>[…] וּמְפוֹרָשִׁים / זוֹנְחֵיהֶם חַיָּבִין מִיתָה וְגֵיהִנָּם יוֹרְשִׁים</p> <p>וְהַמְגַלֵּי פָנִים בָּם בָּאֵשׁ מָחֳלָשִׁים / וְעוֹבֵר בָּהֶם עֲבֵירָה נִשְׁפָּט בְּגַחֲלֵי אִשִּׁים</p> <p>כִּי הֵם אֵשׁ וְאָמְרֵיהֶם אִשִּׁים / ונפצו מִתּוֹךְ לַהֲבֵי חֲשָׁשִׁים</p> <p>וְרֶכֶב אֱלֹהִים רִבּוֹתַיִם אַלְפֵי שִׁנְאָן הַקְּדוֹשִׁים</p> <p>וּבָא לְסִינַי בַּקֹּדֶשׁ בְּשִׁשָּׁה בַשְּׁלִישִׁי בְּפֵירוּשִׁים</p> <p>וַיְהִי קֹלֹת וּבְרָקִים בַּשְּׁלִישִׁי לְקִדּוּשִׁים / וְהַר סִינַי בּוֹעֵר בָּאִשִּׁים</p> <p>וַיֹּאמֶר יְיָ אֶל מֹשֶׁה רֵד הָעֵד בַּקְּדוֹשִׁים / וּבַכֹּהֲנִים הַנִּגָּשִׁים</p> <p>וְיָרַד וְקִדֵּשׁ צֹאן קָדָשִׁים / מִלִּרְאוֹת שֶׁבַח מְקֻדָּשִׁים</p> <p>וּבְגָדִים לְבָנִים לוֹבְשִׁים / וְשָׂמְחוּ אֲנָשִׁים וְנָשִׁים / נְעָרִים זְקֵנִים וִישִׁישִׁים</p> <p>וּלְקַבֵּל עוֹל תּוֹרָה גָּשִׁים / כִּי מֵַתְּנַת עוֹלָם יוֹרְשִׁים</p> <p>וְסִינַי עֻטַּר בְּלַפִּידֵי אִשִּׁים / כִּי רַגְלֵי צוּר בּוֹ דָשִׁים</p> <p>וּבְרָקִים נוֹעֲדוּ לְעֵינֵי קְדוֹשִׁים / וּשְׁבָטִים הָיוּ נִרְעָשִׁים</p> <p>וְכָל צְבָא מָרוֹם רוֹעֲשִׁים / וְאֶרְאֶלֵּי שַׁחַק מִתְרַעֲשִׁים</p> <p>וְזֶה בָּזֶה פּוֹגְשִׁים / וּגְדוּדֵי זְבוּל נִרְעָשִׁים</p> <p>וַהֲמוֹנֵי מָעוֹן מָרְעָשִׁים / וְדָרֵי מָכוֹן רוֹגְשִׁים</p> <p>וְשֹׁכְנֵי עֲרָבוֹת נִרְגָּשִׁים / וְאוֹפַן וְגַלְּגַּל מִתְרַגְּשִׁים</p> <p>לִשְׁמֹעַ מִפִּי קֹדֶשׁ הַקָּדָשִׁים</p> <p>וְשָׁפְכוּ מַעֲנֶה מַקְדִּישִׁים / וּקְדֻשָּׁה לְקָדוֹשׁ מְשַׁלְּשִׁים</p> <p>אָצִים וְחָשִׁים / גּוֹהֲצִים וְגוֹעֲשִׁים / רוֹעֲשִׁים וּמַרְעִישִׁים</p> <p>רוֹגְשִׁים וּמַרְגִּישִׁים / וּמַעֲרִיצִים וּמַקְדִּישִׁים</p> <p>לְהַקְדִּישׁ קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ בְּקֹדֶשׁ הַקָּדָשִׁים</p></blockquote>
<blockquote><p>ככתוב וקרא</p> <br/></blockquote>
