<h1>ו שיר … … אמונה בלחשי</h1>
אלעזר בירבי קליר
<p>קדושת חג שבועות לר׳ אלעזר זל</p>
<p><strong>א</strong></p>
<blockquote><p>[שִׁיר…] אֱמוּנָה בְּלַחְשִׁי</p> <p>יִשָּׁקֵנִי / בְּרֶכֶב רִבּוֹתַיִם כְּבָא לְתָלְשִׁי</p></blockquote>
<blockquote><p>לְרֵיחַ / גַּן נָעוּל […]י זָהָב פִּלְּשִׁי</p></blockquote>
<blockquote><p>מָשְׁכֵנִי / דֶּלֶת מַעְלָן וּמַטָּן בְּחִדּוּשִׁי</p> <br/></blockquote>
<blockquote><p>שְׁחוֹרָה / הִטְעַנִי הַבְדָּלַת קְדוֹשִׁי</p> <p>אַל תִּרְאוּנִי / וְעַל מַיִם הִפְרִיחַ שָׁרְשִׁי</p> <p>הַגִּידָה לִי / זֹאת גְּנָזֶיהָ לְשַׂמֵּחַ נַפְשִׁי</p> <p>אִם לֹא תֵדְעִי / חֵירוּת עוֹלָם לְשַׁלֵּחַ חָפְשִׁי</p> <br/></blockquote>
<blockquote><p>לְסוּסָתִי / טְעָנִי כְּבוֹדִי וּמֵרִים רֹאשִׁי</p> <p>נָאווּ / יְדִידִים זֹהַר מִמַּיִם לְהַמְשִׁי</p> <p>תּוֹרֵי / כְּתָב שָׁלִישִׁים בְּמוֹעֵצוֹת פֵּירְשִׁי</p> <p>עַד שֶׁהַמֶּלֶךְ / לִמֵּד דִּבְּרוֹתָיו בַּחֹדֶשׁ הַשְּׁלִישִׁי</p> <br/></blockquote>
<p><strong>א1</strong></p>
<blockquote><p>סְבִיבָיו / יוֹבִילוּ שַׁי לַמּוֹרָא</p> <p>לְהַנְחִיל דִּבְּרוֹת בְּטָהֳרָה</p></blockquote>
<blockquote><p>לשל[ש… … תורה</p></blockquote>
<blockquote><p>לְהִתְגּוֹנֵן בְּצֵל גָּדוֹל וְנוֹרָא</p> <br/></blockquote>
<p><strong>ב</strong></p>
<blockquote><p>[צְרוֹר הַמֹּר] / מַלְכִּי בַקֹּדֶשׁ מְיַחֲדִים</p> <p>אֶשְׁכּוֹל / נְדִיבֵי עַמִּים וְכָל יְדִידִים</p></blockquote>
<blockquote><p>הִנָּךְ … ס….. ….. ] בְּאֶבְרוֹתָיו מְאֻפָּדִים</p></blockquote>
<blockquote><p>הִנְּךָ יָפֶה / עִטּוּר בְּקוֹלוֹת וְלַפִּידִים</p> <br/></blockquote>
<blockquote><p>קוֹרוֹת / פֹּרְקוּ הָרִים מִקּוֹל פְּחָדִים</p> <p>אֲנִי חֲבַצֶּלֶת / צַגְתִּי בְּיִרְאָה וּרְעָדִים</p> <p>כְּשׁוֹשַׁנָּה / קִוִּיתִי לִמְתוּקִים וְנֶחֱמָדִים</p></blockquote>
