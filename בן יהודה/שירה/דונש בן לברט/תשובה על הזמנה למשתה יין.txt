<h1>תשובה על הזמנה למשתה יין</h1>
דונש בן לברט
<blockquote><p>וְאוֹמֵר "אַל תִּישַׁן! / שְׁתֵה יַיִן יָשָׁן–</p></blockquote>
<blockquote><p>וְכֹפֶר עִם שׁוֹשָׁן / וּמֹר עִם אֲהָלִים</p></blockquote>
<blockquote><p>בְּפַרְדֵּס רִמּוֹנִים / וְתָמָר וּגְפָנִים</p></blockquote>
<blockquote><p>וְנִטְעֵי נַעְמָנִים / וּמִינֵי הָאְשָׁלִים,</p></blockquote>
<blockquote><p>וְרֶגֶשׁ צִנּוֹרִים / וְהֶמְיַת כִּנּוֹרִים</p></blockquote>
<blockquote><p>עֲלֵי פֶה הַשָּׁרִים / בְּמִנִּים וּנְבָלִים,</p></blockquote>
<blockquote><p>וְשָׁם כָּל עֵץ מוּנָף / יְפֵה פֶרִי עָנָף</p></blockquote>
<blockquote><p>וְצִפּוֹר כָּל כָּנָף / יְרַנֵּן בֵּין עָלִים,</p></blockquote>
<blockquote><p>וְיֶהְגּוּ הַיּוֹנִים / כְּהוֹגִים נִגּוּנִים,</p></blockquote>
<blockquote><p>וְהַתּוֹרִים עוֹנִים / וְהוֹמִים כַּחְלִילִים.</p></blockquote>
<blockquote><p>וְנִשְׁתֶּה בַעְרוּגוֹת / בְּשׁוֹשַׁנִּים סוּגוֹת,</p></blockquote>
<blockquote><p>וְנָנִיס הַתּוּגוֹת / בְּמִינֵי הִלּוּלִים,</p></blockquote>
<blockquote><p>וְנֹאכַל מַמְתַקִּים / וְנִשְׁתֶּה מִזְרָקִים</p></blockquote>
<blockquote><p>וְנִנְהַג כַּעְנָקִים / וְנִשְׁתֶּה בִסְפָלִים,</p></blockquote>
<blockquote><p>וְאָקוּם בִּבְקָרִים / אֲנִי לִשְׁחֹט פָּרִים</p></blockquote>
<blockquote><p>בְּרִיאִים נִבְחָרִים / וְאֵילִים וַעְגָלִים,</p></blockquote>
<blockquote><p>וְנִמְשַׁח שֶׁמֶן טוֹב / וְנַקְטִיר עֵץ רָטֹב</p></blockquote>
<blockquote><p>בְּטֶרֶם יוֹם קָטוֹב / יְבוֹאֵנוּ נַשְׁלִים".</p></blockquote>
<blockquote><p>גְּעַרְתִּיהוּ "דֹּם,דֹּם, / עֲלֵי זֹאת אֵיךְ תִּקְדֹּם–</p></blockquote>
<blockquote><p>וּבֵית קֹדֶשׁ וַהְדוֹם / אֱלֹהִים לָעְרֵלִים!</p></blockquote>
<blockquote><p>בְּכִסְלָה דִבַּרְתָּ / וְעַצְלָה בָחַרְתָּ</p></blockquote>
<blockquote><p>וְהֶבֶל אָמַרְתָּ / כְּלֵצִים וּכְסִילִים,</p></blockquote>
<blockquote><p>וְעָזַבְתָּ הֶגְיוֹן / בְּתוֹרַת אֵל עֶלְיוֹן</p></blockquote>
<blockquote><p>וְתָגִיל– וּבְצִיּוֹן / יְרוּצוּן שׁוּעָלִים.</p></blockquote>
<blockquote><p>וְאֵיךְ נִשְׁתֶּה יַיִן / וְאֵיךְ נָרִים עַיִן–</p></blockquote>
<blockquote><p>וְהָיִינוּ אַיִן / מְאוּסִים וּגְעוּלִים!" <br/></p></blockquote>
