<h1>[מי לא]</h1>
שמואל הנגיד
<blockquote><p>— — ∪— — —∪ — —∪ — — — // — — ∪— — —∪ — —∪ — — —</p> <br/></blockquote>
<blockquote><p>מִי לֹא יְשַׁלַּח, יוֹם פְּרִידָה, עֲלֵי פָנָיו // מַבּוּל — יְשַׁלַּח אֵל תְּבַלּוּל בְּאִישׁוֹנָיו<a href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a>!</p></blockquote>
<blockquote><p>אִם אַתְּ<a href="#fn:2" id="fnref:2" title="see footnote"><sup>2</sup></a> בְּעַזּוֹת תַּעֲנֵנִי כְּכָל עִזּוּז<a href="#fn:3" id="fnref:3" title="see footnote"><sup>3</sup></a> — // הֵן מַעֲנֶה רַךְ אֶעֱנֶךָ כְּכָל עָנָו<a href="#fn:4" id="fnref:4" title="see footnote"><sup>4</sup></a>.</p></blockquote>
<blockquote><p>עֹפֶר אֲשֶׁר שָׁפַט וְאָסַר עֲלֵי עֵינַי // שֵׁנָה, וְלֹא אָסַר שְׁפֹךְ דָּם עֲלֵי עֵינָיו<a href="#fn:5" id="fnref:5" title="see footnote"><sup>5</sup></a>,</p></blockquote>
<blockquote><p>תַּחְשֹׁב בְּעֵינָךְ<a href="#fn:6" id="fnref:6" title="see footnote"><sup>6</sup></a> כִּי בְצֶדֶק וּבִנְכוֹחָה // תִּשְׁפֹּט<a href="#fn:7" id="fnref:7" title="see footnote"><sup>7</sup></a> — וְיֵשׁ חוֹשֵׁב וְטוֹעֶה בְּחֶשְׁבּוֹנָיו!</p> <br/></blockquote>
<ol><li id="fn:1"><p><strong>באישוניו</strong> – בעיניו. מי שלא ירבה לבכות ולהזיל דמעות ביום פרידת האוהבים, יקולל. <a href="#fnref:1" title="return to body"> ↩</a></p></li> <li id="fn:2"><p><strong>את</strong> – אתה. <a href="#fnref:2" title="return to body"> ↩</a></p></li> <li id="fn:3"><p><strong>עזוז</strong> – עז פנים. <a href="#fnref:3" title="return to body"> ↩</a></p></li> <li id="fn:4"><p>על פי משלי יח כג: תחנונים ידבר רש ועשיר יענה עזות; תהלים כד ח: יי עזוז וגבור; משלי טו א: מענה רך. <a href="#fnref:4" title="return to body"> ↩</a></p></li> <li id="fn:5"><p>אהובי המדיר שנה מעיני אבל לא אסר על עיניו לשפך את דמי. <a href="#fnref:5" title="return to body"> ↩</a></p></li> <li id="fn:6"><p><strong>בעינך</strong> – על עינך. <a href="#fnref:6" title="return to body"> ↩</a></p></li> <li id="fn:7"><p><strong>תשפט – עינך.</strong> <a href="#fnref:7" title="return to body"> ↩</a></p></li></ol>
