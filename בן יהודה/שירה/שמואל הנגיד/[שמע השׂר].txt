<h1>[שמע השׂר]</h1>
שמואל הנגיד
<blockquote><p>U //– – – U // – – – U – – –// – – –</p></blockquote>
<blockquote><p>שְׁמַע, הַשַּׂר, / אֲשֶׁר כָּל שַׂר / בְּךָ נִקְרָא / וְנִתְכַּנָּה:</p></blockquote>
<blockquote><p>בְּבוֹא סִפְרָךְ / אֲדַמֶּה כִי / בְּיָגוֹן בָּא / וּבִמְגִנָּה,</p></blockquote>
<blockquote><p>וְכִי אֵין בּוֹ / לְבַד תַּגְמוּל / לְחֵטְא אַדִּיר / וְלִתְלֻנָּה</p></blockquote>
<blockquote><p>וְחוֹרֵשׁ תּוֹ־ / אֲנָה יִקְצֹר / מְהֵרָה כַ־ / אֲשֶׁר אִנָּה.</p></blockquote>
<blockquote><p>לְקַחְתִּיהוּ – / וְלִבִּי בִי / כְּמוֹ גַנָּב / בְּתוֹךְ פִּנָּה,</p></blockquote>
<blockquote><p>יְבַקֵּש צַד / לְהִסָּתֵר – / וְאֵין מִסְתּוֹר / וְאֵין צִנָּה!</p></blockquote>
<blockquote><p>וְנִחַשְׁתִּי / בְּחוֹתָמוֹ / אֲשֶׁר הָיָה / כְּשׁוֹשַׁנָּה,</p></blockquote>
<blockquote><p>וְאָמַרְתִּי: / אֲדַמֵּהוּ / אֱלֵי שׁוֹשָׁן / עֲלֵי כַנָּה,</p></blockquote>
<blockquote><p>וְאִם יוּמַר / דְּבַר שׁוֹשָׁן – / יְהִי שָׂשׂוֹן / וְיוֹם רִנָּה.</p></blockquote>
<blockquote><p>קְרָאתִיהוּ – / וְלֹא נֶחְלַף / דְּבַר נַחַשׁ, / וְלֹא שֻׁנָּה!</p></blockquote>
<blockquote><p>מְצָאתִיהוּ / כְּשׁוֹשַׁנָּה / אֲשֶׁר צָצָה / בְּתוֹךְ גִּנָּה,</p></blockquote>
<blockquote><p>וְעָצַם בּוֹ / שְׂשׂוֹן לִבִּי / כְּיוֹם חֻפָּה / וְכַחְתֻנָּה.</p> <br/></blockquote>
