<h1>[לא כמשפטי]</h1>
שמואל הנגיד
<blockquote><p>— ∪ — — —∪ — — — — — // — ∪ — — —∪ — — — — —</p></blockquote>
<blockquote><p>לֹא כְמִשְׁפָּטִי תְדִינוּן עָלֵימוֹ // חַפְּשׂוּ הֵיטֵב וְדִינוּם כִּשְׁלֹמֹה<a href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a></p></blockquote>
<blockquote><p>מַהֲרִי שׁוּבִי, תְּנוּמַת עֵינֵימוֹ, // וַאֲנִי מֵאֵין תְּנוּמָה אֶבְכֵּמוֹ</p></blockquote>
<blockquote><p>הֵן לְבָבִי סָר<a href="#fn:2" id="fnref:2" title="see footnote"><sup>2</sup></a> וְנִמְהַר<a href="#fn:3" id="fnref:3" title="see footnote"><sup>3</sup></a> אַחְרֵימוֹ, // כִּי מְנַהֵג בָּם, לְהַכְעִיס, מִהְרָמוֹ<a href="#fn:4" id="fnref:4" title="see footnote"><sup>4</sup></a>.</p></blockquote>
<blockquote><p>שַׁאֲלוּ נָא אֶת שְׁלוֹמִי<a href="#fn:5" id="fnref:5" title="see footnote"><sup>5</sup></a>, כִּי כָל דּוֹד // אֶל יְדִידוֹ סָר וְשׁוֹאֵל בִּשְׁלוֹמוֹ</p></blockquote>
<blockquote><p>כָּהֲתָה עֵין אוֹהֲבָם<a href="#fn:6" id="fnref:6" title="see footnote"><sup>6</sup></a> כִּי אֵינֵימוֹ, // אַשֲׁרֵי עַיִן אֲשֶׁר עוֹד תֶּחְזֵמוֹ!</p> <br/></blockquote>
<ol><li id="fn:1"><p>אל תדונו, האנשים, את אוהבי כמשפטי הנמהר שרחצתי עליכם בשעת צערי, אלא דרשו וחקרו היטב ודינום בחכמה כשלמה. <a href="#fnref:1" title="return to body"> ↩</a></p></li> <li id="fn:2"><p><strong>סר</strong> – יצא. <a href="#fnref:2" title="return to body"> ↩</a></p></li> <li id="fn:3"><p><strong>ונמהר</strong> – ומהר. <a href="#fnref:3" title="return to body"> ↩</a></p></li> <li id="fn:4"><p><strong>כי מנהג בם להכעיס מהרמו</strong> – כי המוביל אותם בדרך מהר להרחיקם ממני להכעיסני. <a href="#fnref:4" title="return to body"> ↩</a></p></li> <li id="fn:5"><p><strong>שאלו נא את שלומי</strong>־ אתם ידידי שהלכתם ממני, לפחות שאלו לשלומי. <a href="#fnref:5" title="return to body"> ↩</a></p></li> <li id="fn:6"><p><strong>אוהבם</strong> – המשורר. <a href="#fnref:6" title="return to body"> ↩</a></p></li></ol>
