<h1>[עשֵׂה אחי]</h1>
שמואל הנגיד
<p>ולה אלי שכץ כלפה אלאשראף עלי מקאבלה אלמקרא אלדי ככתב בכט ידה</p>
<p>ולו לאדם אשר הטיל עליו את ההשגחה על הקבלת (הגהת) המקרא שכתב בכתב ידו</p>
<blockquote><p>U // – – – U // – – – U // – – – U – – –</p></blockquote>
<blockquote><p>עֲשֵׂה, אָחִי, / בְּחַיֶּיךָ, / שְׁאֵלָה שֶׁ– / שְׁאִלְתִּיךָ,</p></blockquote>
<blockquote><p>וְאַל תִּהְיֶה / כְּקֵץ אָרֹךְ / וְתוֹחֶלֶת / מְמֻשָּׁכָה,</p></blockquote>
<blockquote><p>וְהַסּוֹפֵר / אֲשֶׁר תָּכִין / יְהִי מֵבִין / כְּמוֹתֶךָ,</p></blockquote>
<blockquote><p>וְצַוֵּהוּ / וְתִהְיֶה כָל / עֲבוֹדָתוֹ / עֲלֵי פִיךָ,</p></blockquote>
<blockquote><p>וְיִשָּׁמֵר / בְּכָל טַעַם, / וּבִמְנִידָה / וִּבסְמִיכָה,</p></blockquote>
<blockquote><p>וּבִפְתּוּחִים / וּבִקְמוּצִים, / וּבִמְלֵאָה / וּבַחְשׂוּכָה,</p></blockquote>
<blockquote><p>וְתִהְיֶה אַתְּ / לְעֵזֶר לוֹ – / יְהִי צוּרָךְ / בְּעֶזְרֶךָ! –</p></blockquote>
<blockquote><p>אֲנִי אוֹמֵר, / וְאַדִּירֵי / אֲנָשִׁים יֹא – / מְרוּן כָּכָה:</p></blockquote>
<blockquote><p>בְּישֶׁר כָּל / פְּעָלֶיךָ / וְצֶדֶק מַל – / אֲכוֹתֶיךָ</p></blockquote>
<blockquote><p>וְטוּב טַעַם / בְּךָ נִמְצָא, / וְעֶזְרָה לַ – / חֲבֵרֶיךָ!</p></blockquote>
