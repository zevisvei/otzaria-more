<h1>אֲהָהּ שַׁבְתִּי</h1>
שמואל הנגיד
<p>וזאר קברה פי לאתאני מן מואראתה פי לחדה וקאל</p>
<p>בקר את קברו ביום השני אחרי הטמנתו בקברו ואמר</p>
<blockquote><p>אֲהָהּ, שַׁבְתִּי / בְּצַר רוּחִי, / אֱלֹהִים יָחְ- / נְךָ, אָחִי!</p></blockquote>
<blockquote><p>קְבַרְתִּיךָ / בְּיוֹם אֶמֶשׁ / וְגַם הַיּוֹם / מְרִי שִׂיחִי.</p></blockquote>
<blockquote><p>לְךָ שָׁלוֹם! / הֲלֹא תִשְׁמַע / בְּקְָרִאי לָךְ / בְּכָל כֹּחִי?</p></blockquote>
<blockquote><p>הֲשִׁיבֵנִי: / הֲתַכִּיר מַ- / עֲנֶה קִינַי / בְּהַצְרִיחִי?</p></blockquote>
<blockquote><p>וְאֵיךְ לַנְתָּ / בְּקִבְרָתָךְ / עֲלֵי עָפָר / בְּבֵית דֶּחִי,</p></blockquote>
<blockquote><p>הֲרָפְתָה מִ- / מְּךָ עֶצֶם / וּמַכְתֵּשׁ מִ- / מְּקוֹם לֶחִי,</p></blockquote>
<blockquote><p>עֲזַבְתִּיךָ / בְּכוֹר אָבִי, / לְפִקָּדוֹן / בְּיַד גּוֹחִי,</p></blockquote>
<blockquote><p>וְאֶבְטַח, כִּי / אֱלֵי שָׁלוֹם / תְּהִי הוֹלֵךְ, / בְּמַבְטִיחִי. <br/></p></blockquote>
