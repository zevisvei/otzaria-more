<h1>שמך יגדל</h1>
שלמה אבן גבירול
<blockquote><p>שִׁמְךָ יִגְדַּל / וְלֹא יֶחְדַּל / גָּדְלְךָ מִפִּי אֲיֻמָּה</p> <p>לְהַעֲצִימְךָ / בַּהֲדוֹמְךָ / יְרוּצוּן דָּרֵי אֲדָמָה</p> <p>מֶלֶךְ עָל / אֲשֶׁר פָּעַל / הַכֹּל בְּדַעַת וְחָכְמָה</p> <p>הֲמוֹנֶיךָ / אֱמוּנֶיךָ / עוֹנִים וְאוֹמְרִים בְּאֵימָה</p> <p>יִתְגַּדַּל / וְיִתְקַדַּשׁ / שְׁמֵה רַבָּא בְּעָלְמָא</p> <br/></blockquote>
<blockquote><p>לְךָ אֶקְרָא / בְּרֹב מוֹרָא / וְאַגִּיד גָּדְלְךָ נָעֲלָה</p> <p>וְגַם חִכִּי / יְצַפְצֵף כִּי / לְךָ יְיַָ הַגְּדֻלָּה</p> <p>כְּבוֹדֶךָ / וְהוֹדֶךָ / אֲחַוֶּה יוֹם וָלַיְלָה</p> <p>וּמַקְהֶלְךָ / בְּהֵיכָלְךָ / מְתַנִּים צִדְקְךָ הֵמָּה</p> <br/></blockquote>
<blockquote><p>מְקֻדָּשִׁים / וְתַרְשִׁישִׁים / יְבָרְכוּךָ בְּמַקְהֵלִים</p> <p>יְבַקֵּשׁוּ / וְיִדְרֹשׁוּ / אַיֵּה כְבוֹד אֵל אֵלִים</p> <p>וְיִפְחָדוּ / וְיִרְעָדוּ / בְּעֵת אִי כְבוֹד אֵל שׁוֹאֲלִים</p> <p>כִּי רָמוּ / וְגַם עָצְמוּ / מַעֲשֶׂיךָ הַגְּדוֹלִים</p> <p>וְיָעִידוּ / וְיַגִּידוּ / צִדְקָתְךָ הָעֲצוּמָה.</p> <br/></blockquote>
