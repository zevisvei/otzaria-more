<h1>[אשר גנב]</h1>
שלמה אבן גבירול
<h2 id="אשרגנבשלמהאבןגבירול">[אשר גנב] / שלמה אבן גבירול</h2>
<p>[על פרידת ידידו]</p>
<br/>
<p>È — — — / È — — — / È — — / / È — — — /È — — — / È — —</p>
<br/>
<blockquote><p>אֲשֶׁר גָּנַב לְבַב דּוֹדָיו בְּהוֹדוֹ / וְעַל-כָּל-אֹהֲבָיו הִקִּיף מְצוּדוֹ</p></blockquote>
<blockquote><p>הֲטוֹב לֵאוֹר בְּאוֹר פָּנָיו פְּנֵי-כֹל / וְלֹא-זָרַח כְּבוֹדוֹ עַל-יְדִידוֹ</p></blockquote>
<blockquote><p>וּבֵית-אַהַב תְּמוֹל הֵאִיר בְּיָפְיוֹ / הֲטוֹב כִּי-יֶחֱשַׁךְ הַיּוֹם לְנֹדוֹ.</p></blockquote>
