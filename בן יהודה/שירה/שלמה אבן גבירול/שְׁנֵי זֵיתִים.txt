<h1>שְׁנֵי זֵיתִים</h1>
שלמה אבן גבירול
<blockquote><p>שְׁנֵי זֵיתִים / נִכְרָתִים / בְּגַן נָעוּל יַצְהִירוּ</p> <p>לְרֹאשׁ קְהָתִי / וְאֶפְרָתִי / שְׁתֵּי עֲטָרוֹת יַכְתִּירוּ</p> <p>מוּל מְנוֹרָה / הַטְּהוֹרָה / כְּמוֹ נֵרוֹת יַזְהִירוּ</p> <p>הֵן בְּמַשְׂאֵת / כָּל אִישׁ אֶת / רֵעֵהוּ יַעֲזֹרוּ</p> <p>הֵן בְּמַחֲנֶה / אֶל מוּל פְּנֵי / הַמְּנוֹרָה יָאִירוּ.</p> <br/></blockquote>
<blockquote><p>לְבֶן פֹּרָת / וְצִיץ יִקְרַת / אֲשֶׁר הָיָה כְּצִיץ נוֹבֵל</p> <p>וְלַנִּמְשָׁח / אֲשֶׁר הוּשַׁח / כְּמוֹ אֳנִיָּה בְּלִי חוֹבֵל</p> <p>זְכֹר מַרְאָה / אֲשֶׁר רָאָה / זְכַרְיָהוּ בְּעִיר בָּבֶל</p> <p>כְּמוֹ זְכַרְתָּם / וְהוֹשַׁעְתָּם / אֲזַי עַל יַד זְרֻבָּבֶל</p> <p>זְכֹר זַרְעָם / לְהוֹשִׁיעָם / בְּעֵת תִּקְרָא שְׁנַת יוֹבֵל</p> <p>וְשִׂים אוֹתָם / מְקוֹם אֲבוֹתָם / הֱיוֹת מוֹשְׁלִים בְּכָל תֵּבֵל</p></blockquote>
<blockquote><p>וּמֵעַל יַד / אִישׁ צַיָּד / טַבַּעְתָּם יָסִירוּ.</p> <br/></blockquote>
<blockquote><p>מַמְלָכָה / מְמֻשָּׁכָה / לְעִיר צִיּוֹן תְּשִׁיבֶנָּה</p> <p>וּבַת כְּבוּדָה / בְּרָב עֲבֻדָּה / לְבֵית אִמָּהּ תְּבִיאֶנָּה</p> <p>וְהַגְּבִירָה / וְהַצְּפִירָה / לְרֹאשׁ דּוֹדָהּ תְּקִימֶנָּה</p> <p>וּמִצְנֶפֶת / מְעֻלֶּפֶת / לְרֹאשׁ אַהֲרֹן תְּשִׂימֶנָּה</p> <p>וְהַתָּמִיד / אֲזַי תַּעֲמִיד / וְהַמִּנְחָה תְּרִימֶנָּה</p> <p>וְהַמֶּרְקָח / אֲשֶׁר לֻקַּח / וְלֹא רְאִיתִיו עַד הֵנָּה</p> <p>וְהַקְּטֹרֶת / מְקֻטֶּרֶת / לְאַהֲרֹן יַקְטִירֶנָּה</p> <p>וְהַנֵּרוֹת / בִּמְנוֹרוֹת / בְּנֵי אַהֲרֹן יַבְעִירוּ.</p> <br/></blockquote>
<blockquote><p>הַנְּכֵאָה / וְהַצְּמֵאָה / אֲשֶׁר תָּמִיד לְךָ תֵילִיל</p> <p>סְמֹךְ יָדָהּ / בְּבֶן דּוֹדָהּ / אֲשֶׁר נִקְרָא שְׁמוֹ חַכְלִיל</p> <p>וְהַכְנִיסֵם / לְגַן בֹּשֶׂם / וְחֻפָּתָם בְּהוֹד תַּכְלִיל</p> <p>וְהוֹלִיכֵם / וְהַמְלִיכֵם / בְּהַר צִיּוֹן וְהַגָּלִיל</p> <p>וְחֹק יָשָׁר / אֲזַי יוּשַׁר / עֲלֵי נֵבֶל וְגַם חָלִיל</p> <p>זְעֹם עוֹזְבֵי / שְׁמָךְ וְאוֹהֲבֵי / חֲזוֹן שֶׁקֶר וְשָׁוְא וֶאֱלִיל</p> <p>וְעַם פְּרוּכִים / יְהוּ בְרוּכִים / כָּל רוֹאֵיהֶם יַכִּירוּ.</p> <br/></blockquote>
