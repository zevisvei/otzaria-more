<h2 id="גשולשנישלמהאבןגבירול">[גשו לשני] / שלמה אבן גבירול</h2>
<p>ולה איצ ͘ א</p>
<p>È — — — / È — — — / È — — / / È — — — /È — — — / È — —</p>
<br/>
<blockquote><p>גְּשׁוּ לִשְׁנֵי אֲמִירֵי הַהֲדַסִּים / אֲשֶׁר בַּמֹּר וּבַבֹּשֶׂם יְסוֹדָם</p></blockquote>
<blockquote><p>וְהַשְׁבִּיעוּם בְּאַהְבַת דַּל וְחֹלֶה / כְּרֶגַע לַעֲמֹד עָלַי בְּחַסְדָּם</p></blockquote>
<blockquote><p>אֲשֶׁר עָבְרוּ וְשִׁלַּחְתִּים לְבָבִי / וְשָׁלְחוּ הֵם שְׁלוֹמוֹת לִי בְּיָדָם</p></blockquote>
<blockquote><p>שְׁמֵעוּנִי וְלֹא רָאוּ וְאִם-לֹא / מְצָאוּנִי אֲנִי עֹמֵד לְנֶגְדָּם</p></blockquote>
<blockquote><p>וְשֶׁמֶשׁ סָר וּפָנָה מִפְּנֵיהֶם / כְּאִלוּ מִלְּחָיֵיהֶם אֲדַמְדָּם.</p></blockquote>
