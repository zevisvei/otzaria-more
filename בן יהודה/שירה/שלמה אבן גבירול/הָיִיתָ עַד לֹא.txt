<h1>הָיִיתָ עַד לֹא</h1>
שלמה אבן גבירול
<blockquote><p><a href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a></p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא <strong>א</strong> / וְלָנֶצַח תִּהְיֶה</p></blockquote>
<blockquote><p>לָכֵן <strong>יא</strong> / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא <strong>ב</strong> / וְלָנֶצַח תִּהְיֶה</p></blockquote>
<blockquote><p>לָכֵן <strong>יב</strong> / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<p>וכו׳ – עד <strong>ע</strong></p>
<blockquote><p>הָיִיתָ עַד לֹא / פִנִּיתָ שֻׁלְחָן / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יְפֹאַר יִפָּלֵא / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא / צַלְמָוֶת רָקוּם / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יִצָּרֵף יִצְדַּק / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא / קָרַעְתָּ חַיָּה / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יִקְדַּשׁ יִקָּרֵא / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא / רִצַּפְתָּ שְׁבִיעִי / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יֵרָאֶה יְרוֹמַם / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא / שָׁבַתָּ מִכֹּל / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יִשָּׁפֵר יִשָּׁחֵר / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה</p> <br/></blockquote>
<blockquote><p>הָיִיתָ עַד לֹא / תִּכַּנְתָּ כָּל זֶה / וְלָנֶצַח תִּהְיֶה</p> <p>לָכֵן / יִתָּכֵן יִתָּקֵף / שִׁמְךָ אֶהְיֶה אֲשֶׁר אֶהְיֶה.</p></blockquote>
