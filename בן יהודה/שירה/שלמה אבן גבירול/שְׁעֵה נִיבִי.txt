<h1>שְׁעֵה נִיבִי</h1>
שלמה אבן גבירול
<blockquote><p>שְׁעֵה נִיבִי / בְּהֵיטִיבִי / שִׁגְיוֹנוֹתַי וְהֶגְיוֹנָי</p> <p>לְמַכְאוֹבִי / צְרִי תָבִיא / וְתַקְרִיב יוֹם פִּדְיוֹנָי</p> <p>לְהוֹשִׁיבִי / בַּת נְדִיבִי / בְּאַרְמוֹנַי כְּקַדְמוֹנָי</p> <p>מְחַץ רָבִי / וְסָרָבִי / וְעַקְרַבִּי וְסַלּוֹנָי</p> <p>הֱיֵה סְבִיבִי / וְרִיב רִיבִי / וְהָשֵׁב גְּמוּל גְּאֵיוֹנָי</p> <p>שְׁלַח אָבִי / מְשׂוֹשׂ לְבָבִי / הַמְבַשֵּׂר אֶבְיוֹנָי</p> <p>אֵלִיָּה / הַנָּבִיא / לִפְנֵי בּוֹא יוֹם יְיָ</p> <br/> <p>לְךָ הָאֵל / אֲנִי שׁוֹאֵל / וְגַם אֶפְרֹשׂ לְךָ יָמִין</p> <p>קְרֹץ אַדְבְּאֵל / וּמַגְדִּיאֵל / וְכָל מַכְזִיב בְּךָ יַאֲמִין</p> <p>וְשׁוֹבֵב [אֶ]ל / עִיר אֲרִיאֵל / בְּנֵי יְהוּדָה וּבִנְיָמִין</p> <p>וְחִישׁ גּוֹאֵל / לְיִשְׂרָאֵל / וְקָרֵב קֵץ הַיָּמִין</p> <p>וּמִיכָאֵל / וְגַבְרִיאֵל / יַעַמְדוּ בְּדוּכָנָי</p> <br/></blockquote>
<blockquote><p>מְעַד חוֹמְסַי / וְחֵיל בּוֹשְׁסַי / כְּטִיט חוּצוֹת הָדֵ[ק רְקַע</p> <p>וְלִרְמוּסַי / וְלִרְפוּ]סַי / יָתֵד נֶאֱמָן תִּתְקַע</p> <p>[וְאוֹ]ר חֲמוּסַי / וְנֵר עֲמוּסַי / כַּשַּׁחַר יִבָּקַע</p> <p>וּמַכְעִיסַי / וּמוֹ[אֲ]סַי / שְׁחוֹחַ יֵלְכוּ לְפָנָי</p> <p>הֲדַר יִשַׁי / נְשִׂיא אִישַׁי / כְּאוֹר נוֹצֵץ בְּאַפִּרְיוֹן</p> <p>וְנֶעֱנָשַׁי / וְנֶאֱנָשַׁי / בְּיָמָיו יִמְצְאוּ פִדְיוֹן</p> <p>וְנֶחְבָּשַׁי / וְנִכְבָּשַׁי / בְּרִנָּה יַעֲלוּ צִיּוֹן</p> <p>וְיוּבַל שַׁי / לְמִקְדָּשַׁי / בְּהִגָּיוֹן וְשִׁגָּיוֹן</p> <p>וְאֶל רָשַׁי / וְחַלָּשַׁי / שְׁלַל מוֹנַי וְשַׁלְמוֹנָי.</p> <br/></blockquote>
