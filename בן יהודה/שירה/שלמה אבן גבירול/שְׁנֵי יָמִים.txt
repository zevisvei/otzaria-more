<h1>שְׁנֵי יָמִים</h1>
שלמה אבן גבירול
<blockquote><p>שְׁנֵי יָמִים / מְסֻיָּמִים / נוֹעָדוּ לַעֲמוּסִים</p> <p>זְכִירָתָם / שְׁמִירָתָם / הֻזְהָרוּ בְךָ חוֹסִים</p> <p>חֹק שַׁבָּת / לְבַל יָשְׁבַּת / וְיוֹם זֵכֶר כָל מַעֲשִׂים</p> <p>וְהַיָּמִים הָאֵלֶּה / נִזְכָּרִים וְנַעֲשִׂים</p> <br/></blockquote>
<blockquote><p>לַמִּשְׁפָּט, / הֱיוֹת נִשְׁפָּט / בּוֹ כָל בָּשָׂר, שַׂמְתּוֹ,</p> <p>צוּר עוֹלָם, / לְכָל נֶעְלָם, / וְצוּר לְהוֹכִיחַ יְסַדְתּוֹ</p> <p>עֲלֵי כָל אִישׁ / אֲשֶׁר יַבְאִישׁ / פְּרִי מַעֲלָלָיו לְתִתּוֹ</p> <p>אִישׁ אִישׁ מִמְּלַאכְתּוֹ / אֲשֶׁר הֵמָּה עוֹשִׂים</p> <br/></blockquote>
<blockquote><p>מֵרֵאשִׁית / בּוֹ הֵשִׁית / לְחוֹטְאִים צְרִי וַאֲרוּכָה</p> <p>וְדִין נִקְשָׁב / לְאִישׁ לֹא שָׁב / וְלֹא חָפֵץ בִּבְרָכָה</p> <p>לְהוֹלְכֵי תֹם / יֵשׁ בּוֹ מְתֹם / בְּרִית שְׁמוּרָה וַעֲרוּכָה</p> <p>לְעוֹשֵׂי הַמְּלָאכָה / כִּי בֶאֱמוּנָה הֵם עוֹשִׂים</p> <br/></blockquote>
<blockquote><p>הֵן הַיּוֹם / מְאֹד אָיֹם / וְנוֹרָא כְּמוֹרָאֶךָ</p> <p>בְּעֵת עָבְרָם / בְּמִסְפָּרָם / כִּבְנֵי מֵרוֹן צְבָאֶיךָ</p> <p>כְּמַעֲשֵׂה אֱנוֹשׁ / אַל תַּעֲנֹשׁ / עַל שְׂכָרָם בְּבוֹאֶךָ</p> <p>כִּי בְנִיבְךָ / וְרַב טוּבְךָ / נִשְׁעָנִים נְכֵאֶיךָ</p> <p>צָפַנְתָּ לִירֵאֶיךָ / פָּעַלְתָּ לַחוֹסִים.</p></blockquote>
