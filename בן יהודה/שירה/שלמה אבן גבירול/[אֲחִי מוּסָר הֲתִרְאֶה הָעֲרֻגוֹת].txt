<h1>[אֲחִי מוּסָר הֲתִרְאֶה הָעֲרֻגוֹת]</h1>
שלמה אבן גבירול
<p>ומן אנטֹם תצניפה ואחלא קולה [מחיבוריו הפיוטיים ודבריו הערבים ביותר]</p>
<br/>
<p>È — — — / È — — — / È — — / / È — — — /È — — — / È — —</p>
<br/>
<blockquote><p>אֲחִי מוּסָר הֲתִרְאֶה הָעֲרֻגוֹת / מְשֻׁבָּצוֹת בְּשׁוֹשַׁנִּים וְסוּגוֹת</p> <p>וּבֵית-שֵׁן סֻגְּרוּ עַל-דַּר דְּלָתָיו / וְאַף כַּמֹּר וְעֵינַיִם נְפוּגוֹת</p> <p>אֲשֶׁר יִדְמוּ כְּיוֹנִים בָּאֲרֻבּוֹת / וְכִמְעַט קָט מְתַנּוֹת קוֹל וְהֹגוֹת</p> <p>בְּהַבִּיטָם וּבִפְנוֹתָם דְּאָגוֹת / לְכָל-לִבּוֹת וּמַכְאֹבוֹת וְתוּגוֹת</p> <p>הֲרָגוּהוּ בְּעָרְמַת הַזְּדוֹנִים / וְיֹאמְרוּ כִּי-הֲרָגוּהוּ שְׁגָגוֹת. <br/></p></blockquote>
