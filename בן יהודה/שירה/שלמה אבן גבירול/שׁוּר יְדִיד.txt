<h1>שׁוּר יְדִיד</h1>
שלמה אבן גבירול
<blockquote><p>שׁוּר יְדִיד כְּאֵבִי / וַחֲמַס יְרִיבִי / וְלוֹחֲמִי</p> <p>יִלְעֲגָה לְצִירִי / צַר וְיֵרְדְּ עָרִי / וְהוֹלְמִי</p> <br/> <p>לוֹהֲטִים לְאֵידִי מְהַלְּלִים / אֶת כְּבוֹד זְקֵנַי מְחַלְּלִים / בּוֹזְזִים יְקָרִי וְשׁוֹלְלִים</p> <br/> <p>שִׁלְּחוּ בְקִרְבִּי / גַּחֲלֵי שְׁבִיבִי / וְטַעֲמִי</p> <p>מַר בְּמַאְסָרִי / גַּם דְּמַע נְהָרִי / בְּלִי דְמִי</p> <br/> <p>מוֹעֲדַי כְּקֶדֶם תְּחַדְּשֵׁם / שׁוּב אֱלֵי הֲמוֹנָךְ וְקַדְּשֵׁם / אוֹיְבַי בְּאֵיבָה תְגָרְשֵׁם</p> <br/> <p>לַהֲגִי וְנִיבִי / הַאֲזִין וְרִיבִי / מְרוֹמְמִי</p> <p>רִיב וְגַם יְקָרִי / שָׂא בְיַד הֲדָרִי / וְנָעֳמִי</p> <br/> <p>הַעֲלֵה מְיַחְלֵי פְדוּתְךָ / לַחֲצַר דְּבִירָךְ וּבֵיתְךָ / שָׁם לְךָ תְרַנֵּן עֲדָתְךָ</p> <br/> <p>יַשְּׁרָה נְתִיבִי / יָהּ וּמַחֲרִיבִי / וְזוֹעֲמִי</p> <p>אֹר וְאָז בְּהָרִי / אוֹדְךָ בְּשִׁירִי / וְנָאֳמִי.</p> <br/></blockquote>
