<h1>[במוצאי שבת]</h1>
שלמה אבן גבירול
<p>ולה מרת ͘ יה פי אביה [ולו קינה על אביו]</p>
<p>È — — — — — / È — — — — — / È — — — — —</p>
<blockquote><p>בְּמוֹצָאֵי שַׁבָּת / צְרָרוּנִי רַבַּת / וְנִגְרַזְתִּי חָפְזִי</p></blockquote>
<blockquote><p>וְאֵיכָה לֹא-אִכָּף / וְלֹא-אֶסְפֹּק עַל-כָּף / וְלֹא-אֹמַר רָזִי</p></blockquote>
<blockquote><p>וְאֶשְׁתֶּה קֻבַּעַת / וְאָסִיר טַבַּעַת / וְאַשְׁלִיךְ כּוּמָזִי</p></blockquote>
<blockquote><p>וְאֶשְׁכַּח מַאְוַיַּי / וְאֶמְאַס אֶת-חַיַּי / וְאֹבִידָה עֶלְזִי</p></blockquote>
<blockquote><p>לְהֹוָה נִצַּחַת / וְאֵשׁ מִתְלַקַּחַת / בְּצַמֶּרֶת אַרְזִי</p></blockquote>
<blockquote><p>וּמָתַי אֶתְאַפֵּק / לְשֶׁבֶר מִתְרַפֵּק / וְהַמָּוֶת גּוֹזִי</p></blockquote>
<blockquote><p>וְכַבִּיר יָרוֹעַ / וְלֹא-נִכַּר שׁוֹעַ / כְּדִבְרֵי הַבּוּזִי</p></blockquote>
<blockquote><p>אֲרוֹן הָאֵל נִלְקַח / בְּיוֹם הוֹדִי לֻקַּח / וְתִפְאֶרֶת עֻזִּי</p></blockquote>
<blockquote><p>וְרוּחַ מֵעֹשׂוֹ / נְשָׂאַתְהוּ כִּנְשֹׂא / יְחֶזְקֵאל בֶּן-בּוּזִי</p></blockquote>
<blockquote><p>אֲבָל הָיָה רֵעַ / אֱמוּנַת הוֹשֵׁעַ / וְכָלֵב הַקְּנִזִּי</p></blockquote>
<blockquote><p>בְּאוֹר חַיִּים יֵאוֹר / וְעָלָיו יִשָּׂא אוֹר / אֱלֹהֵי מָעוּזִּי. <br/></p></blockquote>
