<h1>שְׁמוֹנַת אֲלָפִים</h1>
שלמה אבן גבירול
<blockquote><p>שְׁמוֹנַת אֲלָפִים יַבְקִיעַ / גַּלְגַּל תַּחַת רָקִיעַ</p> <p>לַיְלָה וְיוֹמָם יַבִּיעַ / וּתְהִלּוֹת אֵל יַקְבִּיעַ</p> <p>מֵאָז יִסֹּב וְיַפְסִיעַ / וְגַּלְגַּל שֶׁמֶשׁ יַסִּיעַ</p> <p>הוֹלֵךְ עַל גַּב יָצִיעַ / וְאוֹר כּוֹכָבִים יַקְצִיעַ</p> <p>וְהַמַּשְׂכִּילִים יַזְהִירוּ / כְּזֹהַר הָרָקִיעַ.</p> <br/></blockquote>
<blockquote><p>בְּמִזְרָח לֶכְתּוֹ יַצְנִיעַ / וּבִפְאַת מַעֲרָב יַרְגִּיעַ</p> <p>נֶגֶד כֵּס יָהּ יַגִּיעַ / וְשָׁם יִשְׁתַּחֲוֶה וְיַפְגִּיעַ</p> <p>יֵצֵא כְּחָתָן יָרִיעַ / וּבֶטֶן אֲדָמָה יַזְרִיעַ</p> <p>הָדָר לַכֹּל יַשְׁמִיעַ / וּגְבוּרָתוֹ יוֹדִיעַ</p> <br/></blockquote>
<blockquote><p>דּוֹלֶה שֶׁמֶשׁ וּמַטְבִּיעַ / פּוֹתֵחַ יָד וּמַשְׂבִּיעַ</p> <p>הוֹד אֵל צַדִּיק וּמוֹשִׁיעַ / וְהוּא יַשְׁקִיט וּמִי יַרְשִׁיעַ</p> <p>מֵרוֹם תֵּרֵד וְתוֹפִיעַ / זֵדִים וְקָמִים תַּכְנִיעַ</p> <p>אַבְנֵי רָצוֹן תַּבְקִיעַ / תַּרְבִּיץ בַּפּוּךְ וְתַקְבִּיעַ</p> <br/></blockquote>
<blockquote><p>לִבְנוֹת מִקְדָּשׁ לְהוֹפִיעַ / עַל בֵּית שֶׁמֶשׁ וְיָפִיעַ</p> <p>קָנִים תַּצִּיב וְתַקְבִּיעַ / פֶּרַח כַּפְתּוֹר וְגָבִיעַ</p> <p>יַחַד צֶאֱצָא וְצָפִיעַ / נְשָׁמוֹת נֶצַח תַּשְׁפִּיעַ.</p> <br/></blockquote>
