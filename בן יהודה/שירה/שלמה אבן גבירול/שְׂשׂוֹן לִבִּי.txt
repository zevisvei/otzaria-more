<h1>שְׂשׂוֹן לִבִּי</h1>
שלמה אבן גבירול
<blockquote><p>שְׂשׂוֹן לִבִּי / מִשְׂגַּבִּי / הַנִּשְׂגָּב שְׁמוֹ לָעַד</p> <p>סְלַח פִּשְׁעִי / וְחִישׁ יִשְׁעִי / הָבֵא צִיר הַנּוֹעָד</p> <p>אֵלִיָּה / הַתִּשְׁבִּי / מִתּוֹשָׁבֵי גִלְעָד</p> <br/></blockquote>
<blockquote><p>לְנִדְכָּאִים / וְחֵלְכָּאִים / שְׂבֵעֵי בוּז וְגַם לָצוֹן</p> <p>פְּנֵה שַׁדַּי / וְתֹאמַר דַּי / לְאֹרֶךְ קֵץ הַקִּיצוֹן</p> <p>שְׁלַח רוֹעֶה / לְצֹאן תּוֹעֶה / וְאַל תִּטֹּשׁ מְעַט הַצֹּאן</p> <p>אֲשֶׁר נָגֹז / וְגַם נָבֹז / מִלִּפְנִים וְלַחִיצוֹן</p> <p>וְהוּא מְיַחֵל / כְּמוֹ רָחֵל / לְיוֹם תִּקְרָא שְׁנַת רָצוֹן</p> <p>וְצִיץ צַמַּח / וְאָז נִשְׂמַח / בְּשִׂמְחַת בֵּית הַוָּעַד</p> <br/></blockquote>
<blockquote><p>מְחָצוּנִי / לְחָצוּנִי / פְּרִיץ חַיּוֹת וַאֲרָיוֹת</p> <p>הִנְנִי גָר / וְגַם נִסְגָּר / בְּיַד אַרְבַּע מַלְכֻיּוֹת</p> <p>נְבוֹ קָרַס / וְקָם פָּרַס / וֶאֱלִיפַז וּנְבָיוֹת</p> <p>זֶה יִתְפֹּשׂ / וְזֶה יִרְפֹּשׂ / וְזֶה יַדִּיחַ שֵׂיוֹת</p> <p>וּבְכָל זֶה / אֲנִי אֶחֱזֶה / וְעֵינַי לְךָ צוֹפִיּוֹת</p> <p>וְכָל דּוֹרֵשׁ / קֵץ יְפָרֵשׁ / וְיִסְפֹּר הָאוֹתִיּוֹת</p></blockquote>
<blockquote><p>אוּלַי תַּם / אֲשֶׁר נֶחְתַּם / וּ[בֵ]יתִי תָכִין עֲדֵי עַד.</p> <br/></blockquote>
