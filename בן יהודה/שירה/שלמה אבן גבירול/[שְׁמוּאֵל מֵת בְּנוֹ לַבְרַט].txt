<h1>[שְׁמוּאֵל מֵת בְּנוֹ לַבְרַט]</h1>
שלמה אבן גבירול
<p>וקאל איצ͘א</p>
<br/>
<blockquote><p>שְׁמוּאֵל מֵת בְּנוֹ לַבְרַט / וְעָמַדְתָּ עֲלֵי כַּנּוֹ</p> <p>וְכָלִינוּ מְאֹד אֵלָיו / וְאוּלָם הִנְּךָ הִנּוֹ</p> <p>וְלוּ יֵשׁ חַי תְּפַשְׂנוּהוּ / וְרָצַעְנוּ לְךָ אָזְנוֹ</p> <p>הֲמִן הָעֵץ אֲשֶׁר מָנַע / אֱלֹהִים מֵעֲצֵי עֶדְנוֹ</p> <p>תְּמַלֵּא בִטְנְךָ דַּעַת / וְתִתְעַדֵּן בְּטוּב דִּשְׁנוֹ</p> <p>וְתִתְחַכַּם וְתֹאמַר כִּי / אֱלוֹהַּ עֹז וְגַם־קַנּוֹא</p> <p>קְרָאָנוּ לְךָ שִׁיר שָׂם / בְּבֵין כִּימָה וְעָשׁ קִנּוֹ</p> <p>וְלִשְׂמֹאלוֹ מְנַחֵם שָׂם / וְשָׂם אָבוּןְ לִימִינוֹ</p> <p>וּמַה־נִּמְרָץ וּמַה־נַּעְרָץ / בְּמִדְבָּרָיו וְעִנְיָנוֹ</p> <p>וּמֵחֵשֶׁק שְׁלַחְנוּהוּ / בְּיַד־כָּרִי לְאֶרֶץ נֹא</p> <p>וְלַסֹּבֵא נְתַנּוּהוּ / לְהִתְרוֹנֵן בְּאַגָּנוֹ</p> <p>וּבֶן־גִּבּוֹר וּבֶן־חַיִל / כְּתַבְנוּהוּ בְמָגִנּוֹ.</p> <p>אֲהָהּ לַשִּׁיר אֲשֶׁר נָפַל / וְאֵין תִּקְוָה לְפִדְיוֹנוֹ</p> <p>בְיַד־אִישׁ יַעֲנֶה רוּחַ / וְקָדִים מָֽלְאָה בִטְנוֹ</p> <p>וְכָל־עֶצֶם לְעַצְמוֹ שָׁב / וְכָל־מִין שָׁב אֱלֵי מִינוֹ.</p></blockquote>
