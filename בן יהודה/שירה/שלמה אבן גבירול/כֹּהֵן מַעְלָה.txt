<h1>כֹּהֵן מַעְלָה</h1>
שלמה אבן גבירול
<blockquote><br/> <p>כֹּהֵן מַעְלָה / אוֹר פָּנָיו כְּמַרְאֵה בָרָק</p> <p>וְכֹהֵן מַטָּה / בְּאוֹר פְּנֵי מִזְרָח יִבְרָק</p> <p>כֹּהֵן מַעְלָה / גְּוִיָּתוֹ כְתַרְשִׁישׁ</p> <p>וְכֹהֵן מַטָּה / דְּמוּת גְּוִיָּתוֹ מְשֻׁבֶּצֶת כְּמוֹ תַרְשִׁישׁ</p> <p>כֹּהֵן מַעְלָה / הוֹד פָּנָיו כְּלַפִּידֵי אֵשׁ</p> <p>וְכֹהֵן מַטָּה / וְעֵינָיו מַבְהִיקוֹת כְּכוֹכְבֵי אֵשׁ</p> <p>כֹּהֵן מַעְלָה / זְרוֹעוֹתָיו וּמַרְגְּלוֹתָיו נְחֹשֶׁת</p> <p>וְכֹהֵן מַטָּה / חֻקּוֹ לַעֲמֹד עַל מִזְבַּח הַנְּחֹשֶׁת</p> <p>כֹּהֵן מַעְלָה / טָבוּעַ עַל שֵׁמוֹת שִׁבְעִים</p> <p>וְכֹהֵן מַטָּה / יִשָּׂא עַל לִבּוֹ שֵׁמוֹת שִׁבְעִים</p> <p>כֹּהֵן מַעְלָה / כְּסוּתוֹ כְּנָפַים שֵׁשׁ שֵׁשׁ</p> <p>וְכֹהֵן מַטָּה / לָבוּשׁ שְׁמוֹנָה בִגְדֵי שֵׁשׁ</p> <p>כֹּהֵן מַעְלָה / מָתְנָיו חֲגוּרִים בְּכֶתֶם אוּפָז</p> <p>וְכֹהֵן מַטָּה / נֶחְגָּר בְּאַבְנֵט כְּמוֹ פָז</p> <p>כֹּהֵן מַעְלָה / סוֹדֵר מְלֹא מַחְתָּה גַּחֲלֵי אֵשׁ</p> <p>וְכֹהֵן מַטָּה / עוֹרֵךְ מְלֹא מַחְתָּה גַּחֲלֵי אֵשׁ</p> <p>כֹּהֵן מַעְלָה / פִּתּוּחַ שֵׁם קֹדֶשׁ עַל מִצְחוֹ</p> <p>וְכֹהֵן מַטָּה / צִיץ נֵזֶר הַזָּהָב עַל מִצְחוֹ</p> <p>כֹּהֵן מַעְלָה / קוֹפֵץ יְמִינוֹ וּשְׂמֹאלוֹ לְרוֹם מֵצִיץ</p> <p>וְכֹהֵן מַטָּה / רוֹחֵשׁ וְנוֹשֵׂא יָדָיו לְמַעְלָה מִן הַצִּיץ</p> <p>כֹּהֵן מַעְלָה / שׁוֹקֵד בִּפְנִים וְנִקְרָא שַׂר הַפָּנִים</p> <p>וְכֹהֵן מַטָּה / תָּאֳרוֹ בִּפְנִים וּמַסְבִּיר פָּנִים</p> <p>כֹּהֵן מַעְלָה / תְּפִלָּה עוֹרֵךְ אָנָּא הַשֵּׁם</p> <p>וְכֹהֵן מַטָּה / תַּעֲנִית עוֹרֵךְ וּמִתְוַדֶּה אָנָּא הַשֵּׁם</p> <p>כֹּהֵן מַעְלָה / מְמֻנֶּה שַׂר עַל מַלְאֲכֵי שָׁלוֹם</p> <p>וְכֹהֵן מַטָּה / מְמֻנֶּה שַׂר עַל זֶרַע הַשָּׁלוֹם</p> <p>זֶה מְכַפֵּר מִשְּׁמֵי קֹדֶשׁ</p></blockquote>
<blockquote><p>[וְזֶה ]</p></blockquote>
