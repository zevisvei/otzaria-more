<h1>שְׁדוּדִים נְדוּדִים</h1>
שלמה אבן גבירול
<blockquote><p>שְׁדוּדִים נְדוּדִים תְּקַבֵּץ / אֶל נְוַת מַרְבֵּץ / וְלוֹבְשֵׁי תַשְׁבֵּץ</p></blockquote>
<blockquote><p>לְכַפֵּר עַל הַדָּמִים / בֵּין הָעַמִּים / לְבֵית עוֹלָמִים / מֵאָז אָמַר שְׁלֹמֹה</p> <br/></blockquote>
<blockquote><p>לִִבְנֵי קְטוּרָה / שִׂים בָּם מְדוּרָה / וּמִבְּנֵי גְבִירָה</p></blockquote>
<blockquote><p>הָסֵר זַעְמָךְ / וּגְדֹר כַּרְמָךְ / וְיִשְׂרָאֵל עַמָּךְ / כֶּרֶם הָיָה לִשְׁלֹמֹה</p> <br/></blockquote>
<blockquote><p>מַלְכִּי וְחֵילִי / הֱיֵה עוֹזֵר לִי / וְכוֹנֵן זְבוּלִי</p></blockquote>
<blockquote><p>וְשִׁיר לְךָ אָשִׁיר / וְשַׁי לְךָ אַכְשִׁיר / בְּמִזְמוֹר וָשִׁיר / הַשִּׁירִים אֲשֶׁר לִשְׁלֹמֹה</p> <br/></blockquote>
<blockquote><p>הַרְאֵה לַהֲמוֹנַי / טוּבְךָ שׁוֹכֵן מְעוֹנַי / מְיַחֲלִים יְשׁוּעַת אֲדֹנָי</p></blockquote>
<blockquote><p>כְּמֵאָז בֵּין חֲנֵסִים / עֲשֵׂה אוֹת וְנִסִּים / עֲנֵנִי מוֹשִׁיעַ חוֹסִים / גַּם אֵלֶּה מִשְׁלֵי שְׁלֹמֹה</p> <br/></blockquote>
<blockquote><p>חִזְקוּ בְּנֵי נְדִיבִים / זֶרַע אֲהוּבִים / וּבִטְחוּ בְיוֹשֵׁב כְּרוּבִים</p></blockquote>
<blockquote><p>כִּי הַקֵּץ נִגְמַר / וְצַדִּיק יִפְרַח כַּתָּמָר / מְהֵרָה כִּי כֹה אָמַר / יְיָ אֶל שְׁלֹמֹה.</p> <br/></blockquote>
