<h1>שְׁעֵה עֶלְיוֹן</h1>
שלמה אבן גבירול
<blockquote><br/> <p>שְׁעֵה עֶלְיוֹן / לְעַם אֶבְיוֹן / אֲשֶׁר בְּכָל פֶּה הֵם אֲכוּלִים</p> <p>לִפְדּוֹתָם / לְהַנְחוֹתָם / לְיַשֵּׁר לָהֶם שְׁבִילִים</p> <p>מְחֵה דִמְעָה / בְּקוֹל שַׁוְעָה / לִהְיוֹתָם נִגְאָלִים</p> <p>הָחִישָׁה / וְהַגִּישָׁה / זְמַן הַקֵּץ לִשְׁכוּלִים</p> <p>קְרָא לָהֶם / שְׁנַת רָצוֹן / לְנַחֵם לֵב אֲבֵלִים</p> <br/></blockquote>
<blockquote><p>לְךָ יוֹדוּ / וְיַגִּידוּ / פְּלָאֶיךָ בְּתַעֲצוּמִים</p> <p>בְּלֵב נִשְׁבָּר / וְכָל מַעֲבָר / קוֹרְאֵי בְלַחַץ קָמִים</p> <p>הָסֵר יְגוֹנוֹת / וַאֲסוֹנוֹת / מֵעֲלֵיהֶם שׁוֹכֵן מְרוֹמִים</p> <p>פְּדֵה מְיֻסָּר / בְּיַד כָּל צַר / וְגַם צוֹעֵק בְּדַם חֲלָלִים</p> <br/></blockquote>
<blockquote><p>מִבְטֶחָם / לְהַ[רְוִיחָם] / חִישׁ נָא קֵץ וְקָרֵב זְמַן</p> <p>לע…דָ[ה] / וְגַלְמוּדָה / וְגַם מִמְּךָ [מַן</p> <p>                      מַן</p> <p>                      לִים</p> <p>ה</p> <p>                      לִים.]</p></blockquote>
