<h1>שַׁחַר קַמְתִּי גַּם</h1>
שלמה אבן גבירול
<blockquote><p>שַׁחַר קַמְתִּי / גַּם נִכְלַמְתִּי / לָשֵׂאת לְךָ פָנָי</p> <p>בְּךָ חָסִיתִי / וְלֹא כִסִּיתִי / עֲווֹנַי וּזְדוֹנָי</p> <p>כִּי יָדַעְתִּי / גַּם שָׁמַעְתִּי / אֵל עוֹלָם אֲדֹנָי</p> <p>בַּנְתָּ לְרֵעֵנוּ / יְיָ מַלְכֵּנוּ / הוּא יוֹשִׁיעֵנוּ</p> <br/></blockquote>
<blockquote><p>לְךָ צוֹפִיָּה / בַּת הוֹמִיָּה / מִבֵּית מְגוּרֶיהָ</p></blockquote>
<blockquote><p>תַּשְׁחֶה עָרְפָּהּ / תִּפְרֹשׂ כַּפָּהּ / תִּקַּח אֲמָרֶיהָ</p></blockquote>
<blockquote><p>עָנְיָהּ תִּפְנֶה / בֵּיתָהּ תִּבְנֶה / נִפְרְצוּ גְדֵרֶיהָ</p> <p>עַל רֹב פְּשָׁעֵינוּ / יְיָ מַלְכֵּנוּ / הוּא יוֹשִׁיעֵנוּ</p> <br/></blockquote>
<blockquote><p>מֶלֶךְ מַקְשִׁיב / מָתַי תָּשִׁיב / צִפּוֹר גֵּרַשְׁתָּ</p> <p>נָא אַל תִּבְזֶה / עֱנוּת עָנִי זֶה / כַּמֶּה נָטַשְׁתָּ</p> <p>בַּעֲבוּר יְדִידְךָ / אַתָּה יָדְךָ / גּוֹיִים הוֹרַשְׁתָּ</p> <p>וַתִּטָּעֵנוּ / יְיָ מַלְכֵּנוּ / הוּא יוֹשִׁיעֵנוּ</p> <br/></blockquote>
<blockquote><p>הַיּוֹם תִּגְאַל / אֶבְיוֹן יִשְׁאַל / גּוֹאֵל חֲסָדֶיךָ</p> <p>אִם עָנָה בוֹ / רֹעַ לְבָבוֹ / הִנּוֹ בְיָדֶךָ</p> <p>קַח שַׁוְעָתוֹ / וְגַם דִּמְעָתוֹ / שִׂימָה בְנֹאדֶךָ</p> <p>אֱלֹהֵי יִשְׁעֵנוּ / יְיָ מַלְכֵּנוּ / הוּא יוֹשִׁיעֵנוּ</p> <br/></blockquote>
<blockquote><p>חַזֵּק רִפְיוֹן / מֶלֶךְ עֶלְיוֹן / וְרַחֵם עַמֶּךָ</p> <p>זַכֵּה אוֹתָם / בִּזְכוּת אֲבוֹתָם / שְׁלֹשֶׁת תְּמִימֶיךָ</p> <p>קָרֵב קִצָּם / וּגְדֹר פִּרְצָם / לְמַעַן רַחֲמֶיךָ</p> <p>שְׁעֵה שַׁוְעָתֵנוּ / יְיָ מַלְכֵּנוּ / הוּא יוֹשִׁיעֵנוּ.</p> <br/></blockquote>
