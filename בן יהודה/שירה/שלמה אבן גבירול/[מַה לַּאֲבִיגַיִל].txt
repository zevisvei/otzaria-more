<h1>[מַה לַּאֲבִיגַיִל]</h1>
שלמה אבן גבירול
<p>ומן אג͘ד ّ שערה ואעג͘ב קולה [משיריו המוצלחים ודבריו הנפלאים ביותר]</p>
<p>— — È — / — — È — / — — È — // — — È — / — — È — / — — —</p>
<br/>
<blockquote><p>מַה לַּאֲבִיגַיִל אֲשֶׁר לָקָחָה / נַפְשִׁי בְּעֵינֶיהָ וְשָׁם הִנִּיחָה?</p> <p>כָּל חוֹשְׁקֶיהָ דִּבְּרוּ לָהּ כִּי שְׂנֵא- / תִיהָ וְשִׂנְאָתִי מְאֹד נִצָּחָה!</p> <p>עִם זֹאת וְאִם שָׁכְחָה יְדִידוֹתַי, הֲלֹא / אֶשְׁמֹר בְּרִית אַהֲבָה וְלֹא              <br/>                                             אֶשְׁכָּחָה:</p> <p>שָׁלַח בְּנוֹ יִשַׁי לְבֵיתָהּ – וַאֲנִי / אֵלֵךְ אֱלֵי בֵיתָהּ וְלֹא אֶשְׁלָחָה.</p> <p>אִם אֵין בְּיוֹם גָּלוּת לְאֵל קָרְבָּן – הֲלֹא / עוֹלוֹת וְקָרְבָּנוֹת לְזֹאת              <br/>                                               אֶזְבָּחָה!</p></blockquote>
