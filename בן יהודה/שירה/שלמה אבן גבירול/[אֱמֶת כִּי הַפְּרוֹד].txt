<h1>[אֱמֶת כִּי הַפְּרוֹד]</h1>
שלמה אבן גבירול
<p>[לידיד שנפרד ממנו]</p>
<br/>
<p>È — — — / È — — — / È — — / / È — — — /È — — — / È — —</p>
<br/>
<blockquote><p>אֱמֶת כִּי הַפְּרוֹד מַחְלֶה לְבָבוֹת / וּמַבְעִיר בַּקְּרָבוֹת אֵשׁ לְהָבוֹת</p> <p>וְתָמִיד יִבְעֲרוּ מוֹקְדֵי אֲהָבִים / בְּתוֹךְ הַלֵּב כְּאִלּוּ הֵם חֲרָבוֹת</p> <p>עֲלֵי דוֹדַי יְחַיּוּ הַנְּפָשׁוֹת / וְיָאִירוּ לְכָל-דֹּרֵךְ נְתִיבוֹת</p> <p>יְדִיד נַפְשִׁי אֲהָבֶיךָ בְלִבִּי / וְאִם-לֹא אֶזְכְּרֵם תָּמִיד בְּנִיבוֹת</p> <p>וְלֹא אֶשְׁכַּח יְדִידוּתְךָ אֲהוּבִי / וּמִלֶּיךָ בְּתוֹךְ לִבִּי חֲצוּבוֹת</p> <p>וּפֵרוּדְךָ אֲדֹנִי עַז וְקָשֶׁה / יְסַמֵּר שַׂעֲרַת אַנְשֵׁי קְרָבוֹת</p> <p>יְשַׁלַּח-לָךְ אֱלֹהַי טַל רְצוֹנוֹ / וְנַפְשָׁךְ יִשְׁמְרָה שֹׁכֵן עֲרָבוֹת. <br/></p></blockquote>
