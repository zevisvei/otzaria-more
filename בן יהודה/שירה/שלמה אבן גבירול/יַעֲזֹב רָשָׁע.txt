<h1>יַעֲזֹב רָשָׁע</h1>
שלמה אבן גבירול
<blockquote><p>יַעֲזֹב רָשָׁע נְתִיבוֹ / וְיַכְנִיעַ רוּם לְבָבוֹ</p> <p>וְיָשֹׁב אֶל יְיָ / וִירַחֲמֵהוּ בְשׁוּבוֹ</p> <p>בְּטֶרֶם יִסָּפֶה / אוֹ יוֹמוֹ יָבוֹא</p> <br/></blockquote>
<blockquote><p>עֲזָב נָא בֶּן אָדָם / עֲזֹב שֶׁמֶץ וְדִבָּה</p> <p>בְּטֶרֶם תִּתְעוֹפֵף / כְּיוֹנִים אֶל אֲרֻבָּה</p> <p>וְהַנּוֹשֶׁה יָבוֹא / לְבַקֵּשׁ הָעֲרֻבָּה</p> <p>לָכֵן שׁוּב וּזְכֹר / אֶת הַיּוֹם הַבָּא</p> <p>כִּי הַיּוֹם קָצָר / וְהַמְּלָאכָה מְרֻבָּה</p> <p>מִי יָרֵא וְחָרֵד / יִתֵּן אֶל לְבָבוֹ</p> <br/></blockquote>
<blockquote><p>עֲזָב נָא בֶּן אָדָם / עֲזָב הֶבֶל וְהַרְחֵק</p> <p>כִּי מָחָר תִּבְכֶּה / מֵאֲשֶׁר בּוֹ תְשַׂחֵק</p> <p>בִּכְתֹב אִישׁ רִיבְךָ / כְּתָב לֹא יִמָּחֵק</p> <p>וְלֹא יוֹעִיל מַתָּן / וְלֹא שֹׁחַד בַּחֵק</p> <p>וְהַפּוֹעֲלִים עֲצֵלִים / וּבַעַל הַבַּיִת דּוֹחֵק</p> <p>וְאִישׁ הָרִיב נִגָּשׁ / וְגַם נִצָּב לְרִיבוֹ</p> <br/></blockquote>
<blockquote><p>עֲזָב נָא בֶּן אָדָם / עֲזָב יְקָר נוֹדֵד נְדוֹד</p> <p>כִּי עָרוּךְ תָּפְתֶּה / בְּכָל לַהַב וְכִידוֹד</p> <p>זְכָר נָא אַחֲרִיתְךָ / בְּעֵת תּוּשַׁד שָׁדוֹד</p> <p>כִּי כָל אָח יֹאמַר הֶאָח / וְכָל דּוֹר יִדֹּד</p> <p>לָכֵן שׁוּב וְהַתְקֵן / עַצְמְךָ בַּפְּרוֹזְדוֹר</p> <p>כְּדֵי שֶׁתִּכָּנֵס / לַטְּרַקְלִין וְטוּבוֹ</p> <br/></blockquote>
<blockquote><p>עֲזָב נָא בֶּן אָדָם / עֲזָב נַחַת וְשִׂמְחָה</p> <p>בְּטֶרֶם נוּד רוּחֲךָ / בְּכָל פֵּאָה וְרוּחָה</p> <p>וְעֵינֶיךָ יִרְאוּ / גֵיהִנָם פְּתוּחָה</p> <p>וּמֵאַיִן בָּאתָ / מִלֵּחָה סְרוּחָה</p> <p>לְאָן אַתָּה הוֹלֵךְ / לְבוֹר שַׁחַת וְשׁוּחָה</p> <p>וְהַחֶשְׁבּוֹן תִּתֵּן / לְשָׂם עָבִים רְכוּבוֹ.</p> <br/></blockquote>
