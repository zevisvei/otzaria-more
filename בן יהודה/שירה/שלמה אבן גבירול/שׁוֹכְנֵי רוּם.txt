<h1>שׁוֹכְנֵי רוּם</h1>
שלמה אבן גבירול
<blockquote><p>שׁוֹכְנֵי רוּם וְתַחַת / צִבְאוֹתָם וְחֵילָם</p> <p>לְךָ מָעֹז יִתְּנוּ עֹז / יְמֵי עוֹלָם כֻּלָּם</p> <p>מְמַלְּלִים וּמְהַלְלִים / בִּלְבָבָם וְקוֹלָם</p> <p>יְהִי שֵׁם יְיָ מְבֹרָךְ / מֵעַתָּה וְעַד עוֹלָם.</p> <br/></blockquote>
<blockquote><p>בַּשַּׁחַק אֵלִים / חֲצוּבֵי אֵשׁ וָמָיִם</p> <p>וּבָאָרֶץ נְדִיבִים / הוֹגִים דַּת יוֹמַיִם</p> <p>מְמַלְּלִים וּמְהַלְלִים / יִחוּדְךָ פַעֲמַיִם</p> <p>מִקְצֵה הַשָּׁמַיִם / וְעַד קְצֵה הַשָּׁמַיִם.</p> <br/></blockquote>
<blockquote><p>בַּשַּׁחַק שִׁמְךָ מְקֻדָּשׁ / בְּסוֹד כְּרוּבִים</p> <p>וּבָאָרֶץ עֹז לְךָ / יִתְּנוּ עַם נְדִיבִים</p> <p>מְמַלְּלִים וּמְהַלְלִים / לְמוּלְךָ נִצָּבִים</p> <p>מֵעֲלוֹת הַשַּׁחַר / וְעַד צֵאת הַכּוֹכָבִים.</p> <br/></blockquote>
<blockquote><p>בַּשַּׁחַק שִׁמְךָ / יוֹדוּ תַרְשִׁישִׁים</p> <p>וּבָאָרֶץ בְּחִירִים / לְשִׁמְךָ מַקְדִּישִׁים</p> <p>מְמַלְּלִים וּמְהַלְלִים / וְנֶגְדְּךָ נִגָּשִׁים</p> <p>מִנַּעַר וְעַד זָקֵן / וְעַד טַף וְנָשִׁים.</p> <br/></blockquote>
<blockquote><p>[בַּשַּׁחַק</p> <p>וּבָאָרֶץ</p> <p>מְמַלְּלִים וּמְהַלְלִים</p> <p>מ    / וְעַד</p> <br/></blockquote>
<blockquote><p>בַּשַּׁחַק     /   דֶק</p> <p>וּבָאָרֶץ    /   דֶק</p> <p>מְמַלְּלִים וּמְהַלְלִים /   דֶק]</p> <p>בֵּין קוֹץ וְדַרְדַּר / עַד יָבוֹא וְיוֹרֶה צֶדֶק.</p> <br/></blockquote>
