<h1>שַׁחַר קַמְתִּי לְהוֹדוֹת</h1>
שלמה אבן גבירול
<blockquote><p>שַׁחַר קַמְתִּי לְהוֹדוֹת / לְךָ אֱלֹהֵי תְהִלָּתִי</p> <p>וַאֲרַנֵּן לַבֹּקֶר / וְאוֹדִיעֲךָ חַטָּאתִי</p> <p>יִנָּתֶן לִי בָּזֶה / שָׂכָר לִפְעֻלָּתִי</p> <p>נַפְשִׁי בִּשְׁאֵלָתִי / וְעַמִּי בְּבַקָּשָׁתִי</p> <br/></blockquote>
<blockquote><p>לְפָנִים זֹאת בְּיִשְׂרָאֵל / בְּעֵת מַקְרִיבֵי קָרְבָּן</p> <p>אִם חַטָּאתוֹ כְּשָׁנִים / כֻּלּוֹ הָפַךְ לָבָן</p> <p>אֶדַּדֶּה כָל שְׁנוֹתַי / אֲנִי תָמִיד עַל חֻרְבָּן</p> <p>וְאֵין לִי לְבַד מִלָּתִי / וְעוֹלָתִי תְפִלָּתִי</p> <br/></blockquote>
<blockquote><p>מִזְבֵּחַ בִּהְיוֹתוֹ / וְקָדָשִׁים לַזְּבִיחָה</p> <p>וְהֵבִיא אִישׁ אֶת זִבְחוֹ / וְנִכַּפֵּר בִּסְלִיחָה</p> <p>אָפֵס מִנִּי מְכַהֵן / וְאֵין זֶבַח וְאֵין מִנְחָה</p> <p>תּוֹדָתִי זְבִיחָתִי / וְשִׂיחָתִי מִנְחָתִי</p> <br/></blockquote>
<blockquote><p>הֵן בִּהְיוֹת הָעֲבוֹדָה / וְכֹהֲנִים עַל מִשְׁמֶרֶת</p> <p>הֲלֹא חַטָּאת מַכְשֶׁרֶת / וְהָעוֹלָה מְכַפֶּרֶת</p> <p>וְאֵין חַטָּאת וְאֵין עוֹלָה / וְלֹא חֵלֶב וְיוֹתֶרֶת</p> <p>לְבַד אֶשְׁפֹּךְ אֶת רִנָּתִי / וְאַפִּיל אֶת תְּחִנָּתִי</p> <br/></blockquote>
<blockquote><p>קֶדֶם בִּהְיוֹת מִקְדָּשׁ / בְּקוּם זְרִיזִים בְּאַשְׁמֹרֶת</p> <p>טְהוֹרִים נִצְּבוּ לְהָפִיס / הַחֲדָשִׁים לִקְטֹרֶת</p> <p>וְאֵין לְבוֹנָה וְאֵין קְטֹרֶת / וְנִשְׁאַרְתִּי שְׁחַרְחֹרֶת</p> <p>בְּהִתְוַדּוֹת חַטָּאתִי / כְּמוֹ קָרְבָּן תּוֹדָתִי</p> <br/></blockquote>
<blockquote><p>חֲזֵה קָדוֹשׁ כִּי יָדִי / כָּבְדָה עַל אַנְחָתִי</p> <p>וְאֵין מִי יָגֵן בַּעֲדִי / וְאֵין לִי בֵית מְנוּחָתִי</p> <p>לְבַד בְּךָ אַזְכִּיר שְׁמֶךָ / מָעֹז צוּר יְשׁוּעָתִי</p> <p>תַּעֲבִיר אֶת אַשְׁמָתִי / וְעָנְתָה בִּי צִדְקָתִי.</p> <br/></blockquote>
