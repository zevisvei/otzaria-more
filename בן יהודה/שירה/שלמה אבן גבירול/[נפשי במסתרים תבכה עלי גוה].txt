<h1>[נפשי במסתרים תבכה עלי גוה]</h1>
שלמה אבן גבירול
<h2 id="נפשיבמסתריםתבכהעליגוהשלמהאבןגבירול">[נפשי במסתרים תבכה עלי גוה] / שלמה אבן גבירול</h2>
<p>וקאל איצ ͘א</p>
<p>— — È — / — — / — — È — / — — // — — È — / — — / — — È — / — —</p>
<br/>
<blockquote><p>נַפְשִׁי בְּמִסְתָּרִים תִּבְכֶּה עֲלֵי גֵוָה / שֶׁצָּף בְּשֶׁצֶף הַחִבָּה וְהַתַּאְוָה</p></blockquote>
<blockquote><p>כַּמָּה יְיַחֵל אִישׁ כָּלֶה אֲשֶׁר גּוּפוֹ / כָלָה וְאֵין תּוֹחֶלֶת לוֹ וְלֹא-תִקְוָה</p></blockquote>
<blockquote><p>לוּ תִשְׁקְלוּהוּ עִם-הֶבֶל אֲשֶׁר נִשְׁוֶה / גּוּף נֶחֱלָה עִם-נֶפֶשׁ כָּֽלְתָה דָּוָה</p></blockquote>
<blockquote><p>הַשֹּֽׁכְנִי מָרוֹם לָמָּה בְרָאתַנִי / לָמָה הֲרוּג עַיִן מִכָּל-בְּנֵי עַלְוָה.</p></blockquote>
