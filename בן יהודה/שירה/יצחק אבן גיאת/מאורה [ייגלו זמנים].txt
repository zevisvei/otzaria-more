<h1>מאורה [ייגלו זמנים]</h1>
יצחק אבן גיאת
<blockquote><p>יִגָּלוּ זְמַנִּים / זֶה כַמָּה סְתַמְתָּם,</p> <p>יֵאָמְנוּ חֲזוֹנִים / עַד עֵת קֵץ חֲתַמְתָּם.</p> <p>עַל יַד אִישׁ לְפָנִים / תַּתָּם אוֹת, וְשַׂמְתָּם</p> <p>לַעֲזֹר / לִי מִצַּר, וְלִשְׁמֹר / הָאֱמוּנָה.</p> <br/></blockquote>
<blockquote><p>[5] צָבְאוּ עַל הֲמוֹנַי / מִסָּבִיב תְּלָאוֹת,</p> <p>וּלְיֵשַׁע אֱמוּנַי / לֹא אֶרְאֶה פְלָאוֹת.</p> <p>תָּמִיד אֶל מְעוֹנַי / עֵינֵיהֶם נְשׂוּאוֹת</p> <p>בֶּן בְּכוֹר / לִנְטֹעַ, וְלַעְקֹר/ בַּת עֲדִינָה.</p> <br/></blockquote>
<blockquote><p>חַי, אַיֵּה גְבוּרוֹת / עוֹלָמִים קְדוּמִים?</p> <p>[10] הִכִּיתָ בְכוֹרוֹת / לוּדִים וַעֲנָמִים.</p> <p>הִנָּשֵׂא בְעַבְרוֹת / אַכְזָרִים וְקָמִים,</p> <p>שׁוּב נְצֹר / עִיר נוֹתְרָה בְמָצוֹר / כַּמְּלוּנָה!</p> <br/></blockquote>
<blockquote><p>קִוִּיתִי בְצִלָּךְ / לָשֶׁבֶת שְׁלִישִׁית</p> <p>עִיר שָׁלֵם זְבוּלָךְ / לִבְנוֹתָהּ, וְלָשִׁית</p> <p>[15] בָּהּ כִּסֵּא, וּמָלַךְ / שָׁמָּה כַּבְּרֵאשִׁית –</p> <p>אָז כְּאוֹר / הַחַמָּה יְהִי אוֹר / הַלְּבָנָה.</p></blockquote>
