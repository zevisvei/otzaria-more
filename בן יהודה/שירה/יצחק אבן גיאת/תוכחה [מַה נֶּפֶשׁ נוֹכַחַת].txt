<h1>תוכחה [מַה נֶּפֶשׁ נוֹכַחַת]</h1>
יצחק אבן גיאת
<p><strong>סימן: א"ב ויצחק בבית האחרון</strong></p>
<p><strong>יב</strong></p>
<blockquote><p>         מַה נֶּפֶשׁ נוֹכַחַת / תָּשִׁיב בְּיוֹם תּוֹכַחַת / וְהנּוֹשֶׁה בָא לָקַחַת</p> <p>         אֵיכָה עֹרֶף מַקְשֶׁה / וּמֵהִתְרַפֵּס מַחְשֶׁה / אִישׁ אֲשֶׁר לוֹ נוֹשֶׁה</p> <p>         בְּנֵי יָמִים לְרִגְעֵיהֶם / לְנוֹשֵׁיהֶם מַקְרֵיהֶם / נוֹשִׁים בָּהֶם</p> <p>         גַּם מִמַּעַל וְתַחַת / תּוֹכָם מִשְׁתַּלַּחַת / צָרַעת פּוֹרַחַת</p> <p>5        מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         דָּבְקוּ בְצִיץ הֶתְלֵמוֹ / וּבְתוֹהוּ יְמֵי הֶבְלֵמוֹ / יַשִּׂיא מָוֶת עֲלֵימוֹ</p> <p>         הוֹד בְּעָלִים וּרְכוּשׁוֹ / יִקְרָא וְחֵיל כַּחְשׁוֹ / נָשִׁיתִי אֶתְכֶם נָשׂוֹא</p> <p>         וְנָע מְשׁוּבָה וָנַחַת / פֶּתַע לִשְׁאֹל יֵחַת / וַיִּפֹּל אֶל הַפַּחַת</p> <p>         מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         זַחֲלִי נַפְשִׁי לְמִרְיֵךְ / בִּקְרֹא לִיךְ וּלְגֶוְיִךְ / בֹּאִי וְשַׁלְּמִי אֶת נִשְׁיֵךְ</p> <p>         חִקְקֵי לִבֵּךְ בְּעוֹדוֹ / נִגְלוּ וּכְתָבָם בְּסוֹדוֹ / כָּל בַּעַל מַשֶּׁה יָדוֹ</p> <p>         טוּב יָדֶיךָ וּמִנְחַת / שָׁוְא צָרְרוּ בְאַמְתַּחַת / הִנֵּה נִשְׁחַת</p> <p>         מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         יַד אִישׁ תַּכְחִישֵׁהוּ / וְיָחִישׁ לַעְנוֹת פִּיהוּ / אֲשֶׁר יַשֶּׁה בְרֵעֵהוּ</p></blockquote>
<blockquote><p>15          כֵּן חוֹבָתוֹ רְשׁוּמָה / וַעֲלֵי מִצְחוֹ חֲתוּמָה / מַשַּׂאת מְאוּמָה</p></blockquote>
<blockquote><p>         לְעוֹלָמָהּ זוֹנַחַת / וּבָזֶה לֹא מוּנַחַת / נִמְצֵאת מִכָּאן וּמִכָּאן קָרַחַת</p> <p>         מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         מִי יִתֵּן בְּעוֹד טוּבִי / בְּטֶרֶם לָאָרֶץ שׁוּבִי / לֹא נָשִׁיתִי וְלֹא נָשׁוּ בִי</p> <p>         נוֹאַלְתִּי כְּאִיש בּוֹזֶה / וְדָבַקְתִּי בְאוֹחֲזֵי / אֶת הַמַּשָּׂא הַזֶּה</p></blockquote>
<blockquote><p>20       סְפוֹת עַל חֹם קַדַּחַת / וּבַשְּׂאֵת אוֹ סַפַּחַת / כִּי יִהְיֶה בַקָּרַחַת</p></blockquote>
<blockquote><p>         מַה נֶּפֶשׁ נוֹכַחַת…</p></blockquote>
<blockquote><p>         עֲנַן סִכְלוּת עֵינֶיךָ / כִּסָּה וְרַעְיוֹנֶיךָ / כִּי יַשֶּׁה לְךָ אֱלֹהַּ מֵעֲוֹנֶיךָ</p> <p>         פְּנוֹת אֶל חֵטְא בִּקְרוֹתוֹ / בְּלִי הָבִין אַחֲרִיתוֹ / נָשָׂא בוֹ אָלָה לְהֵאָלוֹתוֹ</p> <p>         אֶל צְרוֹר חַטָּאת צוֹרַחַת / לְהִתְחַבֵּר לַנִּשְׁחָת / אַחַת אֶל אַחַת</p> <p>25       מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         קֵרְבוֹ לַמִּשְׁפָּט לְרִיבוֹ / יַרְשִׁיעוֹ פִיו וְנִיבוֹ / הָאִישׁ אֲשֶׁר אַתָּה נוֹשֶׁה בוֹ</p> <p>         רוֹעֶה בְּהִתְיַצֵּב לְיַד / מַעְגָּל וְהָיָה צַיָּד / מַשָּׂא כָל יַד</p> <p>         שְׁפָטוּךָ כְּעִיר הַנִּדַּחַת / בְּלַהֶבֶת קוֹדַחַת / וְאֵשׁ מִתְלַקַּחַת</p> <p>         מַה נֶּפֶשׁ נוֹכַחַת…</p> <br/></blockquote>
<blockquote><p>         תּוֹם תָּם לְאֵין תּוֹם תַּמְשֶׁה / לְנוֹשֶׁה בוֹ כִּי תַשֶּׁה / לֹא תִהְיֶה לוֹ כְנוֹשֶׁה</p></blockquote>
<blockquote><p>30       יַד צֶדֶק הֵן נְטוּיָה / קֶדֶם לְךָ הִיא גְלוּיָה / צִדְקָתְךָ בְּאֶרֶץ נְשִׁיָּה</p></blockquote>
<blockquote><p>         אֲמָתְךָ פְּדֵה מִשַּׁחַת / כִּי לְךָ מִתְוַכַּחַת / אָנֹכִי בוֹרַחַת</p> <p>         וּבְכֵן מָה רַבּוּ מַעֲשֶׂיךָ יְיָ.</p></blockquote>
