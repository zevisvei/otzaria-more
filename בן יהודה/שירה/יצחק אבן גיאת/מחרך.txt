<h1>מחרך</h1>
יצחק אבן גיאת
<blockquote><p>נֶגְדְּךָ / אָשִׂים מְגַמָּתִי,</p></blockquote>
<blockquote><p>כִּי לְךָ / רוּחִי וְנִשְׁמָתִי.</p></blockquote>
<br/>
<blockquote><p>יָקְרוּ / עָצְמוּ פְלָאֶיךָ,</p></blockquote>
<blockquote><p>חֻבְּרוּ / אֶל עַל קְרוּאֶיךָ,</p></blockquote>
<blockquote><p>בָּחֲרוּ / אוֹתְךָ בְרוּאֶיךָ,</p></blockquote>
<blockquote><p>אֶחֱזֶה בִרְאִי מְזִמָּתִי</p></blockquote>
<blockquote><p>כִּי שְׁמָךְ חֵילִי וְחוֹמָתִי</p></blockquote>
<br/>
<blockquote><p>צוֹרֲרִים / שׁוֹעִים לְתַעְתּוּעִים,</p></blockquote>
<blockquote><p>דּוֹבְרִים / כָּזָב וּבוֹ תוֹעִים</p></blockquote>
<blockquote><p>אוֹמְרִים / “שָׁוְא עַמְּךָ רוֹעִים!”</p></blockquote>
<blockquote><p>וַאֲנִי – שִׁבְתִּי וְקִימָתִי</p></blockquote>
<blockquote><p>זִכְרְךָ עָזִּי וְזִמְרָתִי!</p></blockquote>
<br/>
<blockquote><p>חוֹלֲלִי, / אֶעְטֹף לְפָנֶיךָ,</p></blockquote>
<blockquote><p>מַעֲלִי / יִסְלַח רְצוֹנֶךָ,</p></blockquote>
<blockquote><p>מַחֲלִי / תִרְאֶה בְעֵינֶיךָ</p></blockquote>
<blockquote><p>כִּי בְיוֹם שׁוּבִי לְאַדְמָתִי</p></blockquote>
<blockquote><p>אֶקְרְאָה אָחוֹת לְרִמָּתִי.</p></blockquote>
<br/>
<blockquote><p>קוֹמֲמָה, / יָהּ, בֵּיתְךָ! מֵעִיר</p></blockquote>
<blockquote><p>שׁוֹמֲמָה / חֶזְיוֹנְךָ תָעִיר,</p></blockquote>
<blockquote><p>רוֹמֲמָה / וּשְפֹט בְּהַר שֵׂעִיר,</p></blockquote>
<blockquote><p>עוֹרֲרָה וּנְקֹם לְנִקְמָתִי –</p></blockquote>
<blockquote><p>תֵּן פְּאֵר תַּחַת כְּלִמָּתִי!</p></blockquote>
