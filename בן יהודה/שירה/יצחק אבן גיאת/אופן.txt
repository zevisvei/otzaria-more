<h1>אופן</h1>
יצחק אבן גיאת
<blockquote><p>יָקְרוּ לְהַלֵּל, / יָהּ, מַהֲלָלֶיךָ</p></blockquote>
<blockquote><p>וְאֵין מִי יְמַלֵּל / עֹז מִפְעָלֶיךָ</p></blockquote>
<blockquote><p>וִיחַו יוֹם וְלֵיל / מִפְלְאוֹת שִׂכְלֶךָ –</p></blockquote>
<blockquote><p>כִּי אַתְּ מְחוֹלֵל / כֹּל בְּנִיב מִלֶּיךָ.</p></blockquote>
<blockquote><p>רַבּוֹת עָשִׂיתָ / בְּעֹצֶם גָּדְלֶךָ,</p></blockquote>
<blockquote><p>אֲדֹנָי אֱלֹהַי, אֵין עֲרֹךְ אֵלֶיךָ!</p></blockquote>
<br/>
<blockquote><p>צוֹפֶה לְכֹל – אַךְ / בְּעַיִן אֵין צוֹפֶה,</p></blockquote>
<blockquote><p>וּפוֹעֵל בְּלִי יָד / וּמְדַבֵּר וְאֵין פֶּה,</p></blockquote>
<blockquote><p>וְחַי לֹא כְּחַיֵּי / אֱנוֹשׁ לְמָחָר נִסְפֶּה,</p></blockquote>
<blockquote><p>וּמֵאָז נוֹדַע / בְּשֵׁם מוֹחֵץ וְרוֹפֵא,</p></blockquote>
<blockquote><p>וְעוֹשֶׂה טוֹב וְרַע – / אַךְ שְׁנֵיהֶם יָפֶה,</p></blockquote>
<blockquote><p>וְנִצְפֶּה בְשֵׂכֶל / וְלֹא בְעַיִן צוֹפֶה –</p></blockquote>
<blockquote><p>כִּי מִפְעָלֶיךָ / יְעִידוּן עָלֶיךָ.</p></blockquote>
<br/>
<blockquote><p>חָכָם בְּחָכְמָה / נִפְלֵאת נִפְלֵאתָ</p></blockquote>
<blockquote><p>וְדַעַת פְּלִיאָה / מִנִּי נִפְלֵיתָ</p></blockquote>
<blockquote><p>וּמִבּוֹא בְמֶחְקַר / חֲכָמִים נַעֲלֵיתָ.</p></blockquote>
<blockquote><p>מִי זֶה יְמַלֵּא / אֲשֶׁר לֹא מִלֵּאתָ</p></blockquote>
<blockquote><p>וְיָחֵל וִיכַלֶּה / אֲשֶׁר לֹא כִלִּיתָ?</p></blockquote>
<blockquote><p>נִכְסֶה – וְאוּלָם / בְּפִלְאֲךָ נִגְלֵיתָ,</p></blockquote>
<blockquote><p>וְרָם בְּאֵין מָקוֹם / וְעוֹלָם מָלֵאתָ,</p></blockquote>
<blockquote><p>אַתָּה תִּסְבְּלֵהוּ – / וְהוּא לֹא יִסְבְּלֶךָ!</p></blockquote>
<br/>
<blockquote><p>קָדוֹשׁ מְקֹרָא / וּמְשָׁרְתָיו קְדוֹשִים,</p></blockquote>
<blockquote><p>טְהוֹרֵי עֶצֶם / כְּעֶצֶם תַּרְשִׁישִׁים,</p></blockquote>
<blockquote><p>פְּלִיאֵי צוּרָה / וְלֹא תַבְנִית אִישִׁים,</p></blockquote>
<blockquote><p>לְבִלְתִּי שְׁלָט-בָּם / יְסוֹד חֲמִשָּׁה רְגָשִׁים</p></blockquote>
<blockquote><p>וּמַחֲנוֹת נִפְרָשִׁים / לְאַרְבַּעַת רָאשִׁים</p></blockquote>
<blockquote><p>לְהַלֵּל נִגָּשִׁים / לְקֹדֶשׁ הַקֳּדָשִׁים</p></blockquote>
<blockquote><p>וְיִהְיוּ מַקְדִּישִׁים / בְּתוֹךְ הֵיכָלֶךָ.</p></blockquote>
<br/>
