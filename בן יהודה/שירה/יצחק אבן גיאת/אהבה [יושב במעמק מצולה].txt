<h1>אהבה [יושב במעמק מצולה]</h1>
יצחק אבן גיאת
<blockquote><p>יוֹשֵׁב בְּמַעֲמַק / מְצוּלָה, וּמָעֳמָד</p></blockquote>
<blockquote><p>אַיִן, וְחָמַק / דּוֹדוֹ וּמַחְמַד</p></blockquote>
<blockquote><p>עַיִן נָמַק, / יוֹם יוֹם מַלְמָד</p></blockquote>
<blockquote><p>יְשַׂדֵּד לְתֶלֶם / יוֹנַת אֵלֶם,</p></blockquote>
<blockquote><p>וּפַעַם הוֹלֵם / בְּגֹדֶל וְגַאֲוָה.</p></blockquote>
<br/>
<blockquote><p>"צֶדֶק, יְדִידִי, / וְאַהֲבַת כְּלוּלִים</p></blockquote>
<blockquote><p>אֶזְכֹּר בְּחַסְדִּי, / לֶאֱחֹז מְחַבְּלִים</p></blockquote>
<blockquote><p>כְּרָמִים, וְיָדִי / תְרוֹמֵם שְׁפָלִים.</p></blockquote>
<blockquote><p>אֶפְרֹשׂ שְׁלוֹמִי / אֶל בַּת עַמִּי –</p></blockquote>
<blockquote><p>אַךְ אִישׁ חֶרְמִי / לַמַּעֲצֵבָה".</p></blockquote>
<br/>
<blockquote><p>"חֶלְקִי וְצוּרִי / עֲדֵי מָה יְלוֹנֵן</p></blockquote>
<blockquote><p>אוֹיְבִי וְצָרִי / בְּבֵיתְךָ? חוֹנֵן</p></blockquote>
<blockquote><p>אֵין לִי, וְאוֹרִי / הֶחְשִׁיךְ מְעוֹנֵן,</p></blockquote>
<blockquote><p>וְעַל בַּת נְגִידִים / יָצְאוּ גְדוּדִים</p></blockquote>
<blockquote><p>וְעֶבֶד עֲבָדִים / בְּחֶרֶב לְהוּבָה".</p></blockquote>
<br/>
<blockquote><p>"קוּמִי וְשִׁכְנִי / אֶל בֵּית מְנוּחֵךְ!</p></blockquote>
<blockquote><p>אֶשְׁכַּח יְמִינִי / אִם אֶשְׁכָּחֵךְ,</p></blockquote>
<blockquote><p>וְהִנֵּה לְשׁוֹנִי / תִּדְבַּק אֶל חֵךְ –</p></blockquote>
<blockquote><p>אִם לֹא אֶזְכְּרֵךְ / וְאֶחֱבֹשׁ לְשִׁבְרֵךְ</p></blockquote>
<blockquote><p>וְנֵזֶר פְּאֵרֵךְ / תַּחְבְּשִׁי, עֲלוּבָה!" <br/></p></blockquote>
