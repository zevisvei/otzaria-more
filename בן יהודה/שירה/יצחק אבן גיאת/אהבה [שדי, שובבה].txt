<h1>אהבה [שדי, שובבה]</h1>
יצחק אבן גיאת
<blockquote><p>שַׁדַּי, שׁוֹבֲבָה / חוֹלַת אַהֲבָה,</p> <p>הוֹמָה לָרְגָעִים / עַל דָּת הָכְזְבָה.</p> <br/></blockquote>
<blockquote><p>יוֹם רָחַק יְדִידָהּ / נַפְשָׁהּ מֵאֲנָה</p> <p>הִנָּחֵם, וְיָדָהּ / עַל רֹאשׁ נָתְנָה</p> <p>[5] וּבְיַד מַעֲבִידָהּ / דּוּמָה שָׁכְנָה.</p> <p>עַל כִּי שׁוֹבֲבָה / עֵינִי דָאֲבָה</p> <p>וּבְצוּרֵי רְגָעִים / בֵּיתָהּ חָצְבָה.</p> <br/></blockquote>
<blockquote><p>צוֹעֶקֶת וְהוֹמָה / מִיַּד לוֹחֲצִים,</p> <p>כִּי מִשְׁמָע וְדוּמָה / אוֹתָהּ קוֹלְסִים,</p> <p>[10] גַּם גָּדֵר וְחוֹמָה / יַחַד פּוֹרְצִים.</p> <p>אֵיךְ לֹא עֻזְּבָה / פִּתְאֹם עֻזְּבָה,</p> <p>בֵּין שִנֵּי רְשָׁעִים / לוֹהְטִים שָׁכְבָה.</p> <br/></blockquote>
<blockquote><p>חֶזְיוֹנֵי כְתָבָהּ / הֵן מַה נֶּחְתְּמוּ!</p> <p>רַעְיוֹנֵי לְבָבָהּ / לִמְאֹד נִפְעֲמוּ.</p> <p>[15] כָּל יוֹשְׁבֵי סְבִיבָהּ / בָּהּ יִלָּחֲמוּ,</p> <p>עַל כִּי אוֹהֲבָהּ / פִּתְאֹם עָזְבָה</p> <p>בֵּין זֶרַע מְרֵעִים / בָּדָד יָשְׁבָה.</p> <br/></blockquote>
<blockquote><p>"קוֹרְאָה מִמְּצָרִים, / לָמָּה תִירְאִי?</p> <p>עַל כַּנְפֵי נְשָׁרִים / עוֹד תִּנָּשְׁאִי!</p> <p>[20] הֵן אֵשֶׁת נְעוּרִים / לִי עוֹד תִּקְרְאִי,</p> <p>רַעְיָה נֶעְצְבָה – / אוֹתָהּ אֹהֲבָה,</p> <p>כִּי עַל כָּל פְּשָׁעִים / תִּכְסֶה אַהֲבָה!"</p></blockquote>
