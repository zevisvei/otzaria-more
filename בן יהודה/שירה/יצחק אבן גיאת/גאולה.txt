<h1>גאולה</h1>
יצחק אבן גיאת
<blockquote><p>אֶסְבְּלָה נְדוּדִי – / אָגִילָה בְגָלוּתִי,</p></blockquote>
<blockquote><p>אֶעֱבֹד מַעְבִידִי – / אוֹחִילָה בְמַחְלָתִי.</p></blockquote>
<br/>
<blockquote><p>יֶעֱרַב לִי שִׁבְיִי, / לֹא אָקוּץ בְּמוּסָרִי,</p></blockquote>
<blockquote><p>אֶשְּׂאָה אֶת חָלְיִי, / לֹא אֶמְאַס בְּמוּסָרִי,</p></blockquote>
<blockquote><p>אֶתְּנָה עַל לֶחְיִי / עֹל צָרִי וּמוֹסֵרִי –</p></blockquote>
<blockquote><p>אֶעֱלֹז בְּאֵידִי, / אָשִׂישָׂה בְדַלּוּתִי,</p></blockquote>
<blockquote><p>מַחֲלִי וּמְרוּדִי – / הֵמָּה לִי תְהִלָּתִי!</p></blockquote>
<br/>
<blockquote><p>צוֹרֲרַי עַל דָּתְךָ / שִׂנְאַת שָׁוְא שְׂנֵאוּנִי,</p></blockquote>
<blockquote><p>שִׁקְּרוּ בִּבְרִיתְךָ / וּבְלֹא אֵל קִנְאוּנִי,</p></blockquote>
<blockquote><p>לַעֲבֹד זוּלָתְךָ / כִּמְעַט הֶחֱטִיאוּנִי.</p></blockquote>
<blockquote><p>וַאֲנִי בְעוֹדִי – / שַׂמְתִּיךָ אֱיָלוּתִי,</p></blockquote>
<blockquote><p>אֶעֱמֹד עַל עָמְדִי, / לֹא אַט מִמְּסִלָּתִי.</p></blockquote>
<blockquote><p>חָדְלוּ רַעְיוֹנַי / לַחְקֹר קֵץ מְצוּקָתִי,</p></blockquote>
<blockquote><p>נִסְתְּמוּ חֶזְיוֹנַי / אֵין גּוֹלֶה נְבוּאָתִי,</p></blockquote>
<blockquote><p>נֶעְלְמוּ מֵעֵינַי / עִתּוֹתֵי יְשׁוּעָתִי.</p></blockquote>
<blockquote><p>אוֹיְבַי לְנֶגְדִּי / בִּגְבוּלִי וְנַחְלָתִי,</p></blockquote>
<blockquote><p>הֵם לְשִׂכִּים צִדִּי / עֵת שִׁבְרִי וְנַחְלָתִי.</p></blockquote>
<br/>
<blockquote><p>"קוֹל מְבַשֵּׂר פִּדְיוֹן / אַשְׁמִיעַ וְאוֹשִׁיעַ</p></blockquote>
<blockquote><p>אוֹהֲבִי, וּלְצִיּוֹן / אָשׁוּבָה וְאוֹפִיעַ.</p></blockquote>
<blockquote><p>אֶתְּנֵהוּ עֶלְיוֹן / עַל כָּל עָם וְיַרְגִּיעַ</p></blockquote>
<blockquote><p>יֶחֱזֶה בְהוֹדִי / וִיקָר בֵּית תְּפִלָּתִי,</p></blockquote>
<blockquote><p>אֶזְכְּרָה לוֹ חַסְדִּי, / אֶגְאַל אֶת גְּאֻלָּתִי!"</p></blockquote>
<br/>
