<h1>אהבה [שדי שובבה חולת אהבה]</h1>
יצחק אבן גיאת
<blockquote><p>שַׁדַּי, שׁוֹבֲבָה / חוֹלַת אַהֲבָה,</p></blockquote>
<blockquote><p>הוֹמָה לָרְגָעִים / עַל דָּת הָכְזְבָה.</p></blockquote>
<br/>
<blockquote><p>יוֹם רָחַק יְדִידָהּ / נַפְשָׁהּ מֵאֲנָה</p></blockquote>
<blockquote><p>הִנָּחֵם, וְיָדָהּ / עַל רֹאשׁ נָתְנָה</p></blockquote>
<blockquote><p>וּבְיַד מַעֲבִידָהּ / דּוּמָה שָׁכְנָה.</p></blockquote>
<blockquote><p>עַל כִּי שׁוֹבֲבָה / עֵינִי דָאֲבָה</p></blockquote>
<blockquote><p>וּבְצוּרֵי רְגָעִים / בֵּיתָהּ חָצְבָה.</p></blockquote>
<br/>
<blockquote><p>צוֹעֶקֶת וְהוֹמָה / מִיַּד לוֹחֲצִים,</p></blockquote>
<blockquote><p>כִּי מִשְׁמָע וְדוּמָה / אוֹתָהּ קוֹלְסִים,</p></blockquote>
<blockquote><p>גַּם גָּדֵר וְחוֹמָה / יַחַד פּוֹרְצִים.</p></blockquote>
<blockquote><p>אֵיךְ לֹא עֻזְּבָה / פִּתְאֹם עֻזְּבָה,</p></blockquote>
<blockquote><p>בֵּין שִנֵּי רְשָׁעִים / לוֹהְטִים שָׁכְבָה.</p></blockquote>
<br/>
<blockquote><p>חֶזְיוֹנֵי כְתָבָהּ / הֵן מַה נֶּחְתְּמוּ!</p></blockquote>
<blockquote><p>רַעְיוֹנֵי לְבָבָהּ / לִמְאֹד נִפְעֲמוּ.</p></blockquote>
<blockquote><p>כָּל יוֹשְׁבֵי סְבִיבָהּ / בָּהּ יִלָּחֲמוּ,</p></blockquote>
<blockquote><p>עַל כִּי אוֹהֲבָהּ / פִּתְאֹם עָזְבָה</p></blockquote>
<blockquote><p>בֵּין זֶרַע מְרֵעִים / בָּדָד יָשְׁבָה.</p></blockquote>
<br/>
<blockquote><p>"קוֹרְאָה מִמְּצָרִים, / לָמָּה תִירְאִי?</p></blockquote>
<blockquote><p>עַל כַּנְפֵי נְשָׁרִים / עוֹד תִּנָּשְׁאִי!</p></blockquote>
<blockquote><p>הֵן אֵשֶׁת נְעוּרִים / לִי עוֹד תִּקְרְאִי,</p></blockquote>
<blockquote><p>רַעְיָה נֶעְצְבָה – / אוֹתָהּ אֹהֲבָה,</p></blockquote>
<blockquote><p>כִּי עַל כָּל פְּשָׁעִים / תִּכְסֶה אַהֲבָה!"</p></blockquote>
