<h1>אֶפְתַּח פֶּה בְרֶנֶן</h1>
יוסי בן יוסי
<blockquote><p>אֶפְתַּח פֶּה בְרֶנֶן / אֲהַלֵּל לְשָׂם פֶּה</p> <p>בְּשִׂפְתֵי רְנָנוֹת / גֹּדֶל אֶתֵּן לַמֶּלֶךְ.</p></blockquote>
<br/>
<blockquote><p>גָּדְלוֹ אֲשַׁנֵּן / בְּקָהָל וְעֵדָה</p> <p>דְּבָרָיו אֱמוּנָה / וְהוּא לְעוֹלָם מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>הֲלֹא הוּא נוֹתֵן / מַעֲנֵה לָשׁוֹן</p> <p>וּלְכָל אֲשֶׁר יַחְפֹּץ / יַטֶּה לֵב, מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>זִכְרוֹ לְדוֹר דּוֹר / שְׁנוֹתָיו לְאֹרֶךְ יָמִים</p> <p>חַי הוּא לְעוֹלָם / וְאֵין כְּנֶגְדּוֹ מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>טַעַם אַמְתִּיק לוֹ / בְּחֵיךְ וְגָרוֹן</p> <p>יֶהְגֶּה לְשׁוֹנִי צֶדֶק / תְּהִלּוֹת מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>כָּל עַצְמוֹתַי / וְחַדְרֵי קְרָבַי</p> <p>לְשֵׁם קָדוֹשׁ אֲהַלֵּל / כִּי הוּא לְעוֹלָם מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>מַעֲנֶה אֶשְׁאֲלָה מֶנּוּ / וְלֹא מֵאֵל אַחֵר</p> <p>נַפְשִׁי יְשׁוֹבֵב / בְּנַחַת לְגַדֵּל מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>שְׂפָתַי תַּבַּעְנָה / חֵן וְתַחֲנוּנִים</p> <p>עֲנוֹת סִיחַ בְּפִי / בְּסוֹד נֹעַם מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>פָּעֳלוֹ אֲשַׁנֵּן / בְּפֶה וּבְלָשׁוֹן</p> <p>צִדְקוֹ אַגִּיד / כִּי הוּא לְבַדּוֹ מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>קְרָבַי וְכִלְיוֹתַי / זֶמֶר יַבִּיעוּ</p> <p>רֶנֶן יְתַנּוּ / כִּי אֵין זוּלָתוֹ מֶלֶךְ.</p></blockquote>
<br/>
<blockquote><p>שִׁיר חָדָשׁ אָשִׁיר / לְיוֹצֵר בְּרֵאשִׁית</p> <p>תְּהִלָּה אַמְתִּיק לוֹ / כִּי מִמֶּנּוּ מַעֲנֵה לָשׁוֹן.</p></blockquote>
