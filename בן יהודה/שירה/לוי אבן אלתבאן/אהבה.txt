<h1>אהבה</h1>
לוי אבן אלתבאן
<blockquote><p>"לְחוּצָה בְּאֹרֶךְ / גָּלוּת מְשֻׂכֶּלֶת,</p> <p>מְעֻנָּה בְפֶרֶךְ / מֹרְאָה וְנִגְאֶלֶת,</p> <p>לְהֵיכַל דְּבִירֵךְ / אֲשִׁיבֵךְ לְמֶמְשֶׁלֶת</p> <p>וְאֶגְדֹּר פִּרְצֵךְ / וְאָשִׂים לְמַחֲצֵךְ</p> <p>אֲרוּכָה, וְאֶרְצֵךְ / וְאֹהֲבֵךְ נְדָבָה".</p> <p>"וְעַד אָן יְשׁוּעָה / אֶהְיֶה מְחַכָּה –</p> <p>וְאֲנִי פְּרוּעָה / מִצֵּל וְסֻכָּה?</p> <p>כַּשֶּׂה תוֹעָה, / כְּרָחֵל מְבַכָּה,</p> <p>עָבַר קְצִירִי / וְכָלָה בְצִירִי</p> <p>וְנוֹסַף צִירִי – / וְצִירִי לֹא בָא!"</p></blockquote>
<blockquote><br/> <p>"יְחִידָה, לִבְשִׁי / בִגְדֵי תִפְאֶרֶת</p> <p>וְנֵזֶר חִבְשִׁי / לְכָבוֹד וַעֲטֶרֶת</p> <p>כִּי עוֹד תִּנְשִׁי / בשֶׁת, וּמַשְׂכֹּרֶת</p> <p>תִּקְחִי, וְיֵחַת / זֶרַח וְנַחַת,</p> <p>תִּשְׂמְחִי תַּחַת / הֱיוֹתֵךְ עֲזוּבָה".</p> <br/> <p>"חָלְפוּ זְמַנַּי / בְּיָגוֹן וּמְהוּמָה</p> <p>וּפָנַי – עֲוֹנַי / כִּסּוּ בִכְלִמָּה,</p> <p>וּמִכָּל בָּנַי / נוֹתַרְתִּי עֲרֻמָּה.</p> <p>לְרִיק אֶעֱמֹל, / אִם לֹא תִגְמֹל</p> <p>בְּחַסְדְּךָ, וְתַחְמֹל – / וְאָנָה אֲנִי בָא?"</p></blockquote>
<blockquote><p>"קוֹמֵם אֲקוֹמֵם / דְּבִירֵךְ הַנֶּחֱרַָב</p> <p>וְקַרְנֵךְ אֲרוֹמֵם / בְּקִרְיַת מֶלֶךְ רַב</p> <p>וּלְדַל מְשׁוֹמֵם / יְשׁוּעָתִי תִקְרַב</p> <p>וְאַשְׁמִיד רְשָׁעִים / זֶרַע מְרֵעִים –</p> <p>וְעַל כָּל פְּשָׁעִים / תְּכַסֶּה אַהֲבָה".</p></blockquote>
