<h1>[תשובה למשה אבן-עזרא]</h1>
לוי אבן אלתבאן
<blockquote><p>מָה אָשִׁיב עַל / תּוֹכַחַת שַׂר, / עַל מִצְחִי תַּו / עָוֹן הִתְוָה?</p> <p>אִם בִּמְפֹרָד / נַפְשִׁי תֶחְרַד – / אֵיךְ בֶּחָרוּז / יֶשׁ לִי תִקְוָה?</p> <p>וּבְמַעְלוֹת אִם / אֶעְלֶה אֶל שִׁיר / הַמַּעְלוֹת, לִי / תִּגָּל עֶרְוָה.</p> <p>שִׁיר הַשִּׁירִים – / כָּל מִדְבָּרִים / הִשְׁפִּילּו וַ- / יֹּאמֶר “גֵּוָה!”</p> <p>מִי אֶל זוֹלֵל / יַשְׁוֶה יָקָר / אוֹ אִישׁ יָשָׁר / לִבְנֵי עַלְוָה?</p> <p>אֵיכָה יִתְחַר / אֶת הַסּוּסִים / מִי עִם רַגְלִים / אֵין לוֹ שַׁלְוָה?</p> <p>קָצְרָה יָדִי / וּלְשׁוֹנִי מִ- / דַּבֵּר צַחוֹת / עַוָּה עַוָּה –</p> <p>פְּנֵי מַלְוֶה / כָּל שַׂר מֵאוֹר / חָכְמָתוֹ, אַךְ / הוּא לֹא לָוָה!</p> <p>תּוֹדֶה נַפְשִׁי / וּבְמִסְתָּרִים / תִּבְכֶּה נַפְשִׁי / מֵחֵטְא גֵּוָה.</p> <p>אוּלַי יִתֵּן / לִי אוֹת לַחְיוֹת / נַפְשִׁי, וִיצַו / לִקְשֹׁר תִּקְוָה.</p></blockquote>
