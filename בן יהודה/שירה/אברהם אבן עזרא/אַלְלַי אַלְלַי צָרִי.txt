<h1>אַלְלַי אַלְלַי צָרִי</h1>
אברהם אבן עזרא
<p>קינה לט׳ באב</p>
<blockquote><p>אַלְלַי אַלְלַי, / צָרִי / סָמַךְ עָלַי!</p></blockquote>
<blockquote><p>אַלְלַי אַלְלַי, / צוּרִי / סָר מֵעָלַי!</p> <br/> <p>אָמַר אוֹיֵב לְהָתֵל בִּי: / ׳עַם יְיָ אֵלֶּה –</p></blockquote>
<blockquote><p>הֲלֹא פִזַּרְתִּים בְּחַרְבִּי, / לֹא אֲדֹנִים לָאֵלֶּה!׳</p></blockquote>
<blockquote><p>אֵיךְ יַעְמֹד לִבִּי / וְתִקְרֶאנָה אֹתִי כָּאֵלֶּה?</p></blockquote>
<blockquote><p>הֲלֹא עַל כִּי אֵין אֱלֹהַי בְּקִרְבִּי / מְצָאוּנִי הָרָעוֹת הָאֵלֶּה.</p> <br/> <p>בָּנֶיךָ, אֵיךְ תְּשִׁיתֵמוֹ / כִּכְלִי אֵין חֵפֶץ בּוֹ?</p></blockquote>
<blockquote><p>וּבַיִת בָּנוּ אֲבוֹתֵימוֹ, / אֱלֹהֵי יִשְׂרָאֵל בָּא בוֹ –</p></blockquote>
<blockquote><p>הֶחְרִיב צַר בְּיוֹם אֵידֵמוֹ, / שׁוּעָלִים הִלְּכוּ בוֹ,</p></blockquote>
<blockquote><p>וְאָמַר: ׳אֵי אֱלֹהֵימוֹ / צוּר חָסָיוּ בוֹ?׳</p> <br/> <p>רָדָה הַצַּר וְהֵנִיף חַרְבּוֹ / בְּהֵיכַל קָדְשׁוֹ –</p></blockquote>
<blockquote><p>וְשָׁב הַמֶּלֶךְ לִמְסִבּוֹ / בִּמְעוֹן קָדְשׁוֹ.</p></blockquote>
<blockquote><p>וְאֵיךְ נָכְרִי כָּל מוּם בּוֹ / יָקוּם בִּמְקוֹם קָדְשׁוֹ?</p></blockquote>
<blockquote><p>אַיֵֵּה הַשָּׂם בְּקִרְבּוֹ / אֶת רוּחַ קָדְשׁוֹ?</p> <br/> <p>מַה יַּעֲשׂוּ זְרוּיִם / בְּאֶרֶץ לֹא לָהֶם?</p></blockquote>
<blockquote><p>אוֹ מַה יַּעֲנוּ שְׁבוּיִם / לִפְנֵי שׁוֹבֵיהֶם –</p></blockquote>
<blockquote><p>וּלְחָרֵף אֱלֹהִים חַיִּים / שַׁתּוּ בַשָּׁמַיִם פִּיהֶם?</p></blockquote>
<blockquote><p>לָמָּה יֹאמְרוּ הַגּוֹיִם: / ׳אַיֵּה נָא אֱלֹהֵיהֶם?׳</p> <br/> <p>הַנּוֹתֵן בַּיָּם דָּרֶךְ וּבְמַיִם עַזִּים נְתִיבָה,</p></blockquote>
<blockquote><p>נְקֹם אֶת נִקְמָתֵנוּ וְאֶת יְרִיבֵנוּ רִיבָה,</p></blockquote>
<blockquote><p>וְאֶת אַחְרִיתֵנוּ מֵרֵאשִׁיתֵנוּ הֵיטִיבָה!</p></blockquote>
<blockquote><p>הֲשִׁיבֵנוּ יְיָ אֵלֶיךָ וְנָשׁוּבָה.</p></blockquote>
