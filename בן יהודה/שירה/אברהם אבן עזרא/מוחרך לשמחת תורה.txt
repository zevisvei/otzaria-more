<h1>מוחרך לשמחת תורה</h1>
אברהם אבן עזרא
<h2 id="מוחרךלשמחתתורהראברהםאבן-עזרא">מוחרך לשמחת תורה / ר' אברהם אבן-עזרא</h2>
<br/>
<blockquote><p>אֶשְׂמְחָה / כִּי אֶפְתְּחָה / פִי לְהוֹדוֹת,</p></blockquote>
<blockquote><p>אֶעֱנֶה / טוּב מַעֲנֶה / שִׁיר יְדִידוֹת.</p></blockquote>
<br/>
<blockquote><p>בָּא זְמַן / צִיר נֶאֱמָן / לֶאֱמוּנִים.</p></blockquote>
<blockquote><p>עַל שְׁמוֹ / דָּמוֹ כְּמוֹ / הָאֲבָנִים,</p></blockquote>
<blockquote><p>בַּצְּבִי / אוֹ בַּשְּׁבִי / לוֹ נְתוּנִים.</p></blockquote>
<blockquote><p>נִגְאֲלוּ / אִם קִבְּלוּ / הַתְּעוּדוֹת,</p></blockquote>
<blockquote><p>נִכְבְּשׁוּ / אִם נָטְשׁוּ / דָּת וְחִידוֹת.</p></blockquote>
<br/>
<blockquote><p>רַנְנִי, / קוֹלֵךְ תְּנִי, / בַּת סְגֻלָּה,</p></blockquote>
<blockquote><p>שָׁב גְּבִיר / אֶל הַדְּבִיר / בֵּית תְּפִלָּה,</p></blockquote>
<blockquote><p>יִדְּמוּ / גוֹי נִלְחֲמוּ / בַּתְּחִלָּה.</p></blockquote>
<blockquote><p>יִגְוְעוּ / עֵת יִשְׁמְעוּ / קוֹל צְעָדוֹת,</p></blockquote>
<blockquote><p>יִרְגְּזוּ, / בַּל יֶחֱזוּ / אֵשׁ פְּלָדוֹת.</p></blockquote>
<br/>
<blockquote><p>מַה לְּךָ, / נִרְדָּם? לְכָה / אֶל מְקַוֶּה</p></blockquote>
<blockquote><p>בּוֹאֲךָ, / עֵת יַרְאֲךָ / הוֹד בְּנָוֶה –</p></blockquote>
<blockquote><p>בִּלְתְּךָ / וַאְמָתְךָ / לֹא יְאַוֶּה –</p></blockquote>
<blockquote><p>לַחֲזוֹת / הַמַּחֲזוֹת / הָעֲתִידוֹת,</p></blockquote>
<blockquote><p>כִּי כְבָר / נִשְׁמָע דְּבַר / אִישׁ חֲמֻדוֹת.</p></blockquote>
