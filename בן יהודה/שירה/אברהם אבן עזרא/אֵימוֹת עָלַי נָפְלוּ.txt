<h1>אֵימוֹת עָלַי נָפְלוּ</h1>
אברהם אבן עזרא
<p>תוכחה ליום הכיפורים</p>
<blockquote><br/> <p>אֵימוֹת עָלַי נָפְלוּ / וּשְׂעִפַּי בִּי נִבְהֲלוּ,</p></blockquote>
<blockquote><p>בְּהַעֲלוֹתִי עַל לְבָבִי / דְּבַר יוֹנָה הַנָּבִיא.</p> <br/> <p>אֵין יֵשׁ כִּי אִם אִתְּךָ, / וְאֵין לִי מָנוֹס בִּלְתְּךָ,</p></blockquote>
<blockquote><p>וְעָדֶיךָ כָּל בָּשָׂר / יָבוֹא תַּחַת מוּסָר.</p> <br/> <p>בְּךָ מַצַּב עוֹלָמְךָ, / וְכָל שֵׁם נִמְצָא בְּשִׁמְךָ –</p></blockquote>
<blockquote><p>וּמִי יִסָּתֵר מִמְּךָ? עֹז וְתוּשִׁיָּה עִמְּךָ!</p> <br/> <p>רַמְתָּ מֵהַבִּיעַ, / וְאָן אֶסַּק וְאַצִּיעַ –</p></blockquote>
<blockquote><p>עֹז כְּבוֹדְךָ בַּמָּרוֹם / וּשְׁאוֹל נֶגְדְּךָ עָרוֹם?</p> <br/> <p>הֵן כָּאוֹרָה כַּחֲשֵׁכָה / לְךָ, וְהַחֹשֶׁךְ – אֵיכָה</p></blockquote>
<blockquote><p>אֹמַר כִּי יְשׁוּפֵנִי, / וְלַיְלָה אוֹר בַּעֲדֵנִי?</p> <br/> <p>מַחֲשַׁבְתִּי לוֹ נִתְכְּנָה, / וּבְאַחֲרִית יָם אֶשְׁכְּנָה –</p></blockquote>
<blockquote><p>הֲלֹא גַם שָׁם הִנֶּךָּ / וְתֹאחֲזֵנִי יְמִינֶךָ.</p> <br/> <p>עָלֶיךָ אֶשָּׁעֵן, / וְאֵלֶיךָ עֵינִי כְּעֵין</p></blockquote>
<blockquote><p>עֶבֶד אֶל יַד אֲדוֹנָיו, / לָשֵׂאת אֵלָיו פָּנָיו.</p> <br/> <p>זְכֹר, אֲנִי – אִם מוֹבָאִי / לְאַיִן, כְּמוֹ מוֹצָאִי</p></blockquote>
<blockquote><p>מֶנְהוּ, אֵי חֲסָדֶיךָ? / הֲשָׁוְא מַעֲשֵׂה יָדֶיךָ?</p> <br/> <p>רַבִּים הֵם אֲשָׁמַי – / סְלַח נָא, כִּי מְעַט יָמָי;</p></blockquote>
<blockquote><p>כְּחָמְלָךְ עַל פָּעֳלֶךָ / שׁוֹב אָשׁוּב אֵלֶיךָ.</p> <br/> <p>אֶבְטַח עַל רַחֲמֶיךָ / וְסָר מֶנִּי זַעְמֶךָ</p></blockquote>
<blockquote><p>וּמִפִּשְׁעִי לֹא אִירָא, / כִּי שֵׁם יְיָ אֶקְרָא.</p></blockquote>
