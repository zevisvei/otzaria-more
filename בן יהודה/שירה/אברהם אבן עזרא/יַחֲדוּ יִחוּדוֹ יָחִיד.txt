<h1>יַחֲדוּ יִחוּדוֹ יָחִיד</h1>
אברהם אבן עזרא
<blockquote><p>יַחֲדוּ יִחוּדוֹ, / יָחִיד וְאֵין בִּלְעָדוֹ,</p> <p>קַדְמוֹן וְאֵין נֶגְדּוֹ, / בָּרוּךְ שְׁמוֹ וּכְבוֹדוֹ!</p> <br/> <p>אֶפֶס לְיֵשׁ הוֹדַעְתָּ / וּשְׁמֵי מְרוֹם רָקַעְתָּ,</p> <p>נָהָר וְיָם בָּקַעְתָּ, / מִשְׁפָּט וְדִין הִשְׁמָעְתָּ;</p> <p>וְנָהַר מֵעוֹדוֹ / יִתְבַּרַךְ מֵהוֹדוֹ,</p> <p>מַלְאֲכֵי צָבָא יָעִידוּ / כִּי הֵם מְלִיצֵי חֶלְדּוֹ.</p> <br/> <p>בְּאַהְבָה לְעַמּוֹ נִמְצָא, / בְּחָנֵס כְּגִבּוֹר יָצָא</p> <p>לַיְלָה, וְלִשְׁמוֹ חָצָה, / כִּי הוּא גְדֹל הָעֵצָה.</p> <p>שָׁלַח משֶׁה עַבְדּוֹ, / אַהֲרֹן אִישׁ חֲסִידוֹ,</p> <p>לַעֲלוֹת תַּנִּין בְּאֵידוֹ – / לָהֶם שׁוּעַ בְּפִידוֹ.</p> <br/> <p>רֹאשִׁי עַל אוֹיְבַי תָּרִים, / יִחְיוּ שְׁאָר נִקְבָּרִים,</p> <p>תַּהְדֹּף בְּעֻזָּךְ צָרִים / לַיְלָה, כְּלֵיל שִׁמּוּרִים,</p> <p>וּזְכֹר שָׁלַח יָדוֹ / לִשְׁחֹט בְּנוֹ יְחִידוֹ,</p> <p>כַּעֲקֹד אַיִל עֲקָדוֹ, / בְּלִבּוֹ וּבְכָל מְאֹדוֹ.</p> <br/> <p>מֵטִיב וְגָם מַשְׁגִּיחַ / יִשְׁלַח, כְּאוֹר זוֹרֵחַ,</p> <p>תִּשְׁבִּי וְגַם מָשִׁיחַ, / שָׁלוֹם בְּלִי יָרֵחַ,</p> <p>יוֹֹסִיף שֵׁנִית יָדוֹ / לִקְנוֹת שְׁאָר יְדִידוֹ</p> <p>וּלְבוּשׁ נָקָם הוּא מַדּוֹ – / מִי הוּא זֶה בָּא בְסוֹדוֹ?</p></blockquote>
