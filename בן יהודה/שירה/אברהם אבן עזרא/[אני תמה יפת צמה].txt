<h1>[אני תמה יפת צמה]</h1>
אברהם אבן עזרא
<p>אהבה</p>
<br/>
<blockquote><p>אֲנִי תַמָּה / יְפַת צַמָּה – עֲדֶן שָׂמָה / בְּיַד שַׁמָּה, / וְהַחַמָּה</p></blockquote>
<blockquote><p>שְׁזָפַתְנִי, / שְׁאָפַתְנִי, / שְׂרָפַתְנִי; / זְכֹר, דּוֹדִי, בְּרִיתֶךָ!</p></blockquote>
<blockquote><p>נִפְלְאַתָּה לִי אַהֲבָתֶךָ. <br/></p></blockquote>
<blockquote><p>בְּבוֹר שִׁבְיִי, / בְּכוּר עָנְיִי / בְּרֹב מִרְיִי, / עֲדֶן בִּכְיִי / עֲלֵי לֶחְיִי,</p></blockquote>
<blockquote><p>וְאַתְּ עֶזְרִי, / בְּךָ שִׂבְרִי, / חֲבֹשׁ שִׁבְרִי / וְכָל שֶׁבֶר עֲדָתֶךָ! <br/></p></blockquote>
<blockquote><p>רְאֵה לַחְצִי, / רְפָא מַחְצִי, / גְּדֹר פִּרְצִי / וּמִיָּם צִי / לְאֹור תּוֹצִיא,</p></blockquote>
<blockquote><p>תְּקִיֵמִני, / תְּרִימֵנִי, / תְּשִׂימֵנִי / לְרֹאשׁ כִּדְבַר שְׂפָתֶיךָ. <br/></p></blockquote>
<blockquote><p>הֱיֵה סַלְעִי, / בְּיוֹם צַלְעִי / קְרָא יִשְׁעִי / וְסָר נִגְעִי, / וּמַכְנִיעִי</p></blockquote>
<blockquote><p>בְּאַף תַּהְרֹס / כְּגוֹי פַּתְרוֹס, / אֲזַי תִּפְרֹשׂ / כְּנָפְךָ עַל אֲמָתֶךָ. <br/></p></blockquote>
<blockquote><p>מְנַת חֶלְקִי / וְצוּר חִזְקִי / וְצוּף מָתְקִי / וְכָל חִשְׁקִי, / בְּךָ דָּבְקִי,</p></blockquote>
<blockquote><p>חֲצוֹצְרוֹתַי, / נְגִינוֹתַי / וְשִׁירוֹתַי / אֲנַגֵּן עוֹד בְּבֵיתֶךָ. <br/></p></blockquote>
