<h1>[אני גולה וסורה]</h1>
אברהם אבן עזרא
<p>מאורה</p>
<br/>
<blockquote><p>אֲנִי גוֹלָה וְסוּרָה / בֵּינוֹת לְבָאִים פְּזוּרָה,</p></blockquote>
<blockquote><p>וְהֵם שׁוֹקְטִים וְנָחִים / בְּמִשְׁכְּנוֹת מִבְטַחִים.</p></blockquote>
<br/>
<blockquote><p>בַּעְלוֹת עַל לְבָבִי / אוֹמֵר: יְמִין אַל תִּקְצַר,</p></blockquote>
<blockquote><p>אֵיךְ הָיָה כְאֵבִי / נֶצַח וְאֵיךְ צַעֲדִי צַר!</p></blockquote>
<blockquote><p>יוֹם כֹּהֵן וְנָבִיא / הָלְכוּ שְׁבִי לִפְנֵי צָר,</p></blockquote>
<blockquote><p>וּבְנֵי הַשְּׂרָרָה, / קֶדֶם בְּרֹאשָׁם עֲטָרָה,</p></blockquote>
<blockquote><p>חָזְרוּ עַל פְּתָחִים / לוֹבְשֵׁי בְּלוֹיֵי מְלָחִים.</p></blockquote>
<br/>
<blockquote><p>רָמָה יַד מְגָאֵל – / וַאְנִי כְשֶׂה נֶאֱלָמָה,</p></blockquote>
<blockquote><p>יוֹם בָּא עִיר אֲרִיאֵל / וַיַּעֲשֶׂה בִּי נְקָמָה,</p></blockquote>
<blockquote><p>וַיֹּאמֶר: הֲלֹא אֵל / קַנֹּא אֲדֹנָי, וְלָמָּה</p></blockquote>
<blockquote><p>לֹא קִנֵּא לְתוֹרָה? / יוֹם נִלְכְּדָה הַחֲבוּרָה</p></blockquote>
<blockquote><p>וְהֻכּוּ בָּרְמָחִים / גַּם נֻתְּחוּ לִנְתָחִים.</p></blockquote>
<br/>
<blockquote><p>הֶחְשָׁה אֵל אֱמוּנָה / עַד כֹּה – וְנָכְרִי בְּבֵיתוֹ,</p></blockquote>
<blockquote><p>וַיִּתֵּן לְמָנָה / לוֹ כָּל נְאוֹת אַהֲבָתוֹ,</p></blockquote>
<blockquote><p>עַד בִּמְקוֹם שְׁכִינָה / הֶעֱמִיד אֱוִיל תּוֹעֲבָתוֹ,</p></blockquote>
<blockquote><p>וּבְתוֹךְ הָעֲזָרָה / נָטַע עֲצֵי הָאֲשֵׁרָה –</p></blockquote>
<blockquote><p>שָׁם הָיוּ זְבוּחִים / הַכֹּהֲנִים הַמְּשׁוּחִים.</p></blockquote>
<br/>
<blockquote><p>מַה יַּעֲנוּ זְעוּמִים / כִּי נֶהְפְּכָה הַבְּרָכָה?</p></blockquote>
<blockquote><p>בֵּינוֹת כָּל לְאֻמִּים / עָלַי חֲמָתְךָ שְׁפוּכָה.</p></blockquote>
<blockquote><p>הַשְׁקֵף מִמְּרוֹמִים, / קַנֵּא לְיוֹשְׁבֵי חֲשֵׁכָה!</p></blockquote>
<blockquote><p>אוּלַי עֵת בְּשׂוֹרָה / בָּאָה, וְנֵצֵא לְאוֹרָהּ,</p></blockquote>
<blockquote><p>עַל לֵב נֶאֱנָחִים / דַּבֵּר – וְהָיוּ שְׂמֵחִים! <br/></p></blockquote>
