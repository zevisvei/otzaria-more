<h1>[מחשבות אל עמקות]</h1>
אברהם אבן עזרא
<blockquote><p>מַחְשְׁבוֹת אֵל עֲמֻקּוֹת, / מִפְעֲלוֹתָיו יְדוּעוֹת –</p> <p>זוֹרֵעַ צְדָקוֹת / מַצְמִיחַ יְשׁוּעוֹת.</p> <br/> <p>אֵל נוֹרָא תְהִלּוֹת / עַל כִּסְאוֹ בְשִׁבְתּוֹ</p> <p>לִמְשָׁרְתָיו מְסִלּוֹת / שָׂם לַעְשׂוֹת מְלָאכְתּוֹ,</p> <p>חֹק נָתַן לְמַלֹּאות / כָּל דָּבָר בְּעִתּוֹ.</p> <p>הוּא יַקְרִיב רְחוֹקוֹת, / הוּא יַרְחִיק נְגוּעוֹת,</p> <p>הוּא יַרְפֶּה חֲזָקוֹת, / הוּא יְרַפֵּא נְגוּעוֹת.</p> <br/> <p>בַּשַּׁחַק גְּבוּרוֹת / יָדוֹ הָעֲצוּמָה:</p> <p>מֶרְכַּבְתּוֹ בְּצוּרוֹת / טַבַּעְתּוֹ חֲתוּמָה,</p> <p>וּשְׁנֵי הַמְּאוֹרוֹת / הִמְשִׁיל עַל אֲדָמָה.</p> <p>הוּא יָעִיר לְהַשְׁקוֹת, / הוּא יִשְׁמֹר שְׁבוּעוֹת</p> <p>הַקָּצִיר, וְרַקּוֹת / תָּשֹׁבְנָה שְׂבֵעוֹת.</p> <br/> <p>רָץ חֵילוֹ לְפָנָיו / חוּגִים הֶעֱמִיד בָּם –</p> <p>עֵדָיו נֶאֱמָנָיו, / הָאָדָם בְּקִרְבָּם.</p> <p>הוּא נִשְׂגָּב, וְעֵינָיו / צוֹפוֹת מַה בְּלִבָּם,</p> <p>הוּא יִשְׁמַע צְעָקוֹת, / הוּא יֶאְסֹף דְּמָעוֹת,</p> <p>הוּא יַרְחִיק מְצוּקוֹת / בִּפְרֹעַ פְּרָעוֹת.</p> <br/> <p>מִי בָרָא אֲגֻדּוֹת / עַל אֶרֶץ יְסָדָם,</p> <p>שָׁת זִיקֹות לְיַדּוֹת / בַּמַּיִם יְסוֹדָם,</p> <p>וַעְגֻלּוֹת כְּבֵדוֹת / סָבִיב הֶעֱמִידָם?</p> <p>הֵם כִּרְאִי חֲזָקוֹת / הֵם הֵם הַזְּרוֹעוֹת,</p> <p>בָּם חַיּוֹת חֲקוּקוֹת / בָּם אוֹרוֹת זְרוּעוֹת.</p></blockquote>
