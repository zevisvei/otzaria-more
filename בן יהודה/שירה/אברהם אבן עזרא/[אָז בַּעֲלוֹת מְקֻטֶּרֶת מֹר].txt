<h1>[אָז בַּעֲלוֹת מְקֻטֶּרֶת מֹר]</h1>
אברהם אבן עזרא
<p>פדות</p>
<br/>
<blockquote><p>אָז בַּעֲלוֹת מְקֻטֶּרֶת מֹר / רֵיחָהּ לְמַעְלָה</p></blockquote>
<blockquote><p>הִתְרַפְּקָה בְּמֵיטַב טַעַם / עַל דּוֹד כְּיַעֲלָה.</p></blockquote>
<br/>
<blockquote><p>בַּלָּט צְבִי מְשָׁכָהּ בִּימֵי / נֹעַר וְנָהֲרָה</p></blockquote>
<blockquote><p>אֵלָיו כְּרוּץ צְמֵאִים אֶל מֵי / נָהָר, וְעוֹרֲרָהּ.</p></blockquote>
<blockquote><p>לִבָּהּ יְסָעֲרוּ בוֹ יַמֵּי / חִשְׁקָהּ וְאָמְרָה</p></blockquote>
<blockquote><p>"יוֹאֵל יְדִיד רְצוֹנוֹ לֵאמֹר – / אִמְרוֹ אֲקַבְּלָה,</p></blockquote>
<blockquote><p>מִצּוּף לְפִי בְּשַׁלְוָה יִנְעַם / וּצְרִי בְמַחֲלָה".</p></blockquote>
<br/>
<blockquote><p>רֹאשָׁהּ לְכָל עֲדִינֹות הָיְתָה / בִּגְאוֹן מְחוֹבֲבָהּ,</p></blockquote>
<blockquote><p>רַכָּה בְצֵל כְּנָפָיו חָיְתָה, / עַד נָד יְדִיד וּבָהּ</p></blockquote>
<blockquote><p>בָּעַל, וְאֶל דְּרָכִים צָפְתָה / מָתַי יְשׁוֹבֲבָהּ.</p></blockquote>
<blockquote><p>לַיְלָה כְסִיל וְכִימָה תִשְׁמֹר / תַּעֲמדֹ וְשָׁאֲלָה</p></blockquote>
<blockquote><p>בַעֲדוֹ, וְאִם יְאַחֵר פַּעַם – / תִּדֹּם וְיִחֲלָה.</p></blockquote>
<br/>
<blockquote><p>"מַה תִּפְחֲדִי וְלִבֵּךְ זָחַל, / רַעְיָה מְאַהֲבָה?</p></blockquote>
<blockquote><p>אַקְדִּישׁ שְׁמִי וּמִקְדָּשׁ נִחָל / צִיר אֶשְׁלְחָה, וּבָא</p></blockquote>
<blockquote><p>גּוֹאֵל לְגָאֳלֵךְ כִּי אֶנְחַל / אוֹתָךְ, וְאוֹיְבָה</p></blockquote>
<blockquote><p>אִם גִּדְּפָה שְׁנֵינוּ יִגְמֹר / הַקֵּץ וּמִלֲלָה:</p></blockquote>
<blockquote><p>זֶה אֵל אֱמֶת וְאַשְׁרֵי הָעָם / בָּחַר לְנַחֲלָה!"</p></blockquote>
