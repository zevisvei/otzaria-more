<h1>צוּרִי עֲוֹן מִרְיִי</h1>
אברהם אבן עזרא
<blockquote><p>צוּרִי, עֲוֹן מִרְיִי סְלַח / וְשַׂגְּבֵנִי,</p> <p>וּלְחֵיק יְדִיד אָדוֹם וְצַח / תְּשׁוֹבְבֵנִי!</p> <br/> <p>אִם אֶסְכְּלָה – מִצְוָתְךָ / תְחַכְּמֵנִי,</p> <p>אוֹ אֶאֱבַל – אִמְרָתְךָ / תְּנַחֲמֵנִי,</p> <p>אַף מִן קָמֵי תּוֹרָתְךָ / תְּרוֹמֲמֵנִי.</p> <p>עַל מֵי יְשׁוּעָתְךָ שְׁלַח / לְנַהֲלֵנִי</p> <p>וַהְדָרְךָ רְכַב צְלַח / וְהַעֲלֵנִי!</p> <br/> <p>בּוֹגֵד וְצוֹרֵר יֹאמְרוּ / כִּי קֵץ שְׁבוּתִי</p> <p>אָרַךְ, וְעַל כֵּן שִׂבְּרוּ / לִשְׁכֹּן בְּבֵיתִי.</p> <p>רַחְמֵי אֲלֹהִים נִכְמְרוּ / לָחִישׁ פְּדוּתִי.</p> <p>בֶּן עַבְדְּךָ יִשַׁי מְשַׁח / וְשַׁעְשְׁעֵנִי,</p> <p>עַד מָה יְהִי שָׁפָל וְשַׁח / מַלְכִּי לְעֵינִי?</p> <br/> <p>רַגְלַי בְּפַחִים נָפְלוּ – / הַתֵּר אֲסוּרִי,</p> <p>עֵינַי בְּגָלוּת אָפְלוּ - / הָאֵר מְאוֹרִי.</p> <p>מֶלֶךְ וְכֹהֵן יַעֲלוּ / לִבְנוֹת דְּבִירִי.</p> <p>מִכּוּשׁ וְחָבוֹר וַחֲלַח / אָנָּא פְדֵנִי,</p> <p>צָרַי בְּרַגְלֵי אַף דְּלַח / וְעוֹדֲדֵנִי!</p> <br/> <p>מַלְכִּי, בְּפִי וּבְמַעֲנִי / אוֹדֶה שְׁמֶךָ,</p> <p>כִּי עַל עשֶׁר וְעַל עֳנִי / אֲרוֹמֲמֶךָ.</p> <p>חֹן עַמְּךָ, וְלִי אֲנִי / רֹב רַחֲמֶיךָ</p> <p>צַוֵּה לְיָגוֹן נֶאֱלָח / לְשַׁכְּחֵנִי.</p> <p>אוֹרְךָ וַאֲמִתְּךָ שְׁלַח / לְהַנְחֵנִי!</p></blockquote>
