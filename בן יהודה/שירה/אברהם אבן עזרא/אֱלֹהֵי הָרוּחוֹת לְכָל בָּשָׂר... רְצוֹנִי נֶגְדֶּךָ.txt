<h1>אֱלֹהֵי הָרוּחוֹת לְכָל בָּשָׂר... רְצוֹנִי נֶגְדֶּךָ</h1>
אברהם אבן עזרא
<blockquote><p>אֱלֹהֵי הָרוּחוֹת / לְכָל בָּשָׂר אַתָּה</p> <p>וְחָכְמָה בַּטּוּחוֹת / בְּחָכְמָתְךָ שַׁתָּה</p> <p>וְגַם דַּבֵּר צָחוֹת / בְּיַד לָשׁוֹן תָּתָּה —</p> <p>אֱמֶת כֹּל חוֹלַלְתָּ / וּבוֹ לֹא עָמַלְתָּ, / רַק בְּפִיךָ מִלָּלְתָּ!</p> <br/> <p>בְּיָדְךָ עִתּוֹתַי / וּמַה זֶּה פִי יַעְנֶה —</p> <p>וְקָצְרוּ שִׂפְתוֹתָי? / וְכָל רֶגַע הִנֵּה</p> <p>בְקִרְבִּי אוֹתוֹתַי, / וְאָן לִבִּי יִפְנֶה —</p> <p>וְאוֹתוֹ הִגְבַּלְתָּ / בְּדָת שֶׁהִנְחַלְתָּ, / לִי בְרָכָה אָצָלְתָּ!</p> <br/> <p>רְצוֹנִי נֶגְדֶּךָ / וְטוֹב לִי בָחַרְתָּ,</p> <p>וּבִנְפֹל עַבְדֶּךָ / יְמִינוֹ עָזַרְתָּ,</p> <p>וּבִכְסוּת חַסְדֶּךָ / עֲוֹנִי הִסְתַּרְתָּ,</p> <p>וְעָלַי חָמַלְתָּ / וְטוֹבוֹת גָּמַלְתָּ / לִי גְּמוּל לֹא שָׁאָלְתָּ!</p> <br/> <p>הֲכִינוֹתָ סֵפֶר / לְכָל הַנּוֹלָדִים,</p> <p>וּבִכְלֵי עֵט סוֹפֵר / כְּתוּבִים מַעְבָּדִים,</p> <p>וּמָצָאתָ כֹפֶר / לְרוּחוֹת נִכְבָּדִים,</p> <p>וְנַפְשִׁי הִצַּלְתָּ / וְחַיַּי גָּאַלְתָּ, / מֵעֲזֹר לֹא חָדָלְתָּ!</p> <br/> <p>מְלֶאכֶת שָׁמֶיךָ / לְאוֹת לָךְ לֹא נִכְחָד,</p> <p>וְכֵס בִּמְרוֹמֶיךָ / בְּכֻלָּם הִתְאֶחָד,</p> <p>וְיַעְמֹד בִּשְׁמֶךָ / לְמַעַן הוּא אֶחָד,</p> <p>וְהַכֹּל פָּעַלְתָּ / וּבַכֹּל מָשַׁלְתָּ / וַעֲלֵי כֹל גָּדָלְתָּ!</p></blockquote>
