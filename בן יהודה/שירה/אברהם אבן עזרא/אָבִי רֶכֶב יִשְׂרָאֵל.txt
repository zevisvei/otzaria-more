<h1>אָבִי רֶכֶב יִשְׂרָאֵל</h1>
אברהם אבן עזרא
<p>אהבה</p>
<blockquote><br/> <p>אָבִי, רֶכֶב יִשְׂרָאֵל, / שָׂר אֶל מַלְאָךְ, רָד עִם אֵל –</p></blockquote>
<blockquote><p>נָפַל כָּרַע / מֵרַע / מֵרַע / מִסְתּוֹלֵל,</p></blockquote>
<blockquote><p>כִּסְאוֹ מִגֵּר, / יַגֵּר / חוֹגֵר / מִתְהַלֵּל.</p> <br/> <p>בֵּן יַקִּיר מֵחֵיק נִגְרָשׁ, / גַּם בֶּן הָאָמָה יָרַשׁ,</p></blockquote>
<blockquote><p>יַשְׂכִִּיל עַבְדִּי / נֶגְדִּי, / בַּדִּי / בַּל שֹׁרַשׁ,</p></blockquote>
<blockquote><p>מַשְׁחִית הוֹדִי / רוֹדִי; / דּוֹדִי, / בַּל תֶּחְרָשׁ!</p> <br/> <p>רַב לִמְשַׂנְאִי יִתְנַשָּׂא, / יִלְעַג – וּלְשׁוּר וּלְמַשָּׂא</p></blockquote>
<blockquote><p>גִּנַּת בִּיתָן / אֵיתָן / נָתַן / לִמְשִׁסָּה;</p></blockquote>
<blockquote><p>רוּחַ יִבְצֹר, / יַעְצֹר, / יִצֹּר / בַּמַּשָּׂא.</p> <br/> <p>הָשֵׁב לַמִּזְרָח לַצְּבִי / בַּת נָדִיב מִשְּׁבִי,</p></blockquote>
<blockquote><p>וּצְבִי מֵעַם לוֹעֵז / הָעֵז, / צוּר, עֶזְ-/רָתִי בִי,</p></blockquote>
<blockquote><p>מִבֵּית כֶּלֶא / עוֹלֶה / אֶעְלֶה / אֶל אָבִי.</p> <br/></blockquote>
<blockquote><p>׳מוֹשִׁיעֵךְ הִנֵּה / זֶה בָּא, / בִּתִּי, יִרְתֹּם מֶרְכָּבָה,</p></blockquote>
<blockquote><p>עֵדֶן נַחְלִי / יַחְלִי, / בַּחֲלִי / פָז צָבָא,</p></blockquote>
<blockquote><p>יִקַּח רֵיחוֹ / רִקְחוֹ / לֵב חוֹ-/לַת אַהֲבָה.׳</p></blockquote>
