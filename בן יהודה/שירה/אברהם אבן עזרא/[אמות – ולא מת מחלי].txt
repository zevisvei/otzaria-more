<h1>[אמות – ולא מת מחלי]</h1>
אברהם אבן עזרא
<p>אהבה</p>
<br/>
<blockquote><p>אָמוּת – וְלֹא מֵת מֵחֳלִי / הָאַהֲבָה חוֹשֵׁק לְפָנַי!</p> <p>חִשְׁקִי לְמֶלֶךְ רַב בְּלִי / רִפְיוֹן וּבוֹ כָל מַעֲיָנַי,</p> <p>בַּחְדַר לְבָבִי מַחֲלִי / צָפוּן וְלֹא נִכָּר בְּפָנַי,</p> <p>פֶּן יֹאמְרוּ עָלַי: חֵשֶׁק הֲרָגָהוּ / מַה לּוֹ לְבַקֶּשׁ לוֹ גְּדוֹלוֹת?</p> <br/> <p>בַּאַהֲבָה אוֹת נֶאֱמָן / לֹא עָמְדוּ רַבִּים בְּסוֹדוֹ:</p></blockquote>
<blockquote><p>אִם תֶּאֱהַב שָׁפָל – יְמַן / פִּיו תּוֹאֲנוֹת תַּחַת כְּבוֹדוֹ,</p> <p>אִם תֶּאֱהַב אָב רַחְמָן – / לַחְמֹל יְעִיֵרהוּ כְבוֹדוֹ.</p> <p>דּוֹדִי וְרֵעִי זֶה, בּוֹ אֶבְחֲרָה כִּי הוּא / יִצְפּוֹן לְיִשְׁרֵי לֵב גְּמוּלוֹת.</p> <br/> <p>רַבִּים תְּמֵהִים כִּי לְאֹם / עָבַד אֱלֹהָיו כָּל יְמוֹתָיו</p></blockquote>
<blockquote><p>וַיְקַו שְׁנַת פִּדְיוֹם וְיוֹם / רָצוֹן וְלֹא מָלְאוּ שְׁנוֹתָיו.</p> <p>וָאֶעֱנֵם: אִלּוּ בְתֹם / לֵב הֶאֱמִינוּ בוֹ וּמוֹפְתָיו –</p> <p>מוֹפְתֵי יְמֵי יֶשַׁע רָאוּ וְתָמָהוּ / וַיַּחְדְּלוּ בָּרָד וְקוֹלוֹת.</p> <br/> <p>"מֵת בַּשְּׁבִי, הֵא לָךְ צֳרִי – / מַעֲשֶׂה יְדֵי חוֹזֶה וְנָבִיא!</p> <p>לַחְבֹּש כְּאֵב קַח מִפְּרִי / פִיהֶם בְּחֶזְיוֹנֵי כְתָבִי</p> <p>אִמְרוּ לְבֵן בֵּין גּוּר אֲרִי / יָגוּר וּבֵין לַיִשׁ וְלָבִיא:</p> <p>אֹרֶךְ פְּדוּת אַשְׂבִּיעֵהוּ וְאַרְוֵהוּ / וּנְחַדְּשָׁה אַהְבַת כְּלוּלוֹת".</p></blockquote>
