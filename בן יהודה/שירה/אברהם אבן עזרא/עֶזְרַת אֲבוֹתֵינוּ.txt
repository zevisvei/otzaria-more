<h1>עֶזְרַת אֲבוֹתֵינוּ</h1>
אברהם אבן עזרא
<blockquote><p>עֶזְרַת אֲבוֹתֵינוּ / אַתָּה הוּא מֵעוֹלָם,</p> <p>שׁוּבָה אֶת שְׁבִיתֵנוּ / וְנָשׁוּב כִּימֵי עוֹלָם.</p> <br/> <p>אַתָּה אֵל חַי נַעֲלָה, / רֵעֶיךָ מַה יָּקְרוּ,</p> <p>הַבֵּט פּוֹנִים לְמַעְלָה / וּדְבַר עָתָק דִּבְּרוּ!</p> <p>בֵֶּן הַצָּעִיר נִקְלָה / וּבְנֵי הָרָב גָּבְרוּ,</p> <p>בְּרִית אָח לֹא שָׁמָרוּ, / הֵפֵרוּ בְּרִית עוֹלָם.</p> <br/> <p>בִּנְךָ, בְּכוֹרְךָ נִקְרָא – / נִינָיו מָרָה צָרְחוּ.</p> <p>הִכִּיתָם, אֵל, בְּעֶבְרָה / וּכְמֵת מִלֵּב נִשְׁכְּחוּ.</p> <p>תַּחַת מִשְׁפַּט בְּכוֹרָה / כִּפְלַיִם הֵם לָקְחוּ.</p> <p>קָרֵב הַקֵּץ וְיִשְׂמְחוּ / כָל חוֹסֵי בָךְּ לְעוֹלָם.</p> <br/> <p>רָאִיתָ עַם אֵין בּוֹ / כֹחַ: לִדְבַר שׁוֹנִים</p> <p>עָלִיתָ מִקִּרְבּוֹ, / וַיִּבְזוּהוּ מוֹנִים.</p> <p>וַיֹּאמֶר אֶל לִבּוֹ – / וְאִם רַבּוּ הַשָּׁנִים:</p> <p>חַסְדֵי אֵל נֶאֱמָנִים / סְמוּכִים לָעַד לְעוֹלָם!</p> <br/> <p>מַדּוּעַ נִמְשָׁכָה / תּוֹחֶלֶת בְּךָ מְקַוָּה?</p> <p>הַעֲלֵה נָא לָהּ אֲרוּכָה, / כִּי כָל הַיּוֹם דָּוָה,</p> <p>בִּזְכוּת עִיר הַמְּלוּכָה / צִיּוֹן – כִּי שָׁם צִוָּה</p> <p>אֲדֹנָי אֶת הַבְּרָכָה, / חַיִּים עַד הָעוֹלָם.</p></blockquote>
