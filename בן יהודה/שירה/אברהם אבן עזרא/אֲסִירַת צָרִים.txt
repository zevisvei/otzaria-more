<h1>אֲסִירַת צָרִים</h1>
אברהם אבן עזרא
<blockquote><p>אֲסִירַת צָרִים, / שְׁבוּיַת חֶרֶב / מְשַׂנְאָהּ.</p> <p>בְּאַרְצוֹת זָרִים / בְּלִבָּה תָּרֵב / לְחֶטְאָהּ;</p> <p>וְשָׁם קוֹל תָּרִים — / וְקוֹלָהּ עָרֵב — / לְבוֹרְאָהּ:</p> <p>יִשְׁעִי — פְרַח, / צִדְקִי — צְמַח / וְהָיָה כְלָח!</p> <p>וּרְכַב צְלַח, / אָדוֹם וְצָח! / מִרְיִי סְלַח</p> <p>וּכְרוּב מְשָׁח! / אֵיךְ מִדְּבִיר תִּבְרָח?</p> <p>אָן תֶּאֱרַח, / מְקוֹמְךָ תַנָּח?</p> <br/> <p>בְּרוּחַ נִכְאֶה / מְצַעֵק חֵילוֹ / לְפָנָיו,</p> <p>עֲדֵי יֵרָאֶה / כְּקֶדֶם גָּדְלוֹ / לְעֵינָיו;</p> <p>וְאֵיךְ יִשְׁתָּאֶה — / וְעוֹד הוֹד קוֹלוֹ / בְאָזְנָיו:</p> <p>עַם נֶאֱנַח, / אֵבֶל שְׁכַח / וּנְהִי זְנָח!</p> <p>קוּם נָא שְׂמַח, / רִנָּה פְצָח — / כִּי סַף וְשַׁח</p> <p>עֶבֶד וְאָח! / יָדְךָ לְאֵל תִּשְׁטַח,</p> <p>עָלָיו בְּטַח / תְּמוּרַת תּוֹתָח!</p> <br/> <p>רְפָא לֵב רָפֶה / וְהַךְ צָר חֵילוֹ / יְגַבֵּר!</p> <p>קְהָלְךָ צוֹפֶה / וּפִיו בַּצַּר לוֹ / יְדַבֵּר —</p> <p>וְלוּ יִסָּפֶה / לְזוּלָתְךָ לֹא / יְשַׂבֵּר:</p> <p>רוֹצְחַי רְצָח, / זוֹבְחַי זְבָח, / טוֹבְחַי טְבָח!</p> <p>נוֹגְחַי נְגָח, / דּוֹלְחַי דְּלָח! / אֵשׁ אַף קְדַח,</p> <p>בָּהֶם נְפָח — / יוֹם כַּאֲרִי תִצְרָח</p> <p>עַל נֶאֱלָח / כְּבֵדִי פִלָּח.</p> <br/> <p>מְעוֹנִי, הָפֵק / רְצוֹנִי לִרְצוֹן / לְבָבְךָ!</p> <p>וְצַר כַּף סוֹפֵק / תְּשַׂבַּע קָלוֹן / בְּשׁוּבְךָ!</p> <p>וְאֵיךְ תִּתְאַפֵּק — / וְעָרֵל בִּמְעוֹן / כְּרוּבְךָ</p> <p>שָׁכֵן וְנָח? / שׁוּב נָא וְקַח / נִלְכָּד בְּפָח,</p> <p>לֶחְיוֹ בְחָח / בִּגְבוּל חֲלָח! / עָלָיו פְּסָח,</p> <p>אָזְנָיו פְּתַח, / עֵינָיו עֲדֵי תִפְקַח —</p> <p>אוֹרְךָ שְׁלַח / וְעָלָיו תִּזְרָח!</p></blockquote>
