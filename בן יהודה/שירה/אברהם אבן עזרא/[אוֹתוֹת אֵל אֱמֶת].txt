<h1>[אוֹתוֹת אֵל אֱמֶת]</h1>
אברהם אבן עזרא
<blockquote><p>אוֹתוֹת אֵל אֱמֶת עַל / לֵב אֶכְתְּבֵם,</p></blockquote>
<blockquote><p>עֵדוֹתָיו פְּלָאוֹת / וָאֹהֲבֵם.</p></blockquote>
<br/>
<blockquote><p>בָּרָא דַּת יְמִינוֹ / עָצוּם, עֲדֵי</p></blockquote>
<blockquote><p>אֶל כִּסְאוֹ קְרוֹבִים, / שָׁם נוֹעֲדוּ,</p></blockquote>
<blockquote><p>הֵם רָצִים וְשָׁבִים, / לֹא נִפְרְדוּ –</p></blockquote>
<blockquote><p>מַצָּבָם בְּמַעַל / וִישַׂגְבֵם,</p></blockquote>
<blockquote><p>וּשְׁמוֹת הַצְּבָאוֹת / פִּיו יִקְּבֵם.</p></blockquote>
<br/>
<blockquote><p>רֹחַב דַּת יְמִינוֹ / עָצוּם עֲדֵי,</p></blockquote>
<blockquote><p>לִגְדוּדֵי מְעוֹנוֹ / יִדְמֶה וְדֵי</p></blockquote>
<blockquote><p>דֵעוֹתָם; הֲיֶשְׁנוֹ / צוּר בִּלְעֲדֵי</p></blockquote>
<blockquote><p>אֵל? יוֹאֵל וְיִפְעַל / וִיעַצְּבֵם,</p></blockquote>
<blockquote><p>יַעַט סוּת קְנָאוֹת / וִיחוֹבְבֵם.</p></blockquote>
<br/>
<blockquote><p>מַלְאָכָיו בְּחָכְמָה / לוֹ הִלְּלוּ,</p></blockquote>
<blockquote><p>עַד יָרְדוּ בְעָצְמָה / […] לוּ.</p></blockquote>
<blockquote><p>בָּנָיו בָּאֲדָמָה / כֵּן נִמְשְׁלוּ –</p></blockquote>
<blockquote><p>הִרְחִיקָם בְּגֹעַל / וַיְעַקְּבֵם,</p></blockquote>
<blockquote><p>עוֹד יָעִיר נְבוּאוֹת / וִיקָרְבֵם.</p></blockquote>
