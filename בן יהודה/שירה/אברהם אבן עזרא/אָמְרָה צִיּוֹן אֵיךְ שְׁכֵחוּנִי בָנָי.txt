<h1>אָמְרָה צִיּוֹן אֵיךְ שְׁכֵחוּנִי בָנָי</h1>
אברהם אבן עזרא
<p>קינה לט׳ באב</p>
<blockquote><br/> <p>אָמְרָה צִיּוֹן: / ׳אֵיךְ שְׁכֵחוּנִי בָנָי?</p></blockquote>
<blockquote><p>בְּצֵאתָם מֵאֵשׁ דָּת הַנְּתוּנָה בְסִינַי, / בָּעֲרָה בָם אֵשׁ אֲדֹנָי.׳</p></blockquote>
<blockquote><p>אָמַר אוֹיֵב: / ׳אִשִּׁי אָכְלָה אֶתְכֶם!</p></blockquote>
<blockquote><p>אַנְשֵׁי צִיּוֹן, אַיֵּה נָא אֱלֹהֵיכֶם? / יָקוּמוּ וְיַעַזְרֻכֶם!׳</p> <br/> <p>אָמְרָה צִיּוֹן: / ׳בָּנַי כְּצִפּוֹר נִלְקְחוּ,</p></blockquote>
<blockquote><p>בְּאַרְצוֹת זָרִים מֵאֶרֶץ צְבִי נִדְּחוּ,</p></blockquote>
<blockquote><p>בִּגְבוּרָתָם בִּימֵי שַׁלְוָתָם בָּטְחוּ –</p></blockquote>
<blockquote><p>בַּצָּר לָהֶם לְשֵׁם נִשְׂגָּב לֹא בָרְחוּ;</p></blockquote>
<blockquote><p>אֵיכָה שָׁכְחוּ / מִגְדַּל עֹז שֵׁם אֲדֹנָי?׳</p> <br/> <p>אָמַר אוֹיֵב: / ׳בְּנֵי אֵל – אָבַד שִׂבְרָם,</p></blockquote>
<blockquote><p>חָשְׁכָה עֵינָם, כִּי כִבִּיתִי נֵרָם.</p></blockquote>
<blockquote><p>בְּכֹחַ יָדִי גָּבַרְתִּי עַל צוּרָם,</p></blockquote>
<blockquote><p>וְזֶה הָאוֹת: כִּי עַמּוֹ לְעֵינָיו יָחְרָם,</p></blockquote>
<blockquote><p>וְאֹמַר בְּקוֹל רָם / קִרְאוּ בְשֵׁם אֱלֹהֵיכֶם!׳</p> <br/> <p>אָמְרָה צִיּוֹן: / ׳רָעוֹתַי לִי טוֹבוּ,</p></blockquote>
<blockquote><p>עַל שֵׁם קְדוֹשִׁי תַּעְנוּגִים נֶחְשָׁבוּ,</p></blockquote>
<blockquote><p>כִּי תוֹכְחוֹתָיו לִי מִדְּבַשׁ עָרָבוּ,</p></blockquote>
<blockquote><p>וְקִצֵּי יְשׁוּעוֹת בָּנַי עוֹד יִקְרָבוּ,</p></blockquote>
<blockquote><p>כִּי יָשׁוּבוּ – / וְיָשֹׁב חֲרוֹן אַף אֲדֹנָי.׳</p> <br/> <p>אָמַר אוֹיֵב: / ׳רָאִיתִי אֲשֶׁר עֵינַי</p></blockquote>
<blockquote><p>שָׁאֲלוּ, וְדָתִי יְשָׁרָה וְאֵין קֵץ לְשָׁנָי.</p></blockquote>
<blockquote><p>וְאֵיךְ, בְּנֵי צִיּוֹן, תּוּכְלוּ לַעֲמֹד לְפָנָי?</p></blockquote>
<blockquote><p>וּבַמֶּה תְּשׁוּבוּן – וְהִתְעַוְּתָה דָּת סִינַי,</p></blockquote>
<blockquote><p>וְהָיְתָה יַד אֲדֹנָי / בָּכֶם וּבַאֲבֹתֵיכֶם?׳</p> <br/> <p>אָמְרָה צִיּוֹן: / ׳מְשַׂנְאֵי אֵל, הֲלֹא שְׁמַעְתֶּם</p></blockquote>
<blockquote><p>דִּבְרֵי נְבִיאָיו? וְאֵיךְ דָּתוֹ חֵרַפְתֶּם –</p></blockquote>
<blockquote><p>וְתִתְהַלֲלוּ בַּחֹק מִמֶּנָּה גְנַבְתֶּם?</p></blockquote>
<blockquote><p>עַתָּה יָעִיר יְמִין עֻזּוֹ וַאֲבַדְתֶּם,</p></blockquote>
<blockquote><p>יַעַן אַתֶּם הֲמִתֶּם / אֶת עַם אֲדֹנָי.׳</p> <br/> <p>אָמַר אוֹיֵב: / ׳מִכֶּם נְבִיאִים גָּמְרוּ,</p></blockquote>
<blockquote><p>וְהַקַּדְמוֹנִים, אִם אֱמֶת וְתָמִים דִּבְּרוּ –</p></blockquote>
<blockquote><p>בְּבַיִת שֵׁנִי הַנֶּחָמוֹת עָבְרוּ.</p></blockquote>
<blockquote><p>וְעָבְרוּ אֶלֶף שָׁנִים – וְעוֹד מַה תְּשַׂבְּרוּ?</p></blockquote>
<blockquote><p>וְאֵיכָה תֹאמְרוּ: / הוּא יָבוֹא וְיוֹשַׁעֲכֶם?׳</p> <br/> <p>אָמְרָה צִיּוֹן: / ׳דֹּם אוֹיֵב, אֲרִי נוֹהֵם,</p></blockquote>
<blockquote><p>וּנְבוּאוֹת עֲתִידוֹת, אֱמֹר עַתָּה, אַיֵּה הֵם –</p></blockquote>
<blockquote><p>וְכִתְּתוּ גוֹיִם לְאִתִּים חַרְבוֹתֵיהֶם?</p></blockquote>
<blockquote><p>וְנִקְבְּצוּ נְפוּצַי, וְאֵלֶּה שְׁמוֹת שִׁבְטֵיהֶם,</p></blockquote>
<blockquote><p>וְקוֹרָא לָהֶם: / אַתֶּם כֹּהֲנֵי אֲדֹנָי!׳</p></blockquote>
