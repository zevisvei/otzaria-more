<h1>רְצוֹנֶיךָ פְּנֵה אֵל אַדִּיר</h1>
אברהם אבן עזרא
<p>סליחה</p>
<blockquote><p>[……]</p></blockquote>
<blockquote><p>[..] לי […] רְצוֹנֶיךָ / פְּנֵה, אֵל אַדִּיר,</p></blockquote>
<blockquote><p>ולך ל[…]בוניך שְׁמָ[ךְ] עֵת אַזְ[כִּיר] –</p></blockquote>
<blockquote><p>תָּבוֹא לְפָנֶיךָ / אֶנְקַת אָסִיר.</p> <br/> <p>[…]ת הצרים ש[..ה] רוּחִי</p></blockquote>
<blockquote><p>וּמֵעוֹל הֶעָרִים / כָּשַׁל כֹּחִי,</p></blockquote>
<blockquote><p>וּלְךָ בְּמִסְתָּרִים / אֶבְכֶּה, גּוֹחִי,</p></blockquote>
<blockquote><p>וְאֶתְעַטֵּף בְּשִׂיחִי / וְדִמְעִי יַגִּיר.</p> <br/> <p>[וּלְךָ] לְשָׁמֶיךָ / אֶשָּׂא עֵינִי,</p></blockquote>
<blockquote><p>וּלְךָ [בַּ]הֲדוֹמְךָ / אוֹדֶה עֲוֹנִי,</p></blockquote>
<blockquote><p>הַבֵּט בְּרַחֲמֶיךָ / אֵלַי, קוֹנִי,</p> <p>[וְ]עֵינֶיךָ מֶנִּי / אַל נָא תַסְתִּיר.</p> <br/> <p>[…] לְהַאֲבִיד / שְׁמִי מְהוֹלָלִי,</p></blockquote>
<blockquote><p>אֶת עֻלִּי [הִכְ]בִּיד / וְהִגְדִּיל [סִבְ]לִי</p></blockquote>
<blockquote><p>וּבִי בְפֶרֶךְ הֶעֱבִיד; / וָאֱהִי בְמַעֲלִי</p></blockquote>
<blockquote><p>מוּזָר וְנָכְרִי, וְלִי / אֵין עוֹד מַכִּיר.</p> <br/> <p>הַקְשֵׁב קוֹל פִּלּוּל / נִיב מַאֲמָרִי,</p></blockquote>
<blockquote><p>וּרְצֵה נְעִים הִלּוּל / […….רי]</p></blockquote>
<blockquote><p>מִשִׁבְרֵי גִּלּוּל / הַתֵּר אֱ[סוּרִי],</p></blockquote>
<blockquote><p>כִּי בִלְתְּךָ, צוּרִי, / [אין] לִי מַתִּיר.</p></blockquote>
