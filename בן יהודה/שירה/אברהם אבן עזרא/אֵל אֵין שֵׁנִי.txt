<h1>אֵל אֵין שֵׁנִי</h1>
אברהם אבן עזרא
<p>סליחה</p>
<blockquote><p>אֵל אֵין [שֵׁנִי], מַעֲנֵה הֶגְיוֹן / שִׂפְתוֹתַי הַקְשֵׁב,</p></blockquote>
<blockquote><p>[….]ני עֶלְיוֹן / רֵיקָם אַל תָּשֵׁב,</p></blockquote>
<blockquote><p>וְאוֹתִי בְצִיּוֹן / [……] הוֹשֵׁב.</p></blockquote>
<blockquote><p>בְּצִדְקַת אֲבוֹתַי / תְכַפֵּר חוֹבוֹתַי</p></blockquote>
<blockquote><p>וְתָסִיר עַצְּ[בוֹ]תַי / ומ[ס…..ודוֹתי / [….] בִגְלָלָךְ –</p></blockquote>
<blockquote><p>רְפָאָה נַפְשִׁי, כִּי חָטָאתִי לָךְ.</p> <br/> <p>תַּבִּיט בְּרָעַת עַבְדְּךָ […] וְתִשְׁמַע שוע[…] קוֹלִי</p></blockquote>
<blockquote><p>אֵל עֶלְיוֹן, וְתָחֹן דמע [……..לי]</p></blockquote>
<blockquote><p>[……..] לעולי</p></blockquote>
<blockquote><p>וְעֹצֶם שְׁגִיּוֹתַי / שני א[…]ם חֲסִידִים</p></blockquote>
<blockquote><p>[….תי / ….תי אלה [….] לך</p> <br/></blockquote>
<blockquote><p>בִּרְכוֹת אֱמוּנַי / שני[….] חֲסִידִים,</p></blockquote>
<blockquote><p>שְׁמַע תַּחְנוּנַי, / אֵל, [וְהוֹ]שִׁיב יְחִידִים,</p></blockquote>
<blockquote><p>וְ[חוּס] עַל [עַ]מִּים אֲשֶׁר אוֹתְךָ עוֹבְדִים;</p></blockquote>
<blockquote><p>לְבֵיתְךָ יִקָּווּ / וּבִדְבָרִים נָאווּ</p></blockquote>
<blockquote><p>גְּדֻלָּתְךָ יְחַוּוּ, / יַחַד יִשְׁתַּחֲווּ / לָךְ וִיזַמְּרוּ לָךְ.</p> <br/> <p>רוּחִי וְנַפְשִׁי / אֶהֱרֹג עַל דָּתָךְ</p></blockquote>
<blockquote><p>וּבְקוֹל רַחְשִׁי / אֶצְעַק מַר ב[…ת]ך</p></blockquote>
<blockquote><p>עֲנֵנִי, קְדוֹשִׁי, / לְמַעַן תּוֹרָתָךְ,</p></blockquote>
<blockquote><p>וְהַמְצִיא לִי [חֲ]נִינָה / [וְדִי]נִי הָבִינָה,</p></blockquote>
<blockquote><p>וַהֲגִיגִי בִינָה / ומ[….] הַאֲזִינָה / קוֹלִי בְּקָרְאִי לָךְ.</p> <br/> <p>הָאֵל הָרֵם / ת[…] כָּל מ[…]ינַי</p></blockquote>
<blockquote><p>וּמְחֵה זֶרֶם / דְּמָעַי מ[…ני]</p></blockquote>
<blockquote><p>מַהֵר בְּטֶרֶם / […]ות […]ל בָּנַי</p></blockquote>
<blockquote><p>וְעַל עַבְדְּךָ חוּסָה, / אֱלֹ[הַּ] רָם וְנִשָּׂא,</p></blockquote>
<blockquote><p>[……]כסה / וְלָנוּ כָּל מַשָּׂא / [אשר…..] לָךְ.</p></blockquote>
