<h1>אוֹתוֹת נִחוּמִים</h1>
אברהם אבן עזרא
<blockquote><p>אוֹתוֹת נִחוּמִים / אֵחָרוּ –</p></blockquote>
<blockquote><p>אַךְ עַמִּי תָּמִים / בַּחֲזוֹן מְגִלָּתוֹ.</p> <br/> <p>בָּחַר צוּר חֶבְלוֹ / בְּכוֹרוֹ – וְלֹא נִחָם,</p> <p>מֵאָז גּוֹי מוֹשְׁלוֹ / רָב וּבַעֲדוֹ נִלְחָם.</p> <p>הִכָּה בִגְלָלוֹ / כָּל בְּכוֹר בְּאֶרֶץ חָם,</p> <p>גָּעַר בִּתְהוֹמִים – / נִגְזָרוּ,</p> <p>עָבַר לֵב יַמִּים / בַּחֲרֹב מְצוּלָתוֹ.</p> <br/> <p>רָאָה אֶת עַמּוֹ / מֵאֲזַי כְּבִכּוּרָה,</p> <p>עַד בִּלְתִּי שׂוּמוֹ / כָּל יְצוּר וְכָל נִבְרָא</p> <p>הִקְדִּישׁוֹ לִשׁמוֹ / בִּבְרִית וּבְתוֹרָה.</p> <p>טֶרֶם עוֹלָמִים / נוֹצָרוּ</p> <p>נִבְחַר מֵעַמִּים / לִהְיוֹת סְגֻלָּתוֹ.</p> <br/> <p>הֵעָתֵר, נִשָּׂא, / אֶל שְׁפַל הֲמוֹנֶךָ,</p> <p>בֶּאֱדוֹם וּבְמַשָּׂא / הוּא שְׁלוּם אֱמוּנֶיךָ,</p> <p>עוֹמֵד בַּמַּשָּׂא / וַהֲלוּם חֲרוֹנֶךָ.</p> <p>בּוֹעֲלָיו רַחֲמִים / יְגַבֵּרוּ</p> <p>וּכְמִקְטַר סַמִּים / תֶּעֱרַב תְּפִלָּתוֹ.</p> <br/> <p>מִזְמוֹר שִׁיר נָעִים / שׁוֹמְרֵי בְרִית קוֹנָם</p> <p>בַּעֲלוֹת מוֹשִׁיעִים / אָז יְרַנְּנוּ יֵינָם,</p> <p>כִּי הִשְׁקִיט נָעִים / בַּשְּׁבִי, אֲשֶׁר חִנָּם</p> <p>אֶל אַנְשֵׁי דָמִים / נִמְכְּרוּ</p> <p>וּבְאֶפֶס דָּמִים / נִגְאֲלוּ בְחֶמְלָתוֹ.</p> <br/> <p>'חָזוּת נֶחָמָה / אֶל בְּנִי אֲחִישֶׁנָּה,</p> <p>אָקוּם לִנְקָמָה / עַל אֱדוֹם – אֲצִיתֶנָּה.</p> <p>נִשְׁבַּעְתִּי כַמָּה / זֹאת – וְלֹא אֲשִׁיבֶנָּה,</p> <p>וּלְיוֹשֵׁב מַשְׁמִים / בַּשְּׂרוּ</p> <p>כִּי הִנֵּה יָמִים / תִּהְיֶה גְאֻלָּתוֹ.'</p></blockquote>
