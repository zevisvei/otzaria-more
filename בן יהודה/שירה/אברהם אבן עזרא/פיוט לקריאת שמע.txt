<h1>פיוט לקריאת שמע</h1>
אברהם אבן עזרא
<blockquote><p>שְׁמַע יִשְׂרָאֵל / ה' אֱלֹהֵינוּ / ה' אֶחָד/</p></blockquote>
<br/>
<blockquote><p>אֶל אֵל מִי אֲחַבֵּר – / וּשְׁמוֹ אֶחָד,</p></blockquote>
<blockquote><p>הֵא הַלֵּב יְגַבֵּר / מָה הָאֶחָד,</p></blockquote>
<blockquote><p>אֶשְׁמְעָה מַה יְּדַבֵּר / נָבִיא אֶחָד –</p></blockquote>
<blockquote><p>שָׂם בְּיִשְׂרָאֵל / תּוֹרַת אֱלֹהֵינוּ, / מִשְׁפָּט אֶחָד.</p></blockquote>
<br/>
<blockquote><p>בָּרָא לִכְבוֹדוֹ / אוֹפַן אֶחָד,</p></blockquote>
<blockquote><p>וַיַּאֲצֵל הוֹדוֹ / מִן הָאֶחָד,</p></blockquote>
<blockquote><p>כֵּן צִוָּה לְעָבְדוֹ / פֹה שְׁכֶם אֶחָד –</p></blockquote>
<blockquote><p>בְּנֵי יִשְׂרָאֵל / מְשָׁרְתֵי אֱלֹהֵינוּ, / בְּנֵי אִישׁ אֶחָד.</p></blockquote>
<br/>
<blockquote><p>רָם עַל כָּל פָּעֳלוֹ / שֵׁם הָאֶחָד,</p></blockquote>
<blockquote><p>וְאֵיךְ הֶרְאָה גָדְלוֹ / בְּבַיִת אֶחָד,</p></blockquote>
<blockquote><p>עַל כֵּן קֹרָא לוֹ / רוֹעֶה אֶחָד –</p></blockquote>
<blockquote><p>אֵל יִשְׂרָאֵל, / כִּי הוּא אֱלֹהֵינוּ, / מֶלֶךְ אֶחָד.</p></blockquote>
<br/>
<blockquote><p>מִי בָרָא אֵלֶּה? / הֲלֹא אָב אֶחָד!</p></blockquote>
<blockquote><p>מִי לַעֲשׂוֹת יַפְלֵא / עִם עַם אֶחָד?</p></blockquote>
<blockquote><p>הִשְׁקִיף – וַיְמַלֵּא / אֶת יַד אֶחָד.</p></blockquote>
<blockquote><p>צוּר יִשְׂרָאֵל, / אֱלֹהִים אֱלֹהֵינוּ – / שֵׁם הָאֶחָד. <br/></p></blockquote>
