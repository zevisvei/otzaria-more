<h1>אֹמֶן פִּלְאֲךָ הַרְאֵה</h1>
אברהם אבן עזרא
<blockquote><p>אֹמֶן פִּלְאֲךָ / הַרְאֵה לְנִפְעָם,</p> <p>וּכְיוֹם רָאֲךָ / אָז יָשִׁיר בְּנֹעַם:</p> <p>עָזִּי וְזִמְרָת יָהּ!</p> <br/> <p>בַּמֶּה הָיְתָה / חַטַּאת בְּנֶךָ —</p> <p>כִּי עִיר נִבְנְתָה / לַמַּעֲנֶךָ</p> <p>בָּאֵשׁ נִצְּתָה / עִם כֹּהֲנֶיךָ?</p> <p>וּבְבֵית כִּלְאֲךָ / תִּשְׁמַע לְשַׁוְעָם,</p> <p>אֵיךְ צַר יַלְאֲךָ — / כִּי פַעַם בְּפַעַם</p> <p>יֹאמַר: לֹא יִרְאֶה-יָּהּ!</p> <br/> <p>רַבַּת שָׂבְעָה / נַפְשִׁי כְלִמָּה,</p> <p>אַךְ לֹא יָדְעָה / עַד אָן וְכַמָּה</p> <p>בַּטִּיט טָבְעָה / וּבַבּוֹץ; וְשָׁמָּה,</p> <p>אֵל, יִקְרָאֲךָ / בִּנְךָ — הֲיוּעַם?</p> <p>מִמּוֹרָאֲךָ / סָמַר עֵת בְּזַעַם</p> <p>תְּיַסְּרֶנּוּ, יָהּ.</p> <br/> <p>מַחְלָתִי לְמִי / אַגִּיד? וְיֶרֶב</p> <p>בּוּז מִתְקוֹמֲמִי / בֹּקֶר וְעֶרֶב.</p> <p>לֹא אֶתֵּן דֳּמִי / לָךְ עַד תְּקָרֵב</p> <p>מוֹעֵד בּוֹאֲךָ / עַל צַר בְּרָעַם!</p> <p>יָשׁוּב כִּסְאֲךָ / אֶל צִיּוֹן — וְאָז עַם</p> <p>נִבְרָא יְהַלֶּל-יָהּ.</p></blockquote>
