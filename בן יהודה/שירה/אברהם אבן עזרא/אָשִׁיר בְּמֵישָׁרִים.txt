<h1>אָשִׁיר בְּמֵישָׁרִים</h1>
אברהם אבן עזרא
<p>(זמר לשבת)</p>
<blockquote><p>אָשִׁיר בְּמֵישָׁרִים / אֶת שִׁיר יְיָ,</p></blockquote>
<blockquote><p>קוֹל מַהֲלָל אָרִים, / כִּי רָם יְיָ.</p> <br/></blockquote>
<blockquote><p>אֶת סוֹד שְׁמוֹ גִּלָּה / אֶל עַם קְרוֹבוֹ</p></blockquote>
<blockquote><p>יוֹם מַעֲשָׂיו כִּלָּה / בוֹ, כִּי אֲהֵבוֹ;</p></blockquote>
<blockquote><p>לִמְקַדְּשׁוֹ נִגְלָה / יָפְיוֹ וְטוּבוֹ,</p></blockquote>
<blockquote><p>כִּי הַבְּרָכָה בוֹ / צִוָּה יְיָ.</p> <br/></blockquote>
<blockquote><p>בֹּקֶר, בְּהִנָּשְׂאָהּ / שֶׁמֶשׁ וְזָרְחָה,</p></blockquote>
<blockquote><p>מִלֵּב אֱנוֹשׁ רוֹאָהּ / תָּסִיר אֲנָחָה;</p></blockquote>
<blockquote><p>אֶמֶשׁ, בְּעֵת בּוֹאָהּ, / נַפְשִׁי שְׂמֵחָה,</p></blockquote>
<blockquote><p>עָלַי אֲזַי נָחָה / רוּחַ יְיָ.</p> <br/></blockquote>
<blockquote><p>רִאשׁוֹן לְכָל מוֹעֵד / הוּא בַתְּעוּדוֹת,</p></blockquote>
<blockquote><p>יִסְעַד לְבַב סוֹעֵד / שָׁלֹשׁ סְעוּדוֹת;</p></blockquote>
<blockquote><p>לַבָּא לְאוֹת וּלְעֵד / אֵין בּוֹ עֲבוֹדוֹת,</p></blockquote>
<blockquote><p>כִּי אִם דְּבַר הוֹדוֹת / אֶת שֵׁם יְיָ.</p> <br/></blockquote>
<blockquote><p>הַקּוֹל מְאֹד עָרֵב / נִשְׁמַע בְּאָזְנֵי</p></blockquote>
<blockquote><p>בָּנִים: רְאוּ, קָרֵב / לַיְלָה וְהִנֵּה</p></blockquote>
<blockquote><p>׳נֵר בָּעֲרוּ׳ עוֹרֵב / צוֹעֵק וְעוֹנֶה,</p></blockquote>
<blockquote><p>הִתְקַדְּשׁוּ לִפְנֵי / בּוֹא יוֹם יְיָ!</p> <br/></blockquote>
<blockquote><p>מַבְדִּיל בְּאַחְרִיתוֹ / עָשִׁיר וּמִסְכֵּן;</p></blockquote>
<blockquote><p>כִּי אֵל בְּצִדְקָתוֹ / לִבּוֹ יְתַכֵּן,</p></blockquote>
<blockquote><p>וּבְבֵית תְּפִלָּתוֹ / הוֹדוֹ יְשַׁכֵּן –</p></blockquote>
<blockquote><p>אָמֵן וְאָמֵן, כֵּן / יַעְשֶׂה יְיָ!</p></blockquote>
