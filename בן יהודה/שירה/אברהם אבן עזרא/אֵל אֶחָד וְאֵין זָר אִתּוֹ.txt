<h1>אֵל אֶחָד וְאֵין זָר אִתּוֹ</h1>
אברהם אבן עזרא
<blockquote><p>אֵל אֶחָד וְאֵין זָר אִתּוֹ, / מוֹפְתָיו אֶמְצְאָה בִי;</p> <p>אֵיךְ אֶדְרֹש בְּחַיַּי בִּלְתּוֹ — / וּשְׁמוֹ צוּר לְבָבִי?</p> <br/> <p>בָּרָא עַל כְּבוֹדוֹ רִקְמַת / גָּלְמִי עַד הֱיוֹתִי,</p> <p>וַיִּפַּח בְּאַפִּי נִשְׁמַת / חַיִּים עַד חֲיוֹתִי.</p> <p>וַיִּתֵּן בְּיַד לֵב חָכְמַת / סוֹד חַיַּי וּמוֹתִי —</p> <p>אֶעֱרֹךְ אֶת כְּלֵי מִלְחַמְתּוֹ, / אֶקְרַב אֶל קְרָבִי,</p> <p>מָה אָדָם וּמַה מַּחְשַׁבְתּוֹ? / אַךְ עִמִּי יְרִיבִי!</p> <br/> <p>רָחוֹק מִשְּׁמֵי שָׁמָיו — רַק / קָרוֹב מִקְּרָבִים,</p> <p>כָּל חוֹפֵשׂ דְּרָכָיו מֹרַק / שֻׁטַּף מִכְּזָבִים.</p> <p>אֶרְאֶנּוּ בְעַיִן, זֹרַק / עָלָיו מֵי אֲהָבִים —</p> <p>אַךְ דַּעַת יְקַר מַחְשַׁבְתּוֹ / הֶלְאָה מַחֲשָׁבִי,</p> <p>לֹא עָמַד בְּסוֹד מַתְכֻּנְתּוֹ / כָּל חָכָם וְנָבִיא.</p> <br/> <p>מָלֵא פִי וְלִבִּי זִכְרוֹ, / כִּי לִשְׁמוֹ אֲאַוֶּה.</p> <p>בּוֹ אֶשְׁתַּעְשְׁעָה וּדְבָרוֹ / עִם נַפְשִׁי אֲחַוֶּה.</p> <p>אֶמָּשֵׁך לְנַחַל אוֹרוֹ / וּלְטוּבוֹ אֲקַוֶּה.</p> <p>טוֹב חִנָּם בְּדָרְשִׁי תִתּוֹ — / אַל יִחַר לְאָבִי!</p> <p>מָה אַעַן? וְהֵן, עַל דַּעְתּוֹ, / אֵין לִי שַׁי לְהָבִיא.</p></blockquote>
