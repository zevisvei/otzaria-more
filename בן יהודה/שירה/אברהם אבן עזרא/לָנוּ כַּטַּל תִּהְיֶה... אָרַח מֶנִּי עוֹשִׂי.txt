<h1>לָנוּ כַּטַּל תִּהְיֶה... אָרַח מֶנִּי עוֹשִׂי</h1>
אברהם אבן עזרא
<blockquote><p>לָנוּ כַּטַּל תִּהְיֶה, / נִפְרַח כַּשּׁוֹשַׁנָּה.</p> <p>יִמָּלֵא פִינוּ שְׂחוֹק / וּלְשׁוֹנֵנוּ רִנָּה!</p> <br/> <p>אָרַח מֶנִּי עוֹשִׂי / וַיִּגָּלֶה מַחְסִי,</p> <p>מֵי רוֹשׁ מִלֵּא כוֹסִי, / גַּם לַעֲנָה עֲסִיסִי.</p> <p>אֵאָנֵק בְּכַף תּוֹפְשִׂי, / אֶצְעַק בְּיַד חוֹמְסִי;</p> <p>מֵרֹב שִׂיחִי וְכַעְסִי / דִבַּרְתִּי עַד הֵנָּה!</p> <br/> <p>בִּכְלֵי מְצוֹדָתִי / נָפְלָה יְחִידָתִי.</p> <p>עָבְרָה יְשׁוּעָתִי / לִרְאוֹת בְּרָעָתִי.</p></blockquote>
<blockquote><p>אֶקְרָא בְדַלּוּתִי / אֶל אֵל אֱיָלוּתִי:</p> <p>קוּמָה בְעֶזְרָתִי, / הַחְזֵק מָגֵן וְצִנָּה!</p> <br/> <p>רַחֲמֶיךָ נֶאֱמָנִים – / הַחֲדָשִׁים כַּיְשָׁנִים,</p> <p>מִלֶּיךָ נִתְכָּנִים / עִם אָבוֹת גַּם בָּנִים.</p></blockquote>
<blockquote><p>שׁוֹבֵב שִׁבְיַת אֱמוּנִים; / אָז יֹאמְרוּ כָּל מוֹנִים:</p> <p>אֶבֶן מָאֲסוּ הַבּוֹנִים / הָיְתָה לְרֹאשׁ פִּנָּה.</p> <br/> <p>מָתַי ‘כֶּרֶם חֶמֶר’ / לַבַּת אֲשֶׁר שִׂבְּרָה</p> <p>עַל צוּרָהּ יֵאָמֵר? / כִּי כֵן דִּבֶּר צוּרָהּ:</p> <p>'לַיְלָה וְיוֹם שׁוֹמֵר / אֶהְיֶה – וְתַם צִירָהּ!</p> <p>וַאֲנִי יְיָ נוֹצְרָהּ, / לִרְגָעִים אַשְׁקֶנָּה.'</p></blockquote>
