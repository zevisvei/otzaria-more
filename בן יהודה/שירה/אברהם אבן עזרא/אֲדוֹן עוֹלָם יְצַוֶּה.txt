<h1>אֲדוֹן עוֹלָם יְצַוֶּה</h1>
אברהם אבן עזרא
<blockquote><p>אֲדוֹן עוֹלָם יְצַוֶּה / לְדַל צוֹעֵק חַסְדּוֹ,</p> <p>וַעֲצָבָיו יְחַוֶּה / וְיִתְחַנֵּן לְנֶגְדּוֹ</p> <p>וְלִסְלִיחָה יְקַוֶּה / וְלִדְבָרָיו לְעַבְדּוֹ:</p> <p>רוֹצֶה יְיָ אֶת יְרֵאָיו / אֶת הַמְיַחֲלִים לְחַסְדּוֹ.</p> <br/> <p>בְּחוּנִים לְךָ בְּתַחֲנוּן / בָּאוּ וְשִׁיר מַעֲלִים</p> <p>לְךָ, שׁוֹכֵן שְׁחָקִים, / וְרַחֲמֶיךָ מְיַחֲלִים;</p> <p>יֶהֱמוּ מֵעֶיךָ / וּבִנְךָ אַל תַּכְלִים,</p> <p>מָה אָדָם וְחַסְדּוֹ / וּכְאַיִן חֶלְדּוֹ.</p> <br/> <p>רָם עַל כָּל רָמִים / וְיָדוֹ כֹּל יָצְרָה,</p> <p>רְצֵה עַם דַּל וְנִבְדָּל, / עֲנֵהוּ בְּעֵת צָרָה,</p> <p>הַט אֹזֶן לְצַעֲקוֹ / כִּי בְשִׁמְךָ יִקְרָא</p> <p>וְשׁוּר בִּכְיוֹ לְמִרְדּוֹ / וְשַׂק תָּפַר לְגִלְדּוֹ.</p> <br/> <p>הֲפוּכִים בְּתוֹךְ רִשְׁעָם / קוֹל נָתְנוּ בְּבֵיתְךָ,</p> <p>לִקְרַאת רוֹדְפֵיהֶם / נָא הָרֵק חֲנִיתְךָ</p> <p>וְעַל כָּל שׂוֹנְאֵיהֶם / תִּשְׁפֹּךְ אֶת חֲמָתְךָ;</p> <p>רְאֵה אוֹיְבוֹ לְצִדּוֹ / וְחַטָּאתוֹ לְנֶגְדּוֹ.</p> <br/> <p>מָה אַעַן וְאֹמַר – / וְחֶטְאִי יַעֲנֶה בִּי</p> <p>וּפְשָׁעַי אֵדַע / וְאַשְׁמָתִי וְחוֹבִי?</p> <p>וְאֵיכָה רֹאשׁ אָרִים / לְנֶגֶד צוּר לְבָבִי?</p> <p>נִפְּלָה נָּא בְיָדוֹ / כִּי אֵין מִלְּבַדּוֹ.</p></blockquote>
