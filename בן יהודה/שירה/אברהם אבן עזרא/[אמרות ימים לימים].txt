<h1>[אמרות ימים לימים]</h1>
אברהם אבן עזרא
<p>אופן</p>
<br/>
<blockquote><p>אֲמָרוֹת / יָמִים לְיָמִים יֶהְגּוּ טְהוֹרוֹת.</p></blockquote>
<br/>
<blockquote><p>בְּכִסֵּא / רָם מַעֲלוֹת בְּחָכְמָה קְרָצָם,</p></blockquote>
<blockquote><p>וְנוֹשֵׂא / מַחְנוֹת אֲשֶׁר לְפָנָיו הֱפִיצָם,</p></blockquote>
<blockquote><p>וְעוֹשֶׂה / תוֹלְדוֹת אֲדָמָה לֹא כְחֶפְצָם –</p></blockquote>
<blockquote><p>בְּרוּצָם / עַל פִּי אֲדֹנָי כֻּלָּם יְצוּרוֹת</p></blockquote>
<blockquote><p>בְּצוּרוֹת, / עַל מִשְׁפְּטֵיהֶן תָּמִיד נְצוּרוֹת.</p></blockquote>
<br/>
<blockquote><p>רְשׁוּמוֹת / בִּכְתַב אֱמֶת, כְּחַיּוֹת דְּמוּתָן,</p></blockquote>
<blockquote><p>בְּשֵׁמוּת / הוּא יִסְפְּרֵן בְּבוֹאָן וְצֵאתָן,</p></blockquote>
<blockquote><p>וְלִימוֹת / קָצַב יְשׁוֹבְבֵן אֶל נְוָתָן –</p></blockquote>
<blockquote><p>בְּאֵיתָן / שִׁבְתָּן עַל עֲגֻלּוֹת בְּרוּרוֹת,</p></blockquote>
<blockquote><p>צְרוּרוֹת / בְּאוֹר חַיִּים וְדוֹלְקוֹת כְּנֵרוֹת.</p></blockquote>
<br/>
<blockquote><p>מְשָׁרְתֵי / עֶלְיוֹן דְּמוּת מַלְאָכִים בְּכִסְאוֹת</p></blockquote>
<blockquote><p>בְּפַאֲתֵי / שֶׁבַע, וְסוֹד תְּלִי שָׂם בְּפֵאוֹת,</p></blockquote>
<blockquote><p>וְלִמְתֵי / לֵבָב בְּלִי פֶה עוֹנוֹת פְּלִיאוֹת.</p></blockquote>
<blockquote><p>בְּרִיאוֹת / אֶרֶץ בְּסוֹד שְׁנַיִם קְשׁוּרוֹת,</p></blockquote>
<blockquote><p>לְהוֹרוֹת / בָּהּ עָשָׂה אֶת שְׁנֵי הַמְּאוֹרוֹת.</p></blockquote>
