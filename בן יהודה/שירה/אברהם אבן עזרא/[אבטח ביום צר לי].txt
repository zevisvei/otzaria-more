<h1>[אבטח ביום צר לי]</h1>
אברהם אבן עזרא
<p>מאורה</p>
<br/>
<blockquote><p>אֶבְטַח בְּיוֹם צַר לִי / בָּאֵל, וְרָם דִּגְלִי;</p></blockquote>
<blockquote><p>אִם חֵטְא יְהִי נוֹשֵׁךְ – / פִּתְאֹם יְכֻפַּר לִי.</p></blockquote>
<br/>
<blockquote><p>בִּינָה, יְדִיד, שַׁוְעִי / הַקְשֵׁב וְהַאֲזִינָה</p></blockquote>
<blockquote><p>נִיבִי, וְשׁוּר דִּמְעִי, / הוֹד תָּאֳרִי שֻׁנָּה.</p></blockquote>
<blockquote><p>אֵיךְ אֶשְׁקְטָה, רוֹעִי? / הֵן אוֹיְבִי חָנָה</p></blockquote>
<blockquote><p>עָלַי, וְסָר צִלִּי, / גַּם מָעֲדָה רַגְלִי,</p></blockquote>
<blockquote><p>קֶשֶׁת לְחֵץ מוֹשֵׁךְ / נֶגְדִּי, וְיוֹרֶה לִּי.</p></blockquote>
<br/>
<blockquote><p>רוֹעִי, בְּהִנָּתְנִי / תּוֹךְ כּוּר עֳנִי וּבְיַד</p></blockquote>
<blockquote><p>זָרִים נְתַתָּנִי, / וּבְיַד יְלִיד צַיָּד –</p></blockquote>
<blockquote><p>הוֹלֵם הֲלָמַנִי, / גַּם הָיְתָה בִי יָד.</p></blockquote>
<blockquote><p>אַל תַּט בְּאַף, אֵלִי, / אֵלַי וְשָׂא סִכְלִי,</p></blockquote>
<blockquote><p>מֵאָהֳלֵי מֶשֶׁךְ / יֶעְרַב לְךָ קוֹלִי!</p></blockquote>
<br/>
<blockquote><p>מַלְכִּי, עֲנֵה וּפְדֵה / נִבְדָּל לְךָ הוֹמֶה,</p></blockquote>
<blockquote><p>בּוֹ כָּל זְאֵב יִרְדֶּה – / וּלְיֶשְׁעֲךָ צָמֵא,</p></blockquote>
<blockquote><p>מִי בִּלְתְּךָ פוֹדֶה / וּמִי לְךָ יִדְמֶה?</p></blockquote>
<blockquote><p>אִם יַעֲנֶה בַּעְלִי – מַה יַּעֲשֶׂה צַר לִי?</p></blockquote>
<blockquote><p>בַּבּוֹר וּבַחֹשֶׁךְ / שִׁמְךָ יְהִי אוֹר לִי. <br/></p></blockquote>
