<h1>[חיה עצומה מכל בריאה]</h1>
אברהם אבן עזרא
<p>אופן</p>
<br/>
<blockquote><p>חַיָּה עֲצוּמָה / מִכָּל בְּרִיאָה, // מִבֶּן אֲדָמָה / דַעְתָּה פְּלִיאָה.</p></blockquote>
<br/>
<blockquote><p>    אָנָה מְעוֹנָה / מִשָּׁם נְשָׁמָה?</p></blockquote>
<blockquote><p>     בָּאִישׁ נְתָנָהּ, / כַּמָּה חֲכָמָה!</p></blockquote>
<blockquote><p>      אֵל חַי צְפָנָהּ / עַד נֶעֱלָמָה.</p></blockquote>
<blockquote><p>אֵיכָה וְעַל מָה / הָיְתָה כְלוּאָה // בַּגֵּו, וּבַמָּה / שָׁמַע וְרָאָה?</p></blockquote>
<br/>
<blockquote><p>     בַּמֶּה יְקֹרָא / שֵׁם רָם וְגָאָה? –</p></blockquote>
<blockquote><p>     וּדְמוּת וְצוּרָה, / מִשְׁמָע וּמַרְאָה</p></blockquote>
<blockquote><p>     אֵין לוֹ, וְנוֹרָא / זִכְרוֹ לְנִרְאָה!</p></blockquote>
<blockquote><p>שַׁבְתִּי כְדֻמָּה / מִזּוֹ קְרִיאָה // וָאֶשְׁמְעָה מָה / אָמְרָה נְבוּאָה.</p></blockquote>
<br/>
<blockquote><p>     רָצוּ שְׁחָקִים / לַיְלָה וְיוֹמָם,</p></blockquote>
<blockquote><p>     מִצֹּר חֲזָקִים – / מִי זֶה הֱרִימָם?</p></blockquote>
<blockquote><p>     כָּל הַמְּצוּקִים / אֶרֶץ מְקוֹמָם,</p></blockquote>
<blockquote><p>כֻּלָּה בְּחָכְמָה / קִנְיָן מְלֵאָה // אַךְ עַל בְּלִימָה / תּוֹכָם תְּלוּאָה.</p></blockquote>
<br/>
<blockquote><p>     מַרְאוֹת הֲמֻלּוֹת / אֵין קֵץ לְאוֹרָם,</p></blockquote>
<blockquote><p>     חוֹזְריִם חֲלִילוֹת / בִּמְעוֹן יְקָרָם,</p></blockquote>
<blockquote><p>     נוֹשְׂאִים תְּהִלּוֹת / תָּמִיד לְצוּרָם;</p></blockquote>
<blockquote><p>שִׁירָה נְעִימָה / עַל פֶּה נְשׂוּאָה, / עוֹנִים בְּאֵימָה, / אוֹמְרִים בְּיִרְאָה</p></blockquote>
