<h1>רשות לקדיש</h1>
אברהם אבן עזרא
<blockquote><p>אֱלֹהֵי הָרוּחוֹת / לְכָל בָּשָׂר אַתָּה</p> <p>וְחָכְמָה בַטֻּחוֹת / בְּחָכְמָתְךָ שַׁתָּ,</p> <p>וְגַם דַּבֵּר צַחוֹת / בְּיַד לָשׁוֹן תַּתָּ,</p> <p>אֱמֶת כֹּל חוֹלַלְתָּ / וּבוֹ לֹא עָמַלְתָּ, / רַק בְּפִיךָ מִלַּלְתָּ! <br/><br/> בְּיָדְךָ עִתּוֹתַי – / וּמַה זֶּה פִי יַעֲנֶה</p> <p>וְקָצְרוּ שִׂפְתּוֹתַי? / וְכָל רֶגַע הִנֵּה</p> <p>בְּקִרְבִּי אוֹתוֹתַי, / וְאָן לִבִּי יִפְנֶה</p> <p>וְאוֹתִי הִגְבַּלְתָּ? / בְּדָת שֶׁהִנְחַלְתָּ / לִי בְרָכָה אָצַלְתָּ! <br/><br/> רְחָבִים וַאְרֻכִּים / חֲסָדֶיךָ עִמִּי,</p> <p>וְרַעְיוֹנַי רַכִּים / וְלֹא אָבִין גָּלְמִי,</p> <p>וְעֶת נוּם וּבְהַשְׁכִּים / בְּךָ עָצַם עַצְמִי,</p> <p>וְעָלַי חָמַלְתָּ / וְטוֹבוֹת גָּמַלְתָּ / לִי, גְּמוּל לֹא שָׁאַלְתָּ! <br/><br/> הֲכִי טוֹב עָשִׂיתָ / וְאָנֹכִי אֶרְשַׁע,</p> <p>וְחוֹבִי כִסִּיתָ / וְאֶל שַׁוְעִי תִשַׁע,</p> <p>וְאַחַר נִסִּיתָ / לְבָבִי בָּךְ נוֹשַׁע,</p> <p>וְחַיַּי גָּאַלְתָּ / וְנַפְשִׁי הִצַּלְתָּ, / מֵעֲזֹר לֹא חָדַלְתָּ! <br/><br/> מְרוֹמִים שָׁכַנְתָּ / וְשַׁחִים מָלֵאתָ,</p> <p>וְכָל יֵשׁ חוֹנַנְתָּ / וְלַעְשׂוֹת הִפְלֵאתָ,</p> <p>וְכִסֵּא כוֹנַנְתָּ / וְעָלָיו נַעְלֵיתָ,</p> <p>וְהַכֹּל פָּעַלְתָּ / וּבַכֹּל מָשַׁלְתָּ / וְעַלֵי כֹל גָּדַלְתָּ! <br/></p></blockquote>
