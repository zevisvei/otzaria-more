<h1>בּוֹרֵא נַחֲמֵנִי</h1>
אברהם אבן עזרא
<p>גאולה</p>
<blockquote><br/> <p>בּוֹרֵא, נַחֲמֵנִי / וַהֲפֹךְ לְגִיל אֶבְלִי, / עוֹד אֵלִי אַתָּה, אֵל אֵלִי.</p> <br/> <p>אֶת כְּבוֹד שִׁמְךָ לֹא אָמִיר, / בְּךָ אֶעֱדֶה עֲדִי,</p></blockquote>
<blockquote><p>לִהְיוֹת לְצוּר לִי אוֹתְךָ אָמִיר; / אֵיךְ יֹאכְלוּ עַדִּי</p></blockquote>
<blockquote><p>צָרַי, פְּרִי חֹרֶשׁ גַּם אָמִיר / [….]?</p></blockquote>
<blockquote><p>קַנֵּא נָא, קוֹנִי, / כִּי שָׁלְלוּ שֶׁלִי / בַּשֶּׁלִי!</p> <br/> <p>בִּי, לָבוּשׁ בַּדִּים, / הָבֵן לְבֵן בָּא עַל קֵץ לִשְׁאֹל,</p></blockquote>
<blockquote><p>אָבוֹת גַּם בַּדִּים / אָז יִשְׂמְחוּ, עֵת בּוֹא יוֹם לִגְאֹל.</p></blockquote>
<blockquote><p>מֵחֶבְלֵי עֲוֹנִי, / מִבּוֹר שְׁאוֹן מַעֲלִי, / צוּר מַעֲלִי.</p> <br/> <p>רֹב חֶסֶד גּוֹמֵל / הַצּוּר, חֲסָדָיו עוֹד לֹא כָלוּ,</p></blockquote>
<blockquote><p>וּלְכָל אִישׁ גּוֹמֵל / נַפְשׁוֹ, וְכִלְיוֹתָיו לֹא כָלוּ –</p></blockquote>
<blockquote><p>פּוֹרֵחַ גּוֹמֵל / טוּבוֹ, גַּם אֶל כָּל אֱנוֹשׁ כָּלוּא;</p></blockquote>
<blockquote><p>לוֹ אֶשָּׂא עֵינִי: / עַד אָן בְּמוֹ מַקְלִי / אַתְּ מַקְלִי?</p> <br/> <p>הַצַּר בִּי מָשַׁל, / וַיְהִי לְרֹאשׁ הוֹלֵם פַּעַם,</p></blockquote>
<blockquote><p>וָאֶהְיֶה לְמָשָׁל, / יוֹם בּוֹ בְּבֵית צוּר הֵבִיא פַּעַם,</p></blockquote>
<blockquote><p>אָנָּא, אֵל, נִמְשָׁל – / חִישׁ בּוֹ, לְהַנְחִילוֹ טוֹב, פַּעַם,</p></blockquote>
<blockquote><p>וּלְצַר הִגְלַנִי, / בֹּגֵד בְּמַנְחִילִי, / מַן חִילִי!</p> <br/> <p>מִי זֶה בָאֵלִים / יַעֲרֹךְ לְאֵל רָב, חַי לָנֶצַח?</p></blockquote>
<blockquote><p>חוֹמְדֵי הָאֵלִים / יַשְׁמִיד וְהֵם יֹאבְדוּ לָנֶצַח!</p></blockquote>
<blockquote><p>וּלְכָל הָאֵלִים / מֵהֶם, אֱהִי עַד מָה נֶצַח?</p></blockquote>
<blockquote><p>גוֹאֲלִי, חָנֵּנִי / עוֹד כָּל יְמֵי הֶבְלִי, / טוֹב הַב לִי!</p></blockquote>
