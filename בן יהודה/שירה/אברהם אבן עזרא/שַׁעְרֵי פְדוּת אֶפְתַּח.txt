<h1>שַׁעְרֵי פְדוּת אֶפְתַּח</h1>
אברהם אבן עזרא
<blockquote><p>'שַׁעְרֵי פְדוּת אֶפְתַּח – / מִי יִנְעֲלֵם?</p> <p>שׁוֹכְנִים בְּבוֹר שִׁבְיָה / הֵן אֶגְאֲלֵם.</p> <br/> <p>אִם יָדְךָ רָמָה, / מוֹשֵׁל וְרָד,</p> <p>לוֹבֵשׁ לְבוּשׁ רִקְמָה, / בִּגְדֵי שְׂרָד –</p> <p>הַאִם לְיוֹם חֵמָה / לֹא תֶחֱרָד?</p> <p>בָּנַי אֲשֶׁר בַּפָּח – / אֵיךְ תִּלְחֲצֵם?</p> <p>מִיַּד פְּרִיץ חַיָּה / אֲחַלְּצֵם!</p> <br/> <p>בָּנִים אֲמֵלָלִים / וּמְיַחֲלִים</p> <p>בִּידֵי מְגֹאָלִים, / וּמְהוֹלֲלִים,</p> <p>צָרִים וְשׁוּעָלִים / וּמְחַבְּלִים:</p> <p>עוֹד לֹא בְרִית אֶשְׁכָּח / כִּי אֶעְזְרֵם,</p> <p>אֶפְדֵּם שְׁלִישִׁיָּה, / עוֹד אֶזְכְּרֵם.</p> <br/> <p>רַחְמֵי יְשׁוּעָתִי / הֵן אֶגְמְלֵם</p> <p>וּבְאוֹר נְתִיבָתִי / אֲנַטְּלֵם,</p> <p>עַל מֵי מְנוּחָתִי / אֲנַהֲלֵם.</p> <p>צִיר נֶאֱמָן אֶשְׁלָח / וִיקוֹמֲמֵם,</p> <p>לָהֶם יְהִי מִחְיָה / וִירוֹמֲמֵם.'</p> <br/> <p>מַה נָּעֲמָה תִּקְוָה / עִם אַחֲרִית:</p> <p>הַיּוֹם אֲשֶׁר אִוָּה / שׁוֹמֵר בְּרִית</p> <p>יָבוֹא, וְאוֹר קִוָּה / אוֹר שַׁחֲרִית.</p> <p>‘מִפַּאֲתֵי מִזְרָח / לִי אֶמְשְׁכֵם’</p> <p>יִשְׁלַח דְּבָרוֹ יָהּ / וִיבָרְכֵם.</p></blockquote>
