<h1>אֶתְעוֹרֵר וַאְשׁוֹרֵר יוֹם גִּילִי</h1>
אברהם אבן עזרא
<p>גאולה</p>
<blockquote><br/> <p>אֶתְעוֹרֵר / וַאְשׁוֹרֵר יוֹם גִּילִי,</p></blockquote>
<blockquote><p>גִּילִי רַב / יוֹם יֶעְרַב / הוֹד קוֹלִי,</p></blockquote>
<blockquote><p>קוֹלִי רָם, / כִּי הוּרַם / נֵס דִּגְלִי,</p></blockquote>
<blockquote><p>דִּגְלִי עַל / כֹּל פָּעַל / לִי אֵלִי,</p></blockquote>
<blockquote><p>אֵלִי זֶה, / שֶׁאֶחְזֶה / מִמּוּלִי;</p></blockquote>
<blockquote><p>מִמּוּלִי / מִשְׁאָלִי, / הָיָה לִי,</p></blockquote>
<blockquote><p>לִי – טוּבוֹ / הִצִּיבוֹ, / הִצִּיבוֹ / לָשׂוּשׂ בּוֹ.</p> <br/> <p>בּוֹ שָׂמַח, / בּוֹ צָמַח / צִיץ נִזְרִי,</p></blockquote>
<blockquote><p>נִזְרִי גֵא, / כִּי אֶהֱגֶה / דַת צוּרִי;</p></blockquote>
<blockquote><p>צוּרִי בָּא, / כִּי נִכְבָּה / נֵר צָרִי,</p></blockquote>
<blockquote><p>צָרִי סָר / בַּמּוּסָר / מֵעִירִי –</p></blockquote>
<blockquote><p>עִירִי שָׁב / עִיר נוֹשָׁב, / וּדְבִירִי;</p></blockquote>
<blockquote><p>וּדְבִירִי / כִּדְבָרִי, / כִּדְבָרִי / סָר צִירִי,</p></blockquote>
<blockquote><p>צִירִי רַד – / כֹּל חָרַד, / כָּל חָרַד, / יוֹמוֹ רָד.</p></blockquote>
<blockquote><p>רָד גֵּאֶה, / אַךְ נִכְאֶה / רֹאשׁ הֵרִים</p></blockquote>
<blockquote><p>הֵרִים [……….]</p></blockquote>
<blockquote><p>[……….]</p></blockquote>
<blockquote><p>[……….]</p></blockquote>
<blockquote><p>כִּי עָלֶיךָ הָשְׁלַכְתִּי מֵרֶחֶם.</p> <br/> <p>רֹאשׁ כֹּחִי תַּהֲפֹךְ לְרַע, נִגְלָה מְכַסֵּהוּ,</p></blockquote>
<blockquote><p>וְנִלְקַח מִצְּבָאוֹת – חֵלֶק שָׁוְא יְכַסֵּהוּ;</p></blockquote>
<blockquote><p>לְךָ, יְיָ, חֶסֶד – בְּיִרְאָתְךָ מַחֲסֵהוּ,</p></blockquote>
<blockquote><p>כִּי אַתָּה תְשַׁלֵּם לְאִישׁ כְּמַעֲשֵׂהוּ.</p> <br/> <p>הוּבֵאתִי הֵנָּה לְהִתְבּוֹנֵן עִנְיָנַי,</p></blockquote>
<blockquote><p>נִלְחָם בִּי יִצְרִי, הֱמִיתַנִי בַאֲזֵנָי,</p></blockquote>
<blockquote><p>בּוֹשְׁתִּי מִפְּנֵי אֵל – וָאֶשְׁפֹּךְ תַּחְנוּנַי, כִּי שֹׁמֵעַ אֶל אֶבְיוֹנִים יְיָ.</p></blockquote>
