<h1>[הַבּוֹחֵר בְּשִׁירֵי זִמְרָה]</h1>
אברהם אבן עזרא
<p>רשות לקדיש</p>
<br/>
<blockquote><p>הַבּוֹחֵר בְּשִׁירֵי זִמְרָה / אֶקְרָא, / יִתְגַּדֵּל שְׁמוֹ הַנּוֹרָא!</p> <br/> <p>אֶתְבּוֹנֵן – / וּבַכֹּל שָׁפָר</p> <p>וַאְרַנֵּן / לְאֵל כֹּל סָפָר,</p> <p>וַיְכוֹנֵן / אֱנוֹשׁ מֵעָפָר</p> <p>וּבְלִבּוֹ נְשָׁמָה בָרָה / בָּרָא, / כַּמֶּלֶךְ בְּתוֹךְ הַבִּירָה.</p> <br/> <p>בַּכֹּל הוּא / וּמִיָּדוֹ כֹל,</p> <p>כָּמֹהוּ / הֱיוֹת לֹא יָכֹל,</p> <p>כַּתֹּהוּ / לְנֶגְדּוֹ הוּא כֹל!</p> <p>מָה אֹמַר – וְיַד כָּל נִבְרָא / קָצְרָה, / דַּעְתּוֹ מֵאֱנוֹשׁ נִבְצָרָה?</p> <br/> <p>רָם פִּלְאוֹ / וְסוֹד עֶלְיוֹנָיו,</p> <p>עִם כִּסְאוֹ / מְשָׁרְתֵי פָנָיו,</p> <p>מוֹרָאוֹ / עֲלֵי אוֹפַנָּיו,</p> <p>בַּמַּיִם עֲלִיוֹת קֵרָה, / קָרָא – / שָׁמַיִם בְּרוּחוֹ שִׁפְרָה.</p> <br/> <p>מוֹרָאִים / בְּכָל מַעְבָּדוֹ,</p> <p>כִּי נִרְאִים / מְעִידֵי הוֹדוֹ.</p> <p>לִנְבִיאִים / יְגַלֶּה סוֹדוֹ,</p> <p>כִּי נֶפֶשׁ בְּעֵינָיו יָקְרָה / הוֹרָה / לָלֶכֶת בְּדֶרֶךְ יְשָׁרָה.</p></blockquote>
