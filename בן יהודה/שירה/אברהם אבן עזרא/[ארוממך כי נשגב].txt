<h1>[ארוממך כי נשגב]</h1>
אברהם אבן עזרא
<p>לברכו</p>
<br/>
<blockquote><p>אֲרוֹמִמְךָ, / כִּי נִשְׂגָּב כְּבוֹד שִׁמְךָ / וְאוֹדֶךָּ / כִּי נַפְשִׁי בְיָדֶךָ.</p></blockquote>
<br/>
<blockquote><p>בְּרָאתַנִי / מִשַּׁחַק וּמֵאֲדָמָה,</p></blockquote>
<blockquote><p>בְּנִיתַנִי / בִגְוִיָּה וּבִנְשָׁמָה,</p></blockquote>
<blockquote><p>נְשָׂאתַנִי / עַד הֵנָּה – וְאֵין בַּמָּה</p></blockquote>
<blockquote><p>אֲקַדֶּמְךָ, / רַק בָּאתִי בְנָעָמְךָ / וְעֵדֶיךָ, / כִּי אֵין בִּלְעָדֶיךָ.</p></blockquote>
<br/>
<blockquote><p>רְאִינוּךָ / בְעֵינֵינוּ וְלִבֵּנוּ</p></blockquote>
<blockquote><p>קְרָאנוּךָ – / וְאִם אַתָּה בְקִרְבֵּנוּ,</p></blockquote>
<blockquote><p>מְצָאנוּךָ / בְרוּחֵנו וְגַבֵּנוּ,</p></blockquote>
<blockquote><p>וְתַעְצֻמְךָ / בְאַדְמָתְךָ וּבִמְרוֹמְךָ / יְעִידְךָ, / כִּי הַכֹּל עֲבָדֶיךָ.</p></blockquote>
<br/>
<blockquote><p>מִי יְמַלֵּל / קְצָת מִכֹּל דְּבָרֶיךָ?</p></blockquote>
<blockquote><p>וְאַתְּ מְחוֹלֵל / הַכֹּל בַּעֲבוּרֶךָ,</p></blockquote>
<blockquote><p>וְהַמְהַלֵּל / יִתְהַלֵּל בְּזִכְרֶךָ,</p></blockquote>
<blockquote><p>וְיִרְחָמְךָ / כָּל הַבָּא בְעוֹלָמְךָ, / חֲסִידֶיךָ / יְבָרְכוּ שֵׁם כְּבוֹדֶךָ. <br/></p></blockquote>
