<h1>אֵל נִכְסַפְתִּי לִרְאוֹתֶךָ</h1>
אברהם אבן עזרא
<blockquote><p>אֵל, נִכְסַפְתִּי / לִרְאוֹתֶךָ,</p> <p>עַל כֵּן חַשְׁתִּי / אֶל בֵּיתֶךָ;</p> <p>לַיְלָה קַמְתִּי / לְחַלּוֹתֶךָ –</p> <p>אֶשְׂבְּעָה בְהָקִיץ / תְּמוּנָתֶךָ!</p> <br/> <p>בְּשִׁמְךָ קָרְאוּ / אֶבְיוֹנֶיךָ</p> <p>וְעַיִן נָשְׂאוּ / לִמְעוֹנֶךָ,</p> <p>נְשׂוֹא עֹל נִלְאוּ / בֵּין מוֹנֶיךָ –</p> <p>וְהִנָּם שָׁקְדוּ / עַל דְּלָתֶיךָ.</p> <br/> <p>רְצוֹנְךָ יָזְמוּ, / נוֹרָא עֲלִילָה,</p> <p>וְנֶשֶׁף קִדְּמוּ / לְסַדֵּר תְּפִלָּה,</p> <p>וּמִמְּךָ נִכְלְמוּ / לְבַקֵּשׁ מְחִילָה</p> <p>עַל לֹא שָׁמְרוּ / תוֹרָתֶךָ.</p> <br/> <p>הֲגוֹת מַהֲלָלָךְ / בְּפֶה לֹא חָשְׂכוּ</p> <p>וְעֹצֶם גָּדְלָךְ / בְּקָהָל עָרְכוּ;</p> <p>חֲקֹר מִפְעָלָךְ / לְשׁוֹנָם דָּרְכוּ –</p> <p>וּמִי יַשְׁמִיעַ / תְּהִלָּתֶךָ?</p> <br/> <p>מְשֹׁךְ, אֵל, חֶסֶד / אֶל בָּנֶיךָ,</p> <p>וּמִקְדָּשׁ יַסֵּד / בִּרְצוֹנֶךָ,</p> <p>וְכַלֵּה כֶּשֶׂד / בַּחֲרוֹנֶךָ</p> <p>וְשׁוּר, כִּי בָאוּ בְּנַחְלָתֶךָ.</p> <br/> <p>מְחָרְפֵי שִׁמְךָ – / הֲרֹס שִׁנֵּיהֶם</p> <p>וְנַהֵל עַמְּךָ / מֵאָהֳלֵיהֶם;</p> <p>וְהַרְאֵם עָצְמְךָ / וּפְנֵה אֲלֵיהֶם</p> <p>וְלִי, אֲנִי עַבְדְּךָ בֶּן אֲמָתֶךָ.</p></blockquote>
