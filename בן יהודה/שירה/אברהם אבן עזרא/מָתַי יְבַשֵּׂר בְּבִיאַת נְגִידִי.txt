<h1>מָתַי יְבַשֵּׂר בְּבִיאַת נְגִידִי</h1>
אברהם אבן עזרא
<p>(הבדלה)</p>
<blockquote><p>מָתַי יְבַשֵּׂר / בְּבִיאַת נְגִידִי</p></blockquote>
<blockquote><p>אֵלִיָּהוּ / הַגִּלְעָדִי?</p> <br/> <p>אֲיַחֵל לְיִשְׁעִי / בּוֹא יוֹם בְּשׂוֹרָה,</p></blockquote>
<blockquote><p>וּבְבוֹא רֹעִי / שִׁיר אָעִירָה,</p></blockquote>
<blockquote><p>וּבְהַפְגִּיעִי / בוֹ, אָשִׁירָה</p></blockquote>
<blockquote><p>נָא לִידִידִי / שִׁירַת דּוֹדִי.</p> <br/> <p>בְּאַרְצוֹת עָרַי / נַפְשִׁי יָגְעָה,</p></blockquote>
<blockquote><p>יָרְשׁוּ עָרַי / זֵדֵי רִשְׁעָה,</p></blockquote>
<blockquote><p>שָׁמְעוּ צָרַי / אֵין לִי תְשׁוּעָה –</p></blockquote>
<blockquote><p>וְאַתָּה, יְיָ, / מָגֵן בַּעֲדִי.</p> <br/> <p>רוֹדִים וְצָרִים / תּוֹסִיף לְיַסֵּר</p></blockquote>
<blockquote><p>וִיגוֹן בְּחִירִים / בְּיִשְׁעָךְ הָסֵר,</p></blockquote>
<blockquote><p>וְעַל הֶהָרִים / רַגְלֵי מְבַשֵּׂר</p></blockquote>
<blockquote><p>הָחִישׁ, לְבַשֵּׂר / כִּי בָא מוֹעֲדִי.</p> <br/> <p>הַשָּׂם רְכוּבוֹ / עָבִים וְחֵילוֹ,</p></blockquote>
<blockquote><p>הַבּוֹטְחִים בּוֹ / יָשִׁיב לְחֶבְלוֹ,</p></blockquote>
<blockquote><p>וּפִתְאֹם יָבוֹא / אֶל הֵיכָלוֹ –</p></blockquote>
<blockquote><p>יִשְׂמַח לִבִּי / וְיָגֵל כְּבוֹדִי.</p> <br/> <p>מְדֻכָּה, מְעֻנָּה / וְלִבָּהּ רוֹעֵד –</p></blockquote>
<blockquote><p>הָסֵר יְגוֹנָהּ, / שְׁלַח לָהּ סוֹעֵד,</p></blockquote>
<blockquote><p>כִּי עֵת לְחֶנְנָהּ, / כִּי בָא מוֹעֵד –</p></blockquote>
<blockquote><p>אָז מַקֵּל עֹז / אֶקַּח בְּיָדִי.</p></blockquote>
