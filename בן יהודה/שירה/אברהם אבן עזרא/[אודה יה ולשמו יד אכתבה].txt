<h1>[אודה יה ולשמו יד אכתבה]</h1>
אברהם אבן עזרא
<p>אהבה</p>
<br/>
<blockquote><p>אוֹדֶה יָהּ וְלִשְׁמוֹ יַד אֶכְתְּבָה – / הַבּוֹחֵר בְּעַמּוֹ בָּאַהֲבָה.</p></blockquote>
<br/>
<blockquote><p>     בַּמֶּה אֶתְרַצֶּה אֶל אוֹהֲבִי?</p></blockquote>
<blockquote><p>     קָנַנִי תְּחִלָּה מִבֵּית שְׁבִי</p></blockquote>
<blockquote><p>     וָאֶשְׁכֹּן לְבַדִּי אֶרֶץ צְבִי,</p></blockquote>
<blockquote><p>יָשַׁבְתִּי בְּצִלּוֹ וָאֶשְׁכְּבָה / בֶּטַח, וּשְׁנָתִי לִי עָרְבָה.</p></blockquote>
<br/>
<blockquote><p>     רוּחַ מַחֲלָתִי / מֶנִּי דְחֵה</p></blockquote>
<blockquote><p>     וַעְוֹן מַעֲוָתִי, / אֵל, נָא מְחֵה,</p></blockquote>
<blockquote><p>     וּלְהַדְרַת נְוָתִי / אוֹתִי נְחֵה,</p></blockquote>
<blockquote><p>אוּלַי עֵת יְדִידוּתָךְ קָרְבָה? / כִּי נַפְשִׁי לְפֵרוּדָךְ דָאֲבָה.</p></blockquote>
<br/>
<blockquote><p>     הַלְעוֹלָם תִּרְחַק / בֵּית שִׁבְתְּךָ?</p></blockquote>
<blockquote><p>     צַר לוֹחֵץ וְדוֹחֵק / יוֹנָתְךָ,</p></blockquote>
<blockquote><p>     אֵיךְ אָמָה תְּצַחֵק / עַל בִּתְּךָ?</p></blockquote>
<blockquote><p>עַל מָה בַּת כְּבוּדָה – הִיא שָׁכְבָה / לָאָרֶץ, וְשִׁפְחָתָהּ שָׂגְבָה?</p></blockquote>
<br/>
<blockquote><p>     מַחֲסֶה הוּא לְבָנָיו בּוֹ יֶחֱסוּ.</p></blockquote>
<blockquote><p>     יֹום כְּלוֹת מְשַׂנְּאִים דָּת מָאֲסוּ –</p></blockquote>
<blockquote><p>     בּוֹ אֶבְטַח, וְאוֹיְבַי מַה יַעֲשׂוּ?</p></blockquote>
<blockquote><p>כִּי עָבֵי זְבוּל וּתְהוֹם רָחֲבָה / לֹא יוּכְלוּ לְכַבּוֹת הָאַהֲבָה!</p></blockquote>
