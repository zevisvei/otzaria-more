<h1>אָחַז לְבָבִי</h1>
אברהם אבן עזרא
<p>רשות</p>
<blockquote><br/> <p>אָחַז לְבָבִי בְכַנְפֵי / אַהְבַת אֱלוֹהַּ נַעֲלָה,</p></blockquote>
<blockquote><p>מוֹשֵׁל בְּכָל רַע וְיָפֶה – / עַל כֵּן בְּיָדוֹ אֶפְּלָה.</p></blockquote>
<blockquote><p>הַכֹּל, כְּחָזָק כְּרָפֶה, / יַחַד יְמִינוֹ פָּעֲלָה.</p></blockquote>
<blockquote><p>אִם בּוֹ תִּמְעֲלוּ, / אָוֶן וְשָׁוְא תִּפְעֲלוּ.</p> <br/> <p>בֹּאוּ שְׁעָרָיו בְּתוֹדָה / וּבְשִׁיר לְפָנָיו תַּעְלְזוּ</p></blockquote>
<blockquote><p>וּרְצוֹן אֱלֹהִים לְצֵידָה / תִּקְחוּ, וְאַל נָא תַּחְפְּזוּ,</p></blockquote>
<blockquote><p>כִּי כֵן בְּבוֹא יוֹם קְפָדָה / תִּרְאוּ – וְאָז לֹא תִרְגְּזוּ,</p></blockquote>
<blockquote><p>טוּב אֵל תִּנְחֲלוּ / וּבְצוֹרֲרִים תִּמְשְׁלוּ.</p> <br/> <p>רֹב מַהֲלָלָיו וְשִׂיחָיו / הָעֵת בְּבֵיתוֹ אֶקְרְאָה –</p></blockquote>
<blockquote><p>תַּחַת לְשׁוֹנִי שְׁבָחָיו / כִּדְבַשׁ לְמָתוֹק אֶמְצְאָה.</p></blockquote>
<blockquote><p>נַפְשִׁי לְמֵדָה אֳרָחָיו / כִּי מִכְּבוֹדוֹ נִבְרְאָה.</p></blockquote>
<blockquote><p>הֶבֶל יֶהְבְּלוּ / אִישִׁים שְׁמוֹ חִלֲּלוּ.</p> <br/> <p>הַבֵּן, אֲשֶׁר תּוֹךְ מְעַנָּיו / יוֹחִיל רְצוֹן שֹׁכְנִי סְנֶה,</p></blockquote>
<blockquote><p>אוּלַי בְּקָרְאוֹ לְפָנָיו / יַקְשִׁיב וְשַׁוְעוֹ יַעֲנֶה,</p></blockquote>
<blockquote><p>כִּי כָל יְמוֹתָיו וְשָׁנָיו / מֻכֶּה וְנִגַּשׂ נַעֲנֶה,</p></blockquote>
<blockquote><p>קָמָיו הָעֳלוּ / וּמְאַהֲבָיו שָׁפְלוּ.</p> <br/> <p>מָרוֹם, אֲשֶׁר כָּל בְּרוּאִים / יִשְׁתַּחֲווּ לוֹ יִקְּדוּ,</p></blockquote>
<blockquote><p>לִהְיוֹת לְעָבְדּוֹ קְרוּאִים / חִזְקוּ מְאֹד בַּל תִּמְעֲדוּ,</p></blockquote>
<blockquote><p>וּלְמֵי רְצוֹנוֹ צְמֵאִים / תִּהְיוּ וּבוֹ אַל תִּמְרְדוּ,</p></blockquote>
<blockquote><p>אוֹתוֹ הַלֲּלוּ / וּשְׁמוֹ לְבַד גַּדְּלוּ.</p></blockquote>
