<h1>אִמְרָתְךָ אֵלִי הָקֵם לִשְׁבוּיָה</h1>
אברהם אבן עזרא
<blockquote><p>אִמְרָתְךָ, אֵלִי, / הָקֵם לִשְׁבוּיָה / כִּלְבָבִי:</p></blockquote>
<blockquote><p>‘הִנְנִי שׁוֹלֵחַ / לָכֶם אֵלִיָּה / הַנָּבִיא!’</p> <br/> <p>בַּת צִיּוֹן אֵיכָה / בֵּינוֹת אַכְזָרִים / נֻטָּשָׁה?</p> <p>אוֹ אֵיךְ רַעְיָתְךָ / מֵחֲדַר הוֹרִים / גֹּרָשָׁה?</p> <p>וּנְוֵה קָדְשֶׁךָ / אֵיךְ לִבְנֵי זָרִים / מוֹרָשָׁה?</p> <p>יָרְשׁוּ אֶת אָהֳלִי / מִקֶּדֶם הָיָה / מוֹשָׁבִי —</p> <p>וַאֲנִי בוֹרֵחַ / אֶרֶץ נָכְרִיָּה / בֵּין אוֹיְבִי.</p> <br/> <p>רֹאשִׁי הָרִימָה, / כִּי לְךָ מֵאַהֲבָה / מִשְׁתָּאֶה,</p> <p>וּרְאֵה בֶן אָמָה / עַל בֶּן הָאֲהוּבָה / מִתְגָּאֶה.</p> <p>יָמָּה וָקֵדְמָה / נִשְׁבַּר אוֹ נִשְׁבָּה, / אֵין רוֹאֶה.</p> <p>הִכְבִּידוּ עֻלִּי / בּוֹזֵי תוּשִׁיָּה, / יִרְדּוּ בִי —</p> <p>וַאֲנִי צוֹרֵחַ / נַפְשִׁי בּוֹכִיָּה / אֶל אָבִי.</p> <br/> <p>'מֵרוֹם שָׁמַעְתִּי / קוֹלֵךְ, בַּת עַמִּי, / וָאֶעֱנֶה:</p> <p>יִשְׁעִי קֵרַבְתִּי / לִפְדּוֹתֵךְ לִשְׁמִי, / וָאֶקְנֶה</p> <p>אֶת בֵּית תִּפְאַרְתִּי / הָרַס אִישׁ חֶרְמִי / אֵין בּוֹנֶה.</p> <p>לִמְקוֹם הֵיכָלִי / מִקַּצְוֵי נְשִׁיָּה / תָשׁוּבִי,</p> <p>וּבְלֵב שָׂמֵחַ / הַר הַמּוֹרִיָּה / תִקְרָבִי!'</p></blockquote>
