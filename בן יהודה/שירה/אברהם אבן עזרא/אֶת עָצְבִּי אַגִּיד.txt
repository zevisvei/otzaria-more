<h1>אֶת עָצְבִּי אַגִּיד</h1>
אברהם אבן עזרא
<blockquote><p>אֶת עָצְבִּי / אַגִּיד – וְיֶחֱרַד / אוֹיְבִי וְיֵבְךְּ לִי.</p> <p>אֵיךְ לִבִּי / יִשְׁכֹּן – וְנָע בְּיַד / צָרִים, וּבוֹעֲלִי</p> <p>בָּעַל בִּי? / אֵין לִי צֳרִי – לְבַד / תִּקְוָה לְגוֹאֲלִי.</p> <p>לוּלֵי שִׁמְךָ, חִזְקִי וְחִשְׁקִי, / סַפְתִּי בְּבֵיִת שְׁבִי!</p> <br/> <p>בְּחֶבְלִי / הִשְׁכִּין מְחַבְּלַי / אֵלִי בְּזַעֲמוֹ,</p> <p>בֻּלַּע לִי / מִבּוּז מְקַלֲלַי – לוּ כִבְּדוּ שְׁמוֹ.</p> <p>בְּכִסְלִי / רֶשֶׁף, וְשׁוֹאֲלַי / יִרְאוּ אֲשֶׁר כְּמוֹ</p> <p>חוֹלַת אַהְבָה אָנִי – לְרָחְקִי / מֵאֶרֶץ הַצְּבִי.</p> <br/> <p>רַב שִׂיחִי / מִבּוֹא בְמֶחֱקָר / כִּי רַע מְאֹד וּמַר.</p> <p>מַה כֹּחִי עַתָּה – וְנֶעֱקָר / סוֹד הַיְּסוֹד כְּבָר?</p> <p>אֵיךְ אֶחִי – / כִּי אֶחֱזֶה יְקָר / אֵל מִכְּרוּב לְהַר</p> <p>עוֹלֶה? אַחַר שָׁכַב בְּחֵיקִי / בָּרַח כְּמוֹ צְבִי.</p> <br/> <p>'מָה תֶּהֱמִי / עָלַי בְּלִי דֳמִי? / דֹּמִּי – וְהִנְנִי!</p> <p>יוֹם קוּמִי / פִּתְאֹם תְּנֻחֲמִי. / אָכֵן לְמַעֲנִי</p> <p>מִקָּמִי / אֶקֹּם; וְאֵיךְ שְׁמִי / יַכְעִיס – וְאֵל אֲנִי</p> <p>וְאֵלֶּה בָנַי, בְּנֵי חֲשׁוּקִי, / אַבְרָהָם אוֹהֲבִי.'</p></blockquote>
