<h1>אַשְׁרֵי הָאִישׁ יָרֵא</h1>
אברהם אבן עזרא
<blockquote><p>אַשְׁרֵי הָאִישׁ יָרֵא / מִקּוֹל הַקּוֹרֵא, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>בַּמֵֶּה אִישׁ חָשׁוּב? / יֵלֵךְ וְלֹא יָשׁוּב, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>רוּחוֹ נוֹסְסָה בוֹ – / וַיִּגְבַּהּ לִבּוֹ, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>הוֹלֵךְ לַקֶּבֶר – / וּפִתְאֹם יִשָּׁבֵר, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>מַהֲלָכָיו גְּבֹהִים / כְּמַלְאַךְ הָאֱלֹהִים, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>בְּמוֹתוֹ – כַּבְּהֵמָה / לַהֶבֶל דָּמָה, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>רוֹחֵץ אֶת פָּנָיו / טָהוֹר בְּעֵינָיו, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>מִי יִגַּשׁ אֵלָיו – / וְטֻמְאָתוֹ עָלָיו, / בְּשׁוּבוֹ אֶל הָאֲדָמָה?</p> <p>אֶתְמוֹל קוֹל הֵרִים / בְּשִׂמְחָה וּבְשִׁירִים, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>יוֹדְעוֹ בוֹ יֶהְגֶּה / קִינִים וָהֶגֶה, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>רוֹדֵף טוּב רֵיחַ / וְשֶׁמֶן רוֹקֵחַ, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>בְּלוֹיָיו נִקְרָעִים / וְיָרֻם תּוֹלָעִים, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>נִמְשְׁכָה מַכָּתוֹ – / יַרְבֶּה כְּבוֹד בֵּיתוֹ, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>עָבְרוּ יְמֵי מְהוּמָה – / וְאֵין בְּיָדוֹ מְאוּמָה, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>זְרוֹעַ תִּפְאֶרֶת / וְלָשׁוֹן מְדַבֶּרֶת, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>רַד אֶל בּוֹר עָמַק – / וּלְשׁוֹנוֹ תִּמַּק, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>הוֹד וְהָדָר לְפָנָיו, / יִרְאֶה בְעֵינָיו, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>נָפַל לְמַשּׁוּאוֹת / וְחָשְׁכוּ הָרוֹאוֹת, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p> <p>חָמַד סוֹד רֵעִים / וּמַעֲשֵׂה תַעְתּוּעִים, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>מַה יְּבַקֵּשׁ בְּתֵבֵל? / כִּי הַכֹּל הֶבֶל, / בְּשׁוּבוֹ אֶל אֲדָמָה.</p> <p>תְּסוֹבְבוֹ מִלָּתְךָ / וְיַגִּיד תְּהִלָּתְךָ, / בְּעוֹד יֵשׁ בּוֹ נְשָׁמָה;</p> <p>וְעַל עַצְמוֹ תִגְמֹר – / אַךְ אֶת נַפְשׁוֹ שְׁמֹר, / בְּשׁוּבוֹ אֶל הָאֲדָמָה.</p></blockquote>
