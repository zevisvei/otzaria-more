<h1>[אשא עיני לכל דרך]</h1>
אברהם אבן עזרא
<p>מאורה</p>
<br/>
<blockquote><p>אֶשָּׂא עֵינַי / לְכָל דֶּרֶךְ;</p> <p>אָן מִלְּפָנַי / אָן דּוֹדִי בָּרַח?</p> <br/> <p>בָּרַח דּוֹדִי – וּבִכְרוּבִי / שָׁכַן כְּבוֹדוֹ,</p> <p>מִי יִתֵּן לִי יְמֵי טוּבִי / מוּתִי בְיָדוֹ,</p> <p>טֶרֶם בּוֹא צַר וְיִרְדֶּה בִי / מֵאָז, וְעוֹדוֹ</p> <p>עַל בָּמוֹתַי / בְּאַף דּוֹרֵךְ –</p> <p>כֹּחִי בְמָתְנַי / עָלַי לָטֹרַח.</p> <br/> <p>רַד לִי הַיּוֹם אֲשֶׁר שָׁאוּ / עָרֵי תְהִלָּה,</p> <p>סָפוּ חֹזִים וְנִדְכָּאוּ / רָאשֵׁי סְגֻלָּה,</p> <p>חָלְפוּ יָמַי וְלֹא בָאוּ / יְמֵי גְאֻלָּה.</p> <p>קָצְרָה נַפְשִׁי / בְּרֹב אֹרֶךְ</p> <p>מַסְגֵּר, וְעֵינַי / תָּמִיד לָאֹרַח.</p> <br/> <p>"מִי הִשְׁלִיכֵךְ בְּתוֹךְ זָרִים? / הֲלֹא עֲוֹנֵךְ!</p> <p>שׁוּבִי אֵלַי, וְהֵן אָרִים / נִסִּי לְעֵינֵךְ,</p> <p>מִפָּנַי יִרְעֲשׁוּ הָרִים, / אָבוֹא לְמַעְנֵךְ.</p> <p>קוּמִי אוֹרִי / כִּי בָא אוֹרֵךְ</p> <p>וּכְבוֹד אֲדֹנָי / עָלַיִךְ זָרָח!"</p></blockquote>
