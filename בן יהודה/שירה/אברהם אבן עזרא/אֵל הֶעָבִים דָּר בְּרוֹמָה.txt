<h1>אֵל הֶעָבִים דָּר בְּרוֹמָה</h1>
אברהם אבן עזרא
<p>פיוט לשמיני עצרת</p>
<blockquote><br/> <p>אֵל הֶעָבִים, דָּר בְּרוֹמָה, / צַו: הָרִיקוּ רְבִיבֵיכֶם,</p></blockquote>
<blockquote><p>טַל וּמָטָר, לָאֲדָמָה / בֹּאוּ בְּשָׁלוֹם שְׁנֵיכֶם!</p> <br/> <p>בְּחַסְדְּךָ, דָּר עֲרָבוֹת, / שַׂמַּח בְּגִיל הַלְּבָבוֹת,</p></blockquote>
<blockquote><p>תֵּן טַל וּמָטָר לְהַשְׁתּוֹת / אֶרֶץ וְעוֹרֵר אַהֲבוֹת,</p></blockquote>
<blockquote><p>בָּרֵךְ תְּבָרֵךְ תְּבוּאוֹת / וְתִמָּלֵא תֵבֵל תְּנוּבוֹת.</p></blockquote>
<blockquote><p>רַחוּם, לְנִינֵי אֲיֻמָּה / בַּשֵּׂר: שָׁמַעְתִּי בְּקוֹלְכֶם.</p> <br/> <p>הַמּוֹרִיד שִׁפְעַת רְבִיבִים / דָּבָר מִמֶּנּוּ לֹא נִכְחַד,</p></blockquote>
<blockquote><p>מְכַסֶּה שַׁחַק בֶּעָבִים, / תֵּן טַל וּמָטָר בְּיַחַד</p></blockquote>
<blockquote><p>שְׁנֵיהֶם לָאָרֶץ, טוֹבִים / הַשְּׁנַיִם מִן הָאֶחָד,</p></blockquote>
<blockquote><p>מוֹצִיא לָאוֹר תַּעֲלֻמָה / יִפְתַּח אֲרֻבּוֹת שְׁמֵיכֶם.</p> <br/> <p>הָאֵל, הַנּוֹתֵן לְכָל בָּשָׂר / לֶחֶם, לְכָל דַּל וְאֶבְיוֹן,</p></blockquote>
<blockquote><p>דָּגָן וְתִירוֹשׁ וְיִצְהָר / בָּרֵךְ, מְרוֹמָם וְעֶלְיוֹן,</p></blockquote>
<blockquote><p>תְּמַלֵּא הַגְּרָנוֹת בָּר / בִּבְרָכָה, שׁוֹכֵן וְעֶלְיוֹן,</p></blockquote>
<blockquote><p>חַנּוּן בְּרִית הַקְדוּמָה / יִזְכֹּר בְּרַחֲמָיו אֲלֵיכֶם.</p></blockquote>
