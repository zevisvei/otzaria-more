<h1>רשות לקדיש</h1>
אברהם אבן עזרא
<blockquote><p>הַבּוֹחֵר בְּשִׁירֵי זִמְרָה / אֶקְרָא / יִתְגַּדַּל שְׁמוֹ נוֹרָא!</p></blockquote>
<br/>
<blockquote><p>אֶתְבּוֹנֵן – / וּבַכֹּל שָׁפַר,</p></blockquote>
<blockquote><p>וַאְרַנֵּן / לְאֵל כֹּל סָפַר,</p></blockquote>
<blockquote><p>וַיְכוֹנֵן / אֱנוֹשׁ מֵעָפָר</p></blockquote>
<blockquote><p>וּבְלִבּוֹ נְשָׁמָה בָּרָה / בָּרָא – / כְּמֶלֶךְ בְּתוֹךְ הַבִּירָה.</p></blockquote>
<br/>
<blockquote><p>בַּכֹּל הוּא / וּמִיָּדוֹ כֹל,</p></blockquote>
<blockquote><p>כָּמוֹהוּ / הֱיוֹת לֹא יָכֹל,</p></blockquote>
<blockquote><p>כַּתֹּהוּ / לְנֶגְדּוֹ כֹל,</p></blockquote>
<blockquote><p>מָה אֹמַר? וְיַד כָּל נִבְרָא / קָצְרָה, / דַּעְתּוֹ מֵאֱנוֹשׁ נִבְצָרָה!</p></blockquote>
<br/>
<blockquote><p>רָם פִּלְאוֹ / וְסוֹד עֶלְיוֹנָיו,</p></blockquote>
<blockquote><p>עִם כִּסְאוֹ / מְשָׁרְתֵי פָנָיו,</p></blockquote>
<blockquote><p>מוֹרָאוֹ / עֲלֵי אוֹפַנָּיו,</p></blockquote>
<blockquote><p>בַּמַּיִם עֲלִיּוֹת קֵרָה, / קָרָא – / שָׁמַיִם בְּרוּחוֹ שִׁפְרָה.</p></blockquote>
<br/>
<blockquote><p>מוֹרָאִים / בְּכָל מַעְבָּדוֹ,</p></blockquote>
<blockquote><p>כִּי מַרְאִים / מְעִידֵי הוֹדוֹ.</p></blockquote>
<blockquote><p>לַנְּבִיאִים / יְגַלֶּה סוֹדֹו –</p></blockquote>
<blockquote><p>כִּי נֶפֶשׁ בְּפָנָיו יָקְרָה / הוֹרָה / לָלֶכֶת בְּדֶרֶךְ יְשָׁרָה</p></blockquote>
