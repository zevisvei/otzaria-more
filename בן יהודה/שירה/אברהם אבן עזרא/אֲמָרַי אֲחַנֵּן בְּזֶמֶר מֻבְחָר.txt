<h1>אֲמָרַי אֲחַנֵּן בְּזֶמֶר מֻבְחָר</h1>
אברהם אבן עזרא
<p>סליחה (?)</p>
<blockquote><br/> <p>אֲמָרַי אֲחַנֵּן / בְּזֶמֶר מֻבְחָר,</p></blockquote>
<blockquote><p>עֻזְךָ אֲשַׁנֵּן / בְּגָרוֹן נִחָר,</p></blockquote>
<blockquote><p>חַסְדְּךָ אֲרַנֵּן / עֲלוֹת שַׁחַר,</p></blockquote>
<blockquote><p>וְלִמְאֹד [.]קר / עַד אֵין חֵקֶר –</p></blockquote>
<blockquote><p>יְיָ, בֹּקֶר / תִּשְׁמַע קוֹלִי.</p> <br/> <p>בּוֹחֵן לְבָבוֹת / וְרוֹאֶה סְתָרִים</p></blockquote>
<blockquote><p>וְחוֹפֵשׂ קְרָבוֹת / וְצוֹפֶה חֲדָרִים</p></blockquote>
<blockquote><p>וְרוֹצֶה תְּשׁוּבוֹת / כְּעוֹלוֹת פָּרִים</p></blockquote>
<blockquote><p>צוּר דָּר בְּגָבְהֵי / לא יגהה</p></blockquote>
<blockquote><p>יִשְׁמַע מֵהֵי־/כָלוֹ קוֹלִי.</p> <br/> <p>רֵאשִׁית חִכִּי / כְּזֶבַח הַקְשֵׁב</p></blockquote>
<blockquote><p>וְדִמְעִי כְנִסְכִּי / אָז יֵחָשֵׁב,</p></blockquote>
<blockquote><p>וְכוֹנֵן דַּרְכִּי; / בְּעֻזְּךָ הָשֵׁב</p></blockquote>
<blockquote><p>לְקִרְיַת נְבוֹנַי / קְהִלּוֹת אֱמוּנָי –</p></blockquote>
<blockquote><p>בְּקָרְאִי, יְיָ, / שִׁמְעָה בְקוֹלִי.</p> <br/> <p>מְחִילָה הַמְצֵא / לְשָׁבֵי פֶשַׁע</p></blockquote>
<blockquote><p>וְעַמְּךָ הוֹצֵא / מֵחֵטְא וְרֶשַׁע</p></blockquote>
<blockquote><p>וְנִיבוֹ תִרְצֶה / וְשַׁוְעוֹ תִּשַׁע.</p></blockquote>
<blockquote><p>שְׁמָךְ אַקְדִּישָׁה / בְּנֶפֶשׁ חֲלוּשָׁה,</p></blockquote>
<blockquote><p>עֵקֶב אֲשֶׁר שָׁ־/מַעְתָּ בְּקֹלִי.</p></blockquote>
