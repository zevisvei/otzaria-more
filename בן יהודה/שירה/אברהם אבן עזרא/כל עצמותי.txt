<h1>כל עצמותי</h1>
אברהם אבן עזרא
<h2 id="כלעצמותיראברהםאבן-עזרא">כל עצמותי / ר' אברהם אבן-עזרא</h2>
<br/>
<blockquote><p>כָּל עַצְמוֹתַי תֹּאמַרְנָה: יְיָ / מִי כָמוֹךָ?</p></blockquote>
<br/>
<blockquote><p>אֶפֶס בִּלְתְּךָ בְעֵת צָרָה, / עַל כֵּן לְבָבִי</p></blockquote>
<blockquote><p>בָּטוּחַ בָּךְ: וּמָה אִירָא – / וְאַתְּ סְבִיבִי</p></blockquote>
<blockquote><p>מָגֵן בַּעְדִּי בְיוֹם עֶבְרָה / מִצָּר יְרִיבִי?</p></blockquote>
<blockquote><p>         אַתְּ מַחֲנוֹתַי, / אִישׁ מִלְחֲמוֹתַי,</p></blockquote>
<blockquote><p>         שָׂר רִבְבוֹתַי / מוּל שׂוֹנְאוֹתָי!</p></blockquote>
<blockquote><p>           אַתָּה בְחֵיק מוֹנַי / קֶשֶׁת דְּרוּכָה!</p></blockquote>
<br/>
<blockquote><p>בָּךְ לִדְרֹש אֶת רְצוֹנֶךָ – שִׁמְךָ קְרָאתִיו</p></blockquote>
<blockquote><p>וּבְהִתְנַפֵּל לְפָנֶיךָ – / לִבִּי מְצָאתִיו,</p></blockquote>
<blockquote><p>אֵלֶיךָ אֶל מְעוֹנֶךָ / מַשְׂאֵת נְשָׂאתִיו –</p></blockquote>
<blockquote><p>         כִּי מַחְשְׁבוֹתַי / הֵם מַעְשְׂרוֹתַי</p></blockquote>
<blockquote><p>         וְדַבְּרוֹתַי / הֵם מִזְבְּחוֹתַי,</p></blockquote>
<blockquote><p>              אִשַּׁי וְקָרְבָּנַי – / נַפְשִׁי שְׁפוּכָה.</p></blockquote>
<br/>
<blockquote><p>רוּחִי תַיְשַׁר וְתַנְחֵנִי, / יָהּ, לְעָבְדֶךָ,</p></blockquote>
<blockquote><p>תִּפְקַח לִבִּי וְאֶת עֵינִי / לַחְזוֹת כְּבוֹדֶךָ.</p></blockquote>
<blockquote><p>מַעֲנֵה לָשׁוֹן תְּחָנֵנִי, / הַכֹּל בְּיָדֶךָ.</p></blockquote>
<blockquote><p>         טוּב מִפְעֲלוֹתַי / הֵם קָרְבְּנוֹתַי</p></blockquote>
<blockquote><p>         וּמַתְּנוֹתַי / פְּרִי שְׂפָתָי,</p></blockquote>
<blockquote><p>               מִפִּי וְרַעְיוֹנַי / קַח נָא בְּרָכָה.</p></blockquote>
<br/>
<blockquote><p>מֶלֶךְ אֶחָד אֲשֶׁר אֵלָיו / כָּלוּ נְפָשִׁים,</p></blockquote>
<blockquote><p>כִּי מַהֲלָלָיו – בְּמַעְלָלָיו / רָאוּם דְּרוּשִׁים,</p></blockquote>
<blockquote><p>יוֹשֵׁב קֶדֶם – וּמִפְעָלָיו / כָּל עֵת חֲדָשִׁים:</p></blockquote>
<blockquote><p>         עִם דּוֹר אֲבוֹתַי / וְעִם שְׁנוֹתַי</p></blockquote>
<blockquote><p>         וְתוֹלְדוֹתָי; / כִּי כָל יְמוֹתַי</p></blockquote>
<blockquote><p>               בּוֹ יֶחֱזוּ עֵינַי / אוֹת הַמְּלוּכָה.</p></blockquote>
