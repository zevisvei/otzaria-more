<h1>[על אח בערוני]</h1>
אברהם אבן עזרא
<blockquote><p>עַל אָח בִּעֲרוּנִי / עֶבֶד וְאָח, / בַּיָּם נִעֲרוּנִי.</p></blockquote>
<br/>
<blockquote><p>אֶחֱרוּ זְמַנֵּי / קֵץ מוֹעֲדִי –</p></blockquote>
<blockquote><p>וַאֲנִי מְעֻנֶּה, / צָר צַעֲדִי,</p></blockquote>
<blockquote><p>צָרַי יִשְׂטְמוּנִי, / נָתְנוּ בְחָח / אַפִּי, יַחְמְסוּנִי.</p></blockquote>
<br/>
<blockquote><p>בִּתְּקוּ בְנוֹתַי / וַיִּשְׁרְקוּ,</p></blockquote>
<blockquote><p>יִרְאוּ עַצְּבוֹתַי / וַיִּשְׂחֲקוּ,</p></blockquote>
<blockquote><p>יוֹם יוֹם צוֹדֲדוּנִי, / רַגְלַי בְּפַח – / אֵיךְ לֹא יִלְכְּדוּנִי?</p></blockquote>
<br/>
<blockquote><p>רֹעֲעוּ לְחָיַי / בֵּין אוֹיְבִים</p></blockquote>
<blockquote><p>עַד קַצְתִּי בְּחַיַּי / מִסּוֹבֲבִים:</p></blockquote>
<blockquote><p>אֵלֶּה יִמְשְׁכוּנִי / לֵאמֹר “זְבַח!” / אֵלֶּה יִזְבְּחוּנִי.</p></blockquote>
<br/>
<blockquote><p>מָה אֹמַר? וְאוֹיְבַי / בִּי נִלְחֲמוּ,</p></blockquote>
<blockquote><p>נוֹעֲצוּ מְרִיבַי, / לִי זָעֲמוּ,</p></blockquote>
<blockquote><p>לִשְׂחוֹק יִתְּנוּנִי / יֹאְמרוּ “זְנַח!” / חִנָּם יִרְמְסוּנִי.</p></blockquote>
<br/>
