<h1>אֲמָרַי הַקְשֵׁב</h1>
אברהם אבן עזרא
<p>סליחה</p>
<blockquote><br/> <p>אֲמָרַי הַקְשֵׁב / וְאַל נָא תָשֵׁב / פָּנַי בְּחַסְדֶךָ,</p></blockquote>
<blockquote><p>לְשִׁמְךָ אוֹחִיל / וְלִבִּי יָחִיל / מֵרֹב פְּחָדֶיךָ.</p></blockquote>
<blockquote><p>יָהּ, סָמְכֵנִי / גַּם תָּמְכֵנִי / בְּעָמְדִּי לְפָנֶיךָ –</p></blockquote>
<blockquote><p>וְאַל תִּשְׁכָּחֵנִי!</p></blockquote>
<blockquote><p>עֶזְרָתִי הָיִיתָ, אַל תִּטְּשֵׁנִי / וְאַל תַּעַזְבֵנִי.</p> <br/> <p>בְּנָשְׂאִי עֵינַי / נָפְלוּ פָנַי / מֵרֹב זְדוֹנוֹתַי;</p></blockquote>
<blockquote><p>בְּרֹגֶז רַחֵם / תִּזְכּוֹר, וְנַחֵם / לִי מִיגוֹנוֹתַי,</p></blockquote>
<blockquote><p>מֵהֶם דְּלֵנִי / וְהַצִּילֵנִי / וּכְבֹשׁ עֲוֹנוֹתַי –</p></blockquote>
<blockquote><p>וְאַל תַּדִּיחֵנִי!</p> <br/> <p>רָחֲפוּ עֲצָמַי / מֵרֹב אֲשָׁמַי / וְלִמְאֹד נִכְלַמְתִּי,</p></blockquote>
<blockquote><p>וּמֵעַי הָמוּ / לְעֵת בִּי קָמוּ, / דּוּמָם נֶאֱלַמְתִּי,</p></blockquote>
<blockquote><p>וְאֵין לִי פֹעַל / לְכַפֵּר מַעַל, / וּמֵרְאוֹת נֶעֱלַמְתִּי</p></blockquote>
<blockquote><p>יוֹם תּוֹכִיחֵנִי.</p> <br/> <p>הִנֵּה דְּמָעַי / לְגֹדֶל פְּשָׁעַי / עֵינִי שׁוֹטֶפֶת,</p></blockquote>
<blockquote><p>וְנֶפֶשׁ נְמוּכָה / וּבִשְׁמָךְ תְּמוּכָה / בְּשִׂיחָה עוֹטֶפֶת;</p></blockquote>
<blockquote><p>סְלַח שִׁגְיוֹנָהּ / בְּחִין הֶגְיוֹנָהּ / לְמוּלָךְ נוֹטֶפֶת –</p></blockquote>
<blockquote><p>עֲצָתְךָ תַּנְחֵנִי.</p></blockquote>
<blockquote><p>מְחִילָה אֲשַׂבֵּר / וְלָכֵן לְדַבֵּר / הִנֵּה נָא הוֹאַלְתִּי:</p></blockquote>
<blockquote><p>תְּמֹךְ לִי פַעַם / אַךְ הַפַּעַם / אוֹתָהּ שָׁאַלְתִּי,</p></blockquote>
<blockquote><p>כַּפֵּר פִּשְׁעִי / וְשָׂא אֶת רִשְׁעִי / וַאֲשֶׁר נוֹאַלְתִּי –</p></blockquote>
<blockquote><p>וְאַל תִּזְנָחֵנִי!</p></blockquote>
