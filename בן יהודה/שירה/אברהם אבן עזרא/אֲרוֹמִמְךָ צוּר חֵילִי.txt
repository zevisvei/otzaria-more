<h1>אֲרוֹמִמְךָ צוּר חֵילִי</h1>
אברהם אבן עזרא
<blockquote><p>אֲרוֹמִמְךָ, צוּר חֵילִי, / וְנֹעַם שִׁבְחֲךָ אֲיַפֶּה,</p> <p>יְיָ, בֹּקֶר תִּשְׁמַע קוֹלִי, / בֹּקֶר אֶעֱרָךְ לְךָ וַאֲצַפֶּה.</p> <br/> <p>בְּעֵין שִׂכְלִי אֶחֱזֶךָ / וְאָחִיל מִפַּחְדֶּךָ</p> <p>וַאֲנִי אָשִׁיר עֻזֶּךָ / [וַאֲרַנֵּן] לַבֹּקֶר חַסְדֶּךָ.</p> <br/> <p>רַחוּם, סְמוֹךְ עַבְדֶךָ / בְּעָמְדוֹ לְסַלֵּד בְּחִילוֹת,</p> <p>לְהַגִּיד בַּבֹּקֶר חַסְדֶּךָ / וְאֱמוּנָתְךָ בַּלֵּילוֹת.</p> <br/> <p>הִנְנִי, דָּר שָׁמַיִם, / לְמֵי סְלִיחָתְךָ צָמֵא,</p> <p>עֶרֶב וָבֹקֶר וְצָהֳרַיִם / אָשִׂיחָה וְאֶהֱמֶה.</p> <br/> <p>מַהֲלָלְךָ הִבַּעְתִּי / עֲלוֹת שַׁחַר לְר[…]</p> <p>וַאֲנִי אֵלֶיךָ יְיָ שִׁוַּעְתִּי / וּבַבֹּקֶר תְּפִלָּתִי תְקַדְּמֶךָּ.</p> <br/> <p>חֲסָדֶיךָ מַה […]ם / וּפִלְאֵי גְבוּרָתֶךָ –</p> <p>[חֲ]דָשִׁים לַבְּקָרִים, / רַבָּה אֱמוּנָתֶךָ.</p> <br/> <p>‘זַךְ, פְּלָאָיו לְאֵין חֵקֶר’ / יְעִידוּן דַּפֵּי גְבוֹהִים –</p> <p>בְּרָן יַחַד כּוֹכְבֵי בֹקֶר / [וַ]יָּרִיעוּ כָּל בְּנֵי אֱלֹהִים.</p> <br/> <p>קַמְתִּי כְדַל נֶגְדֶּךָ / וְכַפַּי לְךָ שִׁטַּחְתִּי:</p> <p>הַשְׁמִיעֵנִי בַבֹּקֶר חַסְדֶּךָ, / כִּי בְךָ בָטָחְתִּי!</p></blockquote>
