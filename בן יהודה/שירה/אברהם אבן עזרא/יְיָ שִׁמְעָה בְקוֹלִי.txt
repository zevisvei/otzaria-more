<h1>יְיָ שִׁמְעָה בְקוֹלִי</h1>
אברהם אבן עזרא
<p>רשות (?)</p>
<blockquote><br/> <p>יְיָ, שִׁמְעָה בְקוֹלִי, / רְאֵה עָנְיִי וַעֲמָלִי</p></blockquote>
<blockquote><p>וְכַפֵּר חֶטְאִי וּמַעְלִי, / אֱלֹהִים יְיָ חֵילִי.</p> <br/> <p>בְּשָׁפְכִי לְמוּלְךָ זַעֲקִי / לְרַצּוֹתְךָ, צוּר חִזְקִי,</p></blockquote>
<blockquote><p>עֲנֵנִי, אֱלֹהֵי צִדְקִי, / בַּצָּר הִרְחַבְתָּ לִי.</p> <br/> <p>רְצוֹנֶךָ הֲבִינֵנִי / וְאַל בְּקֶצְפְּךָ תְּדִינֵנִי,</p></blockquote>
<blockquote><p>שְׁמַע, יְיָ, וְחָנֵּנִי, / יְיָ הֱיֵה עֹזֵר לִי.</p> <br/> <p>מָעוּזִי בְּיוֹם עֶבְרָה, / מְנוּסִי בְּיוֹם צָרָה,</p></blockquote>
<blockquote><p>יְיָ לִי לֹא אִירָא – / מַה יַּעֲשֶׂה אָדָם לִי?</p> <br/> <p>חַנּוּן, אֶבְיוֹן בְּמַהְלָלוֹ / יְרוֹמִמְךָ כְּפִי יָכְלוֹ,</p></blockquote>
<blockquote><p>כִּי דִּלִּיתָנִי וְלֹא / שִׂמַּחְתָּ אֹיְבַי לִי.</p> <br/> <p>זְמִירוֹת רַנֲנוּ הֲמוֹנַי / לְצוּר, כִּי יִשְׁמַע רְנָנַי,</p></blockquote>
<blockquote><p>אֲרַנֵּן: בָּרוּךְ יְיָ / כִּי הִפְלִיא חַסְדּוֹ לִי.</p> <br/> <p>קָדוֹשׁ, פְּקַח עֵינֶיךָ / וְהַטֵּה נָא אָזְנֶיךָ</p></blockquote>
<blockquote><p>לְמִתְחַנֵּן לְמוּלֶךָ, / יְיָ צוּרִי וְגֹאֲלִי.</p></blockquote>
