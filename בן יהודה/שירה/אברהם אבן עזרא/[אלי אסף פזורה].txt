<h1>[אלי אסף פזורה]</h1>
אברהם אבן עזרא
<p>אהבה</p>
<br/>
<blockquote><p>אֵלִי, אֱסֹף פְּזוּרָה / בֵּין פֶּרֶץ וְלָבִיא –</p></blockquote>
<blockquote><p>הָאוֹמֵר בִּגְבוּרָה / לַצּוּלָה: חֳרָבִי!</p></blockquote>
<br/>
<blockquote><p>בַּמֶּה יְקַו רְצוֹנָךְ / בֵּן רַבּוּ כְעָסָיו?</p></blockquote>
<blockquote><p>אֵיךְ יַעֲבִיר חֲרוֹנָךְ / בִּדְבָרוֹ וּמַעֲשָׂיו?</p></blockquote>
<blockquote><p>הָהּ עַל הֱיוֹת הֲמוֹנָךְ / בֵּין עֶרֶב וְעֵשָׂיו –</p></blockquote>
<blockquote><p>מְאַבְּדֵי חֹק וְתוֹרָה / מִכֹּהֵן וְנָבִיא,</p></blockquote>
<blockquote><p>וּבְשֶׁקֶר וְסָרָה / כָּל עֵת יַעֲנוּ בִי.</p></blockquote>
<br/>
<blockquote><p>רַחֵם לְאֹם בְּחַרְתּוֹ / מִכָּל עַם בְּתֵבֵל,</p></blockquote>
<blockquote><p>וּבְאַהֲבָה מְשַׁכְתּוֹ / לָךְ נַחֲלָה וְחֶבֶל.</p></blockquote>
<blockquote><p>אֵיכָה בְאַף נְתַתּוֹ / נוֹשֵׂא עֹל וְסֵבֶל?</p></blockquote>
<blockquote><p>מָתַי תָּקוּם לְעֶזְרָה, / אֵל יוֹשֵׁב כְּרוּבִי?</p></blockquote>
<blockquote><p>וּלְחוֹמָה בְצוּרָה / תִּהְיֶה נָא סְבִיבִי!</p></blockquote>
<br/>
<blockquote><p>'הִתְנַחֲמוּ, יְלָדִים / מִקֶּדֶם אֲהַבְתִּים,</p></blockquote>
<blockquote><p>כִּי בַעֲבוּר חֲסָדִים / הֵן חֶסֶד מְשַׁכְתִּים</p></blockquote>
<blockquote><p>וְלֶאֱסֹף נְדוּדִים – מַלְאָכַי שְׁלַחְתִּים!</p></blockquote>
<blockquote><p>עַל צַר אָקוּם בְּעֶבְרָה – / וּלְזֶרַע קְרוֹבִי</p></blockquote>
<blockquote><p>אֶהְיֶה לַעֲטָרָה / וּלְכָבוֹד וְלִצְבִי.</p></blockquote>
<br/>
<blockquote><p>מָחָר, צִיּוֹן, יְשׁוּעָה / אֶתֵּנָה לְעַמִּי,</p></blockquote>
<blockquote><p>וּזְרוֹעִי יְדוּעָה / אֶחְשֹׂף, וְזַעְמִי</p></blockquote>
<blockquote><p>אֶשְׁפֹּךְ בְּאַף וְרָעָה / עַל אוֹיְבִי וְקָמִי –</p></blockquote>
<blockquote><p>וּפְדוּת אָחִישׁ מְהֵרָה / אֶל בָּנִים בְּשֶׁבִי,</p></blockquote>
<blockquote><p>נִין אַב הֲמוֹן וְשָׂרָה, / בִּתִּי וַאֲהוּבִי.' <br/></p></blockquote>
