<h1>מוחרך לברכו</h1>
אברהם אבן עזרא
<blockquote><p>כָּל אֲשֶׁר / יוּכַל לְהִתְרַצּוֹת / אֶל אֵל בְּנָאֳמו –</p> <p>גַּדְּלוּ / לַיָי אִתִּי / וּנְרוֹמְמָה שְׁמוֹ! <br/><br/> אַחֲלֵי / לִבִּי שֶׁיֶּחְזֶה / שֵׁם אֵל בְּרָאוֹ.</p> <p>בַּעֲלֵי / לֵב בִּקְּשׁוּ רָזֵי / הַשֵּׁם וּפִלְאוֹ,</p> <p>יַעֲלֶה / עַל נִיב אִישׁ חוֹזֶה, / ‘אֶהְיֶה’ קְרָאוֹ,</p> <p>וַעֲלֵי / דָּתוֹ נִכְתַּב: זֶה / זִכְרוֹ וְזֶה שְׁמוֹ. <br/><br/> בּוֹעֲרִים / אֵיךְ לֹא יַכִּירוּ / שֵׁם צוּר יְלָדָם?</p> <p>סוֹרֲרִים / אֵיךְ לֹא יַיְשִׁירוּ / אֶת מַעֲבָדָם?</p> <p>סוֹחֲרִים / בִּזְבוּל יָעִירוּ: / אַיֵּה כְבוֹדָם?</p> <p>אוֹמְרִים / לִבְנֵי אִישׁ: שִׁירוּ / לוֹ, בָּרְכוּ שְׁמוֹ! <br/><br/> רָשְׁמוּ / צוּרוֹת נִפְרָדוּ / חֻקֵּי אֱמוּנוֹת,</p> <p>עָצְמוּ / עַד לֹא יֻגָּדוּ, / וּבְרֹב תְּבוּנוֹת</p> <p>חָכְמוּ, / כִּי בָם יֻלָּדוּ / כָּל הַתְּמוּנוֹת –</p> <p>יִנְאֲמוּ / זֶה לָזֶה: הוֹדוּ / לוֹ, בָּרְכוּ שְׁמוֹ. <br/></p></blockquote>
