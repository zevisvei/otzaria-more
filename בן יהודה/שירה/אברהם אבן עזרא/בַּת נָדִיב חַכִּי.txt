<h1>בַּת נָדִיב חַכִּי</h1>
אברהם אבן עזרא
<blockquote><p>בַּת נָדִיב, חַכִּי / אֶל צוּר מִשְׂגַּבֵּךְ</p> <p>וְיִגְדַּל שְׁמֵךְ, כִּי / גָּדוֹל בְּקִרְבֵּךְ!</p> <br/> <p>אֵימִים מֵחֲדָרָיו / לִבְשִׁי בְחִילָה</p> <p>וְלַעֲשׂוֹת מַאֲמָרָיו / חוּשִׁי בְגִילָה,</p> <p>בֹּאִי שְׁעָרָיו / בְּתוֹדָה וּתְהִלָּה –</p> <p>רַחֲצִי, הִזַּכִּי, / טַהֲרִי אֶת חֻבֵּךְ!</p> <br/> <p>בַּחֲסָדָיו יַחְלִים / בֵּן חוֹלֶה תְלָאוֹת,</p> <p>וְיחְפִּיר וְיַכְלִים / גּוֹי מַקְנִיא קְנָּאוֹת,</p> <p>וְעַל אֵל אֵלִים / יְדַבֵּר נִפְלָאוֹת –</p> <p>וְכָעֵת תָּשֹׁכִּי / מֵרֻגְזֵךְ וְעָצְבֵּךְ.</p> <br/> <p>'רְאִי, כִּי פָתַרְתִּי / אֶת סוֹד חֲלוֹמֵךְ,</p> <p>וְהִתְגַּדִּלְתִּי / וָאָרוּם לְרַחְמֵךְ.</p> <p>בְּרוּכָה אַתְּ, בִּתִּי / וּבָרוּךְ טַעֲמֵךְ –</p> <p>לָמֶה תִבְכִּי / וְיֵרַע לְבָבֵךְ?</p> <br/> <p>מָה יִּתְרוֹן לִשְׁמִי / בְנוּדֵךְ וּמַה יִּסְכֹּן?</p> <p>אֲחוֹתִי, אֶל מְקוֹמִי / תָּשׁוּבִי וְיִכּוֹן</p> <p>כִּסְאִי וַהֲדוֹמִי / וּבְתוֹכֵךְ אֶשְׁכֹּן</p> <p>וּבְסֻכֵּךְ – סֻכִּי, / מְסִבִּי – בִמְסִבֵּךְ.'</p></blockquote>
