<h1>אוֹר פְּנֵי מֶלֶךְ יְקַו</h1>
אברהם אבן עזרא
<h2 id="אוֹרפְּנֵימֶלֶךְיְקַוראברהםאבן-עזרא">אוֹר פְּנֵי מֶלֶךְ יְקַו / ר' אברהם אבן-עזרא</h2>
<p><strong>[אהבה]</strong></p>
<br/>
<blockquote><p>אוֹר פְּנֵי / מֶלֶךְ יְקַו נִגַּשׂ וְהוּא נַעֲנֶה,</p></blockquote>
<blockquote><p>יַעֲנֶה / מִשֹּׁד מְעַנֶּה לָךְ – וְאֵין מַעֲנֶה,</p></blockquote>
<blockquote><p>יַחֲנֶה / בֵּין לוֹהֲטִים מַחְנֶה לְמוּל מַחְנֶה,</p></blockquote>
<blockquote><p>נִצְּבוּ / לָרִיב וְנֶפֶשׁ נַעֲנָה עִצְּבוּ,</p></blockquote>
<blockquote><p>יֵשְׁבוּ / לֶאְרֹב אֲבָל חִנָּם לְדָם אָרְבוּ.</p></blockquote>
<br/>
<blockquote><p>בֵּית מְרִי / יִתְנַשְּׂאוּ עַל רוּם כְּדֹב עַל מְרִיא,</p></blockquote>
<blockquote><p>כַּאֲרִי / כֵּן שִׁבְּרוּ עַצְמִי בְּשֶׁצֶף חֳרִי,</p></blockquote>
<blockquote><p>עֵץ פְּרִי / אָמְרוּ לְהַשְׁחִית עַד לְאֵין עוֹד צֳרִי;</p></blockquote>
<blockquote><p>כִּזְּבוּ / בִּדְבַר אֱמֶת וּכְתָב אֱמֶת עָזְבוּ,</p></blockquote>
<blockquote><p>כָּתְבוּ / עָלַי מְרוֹרוֹת מֵאֲשֶׁר גָּנְבוּ.</p></blockquote>
<br/>
<blockquote><p>'רַב לְךָ, / הוֹמֶה, לְהַשְׁמִיעַ הֲמוֹן קוֹלְךָ,</p></blockquote>
<blockquote><p>חִילְךָ / אַסְתִּיר וְנָגֹלּוּ שְׁנוֹת גִּילְךָ,</p></blockquote>
<blockquote><p>דִּגְלְךָ / אָרִים עֲדֵי מִיָּם לְיָם מָשְׁלְךָ;</p></blockquote>
<blockquote><p>תִּשְׂגְּבוּ, / עַמִּי, וְעַל לֵב רוֹכְבִים תִּרְכְּבוּ,</p></blockquote>
<blockquote><p>תִּשְׁכְּבוּ / בֶּטַח וּמֵי יֶשַׁע וְגִיל תִּשְׁאֲבוּ.</p></blockquote>
<br/>
<blockquote><p>מַחֲלָה / אֶרְפָּא וְאָסִיר צִיר בְּיוֹם נַחֲלָה,</p></blockquote>
<blockquote><p>אֶגְאֲלָה / גּוֹלִים בְּבוֹר מוֹרְאָה וְגַם נִגְאֲלָה,</p></blockquote>
<blockquote><p>מַעֲלָה / תַעְלוּ וְאַנְחִילְכֶם צְבִי נַחֲלָה;</p></blockquote>
<blockquote><p>תִּרְהֲבוּ / בִשְׁמִי וְלִי כָבוֹד וְעֹז תֶּהֱבוּ,</p></blockquote>
<blockquote><p>אֶהֱבוּ / לַעְבֹד עֲבוֹדָתִי – וְתֵאָהֲבוּ.'</p></blockquote>
