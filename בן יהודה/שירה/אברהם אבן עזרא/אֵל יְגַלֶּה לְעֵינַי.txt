<h1>אֵל יְגַלֶּה לְעֵינַי</h1>
אברהם אבן עזרא
<blockquote><p>אֵל יְגַלֶּה לְעֵינַי / מַעֲבָדָיו,</p> <p>נוֹדְעָה יַד אֲדֹנָי / אֶל עֲבָדָיו.</p> <br/> <p>בּוֹ לְבַדוֹ אַאְמִין; / אֶדְרְשֵׁהוּ,</p> <p>אֶלְמְדָה מַעְשַׁי מִן / מַעֲשֵׁהוּ.</p> <p>אַשְׂמְאִיל אוֹ אֵימִין — / אֶפְגְּשֵׁהוּ,</p> <p>וַעֲלֵי רַעְיוֹנַי / שָׁם פְּקִידָיו,</p> <p>נִצְּבוּ עַל פָּנַי / סוֹד מְעִידָיו.</p> <br/> <p>רָחֲבָה לֹא תִמַּד / דָּת יְקָרוֹ.</p> <p>דֹּק וְחוּגוֹ עָמַד / עַל דְּבָרוֹ.</p> <p>מִשְׁפְּטֵי פִיו לָמַד / עַם בְּחָרוֹ —</p> <p>בָּא אֲזַי מִסִּינַי / עִם גְּדוּדָיו,</p> <p>וַעֲדֶנָּה בְאָזְנַי / קוֹל פְּחָדָיו.</p> <br/> <p>מָה אֲבַקֵּשׁ עוֹד לִי? / רַק רְצוֹנוֹ!</p> <p>כִּי יְסוֹד מִשְׁאָלִי — / הוּא יְבִינוֹ,</p> <p>וַאֲשֶׁר הוּא טוֹב לִי — / הוּא יְכִינוֹ.</p> <p>הֵן לְזֹאת הֶגְיוֹנַי / אֶל יְדִידָיו:</p> <p>אֶהֱבוּ אֶת אֲדֹנָי / כָּל חֲסִידָיו!</p> <br/> <p>מִפְלְאוֹתָיו — כַּמָּה / אִישׁ יְבִינֵם?</p> <p>מַעֲלָתָם עָצְמָה, / מִי יְשַׁנֵּם?</p> <p>מִצְעֲדֵי אִישׁ חָכְמָה — / הוּא יְכִינֵם.</p> <p>יוֹם וְלֵיל הֶגְיוֹנַי / נִיב יְדִידָיו:</p> <p>אֶהֱבוּ אֶת אֲדֹנָי / כָּל חֲסִידָיו!</p></blockquote>
