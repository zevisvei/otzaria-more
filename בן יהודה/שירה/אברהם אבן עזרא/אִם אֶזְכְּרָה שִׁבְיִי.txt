<h1>אִם אֶזְכְּרָה שִׁבְיִי</h1>
אברהם אבן עזרא
<blockquote><p>אִם אֶזְכְּרָה שִׁבְיִי / עֵת נְדוֹד גְּבִיר –</p></blockquote>
<blockquote><p>יִזַּל כַּטַּל בִּכְיִי / וּנְהִי אַחֲבִיר.</p></blockquote>
<blockquote><p>לֹא אֹמְרָה דַיִּי – / עַד בְּנוֹת דְּבִיר</p></blockquote>
<blockquote><p>קָדְשִׁי וְשׁוּב אוֹהֲבִי / אֶל מִשְׁכְּנוֹת הַצְּבִי.</p></blockquote>
<blockquote><p>וְעַד אָן גּוֹאֲלִי / יְרוֹמֵם שׁוֹלֲלִי / וּמַחֲרִיב אָהֳלִי</p></blockquote>
<blockquote><p>וְטִמֵּא אֶת נְוֵה / מְעוֹן שׁוֹכְנִי סְנֶה? <br/><br/> בָּנוּי לְתַלְפִּיּוֹת / מִגְדַּל הַיְּקָר,</p></blockquote>
<blockquote><p>שָׁכַן פְּרִיץ חַיּוֹת / בּוֹ וְנֶעֱקָר.</p></blockquote>
<blockquote><p>אֵיךְ נִקְבְּצוּ דַיּוֹת / שָׁם? וְנֶחֱקָר</p></blockquote>
<blockquote><p>תֹּם קֵץ, וְהִנֵּה נְבִיא / הָאֵל, וְאוֹמֵר: 'חֲבִי</p></blockquote>
<blockquote><p>מְעַט קָט, יַחֲלִי, / וְטוּב אֵל יַחֲלִי, / וְאָז תִּנָּצְלִי;</p></blockquote>
<blockquote><p>וְצוֹעֵק מִדְּוֵה / לְבָבוֹ אֶעֱנֶה!' <br/><br/> רַפֵּא מְשׁוּבָתִי, / רָם וְנֶאֱדָר,</p></blockquote>
<blockquote><p>הָשֵׁב אֱלֵי בֵיתִי / סֹחֶרֶת וְדָר,</p></blockquote>
<blockquote><p>וֶאְסֹף קְהִלָּתִי – / אִישׁ לֹא נֶעֱדָר.</p></blockquote>
<blockquote><p>הַצּוּר, בְּיוֹם רוֹכְבִי – / הַשְׁמֵד מְעוֹן אוֹיְבִי!</p></blockquote>
<blockquote><p>וְתִסְלַח מַעֲלִי / וְתִהְיֶה בוֹעֲלִי / וְתַחְפֹּץ גּוֹאֲלִי,</p></blockquote>
<blockquote><p>וְשָׂא כָל מַעֲוֶה, / וְתוֹךְ בֵּיתָךְ חֲנֵה! <br/><br/> מָתַי תְּנַחֲמֵנִי, / צוּר מְחוֹלֲלִי?</p></blockquote>
<blockquote><p>מָתַי תְּרוֹמֲמֵנִי / עַל מְקַלֲלִי?</p></blockquote>
<blockquote><p>מָתַי תְּקַיְּמֵנִי / מִשַּׁחַת בְּלִי?</p></blockquote>
<blockquote><p>וּשְׁלַח פְּדוּת בַּעֲבִי / וּפְדֵה אֲסִיר מִשְּׁבִי,</p></blockquote>
<blockquote><p>וְתִרְפָּא מַחֲלִי / וְתִבְחַל בּוֹחֲלִי / וְתִגְעַל גּוֹעֲלִי –</p></blockquote>
<blockquote><p>וּמִדָּמָם רְוֵה, / וְהַר קָדְשִׁי בְּנֵה! <br/></p></blockquote>
