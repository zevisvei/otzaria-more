<h1>שַבָּת וְיוֹם חַג יִגָּשׁוּ</h1>
אברהם אבן עזרא
<p>(פיוט לשמיני־עצרת החל בשבת)</p>
<blockquote><p>שַׁבָּת וְיוֹם חַג יִגָּשׁוּ, / אִישׁ בְּאָחִיהוּ יְדֻבָּקוּ,</p></blockquote>
<blockquote><p>חֶסֶד וֶאֱמֶת נִפְגָּשׁוּ, / צֶדֶק וְשָׁלוֹם נָשָׁקוּ.</p> <br/> <p>אָז הוֹד שְׁבִיעִי לְאָב תָּמִים / וּשְׁבִיעִי לְזֶרַע אֲנָשִׁים</p></blockquote>
<blockquote><p>גַּם הַשְּׁבִיעִי בַּיָּמִים, / גַּם הַשְּׁבִיעִי בֶּחֳדָשִׁים,</p></blockquote>
<blockquote><p>וְשָׁנָה שְׁבִיעִית לְעוֹלָמִים – / מִפִּי קָדוֹשׁ מְקֻדָּשִׁים.</p></blockquote>
<blockquote><p>מְקַדְּשֵׁי קֹדֶשׁ יְקֻדָּשׁוּ / וּמַצְדִּיקֵי דָת יִצְדָּקוּ.</p> <br/> <p>בְּסוֹף שֶׁבַע שַׁבְּתֹת שָׁנִים – / שָׁנָה שְׁמִינִית הִיא קֹדֶשׁ,</p></blockquote>
<blockquote><p>גַּם יוֹם שְׁמִינִי – לַבָּנִים, / אָדָם וּבְהֵמָה לְקַדֵּשׁ,</p></blockquote>
<blockquote><p>וּבַשְּׁמִינִי נִקְרְאוּ כֹּהֲנִים / אַחַר מָלְאוּ יְמֵי קֹדֶשׁ –</p></blockquote>
<blockquote><p>שִׁבְעַת יָמִים לֹא מָשׁוּ, / וּבַשְּׁמִינִי הִתְחַזָּקוּ.</p> <br/> <p>רְחָבָה תוֹרַת אֵל וְטוֹבָה / לְטוֹבִים, אֲשֶׁר דַּרְכָּהּ שָׁמְרוּ,</p></blockquote>
<blockquote><p>וּמְצֹרָע וָזָב וְזָבָה / בַּיּוֹם הַשְּׁמִינִי יִטְהֲרוּ,</p></blockquote>
<blockquote><p>וְחֻקַּת הַבַּיִת כְּתוּבָה: / בַּשְּׁמִינִי יְחַטְּאוּ וְכִפְּרוּ</p></blockquote>
<blockquote><p>מְשָׁרְתִים, הָאֱמֶת דָּרָשׁוּ / וּמֵעֲבוֹדַת־שָׁוְא רָחָקוּ.</p> <br/> <p>מְרוּדִים, הַחֲפֵצִים קִרְבַת / אֱלֹהִים, יִהְיוּ לַעֲטֶרֶת,</p></blockquote>
<blockquote><p>כִּי שָׁמְרוּ כַבֵּן כַּבַּת / בְּגָלוּתָם כָּל מִשְׁמֶרֶת –</p></blockquote>
<blockquote><p>הוּא יוֹם שְׁבִיעִי וְשַׁבָּת, / גַּם יוֹם שְׁמִינִי חַג עֲצֶרֶת –</p></blockquote>
<blockquote><p>שָׂשׂוֹן וְשִׂמְחָה יִלְבָּשׁוּ, / יִתְעַנְּגוּ גַּם יִשְׂחָקוּ.</p></blockquote>
