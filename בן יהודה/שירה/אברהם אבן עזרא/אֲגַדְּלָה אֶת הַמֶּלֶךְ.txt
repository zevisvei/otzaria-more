<h1>אֲגַדְּלָה אֶת הַמֶּלֶךְ</h1>
אברהם אבן עזרא
<p>רשות (לקדיש)</p>
<blockquote><br/> <p>אֲגַדְּלָה אֶת הַמֶּלֶךְ / אֲשֶׁר כֹּל פָּעָל,</p></blockquote>
<blockquote><p>וְאֶעֱרֹךְ שִׁיר נָעִים, / שְׁמוֹ יִתְגַּדָּל.</p> <br/> <p>בְּסוֹד רְכוּבוֹ תָּמִיד / תְּנוּ לוֹ מַהְלָל</p></blockquote>
<blockquote><p>וְעַם קְרוֹבוֹ פִּיו מִקַּ-/דְּשׁוֹ לֹא יֶחְדָל.</p> <br/> <p>רְאֵה עֱנוּתוֹ, סֹב וּדְמֵה / לְעֹפֶר אַיָּל,</p></blockquote>
<blockquote><p>וּמֵאֲזַי אַתָּה מִגְדָּל / וּמָעוֹז לַדָּל.</p> <br/> <p>הֲלֹא בְשִׁבְיוֹ, אִם בֵּין עַם / וְעַם יִתְבּוֹלָל –</p></blockquote>
<blockquote><p>בְּרִית אֱמוּנָתְךָ נָצַר / לְךָ רַק נִבְדָּל.</p> <br/> <p>מְלֹךְ, וְלָמָּה תִהְיֶה לִי / כְּאֵין אוֹן וֶאֱיָל?</p></blockquote>
<blockquote><p>אֲבָל כֹּחֲךָ גָּדוֹל, צוּר / וְעַתָּה יִגְדָּל.</p></blockquote>
