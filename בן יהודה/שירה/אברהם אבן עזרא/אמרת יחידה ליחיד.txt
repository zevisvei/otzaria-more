<h1>אמרת יחידה ליחיד</h1>
אברהם אבן עזרא
<blockquote><p>אִמְרַת יְחִידָה לְיָחִיד יָאֲתָה, // נַפְשִׁי לְצוּרָה / נִכְסְפָה וְגַם כָּלְתָה.</p></blockquote>
<br/>
<blockquote><p>בַּעְמֹד פְּעָלִים בְּכֹחַ אוֹנָהּ</p></blockquote>
<blockquote><p>אוֹת עַל הֱיוֹתָהּ, וְחַיָּה יֶשְׁנָהּ,</p></blockquote>
<blockquote><p>אֹמֶן יְסוֹד לָהּ וּמִקְרֶה אֵינָהּ.</p></blockquote>
<blockquote><p>עָמְדָה תְּכוּנָה לְגֵו אוֹ בָלְתָה – // חַיִּים מְקוֹרָהּ, / בִּנְוֵה אֱמֶת הָיְתָה.</p></blockquote>
<br/>
<blockquote><p>רַבּוֹת תְּמוּנוֹת בְּגֵו נִרְכָּבוֹת</p></blockquote>
<blockquote><p>יַחַד, עֲרֵבוֹת בְּקִרְבָּהּ שָׁבוֹת.</p></blockquote>
<blockquote><p>עֵינָהּ כְּשֶׁמֶשׁ בְּאֶפֶס עָבוֹת,</p></blockquote>
<blockquote><p>הֵיכָל מְתֹאָר לְשִׁבְתָּהּ בָּנְתָה, // כִּי מִבְּשָׂרָהּ / הוֹד מְחוֹלְלָהּ חָזְתָה.</p></blockquote>
<br/>
<blockquote><p>הַכֹּל בְּלִי עֵת בְּשִׂכְלָהּ רוֹאָה,</p></blockquote>
<blockquote><p>טֶרֶם בְּכֹחָהּ תְּהִי נִמְצָאָה</p></blockquote>
<blockquote><p>דֵעָה, וְאַחַר לְפֹעַל יָצָאָה.</p></blockquote>
<blockquote><p>תִּשָּׂא – וְהִיא כַנְּשׂוּאָה דָמְתָה, // עַד יַאֲדִירָהּ / אֵל לְרַחֲמָיו פָּנְתָה.</p></blockquote>
<br/>
<blockquote><p>מָעוֹן, בְּךָ נִתְכְּנָה קִימָתִי,</p></blockquote>
<blockquote><p>אַיִן יְרִיבִי וְאַתְּ חוֹמָתִי,</p></blockquote>
<blockquote><p>אָשִׁית לְמָקוֹם לְךָ נִשְׁמָתִי.</p></blockquote>
<blockquote><p>אִם תַּחֲרִישׁ מִקְּרֹא לָךְ – מֵתָה // אָכֵן יְעִירָהּ/ זִכְרְךָ וּבוֹ חָיְתָה. <br/></p></blockquote>
