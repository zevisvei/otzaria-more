<h1>אָנָּא בְּעָרְכֵנוּ</h1>
אברהם אבן עזרא
<blockquote><p>אָנָּא בְּעָרְכֵנוּ, לְקוֹל שַׁוְעֵנוּ יְיָ שְׁמָעָה!</p> <p>אָנָּא לְחֶטְאֵנוּ בַּעֲוֹן בִּצְעֵנוּ יְיָ סְלָחָה!</p> <p>אִמְרֵי מַהֲלָלוֹת / שְׁמַע, יְיָ – / זְדוֹנוֹת וּמְעִילוֹת / סְלַח, יְיָ.</p> <p>בְּעָרְכִי תַחֲנוּנַי / שְׁמַע, יְיָ – / וְהִרְהוּר רַעְיוֹנִי / סְלַח, יְיָ.</p> <p>רְנָנוֹת הֵבֵאתִי / שְׁמַע, יְיָ – / וַאֲשֶׁר הֲרֵעוֹתִי / סְלַח, יְיָ.</p> <p>הַשׁוֹאֲלִים חַסְדָּךְ / שְׁמַע, יְיָ – / וּלְעָנִי וָדָךְ / סְלַח, יְיָ.</p></blockquote>
<blockquote><p>מִצַּר קָרָאתִי / שְׁמַע, יְיָ – / וְיָשָׁר הֶעֱוִיתִי / סְלַח, יְיָ.</p> <p>לְךָ אֶשָּׂא עֵינִי / שְׁמַע, יְיָ – / וְאִם גָּדַל עֲוֹנִי / סְלַח, יְיָ.</p> <p>לְאֶנְקַת אֲסִיר כֶּלֶא / שְׁמַע, יְיָ – / וְלַדַּל הַנִּקְלֶה / סְלַח, יְיָ.</p> <p>שִׂיחַ תְּחִנָּתִי / שְׁמַע, יְיָ – / וְחוֹבִי וְאַשְׁמָתִי / סְלַח, יְיָ.</p> <p>לְשָׁבִים עָדֶיךָ / שְׁמַע, יְיָ – / וּפֶשַׁע עֲבָדֶיךָ / סְלַח, יְיָ.</p> <p>עֲשׁוּקִים וַאֲנוּסִים / שְׁמַע, יְיָ – / וְאִם הִרְבּוּ כְּעָסִים / סְלַח, יְיָ.</p> <p>שִׁבְחֲךָ יֹאמֵרוּ / שְׁמַע, יְיָ – / וְאִם עִצְּבוּ וּמָרוּ / סְלַח, יְיָ.</p> <p>תַּחֲנוּן יְדַבֶּר רָשׁ / שְׁמַע, יְיָ – / וְחֵטְא אוֹתִי גֵרֵשׁ / סְלַח, יְיָ.</p> <p>תְּהִלּוֹת שִׁירוֹתַי / שְׁמַע, יְיָ – / וְרֹע פְּעֻלּוֹתַי / סְלַח, יְיָ.</p></blockquote>
