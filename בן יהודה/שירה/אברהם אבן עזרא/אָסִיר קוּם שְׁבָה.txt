<h1>אָסִיר קוּם שְׁבָה</h1>
אברהם אבן עזרא
<blockquote><p>אָסִיר, קוּם שְׁבָה / וּתְצַפֶּה לְמוֹעֵד מוֹעֲדִים!</p> <p>חָזוּת נִכְתְּבָה / מִפִּי נְבִיאִים לֵב מְעוֹדְדִים:</p> <p>כִּי יַד אַהֲבָה / תִּתְעוֹרֵר לְנָעִים וְנוֹדְדִים.</p> <p>נֶפֶשׁ נַעֲנָה / עוֹד תָּשׁוּב לְעֶדְנָה</p> <p>יוֹם יֶחְזוּ מְעַנִּים / דֶּגֶל יְיָ עָלַי אַהֲבָה.</p> <br/> <p>בַּעְדוֹ שָׁאֲלוּ / פִי חִירוֹת וְיַם סוּף מְקַנְּאִים,</p> <p>עַל לֵב הֶעֱלוּ / יוֹם כִּסּוּ תְהוֹמוֹת חֵיל שׂוֹנְאִים.</p> <p>הָרִים נָזְלוּ, / אֶרֶץ רָעֲשָׁה מִקּוֹל קוֹרְאִים</p> <p>יוֹם בּוֹ נִתְּנָה / עֵדוּת נֶאֱמָנָה</p> <p>עַל יַד צִיר אֱמוּנִים — / וּבְיַד יְיָ עֵדוּת נִכְתְּבָה.</p> <br/> <p>רַב, נָקֵל בְּעֵין / עֻזּוֹ זֹאת — וְתוֹסִיף לַעֲזֹר!</p> <p>הִנֵּנִי כְבֵן, / רִיב לוֹ עִם יְרִיבִי מִדּוֹר.</p> <p>גָּבַרְתִּי — וְכֵן / נִגְבַּר, עַם בְּחָרוֹ לְבֵן בְּכוֹר.</p> <p>יָדוֹ כוֹנְנָה / מִקְדָּשִׁי — וְחָנָה</p> <p>תוֹךְ הֵיכָל וְלִפְנִים / וּכְבוֹד יְיָ בּוֹ שָׁכֵן וּבָא.</p> <br/> <p>מַלְכִּי, עֶדְרְךָ / קַבֵּץ מֵחֲדָרִים לְחֶדְרְךָ!</p> <p>תִּשְׁלַח עֶזְרְךָ / [………]</p> <p>[…………..]</p> <p>גּוֹיִים שָׁכְנָה, / יָבוֹאוּ בְרִנָּה</p> <p>כֻּלָּם רַעֲנַנִּים / אֶל בֵּית יְיָ רָצוּף אַהֲבָה.</p></blockquote>
