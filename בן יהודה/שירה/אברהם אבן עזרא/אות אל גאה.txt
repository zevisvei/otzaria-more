<h1>אות אל גאה</h1>
אברהם אבן עזרא
<blockquote><p>אוֹת אֵל גֵּאֶה / יוֹם יוֹם אֶרְאֶה / כּוֹנַנְתָּ אֶ- / רֶץ וַתַּעְמֹד.</p></blockquote>
<blockquote><p>יְיָ מֶלֶךְ, / יְיָ מָלָךְ, / אֲדֹנָי אֱלֹהַי / גָּדַלְתָּ מְאֹד.</p></blockquote>
<blockquote><p>בְּקִרְבִּי קְדוֹשִׁי, / אֵיכָה אֶשִׁי / זִכְרוֹ – וְנַפְשִׁי / יוֹדַעַת מְאֹד!</p></blockquote>
<blockquote><p>רוּחֲךָ טוֹבָה / תּוֹרֶה נְתִיבָה / לִי, כִּי רְחָבָה / מִצְוָתְךָ מְאֹד.</p></blockquote>
<blockquote><p>הֵן קָדַמְתָּ, / עַל כָּל רַמְתָּ, / כִּי עָצַמְתָּ / מִמֶּנּוּ מְאֹד.</p></blockquote>
<blockquote><p>מַרְאֶה עֵינַי / וּמִשְׁמַע אָזְנַי: / גָּדוֹל אֲדֹנָי / וּמְהֻלָּל מְאֹד! <br/></p></blockquote>
