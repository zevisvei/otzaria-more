<h1>מַחְנוֹת אֵל יְרוּצוּן</h1>
אברהם אבן עזרא
<blockquote><p>מַחְנוֹת אֵל יְרוּצוּן / מִשְׁמָר לְמִשְׁמָר,</p> <p>יִתְאַחְדּוּ – וְקָרָא / זֶה אֶל זֶה וְאָמָר.</p> <br/> <p>אֶתְאַזְּרָה בְשִׁירִים / עַל דַּעְתִּי יְשָׁרִים,</p> <p>כִּי כֹחַ אֲבִירִים / מוֹצִיאִים! נְבָרִים –</p> <p>כַּעְצוּמִים וּבָרִים – / גַּבְרִיאֵל רֹאשׁ גְּבָרִים –</p> <p>תּוּמָתָם יַעֲרִיצוּן! / זֶה הַגָּלִיל מָר</p> <p>עַל הָאָרֶץ, וְנִבְרָא / בְּתוֹכוֹ כְּמוֹ מָר.</p> <br/> <p>בְּכוֹכָב נִכְבַּד אֵל / בְּצוּרִים מְיֻפִּים,</p> <p>חֹק חִכְּמוּ לְשׁוֹאֵל / בְּהָבֵר סְעִפִּים: –</p> <p>שֵׁם הַגְּבִיר רְפָאֵל / רְפוּאָה לְנִרְפִּים –</p> <p>וּכְסוֹבְבָם יְקִיצוּן / נִרְדָּם, וְנִכְמָר</p> <p>כָּל נֶפֶשׁ צְעִירָה, / כִּי חָמַס וְנִשְׁמָר.</p> <br/> <p>רַב מַחְנֵה מַאֲדָרָם, / בַּנֹּגַהּ בְּפָנִים</p> <p>יָפִים עַל הֲדָרָם, / קוֹל עַנּוֹת מְעַנִּים: –</p> <p>וַעֲנִיאֵל שֵׁם גְּבִירָם / הָעוֹנֶה לְנַעְנִים –</p> <p>בִּמְחוֹלָם יְרִיצוּן, / וְסָמַר</p> <p>גֵּו כָּל יְצִיר בְּמוֹרָא / מִפַּחְדּוֹ וְנָמָר.</p> <br/> <p>מַה נִּפְלְאוּ שְׂרָפִים / כַּשֶּׁמֶשׁ יְסוֹדָם,</p> <p>עַצְמֵיהֶם מְרַשְּׁפִים / וּמְפִיצִים מְאוֹרָם</p> <p>וּשְׁפָלִים בּוֹ מְנִיפִים. – / מִיכָאֵל גְּבִירָם –</p> <p>בַּעְדִּי לָאֵל יְלִיצוּן, / יוֹאֵל וְגָמַר</p> <p>עָלַי, בְּטוּב בְּשׂוֹרָה / יְבַשֵּׂר לְלֵב מָר.</p></blockquote>
