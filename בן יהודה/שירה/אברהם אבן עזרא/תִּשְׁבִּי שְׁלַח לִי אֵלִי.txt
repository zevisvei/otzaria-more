<h1>תִּשְׁבִּי שְׁלַח לִי אֵלִי</h1>
אברהם אבן עזרא
<p>(הבדלה)</p>
<blockquote><p>תִּשְׁבִּי שְׁלַח לִי, / אֵלִי, לְרַפֵּא מַחְלִי!</p> <br/> <p>אַל נָא בְחוֹבִי / תַּרְחִיק זְּמַן הַיֶּשַׁע,</p></blockquote>
<blockquote><p>כִּי לֹא בְחַרְבִּי / אֶבְטַח – וְלִי לֹא תוֹשָׁע.</p></blockquote>
<blockquote><p>רִבָה יְרִיבִי / מַהֵר וְעֵינָיו הָשַׁע,</p></blockquote>
<blockquote><p>וּסְמֹךְ בְּנָפְלִי / יָדַי וְהַעְצֵם חֵילִי.</p> <br/> <p>בִּי, אֵל, הֲתַחְרִישׁ / מִבֵּן יְקַו רַחְמֶיךָ?</p></blockquote>
<blockquote><p>וִיטוּר וְנָפִישׁ / אֵיךְ יִירְשׁוּ כַרְמֶךָ?</p></blockquote>
<blockquote><p>קוּמָה, וְעַל אִישׁ / שָׂעִיר שְׁפֹךְ זַעְמֶךָ</p></blockquote>
<blockquote><p>וּשְׁמַע לְקוֹלִי / וּנְשָׂא כְקֶדֶם דִּגְלִי.</p> <br/> <p>רַב מִהְיוֹתִי / יוֹשֵׁב מְשׁוֹמֵם בִּשְׁבִי,</p></blockquote>
<blockquote><p>הָחֵשׁ פְּדוּתִי / עַל יַד נְבִיאָךְ תִּשְׁבִּי;</p></blockquote>
<blockquote><p>שׁוּב אֶל נְוָתִי / וּדְמֵה לְעֹפֶר וּצְבִי,</p></blockquote>
<blockquote><p>יַצֵּב גְּבוּלִי / וּבְנֵה דְּבִיר הֵיכָלִי.</p> <br/> <p>׳מַה לָּךְ בְּשִׂיחֵךְ, / תִּצְעֲקִי כִשְׁבוּיָה?</p></blockquote>
<blockquote><p>הִנֵּה מְשִׁיחֵךְ / נִגְלָה וּבָא אֵלִיָּה!</p></blockquote>
<blockquote><p>רִקְחֵךְ וְרֵיחֵךְ / אָרַח בְּהַר מוֹרִיָּה,</p></blockquote>
<blockquote><p>וַעְלֵי חֲלִילִי / שִׁיר יַעֲנֶה נִין מַחְלִי.׳</p></blockquote>
