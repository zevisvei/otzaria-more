<h1>אֱנוֹשׁ מִיּוֹם גָּדוֹל גּוּרָה</h1>
אברהם אבן עזרא
<blockquote><p>[אֱנוֹשׁ מִיּוֹם] גָּדוֹל גּוּרָה, / כִּי נוֹרָא וּמַר הוּא,</p> <p>הוּא יוֹם […] אֲדוֹן כָּל נִבְרָא / לִפְקוֹד מַעֲשֵׂהוּ.</p> <br/> <p>בֶּן מָוֶת פְּקַח עֵינֶיךָ / וּרְאֵה אֶת מְלוֹנָךְ,</p> <p>יוֹם תִּשְׁמַע בְּבִנְיָנֶךָ: / ‘קוּם צֵא מִמְּעוֹנָךְ!’</p> <p>יוֹם תִּצַּלְנָה שְׁתֵּי [אָזְ]נֶיךָ / וּתְבֻקַּשׁ וְאֵינָךְ –</p> <p>יוֹם גַּבְּךָ לְעָפָר נִקְרָא, / יֶחְזַק מַחֲלֵהוּ,</p> <p>וּלְאַחַר, כְּדוֹרֵךְ פּוּרָה, / מָוֶת יִדְרְכֵהוּ.</p> <br/> <p>רָא[מ]וֹת יַאֲצֹר עִם גָּבִישׁ / לִהְיוֹת מַעֲשִׁירִים,</p> <p>[…] כִּי כֹל הוֹבִישׁ / נֶעְזָב לָאֲחֵרִים,</p> <p>גַּם יֵצֵא לְבֵית עַכָּבִישׁ / מִמַּשְׂכִּית חֲדָרִים,</p> <p>וּבְמוֹתוֹ לְעָשְׁרוֹ יִקְרָא – / אָז לֹא יַעֲנֵהוּ,</p> <p>אַךְ מַעְשֵׂה רְצוֹן אֵל נוֹרָא – / קָרוֹב יֶחֱזֵהוּ.</p> <br/> <p>הֶעָמָל, בְּעֵת בּוֹ יִיגַע – / יִכָּנַע וְיִדְאַב,</p> <p>אוֹ מַחְלָה בְּגוּפוֹ תִּפְגַּע – / יֵאָנַח וְיִכְאָב,</p> <p>יֶחְדַּל בַּעֲדוֹ מִפֶּגַע / אָח, קָרוֹב וְהָאָב.</p> <p>אִם כֶּסֶף וְזָהָב הִפְרָה – / אַחֵר יֹאכְלֵהוּ,</p> <p>אוֹ אַרְמוֹן לְשִׁכְנוֹ קֵרָה – / נָכְרִי יִשְׁכְּנֵהוּ.</p> <br/> <p>מֶלֶךְ בְּלִי דְמוּת וָעֶרֶךְ, / נַעְלֶה עַל זְבוּלָיו,</p> <p>חֹן עַל בֵּן אֲחָזוֹ מֹרֶךְ / מִמָּוֶת וְחִילָיו.</p> <p>אוּלַי, יוֹם בְּךָ יִתְבָּרֵךְ, / יִתְנַחְמוּ אֲבֵלָיו,</p> <p>וּכְמֵאָז בַּעֲבוֹת חֶבְרָה / לְבֵיתָךְ תִּמְשְׁכֵהוּ,</p> <p>לֵאמֹר: ‘כִּי נַעַר יִשְׂרָ- / אֵל וָאֹהֲבֵהוּ.’</p></blockquote>
