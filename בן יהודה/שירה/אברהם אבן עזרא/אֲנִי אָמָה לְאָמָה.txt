<h1>אֲנִי אָמָה לְאָמָה</h1>
אברהם אבן עזרא
<blockquote><p>אֲנִי אָמָה לְאָמָה, / עַל כֵּן דְּבָרַי לָעוּ —</p> <p>מִי יִתֵּן בְּחָכְמָה — / וַיַּעֲבֹר עָלַי מָה.</p> <br/> <p>בַּמִּקְדָּשׁ בְּעָמְדִי / עָמַדְתִּי בִּבְרִית שְׁלוֹמִים</p> <p>עִם אָחִי — וְעַבְדִּי / הִנּוֹ; וְלָמָּה בְקָמִים</p> <p>קָם עָלַי בְּאֵידִי / וַיַּעְקְבֵנִי פְעָמִים?</p> <p>הָאֵיבָה וְשִׂטְמָה / נָטַר אֲשֶׁר נוֹדָעוּ,</p> <p>בָּא אָחִי בְמִרְמָה / גַּם הוּא וְעָשָׂה בְעָרְמָה.</p> <br/> <p>רַעְיַת חֵן יְשִׂיחָהּ / לִבָּהּ וְתֹאמַר בְּנַפְשָׁהּ:</p> <p>אַיֵּה דוֹד לְקָחָהּ / וּפְאֵר נְבוּאָה חֲבָשָׁהּ,</p> <p>צִיץ קֹדֶשׁ בְּמִצְחָהּ / וּצְנִיף מְלוּכָה בְּרֹאשָׁהּ?</p> <p>וּבְיוֹם אַף וְחֵמָה / בִּגְדֵי פְאֵר נִקְרָעוּ,</p> <p>עַד נוֹתְרָה עֲרֻמָּה / וַתַּעֲטֶה סוּת כְּלִמָּה.</p> <br/> <p>הֵן צִירַי חֲזָקִים / עָלַי וְאֶצְעַק כְּחוֹלָה,</p> <p>אַךְ צָרַי מְשַׂחֲקִים — / הֵן הִיא חֲטָאָה גְדוֹלָה!</p> <p>אֵיךְ עָמְדוּ שְׁחָקִים / וּבְנֵי אֱלֹהִים בְּגוֹלָה?</p> <p>עַד כַּמָּה וְכַמָּה / לַעַג וּבוּז יִשְׂבָּעוּ?</p> <p>אֵיךְ רָעָב וְצָמָא / בִּדְבַר נְבוּאָה חֲתוּמָה!</p> <br/> <p>מָתַי קֵץ פְּלָאוֹת — / וּדְבַר אֱלֹהִים צְבָאוֹת</p> <p>‘הוֹאֶל נָא’ בְמַרְאוֹת / אָז, לֹא בְהָקִיץ בְּמַרְאוֹת.</p> <p>עַל דִּבְרֵי נְבוּאוֹת / עוֹדָם עֲתִידוֹת וּבָאוֹת —</p> <p>עַל אָזְנֵי מְזִמָּה / קוֹלוֹת אֱמֶת נִשְׁמָעוּ:</p> <p>‘דֹּמִּי, בַּת תְּמִימָה, / וּתְהִי מַשְׂכֻּרְתֵּךְ שְׁלֵמָה.’</p></blockquote>
