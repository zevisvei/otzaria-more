<h1>אֶל דּוֹד בְּשֵׁם לֹא כִנִּיתִי</h1>
אברהם אבן עזרא
<blockquote><p>אֶל דּוֹד בְּשֵׁם לֹא כִנִּיתִי</p></blockquote>
<blockquote><p>מִימֵי נְעוּרַי פָּנִיתִי,</p></blockquote>
<blockquote><p>דּוֹדוֹ בְנַפְשִׁי קָנִיתִי</p></blockquote>
<blockquote><p>וּבְתוֹךְ לְבָבוֹ חָנִיתִי.</p></blockquote>
<br/>
<blockquote><p>לִי בֵית זְבוּל / כִּדְמוּת שְׁבִיל / דָּרֵי גְלִיל / רוּם וּמַחְמַדִּי, –</p></blockquote>
<blockquote><p>לִהְיוֹת לְאֵל / נַפְשִׁי, וְאֵל / אוֹדֶה בְחֵיל / שִׁיר יְמֵי חֶלְדִּי.</p></blockquote>
<br/>
<blockquote><p>בָּךְ שָׁת, נְדִיבָה, שַׁדַּי אוֹת,</p></blockquote>
<blockquote><p>תִּרְאִי כְּבָר אוֹת בִּפְלִיאוֹת.</p></blockquote>
<blockquote><p>לִדְבֹּק בְּצוּר כָּזֶה יֵאוֹת –</p></blockquote>
<blockquote><p>נִמְשָׁל לְאָדוֹן בִּצְבָאוֹת,</p></blockquote>
<blockquote><p>אֵין לוֹ גְבוּל, / הוּא כֹל – וּבִלְ- / עָדָיו אֱלִיל, / בּוֹ לְבַד תֶּחְדִּי,</p></blockquote>
<blockquote><p>בִּנְטֹש הֲבֵל / חֶלְדֵּךְ בְּלֵיל / בָּרְחוֹ כְצֵל – / עַל אֱמֶת חַדִּי.</p></blockquote>
<br/>
<blockquote><p>מִי יִתְּנֵנִי כָּל חֶדְלִי</p></blockquote>
<blockquote><p>טוֹבוֹת לְקַבֵּל מֵאֵלִי.</p></blockquote>
<blockquote><p>שַׁתִּי פְעָלָיו מִמּוּלִי;</p></blockquote>
<blockquote><p>וּכְפִי תְהִלּוֹת חַסְדּוֹ, לִי</p></blockquote>
<blockquote><p>יָשִׁיב גְּמוּל: / כָּל צִיר וְחִיל / יַהְפֹּךְ לְגִיל – / לֹא כְפִי חַסְדִּי –</p></blockquote>
<blockquote><p>וּבְמַאֲפֵל / דּוֹמֶה לְלֵיל / הוּא יַאֲהֵל / שֶׁמֶשׁ נֶגְדִּי.</p></blockquote>
