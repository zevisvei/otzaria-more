<h1>[אודה לשם באפי נשמה]</h1>
אברהם אבן עזרא
<p>מוחרך</p>
<br/>
<blockquote><p>אוֹדֶה לְשָׂם בְּאַפִּי נְשָׁמָה / רָמָה.</p></blockquote>
<br/>
<blockquote><p>בִּי שָׂהֲדִי לְנוֹרָא עֲלִילָה,</p></blockquote>
<blockquote><p>אֶתְעַנְּגָה בְזִכְרוֹ בְּגִילָה.</p></blockquote>
<blockquote><p>בִּי חֻדְּשָׁה, עֲלֵי כָּל אֲכִילָה / רָמָה,</p></blockquote>
<blockquote><p>וּשְׁמוֹ מְקוֹר תְּבוּנָה – וְלָמָּה / אֶצְמָא?</p></blockquote>
<br/>
<blockquote><p>רוּחַ אֱנוֹשׁ בְּהֵיכַל נְוָתוֹ,</p></blockquote>
<blockquote><p>בּוֹ נִמְשְׁלָה כְּמֶלֶךְ בְּבֵיתוֹ,</p></blockquote>
<blockquote><p>לָהּ מַעֲלָה בְחַיָּיו וּבְמוֹתוֹ – / רָמָה,</p></blockquote>
<blockquote><p>תַּעֲמֹד, וְלֹא לְרוּחַ בְּהֵמָה / דּוֹמָה.</p></blockquote>
<br/>
<blockquote><p>הֶבֶל וְשָׁוְא יְקַר הַגְּוִיָּה,</p></blockquote>
<blockquote><p>מָה אֶחְמְדָה בְאֶרֶץ נְשִׁיָּה?</p></blockquote>
<blockquote><p>אֶרְאֶה זְמַן כְּקֶשֶׁת רְמִיָּה / רָמָה</p></blockquote>
<blockquote><p>וִימֵי אֱנוֹשׁ כְּצֵל עַל אֲדָמָה / הֵמָּה.</p></blockquote>
<br/>
<blockquote><p>מִיּוֹם הֱיוֹת לְבָבִי וְעַד כֹּה</p></blockquote>
<blockquote><p>יֵיטִיב וְיִצְפְּנֵנִי בְּסֻכֹּה.</p></blockquote>
<blockquote><p>בַּמָּה אֲקַדְּמֶנוּ – וְיָדוֹ / רָמָה</p></blockquote>
<blockquote><p>עַל כֹּל, וְאֵין בְּיָדִי מְאוּמָה / בַּמָּה?</p></blockquote>
