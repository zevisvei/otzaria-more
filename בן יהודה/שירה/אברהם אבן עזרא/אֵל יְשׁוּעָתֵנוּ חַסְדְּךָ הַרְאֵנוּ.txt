<h1>אֵל יְשׁוּעָתֵנוּ חַסְדְּךָ הַרְאֵנוּ</h1>
אברהם אבן עזרא
<blockquote><p>אֵל יְשׁוּעָתֵנוּ, / חַסְדְּךָ הַרְאֵנוּ,</p> <p>מַעֲשֵׁה יָדֵינוּ / כּוֹנְנָה עָלֵינוּ!</p> <br/> <p>בַּעֲלוֹת עַל לִבּוֹת / בָּךְ לְבַד הֶאְמִינוּ</p> <p>כָּל פְּלִיאוֹת רַבּוֹת — / דַּרְכְּךָ יָבִינוּ,</p> <p>נִתְכְּנוּ לָךְ סִבּוֹת, / פַּעֲמָם יָכִינוּ.</p> <p>צִדְקְךָ הוֹרֵנוּ; / נִכְבְּשָׁה יִצְרֵנוּ</p> <p>בָּךְ, וְאַל יַתְעֵנוּ / עוֹד עֲוֹן בִּצְעֵנוּ.</p> <br/> <p>רָשְׁמוּ הַלּוֹמְדִים / כָּל תְּבוּנוֹת פִּיהֶם,</p> <p>קֵץ לְמוֹעֵד מוֹעְדִים / תַּעֲלֻמוֹת מֵהֶם.</p> <p>כֵּן כְּעֵינֵי עוֹבְדִים / אֶל יְדֵי מוֹרֵיהֶם —</p> <p>כֵּן לְךָ עֵינֵינוּ / עַד שֶׁתְּחָנֵּנוּ.</p> <p>אוֹת בְּרִית בֵּינֵינוּ / עֵד אֱמֶת יַעְנֵנוּ.</p> <br/> <p>מִפְּדוּת נוֹאָשָׁה / עַד הֲלֹם רוּחֵנוּ,</p> <p>כִּי עֲבוֹדָה עָשָׁה / מִדְּבִיר נִצְחֵנוּ;</p> <p>בַּעֲבוֹדָה קָשָׁה / תַּם לְרִיק כֹּחֵנוּ.</p> <p>מַהֲרָה הַמְשֵׁנוּ / עַד כְּלוֹת נַפְשֵׁנוּ,</p> <p>אָז שְׁלַח אֵלֵינוּ / צִיר וְיִגְאָלֵנוּ!</p></blockquote>
