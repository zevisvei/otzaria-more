<h1>אֶשְׁפֹּךְ שִׂיחִי לְפָנֶיךָ</h1>
אברהם אבן עזרא
<blockquote><p>אֶשְׁפֹּךְ שִׂיחִי לְפָנֶיךָ, / רוּחִי אַפְקִיד בְּיָדֶךָ / וְנִשְׁמָתִי.</p> <br/> <p>בּוֹרֵא תֵבֵל בְּמַאֲמָרוֹ / וְחָכְמָתוֹ,</p> <p>נוֹטֶה שַׁחַק בְּמִדְבָּרוֹ / וּבִינָתוֹ,</p> <p>נִשָׂא עַל כִּסֵּא יְקָרוֹ /וְיִפְעָתוֹ –</p> <p>הַטֵּה כַיּוֹם אָזְנֶךָ /וּשְׁמַע שַׁוְעַת עֲבָדֶיךָ / וְשַׁוְעָתִי!</p> <br/> <p>רָאֹה תִרְאֶה בְעָנְיֵנוּ / וְלַחֲצֵנוּ,</p> <p>כִּי אַתָּה צוּר יִשְׁעֵנוּ / וּפוֹצֵנוּ</p> <p>וּשְׁמַע אֶנְקַת אֲסִירֵינוּ, / קְדוֹשֵׁנוּ.</p> <p>שִׂים עָלֵינוּ עֵינֶיךָ, / גַּם תָּשִׁים נָא בְּנֹאדֶךָ / דִּמְעָתִי!</p> <br/> <p>מַלְכִּי, בְּךָ בָטַח לִבִּי /בְּגָלוּתִי,</p> <p>כִּי אַתְּ סַלְעִי וּמִשְׂגַּבִּי / בְּצָרָתִי.</p> <p>מַרְפֵּא תִשְׁלַח בְּיַד תִּשְׁבִּי / לְמַכָּתִי –</p> <p>תִּשָּׂא אֵלַי פָּנֶיךָ / וְתָאִיר נָא מֵהוֹדֶךָ / אֲפֵלָתִי!</p></blockquote>
