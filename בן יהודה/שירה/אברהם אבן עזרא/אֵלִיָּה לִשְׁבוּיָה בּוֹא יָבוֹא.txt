<h1>אֵלִיָּה לִשְׁבוּיָה בּוֹא יָבוֹא</h1>
אברהם אבן עזרא
<blockquote><p>אֵלִיָּה / לִשְׁבוּיָה / בּוֹא יָבוֹא בִּבְשׂוֹרָה</p> <p>לִפְנֵי בּוֹא / יוֹם יְיָ / הַגָּדוֹל וְהַנּוֹרָא.</p> <br/> <p>בְּתַחֲנוּנִים / כְּאֶבְיוֹנִים / נְקַדֵּם אֶת פָּנֶיךָ,</p> <p>זְכֹר הַבְּרִית / לִשְׁאֵרִית / צֶאֱצָאֵי אֱמוּנֶיךָ</p> <p>וְאֵיךְ אֲחֵיהֶם / וְעַבְדֵיהֶם / יַעֲבִידוּם לְפָנֶיךָ —</p></blockquote>
<blockquote><p>וְהֵם חֶבְלָךְ, / עֲבָדִים לָךְ, / כַּכָּתוּב בַּתּוֹרָה!</p> <br/> <p>רְאֵה עַתָּה — / הֲלֹא אַתָּה / אֵל אֶחָד וְלֹא תִשְׁנֶה!</p> <p>אֲנִי בָזוּז / כְּשֶׂה גָזוּז / וְאֶקְרָאֲךָ וְלֹא תַעֲנֶה.</p> <p>שְׁלַח מַרְפֵּא / לְלֵב נִכְפֶּה / וְתָבִיא מַגִּיד מִשְׁנֶה</p> <p>כִּדְבָרָךְ, / לִבְכוֹרָךְ / הֲלֹא מִשְׁפַּט הַבְּכוֹרָה!</p> <br/> <p>'מַה תּוֹחִיל, / עַם הוֹחִיל — / וּמוֹעֵד מוֹעֲדִים יִשְׁלָם!</p> <p>וַאְשִׁיבֵךְ / וְאוֹשִׁיבֵךְ / בְּהֵיכַל חֵן וְהוֹד אוּלָם,</p> <p>וְשָׁם גּוֹאֵל / לְיִשְׂרָאֵל / יִהְיֶה לָךְ לְאוֹר עוֹלָם</p> <p>וְלַיְלָה כַּ-/ יוֹם יָאִיר, / כַּחֲשֵׁיכָה כָּאוֹרָה.'</p></blockquote>
