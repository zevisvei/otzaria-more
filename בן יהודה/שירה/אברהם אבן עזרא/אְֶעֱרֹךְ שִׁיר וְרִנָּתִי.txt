<h1>אְֶעֱרֹךְ שִׁיר וְרִנָּתִי</h1>
אברהם אבן עזרא
<p>אהבה</p>
<blockquote><br/> <p>אְֶעֱרֹךְ שִׁיר וְרִנָּתִי / עַל שְׂפָתִי.</p> <br/> <p>בָּךְ, אֵלִי, שִׁבְרִי וּבָךְ כִּסְלִי / בַעֲמָלִי,</p></blockquote>
<blockquote><p>שֵׁם כְּבוֹדְךָ לָעַד מְשׂוֹשׂ גִּילִי / הוּא לִי,</p></blockquote>
<blockquote><p>מַאֲמָרְךָ תָּמִיד בְּמַאְפֵלִ[י] / נֵר לְרַגְלִי –</p></blockquote>
<blockquote><p>רֹאשׁ דְּבָרְךָ אֱמֶת וּבַחְלוֹתִי / הוּא חֲיוֹתִי.</p> <br/> <p>רִצֲצוּ אֶת לִבִּי וְעַצְמוֹתַי / מַעֲווֹתַי</p></blockquote>
<blockquote><p>וְיָרְשׁוּ [אֹהָלַי] וְחַצְרוֹתַי / שׂוֹנְאוֹתַי,</p></blockquote>
<blockquote><p>הֵן עֲלֵי זֹאת תָּמִיד בְּאַנְחוֹתַי / […..<תי>]</p></blockquote>
<blockquote><p>[….]בה [עלות שכינתי] מַעֲוָתִי.</p> <br/> <p>׳הֶעֱבַדְתִּיךְ לְזָר וְיִסַּרְתִּיךְ / בַּעֲוֹנֵךְ –</p></blockquote>
<blockquote><p>הֵן אֲנִי שָׁב בְּחֵן וְרִחַמְתִּיךְ / עוֹד וְאֶבְנֵךְ.</p></blockquote>
<blockquote><p>אִם עֲזַבְתִּיךְ בְּאַף וְהִכְעַסְתִּיךְ – / לֹא אֲעַנֵּךְ,</p></blockquote>
<blockquote><p>עֵת חֲמָתִי אֶזְכֹּר לְרַעֲיָתִי / אַהֲבָתִי.</p> <br/> <p>מִבְּרוּאִים חֶלְקִי אַתְּ וְגוֹרָלִי / אַתְּ וְחֶבְלִי,</p></blockquote>
<blockquote><p>הֵן אֲיַשֵּׁר לַעְלוֹת לְהֵיכָלִי / לָךְ שְׁבִילִי,</p></blockquote>
<blockquote><p>יוֹם רְצוֹנִי ״אִישִׁי״ וְלֹא ״בַעְלִי״ / תִקְרְאִי לִי –</p></blockquote>
<blockquote><p>אָז אֲעוֹרֵר לַבַּת אֲהוּבָתִי / אֶת בְּרִיתִי.׳</p></blockquote>
