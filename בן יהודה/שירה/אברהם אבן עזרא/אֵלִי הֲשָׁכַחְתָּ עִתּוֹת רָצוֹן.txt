<h1>אֵלִי הֲשָׁכַחְתָּ עִתּוֹת רָצוֹן</h1>
אברהם אבן עזרא
<p>זולת (לט׳ באב?)</p>
<blockquote><br/> <p>אֵלִי, הֲשָׁכַחְתָּ / עִתּוֹת רָצוֹן?</p></blockquote>
<blockquote><p>וְעַל מִי נָטַשְׁתָּ / מְעַט הַצֹּאן?</p> <br/> <p>אֵיךְ בָּא עָר וְקָדֵשׁ / בַּזֶּה הַחֹדֶשׁ</p></blockquote>
<blockquote><p>בְּהֵיכַל הַקֹּדֶשׁ, / פְּנִימִי וְחִיצוֹן?</p></blockquote>
<blockquote><p>בָּא בִדְבִיר הַיְּקָר / גּוֹי בְּאֵל חַי שָׁקַר,</p></blockquote>
<blockquote><p>בְּלִי פַר בֶּן בָּקָר / וְאַיִל מִן הַצֹּאן.</p></blockquote>
<blockquote><p>רֹאשׁ נִזְרִי מָרְטוּ, / עֲדַת עִירִי בָּעֲטוּ,</p></blockquote>
<blockquote><p>וְשֶׂה וּבְנוֹ שָׁחֲטוּ / וּבַעֲלֵי הַצֹּאן.</p></blockquote>
<blockquote><p>הָאוֹיְבִים דְּלָקוּם, / הֶעָלוֹת – הֱצִיקוּם,</p></blockquote>
<blockquote><p>וְיוֹם אֶחָד דְּפָקוּם / וּמֵתוּ כָּל הַצֹּאן.</p></blockquote>
<blockquote><p>עוֹד אַנְשֵׁי קֹדֶשׁ / לְנֶאְדָּר בַּקֹּדֶשׁ</p></blockquote>
<blockquote><p>הִתְקַדְּשׁוּ בְקֹדֶשׁ / עַל זִבְחֵי הַצֹּאן.</p></blockquote>
<blockquote><p>שׁוּב לְעַמְּךָ, הָאֵל, / הָבָה פְדוּתָם וְגוֹאֵל,</p></blockquote>
<blockquote><p>כִּי אֵין זוּלָתְךָ גּוֹאֵל, / וּתְבַשְּׂרֵם בְּעִלָּצוֹן.</p></blockquote>
