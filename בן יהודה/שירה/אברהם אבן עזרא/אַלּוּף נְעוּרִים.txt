<h1>אַלּוּף נְעוּרִים</h1>
אברהם אבן עזרא
<blockquote><p>אַלּוּף נְעוּרִים / אָז יְמִינוֹ הֵרִים –</p></blockquote>
<blockquote><p>יִזְכֹּר בְּרִיתוֹ, / בַּעֲבוּר אַהֲבָתוֹ / יָשׁוּב לְבֵיתוֹ. <br/><br/> בְּחֵי אֲבִיהֶם / כַּיְּתוֹמִים בָָּניו,</p></blockquote>
<blockquote><p>וְהֵם בְּנֵיהֶם / יִצְעֲקוּ עַל פָּנָיו:</p></blockquote>
<blockquote><p>הֲרֹב מְרִיתֶם? / הֶחֱרִיב מִשְׁכָּנָיו?</p></blockquote>
<blockquote><p>אֲשֶׁר כְּעָרִים / קָם וְנָתַשׁ עָרִים,</p></blockquote>
<blockquote><p>וּבַחֲמָתוֹ / נוֹחֲלָה נַחְלָתוֹ / מִיּוֹם עֲלוֹתוֹ. <br/><br/> רְאוּ, שְׁכֵנִים / מַחֲלָתִי וּלְכוּ</p></blockquote>
<blockquote><p>לְדוֹד, לְפָנִים / בּוֹ מְתָיו הִתְבָּרְכוּ,</p></blockquote>
<blockquote><p>וְתַחֲנוּנִים / עַל צְעִירוֹ שָׁפְכוּ.</p></blockquote>
<blockquote><p>בְּסוֹד נְעָרִים / קוֹל לְאָבִיו יָרִים,</p></blockquote>
<blockquote><p>וְעַל שְׁבוּתוֹ / עַל שְׁמוֹ – עַד מוֹתוֹ / יוּכַל שְׂאֵתוֹ. <br/><br/> מְשַׁל אֲדָמָה / עַד הֲלוֹם בֶּן עֶלְיוֹן</p></blockquote>
<blockquote><p>יִהְיֶה, וְתַמָּה / עֵת אֲשֶׁר בָּהּ פִּדְיוֹן,</p></blockquote>
<blockquote><p>וְנֶעֱלָמָה / כָּל נְבוּאַת חֶזְיוֹן</p></blockquote>
<blockquote><p>וְסוֹד סְפָרִים, / יוֹם לְיוֹם נִזְכָּרִים</p></blockquote>
<blockquote><p>עֲלֵי שְׂפָתוֹ, / כִּי תְשׁוֹבֵב אוֹתוֹ / אֶל בֵּית נְכוֹתוֹ. <br/></p></blockquote>
