<h1>עֱנוּ בַהֲמוֹן שִׁירִים</h1>
אברהם אבן עזרא
<blockquote><p>עֱנוּ בַהֲמוֹן שִׁירִים / לְעוֹשֵׂה שְׁנֵי אוֹרִים!</p> <br/> <p>אֲלָפִים וְרִבּוֹאוֹת / וּמֵאוֹת וּצְבָאוֹת</p> <p>מְעִידִים בְּכָל פֵּאוֹת / לְעוֹשֵׂה נִפְלָאוֹת,</p> <p>מְגַלֶּה נִסְתָּרִים / וְדוֹבֵר מֵישָׁרִים.</p> <br/> <p>בְרֹחַב וּבָאֹרֶךְ, / בְּכָל צַד וְכָל דֶּרֶךְ,</p> <p>בְּשֵׁם אֵין לוֹ עֶרֶךְ / לְאֻמּוֹ יִתְבָּרֵךְ –</p> <p>קוֹרֵא הַדּוֹרִים, / מַשְׁפִּיל וְגַם מֵרִים.</p> <br/> <p>רְעֵה בֵן כִּנִּיתוֹ / בְּכוֹרָךְ וְהִרְבִּיתוֹ,</p> <p>וְכֵן מֵרֵאשִׁיתוֹ / תֵּיטִיב אַחֲרִיתוֹ,</p> <p>וְיַד אוֹהֲבָיו תָּרִים / וּמִתְקוֹמְמָיו תַּחְרִים.</p> <br/> <p>הַקְשֵׁב נִיב עֶרְכִּי / וְיַשֵּׁר אֶת דַּרְכִּי</p> <p>וְאַמֵּץ אֶת בִּרְכִּי – / וְיָדְעוּ עַמִּים, כִּי</p> <p>אֱלֹהֵי הָעִבְרִים / אֱלֹהִים אַדִּירִים.</p> <br/> <p>מַחֲסֶה בְּיוֹם עֲבָרוֹת / וְעֶזְרָה בְּעֵת צָרוֹת,</p> <p>מוֹצִיא מַזָּרוֹת / וְהָיוּ לִמְאוֹרוֹת –</p> <p>בְּרוּם שַׁחַק מְאִירִים / וְתֵבֵל מַזְהִירִים.</p></blockquote>
