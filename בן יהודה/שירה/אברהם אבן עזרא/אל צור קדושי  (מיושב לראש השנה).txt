<h1>אל צור קדושי (מיושב לראש השנה)</h1>
אברהם אבן עזרא
<blockquote><p>אֶל צוּר קְדוֹשִׁי / אֶשָּׂא נַפְשִׁי;</p></blockquote>
<blockquote><p>הַאֲזֵן לַחְשִׁי / וְהַעֲבֵר כַּחְשִׁי,</p></blockquote>
<blockquote><p>אֵל קְדוֹשִׁי, / כְּבוֹדִי וּמֵרִים רֹאשִׁי!</p> <br/> <p>בְּמַעֲמָד וּמוֹשָׁב / שְׁעֵה גֵּר תּוֹשָׁב,</p></blockquote>
<blockquote><p>בְּכָל לֵבָב שָׁב / לָךְ, דֹם יָשָׁב;</p></blockquote>
<blockquote><p>מְהֵרָה יוּשָׁב / לְעִיר קְדוֹשִׁי!</p> <br/> <p>הַעֲלֵה, צוּרִי, / צֳרִי לִמְזוֹרִי</p></blockquote>
<blockquote><p>וְנַהֵל עֶדְרִי / לְצִיּוֹן עִירִי,</p></blockquote>
<blockquote><p>עֲבוּר ׳מַהֲרִי / שְׁלֹשׁ סְאִים וְלוּשִׁי׳.</p> <br/> <p>מְסוּרִים בִּידֵי / צוֹרֵר רוֹדֶה</p></blockquote>
<blockquote><p>חוּשָׁה וּפְדֵה; / וְסוּת יֶשַׁע עֲדֵה</p></blockquote>
<blockquote><p>לְבַשְּׂרָהּ: ׳בִּגְדֵי / יֶשַׁע לִבְשִׁי!׳</p> <br/> <p>חָזוּת ׳וּלְקָחוּם / אֲשְֶׁר הִדִּיחוּם׳</p></blockquote>
<blockquote><p>תְּמַהֵר, רַחוּם, / לְדַלִּים נָחוּם,</p></blockquote>
<blockquote><p>וְחֲזוֹן נַחוּם / הָאֶלְקוֹשִׁי.</p></blockquote>
