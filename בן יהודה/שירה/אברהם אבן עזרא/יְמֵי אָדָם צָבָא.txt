<h1>יְמֵי אָדָם צָבָא</h1>
אברהם אבן עזרא
<blockquote><p>יְמֵי אָדָם צָבָא / וְהוּא בָעוֹלָם הֵלֶךְ,</p> <p>כִּי בַהֶבֶל בָּא / וּבַחשֶׁךְ יֵלֵךְ.</p> <p>שׁוֹגֶה בְּאַהֲבַת תֵּבֵל / סָכַל, כִּי הוּא הֶבֶל;</p> <p>וְנִפְתָּה בְּמֶתֶק שִׂיחָהּ, / הִטַּתּוּ בְּרֹב לִקְחָהּ.</p> <p>לְרוֹשֵׁשׁ אוֹתוֹ – תַּעֲשִׁיר, / לְקוֹנֵן עָלָיו – תָּשִׁיר,</p> <p>וְצוּף פִּיהָ וְחֶלְבּוֹ – / מְרוֹרַת פְּתָנִים בְּקִרְבּוֹ.</p> <p>שְׂפָתֶיהָ בְּצוּף לֶקַח / יְפַתּוּ כְּיֵין הָרֶקַח –</p> <p>וּמִגֶּפֶן סְדוֹם גַּפְנָם, / חֲמַת תַּנִּינִים יֵינָם.</p> <p>עוֹלָם זְנַח וּשְׁלוֹמָיו / וְאַל תִּתְאָו לְמַטְעַמָּיו –</p> <p>וְיִהְיֶה בְּעֵינֶיךָ נִבְזֶה / מְעַט הַדְּבַשׁ הַזֶּה.</p> <p>אֲנִי אִישׁ בָּעוֹלָם צוֹעֶה / וּבְעִי סְעִפָּיו תּוֹעֶה,</p> <p>נָגוּעַ וּמְנֻסֶּה, / כִּי רַע עָלַי הַמַּעֲשֶׂה.</p> <p>בְּיֵין נֹעַר שָׁכָרְתִּי / וְאַחֲרֵי עֵינַי תָּרְתִּי,</p> <p>וְעֵקֶב הִסְכַּלְתִּי עֲשֹה – / גָּדוֹל עֲוֹנִי מִנְּשׂוֹא.</p> <p>רָמוֹת הָלְכוּ עֵינָי / וּלְזֹאת נָפְלוּ פָנָי,</p> <p>וְכָל מַכְמוֹרֵי כַחֲשִׁי / עֵינִי עוֹלְלָה לְנַפְשִׁי.</p> <p>מִתְּנוּמַת יְמֵי עֶדְנָה / הֱקִיצוּנִי יְמֵי זִקְנָה –</p> <p>וְהִנֵּה כָּל פְּרִי נֹחַ / הֶבֶל וּרְעוּת רוּחַ.</p> <p>יַלְדוּת אֱנוֹשׁ וְשַׁחֲרוּתוֹ / כַּחֲלוֹם רָאָה אוֹתוֹ,</p> <p>וַחֲטָאָיו נִצְמָתִים / וְאַחֲרָיו אֶל הַמֵּתִים.</p> <p>וְיוֹם יִדַּל הַכֹּחַ / אֲזַי יַחֲלֹף הָרוּחַ,</p> <p>וְהַגּוּף בִּשְׁאוֹל מִרְמָס / יִהְיֶה, כְּפֶגֶר מוּבָס.</p> <p>מִשְׁכָּנוֹ יְקִיאֶנּוּ / וְכַנּוֹ יַבְהִילֶנּוּ</p> <p>לָצֵאת מֵחֲדַר בֵּיתוֹ, / מֻשְׁכָּב עַל מִטָּתוֹ.</p> <p>וּמוֹדָעָיו יְרִיצוּהוּ / וּמִתֵּבֵל יְנִידוּהוּ</p> <p>וּבַבּוֹר יֻשְׁלַךְ קִרְבּוֹ, / כִּכְלִי אֵין חֵפֶץ בּוֹ.</p> <p>וְתִיקַר אָז הַנְּשָׁמָה, / אֲשֶׁר מִתַּאֲוָה עֲרֻמָּה,</p> <p>וְתַעֲלֶה, אִם לֹא נִטְמֵאת / הַנֶּפֶשׁ הַחֹטֵאת.</p> <p>הֲשִׁיבוּהָ אֶל מְעוֹנָהּ / כַּאֲשֶׁר הָאֵל נְתָנָהּ –</p> <p>וְאָז אוֹרָהּ יַבְקִיעַ / כְּזֹהַר הָרָקִיעַ.</p> <p>זְנַח מָשׂוֹשׂ וַהֲנָחָה, / כִּי לֵב כְּסִיל בְּבֵית שִׂמְחָה;</p> <p>וּבְכֵה וַחֲגֹר חֶבֶל, / לֵב חֲכָמִים בְּבֵית אֵבֶל.</p> <p>וְהָבָה נִשְׁפֹּךְ שִׂיחָה / לִפְנֵי אֵל וְנִוָּכְחָה;</p> <p>לְכוּ נָא, זֶרַע אֱמוּנָי, / וְנָשׁוּבָה אֶל יְיָ!</p></blockquote>
