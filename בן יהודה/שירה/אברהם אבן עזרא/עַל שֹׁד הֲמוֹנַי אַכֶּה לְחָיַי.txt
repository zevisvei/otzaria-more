<h1>עַל שֹׁד הֲמוֹנַי אַכֶּה לְחָיַי</h1>
אברהם אבן עזרא
<p>(מחרך)</p>
<blockquote><br/> <p>עַל שֹׁד הֲמוֹנַי / אַכֶּה לְחָיַי,</p></blockquote>
<blockquote><p>כִּי מִפְּנֵי צָר / קַצְתִּי בְחָיַי.</p> <br/> <p>אַיֵּה כְבוֹדִי? – / נִמְאָס וְנִרְמָס!</p></blockquote>
<blockquote><p>נֶחְמָס יְסוֹדִי, / יִמַּס בְּעֹל מָס.</p></blockquote>
<blockquote><p>נִשְׁפַּךְ כְּבֵדִי / מִתּוֹךְ וְחָמָס,</p></blockquote>
<blockquote><p>חַיַּי – בְּמוֹתִי, / מוֹתִי – בְּחַיָּי.</p> <br/> <p>בַּת הַשְּׁתַּיִם / נָדֲדָה בְאַרְבַּע,</p></blockquote>
<blockquote><p>קוֹרוֹת שְׁתַּיִם / מָצְאָה בְאַרְבַּע,</p></blockquote>
<blockquote><p>עָזְבָה שְׁתַּיִם, / דָּבְקָה בְאַרְבַּע –</p></blockquote>
<blockquote><p>גָּלְתָה, וְאָרַךְ / קֵץ מָאֲוַיָּי.</p> <br/> <p>רוֹעִים שְׁלֹשָׁה / גָּזְרוּ חֲמִשָּׁה,</p></blockquote>
<blockquote><p>כָּתְבוּ שְׁלֹשָׁה / סִפְרֵי חֲמִשָּׁה,</p></blockquote>
<blockquote><p>עַל כֵּן שְׁלֹשָׁה / עוֹרְרוּ חֲמִשָּׁה –</p></blockquote>
<blockquote><p>אָסְפוּ זְקֵנַי, / כֹּהֲנַי, לְוִיָּי.</p> <br/> <p>מָעוֹן, לְשִׁמְךָ / קַמְתִּי לְשַׁחֵר,</p></blockquote>
<blockquote><p>הִנְנִי לְעָבְדָךְ, / אוֹהֵב וּבוֹחֵר,</p></blockquote>
<blockquote><p>הַקְשֵׁב לְשַׁוְעִי: / נָא אַל תְּאַחֵר –</p></blockquote>
<blockquote><p>נַפְשִׁי הֲלֹא לָךְ / וְנֶפֶשׁ עֲנִיָּי.</p></blockquote>
