<h1>יֶשַׁע בַּשְּׂרוּ אֹם גּוֹלָה</h1>
אברהם אבן עזרא
<blockquote><p>פיוט לחנוכה</p> <br/></blockquote>
<blockquote><p>יֶשַׁע בַּשְּׂרוּ אֹם גּוֹלָה / כִּי בָא יוֹם גְּאוּלִים,</p></blockquote>
<blockquote><p>וִיהִי לָהּ לְשֵׁם וּתְהִלָּה / אֵל שׁוֹכֵן זְבוּלִים.</p> <br/> <p>אַהְבַת אַב הֲמוֹנִים הֵעִיר / הַיּוֹם לַעֲדָתוֹ –</p></blockquote>
<blockquote><p>וַיִּשְׁפֹּךְ עֲלֵי אִישׁ שָׂעִיר / כָּאֵשׁ אֶת חֲמָתוֹ</p></blockquote>
<blockquote><p>וַיַּאְבִיד שְׂרִידָיו מֵעִיר / וַיַּשִּׁים נְוָתוֹ;</p></blockquote>
<blockquote><p>וַיַּמְצִיא צֳרִי וּתְעָלָה / לַעְנִיִּים אֲמֵלִים,</p></blockquote>
<blockquote><p>הַחוֹכִים פְּדוּת וּגְאֻלָּה / וּלְיֶשַׁע מְיַחְלִים.</p> <br/> <p>בֹּאוּ לַהֲגוֹת רֹב טוֹבָה / עָשָׂה מִלְּפָנִים</p></blockquote>
<blockquote><p>וּתְנוּ לוֹ תְּהִלָּה עָרְבָה / וַחְמוּדֵי רְנָנִים,</p></blockquote>
<blockquote><p>כִּי הִשְׁמִיד בְּחֵמָה רַבָּה / מַלְכוּת הַיְּוָנִים –</p></blockquote>
<blockquote><p>יוֹם בָּאוּ לְהָשִׂים תָּהְלָה / בִּילָדִים אֲצִילִים</p></blockquote>
<blockquote><p>וּלְתִתָּן מְעִי מַפָּלָה / מִקְדָּשׁ עִם גְּבוּלִים.</p> <br/> <p>רָם לֵב צַר בְּבוֹאוֹ לִפְנִים / וַיִּגְדַּל גְּאוֹנוֹ,</p></blockquote>
<blockquote><p>וּלְכַלּוֹת בְּנֵי אֵיתָנִים / הִשִּׂיאוֹ זְדוֹנוֹ,</p></blockquote>
<blockquote><p>אוּלָם שָׁב בְּבֹשֶׁת פָּנִים / וַיִּכְלֶה הֲמוֹנוֹ,</p></blockquote>
<blockquote><p>מָתְנָם אָחֲזָה חַלְחָלָה – / כָּל עוֹבְדֵי אֱלִילִים –</p></blockquote>
<blockquote><p>וּלְדָךְ נִתְּנָה הַצָּלָה / וּגְבוּרָה לְכוֹשְׁלִים.</p> <br/> <p>הַכֹּהֲנִים בְּבוֹאָם רָאוּ / פַּךְ אַחַת חֲתוּמָה,</p></blockquote>
<blockquote><p>שֶׁמֶן בָּהּ לְהַדְלִיק מָצְאוּ / יוֹם אֶחָד, וְהֵמָּה</p></blockquote>
<blockquote><p>הִדְלִיקוּ שְׁמוֹנָה מָלְאוּ / בִּבְרָכָה עֲצוּמָה.</p></blockquote>
<blockquote><p>אָז הוֹדוּ לְשׁוֹכֵן מַעְלָה, / כִּי מַעְשָׂיו גְּדוֹלִים,</p></blockquote>
<blockquote><p>כִּי רִפְאוּת לְמַחְלָה הֶעְלָה / וּצְרִי לַעֲמֵלִים.</p> <br/> <p>מַשְׂכִּילִים שְׁמוֹנַת יָמִים / הָאֵלְֶּה לְדוֹרִים</p></blockquote>
<blockquote><p>קָבְעוּ, לִהְיוֹתָם שׂוּמִים / יוֹם שִׂמְחָה וְשִׁירִים,</p></blockquote>
<blockquote><p>גַּם נֵרוֹת יְהוּא מוּקָמִים / עַל פִּתְחֵי שְׁעָרִים:</p></blockquote>
<blockquote><p>זִכָּרוֹן לְנֵס הַנִּפְלָא; / לָכֵן עַל מְחוֹלִים</p></blockquote>
<blockquote><p>שִׁירוּ, וַעֲשׂוּ יוֹם צָהְלָה / וּתְנוּ מַהֲלָלִים.</p> <br/> <p>׳בַּת צִיּוֹן, בְּכָל לֵב עִלְזִי / וּשְׁתִי כוֹס יְשׁוּעוֹת!</p></blockquote>
<blockquote><p>הִנֵּה לָךְ אֲגַלֶּה רָזִי / וַאְקַיֵּם שְׁבוּעוֹת.</p></blockquote>
<blockquote><p>נִשְׁבַּעְתִּי עֲלֵי יַד חוֹזִי / לֶאְסֹף מִתְּנוּעוֹת</p></blockquote>
<blockquote><p>הָעָם, שֶׁלְּקַחְתִּיו נַחְלָה / לִי, וּבְכָל שְׁבִילִים</p></blockquote>
<blockquote><p>אֶקְרָא: סַקְּלוּ הַמְסִלָּה / חִישׁ לַעְבֹר גְּאוּלִים!׳</p></blockquote>
