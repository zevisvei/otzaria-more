<h1>אוֹי כִּי לִבִּי יִדְוֶה בְחֻמִּי</h1>
אברהם אבן עזרא
<p>קינה לט׳ באב</p>
<blockquote><br/> <p>אוֹי כִּי לִבִּי יִדְוֶה בְחֻמִּי, / וְדֶמַע תֵּרֵד עֵינִי וְדָמִי,</p></blockquote>
<blockquote><p>כִּי רֻתְּחוּ וְלֹא דָמּוּ / עַל שְׂרֵפַת אוּלַמִּי,</p></blockquote>
<blockquote><p>יוֹם, אֲשֶׁר בּוֹ אֵרַרְתִּי / אֲשֶׁר יְלָדַתְנִי אִמִּי –</p></blockquote>
<blockquote><p>פַּלְגֵי מַיִם תֵּרֵד עֵינִי עַל שֶׁבֶר בַּת עַמִּי.</p> <br/> <p>יוֹם, אֲשֶׁר בּוֹ יַד בָּבֶל וְרַהַב / בַּהֵיכָל הִשִּׂיק לַהַב,</p></blockquote>
<blockquote><p>בְּיוֹם גָּמַר גְּאוֹן יַעֲקֹב הַנֶּאֱהָב, / אֲשֶׁר הוּעַם זָהָב –</p></blockquote>
<blockquote><p>יוֹם תֵּת צוּרִי קֹדֶשׁ אֲשֶׁר אָהַב / בְּיַד אֲדוֹמִי וַאֲרָמִי.</p> <br/> <p>יוֹם, אֲשֶׁר בְּרֹב חוֹבָתִי הוּבָא / אֶל צִיּוֹן הָאֲהוּבָה,</p></blockquote>
<blockquote><p>וַתְּהִי לְבִזָּה כְאִשָּׁה עֲזוּבָה, / בְּיַד צָרֶיהָ עֲלוּבָה –</p></blockquote>
<blockquote><p>בַּת עַמִּי, בְּכִי וְאַל תִּדֹּמִי, / לְשֹׁד יוֹם זֶה הָשַּׁמִּי!</p> <br/> <p>יוֹם, אֲשֶׁר בָּנַי יְצָאוּנִי – וְאֵינָם, / וְנָמַקּוּ בַּעֲוֹנָם,</p></blockquote>
<blockquote><p>וַאֲבִיהֶם רוֹאֶה עֶלְבּוֹנָם / בְּעֵת צָרָתָם – וְאֵין עוֹנָם,</p></blockquote>
<blockquote><p>אֶל מִי אֶדְרֹשׁ, וּמִמִּי? / אָבִי, רְאֵה עָנְיִי וּמַעֲרֻמִּי!</p> <br/> <p>כְּבֵדִי זוּבִי, זוּבִי כְּבֵדִי / וּמֵעַפְעַפַּי רְדִי!</p></blockquote>
<blockquote><p>וּמְרֵרָתִי, קֵד הוֹקִידִי / וְקִירוֹת לִבִּי הָתֵמִּי!</p></blockquote>
