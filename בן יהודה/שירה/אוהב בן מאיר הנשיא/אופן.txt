<h1>אופן</h1>
אוהב בן מאיר הנשיא
<blockquote><p>אֶרְאֶלִּים / וְחַשְׁמַלִּים / לַאֲלָפִים / וְרִבּוֹאוֹת</p> <p>נִקְבָּצִים / וּמַעֲרִיצִים / בְּרֹב אֵימוֹת / וְנוֹרָאוֹת,</p> <p>לְהַקְדִּישָׁם / יַחַד שֵׁם / אֵל עוֹשֵה / נִפְלָאוֹת –</p> <p>אוֹמְרים “קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!” <br/><br/> וּשְׂרָפִים / אֲשֶׁר עָפִים, / בַּעֲלֵי שֵׁשׁ / כְּנָפַיִם,</p> <p>מִמַּעַל / אֶל מַעַל / חוּג אֶרֶץ / וְשָׁמַיִם –</p> <p>דִּבְרֵיהֶם / וְכַנְפֵיהֶם / כְּקוֹל הָמוֹן / כְּקוֹל מַיִם,</p> <p>כְּנִיב נִמְרָץ / לְאֵל נַעֲרָץ / בִּנְבוּאוֹת / נִבָּאוֹת. <br/><br/> בְּכֵס נִקְבָּע / בְּרֹאשׁ אַרְבַּע / דְמוּת סַפִּיר / הַטְּהוֹרָה.</p> <p>נִיב רַחְשָׁם – / וְעַל רֹאשָׁם / כְּעֵין קֶרַח / הַנּוֹרָא –</p> <p>דְּמוּת בָּזָק / בְּקוֹל חָזָק / מְבָרְכִים בְּשָׂ- / פָה בְרוּרָה,</p> <p>יְרוֹמֵמוּ / וְיַנְעִימוּ / קְדֻשָּׁה כַּ- / אֲשֶׁר תֵּאוֹת. <br/></p></blockquote>
