<h1>מעמד ב: עַל סְגַן</h1>
שלום שבזי
<p>החתימה: שלם יוסף</p>
<blockquote><p>עַל סְגַן רֹאשׁ גְּאוֹנִי / אִישׁוֹנִי בָכָה</p> <p>יוֹם פָּסַע לְפָנַי / וְהָעִיר נֶהְפָּכָה</p> <p>יָהּ, פְּדֵה קְהַל הֲמוֹנַי / מִיּוֹם קִצְפְּךָ</p> <p>אַתָּה יְוֹיָ / לֹא תִכְלָא רַחֲמֶיךָ:</p> <br/></blockquote>
<blockquote><p>שְׂשׂוֹן הָעִיר עָבַר / בְּעֵת כִּי בָא שָׁמְעוֹ</p> <p>וְהָאֵבֶל גָּבַר / כִּי נֶעֱִזַב גִּזְעוֹ</p> <p>וְנִכְנָס לַקֶּבֶר / וּמִי זֶה יוֹשִׁיעוֹ</p> <p>וְנַפְשׁוֹ תִּתְחַבַּר / בְּעֵדֶן תִּדְרוֹכָה:</p> <br/></blockquote>
<blockquote><p>לְקָחוֹ הָאוֹרֵב / שְׁהוּא מַלְאָךְ אַכְזָר</p> <p>וְעוֹדֶנּוּ קָרוֹב / בְּחָכְמָה הִתְאַזָּר</p> <p>חֲצֵרוֹ נִתְחָרֵב / בְּיוֹם עָלָיו נִגְזָר</p> <p>זְכוֹר רַחֲמֶיךָ יְוֹיָ / וַחֲסָדֶיךָ:</p> <br/></blockquote>
<blockquote><p>מֵרֹב הַבִּכְיָה / לְבָבִי נִתְאָנַח</p> <p>וְנַפְשִׁי נָמוֹכָה / כִּי הֲדָרָהּ צָנַח</p> <p>וְהָעִיר נָבוֹכָה / וּכְבוֹדָהּ נִזְנַח</p> <p>יוֹדוּךָ יְוֹיָ / כָּל מַעֲשֶׂיךָ:</p> <br/></blockquote>
<blockquote><p>יְחִידָתִי סִפְדִי / עַל גְּדוֹל הַחֶבְרָה</p> <p>וּבְקִינָה עִמְדִי / בְּחוּצוֹת וּבְחֶדְרָה</p> <p>וּלְצוּרֵךְ הוֹדִי / וְזֶה סוֹף כָּל נִבְרָא</p> <p>כִּי הַכֹּל הוֹלֵךְ / לְעֵמֶק הַבָּכָא:</p> <br/></blockquote>
<blockquote><p>וּמַה-יּוֹעִיל עֹשֶׁר / וְהַהוֹן לִבְעָלָיו</p> <p>רַק עֲשׂוֹת הַיֹּשֶׁר / בְּמֵיטַב מִפְעָלָיו</p> <p>וְיוֹם בּוֹא הַקֶּשֶׁר / לְהִתְעוֹלֵל עָלָיו</p> <p>אִם צַדִּיק נִמְצָא / בְּחֵלֶק טוֹב זָכָה:</p> <br/></blockquote>
<blockquote><p>סוּר מֵרָע בֶּן אִישׁ / וַעֲשֵׂה-טוֹב תָּמִיד</p> <p>וְרֵיחֲךָ אַל תַּבְאִישׁ / אָשׁוּרֶיךָ תַמְעִיד</p> <p>לְאִוַּלְתָךְ תַּכְתִּישׁ / וּבַתּוֹרָה תַצְמִיד</p> <p>כִּי הִיא חַיֶּיךָ / וְאוֹרֶךְ יָמֶיךָ:</p> <br/></blockquote>
<blockquote><p>פְּדֵה נַפְשָׁךְ הַיּוֹם / תַּחְזוֹר לִיסוֹדָךְ</p> <p>וְהַמְצִיא לָהּ פִּדְיוֹם / כְּפִי הֶשֵּׂג יָדָךְ</p> <p>כִּי גָּדוֹל הַיּוֹם / אֲשֶׁר הוּא יוֹם אֵידָךְ</p> <p>כִּי הָרַע וְהַטּוֹב / נָתוּן לְפָנֶיךָ:</p> <br/></blockquote>
