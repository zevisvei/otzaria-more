<h1>חוּס אֱלֹהַי</h1>
שלום שבזי
<p>חסדאי; בשיבוץ עם השיר <strong>יעבר עלי רצונך</strong> של <strong>יהודה הלוי</strong></p>
<blockquote><p><strong>ח</strong>וּס אֱלֹהַי מִמְּעוֹנְךָ / עַל חֲבַצֶּלֶת שְׁרוֹנְךָ</p></blockquote>
<blockquote><p><strong>יַ</strong>עֲבֹר עָלַי רְצוֹנְךָ / כַּאֲשֶׁר עָבַר חֲרוֹנְךָ</p></blockquote>
<br/>
<blockquote><p><strong>שַׂ</strong>גְּבֵנִי מִזְּדוֹנִי / כִּי לְךָ לִבִּי וְעֵינִי</p></blockquote>
<blockquote><p><strong>הַ</strong>לְעוֹלָמִים עֲווֹנִי / יַעֲמֹד בֵּינִי וּבֵינְךָ</p></blockquote>
<br/>
<blockquote><p><strong>דּ</strong>וֹד מְחֵה אוֹיֵב מְעַקֵּשׁ / כִּי לְנֶגְדִּי הוּא לְמוֹקֵשׁ</p></blockquote>
<blockquote><p><strong>וַ</strong>עֲדֵי מָתַי אֲבַקֵּשׁ / אוֹתְךָ עִמִּי וְאֵינְךָ</p></blockquote>
<br/>
<blockquote><p><strong>אַ</strong>תְּ קְרוֹבִי מִקְּרוֹבִים / גַּם סְבִיבֶיךָ שְׁבִיבִים</p></blockquote>
<blockquote><p><strong>דָּ</strong>ר בְּכַנְפֵי הַכְּרוּבִים / הַפְּרוּשִׂים עַל אֲרוֹנְךָ</p></blockquote>
<br/>
<blockquote><p><strong>יָ</strong>דְךָ תַּנְחֶה פְּזוּרִים / בַּעֲבוּר חֶסֶד נְעוּרִים</p></blockquote>
<blockquote><p><strong>הֶ</strong>עֱבַדְתַּנִי לְזָרִים / וַאֲנִי כַּנַּת יְמִינְךָ</p></blockquote>
<br/>
<blockquote><p><strong>גָּ</strong>דְעָה קֶרֶן מְעַנָּי / בַּעֲבוּר שִׁמְךָ יְיָ</p></blockquote>
<blockquote><p><strong>גּ</strong>וֹאֲלִי, לִגְאֹל הֲמוֹנַי / רוּם וְהַשְׁקֵף מִמְּעוֹנְךָ . <br/></p></blockquote>
