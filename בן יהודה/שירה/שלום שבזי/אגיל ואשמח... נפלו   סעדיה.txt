<h1>אגיל ואשמח... נפלו / סעדיה</h1>
שלום שבזי
<p>חתימה: אני סעדיה</p>
<blockquote><p>אָגִיל וְאֶשְׂמַח / וְלָאֵל אֲשַׁבַּח / וְאֶזְבַּח זֶבַח / בְּיוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>נָפְלוּ אוֹיְבַי / וְהֻכּוּ מְרִיבַי / וְנִמְחֲלוּ חוֹבַי / בְּיוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>יוֹם זֶה קָדוֹשׁ / לַמּוֹעֲדִים רֹאשׁ / בָּרוּךְ מְקַדֵּשׁ / יוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>סְבִיב הֵיכָלוֹ / עֲנוּ מַהֲלָלוֹ / וְהַקְרִיבוּ לוֹ / קָרְבַּן פָּסַח:</p></blockquote>
<blockquote><p>עַל שֵׁם גְּאֻלָּה / וְדִיצָה וְצַהְלָה / וְחֶסֶד וְחֶמְלָה / בְּיוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>דָּן מִצְרַיִם / צוּר שָׁמַיִם / בְּתוֹךְ הַמַּיִם / בְּיוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>יָרַד פַּרְעֹה / וְנִכְרַת זַרְעוֹ / וְאָבַד יִשְׁעוֹ / בְּיוֹם הַפָּסַח:</p></blockquote>
<blockquote><p>הִלְּלוּ אֱמוּנִים / אֲדוֹן הָאֲדוֹנִים / וּפָצְחוּ רְנָנִים / בְּיוֹם הַפָּסַח:</p></blockquote>
