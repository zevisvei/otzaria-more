<h1>שְׁלַח יָהּ</h1>
שלום שבזי
<p>חתימה: שלם שץ</p>
<blockquote><p>שְׁלַח יָהּ יֶשַׁע / לְעָם נֶחְמָסִים</p> <p>אֲשֶׁר בְּגָלוּתָם / הֵם נִרְמָסִים</p> <p>בְּיַד צָר רוֹדֶה / וְרָע נֶאְנָסִים</p> <p>הַפְלֵה חֲסָדֶיךָ / מוֹשִׁיעַ חוֹסִים:</p> <br/></blockquote>
<blockquote><p>לְבֵיתְךָ מִתְאַוִּים / לְבַקֵּשׁ סְלִיחָה</p> <p>תָּסִירָה מֵהֶם / יָגוֹן וַאֲנָחָה</p> <p>אֲיֻמָּתְךָ תָּמִיד / עָלֶיךָ בְּטוּחָה</p> <p>יְוֹיָ שִׁמְעָה / יְוֹיָ סְלָחָה</p> <p>וְהַצִּילָה עַמְּךָ / מִיַּד שׁוֹסִים:</p> <br/></blockquote>
<blockquote><p>מָהוּל לִבָּם / בְּחֵטְא וַעֲווֹנִים</p> <p>כִּי נִתְעָרְבוּ / בְּגוֹי עַז פָּנִים</p> <p>וּבְתוֹךְ אֹהֶל / קֵדָר חוֹנִים</p> <p>וְאִם לֹא תִּזְכּוֹר / זְכוּת רִאשׁוֹנִים</p> <p>יִהְיוּ בְּחֶטְאָם / לָעַד מְאוּסִים:</p> <br/></blockquote>
<blockquote><p>שְׁאוֹן אוֹיְבֵיהֶם / בָּהֶם שָׁלַט</p> <p>וּמִי זֶה מֵהֶם / עוֹד יִמָּלֶט</p> <p>וְלוּלֵי חַסְדָּךְ / אוֹתָם קָלַט</p> <p>שָׂרִיד מֵהֶם / לֹא יִפָּלֶט</p> <p>כִּי בְמַעֲלָלָם / הֵם נֶעְלָסִים:</p> <br/></blockquote>
<blockquote><p>צָלַל לִבָּם / בְּגָלוּת אוֹיְבָם</p> <p>וּבִדְבַר שֶׁקֶר / גָּבַהּ לִבָּם</p> <p>כְּאֵבָם צָפוּן / בִּצְפוּן חֻבָּם</p> <p>מִתִּגְרַת צָר / הַמּוֹשֵׁל בָּם</p> <p>מָתַי תַּעֲשֶׂה / לָהֶם נִסִּים:</p> <br/></blockquote>
