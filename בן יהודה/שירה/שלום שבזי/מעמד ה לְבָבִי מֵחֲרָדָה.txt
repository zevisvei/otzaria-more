<h1>מעמד ה: לְבָבִי מֵחֲרָדָה</h1>
שלום שבזי
<p>החתימה: לשלם הקטן</p>
<blockquote><p>לְבָבִי מֵחֲרָדָה / בְּגוּפִי נִתְעַלָּם</p> <p>עַל צִיר הַתְּעוּדָה / הַגְּבִיר הַנִּשְׁלָם</p> <p>הוֹי עַל כְּלִי חֶמְדָּה / מְסַלְסֵל סוֹד נֶעְלָם</p> <p>יְהִי שֵׁם יְוֹיָ מְבוֹרָךְ / מֵעַתָּה וְעַד עוֹלָם:</p> <br/></blockquote>
<blockquote><p>שָׁקַט הַמִּדְרָשׁ / וְגַם בֵּית הַכְּנֶסֶת</p> <p>עַל זֶה שְֶׁנִּגְרָשׁ / שְׂשׂוֹנָהּ מוֹאֶסֶת</p> <p>כִּי מָוֶת בָּלַשׁ / וּבָא לוֹ בְּהֶסֶּת</p> <p>וְאֵיכָה מִמֶּנּוּ / נוּכַל נִתְעַלָּם:</p> <br/></blockquote>
<blockquote><p>לַכֹּל הוּא שׂוֹטֵם / וְשׁוֹלֵט כַּמֶּלֶךְ</p> <p>לָרוּחַ אוֹטֵם / וּבַגּוּף לֹא תֵלֵךְ</p> <p>בְּאַף הַחַי חוֹטֵם / בְּזַעְפּוֹ עַד יֻשְׁלַךְ</p> <p>עוֹכֵר הַבִּרְיוֹת / וּמֵסִיר אֶת צִלָּם:</p> <br/></blockquote>
<blockquote><p>מַה-יַּעֲשֶׂה אָדָם / וְאֵין בְּיָדוֹ יְכוֹלֶת</p> <p>וְזֶה שִׂטְנוֹ קָדַם / שְׁהוּא אֵשׁ אוֹכֶלֶת</p> <p>וְכָל הַיצוּר יְסוֹדָם / מִגּוּשׁ נִגְבֶּלֶת</p> <p>וְחוֹזְרִים לִיסוֹדָם / וְיִתְחַדֵּשׁ אֶבְלָם:</p> <br/></blockquote>
<blockquote><p>הֵיךְ תְּהֶא לוֹ תְקוּמָה / אִישׁ מִן הֶעָפָר</p> <p>כִּי נַפְשׁוֹ עֲמוּמָה / מֵעֵת שֶׁנּוּצָּר</p> <p>וּבְעָווֹן וְאַשְׁמָה / יִצְרוֹ בוֹ יִפְצָר</p> <p>וּמִתִּגְרַת מַעֲשָׂיו / יָשׁוּב דָּךְ נִכְלָם:</p> <br/></blockquote>
<blockquote><p>קוֹדֶם יְצִירָתוֹ / מִטִּפָּה סְרוּחָה</p> <p>וְאַחַר בְּרִיאָתוֹ / צוּרָה נִמְתָּחָה</p> <p>וְיוֹם יָבוֹא עִתּוֹ / וְנַפְשׁוֹ נִלְקָחָה</p> <p>שָׁב אֶל הֶעָפָר / מִתְוַדֶּה וְנֶאֱלָם:</p> <br/></blockquote>
<blockquote><p>טוֹב יָחִיל וְדוּמָם / וְאִם לָאו מַה-יָּשִׁיב</p> <p>לְאַחַר שֶׁנֶּעְמָם / בַּהֲבָלִים יַחְשִׁיב</p> <p>וּמָוֶת פִּיו זָמָם / וְאָזְנוֹ מִלְּהַקְשִׁיב</p> <p>וְאֵידוֹ אָרַב לוֹ / וְאֶת רֹאשׁוֹ הָלָם:</p> <br/></blockquote>
<blockquote><p>נַחֵם אֲבֵלֵינוּ / אֱלֹהִים מִמְּרוֹמָךְ</p> <p>וְהַשְׁקֵף לְבָרְכֵנוּ / מִתּוֹךְ שְׁמֵי מְרוֹמָךְ</p> <p>וַעֲנֵה קָרְאֵנוּ / וַעֲשֵׂה לְמַעַן שְׁמָךְ</p> <p>וּבְיוֹם שַׁוְעֵנוּ / יָהּ אַל תִּתְעַלָּם:</p> <br/></blockquote>
