<h1>אֲיֻמָּה הַמְשִׁי</h1>
שלום שבזי
<h2 id="רשלוםשבזי">/ ר' שלום שבזי</h2>
<p>חתימה: אלשבזי (ילוד משתא)</p>
<h3 id="א">א</h3>
<blockquote><p>אֲיֻמָּה הַמְשִׁי / דִּגְלֵי הֲמוֹנַי מִמְּצוּלָה</p></blockquote>
<blockquote><p>לְדוֹדֵךְ בִּקְשִׁי / יָקִים לְסֻכָּה הַנְפוּלָה</p></blockquote>
<blockquote><p>לְחַסְדוֹ דִרְשִי / תִּהְיִי בְּרֹב שִׂמְחָה סְגֻלָה</p></blockquote>
<blockquote><p>וְהַנְחִי בָנִים</p></blockquote>
<blockquote><p>עֲדַת אֵיתָנִים</p></blockquote>
<blockquote><p>שֶׁהֶם נֶאֱמָנִים</p></blockquote>
<blockquote><p>בְּאֶרֶץ קָדְשִׁי / נִשְׂמַח בְּבוֹא כַּלָּה כְּלוּלָה</p></blockquote>
<blockquote><p>וּמַה טוֹב שָׁרְשִׁי / צַמֶּרֶת אֶרֶז הַשְּׁתוּלָה: <br/></p></blockquote>
<h3 id="ב">ב</h3>
<blockquote><p>לְדוֹדִי חוֹשֵׁק / כִּי הוּא מְנַת כּוֹסִי וְחַלְקִי</p></blockquote>
<blockquote><p>וּמִיַּד עוֹשֵׁק / הוּא גּוֹאֲלִי תָּמִיד וְחַזְקִי</p></blockquote>
<blockquote><p>וְדָתוֹ נוֹשֵׁק / הוּא יִתְּנָה צָרְכִּי וְסָפְקִי</p></blockquote>
<blockquote><p>מְפַרְנֵס לְכֹל</p></blockquote>
<blockquote><p>וְזַן וּמְכַלְכֵּל</p></blockquote>
<blockquote><p>עַל יַד בַּת כֹּל</p></blockquote>
<blockquote><p>בְּחֵן וּבְרַחְשִׁי / טוּבוֹ בְּכָל מָקוֹם מְמֻלָּא</p></blockquote>
<blockquote><p>וְקָו הַשִּׁשִּׁי / הִנְחִיל עֲדָתוֹ הָאֲצֻלָּה:</p></blockquote>
<br/>
<h3 id="ג">ג</h3>
<blockquote><p>שְׁמוֹ יִתְבָּרַךְ נוֹרָא, / מָעוֹן כָּל הַבְּרָכוֹת</p></blockquote>
<blockquote><p>לְשֻׁלְחָן עָרַךְ לָנוּ, / וְרֹב טוֹבוֹת מְשׁוּכוֹת</p></blockquote>
<blockquote><p>יְעוֹרֵר חַדְרֵךְ / וִיפַרְנְסָה שוֹנֵי הֲלָכוֹת</p></blockquote>
<blockquote><p>יְצַו אֶרְאֶלִּים</p></blockquote>
<blockquote><p>וְהַגַּלְגַּלִים</p></blockquote>
<blockquote><p>לְהוֹסִיף טַלִּים</p></blockquote>
<blockquote><p>וְיַזְרִיח שִׁמְשִׁי / וִיקוֹמְמָה קִרְיָה שְׁכוּלָה</p></blockquote>
<blockquote><p>וְיָרוּם רָאשִׁי / נַפְשִׁי בְּחֵן תִּהְיֶה גְּאֻלָּה:</p></blockquote>
<br/>
<h3 id="ד">ד</h3>
<blockquote><p>בְּלִיל חָלַמְתִּי / וַאֲנִי בְּאוּזַל הַבְּנוּיָה</p></blockquote>
<blockquote><p>וְנִתְחַבַּרְתִּי / עִם הַיְּשִׁיבָה הַשְׁרוּיָה</p></blockquote>
<blockquote><p>וְאָז שָׂמַחְתִּי / לִשְׁתוֹת בְּחֵן כּוֹסִי רְוָיָה</p></blockquote>
<blockquote><p>וְשָׂמַח לִבִּי</p></blockquote>
<blockquote><p>בַּמִּשְׁתֶּה עִנְבִּי</p></blockquote>
<blockquote><p>מִשְׁמָר אִנְבִּי</p></blockquote>
<blockquote><p>וְגֶפֶן שִׁבְשִׁי / הֵם כַּחֲצִי גֹּרֶן עֲגֻלָה</p></blockquote>
<blockquote><p>לִבְשִׁי בִּגְדֵי פָּאֵר, / שִׂמְחִי כְּאַוְלָא</p></blockquote>
<br/>
<h3 id="ה">ה</h3>
<blockquote><p>זְכוּת הַתּוֹרָה / תָּגֵן עָלַי שִׁבְטֵי עֲדָתִי</p></blockquote>
<blockquote><p>בַּקֶּרֶן אוֹרָה / אַל חַי יְסוֹבֵב נַחַלָתִי</p></blockquote>
<blockquote><p>יְצַו מְהֵרָה חַסְדוֹ, / יְהֵא חֵלֶק מָנָתִי</p></blockquote>
<blockquote><p>יְקַנֵּא לִשְׁמוֹ</p></blockquote>
<blockquote><p>וְיַצִּיל עַמּוֹ</p></blockquote>
<blockquote><p>וְאֵשׁ דָּת לָמוֹ</p></blockquote>
<blockquote><p>בְּיוֹם וּבְרַמְשִׁי / תַּנְהִיר לְדַרְכִּי הַסְּלוּלָה</p></blockquote>
<blockquote><p>יְחִידָה חוּשִׁי / רַב לָךְ בְּיָם טָבַע מְצוּלָה:</p></blockquote>
<br/>
<h3 id="ו">ו</h3>
<blockquote><p>יִלּוֹד מִשַּׁתָּא / שִׁית לִבְּךָ לְחָבֵר שִׁיר נְגִינָה</p></blockquote>
<blockquote><p>וְקַח לְךָ רֵאשִׁית / תּוֹרַת אֱמֶת חָכְמָה וּבִינָה</p></blockquote>
<blockquote><p>בְּמִּדָּה שִּׁשִּׁית / קוּמָה אֲמָרַי הֶאֱזִינָה</p></blockquote>
<blockquote><p>לְצוּר יַחֵדָה</p></blockquote>
<blockquote><p>וְהָכֵן צֵידָה</p></blockquote>
<blockquote><p>וְקָרְבָּן תּוֹדָה</p></blockquote>
<blockquote><p>וְטוֹב לֹא תִּנְשִׁי / סוֹד הַתְּעוּדָה לֹא יְסוּלָּא</p></blockquote>
<blockquote><p>וְתָשׁוּב חָפְשִׁי / חַטָּתְךָ תִּהְיֶה מְחוּלָה <br/></p></blockquote>
