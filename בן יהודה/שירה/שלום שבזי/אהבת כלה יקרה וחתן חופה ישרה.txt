<h1>אהבת כלה יקרה וחתן חופה ישרה</h1>
שלום שבזי
<h2 id="אַהֲבַתכַּלָּהיְקָרָהוַחֲתַןחוּפָּהיְשָׁרָהרשלוםשבזי">[אַהֲבַת כַּלָּה יְקָרָה וַחֲתַן חוּפָּה יְשָׁרָה] / ר' שלום שבזי</h2>
<br/>
<blockquote><p>אַהֲבַת כַּלָּה יְקָרָה,</p></blockquote>
<blockquote><p>וַחֲתַן חוּפָּה יְשָׁרָה,</p></blockquote>
<blockquote><p>דּוּגְמַת אַבְרָהָם וְשָׂרָה,</p></blockquote>
<blockquote><p>רֹאשׁ תְּחִלַּת אַהֲבָתִי.</p></blockquote>
<br/>
<blockquote><p>לָהּ מְאֹד הוֹמֶה לְבָבִי,</p></blockquote>
<blockquote><p>חוֹשְׁקִים לִבְיָה וְלָבִיא,</p></blockquote>
<blockquote><p>לַעֲרוֹךְ נֵר מַעֲרָבִי,</p></blockquote>
<blockquote><p>אֶלְמְדָה חֻקִּי וְדָתִי.</p></blockquote>
<br/>
<blockquote><p>שׁוּר גְּבֶרֶת, דַע בְּסוֹדָהּ,</p></blockquote>
<blockquote><p>חוֹשְׁקָהּ תָּמִיד לְדוֹדָהּ,</p></blockquote>
<blockquote><p>כִּי בְּאוֹר הוֹדוֹ יְסוֹדָהּ,</p></blockquote>
<blockquote><p>יִקְרְאָה לָהּ רַעֲיָתִי.</p></blockquote>
<br/>
<blockquote><p>בָּא בְּרֹאשׁ הָרִים, יְדַלֵּג</p></blockquote>
<blockquote><p>בֵּין הֲדַס, לוֹלָב וְאֶתְרוֹג,</p></blockquote>
<blockquote><p>זֶה וְזֶה יַחְדָו יְשָׂרֵג,</p></blockquote>
<blockquote><p>וַעֲרָבָה סוֹף קְצָתִי.</p></blockquote>
<br/>
<blockquote><p>זוּמְנוּ שִׁבְטַי בְּהַלֵּל,</p></blockquote>
<blockquote><p>לָבְשׁוּ יוֹפִי וּמִכְלוֹל,</p></blockquote>
<blockquote><p>סוֹדְרִים שִׁירָה לְגוֹלֵל,</p></blockquote>
<blockquote><p>אוֹר מְהוּדָּר לַעֲדָתִי.</p></blockquote>
<br/>
<blockquote><p>יָרְדוּ צִבְאוֹת שְׁחָקִים,</p></blockquote>
<blockquote><p>שָׁמְעוּ קוֹלוֹת בְּרָקִים,</p></blockquote>
<blockquote><p>דִּבְּרוֹת עֶשֶׂר חֲקוּקִים</p></blockquote>
<blockquote><p>הֵן בְּלוּחוֹת שֶׁנְּחַלְתִּי.</p></blockquote>
<br/>
<blockquote><p>מעננים וערפל</p></blockquote>
<blockquote><p>מקהלי כרע ונפל</p></blockquote>
<blockquote><p>החיים צורי, וכפל</p></blockquote>
<blockquote><p>את שׂכר עמי ובאתי.</p></blockquote>
<br/>
<blockquote><p>שָׁמְעוּ עַמִּים וְזָעוּ,</p></blockquote>
<blockquote><p>רָעֲשׁוּ קָדִים וְנָעוּ,</p></blockquote>
<blockquote><p>נִבְהֲלוּ אוֹיְבַי וְלָעוּ</p></blockquote>
<blockquote><p>מֵהֲדָרִי יוֹם עֲלוֹתִי.</p></blockquote>
<br/>
<blockquote><p>תִּקְּנוּ שִׁבְטֵי יְשָׁרִים</p></blockquote>
<blockquote><p>שָׁם שְׁנֵים עָשָׂר סְפָרִים,</p></blockquote>
<blockquote><p>גַּם וְשִׁשָּׁה הַסְּדָרִים,</p></blockquote>
<blockquote><p>הֵם וּמֹשֶׁה הָאֲמִתִּי.</p></blockquote>
<br/>
<blockquote><p>אַהֲבָתוֹ הָעֲדוּפָה</p></blockquote>
<blockquote><p>הִיא בְּאַפִּרְיוֹן רְצוּפָה,</p></blockquote>
<blockquote><p>זוֹרְחָה רֹאשׁ כָּל תְּקוּפָה,</p></blockquote>
<blockquote><p>אָף תְּקַם עַל רֹאשׁ שְׂפָתִי.</p></blockquote>
<br/>
<blockquote><p>שַׁאֲלִי רַעְיָה בְּעָבוּר</p></blockquote>
<blockquote><p>הַנְהֲרִי, יָחִיד וְצִבּוּר,</p></blockquote>
<blockquote><p>רֹאשׁ חֳדָשִׁים בָּא בְּדִבּוּר,</p></blockquote>
<blockquote><p>וַאֲנִי בִּנְהַר פְּרָתִי.</p></blockquote>
<br/>
<blockquote><p>צָלְחוּ יַחְדָּו חֲתָנַי,</p></blockquote>
<blockquote><p>יוֹם עֲלוֹת בִּשְׁנֵי אֲרוֹנַי,</p></blockquote>
<blockquote><p>אַהֲרֹן וּמֹשֶׁה לְפָנַי,</p></blockquote>
<blockquote><p>נִשְׁמְעָה שִׁירַת קְהָתִי.</p></blockquote>
