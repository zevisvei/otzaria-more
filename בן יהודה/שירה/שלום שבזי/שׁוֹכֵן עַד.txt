<h1>שׁוֹכֵן עַד</h1>
שלום שבזי
<p>חתימה: שלם שץ</p>
<blockquote><p>שׁוֹכֵן עַד וְקָדוֹשׁ / בְּחַשְׁמַלֵּי מְרוֹמִים</p> <p>יָחִישׁ יְחַיֵּנִי / וְיִמְחוֹל לַאֲשָׁמִים</p> <p>וְיִהְיֶה לִי לְעֶזְרָה / מֵאוֹיְבִים וְקָמִים</p> <p>וְנָשׁוּבָה לְקַדְּשׁוֹ / וְנַקְרִיבָה שְׁלָמִים</p> <p>שִׁבְתִּי בְּבֵית יְוֹיָ / לְאוֹרֶךְ יָמִים:</p> <br/></blockquote>
<blockquote><p>לְבֵית קָדְשׁוֹ יְכוֹנֵן / הֵיכָל וַעֲזָרוֹת</p> <p>וְנָשׁוּבָה בְּחֶמְלָה / וְנִשְׁמַע קוֹל בְּשׂוֹרוֹת</p> <p>וְנִרְאֶה חֵל וְחוֹמָה / וְעֶשֶׂר הַמְּנוֹרוֹת</p> <p>וְכֹהֵן הַמְקֻדָּשׁ / בְּיוֹם כִּפּוּר יְשָׁרֵת</p> <p>וּלְוִיִּם בְּדוּכָן / בְּשִׁירוֹתָם נְעִימִים:</p> <br/></blockquote>
<blockquote><p>מִזְבֵּחַ וְקָרְבָּן / וַעֲלִיַּת רְגָלִים</p> <p>וּמָשִׁיחַ וְסֶגֶן / וְאַרְבָּעָה דְּגָלִים</p> <p>וּשְׂעִירִים שְׁנַיִם / בְּגוֹרָל וַחֲבָלִים</p> <p>וּפָרָה הַעֲשׂוּיָה / לְטַהַר מִגְּעָלִים</p> <p>וְתוֹרָה הַהֲדוּרָה / בְּכִנּוֹר וַחֲלִילִים</p> <p>וְשָׁם נַקְרִיב לְרָצוֹן / קָרְבָּנִים תְּמִימִים:</p> <br/></blockquote>
<blockquote><p>שְׂכָרֵנוּ יְהֶא שָׁם / לְמוּל הָאֵל מְרֻבָּה</p> <p>כְּמוֹ גַּם אִישׁ חֲמוּדוֹת / בְּדִבּוּרוֹ וְנִבָּא</p> <p>וְאַשְׁרֵי הַמְחַכֶּה / כִּי יְשׁוּעָה קְרוֹבָה</p> <p>וְנִרְאֶה חֶפְצֵנוּ / בְּרוֹב שָׂשׂוֹן וְאַהְבָה</p> <p>וְאָז נִשְׂמַח וְנִרְאֶה / בְּשִׂמְחַת הַשְּׁאוּבָה</p> <p>וְסַנְהֶדְרֵי גְּדוֹלָה / סְגֻלַּת הַחֲכָמִים:</p> <br/></blockquote>
<blockquote><p>צְלַח נָא בֶּן גְּבִירָה / בְּרוֹב שִׂמְחָה וְצַהֲלָה</p> <p>וְשַׁבֵּחַ לְצוּרָךְ / שְׁהוּא אַדִּיר וְנַעֲלָה</p> <p>וְהָבִין מַעֲשֵׂהוּ / שְׁהוּא נוֹרָא וְנִפְלָא</p> <p>וְאָז תִּזְכֶּה וְתִרְאֶה / לְיוֹם בּוֹא הַגְּאֻלָּה</p> <p>תְּבָרֵךְ אֵל וְתוֹדֶה / אֲשֶׁר לוֹ הַגְּדֻלָּה</p> <p>וְאִם הֵם הַדְּבָרִים / סְתוּמִים וַחֲתוּמִים:</p> <br/></blockquote>
