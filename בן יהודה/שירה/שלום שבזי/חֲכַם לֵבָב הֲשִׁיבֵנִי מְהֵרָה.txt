<h1>חֲכַם לֵבָב הֲשִׁיבֵנִי מְהֵרָה</h1>
שלום שבזי
<p>חתימה: (20: שלם ילוד יוסף)</p>
<blockquote><br/> <p>חֲכַם לֵבָב הֲשִׁיבֵנִי מְהֵרָה / וְהִתְבּוֹנֵן בְּסוֹד חָכְמָה וְתוֹרָה:</p></blockquote>
<blockquote><p>דְּעֶה הָבֵן וְהָשֵׁב מִשְׁאֲלוֹתַי / וְעַבְדֶּךְ אֲהוּבִי תַעֲזֹרָה:</p></blockquote>
<blockquote><p>הֲשִׁיבֵנִי בְּעִנְיַן הַיְּצוּרִים / וְחִלּוּקָם בְּסוֹד גֹּלֶם וְצוּרָה:</p></blockquote>
<blockquote><p>וְלָמָּה נִבְרְאוּ הַכֹּל לְשִׁשָּׁה / דְּרֹשׁ רִבִּי עֲלֵי זֹאת וַחֲקֹרָה:</p></blockquote>
<blockquote><p>וְכֵיצַד נִדְבְּקָה נֶפֶשׁ בְּתוֹךְ גּוּף / וְגַם כֵּיצַד בְּעֵת כִּי תַחֲזֹרָה:</p></blockquote>
<blockquote><p>וְלָמָּה גּוּף יְהֶא יוֹרֵד לְמַטָּה / וְהִיא תַעְלֶה לְמַחְצֵבָהּ טְהוֹרָה:</p></blockquote>
<blockquote><p>וְכֵיצַד תִּדְבְּקָה עֵת דִּין וְחֶשְׁבּוֹן / בְּעֵת תֵּצֵא לְחשֶׁךְ אוֹ לְאוֹרָה:</p></blockquote>
<blockquote><p>וְהֵיכָן תַּעֲרוֹכָה דִּין וְהָשֵׁב / וְהֵיכָן גּוּף יְהֶא שָׁעַת גְּזֵרָה:</p></blockquote>
<blockquote><p>וְלָמָּה נֶחְלְקוּ צַדִּיק וְרָשָּע / גְּזֵרַת אֵל אֲשֶׁר יָצַר וּבָרָא:</p></blockquote>
<blockquote><p>וְהוּא צַדִּיק טְהוּר עַיִן וְתָמִים / וְגַם אוֹהֵב בְּנֵי דֶרֶךְ יְשָׁרָה:</p></blockquote>
<blockquote><p>וְלָמָּה זֶּה יְהֶא צַדִּיק וְיָשָׁר / וְזֶה רָשָׁע וּמַרְבֶּה בַּעֲבֵרָה:</p></blockquote>
<blockquote><p>וְלָמָּה נִמְסְרָה נֶפֶשׁ יְקָרָה / לְגּוּף טָמֵא וְהָיַת בּוֹ קְשׁוּרָה:</p></blockquote>
<blockquote><p>וְלָמָּה נִמְסְרוּ שָׁלֹשׁ נְפָשׁוֹת / וְאֵיזוֹ הִיא בְּחִבָּתָהּ יְתֵרָה:</p></blockquote>
<blockquote><p>וְאֵיבָרִים רְמַ“ח לָמָּה בְּעִנְיָן / וְהַגִּידִים שְׁסַ”ה אוֹתָם סְפֹרָה:</p></blockquote>
<blockquote><p>וְאֶצְבָּעוֹת שְׁהֶן עֶשֶׁר בְּעֶשֶׁר / הֲשִׁיבֵנִי וְחָצְנֶךָ נְעָרָה:</p></blockquote>
<blockquote><p>וְלָמָּה נִבְרְאוּ אַרְבַּע יְסוֹדוֹת / וְעוֹד כֵּיצַד תְּהֶא לָהֶם חֲבוּרָה:</p></blockquote>
<blockquote><p>וְאֵיזֶה מַחֲצֵב עָנִי וְעָשִׁיר / מְבוּכָתִי אֲהוּב לִבִּי הֲסִירָה:</p></blockquote>
<blockquote><p>הֲשִׁיבֵנִי בְּחַיֶּיךָ אֲהוּבִי / הֲלֹא נַפְשִׁי תְּשׁוּבָתְךָ סְבוּרָה:</p></blockquote>
<blockquote><p>וְאַל תַּטִּיח עֲלֵי עַבְדָּךְ דְּבָרִים / עֲלֵי הַשִּׁיר אֲשֶׁר בָּא בַהֲעָרָה:</p></blockquote>
<blockquote><p>שְּמִי שָׁלֹם יְלוּד יוֹסֵף אֲשַׁלַּח / לְרָב דָּוִד נְשִׂיא רֹאשׁ הַחֲבוּרָה:</p></blockquote>
<blockquote><p>וְשָׁלוֹם רָב בְּכָל רֶגַע יְסוֹבֵב / וְגַם יַזְהִיר כְּמוֹ אוֹר הַמְּנוֹרָה:</p></blockquote>
