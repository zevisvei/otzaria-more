<h1>אִם לִבִּי</h1>
שלום שבזי
<p>חתימה: אלשבזי</p>
<blockquote><p>אִם לִבִּי / בְּעַרְבִי / מִמַּיִם זֵדוֹנִים</p> <p>וָאֶצְמָא / לְמֵימָה / וְאֵין בִּי כֹּחַ אוֹנִים</p> <p>וְעֵין שִׂכְלִי / יוֹם חֶדְלִי / תַּזְכִּירֵנִי יְגוֹנִים</p></blockquote>
<blockquote><p>רַחֲמֵנוּ / צוּרֵנוּ / כְּרַחֵם אָב עַל בָּנִים:</p> <br/></blockquote>
<blockquote><p>לְהַקְבִּיל דּוֹד / אֶתְנוֹדֵד / וְאֵין בְּפַעֲמַי כּוֹחַ</p> <p>כִּי פִשְׁעִי / שָׂם צַלְעִי / כְּשֶׂה מוּכָן לִטְבּוֹחַ</p> <p>בְּצַלְמָוֶת / גּוּף עִוֵּת / עֲדֵי יִדְכֶּה יָשׁוֹחַ</p></blockquote>
<blockquote><p>וְכִמְעַט אֶמְ/צָא רוּחַ / וּמָשַׁל גּוֹי עַז פָּנִים:</p> <br/></blockquote>
<blockquote><p>שְׁכִינַת אֵל / רְצִי שׁוֹאֵל / וּזְכוּת לַמְּדִי עָלָיו</p> <p>וְיִתְבּוֹנֵן / לְהִתְכּוֹנֵן / בְּאוֹת אַרְבַּעַת דְּגָלָיו</p> <p>בְּנִיב רַחְשׁוֹ / וְסוֹד נַפְשׁוֹ / יְוֹיָ נִצָּב עָלָיו</p></blockquote>
<blockquote><p>וּבְטוֹבַת / מַהֲלָלָיו / יְדַבֵּר שִׁיר וּרְנָנִים:</p> <br/></blockquote>
<blockquote><p>בְּאַהְבַת בַּת / נִצֶּבֶת / לְמַעְלָה מִמֶּרְכָּבָה</p> <p>רְצֵה שַׁוְעִי / צוּר יִשְׁעִי / לְעַמָּךְ כֹּפֶר הָבָה</p> <p>בִּזְכוּת חֵן / בְּנֵי בִטְחוֹן / דְּרוֹשׁ סְגֻלָּה נֶאְהָבָה</p></blockquote>
<blockquote><p>רְפָא נֶפֶשׁ / נֶעְלָבָה / בֵּין אוֹיְבִים חַמָּנִים:</p> <br/></blockquote>
<blockquote><p>זֶה אֵלִי / שְׁמַע קוֹלִי / לְעַמָּךְ הַמְצִיא כֹפֶר</p> <p>רְפָא חוֹלֶה / שְׁהוּא גּוֹלֶה / בְּרִית אָבוֹת אַל תּוּפַר</p> <p>וְשִׂים שָׁלוֹם / לְדָל חוֹלֵם / כְּמֵי גִּיחוֹן וּמֵי פַרְפַּר</p></blockquote>
<blockquote><p>לְעֶזְרָתִי / צְלַח עוֹפֶר / וְרַחֵם עָם נֶאֱמָנִים:</p> <br/></blockquote>
<blockquote><p>יְצַו חַסְדּוֹ / לִידִידוֹ / רָם וְנִשָּׂא עֲלֵי הַכֹּל</p> <p>וְיוֹפִיעַ / לְהַשְׁפִּיעַ / לִבְנֵי דַּעַת וְהַשְׂכֵּל</p> <p>וְטוּב מָזוֹן / לְדָל נִזּוֹן / מֵחַסְדּוֹ וּמִתְכַלְכֵּל</p></blockquote>
<blockquote><p>וְהוּא נִשְׁעָן / עֲלֵי בַת כֹּל / אֲשֶׁר הִיא אֵם הַבָּנִים:</p> <br/></blockquote>
