<h1>מעמד ו: הוֹי עַל רֵעַ</h1>
שלום שבזי
<p>החתימה: שלם יושף</p>
<blockquote><p>הוֹי עַל רֵעַ טוֹב / רֹאשׁ חַכְמֵי הַדּוֹר</p> <p>אֲשֶׁר הוּא כְּגַן רָטוֹב / עֲלֵי נַחַל גּוֹבֵר:</p> <br/></blockquote>
<blockquote><p>שׁוֹפֵעַ בְּסוֹדוֹת / כְּמוֹ אַב הַיְּשִׁיבָה</p> <p>וּמַבִּיעַ חִידוֹת / בְּכָל סוֹד יְשִׁיבָה</p> <p>וּבְחֻקִּים וְעֵדוֹת / לְכָל לֵב יֵיטִיבָה</p> <p>יְשַׂמַּח עָצֵב / בְּעֵת יִהְיֶה דוֹבֵר:</p> <br/></blockquote>
<blockquote><p>לְבָבִי יוֹחִילָה / בְּזָכְרִי שֶׁנִּפְטָר</p> <p>וְנַפְשִׁי תֵילִילָה / בְּמִי עוֹד תִּתְעַטָּר</p> <p>וּמָוֶת בַּעֲלִילָה / בָּא עָלָיו וְנִפְטָר</p> <p>וְאָרַב לוֹ כַדֹּב / בְּעֵמֶק הַמָּצוֹר:</p> <br/></blockquote>
<blockquote><p>מִי זֶה יְנַחֲמֵנוּ / עַל רֹאשׁ הַתְּעוּדָה</p> <p>אֲֲשֶׁר הוּא מַשְׁקֵנוּ / מִמַּעְיַן הַנְּגוּדָה</p> <p>הֲדָרוֹ בֵינֵינוּ / כְּמוֹ רַבִּי יְהוּדָה</p> <p>לְתוֹרַת אֵל אוֹהֵב / וְהַמִּצְווֹת שׁוֹמֵר:</p> <br/></blockquote>
<blockquote><p>יָשָׁר בְּמִפְעָלוֹ / וְהוֹסִיף בְּחָכְמָה</p> <p>בְּכָל יוֹמוֹ וְלֵילוֹ / כְּמוֹ גִּבּוֹר נִדְמָה</p> <p>מְרַוֶּה מִלּוּלוֹ / לְכָל מִי שֶׁיִּצְמָא</p> <p>זִכְרוֹ כְּיֵין הַטּוֹב / נָבוֹן לֵב הַטָּהוֹר:</p> <br/></blockquote>
<blockquote><p>וְאֵיךְ אֶמְצָא דִמְיוֹן / לְחָבֵר הַנָּעִים</p> <p>לְבָבִי כָּאֶבְיוֹן / עַל נְטַע שַׁעְשׁוּעִים</p> <p>וְיָדַי מֵרִפְיוֹן / טוֹפְחוֹת בִּזְוָעִים</p> <p>וּמִי יוּכַל יָשׁוֹב / וּמִסָּבִיב מָגוֹר:</p> <br/></blockquote>
<blockquote><p>שְׂשׂוֹנִי פָנָה / אַחַר אִישׁ חֲמוּדוֹת</p> <p>וְגַם לִבִּי עָנָה / נְהִיָּה וּסְפִידוֹת</p> <p>וְחָכְמָה נִצְפָּנָה / וּרְמָזִים וְסוֹדוֹת</p> <p>וְלִבִּי הוּא כוֹאֵב / וְאֵינוּ מִתְעוֹרֵר:</p> <br/></blockquote>
<blockquote><p>פָּנָה הֲדַר הָעִיר / אַחַר שֶׁנֶּאֱסָף</p> <p>גָּדוֹל וְגַם צָעִיר / כְּאֵב לִבָּם יָסָף</p> <p>וְאֵיךְ לִבִּי אָעִיר / וְלַחָכְמָה יִכְסָף</p> <p>וּמִי אֶמְצָא אוֹהֵב / לְתוֹרַת אֵל חוֹקֵר:</p> <br/></blockquote>
