<h1>מעמד ג: עֵינִי רַב</h1>
שלום שבזי
<p>החתימה: בן יוסף</p>
<blockquote><p>עֵינִי רַב בִּכְיָה / מִקּוֹל הַמִּסְפֵּד</p> <p>מִי יְחַזֵּק רִפְיָהּ / כִּי דִמְעָה יֵרֵד</p> <p>עַל סְגֻלַּת שִׁבְיָה / הָרָב הַמְלַמֵּד</p> <p>אִם עֲווֹנוֹת תִּשְׁמָר-יָהּ / יְוֹיָ מִי יַעֲמוֹד:</p> <br/></blockquote>
<blockquote><p>בִּרְכַּי יוֹחִילוּ / עַל צְנִיף תִּפְאָרָה</p> <p>בְּרַעַד יֵלִילוּ / וּבֵיהוּל וּגְעָרָה</p> <p>וְאֵיכָה יוֹבִילוּ / כְּלִי שֶׁלַּתּוֹרָה</p> <p>לְדִירַת הֶעָפָר / כִּי בָא הַשּׁוֹדֵד:</p> <br/></blockquote>
<blockquote><p>נְעִימָה לוֹ, אִם בָּא / לִדְרוֹשׁ בָּעִנְיָן</p> <p>בְּשִׂמְחָה וּבְחִבָּה / וְלֵב טָהוֹר מְצֻיָּן</p> <p>וּבְנֶפֶשׁ טוֹבָה / יְתַקֵּן הַבִּנְיָן</p> <p>לְתַלְמִידֵי הַדּוֹר / מְלַמֵּד וּמְחַדֵּד:</p> <br/></blockquote>
<blockquote><p>יָשָׁר הוּא וְנָבוֹן / בְּשָׁעַת תַּלְמוּדוֹ</p> <p>כְּמוֹ שָׂר אוֹ רִבּוֹן / אֲדַמֶּה לִכְבוֹדוֹ</p> <p>וּמִסִּיג סוֹד חֶשְׁבּוֹן / כְּאוֹר שֶׁמֶשׁ הוֹדוֹ</p> <p>הוֹמֶה לִי לִבִּי / עָלָיו גַּם חָרֵד:</p> <br/></blockquote>
<blockquote><p>וּמַה-יּוֹעִיל בִּכְיִי / אַחַר אִישׁ נֶחְמָד</p> <p>וּמִי יְרַפֵּא חָלְיִי / בְּשָׁעַת כִּי יִלְמָד</p> <p>יְחַזֵּק אֶת רִפְיִי / וּבַתּוֹרָה נִצְמָד</p> <p>וְקוֹלוֹ כַּשַּׁחַל / בְּמִדְרָשׁוֹ שׁוֹקֵד:</p> <br/></blockquote>
<blockquote><p>סְגַן הַמִּשְׁפָּחָה / וְרֹאשׁ הוּא לַחֲכָמִים</p> <p>בְּעֵת כִּי יִפְתָּחָה / בְּמִדְרָשׁ וּטְעָמִים</p> <p>לְבָבוֹ יְפַעֲנֵחָה / לְסוֹדוֹת נֶעְלָמִים</p> <p>וְחוּט חֶסֶד מָשׁוּךְ / עֲלֵי רֹאשׁוֹ עוֹנֵד:</p> <br/></blockquote>
<blockquote><p>פְּאֵר יֶשַׁע קָשׁוּר / תָּמִיד עַל זְרוֹעוֹ</p> <p>וּבְחָכְמָה יָשׁוּר / לְכָל סוֹד יַשְׁמִיעוֹ</p> <p>עֲלֵי דוֹרוֹ כַּשּׁוּר / הָיָה לְהַצְנִיעוֹ</p> <p>אַחֲרָיו אֵיךְ נִשְׂמַח / בְּשַׁבָּת וּבְמוֹעֵד:</p> <br/></blockquote>
