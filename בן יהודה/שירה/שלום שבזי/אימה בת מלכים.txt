<h1>אימה בת מלכים</h1>
שלום שבזי
<p>חתימה: אלשבזי משתא זיו</p>
<blockquote><br/> <p>אֲיֻמָּה בַּת מְלָכִים הַתְּיֻמָּה / עֲדֵי כַמָּה / בְּגָלוּתֵךְ עֲלוּבָה</p></blockquote>
<blockquote><p>לְדוֹד בִּקְשִׁי / וְהַפְרִיחִי לְשָׁרְשִׁי / בְּהַר קָדְשִׁי / הֲעִידִינִי אֲהוּבָה</p></blockquote>
<blockquote><p>שְׁאוֹן גַּלִּים / מְשָׁכַנִי וְהִעְלִים / בְּנֵי אֵלִים / וְנִשְׁמָתִי עֲזוּבָה</p></blockquote>
<blockquote><p>בְּחֵיק רַעְיָה / יְחִידָתִי עֲנִיָּה / וּבוֹכִיָּה / בְּיַד פֶּרֶא עֲצוּבָה</p></blockquote>
<blockquote><p>זְמָן בַּחְרוּת / מְשָׁכַנִי בְּיַשְׁרוּת / בְּדָת חָרוּת / בְּתוֹרַת אֵל כְּתוּבָה</p></blockquote>
<blockquote><p>יְקָרָה מִפְּנִינִים / לַמְּבִינִים / וְהֵם חוֹנִים / בְּתַחְלוּאֵי מְשׁוּבָה.</p></blockquote>
<blockquote><p>מְתֵי שֶׁקֶר / עֲוֹנָם לֹא יְבֻקַּר / עֲדַת עֵקֶר / וְחָכְמָתָם גְּנוּבָה</p></blockquote>
<blockquote><p>שְׁפָלִים / כַּבְּהֵמוֹת הֵם מְשׁוּלִים / מְבֹהָלִים / בְּעֵץ דַּעַת תְּנוּבָה</p></blockquote>
<blockquote><p>תְּכוּנָתָם / מְכֹעֶרֶת בְּדָתָם / וְאַהְבָתָם / בְּסוֹד אֵיבָה חֲשׁוּבָה</p></blockquote>
<blockquote><p>וְאֵין חָכְמָה / וְלֹא דַעַת וְעָצְמָה / אֲבָל עָרְמָה / בְּלִבּוֹתָם נְצוּבָה</p></blockquote>
<blockquote><p>זְמוּנִים לַעֲשׂוֹת / רָאשֵׁי גְּיָסוֹת / וּמַחְמִיסוֹת / בְּלִי מַתָּן קְצוּבָה</p></blockquote>
<blockquote><p>יְדִיד חוּסָה / עֲלֵי עֵדָה אֲנוּסָה / וְתָנוּסָה / לְתוֹךְ צַעְרֵי תְשׁוּבָה</p></blockquote>
<blockquote><p>וְהָגֵן עַל בְּנֵי אִישׁ תָּם / בְּלֶכְתָּם / תְּהֶא דַעְתָּם / בְּחֶסֶד אֵל רְהוּבָה.</p></blockquote>
