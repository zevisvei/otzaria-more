<h1>נוֹחֲלֵי יוֹם</h1>
שלום שבזי
<p>חתימה: שלום</p>
<blockquote><p>נוֹחֲלֵי יוֹם מְנוּחָה / שִׁירוּ נָא שִׁיר לָאֵל</p> <p>כִּי אוֹרָה וְשִׂמְחָה / יוֹם זֶה לְיִשְׂרָאֵל:</p> <br/></blockquote>
<blockquote><p>שִׂמְחַת גִּיל לְבָבִי / הִנֵּה מְאֹד נִגְדְּלָה</p> <p>עֵת אֵצֵא בְּעַרְבִי / לִקְרַאת פְּנֵי כַלָּה</p> <p>לְזֹאת אֶהְגֶּה בְּנִיבִי / לָהּ שִׁיר עִם תְּהִלָּה</p> <p>בְּשׁוּבָה וַהֲנָחָה / בְּהַדְרַת עַם הָאֵל</p> <p>עַד לְבֵית מְנוּחָה / כְּמַלְכְּתָא לְיִשְׂרָאֵל:</p> <br/></blockquote>
<blockquote><p>לִפְנֵי הוֹד כְּבֻדָּה / בַּת מֶלֶךְ פְּנִימָה</p> <p>יָחִישׁ עָם אֲשֶׁר נִפְדָּה / לַעְרוֹךְ שֻׁלְחָן בְּנַהְמָא</p> <p>וּמִטָּה בִּכְלֵי חֶמְדָּה / וּמְנוֹרָה נְעִימָה</p> <p>וְהֵן מֵעֵת הַמִּנְחָה / יַעֲשֶׂה וְגַם יוֹאֵל</p> <p>בְּעַצְמוֹ לֹא שְׁלוּחָה / כִּגְדוֹלֵי יִשְׂרָאֵל:</p> <br/></blockquote>
<blockquote><p>וְזֹאת נֹאמַר כַּהֲלָכָה / בְּדָת זְכוּת הֲכָנָה</p> <p>כִּי כֵן יִשָּׂא בְרָכָה / מִפִּי מַלְאַךְ שְׁכִינָה</p> <p>לְשַׁבָּת אַחֶרֶת כָּכָה / תְּהֶא כֵן בִּרְנָנָה</p> <p>וּמַלְאַךְ אַחֵר בְּהַכְרָחָה / עוֹנֶה בְכֹחַ וָאֵל</p> <p>אָמֵן וְהִיא הַבְטָחָה / גְּדוֹלָה לְיִשְׂרָאֵל:</p> <br/></blockquote>
<blockquote><p>מִצְוַת זָכוֹר וְשָׁמוֹר / כּוֹלֶלֶת הַתּוֹרָה</p> <p>בְּדִבּוּר אֶחָד לֵאמֹר / מִפִּי הַגְּבוּרָה</p> <p>וּמִקְדָּשׁ עַל הַר הַמּוֹר / בִּזְכוּתָהּ בִּמְהֵרָה</p> <p>וּכְאַיָּלָה שְׁלוּחָה / יְצַו הָאֵל וְיוֹאֵל</p> <p>בְּהַשְׁקֵט וּבִטְחָה / מָשִׁיחַ לְיִשְׂרָאֵל:</p> <br/></blockquote>
<blockquote><p>עֲדַת קְדוֹשֵׁי עֶלְיוֹן / דְּגוּלֵי סְגֻלָּה</p> <p>נֶחָמַת בִּנְיַן אַפִּרְיוֹן / יֶחֱזוּ בִּיקָר וּגְדֻלָּה</p> <p>חַסְדָּךְ שׁוֹכֵן צִיּוֹן / זָכְרָה נָא לְעַם נַחֲלָה</p> <p>קוּמָה נָּא לְשָׁלְחָה / מְבַשֵּׂר וְגַם גּוֹאֵל</p> <p>וּבְקוֹל שָׂשׂוֹן וְשִׂמְחָה / אָז יָשִׁיר יִשְׂרָאֵל:</p> <br/></blockquote>
