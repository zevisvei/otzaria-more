<h1>שָׁלוֹם לְךָ רִבִּי מִמַּלְאֲכֵי מַעְלָה</h1>
שלום שבזי
<p>חתימה: שלם יצ [ישמרהו צורו]</p>
<blockquote><p>שָׁלוֹם לְךָ רִבִּי / מִמַּלְאֲכֵי מַעְלָה</p></blockquote>
<blockquote><p>אָהְבַךְ בְּתוֹךְ לִבִּי / יָדוּעַ וְגַם נִגְלָה:</p> <br/> <p>לִבִּי בְּאָהְבֶךָ / נִמְשָׁךְ וְנִתְקָרֵב</p></blockquote>
<blockquote><p>עִם אַהֲבָתֶךָ / נֶחְבָּר וְנִתְעָרַב</p></blockquote>
<blockquote><p>אַזְכִּיר שְׁלוֹמֶךָ / בֹּקֶר וְגַם עֶרֶב</p></blockquote>
<blockquote><p>אַתָּה כְמוֹ אָבִי / אַזְכִּירְךָ סֶלָה:</p> <br/> <p>מַהֵר שְׁמָעֵנִי / בֵּין שִׁיר אֲשֶׁר נִכְתָּב</p></blockquote>
<blockquote><p>מָרִי וְרִבּוֹנִי / הַתָּם וְהַנֶּאְהָב</p></blockquote>
<blockquote><p>חַזֵּק לְרִפְיוֹנִי / לִי הַגְּהָה וּכְתָב</p></blockquote>
<blockquote><p>תַּקֵּן לְמִכְתָּבִי / אָבִין בְּכָל מִלָּה:</p> <br/> <p>יִהְיֶה כְתָב שִׁירִי / מֻגָּהּ כְּתָב סוֹפֵר</p></blockquote>
<blockquote><p>הִתְכַּוְנָה מָרִי / מִלַּי וְגַם תִּסְפֹּר</p></blockquote>
<blockquote><p>כִּי לָךְ בְּעֵת זָכְרִי / אֶשְׁאַף כְּמוֹ צִפּוֹר</p></blockquote>
<blockquote><p>הָבֵן בְּמַחְשָׁבִי / נַחַת וְגַם צַהְלָה:</p> <br/> <p>צַהְלָה וְקוֹל רִנָּה / יַשְׁמִיעֲךָ הָאֵל</p></blockquote>
<blockquote><p>יִבְנֶה לְצִיּוֹנָה / נִזְכֶּה לְבֹא גוֹאֵל</p></blockquote>
<blockquote><p>נָשִׁיר נְרַנֵּנָה / נִמְצָא לְבֵית הַרְאֵל</p></blockquote>
<blockquote><p>נִשְׁמַע דְּבַר תִּשְׁבִּי / יְבַשֵׂר בֹּרב גִּילָה:</p></blockquote>
