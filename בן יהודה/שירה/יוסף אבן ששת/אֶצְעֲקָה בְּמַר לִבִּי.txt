<h1>אֶצְעֲקָה בְּמַר לִבִּי</h1>
יוסף אבן ששת
<blockquote><p>אֶצְעֲקָה בְּמַר לִבִּי / “אוֹהֲבִי, זְנַחְתַּנִי!”</p> <p>אוֹמְרָה לְאֵל סַלְעִי / “לָמָּה שְׁכַחְתַּנִי?”</p> <br/></blockquote>
<blockquote><p>בֵּן בְּכוֹר קְרָאתַנִי, / לִכְבוֹדְךָ בְרָאתַנִי,</p> <p>חֶסֶד מְשַׁכְתַּנִי, / מִנֹּף גְּאַלְתַּנִי.</p> <p>מַה לְּךָ שְׂנֵאתַנִי / אַחֲרֵי פְדִיתַנִי</p> <p>לֹא לְךָ לְקַחְתַּנִי</p> <p>אוֹהֲבִי, אֲהָהּ לִי, / אֵיךְ הֶעֱבַדְתַּנִי!</p> <br/></blockquote>
<blockquote><p>נָע וְנָד אֲנִי מֻשְׁלָךְ / מִמְּךָ וּמִשֶּׁלָּךְ</p> <p>וּזְמָן יְקָר הָלַךְ / אַחֲרֵי זְמָן מָשְלָךְ</p> <p>וִימֵי עֱזוּז גָּדְלָךְ / עַבְדִּי, אֲהָהּ, מָלָךְ</p> <p>וַהֲלֹא לְעֶבֶד לָךְ</p> <p>אוֹהֲבִי, אֲהָהּ לִי, / בִּלְבַד יְצַרְתַּנִי!</p> <br/></blockquote>
<blockquote><p>שַׁחוּ בְנוֹת שִׁירִי / חִלֵּל זְמָן שָׂרִי</p> <p>וַיֶּחֱזַק צִירִי / מֵאֵין לְלֵב צֹרִי</p> <p>אֶבְכֶּה בְהִמָּכְרִי / חִנָּם אֱלֵי צָרִי</p> <p>כַּאְשֶׁר לְךָ נָכְרִי</p> <p>אוֹהֲבִי, אֲהָהּ לִי, / אָבִי חֲשַׁבְתַּנִי!</p> <br/></blockquote>
<blockquote><p>שִׂיחִי לְךָ שַׂחְתִּי, / נִגְעִי וְתוֹכַחְתִּי</p> <p>אֶצְעַק: אֲהָהּ, שַׁחְתִּי, / סַפְתִּי וְנִסַּחְתִּי,</p> <p>נַדְתִּי וְנִדַּחְתִּי, / נַחְתִּי וְלֹא נַחְתִּי</p> <p>בַּשְּׁבִי וְנִשְׁכַּחְתִּי,</p> <p> אוֹהֲבִי, אֲהָהּ לִי, / כִּי אִם זְנַחְתַּנִי!</p> <br/></blockquote>
<blockquote><p>תָּנִיף כְּיָם נִדְבַת / יָד, כּוֹנֲנָה טוֹבַת</p> <p>מָשׂוֹשׂ אֲשֶׁר שָׁבַת, / יוֹם תַּעֲלֶה שִׁכְבַת</p> <p>טַל לַעֲזֹר מְשִׁיבַת / נֶפֶשׁ, וְתִפְדֶּה בַת-</p> <p>נָדִיב כְּמוֹ אַהְבַת / עוֹלָם אֲהַבְתַּנִי !</p> <br/></blockquote>
