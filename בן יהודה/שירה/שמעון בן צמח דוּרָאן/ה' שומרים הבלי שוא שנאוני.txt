<h1>ה' שומרים הבלי שוא שנאוני</h1>
שמעון בן צמח דוּרָאן
<p><a href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a></p>
<blockquote><p>ה' שׁוֹמְרִים הַבְלֵי שָׁוְא שְׂנֵאוּנִי / בְּהַבְלֵיהֶם כִּעֲסוּנִי / בְּזָרִים הִקְנִיאוּנִי</p></blockquote>
<blockquote><p>וּמֵאָז אָהַבְתָּ / יַעֲקֹב וְהִקְרַבְתָּ / וְזַרְעוֹ חִבַּבְתָּ</p></blockquote>
<blockquote><p>כִּי הֵם זֶרַע בֵּרַךְ ה'.</p></blockquote>
<br/>
<blockquote><p>מָאַסְתָּ שָׂעִיר / קִנְאָה יָעִיר / וּבָחַרְתָּ בַּצָּעִיר</p></blockquote>
<blockquote><p>קַרְנוֹ תָּרִים / בְּבִרְכוֹת הוֹרִים / אָב מָצָא מֵאָה שְׁעָרִים</p></blockquote>
<blockquote><p>וַיבָרְכֵהוּ ה'.</p></blockquote>
<br/>
<blockquote><p>עָלָיו צִוָּה לְשָׁמְרוֹ / מֶלֶךְ עִמּוֹ בְּהַזְהִירוֹ / הַנּוֹגֵעַ בּוֹ בַּחֶרֶב לְדָקְרוֹ</p></blockquote>
<blockquote><p>וְכַאֲשֶׁר לֹא נְגַעֲנוּךָ / וְכָבוד כִּבַּדְנוּךָ / וּבַשָׁלוֹם שְׁלַּחְנוּךָ</p></blockquote>
<blockquote><p>אִתָּהּ עַתָּה בָּרוּךְ ה'.</p></blockquote>
<br/>
<blockquote><p>וְשָׁאַל מַטְעַמִּים / לְבָרֵךְ יַעַבְדוּהוּ עַמִּים / וְיִשְׁתַּחֲווּ לוֹ לְאֻמִּים</p></blockquote>
<blockquote><p>וְהָאֵל נֶפֶשׁ עֲבָדָיו פּוֹדֶה / נָתַן חָלָק תַּחַת צוֹדֶה / רֵיחַ בְּגָדָיו כְּרֵיחַ הַשָּׂדֶה</p></blockquote>
<blockquote><p>אֲשֶׁר בֵּרְכוֹ ה'.</p></blockquote>
<br/>
<blockquote><p>נְעִימוֹת בִּימִינְךָ / תּוֹדִיעַ לְבָנֶיךָ / רֵאשִׁית קִנְיָנֶיךָ</p></blockquote>
<blockquote><p>מוֹרֵה חַטָּאִים בַּדֶּרֶךְ / בֵּית יִשְׂרָאֵל לְבָרֵךְ / בֵּית אַהֲרֹן יְבָרֵךְ</p></blockquote>
<blockquote><p>יְבָרֵךְ יִרְאֵי ה'.</p></blockquote>
<br/>
<ol><li id="fn:1"><p>(מאיורקה, 1361 – אלג'יר, 1444) <a href="#fnref:1" title="return to body"> ↩</a></p></li></ol>
