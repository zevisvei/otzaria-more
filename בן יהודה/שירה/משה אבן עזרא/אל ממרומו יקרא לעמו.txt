<h1>אל ממרומו יקרא לעמו</h1>
משה אבן עזרא
<h2 id="אֵלמִמְרוֹמוֹיִקְרָאלְעַמּוֹרמשהאבןעזרא">אֵל מִמְרוֹמוֹ יִקְרָא לְעַמּוֹ / ר' משה אבן עזרא</h2>
<p>סימן: אני משה קטן</p>
<blockquote><p>אֵל מִמְרוֹמוֹ / יִקְרָא לְעַמּוֹ</p></blockquote>
<blockquote><p>אִישׁ עַל מְקֹמוֹ / יָבֹא בְשָׁלוֹם!</p></blockquote>
<p>שמ' יח, כג</p>
<br/>
<blockquote><p>נְדָרַי אָבִיא / וְשִׁיַרי אַצְבִּיא</p></blockquote>
<blockquote><p>כִּי לְבֵית אָבִי / שַׁבְתִּי בְשָׁלוֹם!</p></blockquote>
<p>ברא' כח, כא</p>
<br/>
<blockquote><p>יְמֵי גִיל תַּמּוּ / וְשִׁירֵי דָּמְמוּ</p></blockquote>
<blockquote><p>כִּי נָדַמּוּ / נְאוֹת הַשָּׁלוֹם:</p></blockquote>
<p>ירמ' כה, לז</p>
<br/>
<blockquote><p>מֶלֶךְ יַעֲצֹר / וְרוּחוֹת יִבְצֹר</p></blockquote>
<blockquote><p>וְיֵצֶר תִּצֹּר / שָׁלוֹם שָׁלוֹם!</p></blockquote>
<p>ישע' כו, ג</p>
<br/>
<blockquote><p>שָׁמָּה יִשְׂגּוּ / וְלָרֹב יִדְגוּ</p></blockquote>
<blockquote><p>וְהִתְעַנְּגוּ / עַל רֹב שָׁלוֹם:</p></blockquote>
<p>תה' לז, יא</p>
<br/>
<blockquote><p>הַיּוֹם לְעֵינַי / יְגַלֶּה לְאָזְנַי</p></blockquote>
<blockquote><p>קוֹל רָם: אֲדֹנִי / תִּשְׁפֹּת שָׁלוֹם!</p></blockquote>
<p>ישע' כד, יב</p>
<br/>
<blockquote><p>קָרֵב מַחֲזֶה / וּמַאֲמַר חוֹזֶה:</p></blockquote>
<blockquote><p>וּבַמָּקוֹם הַזֶּה / אֶתֵּן שָׁלוֹם!</p></blockquote>
<p>חג' ב, ט</p>
<br/>
<blockquote><p>טוֹב וְעֹז וְשָׁלוֹם / יְצַו צוּר לְעֵילוֹם</p></blockquote>
<blockquote><p>אַתָּה שָׁלוֹם / וּבֵיתְךָ שָׁלוֹם:</p></blockquote>
<p>ש"א כה, ו</p>
<br/>
<blockquote><p>נְכֵאִים שַׁבְתֶּם / כִּי נִמְכַּרְתֶּם</p></blockquote>
<blockquote><p>תִּשְמְעוּ אַתֶּם: / עֲלוּ לְשָׁלוֹם:</p></blockquote>
<p>ברא' מד, יז <br/></p>
