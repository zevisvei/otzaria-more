<h1>בלילי על משכבי</h1>
משה אבן עזרא
<h2 id="בְּלֵילִיעַלמִשְׁכָּבִירמשהאבן-עזרא">בְּלֵילִי עַל מִשְׁכָּבִי / ר' משה אבן-עזרא</h2>
<p>סימן: משה</p>
<blockquote><p>בְּלֵילִי עַל מִשְׁכָּבִי / עֵצוֹת בְּנַפְשִׁי שַׂמְתִּי</p></blockquote>
<blockquote><p>וּלְהִתְוַדּוֹת עַל חוֹבִי / אַשְׁמוּרוֹת קִדַּמְתִּי</p></blockquote>
<blockquote><p>וְלַעֲמֹד עַל מַצָּבִי / לִפְנֵי אֲדֹנָי קַמְתִּי</p></blockquote>
<blockquote><p>לִרְאוֹת מַה יְּדַבֶּר בִּי / וּמָה אָשִׁיב עַל תּוֹכַחְתִּי:</p></blockquote>
<p>חבק' ב 1</p>
<blockquote><p>מִי זֶה יַעֲמֹד לְפָנָיו / וְלֹא יִירָא חֲרוֹן אַפּוֹ</p></blockquote>
<blockquote><p>וְאֵיכָה יִדְרֹשׁ רְצוֹנָיו / וְדָמִים מִלֵּא כַפּוֹ?</p></blockquote>
<blockquote><p>וּמִמַּחֲשַׁכֵּי עֲוֹנָיו / יֶחְשְׁכוּ כּוֹכְבֵי נִשְׁפּוֹ</p></blockquote>
<blockquote><p>וְהִקִּיפוּהוּ זְדוֹנָיו / כָּאֲרִי מִפֹּה וּמִפֹּה</p></blockquote>
<blockquote><p>מִפְּחָדָיו צָרַרְתִּי / וְאֶל חַסְדּוֹ בָּרַחְתִּי:</p></blockquote>
<br/>
<blockquote><p>שׁוֹכְחֵי טוּב אֲדוֹנֵיהֶם / וְנוֹשִׁים חֶסֶד פָּעֳלוֹ</p></blockquote>
<blockquote><p>כִּי אַחֲרֵי עֵינֵיהֶם / תָּרוּ בְעוֹלָם וְהֶבְלוֹ</p></blockquote>
<blockquote><p>וְשָׁוְא לָקְחוּ בִידֵיהֶם / וּבָאוּ אֶל הֵיכָלוֹ</p></blockquote>
<blockquote><p>וַיְפַתּוּהוּ בְּפִיהֶם / וּבִלְשׁוֹנָם יְכַזְּבוּ לוֹ</p></blockquote>
<blockquote><p>אֲהָהּ, כִּי תָפֵל טַחְתִּי / וּבַהֶבֶל בָּטַחְתִּי:</p></blockquote>
<p>יחז' יג 11</p>
<blockquote><p>הָאוֹמֵר וְהוּא עוֹשֶׂה / אַל תֵּפֶן לְמַעֲשֵׂהוּ</p></blockquote>
<blockquote><p>וְיוֹם בּוֹ אֲשֵׁמִים תְּנַסֶּה / בַּדִּין אַל תְּנַסֵּהוּ</p></blockquote>
<blockquote><p>וּמַעֲרוּמֵינוּ תְכַסֶּה / וּמִי זֶה בִּלְתְּךָ תְּכַסֶּנּוּ?</p></blockquote>
<blockquote><p>מָעוֹז לַדַּל וּמַחְסֶה / בַּשֶּׂר-נָא חוֹסֵנוּ:</p></blockquote>
<blockquote><p>מַאֲסָרֶיךָ פִּתַּחְתִּי / וְכִדְבָרְךָ סָלַחְתִּי:</p></blockquote>
<p>תה' קטז 17</p>
<p>מד' יד 20</p>
