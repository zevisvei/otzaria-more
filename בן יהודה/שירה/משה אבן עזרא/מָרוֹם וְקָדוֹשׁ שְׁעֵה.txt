<h1>מָרוֹם וְקָדוֹשׁ שְׁעֵה</h1>
משה אבן עזרא
<p>לקרובה א'</p>
<br/>
<blockquote><p>מָרוֹם וְקָדוֹשׁ שְׁעֵה שַׁוְעַת שְׁבוּיִם / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>הִתְעַשֵּׁת לָהֶם וְכָתְבֵם לְחַיִּים / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>וּקְרָא שְׁנַת רָצוֹן לְרִיב הָעֲנִיִּים / וְהָאֶבְיוֹנִים: / ישע' סא,כ;מא,יז</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<br/>
<blockquote><p>שְׁלוֹמִים נִפְלָאוּ / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>אֶל עַם נִצְבָּאוּ / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>בִּבְכִי יָבֹאוּ / וּבְתַחֲנוּנִים: / ירמ' לא,ח</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<br/>
<blockquote><p>הַעֲלֵם עִם בְּנֵיהֶם / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>וּסְלַח חֲטָאֵיהֶם / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>אֲשֶׁר כָּשְׁלוּ בָהֶם / אָבוֹת וּבָנִים: /שם ו,א</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<br/>
<blockquote><p>קְשֹׁב נָא בָנֶיךָ / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>וְהַשְׁקֵם לְפָנֶיךָ / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>בְּאֵר מַיִם חַיִּים / עִם גַּנִּים: / שה"ש ד,טו</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<br/>
<blockquote><p>טַל סְלִיחָה הָרֵק / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>שַׁעֲרֵי חֵן הָפֵק / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>וְיָבֹא גוֹי צַדִּיק / שֹׁמֵר אֱמֻנִים: / ישע' כו,ב</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<br/>
<blockquote><p>נְבָרִים הֵבִינוּ / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>כַּשֶּׁלֶג יַלְבִּינוּ / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>אִם יִהְיוּ חֲטָאֵיהֶם / כּשָּׁנִים: / שם א, יד</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
<blockquote><p>שְׁלַח קוֹל חֲסָדֶיךָ / מִמְּעוֹנִים</p></blockquote>
<blockquote><p>לְהוֹשִׁיב עֲבָדֶיךָ / עִם אֱמוּנִים</p></blockquote>
<blockquote><p>מֵעַבְדֵי אֲדֹנָי / הַקְּטַנִּים: /ישע' לו,ט</p></blockquote>
<blockquote><p>התעשת</p></blockquote>
