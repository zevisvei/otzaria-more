<h1>מִבֵּית מְלוֹנִי קַמְתִּי בְצוּקִי</h1>
משה אבן עזרא
<p>לחן: אחות קטנה תפילותיה עורכ</p>
<p>סימן: משה הק[טן]</p>
<blockquote><p>מִבֵּית מְלוֹנִי / קַמְתִּי בְצוּקִי</p></blockquote>
<blockquote><p>וּלְבֵית אֲדוֹנִי / בָּאתִי כְחֻקִּי</p></blockquote>
<blockquote><p>וְאֵלָיו עֵינִי / תְּלוּיָה בְצַעֲקִי</p></blockquote>
<blockquote><p>בְּקָרְאִי עֲנֵנִי / אֱלֹהֵי צִדְקִי: תה' ד, ב</p></blockquote>
<br/>
<blockquote><p>שְׁרִירוּת יִצְרִי - / יַשְׁפִּילֵנִי</p></blockquote>
<blockquote><p>בְּהָלְכִי בְקָרִי / יַבְהִילֵנִי</p></blockquote>
<blockquote><p>וְחַטַּאת מֶרִי / יַנְחִילֵנִי</p></blockquote>
<blockquote><p>לֹא יַרְפֵּנִי / עַד בִּלְעִי רֻקִּי איו' ז, ב</p></blockquote>
<br/>
<blockquote><p>הִבְאִישׁ רֵיחִי / כִּי בִי רָדָה</p></blockquote>
<blockquote><p>וְהֵמַר רוּחִי / וְעָלַי עָדָה</p></blockquote>
<blockquote><p>הִכְשִׁיל כֹּחִי / וְאוֹתִי צָדָה</p></blockquote>
<blockquote><p>שָׂם לַחֲרָדָה / נֶשֶׁף חִשְׁקִי: ישע' כא, ד</p></blockquote>
<br/>
<blockquote><p>הָקֵץ מִשְׁנַת / יְגוֹנָיו הוֹזֶה</p></blockquote>
<blockquote><p>וְהַקְרִיבָה שְׁנַת / פְּדוּת לְעָם נִבְזֶה</p></blockquote>
<blockquote><p>וּרְצֵה רִנַּת / כּוֹאֵב וְרָזֶה</p></blockquote>
<blockquote><p>כִּי אַתָּה מְנָת / חֶבְלִי וְחֶלְקִי: תה' טז, ח <br/></p></blockquote>
