<h1>[מַצִּיב אֲדָמָה וְכָל גְּבוּלֶיהָ]</h1>
משה אבן עזרא
<p>סימן: מ[שה] ק[ט]ן</p>
<br/>
<blockquote><p>מַצִּיב אֲדָמָה / וְכָל גְּבוּלֶיהָ</p> <p>נֹתֵן נְשָׁמָה / לָעָם עָלֶיהָ: [ישע' מב, ה] <br/></p></blockquote>
<blockquote><p>קַנֵּא לְכַנָּה / יְמִינְךָ שָׁתָלָה</p> <p>יוֹם דַּת אֱמוּנָה / וְצֶדֶק הִנְחִילָה</p> <p>וְשׁוֹד וּמְגִנָּה / הַיּוֹם סָבִיב לָהּ</p> <p>וְהִנֵּה כְּגַנָּה / אֲשֶׁר אֵין מַיִם לָהּ</p> <p>דּוֹמָה לְאֵלָה / נָבְלוּ עָלֶיהָ: [שם א, ל] <br/></p></blockquote>
<blockquote><p>נַהֵל אֵל עֶלְיוֹן / לְאַט הַקְּהִלָּה</p> <p>לִנְוֵה אַפִּרְיוֹן / הֱיוֹת לָהּ סְגֻלָּה</p> <p>וְקוֹמֵם צִיּוֹן / וְנִבְנְתָה עַל תִּלָּהּ</p> <p>וְחַזֵּק רִפְיוֹן / יָדָיו כְבַתְּחִלָּה</p> <p>וְתִהְיֶה כַכַּלָּה / תַּעְדֶּה כֵלֶיהָ: [שם מא, י] <br/></p></blockquote>
