<h1>מפני פחד אדני לבשתי סות אימים</h1>
משה אבן עזרא
<blockquote><p>מִפְּנֵי פַחַד אֲדֹנָי / לָבַשְׁתִּי סוּת אֵימִים.</p></blockquote>
<blockquote><p>הֱקִיצוּנִי רַעְיוֹנַי – / וְאִישׁוֹנַי נִרְדָּמִים,</p></blockquote>
<blockquote><p>וּלְקוֹל קוֹרֵא פָנַי / נָפְלוּ וְהֵם נִכְלָמִים:</p></blockquote>
<blockquote><p>“נִצָּב לָרִיב אֲדֹנָי / וְעוֹמֵד לָדִין עַמִּים!”</p> <br/></blockquote>
<blockquote><p>שִוַּעְתִּי וְזָחַלְתִּי / מֵאֵימַת הַקּוֹרֵא,</p></blockquote>
<blockquote><p>וְזַעְתִּי עֵת שָׁמַעְתִּי / “מִי הָאִישׁ הַיָּרֵא?”</p></blockquote>
<blockquote><p>כִּי מָרַדְתִּי וְזַדְתִּי - / וְאֵיךְ יִתְאַפֵּק מוֹרֶה?</p></blockquote>
<blockquote><p>בְּזָכְרִי כִי שָׁכַחְתִּי / יוֹם לְמִשְׁפָּט אֶקָּרֵא –</p></blockquote>
<blockquote><p>בְּעֵינַי נַחֲלֵי קְדוּמִים, / וּמֵעַי גַחֲלֵי רְתָמִים!</p> <br/></blockquote>
<blockquote><p>הֲיֵדַע אִישׁ מִצְעָדָיו / כָּל יְמֵי הֶבְלוֹ חֲשׁוּבִים?</p></blockquote>
<blockquote><p>וְהֵבִין כִּי מַעְבָּדָיו / בְּעֵט בַּרְזֶל נִכְתָּבִים?</p></blockquote>
<blockquote><p>כִּי שׁוֹפְטָיו וּמְעִידָיו / לַעֲדֵי עַד נִצָּבִים,</p></blockquote>
<blockquote><p>יוֹם יִבְגְּדוּ בוֹ יוֹלְדָיו / וִיכַחֲשׁוּ בוֹ אוֹהֲבִים,</p></blockquote>
<blockquote><p>שְׂבַע רֹגֶז קְצַר יָמִים – / הֵם בְּעֵינַי כַחֲלוֹמִים.</p> <br/></blockquote>
<blockquote><p>אֱנוֹשׁ הֶבֶל וּמַחְשְׁבוֹתָיו, / כָּל יָמָיו כְּצֵל עוֹבֵר,</p></blockquote>
<blockquote><p>כְּעוֹף יִהְיוּ שְׁנוֹתָיו / יְסוּפוּן מִבְּלִי אֵבֶר,</p></blockquote>
<blockquote><p>כִּי כָל אֲשֶׁר בְּאוֹצְרוֹתָיו / בְּאֶפֶס יָד יִשָּׁבֵר,</p></blockquote>
<blockquote><p>אֲבָל יֹשֶׁר מִפְעֲלוֹתָיו / יְלַוּוּהוּ לַקֶּבֶר –</p></blockquote>
<blockquote><p>וְאָז אֵל מִכֵּס זְעָמִים / יָשׁוּב לְכִסֵּא רַחֲמִים.</p></blockquote>
