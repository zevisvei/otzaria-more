<h1>מַה לְּךָ בֶּן אָדָם</h1>
משה אבן עזרא
<p>סימן: מ(שה בר) יע(ק)ב</p>
<blockquote><p>מַה לְּךָ בֶּן אָדָם / נְשָׁמָה בְאַפּוֹ</p></blockquote>
<blockquote><p>מַה לְּךָ נִרְדָּם / וְהַיָּם בְּזַעְפּוֹ?</p></blockquote>
<blockquote><p>שׁוּר בָּשָׂר וָדָם / קָשֶׁה עָרְפּוֹ</p></blockquote>
<blockquote><p>שׁוּב וְיָשׁוּב אֱלֹהִים / מֵחֲרוֹן אַפּו: / דב' יג, יח</p></blockquote>
<br/>
<blockquote><p>יָפְיְךָ הִשִּׁיאֲךָ / בִּימֵי שַׁחֲרוּת</p></blockquote>
<blockquote><p>וְתַאֲוָה תַשִּׁיאֲךָ / הֲלֹךְ בִּשְׁרִירוּת</p></blockquote>
<blockquote><p>לֵב, וְזֹקֶן יִרְאֲךָ / כְּתַב בִּמְרִירוּת</p></blockquote>
<blockquote><p>מָוֶת יִקְרָאֲךָ: / סוּרָה שְׁבָה-פֹה: / רו' ד, א</p></blockquote>
<br/>
<blockquote><p>עֲתְּדָה לְךָ צֵדָה / לְדֶרֶךְ יִשְׁגֶּה</p></blockquote>
<blockquote><p>וּתְהִי לְךָ לְעֵדָה / בְּיֵצֶר מַשְׁגֶּה</p></blockquote>
<blockquote><p>בְּטֶרֶם בֹּא קְפָדָה / וְאוֹתְךָ יַגֶּה</p></blockquote>
<blockquote><p>כַּאֲשֶׁר עֵת יֶהְגֶּה / כְּפִיר עַל טַרְפּוֹ: / ישע' לא ד <br/><br/> בֶּן כְּעֶבֶד לִפְנֵי / אֲדֹנָיו נַעֲנֶה</p></blockquote>
<blockquote><p>בְּפִיהוּ אֵין מַעֲנֶה / מִלֵּב זוֹנֶה –</p></blockquote>
<blockquote><p>אַךְ לְחֶסֶד מִשְׁנֶה / חֲטָאָיו יִפְנֶה,</p></blockquote>
<blockquote><p>טוֹב תְּמַלֵּא כַפּוֹ / וְכוֹנֵן טַרְפּוֹ: <br/></p></blockquote>
