<h1>[יְעִירוּנִי סְעִפַּי לַחֲזוֹתֶךָ]</h1>
משה אבן עזרא
<p>“לרבי משה בן עזרא ז”ל"</p>
<br/>
<blockquote><p>יְעִירוּנִי / סְעִפַּי לַחֲזוֹתֶךָ</p> <p>וְיוֹרוּנִי / לְהַגִּיד נִפְלְאוֹתֶיךָ</p> <p>וְיַרְאוּנִי / בְּעֵין לֵב נוֹרְאוֹתֶיךָ</p> <p>כִּי אֶרְאֶה / שָׁמֶיךָ מַעֲשֵׂה אֶצְבְּעֹתֶיךָ: [תה' ח, ד] <br/><br/> מִתְהַלֵּךְ / עֲלֵי קָו חוּג שָׁמָיִם</p> <p>וּמְסַבֵּב / מְסִבָּתוֹ כְאָבְנָיִם</p> <p>וּמְסַפֵּר / כְּבוֹדְךְ בְּלִי שְׂפָתָיִם</p> <p>וְהָאָרֶץ / עוֹמֶדֶת בִּנְתָּיִם</p> <p>וְהִיא תְלוּיָה / בְּחַבְלֵי אַהֲבָתֶךָ: <br/><br/> שָׁם שׁוֹאֵף / הַשֶׁמֶשׁ וְזוֹרֵח</p> <p>וְהוּא אוֹצֵל / מְאוֹרוֹ לַיָרֵחַ</p> <p>וְהַגַּלְגַּל / כְּמוֹ אֹהֶל מוֹתֵחַ</p> <p>וְכוֹכָבִים / עָלָיו כְּגַן פּוֹרֵחַ</p> <p>לְהוֹדִיעַ / עֹמֶק מַחְשְׁבוֹתֶיךָ <br/></p></blockquote>
<blockquote><p>הֶאָפֵס / חֶסֶד אֵל לַעֲדֵי עַד?</p> <p>וְאִם גָּמַר / דְּבַר נִחֻמִים וּמִסְעָד,</p> <p>אִם רוֹפֵא / אוֹ הַצֳּרִי אֵין בְּגִלְעָד</p> <p>יְהִי מַחֲזִיק / לֵב יֶלֶד לְבַל יִמְעד? – – –</p> <p>וְהוּא דוֹפֵק / שַׁעֲרֵי חֶמְלָתֶךָ!</p></blockquote>
