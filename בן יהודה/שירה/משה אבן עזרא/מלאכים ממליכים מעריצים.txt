<h1>מלאכים ממליכים מעריצים</h1>
משה אבן עזרא
<blockquote><p>מַלְאָכִים / מַמְלִיכִים, / מַעֲרִיצִים / בְּכָל פֵּאוֹת,</p></blockquote>
<blockquote><p>מַקְדִּישִׁים / מַרְעִישִׁים, / מַרְגִּישִׁים / בְּרֹב תְּשׁוּאוֹת,</p></blockquote>
<blockquote><p>נוֹרָאִים, / וְאֵין נִרְאִים, / נִפְלָאִים / וְנִפְלָאוֹת –</p></blockquote>
<blockquote><p>אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!</p></blockquote>
<br/>
<blockquote><p>שְׁמוֹ לָעַד / בְּרָב-רַעַד / בְּאֵימָה הֵן / יַעֲרִיצוּן,</p></blockquote>
<blockquote><p>שְׁאוֹן שְׂרָפִים / בְּרוֹם עָפִים, / כַּבְּרָקִים / יְרוֹצֵצוּן.</p></blockquote>
<blockquote><p>שׁוֹאֲלִים זֶה / רְשׁוּת מִזֶּה / לְהַקְדִּישׁ אֵל בְּמוֹרָאוֹת,</p></blockquote>
<blockquote><p>אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!</p></blockquote>
<br/>
<blockquote><p>הֲמוֹן עֵדִים / מַרְעִידִים, / בְּרֶגֶשׁ יַ- / עֲרִיצוּ אֵל,</p></blockquote>
<blockquote><p>הֲמֻלָּתָם / קְדֻשָּׁתָם / בְּעֵת שִׁלּוּשׁ / קְדֻשַּׁת אֵל,</p></blockquote>
<blockquote><p>בְּקוֹל רַעַם / בְּרֹב נֹעַם / בְּקוֹל מַרְאֶה / לְהִתְרָאוֹת</p></blockquote>
<blockquote><p>אוֹמְרִים: קָדוֹשׁ קָדוֹשׁ קָדוֹשׁ יְיָ צְבָאוֹת!</p></blockquote>
