<h1>הא לך מכתב מאח נכאב</h1>
משה אבן עזרא
<blockquote><p>הֵא־לָךְ מִכְתָּב / מֵאָח נִכְאָב / חֶטְאוֹ פָרַשׂ / פַּח לִפְעָמָיו</p></blockquote>
<blockquote><p>וּזְמָן הִצְבִּיא / צָרוֹת עָלָיו / עַד כִּי־שָׁלַח / יָד בִּשְׁלֹמָיו</p></blockquote>
<blockquote><p>מֵי־רֹאשׁ בִּלְבַד / נָתַן מַשְׁקָיו / שֹׁרֶשׁ רֹתֶם / רַק מַטְעַמָּיו</p></blockquote>
<blockquote><p>אַחְלָיו אִלּוּ / בִימֵי נֹעַר / יֹאבַד וִיהִי / מֵת בַּעְלוּמָיו</p></blockquote>
<blockquote><p>אָז יָנוּחַ / לוֹ כִּי אַחַר / כָּבוֹד יִהְיֶה / בָא אֶל־עַמָּיו</p></blockquote>
<blockquote><p>הֶן מַאְוַיָּו / לִכְלוֹת חַיָּיו / מָוֶת יִבְחַר / לוֹ מֵעְצָמָיו</p></blockquote>
<blockquote><p>כִּי לֹא־יִשְׁכַּב / לַיְלָה לִבּוֹ / גַּם מַכְאֹבִים / הֵם כָּל־יָמָיו</p></blockquote>
<blockquote><p>פָּצוּ עָלָיו / פִּיהֶם צָרָיו / גַּם כָּל־הַיּוֹם / שִׂפְתֵי קָמָיו</p></blockquote>
<blockquote><p>עַד כִּי־אָחִיו / הִקְשָׁה לִבּוֹ / וּבְלִי־פֶשַׁע / שִׁחֵת רַחְמָיו</p></blockquote>
<blockquote><p>לוּ דִבַּרְתִּי / מֵאָז כָּזֹאת / אֶל־הַסֶּלַע / נָתַן מֵימָיו. <br/></p></blockquote>
