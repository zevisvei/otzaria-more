<h1>מְפַחֵד מִפְּנֵי עֲוֹנִי</h1>
משה אבן עזרא
<p>לנועם: מדי ימי ושני</p>
<br/>
<blockquote><p>מְפַחֵד מִפְּנֵי עֲוֹנִי / בָּאתִי וְעָטוּי אֵימָה</p> <p>אוּלַי בְּמַר תַּחֲנוּנַי / אֲכַבֶּה יְקֹד הַחֵמָה</p> <p>וְעַיִש תָּעִיד בְּפַנָי בְּלֵילִי, / וְתַעַן כִּימָה</p> <p>אִם אֶתֵּן שְׁנָת לְעֵינָי / לְעַפְעַפַּי תְּנוּמָה: <br/><br/> מִבְּלִי עַיִן אֲצַפֶּה / חֶבְיוֹן תְּהִלָּתֶךָ</p> <p>וְאֶתְיַצְּבָה עַל מִצְפֶּה / לִרְאוֹת נִפְלְאוֹתֶיךָ</p> <p>וַאֲסַפֵּר וְאֵין לִי פֶה / עֹמֶק מַחְשְׁבוֹתֶיךָ _</p> <p>וְיַד שִׂכְלִי יִתְרַפֶּה / בְּמַעֲשֵה אֶצְבְּעוֹתֶיךָ,</p> <p>אַף לְלִבִּי מַרְפֵא / בְּזָכְרִי נִפְלְאוֹתֶיךָ</p> <p>כִּי מִמְּךָ סוֹד חָכְמָה / וְעִמְּךָ יְסוֹד הַנְּשָׁמָה: <br/><br/> שָׁמָיו מִבְּלִי מִלָּה / וְאַרְבַּעַת שָׁרָשָיו</p> <p>יְעִידוּן כִּי אֵין תְּחִלָּה / לְשָׁנָה וְסוֹף חֳדָשָׁיו</p> <p>וְכִי יֵשׁ תְּחִלָּה וְתִכְלָה / לִישָׁנָיו וַחֲדָשָׁיו?</p> <p>וְתֵבֵל כֻּלָּה – / וְהָעוֹלָם וַאֲנָשָׁיו</p> <p>בְּמַלְאָכָיו יְשַׁו תָּהֳלָה / וְלֹא יַאֲמִין בִּקְדוֹשָׁיו</p> <p>וְאֵיךְ יִצְדַּק אֱנוֹשׁ רִמָּה / עָפָר מִן הָאֲדָמָה?! <br/><br/> הִקְרִיב אִמְרֵי שֶׁפֶר / לִבִּי, וְשִׁירֵי תֻמָּה</p> <p>וּמַחְשְׁבוֹת עָוֶל הֵפֵר / לְהָשִׁיב אֶת הַחֵמָה</p> <p>אַל נָא עָפָר וְאֵפֶר / יַעַל בְּמֹאזְנֵי מִרְמָה</p> <p>וּלְעַם נְבָרִים יְכַפֵּר / וְאִם הוּא הִרְבָּה אָשְׁמָה</p> <p>כִּי לְקוֹל שֹׁקֵל וְסֹפֵר / לִבּוֹ יֶהְגֶּה אֵימָה</p> <p>נָא הֶרֶף יוֹם נְקָמָה / וַעֲזֹב חֵמָה:</p></blockquote>
