<h1>יְנֻחָמוּ יְרֻחָמוּ</h1>
משה אבן עזרא
<p>“לרבי משה ן' עזרא ז”ל"</p>
<p>סימן: משה</p>
<blockquote><p>יְנֻחָמוּ / יְרֻחָמוּ / בְּנֵי אִישׁ תָּם, רַב אוֹנָם</p></blockquote>
<blockquote><p>בְּגָלוּתָם, / בְּבוֹר עֱנוּתָם / דָּם תֵּרַדְנָה עֵינָם –</p></blockquote>
<blockquote><p>לְהַצִּילָם / וּלְגָאֳלָם / צִיר אֶשְׁלַח לְמַעֲנָם</p></blockquote>
<blockquote><p>כְּהִבְטַחְתִּים / וְנִחַמְתִּים / וְשִׂמַּחְתִּים מִיגוֹנָם:</p></blockquote>
<p>ירמ' לא, יב</p>
<br/>
<blockquote><p>מְנֻדָּחִים / וְנֶאֱנָחִים / הֵם שׁוֹכְחִים אֶת רִבְצָם</p></blockquote>
<blockquote><p>מְקֻבָּצִים / וּמַעֲרִיצִים / יְשׁוּבוּן אֶל מְחוֹז חֶפְצָם</p></blockquote>
<blockquote><p>בּרֹן יִשְׂאוּ / וְיָבֹאוּ / כִּי מָלְאוּ יְמֵי קִצָּם</p></blockquote>
<blockquote><p>לְהוֹשִׁיעַ / לְהַשְׁמִיעַ / לְצִיּוֹן הִנֵּה הִנָּם:</p></blockquote>
<p>ישע' מא, כז</p>
<br/>
<blockquote><p>שְׁבוּת צִיּוֹן / בְּשׁוּב עֶלְיוֹן / לְמִקְדַּשׁ אֵל יָצְאוּ מִשָּׁם</p></blockquote>
<blockquote><p>בְּאֶקְדָּחִים / מְפֻתָּחִים / יְכוֹנֵן אֶת מִקְדָּשָׁם</p></blockquote>
<blockquote><p>בְּאֵין אֶנְחָה / וּצְוָחָה / וְשִׂמְחַת עוֹלָם עַל רֹאשָׁם</p></blockquote>
<blockquote><p>וְצוּר עֻזָּם / וּמָעֻזָּם / מוֹשִׁיעָם וּמָגִנָּם:</p></blockquote>
<br/>
<blockquote><p>(וְ)הַמְרִיצוּ / וְהַעֲרִיצוּ / לְאֵל רָם קְדוֹשׁ יִשְׂרָאֵל</p></blockquote>
<blockquote><p>יְנַחֶמְכֶם / יְקַבֶּצְכֶם / אֶל מִקְדַּשׁ הַהַרְאֵל</p></blockquote>
<blockquote><p>וּבְנֵי שְׁלֹשָׁה / בִקְדֻשָּׁה / יַקְדִּישׁוּ שֵׁם הָאֵל</p></blockquote>
<blockquote><p>כְּאֶרְאֶלִּים / כְּחַשְׁמַלִּים / כְּשִׁנְאַנִּים בִמְעוֹנָם: <br/></p></blockquote>
