<h1>זְּמָן יִקְרָא עֵדָיו</h1>
משה אבן עזרא
<h2 id="זְּמָןיִקְרָאעֵדָיורמשהאבן-עזרא">זְּמָן יִקְרָא עֵדָיו / ר' משה אבן-עזרא</h2>
<br/>
<blockquote><p>                        זְּמָן יִקְרָא עֵדָיו / רְאוֹת מַעְשֵׂי יָדָיו / אֲשֶׁר הֵמָּה עֹשִׂים</p></blockquote>
<blockquote><p>                        הֲכִי כִסָּה פָנָיו / בְּתַשְׁבֵּץ נִצָּנָיו / וּבִכְתֹנֶת פַּסִּים</p></blockquote>
<blockquote><p>                        וּמֵאֵין לֵב יָגִיל / כְּבַת תַּעְדֶּה עָגִיל / וְתִתְיַף בִּשְׁבִיסִים</p></blockquote>
<blockquote><p>                        וְרוּח גַּן יִשְׁאַף / וְיִשָּׂא אֶל כָּל אַף / כְּפָרִים וַהְדַסִּים</p></blockquote>
<blockquote><p>5                        וּמַחְמַדֵּי נִרְדָּם / יְעִירוּן כָּל נִרְדָּם / וְלֵב חשְׁקִים שֹׁסִים</p></blockquote>
<blockquote><p>                        וּבֵינוֹת עָלֵיהֶם / יְעוּפוּן סִיסֵיהֶם / וְעַל אֶרֶץ טָשִׂים</p></blockquote>
<blockquote><p>                        וְלַחְשׂף סוֹדֵיהֶם / יְחוּשׁוּן סוֹדֵיהֶם / וְכֹסֵינוּ כֹסִים</p></blockquote>
<blockquote><p>                        לְאִישִׁים צִלְלֵיהֶם / וְרָאשֵׁי עַנְפֵיהֶם / לְתֹרִים וּלְסִיסִים</p></blockquote>
<blockquote><p>                        וּמֵי שֹׁקֶט צַחוּ / הֲכִי לֹא נִדְלָחוּ / וְאֵינָם נִרְפָּשִׂים</p></blockquote>
<blockquote><p>10                        בְּלִי פֶה מַטִּיפִים / וְהֵם כִּפְצִירָה פִּים / וְשִׁרְיוֹן קַשְׂקֵשִּׂים</p></blockquote>
<blockquote><p>                        וְרִשְׁפֵּי מִזְרָקִים / עֲלֵי כַפֵּי מַשְׁקִים / כְּאֵשׁ מֵאֵין מֵשִׂים</p></blockquote>
<blockquote><p>                        לְהָקֵל מַעְצֵבִים / יְרֻצוּן כֶּעָבִים / וְכִמְרֻצַת סוּסִים</p></blockquote>
<blockquote><p>                        לְזֹאת אֶשְׁאַל רֵעַי / לְהָבִין אֶת רֵעַי / וּבוֹ נִהְיֶה שָׂשִׂים</p></blockquote>
<blockquote><p>                        הֲכִי רִגְעֵי נוֹחַ / יְחוּשׁוּן לִבְרוֹחַ / וְכַצֵּל הֵם נָסִים.</p></blockquote>
