<h1>לְמַעַן הֵפַרְנוּ אֶת בְּרִיתֶךָ</h1>
משה אבן עזרא
<blockquote><p>לְמַעַן הֵפַרְנוּ / אֶת בְּרִיתֶךָ</p></blockquote>
<blockquote><p>הַצוּר, מָרַדְנוּ / בְאִמְרָתֶךָ</p></blockquote>
<blockquote><p>וְעֵקֶב מָעַלְנוּ / ְתוֹרָתֶךָ</p></blockquote>
<blockquote><p>בְּעָלוּנוּ אֲדֹנִים / זוּלָתֶךָ</p></blockquote>
<p>ישע' כו יג</p>
<br/>
<blockquote><p>מַחֲמַדַּי כִּלּוּ / בְּנֵי יִשְׁמָעאֵל</p></blockquote>
<blockquote><p>וּבשָׂרִי בִּלּוּ / קֵדָר וְאַדְבְּאֵל</p></blockquote>
<blockquote><p>נוֹעֲדוּ וַיֹּאכְלוּ / אֶת יִשְׂרָאֵל</p></blockquote>
<blockquote><p>וְהֵם נַחֲלָתְךָ / בְּנֵי בְרִיתֶךָ:</p></blockquote>
<p>מל"א ח, נא</p>
<br/>
<blockquote><p>שִׁמָּמוֹן הוּשַׂם / מַחְמַד מְעוֹנָי</p></blockquote>
<blockquote><p>וּנְבָיוֹת וּמִבְשָׁם / פָּשְׁטוּ עֲדָנָי</p></blockquote>
<blockquote><p>וַיְמַלְאוּ כְרֵסָם / מִמָּעֲדָנָי</p></blockquote>
<blockquote><p>וְהִצִּילוּ עֵינִי / לְעֻמָּתֶךָ:</p></blockquote>
<p>ש"ב כ. ו</p>
<br/>
<blockquote><p>הָיִינוּ לְשַׁמָּה / בִּידֵי שַׁמָּה</p></blockquote>
<blockquote><p>וּלְבִזָּה לְמִזָּה / וְנָפִישׁ וְקֵדְמָה</p></blockquote>
<blockquote><p>וְיָרַדְנוּ דוּמָה / בְּפֶרֶךְ דּוּמָה</p></blockquote>
<blockquote><p>אַיֵּה גְבוּרָתְךָ / וְקִנְאָתֶךָ?</p></blockquote>
<p>ישע' סג, טו</p>
<br/>
<blockquote><p>נְקֹם נִקְמָתֵנוּ / אֱלֹהֵי עוֹלָם</p></blockquote>
<blockquote><p>וְדוֹן בָּם דִּינֵנוּ / חוֹקֵר נֶעֱלָם</p></blockquote>
<blockquote><p>וְאֶל חֵיק אוֹיְבֵינוּ / נְשַׁלֵּם גְּמוּלָם</p></blockquote>
<blockquote><p>וְנוֹדֶה וּנְסַפֵּר / תְּהִלָּתֶךָ:</p></blockquote>
<p>תה' עט, יג <br/></p>
