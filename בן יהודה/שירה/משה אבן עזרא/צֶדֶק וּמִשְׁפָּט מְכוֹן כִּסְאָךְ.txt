<h1>צֶדֶק וּמִשְׁפָּט מְכוֹן כִּסְאָךְ</h1>
משה אבן עזרא
<p>סימן: משה</p>
<blockquote><p>צֶדֶק וּמִשְׁפָּט / מְכוֹן כִּסְאָךְ</p></blockquote>
<blockquote><p>חֶסֶד וֶאֱמֶת / יְקַדְמוּ פָנֶיךָ: / תה' פט, טו</p></blockquote>
<br/>
<blockquote><p>מָרוֹם וְקָדוֹשׁ / גָּדְלְךָ לֹא נֶחְקָר</p></blockquote>
<blockquote><p>אַשְׁרֵי אָדָם / בַּאֲמִתְךָ לֹא שֶׁקֶר</p></blockquote>
<blockquote><p>בַּהֲבִיאֲךָ לִבּוֹת / יְצוּרִים בַּמֶּחְקָר –</p></blockquote>
<blockquote><p>בְּחַסְדְּךָ תִּיקַר / נַפְשִׁי בְּעֵינֶיךָ: / מל"ב א, יד</p></blockquote>
<br/>
<blockquote><p>שְׁעֵנִי בְקָרְאִי / כְּקָרְבָּן נִרְצָה</p></blockquote>
<blockquote><p>וּבַדִּין בְּעָמְדִי / הֲבִינֵנִי מְלִיצָה-</p></blockquote>
<blockquote><p>וְשָׁפַךְ דִּמְעִי / כְּדַם עוֹלוֹת נִמְצָה</p></blockquote>
<blockquote><p>לְמַעַן אֶמְצָא / חֵן בְּעֵינֶיךָ: / שמ' לג, יב</p></blockquote>
<br/>
<blockquote><p>הַקְשֵׁב רִנָּתִי / מְקוֹם דַּם עוֹלָתִי</p></blockquote>
<blockquote><p>כִּי לְךָ נָשָׂאתִי / רוּחִי וְנִשְׁמָתִי –</p></blockquote>
<blockquote><p>וְעַל חַטָּאתִי / לְהִתְוַדּוֹת בָּאתִי</p></blockquote>
<blockquote><p>תִּכּוֹן תְּפִלָתִי / קְטֹרֶת לְפָנֶיךָ: / תה' קמא, ב</p></blockquote>
