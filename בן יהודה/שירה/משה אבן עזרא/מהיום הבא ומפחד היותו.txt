<h1>מהיום הבא ומפחד היותו</h1>
משה אבן עזרא
<h2 id="מהיוםהבאומפחדהיותומשהאבןעזרא">[מהיום הבא ומפחד היותו] / משה אבן עזרא</h2>
<br/>
<blockquote><p>מֵהַיּוֹם הַבָּא / וּמִפַּחַד הֱיוֹתוֹ</p></blockquote>
<blockquote><p>כֶּעָשָׁן מֵאֲרֻבָּה / תַעַל מְהוּמָתוֹ,</p></blockquote>
<blockquote><p>תְּבוּסָתוֹ רַבָּה – / וּמִי יוּכַל שְׂאֵתוֹ</p></blockquote>
<blockquote><p>וּמִי הָעוֹמֵד / בְּהֵרָאוֹתוֹ?</p></blockquote>
<br/>
<blockquote><p>שׁוֹקֵט וּבוֹטֵחַ – / אֵימָה תְבַעֲתֵהוּ,</p></blockquote>
<blockquote><p>וּמָוֶת שׁוֹכֵחַ – / שׁוֹאָה תְבוֹאֵהוּ,</p></blockquote>
<blockquote><p>וְשׁוֹכֵב שָׂמֵחַ – / צְעָקָה תְעִירֵהוּ,</p></blockquote>
<blockquote><p>תֵּצֵא נִשְׁמָתוֹ / יָשׁוּב לְאַדְמָתוֹ.</p></blockquote>
<br/>
<blockquote><p>הוּא הַיּוֹם הַמַּר,/ כַּתַּנּוּר בּוֹעֵר,</p></blockquote>
<blockquote><p>כְּמֵי הַיָּם יֵמַר / וְכָמוֹהוֹ סוֹעֵר,</p></blockquote>
<blockquote><p>וְכוֹס יַיִן חָמַר / מְשַׁכֵּר כָּל נוֹעֵר,</p></blockquote>
<blockquote><p>מְמַהֵר תְּנוּמָתוֹ / לַעֲדֵי-עַד שְׁנָתוֹ!</p></blockquote>
<br/>
<blockquote><p>אַשְׁרֵי אֱנוֹשׁ יִרְאֶה / בְאִישׁוֹנֵי מְזִמָּה,</p></blockquote>
<blockquote><p>כִּי גָאֹה תִגְאֶה / בְצֵאתָהּ הַנְּשָׁמָה,</p></blockquote>
<blockquote><p>וְתַעַל וְתִשְׁתָּאֶה / לְהֵיכְלֵי הַחָכְמָה,</p></blockquote>
<blockquote><p>וְתָתוּר צִדְקָתוֹ / מְנוּחָה לַחֲנוֹתוֹ.</p></blockquote>
