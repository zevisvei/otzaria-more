<h1>הֶרֶף בֹּכֶה חֻקֵּי בָתִּים</h1>
משה אבן עזרא
<blockquote><p>הֶרֶף בֹּכֶה / חֻקֵּי בָתִּים / אַחְרֵי יַלְדֵי / אַהַב נִצְדִּים</p> <p>וּנְטֵה עִמִּי / אֶל גַּן בֹּשֶׂם / לָשֶׁבֶת בֵּין / צִלֵּי בַדִּים</p> <p>וּפְנֵי אֶרֶץ / לָבְשׁוּ שָׁנִי / אוֹ מִתַּשְׁבֵּץ / לָהּ מַרְבַדִּים</p> <p>הֵסִירָה אֶת / בִּגְדֵי אַלְמֹן / וַתִּפְשֹׁט כָּל / סוּתֵי עִדִּים</p> <p>וַתַּעַד הוֹד / כֶּלְיֵי כַלָּה / אוֹ מִמֶּשִׁי / תַּעְטֶה בַדִּים</p> <p>תִּשְׁלַח לִקְרַאת / כָל אַף עַל פִּי / רוּחֵי נֶשֶׁף / כָּל מַחְמַדִּים</p> <p>תֵּן כּוֹס מֵעֵין / שֶׁלֶג יַלְבִּין / רַק כַּתּוֹלָע / יֵינוֹ יַאְדִּים</p> <p>תָּעִיד כִּי אֵשׁ / הוֹא בֵּין בָּרָד / אוֹ לַפִּדִים / תּוֹךְ הַכַּדִּים. <br/></p></blockquote>
