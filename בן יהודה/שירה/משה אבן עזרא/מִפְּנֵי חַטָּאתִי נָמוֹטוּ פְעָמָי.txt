<h1>מִפְּנֵי חַטָּאתִי נָמוֹטוּ פְעָמָי</h1>
משה אבן עזרא
<blockquote><p>מִפְּנֵי חַטָּאתִי / נָמוֹטוּ פְעָמָי / וְלַעֲוֹן שַׁחֲרוּתִי</p></blockquote>
<blockquote><p>וּזְדוֹנֵי עֲלוּמָי / אָבְדָה תִקְוָתִי</p></blockquote>
<blockquote><p>וְאֵין שָׁלוֹם בַּעֲצָמַי:</p></blockquote>
<br/>
תה' לח, ד
<br/>
<blockquote><p>מִפְּנֵי חַטָּאתִי / מְהַלֵּךְ אֲנִי שׁוֹלָל / כָּל הַיּוֹם קֹדֵר</p></blockquote>
<blockquote><p>דְּוֵה לֵבָב וְאֻמְלָל / וְכָלוּא לְחֵטְא עוֹלַל</p></blockquote>
<blockquote><p>וּבַהֶבֶל יָמָי:</p></blockquote>
<br/>
איוב ז, טז
<br/>
<blockquote><p>מִפְּנֵי חַטָּאתִי / שְׂשׂוֹן נַפְשִׁי נִגְמָר / וְצוּפִי בְּרוֹשׁ הֻמָּר</p></blockquote>
<blockquote><p>וְלִמְאׁד מַר לִי מָר / וְשָׁלַח יָד וְגָמַר</p></blockquote>
<blockquote><p>יָדָיו בִּשְלֹמָי:</p></blockquote>
<br/>
<p>תה' נה, כא</p>
<br/>
<blockquote><p>מִפְּנֵי חַטָּאתִי / הֵן יָזוּב כְּבֵדִי / וְגַם גָּלָה כְבוֹדִי</p></blockquote>
<blockquote><p>וַהֲדָרִי וְהוֹדִי / וְהֵבֵאתִי לְפִידִי</p></blockquote>
<blockquote><p>טְעָמַי מְקוֹם שְׁלָמָי:</p></blockquote>
<br/>
<blockquote><p>מִפְּנֵי חַטָּאתִי / נִגְלוּ כָל עֲלֻמָי / וְתַמּוּ כָל זְמַנַּי</p></blockquote>
<blockquote><p>וְסָפוּ כָל תְּמִימָי / וּבוֹ קָמוּ קָמַי</p></blockquote>
<blockquote><p>וּבוֹ נִתְּקוּ זְמָמָי / אָבְדָה תִקְוָתִי</p></blockquote>
<blockquote><p>וְאֵין שָׁלוֹם בַּעֲצָמַי:</p></blockquote>
<br/>
<p>תה' לח, ד</p>
