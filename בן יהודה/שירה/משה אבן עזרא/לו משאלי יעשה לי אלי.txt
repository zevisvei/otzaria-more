<h1>לו משאלי יעשה לי אלי</h1>
משה אבן עזרא
<blockquote><p>לוּ מִשְׁאָלִי / יַעֲשֶׂה לִּי אֵלִי</p> <p>וְיַמְשֵׁנִי / מִתְּהוֹם נַחֲלָתִי / אֶל נַחֲלָתִי!</p> <br/></blockquote>
<blockquote><p>מָתַי צֹרִי / יִשְׁלְחָה לִי צוּרִי</p> <p>עַל יַד צִירִי / לַחֲבוֹשׁ אֶת צִירִי?</p> <p>וּפְנֵי צָרִי / אֲשֶׁר סָבִיב צָרִי</p> <p>יָשִׂים כִּבְלִי / כִּי הִכְבִּיד כַּבְלִי</p> <p>כִּי שָׂמַנִי / מִבְּלִי תַאֲוָתִי / גַּם תַּאֲוָתִי.</p> <br/></blockquote>
<blockquote><p>שָׁמֵם מִבְצָר / בִּידֵי יְלִיד נִבְצָר</p> <p>וּפְנֵי הַצָּר / נֶחְשְׁבוּ לִי כַּצָּר</p> <p>עַד כִּי קָצַר / יָד וּמַצָּע קָצַר</p> <p>וַיּוֹר כִּסְלִי / אַךְ עֲלֵי אֵל כִּסְלִי</p> <p>לַהֲרִימֵנִי / מִבּוֹר דַּלּוּתִי / חִישׁ לְדַלּוֹתִי.</p> <br/></blockquote>
<blockquote><p>הִנֵּה אֵילַי / סָפוּ כִּי אֵילַי</p> <p>יָבְשׁוּ. וְאֵילַי / שָׁמְמוּ כִּי אֵלַי</p> <p>צָפוּ וְאֵילַי / יִקְרְאוּ נָא, אוּלַי</p> <p>יָשִׁיב חֵילִי / אֶל הֲדוֹמִי וְחֵלִי</p> <p>וְיַנִּיחֵנִי / לַעֲרוֹךְ זִמְרָתִי / עַל זִמְרָתִי.</p> <br/></blockquote>
<blockquote><p>בִּידֵי דוּמָה / יָרַדְתִּי דוּמָה</p> <p>הַשְׁקוּ חֵמָה / לִי תַחַת חֵימָה</p> <p>וַאֲנִי חוֹמָה / …רעית חוֹמָה</p> <p>מִבּוֹר חֵילִי / אֶצְעֲקָה אֶל חֵילִי</p> <p>לְנַהֲלֵנִי / וְלִגְאוֹל אַדְמָתִי.</p> <br/></blockquote>
