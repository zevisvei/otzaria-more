<h1>מִתִּגְרַת כַפֶּיךָ רוֹפְפוּ שָׁמַיִם</h1>
משה אבן עזרא
<p>“לרבי משה ז”ל"</p>
<p>סימן: משה קטן</p>
<blockquote><p>מִתִּגְרַת כַּפֶּיךָ / רוֹפְפוּ שָׁמַיִם</p></blockquote>
<blockquote><p>וּבְרוּחַ אַפֶּיךָ / נֶעֶרְמוּ מַיִם: / שמ' טו, א</p></blockquote>
<br/>
<blockquote><p>שִׁירִים עִרְכִי, / יוֹנָה, בְּמַחְבָּר</p></blockquote>
<blockquote><p>וְזֶמֶר שִׁפְכִי / בְּמֵיטַב מִדְבָּר</p></blockquote>
<blockquote><p>וְתָמִיד בָּרֲכִי / עַל צָר תִּתְגַּבָּר</p></blockquote>
<blockquote><p>הַצּוּר הַהֹפְכִי / אֲגַם מָיִם: / תה' קיד, ח</p></blockquote>
<br/>
<blockquote><p>הוֹד זְרוֹעוֹ שַׁרְנוּ / וְהוֹדִינוּ בָדָד</p></blockquote>
<blockquote><p>כִּי לֹא נִכְלַמְנוּ / וְאוֹיְבֵנוּ נָדָד</p></blockquote>
<blockquote><p>תַּחְתֵּינוּ קַמְנוּ / וַנִּתְעוֹדָד</p></blockquote>
<blockquote><p>כִּי מִי בִלְתּוֹ מָדַד / בְּשָׁעֳלוֹ מַיִם: / ישע' מ, יב</p></blockquote>
<br/>
<blockquote><p>קֵץ שָׂם הַצּוּר / וּזְמָן לַגּוֹזֵל</p></blockquote>
<blockquote><p>וְהוֹצִיא עָצוּר / מִכּוּר הַבַּרְזֶל</p></blockquote>
<blockquote><p>וְטַל יֶשַׁע נָצוּר / עָלָיו נוֹזֵל</p></blockquote>
<blockquote><p>וְגַם הִכָּה צוּר / וַיָּזֻבוּ מָיִם: / שם מה, כא</p></blockquote>
<br/>
<blockquote><p>טִבַּע עֲצֶרֶת / בֹּגְדִים בִּתְהוֹמִים</p></blockquote>
<blockquote><p>וְדָרְכָה גְבֶרֶת / עַל בָּמֳתֵי קָמִים</p></blockquote>
<blockquote><p>וּזְרוֹעַ תִּפְאֶרֶת / פָּרְשָׂה חֲרָמִים</p></blockquote>
<blockquote><p>לָהּ וּמִכְמֹרֶת / עַל פְּנֵי הַמָּיִם: / ישע' יט, ח</p></blockquote>
<br/>
<blockquote><p>נַחֵם מְחַכֶּה / לְיֶשַׁע אֲדֹנָיו</p></blockquote>
<blockquote><p>נָגוּעַ וּמֻכֶּה / מְדֻכָּא מֵעֲוֹנָיו</p></blockquote>
<blockquote><p>וְיוֹם לְיוֹם מְפַכֶּה / נַחַל מֵעֵינָיו</p></blockquote>
<blockquote><p>וְדָמוֹ עֵת יִבְכֶּה / מָהוּל בַּמָּיִם: / שם א, כב</p></blockquote>
