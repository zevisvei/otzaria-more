<h1>[מִפִּלְאִי מְרוֹמוֹת]</h1>
משה אבן עזרא
<blockquote><p>מִפִּלְאִי מְרוֹמוֹת / נִטָּיוּ בְחָכְמָה</p></blockquote>
<blockquote><p>וּמִמּוֹפְתֵי אֲדָמוֹת / נִתְלוּ עַל בְּלִימָה</p></blockquote>
<blockquote><p>יָדַעְתִּי כִּי כֹל תּוּכָל / בְּאוֹן וְעָצְמָה</p></blockquote>
<blockquote><p>וְלֹא יִבָּצֵר / מִמְּךָ מְזִמָּה:</p></blockquote>
<br/>
<blockquote><p>שְׁחָקִים אֲשֶר נִתְלוּ / כְּאֹהֶל נִמְתָּחוּ</p></blockquote>
<blockquote><p>לָעַד לֹא יִבְלוּ / וְגַם לֹא יִזָּחוּ</p></blockquote>
<blockquote><p>וְהָאֲרָצוֹת נִתְלוּ / וּלְמַטָּה שָׁחוּ –</p></blockquote>
<blockquote><p>וְלוּ יְאַו אֵל יִכְלוּ / כֶּעָשָׁן נִמְלָחוּ</p></blockquote>
<blockquote><p>וְעִמָּם הָאֲדָמָה / תִּשָּׁאֶה שְׁמָמָה:</p></blockquote>
<br/>
<blockquote><p>הְכִינוֹתָ פְּגָרִים / וְנָפַחְתָּ נְשָׁמוֹת</p></blockquote>
<blockquote><p>אֵלֶּה לָעֲפָרִים / וְאֵלֶּה לַמְּרוֹמוֹת</p></blockquote>
<blockquote><p>וְכֵן חֵלֶק יְצוּרִים / נְקָמוֹת וּנְעִימוֹת</p></blockquote>
<blockquote><p>וּנְפָשִׁים טְהוֹרִים / יִרְאוּ בְעֵין מְזִימּוֹת</p></blockquote>
<blockquote><p>אוֹתוֹת תַּעֲלֻמָה / הֲכִינוֹתָ בְּחָכְמָה:</p></blockquote>
<br/>
<blockquote><p>יְגִיעַ כַּפֶיךָ / אֳלוֹהַּ אַל תְּחַבֵּל</p></blockquote>
<blockquote><p>וּבַעֲלוֹת קִצְפֶךָ / מְלַאכְתְּךָ אַל תְּנַבֵּל</p></blockquote>
<blockquote><p>הֲיֶחֱרֶה אַפֶּךָ / עַל סָס כְּצִיץ נוֹבֵל</p></blockquote>
<blockquote><p>אוֹ תִקַּח נְקָמָה / מֵעָפָר וְרִמָּה?!</p></blockquote>
