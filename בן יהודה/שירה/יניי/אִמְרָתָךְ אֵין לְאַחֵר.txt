<h1>אִמְרָתָךְ אֵין לְאַחֵר</h1>
יניי
<blockquote><p>אִמְרָתָךְ / אֵין לְאַחֵר / בְּ[רִיתָ]ךְ / אֵין לְבַלּוֹת</p></blockquote>
<blockquote><p>גְּזֵירָתָךְ / אֵין לִגְרֹעַ / דִּבְרָתָךְ / אֵין לִדְחוֹת</p></blockquote>
<blockquote><p>הַדְרָתָךְ / אֵין לְהַרְהֵר / וִתּוּרָךְ / אֵין לֹאמַר</p></blockquote>
<blockquote><p>זִכְרוֹנָ[ךְ] / אֵין לְזַיֵּף / חֲנִיָּתָךְ / אֵין לַחֲקוֹר</p></blockquote>
<blockquote><p>טִירָתָךְ / אֵין לִטְפֹּף / יע… / [אֵין לֵידַ]ע</p></blockquote>
<blockquote><p>כּוֹחָךְ / אֵין לכ… / לִינָתָךְ / אֵין לִלְחֹץ</p></blockquote>
<blockquote><p>ממ…. / [אֵין לִ]מְרוֹד / נִיצְחָ[ךְ] / אֵין לְנַסּוֹת</p></blockquote>
<blockquote><p>סְפוּנָךְ / אֵין לִסְקֹר / ע….. / אֵין לַעֲלוֹת</p></blockquote>
<blockquote><p>פִּלְאָךְ / אֵין לִפְלוֹג / צוּרָתָךְ / אֵין לָצוּר</p></blockquote>
<blockquote><p>ק…תך / אֵין לִקְרַב / רוּחָךְ / אֵין לִרְכַּב</p></blockquote>
<blockquote><p>שִׁלְטוֹנָךְ / אֵין לְשַׁעֵר / [תִּלּ]וּלָךְ / אֵין לִתְלוֹת.</p> <br/></blockquote>
