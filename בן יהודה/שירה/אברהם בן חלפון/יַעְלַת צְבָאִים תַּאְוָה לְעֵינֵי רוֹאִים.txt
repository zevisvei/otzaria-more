<h1>יַעְלַת צְבָאִים תַּאְוָה לְעֵינֵי רוֹאִים</h1>
אברהם בן חלפון
<blockquote><p>יַעְלַת צְבָאִים / תַּאְוָה לְעֵינֵי רוֹאִים:</p></blockquote>
<blockquote><p>בַּת נֶאֱמָנָה / תִּדְמֶה אֱלֵי אַיֶּלֶת</p> <p>בִּסְעִיף אֱמוּנָה / צוֹמְחָה וּמִתְגַּדֶּלֶת</p> <p>וּכְמוֹ לְבָנָה / בַּעְלַת יְפִי מַשְׂכֶּלֶת</p></blockquote>
<blockquote><p>יֵשׁ בָּהּ פְּלָאִים / כִּי תַחֲבוֹשׁ תַּחְלוּאִים:</p> <br/></blockquote>
<blockquote><p>רֵיחָהּ יְפָרֵשׁ / לָעוֹבְרִים מִמֶּרְחָק</p> <p>מַגִּיד לְדוֹרֵשׁ / בַּעְדָהּ: הֲזוֹ בַת יִצְחָק</p> <p>עָנָף כְּשֹׁרֶשׁ / יֹאמַר: רְאֵה, אֵיךְ הוּחָק</p> <p>כִּי בַקְּרוּאִים / הוּא רֹאשׁ וְשָׂר לִנְשִׂיאִים:</p> <br/></blockquote>
<blockquote><p>גַּן אַהֲבָתָהּ / הֵנֵץ וְנָתַן רֵיחוֹ</p> <p>קוּם רֵד לְבֵיתָהּ, / דּוֹד, וַחֲנֵה נִכְחוֹ</p> <p>שָׁמְרָה בְרִיתָהּ / לֵךְ בּוֹא רְקַח אֶת רִקְחוֹ</p> <p>טוּבָיו כְּלוּאִים / מִזָּר, וְלָךְ מוּבָאִים:</p> <br/></blockquote>
<blockquote><p>לִקְחָהּ פְּגָרִים / יָקִים, וְיָשִׁיב נֶפֶשׁ</p> <p>מַמְצִיא אֲסִירִים / בִּכְלוּא אֲהָבִים חוֹפֶשׁ</p> <p>וּלְלֵב מְצֵרִים / מַעְלֶה צֳרִי וָנֶפֶשׁ</p> <p>וּצְמָא צְמֵאִים / מַשְׁבִּיר בְּמִלִּים נָאִים:</p> <br/></blockquote>
<blockquote><p>בָּא עֵת יְדִידוֹת / לִרְווֹת בְּדַדֵּי אַהְבָה</p> <p>לֵךְ אִישׁ חֲמוּדוֹת / בּוֹא סֹב בְּרֹאשׁ עַם וּשְׁבָה</p> <p>כֵּן קֵץ עֲתִידוֹת / יָחִישׁ לְהַצְבִּיא צָבָא</p> <p>לִדְבַר נְבִיאִים / חוֹכִים וְלָאֵל קוֹרְאִים:</p> <br/></blockquote>
