<h1>ארץ חפץ נדחיך</h1>
יצחק בר שֵׁשֶׁת בֶּרְפֶת
<h2 id="אֶרֶץחֵפֶץנִדָּחַיִךְרבייצחקברששתהריבש">[אֶרֶץ חֵפֶץ, נִדָּחַיִךְ] / רבי יצחק בר ששת (הריב"ש)</h2>
<blockquote><p>אֶרֶץ חֵפֶץ, נִדָּחַיִךְ / יוֹם יוֹם שוֹאֲלִים בִּגְלָלֵךְ,</p></blockquote>
<blockquote><p>צוֹפִים שוֹאֲפִים לִמְנוּחַיִךְ / לֵאמֹר ‘בֵּית ה’ נֵלֵךְ!'</p></blockquote>
<blockquote><p>שָלוֹם רַב, קִרְיַת מֶלֶךְ, / שָלוֹם מֵאֵת צִבְאוֹתַיִךְ,</p></blockquote>
<blockquote><p>יְהִי שָלֹום בְּחֵילֵךְ, / שַלְוָה בְּאַרְמְנוֹתַיִךְ!</p></blockquote>
<br/>
<blockquote><p>5          צִיּוֹן הָעִיר הַהֻלָּלָה / מֵאָז לִצְבִי וּלְכָבוֹד,</p></blockquote>
<blockquote><p>עָלַיִךְ רוּחִי חֻבָּלָה, / עָלַיִךְ לִבִּי יִדֹּד.</p></blockquote>
<blockquote><p>יִדֹּדוּן בִּשְבִי וּנְדוֹד, / יִדֹּדוּן כָּל מַחֲנוֹתַיִךְ,</p></blockquote>
<blockquote><p>כִּי אֵש אָכְלָה הֵיכָלֵךְ / וַתֹּאכַל יְסוֹדוֹתַיִךְ.</p></blockquote>
<br/>
<blockquote><p>חִשַּבְתִּי יָמִים מִקֶּדֶם / דוֹרוֹת וּשְנוֹת עוֹלָמִים,</p></blockquote>
<blockquote><p>10          פָּרַצְתִּי יָמָּה וָקֶדֶם / מָשַלְתִּי עַל הָעַמִּים.</p></blockquote>
<blockquote><p>כַּמָּה לִי אֵשֵב מַשְמִים, / צִיּוֹן, עַל שֹד חוֹמוֹתַיִךְ?</p></blockquote>
<blockquote><p>לָמָּה קוֹדֵר אֶתְהַלֵּךְ / עַל תִּלֵּךְ וְשוֹמְמוֹתַיִךְ?</p></blockquote>
<br/>
<blockquote><p>קִרְיַת מָשׂוֹשׂ עִיר עָז לָנוּ / מָלֵאת קִצֵּךְ וּצְבָאֵךְ,</p></blockquote>
<blockquote><p>כִּי תַם פֶּשַע רַב הֶגְלָנוּ, / תַּם עָוֹן, נִרְצָה חֶטְאֵךְ.</p></blockquote>
<blockquote><p>15          צוּרֵךְ לָאוֹר יוֹצִיאֵךְ, / יִרְפָּאֵךְ מִמַּכּוֹתַיִךְ,</p></blockquote>
<blockquote><p>כִּי נִחַם אֶת הֵיכָלֵךְ, נִחַם אֶת כָּל חָרְבוֹתַיִךְ.</p></blockquote>
<br/>
<blockquote><p>בַּת צִיּוֹן, הַשְפִּילִי שֵבִי / עַל כָּבוֹד גָּלָה מִמֵּךְ,</p></blockquote>
<blockquote><p>עַל בָּנִים הָלְכוּ בַשֶּבִי / מָה אַשְוֶה לָךְ וַאֲנַחֲמֵךְ?</p></blockquote>
<blockquote><p>אֵין שַׂר, אֵין שֵבֶט תּוֹמֵךְ, / אֵין מֵקִים יְרִיעוֹתַיִךְ,</p></blockquote>
<blockquote><p>20          שִיתִי כַלַּיִל צִלֵּךְ, / חִגְרִי שַׂק בְּחוּצוֹתַיִךְ!</p></blockquote>
<br/>
<blockquote><p>שוּבִי יוֹנַת חֵן, נֶעֱצָבָה / יוֹנָה מֵאֶרֶץ אַשּוּר,</p></blockquote>
<blockquote><p>שִׂמְחִי, כִּי אָשִׂים בָּעֲרָבָה / בְרוֹש תִּדְהָר וּתְאַשּוּר.</p></blockquote>
<blockquote><p>מֵחֶרְמוֹן וּשְׂנִיר יָשוּר / נִדְּחֵךְ וּנְפוּצוֹתַיִךְ,</p></blockquote>
<blockquote><p>הַרְחִיבִי מְקוֹם אָהֳלֵךְ / וִירִיעוֹת מִשְכְּנוֹתַיִךְ</p></blockquote>
<br/>
<p>(ברצלונה, פ“ו [1326] – אלג’יר, קס”ח [1407])</p>
