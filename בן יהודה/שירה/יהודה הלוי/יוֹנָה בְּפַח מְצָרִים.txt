<h1>יוֹנָה בְּפַח מְצָרִים</h1>
יהודה הלוי
<p>אהבה. חתימה: יהודה</p>
<blockquote><p>יוֹנָה בְּפַח מְצָרִים / בֵּין אוֹיְבִים וְצָרִים, / קִנָּהּ עָזְבָה –</p> <p>לָמָּה בְּאַף נְטַשְׁתָּהּ / אַחַר אֲשֶׁר מְשַׁכְתָּהּ / בַּעְבוֹת אַהֲבָה?</p> <br/></blockquote>
<blockquote><p>הוֹמִים לְנוֹד יְדִידִי / מֵעַי וְכוֹאֲבִים,</p> <p>אֵיךְ יַעֲטוּ רְדִידִי / צָרִים וְאוֹיְבִים?</p> <p>אֵיךְ נָשְׂאוּ רְבִידִי / בּוֹגְדִים וְאוֹרְבִים?</p> <p>אֵיךְ רַעֲיָה עֲרוּמָה / מִסּוּת יְקָר עֲרֻמָּה / דּוּמָה שָׁכְבָה,</p> <p>אַחַר אֲשֶׁר שְׂמֹאלוֹ / לִי מַחֲסֶה, וְדִגְלוֹ / עָלַי אַהֲבָה?</p> <br/></blockquote>
<blockquote><p>וּזְכֹר בְּרִית נְעוּרִים, / דּוֹדִי וְאוֹהֲבִי!</p> <p>אֵיךְ תַּעֲזֹב לְזָרִים / נַחְלָתְךָ שְׁבִי?</p> <p>אוֹ אִם עֲלֵי בְתָרִים / תִּבְרַח כְּמוֹ צְבִי –</p> <p>אֵלַי נְטֵה, אֲדוֹנִי! / אַךְ אִם בְּרֹב עֲווֹנִי / עִירִי חָרְבָה –</p> <p>סוּרָה שְׁכֹן בְּקִרְבִּי! / הֵן אָהֳלָךְ בְּלִבִּי / רָצוּף אַהֲבָה.</p> <br/></blockquote>
<blockquote><p>דּוֹרְשֵׁי פְדוּת וְצִלָּם / נִגְלָה וְנֶחֱשַׂף,</p> <p>הוֹד יַעֲטוּ, וְחִילָם / צוּר יַחֲבֹשׁ, וְאַף</p> <p>יִכְבֹּשׁ, וְלֹא לְעוֹלָם / יִקְצֹף וְיֶאֱנַף.</p> <p>שָׂרִיד יְצַו לְהַשְׁאִיר / מֵהַשְּׁבִי, וְיָאִיר / שֶׁמֶשׁ עָרְבָה,</p> <p>עֶזְרִי, חֲמֹל עֲלֵי עִיר / קָדְשִׁי, וְשׁוּב וְהָעִיר / אֶת הָאַהֲבָה!</p> <br/></blockquote>
<blockquote><p>הַעוֹד אֲקַו לְחַלּוֹת / הַר הַצְּבִי, וְעוֹד</p> <p>אָשִׁיר יְמֵי כְלוּלוֹת / שִׁיר אַהֲבָה לְדוֹד?</p> <p>מַה תִּפְרְצוּ לְגַלּוֹת / תַּכְלִית יְמֵי נְדוֹד?</p> <p>אַשְׁבִּיעֲכֶם בְּעֹפֶר / הִרְחִיק נְדֹד וְהֵפֵר / אֶת הָאַחֲוָה,</p> <p>אִם הַבְּכִי תָעִירוּ, / רֵעַי, וְאִם תְּעוֹרְרוּ / אֶת הָאַהֲבָה.</p> <br/></blockquote>
