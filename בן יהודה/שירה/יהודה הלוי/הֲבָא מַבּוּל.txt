<h1>הֲבָא מַבּוּל</h1>
יהודה הלוי
<p><a href="#fn:1" id="fnref:1" title="see footnote"><sup>1</sup></a></p>
<blockquote><p>הֲבָא מַבּוּל וְשָׂם תֵּבֵל חֳרָבָה<a href="#fn:2" id="fnref:2" title="see footnote"><sup>2</sup></a>,</p> <p>וְאֵין לִרְאוֹת פְּנֵי אֶרֶץ חֲרֵבָה<a href="#fn:3" id="fnref:3" title="see footnote"><sup>3</sup></a>,</p> <p>וְאֵין אָדָם וְאֵין חַיָּה וְאֵין עוֹף?</p> <p>הֲסָף הַכֹּל וְשָׁכְנוּ מַעֲצֵבָה<a href="#fn:4" id="fnref:4" title="see footnote"><sup>4</sup></a>?</p> <p>וּבִרְאוֹת הָר וְשׁוּחָה לִי מְנוּחָה<a href="#fn:5" id="fnref:5" title="see footnote"><sup>5</sup></a>,</p> <p>וְאֶרֶץ הָעֲרָבָה לִי עֲרֵבָה.</p> <p>וְאַשְׁגִּיחַ לְכָל־עֵבֶר וְאֵין כֹּל,</p> <p>אֲבָל מַיִם וְשָׁמַיִם וְתֵבָה.</p> <p>וְלִוְיָתָן בְּהַרְתִּיחוֹ מְצוּלָה –</p> <p>וְאֶחְשֹׁב כִּי תְהוֹם יַחְשֹׁב לְשֵׂיבָה<a href="#fn:6" id="fnref:6" title="see footnote"><sup>6</sup></a></p> <p>וְלֵב הַיָּם יְכַחֵשׁ בָּאֳנִיָּה,</p> <p>כְּאִלּוּ הִיא בְּיַד הַיָּם גְּנֵבָה<a href="#fn:7" id="fnref:7" title="see footnote"><sup>7</sup></a>.</p> <p>וְיָם יִזְעַף וְנַפְשִׁי תַעֲלֹז <a href="#fn:8" id="fnref:8" title="see footnote"><sup>8</sup></a> כִּי</p> <p>אֱלֵי מִקְדַּשׁ אֱלֹהֶיהָ קְרֵבָה:</p></blockquote>
<ol><li id="fn:1"><p>שיר על הים. <a href="#fnref:1" title="return to body"> ↩</a></p></li> <li id="fn:2"><p><strong>הבא מבול</strong>: המשורר אינו רואה עוד היבשה ושואל הבא מבול <strong>חרבה</strong>: חרבן <a href="#fnref:2" title="return to body"> ↩</a></p></li> <li id="fn:3"><p><strong>חרבה</strong>: יבשה. <a href="#fnref:3" title="return to body"> ↩</a></p></li> <li id="fn:4"><p><strong>הסף</strong>: התם <strong>מעצבה</strong>: ביגון, מליצה עפ“י ישעיה פ”ג פסוק י"א שם ושכנו מעצבה. <a href="#fnref:4" title="return to body"> ↩</a></p></li> <li id="fn:5"><p><strong>ובראות הר</strong> וכו': כשאני רואה אפילו הר ואפילו <strong>שׁוחה</strong>, בור עמוק אני שמח ולבי נח. <a href="#fnref:5" title="return to body"> ↩</a></p></li> <li id="fn:6"><p><strong>ולויתן</strong>: התנין הגדול. <strong>מצולה</strong>: תהום. <strong>לשּיבה</strong>: מפני הגלים המכים קצף הדומים לתלתלי שבה. והקצף עולה על־ידי זה שלויתן מרתיח את המצולה. <a href="#fnref:6" title="return to body"> ↩</a></p></li> <li id="fn:7"><p><strong>ולב הים</strong> וכו': הים מכסה את האניה וכאלו בולע אותה אל תוך לבו ומכחש לאמר, לא לקחתי מאומה, כאלו היתה בידו גנבה. <a href="#fnref:7" title="return to body"> ↩</a></p></li> <li id="fn:8"><p><strong>וים יזעף</strong> וכו': בשעה שהים זועף תעלז נפשי. <a href="#fnref:8" title="return to body"> ↩</a></p></li></ol>
