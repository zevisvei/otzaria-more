<h1>הִגִּיעַ זְמַן קֵץ לִפְקֹד</h1>
יהודה הלוי
<p>לשמואל הנגיד</p>
<blockquote><p>הִגִּיעַ זְמַן קֵץ לִפְקֹד / אֲרִיאֵל –</p> <p>וַיּוֹסֶף אֲדֹנָי קְרֹא עוֹד: / שְׁמוּאֵל!</p> <br/></blockquote>
<blockquote><p>הַקּוֹל עַד לְשִׁנְעָר עָלָה – / וְיָרַד</p> <p>לַמַּעְרָב, וְקוֹלָהּ צָהֲלָה / סְפָרַד,</p> <p>וּלְבָבִי לְפָגְעוֹ כָּלָה / וְחָרַד,</p> <p>כִּי אֶעְמֹד בְּבֵיתוֹ כַּעְמֹד / בְּבֵית אֵל,</p> <p>גַּם אֶחְזֶה בְּהוֹד פָּנָיו הוֹד / אֲרִיאֵל.</p> <br/></blockquote>
<blockquote><p>לִבּוֹ בַזְּמָן כָּאְנִיּוֹת / סוֹחֵר,</p> <p>חֶלְקוֹ יַעֲזֹב מַשְׂכִּיּוֹת / לְאַחֵר –</p> <p>אַךְ הוּא סוֹד נְשָׁמוֹת חַיּוֹת / מְשַׁחֵר.</p> <p>לֹא יַחְמֹד יְקָר, אַךְ יַחְמֹד / כְּבוֹד אֵל</p> <p>לִהְיוֹת סוֹד לְעוֹלָם וִיסוֹד / לְהַרְאֵל.</p> <br/></blockquote>
<blockquote><p>יָצָא עַל גְּבוּל מִצְרַיִם / כְּיוֹסֵף</p> <p>אֶת נִדְחֵי יְרוּשָׁלַם / לְאַסֵּף –</p> <p>אַךְ כָּל צוֹרְרֵי אֶפְרַיִם / יְשַׁסֵּף,</p> <p>כִּי מִפְּנֵי חֲצָצָיו יִדּוֹד / מְגָאֵל,</p> <p>עַד לִהְיוֹת קְבוּצָיו כָּל דּוֹד / וְגוֹאֵל.</p> <br/></blockquote>
<blockquote><p>כֹּל פָּעַל לְמַעְנוֹ, עַנְוָה / וְגַאְוָה,</p> <p>חָלַק הַזְּמָן בֵּין חֶדְוָה / וּמִצְוָה,</p> <p>גַּם נַפְשׁוֹ כְּמוֹ כֵן דָּוָה / וְרָוָה:</p> <p>הִכָּבְדוֹ בְּרָב עָם כִכְבוֹד / לְמוּאֵל –</p> <p>אַךְ לִפְנֵי אֱלֹהָיו יִקֹּד / כְּשׁוֹאֵל.</p></blockquote>
