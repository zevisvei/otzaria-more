<h1>[יודעי מכאובי]</h1>
יהודה הלוי
<blockquote><p>יוֹדְעֵי מַכְאוֹבִי, / אַל תִּתְמַהְמֶהוּ / מֵהָבִיא</p> <p>צָרְיִי מִגִּלְעָד ‑ / הוּא אֵלִיָּהוּ / הַנָּבִיא!</p> <br/> <p>הַעתִּירוּ בַּעְדִי / אֶל אָבִי – אוּלַי / יֵעָתֵר</p> <p>וִיגַלֶּה סוֹדִי / וְחָזוֹן מַשְׂכִּילַי / יִפָּתֵר.</p> <p>עַד כַּמָּה, דּוֹדִי, / מִלִּפְנוֹת אֵלַי / תִּסָּתֵר?</p> <p>וּצְבִי מָאַס בִּי ‑ / מָתַי אַעְלֵהוּ / בֵּית אָבִי,</p> <p>הַדּוֹד הַנּוֹעָד / לִהְיוֹת מַחְנֵהוּ / בִּלְבָבִי?</p> <br/> <p>וַיְבוֹשֵׁשׁ רִכְבּוֹ, / תַּמּוּ מוֹעֲדָיו / וָאָיִן,</p> <p>מֵאַיִן יָבֹא / עֵזֶר לִידִידָיו, / מֵאָיִן?</p> <p>אֶצְמָא אֶל נִיבוֹ, / כִּי טוֹבִים דֹּדָיו / מִיָּיִן.</p> <p>מַלְכִּי וּכְרוּבִי, / כַּסְפִּי אֶעְדֵּהוּ / וּזְהָבִי,</p> <p>וּבְבֵית הַוַּעַד / יֵינִי אַשְׁקֵהוּ / וַחְלָבִי.</p> <br/> <p>דּוֹדִי, זָנַחְתָּ / אוֹתִי שׁוֹמֵמָה / תּוֹךְ מוֹקְשִׁי</p> <p>אָכֵן שָׁלַחְתָּ / פֶּרֶא בֶּן‑אָמָה / לַחָפְשִׁי,</p> <p>וֶאְדוֹם שִׂמַּחְתָּ, / כִּי יָדוֹ רָמָה / עַל קָדְשִׁי.</p> <p>שִׂמַּחְתָּ אוֹיְבִי / וַתַרְכִּיבֵהוּ / מֶרְכָּבִי,</p> <p>תֵּן מָתְנָיו הַמְעֵד / עַד תַּשְׁכִּיבֵהוּ / מִשְׁכָּבִי!</p> <br/> <p>הַלֵּב לֹא יֵרַךְ, / אִם הַקֵּץ אָרַךְ / וַיִּרְבֶּה,</p> <p>אַךְ בָּךְ יִתְבָּרַךְ / שְׁאֵרִית עֶדְרָךְ / הַנִּשְבֶּה.</p> <p>גֵר מֵאָז נֶעְרַךְ / וְכוֹכָב דָּרַךְ / לֹא יִכְבֶּה.</p> <p>אִם עָבַר טובִי / וְעֵינַי כָּהוּ ‑ / כּוֹכָבִי</p> <p>לֹא יִכְבֶּה לָעַד, / כִּי הַנֵּר הַהוּא / מַעְרָבִי!</p></blockquote>
