<h1>יַד מַעֲלָלַי תַּכְבִיד כְּבָלַי</h1>
יהודה הלוי
<blockquote><p>יַד מַעֲלָלַי / תַּכְבִיד כְּבָלַי</p></blockquote>
<blockquote><p>וְרֹעַ פְּעָלַי / מִנִּי וְאֵלָי –</p></blockquote>
<blockquote><p>מַה תִּשְׁתּוֹחֲחִי, נַפְשִׁי, וַתֶּהֱמִי עָלָי?</p></blockquote>
<br/>
<blockquote><p>הַבְלֵי נְעוּרִים / בְּלִבִּי קְשׁוּרִים,</p></blockquote>
<blockquote><p>וְשׁוֹכְנֵי קְבָרִים / אֵינָם זְכוּרִים,</p></blockquote>
<blockquote><p>כְּהֶמְיַת נְבָלָי / וְהַבְלֵי הֲבָלָי.</p></blockquote>
<br/>
<blockquote><p>וּמֶה לְנִשְׁמָתִי / עַל אַדְמָתִי?</p></blockquote>
<blockquote><p>אַךְ כְּלִמָּתִי / כְּפִי אַשְׁמָתִי,</p></blockquote>
<blockquote><p>וְכֵן מַהֲלָלָי / לְפִי מַעֲלָלָי.</p></blockquote>
<br/>
<blockquote><p>דּוֹרְשָׁה תַאֲוָה / וְנוֹטְשָׁה תִקְוָה,</p></blockquote>
<blockquote><p>הַיּוֹם רָוָה ‑ / וּמָחָר דָּוָה,</p></blockquote>
<blockquote><p>וְהַיּוֹם – אַחְלַי, / וּמָחָר – אַלֲלַי!</p></blockquote>
<br/>
<blockquote><p>הֲלֹא אִם שְׁבִילֵךְ / יִישַׁר לְפוֹעֲלֵךְ</p></blockquote>
<blockquote><p>יְהִי שְׂכַר פָּעֳלֵךְ / כּיַד הַמֶּלֶךְ</p></blockquote>
<blockquote><p>וְאֵין מִשְׁאָלַי / לִפְנֵי כֵלָי.</p></blockquote>
